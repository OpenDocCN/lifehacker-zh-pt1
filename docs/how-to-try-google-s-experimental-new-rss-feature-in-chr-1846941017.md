# 如何在 Chrome Canary 中尝试谷歌的实验性新 RSS 功能

> 原文：<https://lifehacker.com/how-to-try-google-s-experimental-new-rss-feature-in-chr-1846941017>

谷歌正在 Android 平台的 Chrome Canary 中测试一项基于 RSS 的实验性“关注”功能。谷歌表示，该功能目前只是一个“实验”，并将根据用户和开发者的反馈决定是否公开实施该功能，但这对像我这样的 RSS 爱好者来说似乎是个好消息。

Watch

如果你对 RSS 不熟悉，这是一种每当网站发布新内容时就会更新的网络订阅源，RSS 阅读器将所有这些订阅源收集到一个列表中，你可以随时查看。RSS 阅读器不像以前那么受欢迎了，像谷歌新闻和火狐的内置 RSS 提要这样的众所周知的选项在几年前就已经停止了。尽管如此，RSS 仍然是许多人与他们喜欢的新闻和博客保持联系的首选方式。您甚至可以将播客和 YouTube 频道添加到 RSS 提要中。这比直接去网站获取最新更新要容易得多，也比在 Twitter、脸书或 Reddit 上获取头条新闻更容易分散注意力。

目前，谷歌的 Follow list 只是一个实验，谷歌希望将“在 Chrome 中建立用户和网络出版商之间更深的互动”，但最终版本取决于用户和开发者的反馈。

想要参加 Chrome 后续功能测试的人需要 [安装 Android 版 Chrome Canary](https://www.google.com/chrome/canary/)。正如我们经常提醒的那样，Chrome Canary 是浏览器最不稳定的版本，充满了实验性和未完成的功能，容易崩溃。也就是说，Canary 是作为一个独立于其他版本 Chrome 的应用程序安装的，因此如果它运行不佳，你可以随时切换到不同的浏览器。

未来几周，Follow 功能将慢慢向 Chrome Canary Android 用户推出，但并非所有用户都可以使用它(我还没有看到)。但是一旦跟随功能*可用*，导航到一个网站，然后点击右上角的三点图标。点击菜单底部的 **"+关注"**，将网站添加到您的关注列表中。每当您打开新标签时，来自被关注网站的最新更新将出现在新的“关注”订阅源中。

如果你像我一样是一个 RSS 转换者，但不想一直使用 Chrome Canary，有许多第三方 RSS 网站、应用程序和每个浏览器的插件，包括我的首选 RSS 阅读器 Feedly。
