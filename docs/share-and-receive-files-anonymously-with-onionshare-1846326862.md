# 通过“OnionShare”匿名共享和接收文件

> 原文：<https://lifehacker.com/share-and-receive-files-anonymously-with-onionshare-1846326862>

无论你是间谍还是极客，有很多方法可以在不暴露任何人身份的情况下与其他人分享和接收文件。无论你需要这种保密的原因是什么，无论是作为主机还是提交者，都很容易为你的数据建立一个虚拟的转储场——绝密的或平凡的。



流行的[**OnionShare**](https://onionshare.org/)应用程序的全新版本前几天发布了，我建议如果你想与朋友或网络陌生人建立秘密联系，不妨试一试。这款开源应用利用 Tor 的力量来管理转账，基本上保证了人们的匿名性——只要你聪明，比如说，不使用你现实生活中的脸书账户来分享你基于 OnionShare 的 dropbox 的链接。

首先，下载并安装免费的 OnionShare 应用程序(适用于 Windows/macOS/Linux)。启动它，你将自动开始连接 Tor 网络。一旦完成，你就可以在 OnionShare 的主屏幕上选择几个不同的选项:

OnionShare 2.3 中的一个有趣的升级是，这些功能现在都在自己的标签中启动。在之前版本的应用中，你不能选择同时接收文件*和发送文件给其他人。现在，你可以同时做所有的事情——文件共享、网站托管和匿名聊天。*

如果您想为收到的文件设置一个转储区，只需点击**接收文件**下的**“开始接收”**链接。系统会询问您希望将这些文件保存在系统的什么位置:

点击“开始接收模式”，你会得到一个特殊的网址。将其复制并粘贴给人们——最好是你认识的个人，而不是网络论坛——他们将能够使用 Tor 浏览器向你发送文件。

仅此而已。将文件从一个系统发送到另一个系统很简单，但是需要花费一些时间。至少，感觉我通过 Tor 传输的 86MB 文件比你从网上上传/下载文件花费的时间要长得多(正如你所料，考虑到[Tor 如何工作](https://lifehacker.com/what-is-tor-and-should-i-use-it-1527891029) )。

点击 OnionShare 右上角的小向下箭头图标，查看人们在何时向您发送了什么。您还可以通过单击文件列出大小右侧的图标，快速跳转到文件在系统中的位置:

当你的接收线路运行时，你也可以使用 OnionShare 窗口左上角的 **"+"** 图标打开一个新标签，做些别的事情——比如和别人分享一个文件。从 OnionShare 的主菜单中启动**“共享文件”**功能，将一些东西拖放到窗口中:

你需要点击大的**开始分享**框来实现。当你这样做时，你会得到另一个 OnionShare URL，你需要将它发送给其他人。他们会启动 Tor 浏览器中的 URL 来获取你的文件。

我没有玩过 OnionShare 的**主持网站**或**匿名聊天**功能，因为我对这两个功能都不感兴趣——尤其是后者。另外，你用自己的带宽托管了一个网站，任何人只要有特殊的 URL(当然还有 Tor 浏览器)就可以访问，如果你非常受欢迎，你的 ISP 可能会反对。公平地说，聊天功能听起来比实际情况更加不祥；它实际上启动了一个聊天*服务器*，其他人通过 Tor 浏览器连接到这个服务器。你也是，因为你不能仅仅通过 OnionShare 与连接到你服务器的人聊天；你必须启动 Tor 浏览器才能参与。你所能做的就是聊天——例如，不能共享文件或图片。
