# 如何在信号中发送消失的消息

> 原文：<https://lifehacker.com/how-to-send-disappearing-messages-in-signal-1846012113>

许多消息应用程序可以让你发送消失的消息，但 Signal 处理过程的方法 [是最好的](https://lifehacker.com/for-better-privacy-use-these-apps-instead-of-whatsapp-1845551260) 。如果你正在寻找以隐私为中心的消息传递，或者如果你是埃隆·马斯克的粉丝，这是另一个开始使用加密聊天应用的好理由。

Watch

顾名思义，Signal 的消失信息会在一定时间后自毁。对于对话中的两个用户，已删除的消息将从聊天线程中永久删除。当您发送潜在的敏感信息，并且不希望在不必要的时间内访问这些信息时，这是一个很好的选择。

显然，消失的消息可以增强你的信号对话的隐私性和安全性，但也有局限性。即使消失的消息从两个用户的消息历史中被删除，用户仍然可以在它们被删除之前拍摄一张屏幕截图，或者复制并将它们的内容粘贴到其他地方。如果你的收件人特别聪明，他们可以使用另一种设备来拍摄问题中消失的材料。

虽然你可以在自己的设备 上 [屏蔽 Signal 中的截屏，但没有办法阻止收件人在消息被删除前保存它。](https://support.signal.org/hc/en-us/articles/360043469312-Screen-Security) [只发送私人信息给你信任的联系人](https://lifehacker.com/how-to-make-sure-your-disappearing-signal-messages-actu-1825921690) 。

也就是说，您可以控制消息在被删除之前在聊天中保留多长时间。时间窗口越小，有人成功挽救的可能性就越小。

### 如何在 Signal 中打开消失信息

默认情况下，信号消息不会消失，但是可以使用以下两种方法中的任何一种随时为单独的对话打开或关闭该功能。请注意，消失消息仅适用于同时使用 Signal 的用户之间的一对一对话；它在群聊中不可用，如果对方使用不同的短信应用程序，它也不能使用。

**从快捷菜单中打开消失消息:**

1.  打开信号，然后打开对话或开始新的对话。
2.  轻按对话右上角的三点按钮。
3.  轻按**“消失的信息”**
4.  选择邮件被删除前显示的时间。最快 5 秒，最长 1 周。

**在对话设置中打开消失消息:**

1.  打开一个对话，然后轻按对话窗口顶部的联系人姓名。
2.  在对话菜单中，轻按**“消失的信息”**以开启(或关闭)本次对话
3.  使用菜单选择消息删除时间限制。
