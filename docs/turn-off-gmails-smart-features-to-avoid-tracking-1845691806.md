# 关闭 Gmail 的“智能”功能以避免被跟踪

> 原文：<https://lifehacker.com/turn-off-gmails-smart-features-to-avoid-tracking-1845691806>

谷歌收集你在使用其产品和服务时所做的一切的数据。有时候这些数据会卖给广告商；有时候是为了更方便的使用谷歌的产品。

Watch

例如，谷歌使用来自 Gmail、Chat 和 Meet 的数据来使整个谷歌用户体验更加方便，比如它将如何学习你如何键入和建议听起来(希望)自然的自动邮件回复。

然而，有些人不希望他们的数据被跟踪和存储，不管这会带来什么便利。虽然已经有一些方法可以改变 Gmail 的数据收集行为——稍后会详细介绍——但由于两个新的 Gmail 设置 [将在未来几周推出](https://blog.google/products/gmail/new-settings-smart-features-and-personalization-gmail/) ，退出谷歌的跟踪将很快变得更容易。

首先，用户现在可以选择禁用 Gmail 的“智能功能”，如智能撰写、智能回复等。虽然这些都很方便，但谷歌必须彻底检查你的 Gmail、会议和聊天数据才能让它们工作。

另一项设置允许用户阻止 Gmail、聊天和会议数据在其他谷歌应用中使用。就像禁用智能功能一样，关闭个性化会删除某些便利，如谷歌搜索中的内容建议、谷歌地图中的餐厅推荐或谷歌助手中的自动账单支付提醒。你可能会失去一点帮助，但你至少会让谷歌远离你的收件箱。

### 在设置菜单中关闭 Gmail 的智能功能和个性化选项

这些新权限将在未来几周内自动提示所有 Gmail 用户。也就是说，如果你不想等待，你可以现在就在 Gmail 的设置中更改(一些)你的 Gmail 数据偏好设置。调出 Gmail，然后访问**设置>常规**关闭以下选项:

*   智能作曲
*   智能撰写个性化
*   推动
*   智能回复

接下来，进入**设置>收件箱**。取消选择所有收件箱类别，关闭**“重要性标记”**

您可以通过进入**设置>账户并导入>其他谷歌账户设置来更改您的其他 Gmail 数据的使用方式。**

如果您不知道这些功能会收集您的数据，请不要担心；Gmail 的设置里哪里都没有提到。令人欣慰的是，新的用户选项对智能功能如何工作以及谷歌如何使用它收集的数据更加清晰。
