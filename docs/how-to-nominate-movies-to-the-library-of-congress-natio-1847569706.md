# 如何向国会图书馆国家电影登记处提名电影

> 原文:[https://gizmodo . com/how-to-nominate-movies-to-the-library-of-congress-natio-1847569706](https://gizmodo.com/how-to-nominate-movies-to-the-library-of-congress-natio-1847569706)

每年，国会图书馆都会挑选25部电影作为国家电影登记处的一部分进行保存。去年，图书馆选择承认并保存所有的东西，从经典的《金臂人》(1955年)和《发条橙》(1971年)，到21世纪更近的电影，如《T4》、《怪物史莱克》(2001年)和《黑暗骑士》(2008年)。

但是你知道吗，国会图书馆每年都从公众那里收集建议？公开提名的截止日期快到了，但你仍有时间发表自己的意见。

自1989年开始以来，国家电影登记处已经有了800部电影，国会图书馆保存了一份很长的电影名单，还没有添加进去，如果你想要 [一些想法](https://www.loc.gov/programs/national-film-preservation-board/film-registry/films-not-yet-named-to-the-registry/) 。公开提名的截止日期是9月15日，所以你最好快点拿到你的名单。唯一的限制？电影必须在2011年之前发行。

图书馆在其 [网站](https://www.loc.gov/programs/national-film-preservation-board/film-registry/nominate/) 上解释说:“当图书馆员和电影委员会考虑他们的最终选择时，公众提名起着关键作用”。

“注册标准没有明确禁止电视节目、商业广告、音乐视频或外国作品，但是，设立注册的立法初衷是为了保护美国电影。因此，国家电影保存委员会和国会图书馆馆长首先考虑美国电影，”图书馆解释说。

任何人都可以通过 [在线表格](https://www.research.net/r/national-fim-registry-nomination-form) 提交他们的选择，尽管你每年只能提名50部电影。请不要用数百个条目在邮箱里发垃圾邮件，因为这很有趣。

“注册表旨在反映美国社会和自1890年以来美国电影的丰富多彩。为此，我们强烈鼓励提名全方位的美国电影制作:不仅仅是好莱坞经典或其他知名作品，还包括无声时代的标题、纪录片、先锋、教育和工业电影，以及代表美国文化无与伦比的多样性的电影，无论是在内容上还是在创造美国社会快照的所有人身上:导演、作家、演员、电影摄影师和其他工艺，“图书馆说。

如果你出于某种原因更喜欢平信——也许一位年长的亲戚没有电子邮件，但想参加——参赛作品也可以通过普通的美国邮件发送:
