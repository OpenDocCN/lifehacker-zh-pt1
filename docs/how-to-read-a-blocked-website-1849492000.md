# 如何阅读被封锁的网站

> 原文：<https://lifehacker.com/how-to-read-a-blocked-website-1849492000>

我们经常把互联网想象成一个无限信息的开放来源，但是根据你住在哪里或者你连接了谁的网络，情况并不总是这样。在许多地方，某些网络活动或内容将被彻底屏蔽，我们不只是在谈论烦人的付费网站。例如，您的工作场所或学校网络可能会屏蔽某些网站以防止分心，而一些在线内容受到当地政府的严格监管或限制。然而，如果你有合适的工具，你可以绕过这些封锁。



这些工具可以帮助你颠覆大多数内容块，尽管我们强烈建议你在使用它们之前确保你没有违反任何法律。虽然我们下面介绍的所有内容在美国都是合法的，但是在其他国家使用这些工具——或者使用它们访问非法内容——可能会给你带来很大的麻烦。比如，几年监禁或者巨额罚款。做你的研究，并使用他们在你自己的风险。

阅读被屏蔽网站的最简单方法是用代理连接隐藏你的真实 IP 地址。通过连接到不同地区的代理服务器，你可以访问网飞、YouTube 或 Spotify 等流媒体网站上的区域锁定内容。您连接到的网站将识别代理的 IP 地址，而不是您的 IP 地址。有免费和付费服务 可供选择，付费选项提供更好的隐私功能和更可靠的连接，但免费代理也很好，只要你检查专业评论以验证它们是安全的。

代理增加了你的互联网隐私，使公司(或政府)更难跟踪你的浏览活动，但它们并不完美。代理服务器最适合访问被地区屏蔽的流媒体内容，而不是破坏当地法律或其他高级限制，因为许多组织使用复杂的跟踪或屏蔽技术，你不能只通过简单的代理连接来解决。在这些情况下，VPN 是更好的选择。

我们的关于 VPN 的指南 [以及如何为你和你的设备找到最好、最安全的选择](https://lifehacker.com/how-to-find-a-trustworthy-vpn-1833045522) 是对该技术的一个很好的介绍，但最快的解释是这样的:VPN 就像代理，但有更多的保护措施(如加密连接)。它们可以保护*你设备上的所有*在线活动(甚至你的整个家庭网络)，而不仅仅是你的互联网浏览器，如果你正在下载种子或访问你居住的高度敏感的内容，它们会更加有效。最好的 VPN 是付费产品，我们强烈建议不要使用免费的 VPN，因为它们要么质量很差，要么完全是恶意软件的骗局。

也就是说，VPN 也不是完美的，即使是最好的 [付费服务也有问题](https://lifehacker.com/avoid-these-no-log-vpn-services-that-leaked-millions-of-1844445123) 或者可以意外停止工作( [)确保你的 VPN 有一个终止开关](https://lifehacker.com/do-you-need-a-vpn-kill-switch-1847431876) ！)，有些防火墙会识别并阻止 VPN 连接。有一些被称为“分散”VPN(dVPN)的专用 VPN 可以解决这个问题，它们通过分散网络传播您的连接，并创建更可信的欺骗 IP 地址来绕过超级严格的过滤器，但这些对大多数人来说不是一个选项，因为它们需要基于加密的支付(如果您感兴趣，请查看如何使用极客的 dVPN 指南 )。

相反，您应该尝试使用 TOR。TOR(或洋葱浏览器)本质上是一个为匿名、打包层(懂吗？)变成了一个大多数互联网用户可以自己安装的简单工具。查看 TOR 上的 [指南](https://lifehacker.com/what-is-tor-and-should-i-use-it-1527891029) 和 [如何使用它](https://lifehacker.com/how-to-anonymize-your-browsing-with-a-tor-powered-raspb-1793869805) 了解更多信息，但是如果你已经使用了桌面浏览器，TOR 将是不言自明的。

TOR 有自己的警告，如连接速度慢和 [减少了恶意软件保护](https://lifehacker.com/dont-use-tor-right-now-if-youre-working-from-home-1842318515) ，这限制了它的可用性(最好坚持使用代理或 VPN 进行流媒体、下载或 P2P 文件传输)，但 TOR 的主要好处是网站——以及其他任何监视你浏览活动的人，如你的 ISP、广告商或政府组织——将无法跟踪你。或者至少，他们会有一段*真正*艰难的时间去弄清楚你是谁，你住在哪里。这使得它成为一般网络浏览的绝佳选择——甚至是 [访问黑暗网络](https://lifehacker.com/how-to-browse-the-dark-web-if-youre-shady-or-just-curi-1848643880) 。
