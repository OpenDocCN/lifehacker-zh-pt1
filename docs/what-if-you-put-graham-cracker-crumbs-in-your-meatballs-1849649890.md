# 如果你在肉丸里放全麦饼干屑会怎么样？

我主要是将全麦饼干与奶酪蛋糕和课后零食联系在一起。传统上，我没有把和肉丸联系在一起，更不用说火腿丸了，这是一种爱荷华州的美食 [我几天前才知道](https://twitter.com/clairelizzie/status/1579877692333518849) 。

火腿球就像它们听起来的那样——由磨碎的火腿和磨碎的牛肉卷成的球。很简单，但真正的难点在于面包屑，它们不是由不新鲜或干燥的面包制成的，而是由全麦饼干制成的。(它们还涂有番茄和红糖釉。我爱爱荷华州。)“那不是挺甜的吗？”你问。是的。是的，它是。但是糖、盐和脂肪放在一起味道很好。我认为全麦面粉的坚果味道会和肉搭配得很好。(我是对的。)

我没有开发自己的火腿丸子(已经有做了)，而是决定接受这个提示，在简单的肉丸食谱中使用全麦饼干，用50/50的碎牛肉和猪肉混合，加上一些犹太盐、白胡椒和伍斯特郡酱。我没有把面包屑泡在牛奶里——火腿球的食谱没有要求这么做，而且看起来也没有必要，因为它们超级好吃。我把所有的东西混合在一起， [把混合物](https://lifehacker.com/this-is-the-easiest-fastest-way-to-portion-your-homema-1849385658) 分成小球，然后用平底锅在黄油里煎，直到它们变成棕色并熟透。

他们令人难以置信。全麦饼干面包屑做了他们的工作，而且做得很好。肉丸多汁鲜嫩，然而在烹饪时保持了它们的形状，即使用锅铲在锅里滚来滚去。是的，它们有点甜。

事实上，第一批T1对我来说有点太甜了，这就是为什么要添加T2伍斯特郡酱的原因。凤尾鱼的腥味和罗望子的唐味抵消了烘焙香料和糖的味道，创造出一种平衡的，有人可能会说是“细致入微”的肉球。

我有一个食谱，但是你可以自己随意摆弄肉和全麦饼干屑。试着用一半的面包屑代替全麦饼干，或者全力以赴。我不会尝试用诺娜的肉丸食谱做这个动作(老实说，我甚至不会和诺娜讨论这个话题)，但我认为用瑞典肉丸即兴重复会很合适。

## 香甜可口的全麦饼干肉丸(约5大份)

#### 成分:

*   1磅碎牛肉
*   1磅碎香肠
*   1杯全麦饼干屑(大约7块饼干)
*   1个鸡蛋
*   3/4杯牛奶
*   2茶匙伍斯特郡酱
*   1茶匙粗盐
*   3/4茶匙白胡椒
*   油炸用黄油

在一个大碗中混合除黄油以外的所有材料。按照你想要的方式分割肉丸——我使用这种方法——但是不要让它们的直径超过1 1/4英寸。

在不粘锅中用中火融化一汤匙黄油。一旦黄油开始起泡，加入肉丸，分批加入，以免挤到肉丸。将所有的面都煎至焦黄，中间没有粉红色。(你将不得不牺牲一个肉丸来检查它。厨师的生活也是如此。)重复，直到你把所有的丸子都煮熟，或者你想吃多少就吃多少。(肉丸混合物在冰箱里保存得很好。)和锅里的棕色黄油一起上桌。
