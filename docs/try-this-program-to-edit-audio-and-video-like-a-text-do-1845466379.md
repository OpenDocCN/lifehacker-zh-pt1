# 使用 Descript 像编辑文本文档一样编辑音频和视频

> 原文：<https://lifehacker.com/try-this-program-to-edit-audio-and-video-like-a-text-do-1845466379>

学习编辑视频和音频很难，复杂的软件界面也不会让事情变得更容易。如果这符合你的编辑体验，你可能想试试 [Descript](https://www.descript.com) ，这是一个用于 Windows 和 Mac 的桌面视频和音频编辑程序，可以转录视频，让你只需移动文本就可以修剪、重新排列甚至更新剪辑。



我查看了 Descript 的免费版本，并通过附带的教程项目预览了它的一些高级功能，我肯定看到了它对播客、YouTubers 和其他内容创作者的吸引力——只要你愿意为最好的功能付费。但是在我们去那里之前，让我们回顾一下 Descript 实际上是如何工作的，因为即使没有那些花里胡哨的东西，它也真的很酷。

### Descript 的基于文本的编辑器如何工作

Descript 可以为你创建一个视频的转录，但你仍然需要梳理人工智能的脚本并纠正任何错误。也就是说，只要扬声器清晰，音频质量足够好，您可能不会有太多需要编辑的内容。

您还可以手动键入并标记转录的时间戳，或者甚至复制并粘贴脚本(如果用于录制的话)。手动转录很繁琐，但最终结果是一样的。

转录完成后，您可以通过点按转录的特定部分来跳过视频的时间线，然后您可以通过修剪文本中的文字来编辑视频录制。一个“填充词”功能——仅限高级用户——甚至会突出显示不需要的词，如“uhs”和“uhms”，从而更容易找到并删除它们。

你也可以用 Descript 的“Overdub”功能来替换单词，该程序会将新单词添加到音轨*中，而无需重新录制。*你所要做的就是从文字稿中选择一个单词或句子，然后重新键入这些单词。程序会在说话者的声音中合成被替换的单词，结果令人惊讶地令人信服。

显然，这里或那里替换的单词听起来很好，而整个句子听起来可能有点不自然，但它使用说话者自己的声音作为基线，所以至少听起来像他们。

我把制作播客和 YouTube 视频作为一种爱好，修复微小的语音错误是整个过程中最乏味的部分。更糟糕的是，如果您将多个音频质量或说话节奏略有不同的录音拼接在一起，那么重新录制片段听起来可能“不合适”。Descript 可以完全缓解这些问题——尽管如果你想使用 Overdub 和 Filler Words，你还需要购买另一个编辑工具。

### Descript 值这个价吗？

这给我们带来了最重要的问题:Descript 是否足以取代 Adobe Premiere Pro 和 Audition？你将为每一个支付大致相同的费用，所以除非你有足够的现金，否则你可能只想选择一个。

有免费版，但是超级限量。免费帐户获得一次性的三小时转录时间，之后他们需要升级 [才能获得更多。](https://www.descript.com/pricing) 免费用户还被限制只能录制 20 个最高分辨率为 720p 的屏幕录像，以及单个项目编辑(这意味着你必须完成或删除你正在处理的项目，然后才能开始新的项目)。

“创建者”订阅(15 美元/月或 144 美元/年)允许每月多达 10 小时的转录音频/视频，无限制的屏幕录制和正在进行的项目。然而，Descript 的 Overdub 和 Filler Word 功能被锁定在“专业”订阅(30 美元/月或 288 美元/年)之后，这与你在其他具有更深入编辑功能的订阅软件上花费的一样多(或更多)。在使用它一段时间后，我不推荐 Descript 作为你可能已经在使用的专业编辑软件的全面替代品。

明确地说，Descript 是一个完全胜任的多轨视频和音频编辑程序。它甚至有内置的屏幕捕捉和录音工具，它可以让你与多个用户远程协作编辑。这些都是有用的功能，它的文本文档 UI 很直观，但是 Descript 不像其他编辑软件那样强大或功能丰富。它缺少许多重要的视频工具，如色彩校正和独特的过渡，只有少量的音频润色。

也就是说，如果你刚刚起步或者买不起更昂贵的软件，Descript 可以满足基本的编辑需求，并且改变游戏规则的 Overdub 和 Filler Word 功能可以使 Descript 成为你现有制作工作流程的一个受欢迎的补充——如果你有多余的预算。如果你不确定的话，你可以随时试用免费版本 ，并一瞥预含教程项目的高级功能。
