# 信用卡保险值得吗？

> 原文：<https://lifehacker.com/is-credit-card-insurance-worth-it-1845560093>

支付保护计划是你信用卡的一种保险形式，它听起来很诱人:在困难的情况下，你可以用它来连续几个月暂停最低月付款，避免额外费用，让的信用评分不受影响。但是它值得每月花费的费用去注册它？



### **什么是支付保障计划？**

支付保护(又名“信用卡保护保险”、“信用盾”或“信用保障”)是一项附加服务，通常每月收取几美元费用，金额基于您总余额的一小部分。如果“触发事件”导致失业或残疾，该服务将暂停支付(通常长达 18 个月)。此外，一些支付保护计划将取消信用卡欠款在死亡事件。到目前为止一切顺利，对吧？

### **不利的一面**

即使你选择在经历的困难之前支付费用，获得支付保护的资格也不是一帆风顺的，因为许多计划都有一长串条件和排除条款隐藏在细则中。美国政府问责局的一份报告发现， [24%的福利申请被拒绝](https://www.gao.gov/new.items/d11311.pdf) ，其中超过一半的拒绝是因为持卡人无法提供足够的文件。这种缺乏透明度的做法在 2012 年引起了监管机构的审查，导致一些大银行完全放弃了这种产品(这也可以解释为什么它目前以如此多的不同名称上市)。

*   你已经有健康问题了。
*   如果你是季节性工人，或者是兼职工人，或者是个体经营者，你可能没有资格领取失业救济金。
*   您达到了每年触发事件的数量上限。
*   像取消债务这样的大额支出可能会有上限。
*   你可能需要在注册前几个月被雇佣才有资格。
*   证明您残疾、失业、或其他状况的文件不足(由您的保险公司定义)。
*   你可能不得不为任何你有资格做的工作而残疾，不仅仅是你现在的工作。

漏洞的清单还在继续。你真的必须仔细阅读政策的每一个细节——，即使你认为你是睁着眼睛进去的，对条款的解释也可能是你和你的贷款人之间的一场战斗。

### 底线

支付保障计划提供了有限的福利，通常只在短时间内暂停你的最低还款额和利息，这些福利可能无法覆盖*所有的*紧急情况。花在月租费上的钱可以代替花在 [人寿或伤残保险](https://twocents.lifehacker.com/what-types-of-insurance-do-i-really-need-1488536904) 上，这样会提供更好的保障。另一个选择是建立一个应急基金，以应对未来可能出现的困难。更妙的是，在期间，疫情许多信用卡都已经有了 [优惠的隐忍计划；你需要做的就是直接联系他们](https://twocents.lifehacker.com/get-all-your-credit-card-payments-deferred-with-one-cal-1843137590) 向索赔困难。毕竟，为什么要为可以免费得到的东西付费呢？
