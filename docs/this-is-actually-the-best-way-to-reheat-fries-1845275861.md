说我买空气油炸锅只是为了重新加热炸薯条是不公平的，但是说我想重新加热炸薯条与购买没有任何关系是不真诚的。回到今年年初——那时我们还年轻，空气清新——我写了一个小博客，声称 [华夫饼干悲伤的冷薯条](https://skillet.lifehacker.com/the-best-way-to-reheat-french-fries-1840958947) 是重新加热它们的最佳方式。

小巧而强大的对流烤箱——从技术上讲，它不会油炸任何东西——非常方便。我已经有了一个我打算油炸的东西的完整列表，但我从冷薯条开始(并在早餐时吃了它们)，因为这是最初把我们带到这一点的原因。

我的朋友们，你们(以及所有对我大喊大叫的人)没有撒谎。当谈到将松软的纸板状薯条恢复到以前的松脆、金黄色时，空气油炸锅击败了华夫饼机(尽管我认为华夫饼干剩余的薯条是很好的早餐土豆)。

由于水分迁移，剩余的薯条令人伤心和潮湿，空气油炸锅在短时间内解决了这一问题。一旦油炸食品开始冷却，蓬松的淀粉颗粒中的水分就会向外壳移动，使油炸食品内部呈颗粒状，外部呈糊状。空气油炸锅不能再水合这些淀粉颗粒，但它肯定能使油炸锅潮湿的外表复活。循环的热空气带走水分，并使任何休眠的油炸油脂移动和生长，使马铃薯外壳变脆。虽然内部不像你第一次从油炸锅里拿出来时那么嫩滑，但它们非常接近。我今天早上吃的那些和新鲜的薯条几乎没有区别，尽管值得注意的是它们似乎是“新鲜切割，一次煮熟”的薯条，所以这可能只是他们的第二次(而不是第三次)加热。

除了重新加热完全冷了的薯条，这是一个很好的恢复外卖薯条的方法，这些薯条可能在纸袋或塑料容器中放得太久了。在375度的空气油炸锅里只需几分钟就能让它们恢复活力。不同空气油炸锅的时间会有所不同，但是不像速溶锅或真空循环器，它很容易检查你的空气油炸食品的烹饪过程——只需将小篮子滑出即可。我的空气油炸锅在375华氏度下只用了五分钟就让冰冷、没有生气、切得相当厚的早餐薯条恢复了往日的辉煌，这是危险的快速，尤其是在一个倾向于过量订购薯条的家庭。
