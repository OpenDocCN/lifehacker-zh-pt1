# Vicks VapoRub真的能治愈趾甲真菌吗？

长期以来，那些饱受足部真菌困扰的人一直认为，非处方的Vicks VapoRub有一种隐藏的足疗功效:据称它可以治愈趾甲真菌。

《纽约时报》2009年的一篇文章支持了这种不知不觉的奇迹般的治疗，引用了一项研究，指出药膏的活性成分之一麝香草酚具有意想不到的抗真菌特性。草药爱好者可能喜欢科学验证，因为麝香草酚是一种在普通家用草药百里香油中发现的化合物。

也许由于趾甲真菌与其他更紧迫的健康问题相比微不足道，围绕这种所谓的治疗方法的阴谋仍然存在。问题显然依然存在:它真的有用吗？

### 《纽约时报》的文章说了什么？

研究人员一直在研究百里香这种草药对治疗趾甲真菌病的影响——虽然这种病通常没有害处，但预计每年 [会吃掉美国人医疗账单中的很大一部分](https://www.npr.org/sections/health-shots/2015/12/23/460806150/its-costing-a-lot-of-money-to-make-those-toenails-fungus-free) 。消除指甲真菌的非处方药有很多缺点:价格昂贵，效果不佳，而且有一大堆副作用。以下是《纽约时报》在2009年关于使用百里香等草药治疗皮肤癣菌的研究报道——皮肤癣菌是一种常见的真菌，人类通过与其他人、动物和土壤接触而感染。

> 在一项研究中，科学家们测试了一种普通药物胸部按摩成分的抗真菌效果。在这七种成分中，麝香草酚是抑制引起指甲真菌的皮肤真菌生长最有效的成分之一。对动物的其他研究也表明麝香草酚油对皮肤真菌有效。研究还表明，麝香草酚油通过破坏细胞膜和新陈代谢来破坏指甲真菌的另一个原因，念珠菌。

### 足疗界的其他人相信吗？

不完全是。没有足够的科学文献直接应用于Vicks VapoRub和麝香草酚的使用，以确定它是否有助于您的趾甲真菌。很可能你的足科医生不会建议你用它来达到这个目的。

2016年，特雷西·弗拉霍维奇(Tracey Vlahovic)查阅了该领域最有影响力的研究之一，推断出没有足够的证据支持这种特殊的药膏是一种良好的治疗方法。

> 时间会证明这种软膏是否真的能根除真菌，或者，凭借其软膏的特性，简单地创造一个更加水合的指甲单元，使指甲看起来更加健康。

此外，犹他大学的皮肤病学家克里斯多夫·赫尔博士最近声称T3疗法“可能没什么用”。但也不会伤害任何东西。”

赫尔继续解释了为什么主题治疗通常没有多大作用:

> 外面有很多这样的东西。很多人用醋，茶树油。这些局部药物的难点在于它们不能很好地穿透指甲。所以，实际上，让它们接触指甲里的真菌是非常困难的。这就是为什么许多处方外用药物的效果如此有限。

最好的办法是服用医生推荐的任何抗真菌药物。你的药柜里可能没有令人兴奋的非处方药，但至少你的医生可以开出一种可以的药。
