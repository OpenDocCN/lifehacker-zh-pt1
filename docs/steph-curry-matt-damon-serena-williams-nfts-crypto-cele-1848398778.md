# 我们输给 NFTs 的名人

> 原文：<https://gizmodo.com/steph-curry-matt-damon-serena-williams-nfts-crypto-cele-1848398778>

我们都经历过。你崇拜的人突然在头像上加了激光眼。他们已经进入了密码世界。

尽管如此，当他们开始为 [【死亡交叉】](https://gizmodo.com/bad-days-for-bitcoin-are-back-as-death-cross-approaches-1848333064) 而烦恼，或者考虑他们是否应该用他们的 [狗币去买一辆特斯拉](https://gizmodo.com/dogecoin-surges-18-after-elon-musk-says-the-crypto-wil-1848211210) 的时候，你还是要努力保持友好的关系。然后有一天你登录 Twitter，看到他们的头像是一只无聊的猿，他们在 Twitter 上发“gm”给其他几十个有猿头像的 Twitter 用户。他们去了 NFT。他们在推特上谈论购买加密市场的股票，尽管价值 1 万亿美元的股票一夜之间消失了。没有比抢购不可兑现的资产更好的方法了。

虽然许多随机的人一头扎进了加密货币和非金融交易的世界，但名人在推动市场达到泡沫新高度方面发挥了不小的作用。有些是意料之中的。(我们在看你，金·卡戴珊和汤姆·布拉迪。)但随着 NFT 的兴起，赚钱的诱惑越来越大，加入 NFT 的名人也越来越多。这份名单绝不是详尽无遗的，但这里有一些名人，虽然仍然在地球上的身体，但已经试图去月球的秘密世界。人们会怀念他们的。

当抖音宣布一项 NFT 计划时，Lil Nas X 被定为该平台第一批参与 drop 的用户之一。一方面，这是有道理的。Lil Nas X 如果不是一个潮流的引领者，那就什么都不是。在获得跨越说唱和乡村音乐的突破后，他成为了一个定义流派的嘻哈明星。从这个角度来看，进入 NFTs 是有意义的。就算你觉得大部分 NFT 的审美相当于一袋火上浇油的狗屎，你至少可以把的*种* *搞定。*

但是这位嘻哈明星也直言不讳地谈到了气候危机的风险，甚至 [在推特上](https://twitter.com/LilNasX/status/1137502076131565568) 说，他的突破性轰动作品《老城之路》包括“提及扭转正在发生的气候变化、水污染以及我们在这些艰难时刻目睹的灾难性政治气候。”大多数 NFT 都是在以太坊区块链制造的，它使用碳密集型的工作方法来维护其安全性。尽管抖音·NFT 的项目 [在混乱中崩溃](https://gizmodo.com/tiktok-ousts-marketing-guy-responsible-for-tiktok-kitch-1848395820) ，但看到霍米·蒙特罗与它扯上关系还是令人难过。

1 月 20 日，我们醒来时发现瑟琳娜·威廉姆斯 [在推特上发](https://twitter.com/serenawilliams/status/1484159217791647751)【GM】并贴出一只粉红色的无聊猿。令人心碎。她接着又发了一条“GM”[推文](https://twitter.com/serenawilliams/status/1484186070455500805) 关于一家名为 [Sorare](https://twitter.com/Sorare) 的公司，该公司提议让用户“在全球梦幻足球游戏中体验真正的足球联系！收集&交易数字卡。”你真的不想看到它。

大满贯时代赢球最多的网球选手，小威真的没必要走上这条路。当然，网球生涯只能持续这么长时间，塞丽娜可能想做点什么。但是这个呢？真的吗？

她的丈夫是 Reddit 的创始人亚历克西斯·奥哈尼安(一个无聊的猿类化身哈弗)，所以这个支点也许并不令人惊讶。他去年给她买了一只猩猩和一只隐朋克。去年，奥哈尼安甚至戴着它的胸针参加大都会歌剧院的晚会，在他的推特结尾用“WAGMI”炫耀它(“我们都会成功的”，对于我们当中的非红色 pilled。)小威还没到那种程度。还没有。

这个疼。去年四月，著名的告密者爱德华·斯诺登 [创作并出售了](https://foundation.app/@Snowden/foundation/24437) 一幅 NFT 的艺术作品，这幅作品将他的脸叠加在完整的法庭裁决上，描述了国家安全局 [的大范围监控如何违反了法律](https://www.theguardian.com/us-news/2020/sep/03/edward-snowden-nsa-surveillance-guardian-court-rules) 。这是一项基于斯诺登爆料的具有里程碑意义的裁决，毫无疑问是一项大交易，甚至在 他首次揭露该机构内部真实情况的这些年后 [。](https://gizmodo.com/one-year-after-the-first-snowden-leak-whats-really-cha-1586213046)

这就是为什么看到斯诺登如此陷入 NFT 的圈套；尽管这次拍卖是代表新闻自由基金会进行的，但它仍然是一次拍卖，*，*本质上把隐私新闻中最不朽的时刻之一变成了...有点抢钱。更不用说 NFT 的 [隐私和安全问题](https://101blockchains.com/nft-vulnerabilities-and-security-concerns/#:~:text=Hackers%20were%20able%20to%20gain,payment%20cards%20available%20on%20file.) 了，这一点斯诺登本人完全清楚，甚至在他自己的 NFT 首次公布后几个月就警告买家注意这些问题 [！来吧艾德。你不应该这样。](https://cointelegraph.com/news/edward-snowden-says-gamers-could-be-vulnerable-to-exploitation-using-nfts)

托尼·霍克似乎很冷静。当然，滑板先锋的角色是规则。但他真正闪耀的是他的[系列](https://twitter.com/tonyhawk/status/1482799721643085824)[推文](https://twitter.com/tonyhawk/status/1403551144064872451) 关于人们总是告诉他他*看起来*像托尼·霍克，他只是随大流。这种谦逊的氛围使得霍克跌入 NFTdom 令人惊讶。当然，他兜售。但是，让托尼·霍克选秀之王变成一只表演各种特技的骷髅鹰来做一次 NFT 空投有点儿傻。尽管这比机器人酷多了(？？)制作静态全息图(？？？？？？)托尼·霍克在数字渲染的半管中间的基座上。

如果你还没有看过马特·达蒙为 Crypto.com 的加密和 NFT 平台制作的令人作呕的广告，你应该现在就去看看。我们会等的。

好了，既然你回来了。*uggggghhhhhhhhhh*！这是达蒙，穿着一身黑色，走过一个画廊，同时告诉观众在加密货币上“冒险”，就像过去的登山者和冒险家在自己的想法上冒险一样。莱特兄弟出现了。有宇航员。换句话说，这是对巨大的 [充满骗局的庞氏骗局](https://jacobinmag.com/2022/01/cryptocurrency-scam-blockchain-bitcoin-economy-decentralization) 的完全不准确的描述，而加密世界经常如此。

更糟糕的是，这位明星 [最近还宣布](https://www.forbes.com/sites/jeffconway/2021/10/25/matt-damon-announces-waterorg-partnership-with-cryptocom/) 他十多年前共同创立的环保非营利组织 Water.org 将与加密货币平台合作。当时，Crypto.com 发表了 [声明](https://blog.crypto.com/matt-damon-water-org-partner-with-crypto-com/) 称此举有望鼓励对非营利组织的“秘密捐赠”，该组织旨在改善受气候变化影响最严重地区获得清洁水的机会。因为，你知道，没有什么比 [用 crypto](https://www.theguardian.com/technology/2021/nov/18/cryptocurrency-bitcoin-environmentalist-alarm-us-politicians) 支付更能说明“抗击气候变化”的了。

不管是好是坏，沙特纳搭乘 NFT 炒作列车的时间比这个名单上的大多数其他名字都要长。它最早开始于两年前的夏天，当时他收集的 125，000 张虚拟“交易卡”在大约 10 分钟内售罄，留下了无数演员的经典镜头...拿着艾美奖。做营地辅导员。穿得像巴克斯十九世国王。 [和更](https://shatner.wax.io/catalog) ！现在，只有一幅《盖伊》的 NFT 供竞拍——以喜怒无常的黑白照片为特色——目前有 11 幅作品的总出价，很明显，他仍然有他的粉丝愿意为任何以他们敬爱的柯克船长为特色的东西花钱。

有史以来最伟大的三分射手有点像个傻瓜。斯蒂芬既创造了*[，又购买了](https://opensea.io/SC30)*，尤其是在 8 月份整个事情真正开始起飞的时候买了一只无聊的猿。他也有一些来自一个叫 [隆隆孔联盟](https://www.rumblekongleague.com/) 的类人猿，我们刚刚得知这是元宇宙的一个体育联盟，承诺提供“前所未见的篮球体验”库里还拥有他打破三分球记录时所穿鞋子的数字效果图。他是否像重视可替代版本一样重视它们，这取决于 TBD。

考虑到阿姆是如何，至少，艰难地度过了 [【职业生涯中期危机】](https://www.npr.org/2018/09/10/646247737/eminem-became-a-parody-of-himself-and-everybodys-laughing) 这些收购可能只是对文化合法性的一种尝试。或者这家伙真的真的很喜欢花几十万买一部看起来有点像他的猿卡通。只有阿姆知道。

瑞茜·威瑟斯彭曾经是国际环保组织的代言人，她在 2015 年为该组织做了一场关于应对气候变化重要性的的大型公益广告。所以你不会想到，不到十年后，她会紧跟在 [环保恐怖的](https://gizmodo.com/carbon-offsets-for-nfts-dont-address-the-deeper-problem-1846520183) NFT 潮流上。但是 2020 年充满了惊喜，威瑟斯彭对 NFT 的痴迷显然是其中之一。从去年 10 月开始，她就相当定期地在推特上发布代币，甚至现在还在推特 上展示一幅 NFT 的作品作为她的 [个人资料图片。](https://twitter.com/ReeseW)

但也许她对“艺术形式”(如果你可以称之为艺术形式的话)的最令人畏缩的声明是在本月早些时候，当时威瑟斯彭在推特上说，“在不久的将来，每个人都将拥有一个平行的数字身份。”

“头像、加密钱包、数字商品将成为常态，”她写道。“你有这个打算吗？”

不管这是否真的会发生，里斯仍然完全投入到加密炒作中。我想，对她来说不错吧？

Gronk 看起来像一只超大的小狗，所以完全有可能它只是跟着汤姆·布拉迪走进了元宇宙兔子洞，以为那里会有零食。然而不幸的是，[gronknft . com](https://gronknft.com/)现在是一个存在的 URL。那里所有的 NFT 都卖完了。但是你也可以在转售市场上找到可爱的 tight end 自己做 GRONK 事情的各种数字效果图，如在 [OpenSea](https://opensea.io/collection/rob-gronkowski-championship-series-nfts) 接球或扣球，如果你想得到自己的“Gronk 数字签名和编号”

帕丽斯·希尔顿对尝试任何形式的商业冒险都不陌生，不管有没有问题。这位女继承人至少有一条 [失败的时尚路线](http://www.starstyle.com/paris-hilton-promoting-her-dollhouse-fashion-line-magic-marketplace-convention-las-vegas-sp13958/) ，在制作了一张 [单曲专辑](https://www.cheatsheet.com/entertainment/paris-hilton-1-hit-song-triggered-lawsuit.html/) 后一蹶不振的音乐生涯，以及更多你可能数不清的 [B 级表演角色](https://www.dontdiewondering.com/paris-hilton-from-ultimate-00s-socialite-to-serial-entrepreneur/) 。所以，当 NFT 的时装在去年年中成为新的热门时，希尔顿以比你说“简单的生活”更快的速度赶上了潮流，并在去年四月卖出了她的第一个时装系列，这并不令人惊讶。

如果“帕丽斯·希尔顿·NFT 系列”这句话没有让你晕头转向，那么也许这些作品的名字会让你晕头转向；有“我的元宇宙里的蜂鸟”，这只是一只粉红色蜂鸟溅在粉红色丽莎弗兰克风格背景上的图像。还有《传奇之爱》(Legendary Love)，其中有一个希尔顿标志性的狗小叮当的雕像，它于 2015 年去世，另一个名为“标志性加密女王”(Iconic Crypto Queen)，它是希尔顿本人的光滑机器人复制品。
