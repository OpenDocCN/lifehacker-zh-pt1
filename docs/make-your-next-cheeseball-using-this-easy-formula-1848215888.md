# 用这个简单的公式制作你的下一个芝士球

> 原文：<https://lifehacker.com/make-your-next-cheeseball-using-this-easy-formula-1848215888>

带着奶酪球参加办公室假日聚会或其他季节性聚会是赢得朋友和影响同事的一种简单、划算的方式。奶酪球吃起来很有趣，也很容易制作，在这个广阔的信息网上有很多关于它们的食谱。



我不认为我做过两次同样的奶酪球。我喜欢尝试各种口味，但我也讨厌照着菜谱做。然而，我确实喜欢即插即用的配方，它能给你指导，而不会让你受限于严格的成分列表，而我恰好有这样一个制作简单美味的奶酪球的配方

## 克莱尔简单灵活的奶酪球配方

> 8 盎司可涂抹的奶酪+ 8 盎司切碎的硬奶酪+ 1 汤匙混合调料+ 2 茶匙酸性液体+松脆的东西

就是这样。

对于可涂抹的奶酪，你会想要奶油奶酪、山羊奶酪、奶油奶酪和山羊奶酪的混合物，或者像 Alouette 一样的预先调味的奶酪涂抹料(尽管如果你使用最后一种，你可能不得不减少调料)。

对于硬奶酪，切达奶酪是首选，但不要害怕尝试一些陈年的东西，如有水晶斑点的陈年高达奶酪、gruyère 奶酪或 pecorino Romano 奶酪。

你应该使用的调味料的确切数量可以根据它的咸度而变化，所以如果它的钠含量非常高，就从两茶匙开始。仅在 Trader Joe's，就有看似无穷无尽的混合调料可供购买，如果你愿意，你可以尽情享用，但不要低估了一包经典的意大利调料，它的大小正好可以给一个奶酪球调味。

一点酸性液体可以防止你的牛奶球感觉沉重，让你的味觉充满脂肪。柠檬汁、红酒醋和泡菜盐水都不错，但最后一种绝对是我的最爱。

最后，你需要一些松脆的东西来包裹它。炸蒜是我最近喜欢的果酱，但我也喜欢传统的碎山核桃(或任何其他坚果)和百吉饼调料组合。

## 如何组装你的奶酪球

当你的软的、可涂抹的奶酪在室温下时，如果你的硬奶酪没有被切碎，T2 就把它切碎。将两种奶酪放入碗中，用木勺充分搅拌。加入几茶匙混合调料，再次混合，品尝混合物。如果你认为需要的话，再加一两茶匙。加入你的酸，再次混合，然后给你的未成球的奶酪一个最后的味道，再加入一点它需要的任何东西。

用塑料包好奶酪球，放入冰箱，让各种口味融合至少一个小时。当你准备上菜时，打开包装，把松脆的东西撒在盘子上，把奶酪卷成小块，根据需要把它们压成球，帮助它们粘在一起。与饼干一起食用。
