# 为什么你可能想在跑步前淋浴

> 原文:[https://life hacker . com/why-you-may-want-to-shower-you-run-1849131818](https://lifehacker.com/why-you-might-want-to-shower-before-your-run-1849131818)

说到 [在热火中奔跑](https://lifehacker.com/here-s-how-much-slower-you-run-in-the-heat-1782687837) ，我就是个软骨头。(说到很多事情，我就是个软骨头。)在跑完春季 [马拉松](https://lifehacker.com/how-young-is-too-young-to-run-a-marathon-1848901266) 后，我努力恢复训练，让自己走出家门，进入夏季天气。然后我想起了我通常为赛前紧张保留的一个窍门:快速淋浴。

Watch

你可能会认为在你汗流浃背，需要再次淋浴之前，淋浴是没有意义的。但是如果你像我一样，跑前快速冲洗可能正是你解决夏季跑步低迷所需要的。这篇文章的灵感来自于我自己的经历，并得到了 [这篇来自Ready Set Marathon](https://readysetmarathon.com/should-you-shower-before-running/) 的博客文章的支持。

## 跑步前淋浴的好处

对我来说，快速淋浴的最大好处是在跑步前让我清醒。我的秘诀是坚持用凉水，淋浴保持一两分钟，直到我感觉精力充沛。虽然冷水淋浴经常与冷却联系在一起，但快速冲洗是唤醒你开始早晨锻炼的有效方法。

第二，冷水能奇迹般地让人们更容易进入高温环境。这种效果不会持续很长时间，但它足以让你在夏天出门时精神振奋。显然，泡在凉爽的游泳池里会更好；我凑合。

## 跑步前淋浴的缺点

避免运行前冲洗的最明显的原因是，从几个不同的方面来看，这是一种浪费。首先，如果你知道锻炼后你也需要洗澡，这可能会感觉浪费你的时间和精力。

在更大的范围内，两个淋浴是浪费水。为什么会造成额外的环境压力， [尤其是现在的](https://gizmodo.com/supreme-court-west-virginia-epa-power-pollution-ruling-1849112254) ？也许坚持一场 [冷场](https://lifehacker.com/create-the-most-bracingly-cold-shower-experience-ever-1827242963) r赛后跑是你能做的最负责任的事情。

## 底线

如果你需要在炎热的夏日早晨尝试一些新的跑步方式，试着快速冲个澡。如果这对你不起作用，那么至少你在节约用水。大多数跑步者都有某种仪式，这种仪式更多的是精神上的，而不是科学上的。对我来说，淋浴一两分钟对我的夏季训练有很大影响。
