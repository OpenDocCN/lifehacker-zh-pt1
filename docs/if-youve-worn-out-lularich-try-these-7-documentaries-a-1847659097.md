不要与网飞的另一部耸人听闻、以恶作剧为主题的系列电影 、*抢劫*混淆，它绝对是真实犯罪的较轻选择之一。这部纪录片由《肮脏的强盗》和艾美奖提名导演德里克·多尼恩(Derek Doneen)执导，涵盖了从大胆到细致的真实盗窃故事。每一个主题都有两集，包括一个从《盗亦有道》和《犯罪程序》中获得灵感的男人，一个参与了拉斯维加斯历史上最大胆的抢劫案之一的女人，以及一个被指控偷了世界上最昂贵的波旁酒的肯塔基州父亲。《抢劫》在重演方面有点重，尤其是在第一集，虚构的复述比现实生活中的犯罪者/被指控的犯罪者的对话占据了更多的屏幕时间。他们确实增强了这部剧的趣味性，这部剧对与之相关的犯罪采取了非常程式化的方式。每一个主体的故事的后半部分都要面对后果。但是有了“性魔法金钱谋杀”这样的剧集标题，《抢劫》自始至终都是一部轻松的电影。[达内特·查韦斯]
