# 这些是飞机扶手分配的明确规则

> 原文：<https://jalopnik.com/these-are-the-definitive-rules-for-airplane-armrest-all-1847095061>

你可能已经听说，随着人们从他们的 COVID 诱导的禁闭中走出来，再次飞向天空，飞行中的打架事件、、疯狂的场景[一般不守规矩的行为](https://www.npr.org/2021/06/04/1003191559/as-covid-19-restrictions-ease-airline-employees-are-seeing-trouble-on-flights) 达到了惊人的高水平。虽然其中一些与面具和酒精有关，但仍有经典的飞行冲突主题，如扶手分配。扶手领土争端的事情是，这应该是一个很好解决的问题。如果你不知道，我现在要提醒你扶手的规则。

在飞机上发生争吵的所有事情中，这可能是最愚蠢的，因为扶手到座位的分配规则早已确立，每个乘客都应该意识到我们的前辈在空中努力了这么久才建立的重大妥协。

但这是既定的。我们知道什么座位有什么扶手，也知道为什么。原因不是武断的，而是基于合理的、不屈不挠的逻辑和每个人与生俱来的基本体面感。

这些是规则:

如果你看不到图表或者你的贴身男仆给你读这个，我会一个座位一个座位地解释:

### **过道**

这个座位只有外侧扶手。靠过道的座位可以自由离开这一排使用卫生间，过道一侧有相当大的伸展空间，这是这一排最开阔的感觉。

过道座位的职责是站起来让你的同排座位的人去洗手间或飞机的其他地方。

### **居中**

这个座位有两个扶手。中间的座位没有过道或窗户的好处，因此可以通过使用座位两侧的扶手来弥补。

人们普遍认为这是最差的座位，因此应该有额外的补偿扶手。

### **窗口**

这个座位有靠墙的扶手。靠窗座位当然有窗户，它向那些仍能感受到这种快乐的人展示了重于空气飞行的奇迹，因此也是最受儿童欢迎的座位选择。这堵墙还提供了宝贵的靠着睡觉的选择，以及一点隐私，如果你在打电话或做其他事情时把脸贴在墙上的话。

缺点是你就像中间的座位一样被困住了，你可能会被要求调整窗帘，如果合理的要求，你真的应该遵守。

就是这样。这就是规则，而且毫不动摇。如果需要，您和您的同排乘客可以协商其他安排，包括将内部扶手移开，但大扶手妥协是默认协议，必须始终遵守，除非获得该排所有三名成员的完全同意。
