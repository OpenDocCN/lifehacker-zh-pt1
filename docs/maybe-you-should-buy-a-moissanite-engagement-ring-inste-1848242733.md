# 也许你应该买一枚碳硅石订婚戒指，而不是钻石

婚姻和钻石是千禧一代被指责杀害的和中的几件。虽然说千禧一代不买钻石(或结婚，就此而言)并不准确，但*确实以不同的方式购买。*

高昂的价格和对开采钻石的道德担忧只是人们选择实验室培育的宝石或替代宝石(如类似的碳硅石)作为订婚戒指的几个原因。

碳硅石是为数不多的 [普通钻石替代品](https://www.brides.com/diamond-alternatives-5186406) 之一，包括蓝宝石、红宝石和祖母绿等有色宝石。但与其他宝石相比，碳硅石几乎无法从视觉上与钻石区分开来，作为珠宝制造商使用的第二硬的石头，其耐用性也相似。

这里有几个理由来考虑碳硅石订婚戒指。

## 碳硅石和钻石有什么区别？

如果你想要钻石的外观，但又不能忍受T2购买钻石，碳硅石是你能得到的最接近的了。

### 价格:碳硅石更便宜

购买钻石的最大障碍之一是成本。即使是非常小的优质钻石，对许多人来说也是买不起的——在预算紧张的情况下是买不起的，对那些只想把钱花在别处的人来说也是荒谬的。

碳硅石 [大约是大小和质量相似的钻石](https://www.brides.com/moissanite-vs-diamond-5094238) 的10%。所以，如果你想要一块更大的石头，碳硅石会给你更大的回报。

### 伦理:碳硅石是合成的

你当然可以买到实验室生产的钻石，但所有的碳硅石都是合成的，而不是开采的，这减轻了人们对钻石行业道德的担忧。碳硅石是自然形成的，但块度不够大，无法切割成可用的宝石。

### 外观:碳硅石几乎相同

对一般人来说，碳硅石和钻石是难以区分的，不太可能有人仅仅通过看你的石头就能分辨出来。

几个 [非常微小的区别](https://www.gemsociety.org/article/moissanite-vs-diamond/) :碳硅石可能会因光照而略微变色，尽管有可能得到“无色”的石头。它也可以反射彩色光而不是白光。然而，它不太可能包含任何可见的瑕疵，而较便宜的钻石往往会有。

### 耐用性:碳硅石几乎和钻石一样坚固

除了钻石，宝石的一个缺点是它们更软，更容易损坏，这对于你每天佩戴的订婚戒指来说并不理想。说到耐用性，碳硅石是你能得到的最接近钻石的材料，而且不太可能碎裂或破裂。

## 你应该买碳硅石还是立方氧化锆戒指？

立方氧化锆是另一种看起来类似于钻石的透明石头，但它的耐用性差得多，多孔性也更强，这意味着随着时间的推移，它容易出现划痕、断裂和变暗。这是一个相对便宜(和合成)的替代品，但立方氧化锆石可能需要在某些时候被取代。还有其他透明宝石选择，尽管它们在亮度和耐用性方面差异很大。

## 如何选择碳硅石

如果你想要一枚订婚戒指，并决定选择碳硅石，有几个购物策略 要记住:

*   比较价格:网上珠宝商的价格可能比你当地的珠宝店要低，但价格并不代表一切。货比三家至少会让你更好地了解什么是合理的，并有一些讨价还价的筹码。

*   **越大未必越好:**仅仅因为你买得起大得多的碳硅石，并不意味着你应该走这条路。较大的石头可能会有更明显的色差。

*   **不要满足于不完美:**大多数碳硅石是所谓的“这就是为什么没有理由为暗淡、有表面划痕或颜色不佳的石头付钱。
