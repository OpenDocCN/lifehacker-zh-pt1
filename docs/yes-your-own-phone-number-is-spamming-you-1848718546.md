# 是的，您自己的电话号码正在向您发送垃圾邮件

> 原文：<https://lifehacker.com/yes-your-own-phone-number-is-spamming-you-1848718546>

在这一点上，我们都习惯于接收垃圾短信:声称来自你的运营商为你支付账单提供“礼物”的消息，或者来自“网飞”要求你打开一个链接并“更新你的订阅”这些敲诈你的企图并不罕见。不同寻常的是，这些短信不是来自陌生的来源，而是来自你自己的电话号码。



收到自己号码发来的垃圾短信并不意味着你被黑了，尽管乍一看可能是这样。你抱怨又收到一条糟糕的垃圾短信，却发现攻击者不是某个不知名的骗子，而是你。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-O-YX6pMLMfA&start=4](https://lifehacker.com/embed/inset/iframe?id=youtube-video-O-YX6pMLMfA&start=4) 

但是不，你的号码没有被泄露，也不代表黑客窃取了你的信息。这种类型的垃圾邮件攻击被称为“欺骗”，其中不良行为者不是通过他们实际的电子邮件地址或电话号码发送消息，而是从他们喜欢的任何来源发送消息，包括越来越多的您自己的电话号码。

这些网络钓鱼企图既可笑又透明。干得好，垃圾邮件发送者:你知道如何从我自己的电话号码发送文本，并且你正在使用这种能力来执行你以前一样的令人厌倦的骗局。*为什么*这次我会认为骗局是真的，因为它看起来像是我发给自己的？

对于这个问题你能做些什么吗？当你收到任何垃圾短信时，你有两个选择:你可以向你的运营商举报，或者向 FCC 投诉。

但是当你向你的运营商报告你自己的电话号码时会发生什么呢？要举报垃圾短信，您首先要将其转发到 7726(垃圾短信)，然后跟踪该短信，找到发件人的电话号码。嗯，你好，这是你的号码。那有什么好处呢？此外，运营商已经很清楚这些垃圾短信的内容，他们似乎不愿意或无法有效地处理它们。

这就剩下了向 FCC 报告的选项，这可能是你的最佳选择——组织收到的关于这个问题的正式投诉越多，他们就越有可能迫使运营商采取行动，并找到实际的解决方案。然而，这两种做法都不太有效。作为消费者，你根本无法阻止大量垃圾邮件向你袭来。运营商真的需要解决这个问题。

不言而喻，但是*请*不要与这些垃圾链接互动或打开它们。虽然有时它们是无害的——最近有报道称，虚假短信无非是 [指向俄罗斯国家电视台网站](https://www.youtube.com/watch?v=O-YX6pMLMfA&t=4s)——但一些链接可能会指向更邪恶的地方。
