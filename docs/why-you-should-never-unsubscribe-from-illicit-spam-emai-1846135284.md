# 为什么永远不要“退订”非法垃圾邮件和短信

> 原文:[https://life hacker . com/why-you-should-never-unsubscribe-from-illicit-spam-email-1846135284](https://lifehacker.com/why-you-should-never-unsubscribe-from-illicit-spam-emai-1846135284)

在处理电子邮件或垃圾短信时，我们中的一些人可能有点太*勤奋了——因为无论你做什么，你都不应该点击“取消订阅”链接或回复短信“停止”，因为它们的字面意思是“订阅”和“请继续”给非法垃圾邮件发送者。*

### **与垃圾邮件发送者的任何互动都是有害的**

非法垃圾邮件向你发送恶意软件链接和销售假保险费率是有区别的，和你注册了但可能不再需要的时事通讯，在这种情况下，我们指的是前者。随着这些垃圾邮件发送者每天发出数百万条短信或电子邮件，他们实际上并不是专门针对你的——事实上，他们可能不知道你的电子邮件或电话号码是否有效。然而，他们*正在*寻找活跃账户的迹象，因为有效的电子邮件地址或电话号码对于实施进一步的诈骗很有价值。通过切换“取消订阅”或以任何方式回复，您验证了您的联系信息，并冒着招致更多垃圾邮件的风险。

### **如何处理垃圾邮件**

简单的部分是你基本上什么都不做。当你收到一封垃圾邮件，将其标记为垃圾邮件，并在打开之前删除它。同样，对于短信，你可以直接删除它们而不回复，并屏蔽该号码。也就是说， [FTC建议](https://www.consumer.ftc.gov/articles/0038-spam) 通过将垃圾邮件转发到号码7726(垃圾邮件)来报告垃圾邮件——然而，如果你不精通技术，篡改垃圾邮件文本会*增加*意外点击邮件中某处恶意软件链接的机会(你永远不应该点击未经请求的链接)。如果你不喜欢转发短信，不要担心——删除、屏蔽和忽略就好。否则，你可以使用这些步骤转发垃圾短信 [，无论是对于你的Android还是iPhone](https://www.wikihow.com/Report-a-Spam-Text#:~:text=Enter%207726%20as%20the%20recipient,to%207726%20free%20of%20charge.) 。

### **如何避免垃圾邮件**

垃圾邮件是生活的一部分，但正如美国联邦贸易委员会(FTC)所建议的，有一些简单、有用的提示可以帮助你防止垃圾邮件:

*   避免在公共场合显示你的电子邮件地址。垃圾邮件发送者会抓取博客帖子、聊天室、社交网站和论坛，所以你在那里的次数越少越好。

*   使用两个电子邮件地址——一个用于发送个人信息，一个用于发送其他信息。理想情况下，第二个面向公众的电子邮件地址应该是你愿意在某一天删除的，如果需要的话。就我个人而言，我使用第二个假名电子邮件地址，我称之为“垃圾邮件”，我用它来注册促销或时事通讯。现在来看，与我的常规地址相比，它的垃圾邮件数量是它的两倍。
*   使用真正原始的地址，不太可能是垃圾邮件发送者创建的。垃圾邮件发送者向大型ISP和电子邮件服务中可能的名称组合发送数百万封邮件，希望找到一个有效的地址。这意味着普通的名/姓电子邮件地址更容易吸引垃圾邮件(作为Hotmail的第一个Mike Winters，我可以不耐烦地确认这是真的)。
