# 如何使用抖音新的反欺凌工具

> 原文:[https://life hacker . com/how-to-use-tiktoks-new-anti-欺负-tools-1846456341](https://lifehacker.com/how-to-use-tiktoks-new-anti-bullying-tools-1846456341)

任何社交媒体平台成功的一个不幸的副作用是相关的网络欺凌和骚扰的增加，抖音也不例外。

Watch

幸运的是，该公司已经开始实施新的反欺凌战略来保护用户的安全，开始于2020年12月推出的更强有力的反骚扰和网络欺凌政策。抖音反欺凌倡议的最新部分是两个旨在减少有害评论的新功能。

### 评论批准

第一个是新的评论批准选项。抖音已经允许用户 [通过关键词](https://support.tiktok.com/en/using-tiktok/messaging-and-notifications/comments) 过滤评论——这样你就可以防止用户在你的视频上发布诋毁、亵渎或其他有问题的内容——但该应用现在可以隐藏所有新的评论，直到你审查并批准它们。你认可的将公开出现在视频的评论部分，而你驳回的将不会被发布。

在抖音应用中查找并启用新的评论审核工具:

1.  打开您的抖音个人资料。
2.  轻点**“…”**图标。
3.  去**隐私与安全>那里可以评论我的视频。**
4.  选择**“注释过滤器”**
5.  打开**“过滤所有评论”**

要查看所有待定评论的列表，请返回评论过滤器菜单，然后点击**“查看过滤的评论”**轻按评论，然后选择是删除还是批准它。

### 预防性反欺凌提示

除了新的评论审查工具，抖音现在可以在用户发布评论之前自动筛选评论。

如果应用程序的人工智能识别出与抖音政策相悖的语言，应用程序会警告用户，他们的评论可能会违反规则，并给他们一个修改或放弃评论的机会。

然而，用户不会被阻止写可能有害的评论——用户可以忽略抖音的提示，重新考虑并发表他们的评论。如果他们这样做了，扩展的评论过滤器将有望阻止那些粗糙的回复最终出现在评论区。尽管如此，该公司希望这一额外措施将减少其平台上的骚扰事件——以及普遍的负面影响。
