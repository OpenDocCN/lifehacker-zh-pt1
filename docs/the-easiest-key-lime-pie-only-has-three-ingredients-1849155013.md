# 最简单的酸橙派只有三种配料

> 原文：<https://lifehacker.com/the-easiest-key-lime-pie-only-has-three-ingredients-1849155013>

有时候——尤其是在炎热的夏天——你只想甜点尽可能简单。由 [Kirbie 的](https://kirbiecravings.com/) 发布的这款新的 [三种成分的无烘焙关键酸橙派配方](https://kirbiecravings.com/3-ingredient-no-bake-key-lime-pie/) 正是如此。食谱只需要三种配料:酸橙、甜炼乳和预制的全麦饼干皮。它能做出又甜又酸又美味的青柠派。

最初的食谱要求将两罐甜炼乳与 3/4 杯酸橙汁和一汤匙酸橙汁混合，然后倒入全麦饼干外壳中，放入冰箱。如果你没有关键的酸橙或者不想要挤压一大堆迷你酸橙的额外工作，普通酸橙就可以了。

如果原始配方中的两罐 14 盎司的甜炼乳对你来说太甜了，你可以用其中一罐换成 8 盎司的奶油奶酪，你可以把它搅成软峰，并搅拌到莱姆和甜炼乳的混合物中。当我做那个版本的时候，我发现馅饼的馅有点软，最后我把剩下的馅饼放进了冰箱。但绝对是我以后再做的馅饼。

## **这种酸橙派很容易做成纯素食派**

在测试了原始配方后，我想知道是否可以通过使用加糖的浓缩椰子汁来制作纯素食版本。事实证明，只要你把馅饼放在冰箱里，答案是肯定的——你可以做一个奶油软糖馅的纯素青柠馅饼。我原以为会有强烈的椰子味，但这只是一个暗示，酸橙味占据了中心位置。(你需要冷冻馅饼，并在冷冻时食用，因为将其放入冰箱会产生无法凝固的馅料。)

你需要:

*   4 罐 7 盎司的甜浓缩椰奶
*   1 汤匙酸橙皮
*   一杯鲜榨酸橙汁
*   预制全麦饼干外壳

## **如何制作素食青柠派**

1.  将酸橙汁和酸橙皮搅拌到加糖的浓缩椰奶中。搅拌直至混合均匀。
2.  将混合物倒入全麦饼干馅饼皮。
3.  放入冰箱，直到混合物冻结成类似软冰淇淋的稠度。
4.  冷冻后食用。如果你愿意，可以在上面放些鲜奶油和额外的酸橙皮。
