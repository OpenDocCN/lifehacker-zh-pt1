# 别老是检查你的 401k

> 原文：<https://lifehacker.com/stop-checking-your-401k-so-much-1849054622>

现在比以往任何时候都更注意这个建议:停止看你的 401k 那么该死多。当有关于股票市场的令人担忧的消息时(见:此刻每一条最后的经济新闻)，你可能会觉得不得不纠结于你的退休储蓄情况如何，但是不要这样——即使余额已经清楚地列在你的银行应用的仪表板上。别理它。原因如下。



## **你的 401(k)是为了以后**

Jim Keenehan 是 AFS 401(k)退休服务有限责任公司的高级顾问，多年来一直担任财务顾问。他建议你不要把你的 401(k)计划当成银行存款。相反，“你可以把它想成是一种退休后能够为自己提供薪水的方式，”他说。你停止了工作，你的薪水也停止了，你可以依靠你通过与雇主的计划存起来的钱。这是以后的钱。平衡看起来怎么样对现在已经不重要了，除非你现在已经到了或者非常接近退休年龄。

当你知道股市下跌意味着你的余额下降时，这就很难记住了。人们很自然地希望多次赶去查看损失程度，尤其是在经济动荡时期。但是，基恩汉说，这样做只会对你不利。

“当股票市场表现良好，你的投资也表现良好时，这可能是一件有趣的事情，”他承认道。"然而，股票市场并不总是上涨."不管是上涨还是下跌，你都要长期坚持。

## 你应该多久检查一次你的 401k？

Keenehan 说，除了寻求余额上升的快感或预期余额下降的压力，还有其他原因可以让你检查自己的 401k，比如改变你的缴款额或更新你的受益人。

没关系。否则？他说，每年做一次。事实上，尽你所能，也只需要每年做些改变，如果可以的话，在财务顾问的帮助下完成。

## 你能做些什么让自己感觉更好，不再检查你的 401k？

基恩汉警告说，不要因为关注日常平衡而让自己发疯。取而代之的是，通过审视你的财务状况，收回一些控制措施。找一个财务顾问，或者和你的雇主核实一下，看看他们是否能给一个能够回答你问题的人。(如果他们提供 401k 计划，他们应该。)

“和一个信托理财顾问坐下来谈谈。“受托人”是一个关键词，因为这意味着财务顾问将为这个人的最大利益服务，”他说。“很多时候，人们都在看关于股票市场崩盘的头条新闻—,这实际上意味着震惊价值。他们试图让人们点击链接，他们想在屏幕上吸引眼球。这并不是任何人都应该在自己的退休账户中遵循的建议。”

简而言之，不要根据一篇文章或者一条推特或者一种直觉来做财务决策。和一个有知识和经验的顾问坐下来谈谈，他能理解波动的市场，他(T4)能稍微冷静地指导你。每年做一次。否则，停止登录。
