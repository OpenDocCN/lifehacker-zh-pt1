# 为什么你应该知道你的脚的厘米尺寸

> 原文：<https://lifehacker.com/why-you-should-know-your-foot-size-in-centimeters-1847435887>

不同品牌的鞋码差异如此之大，以至于买一双新的会感觉像是在赌博。很明显，最好的合脚方式就是试穿，但这并不总是可能的。如果你别无选择，只能在网上订购，你应该知道你的脚实际上有多大。



为什么是厘米而不是英寸？不出所料，它们被广泛使用:几乎地球上的每一个鞋制造商都在他们的尺码表中包括厘米尺寸，所以一旦你知道了你的号码，你就可以很容易地找到你在任何品牌的尺码。它还可以快速准确地将男性尺寸转换为女性或儿童尺寸(反之亦然)。基本上，如果你在网上买鞋，你需要测量你的脚。

谢天谢地，这非常容易做到。两个最好的方法需要一把尺子或卷尺，一支笔和纸，以及有人帮助。厘米标记是理想的，但如果你没有公制尺或卷尺也可以。只是一定要把每一个以英寸为单位的测量值乘以 2.54 来得到以厘米为单位的等值数字。

使用软卷尺更容易，因为你可以简单地站在上面，让你的助手计算你脚趾和脚跟位置的差异。 [来自圣安东尼奥的 Good Sports Outdoor Outfitters](https://www.youtube.com/watch?v=Q6WhrvlSlBY)的这段视频详细描述了这项技术:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-Q6WhrvlSlBY&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-Q6WhrvlSlBY&start=0) 

如果你只有一把尺子或其他你不想站在上面的东西，你可以用它来测量长度。密歇根足科医生 YouTube 频道有 [关于正确追踪和测量](https://youtu.be/-07Gllupv48?t=161) 的深度指南:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video--07Gllupv48&start=161](https://lifehacker.com/embed/inset/iframe?id=youtube-video--07Gllupv48&start=161) 

无论你使用哪种方法，确保你在测量的时候把重量放在每只脚上，这样你就能得到尽可能精确的数字。完成后，将两次测量结果保存在手机、Google Drive、Dropbox 或密码管理器的备忘录中。(冰箱上的便利贴也很有效。)

使用你的尺寸真的可以消除找到你的尺寸的猜测。假设你的左脚 26.3 厘米，右脚 26.5 厘米。对于需要合脚和/或可能会变大的鞋子，寻找对应于 26.5 或 27 厘米的尺寸。如果你想要一双更宽敞的鞋子——比如你会搭配厚袜子穿的靴子——那就穿 27.5 码甚至 28 码的。

就是这样！你可能不会每次都得到完美的尺码，但是你会比传统的尺码更接近。“9 号”可以有一百万种不同的含义，但是你可以永远相信厘米。
