# 你可以像使用鸡尾酒一样使用 Fernet

> 原文：<https://lifehacker.com/you-can-use-fernet-like-cocktail-bitters-1845940913>

昨晚，当我制作三瓶 [自制爱尔兰奶油](https://skillet.lifehacker.com/make-your-own-irish-cream-immediately-1845788014) 利口酒(作为礼物)时，我决定其中一瓶应该是肯塔基奶油，于是伸手去拿波旁威士忌而不是爱尔兰威士忌。在混合可可粉、炼乳、奶油和威士忌(还有一小撮盐和少量香草)后，我把它们搅拌起来，给混合物一点味道。这很好，但它需要一些东西。那东西，我的朋友们，是 fernet 的飞溅。



虽然我一杯一杯地喝 fernet(当然是加冰的)，但并不是每个人都像我一样喜欢它苦涩、提神的味道。但是，即使全面的 fernet 有点多，这并不意味着你应该完全避免它。很像鸡尾酒苦味剂，Fernet 有强烈的草药味道，如果你谨慎使用，可以给饮料增加一点复杂性。 [比如花心的朱岳鹏](https://lifehacker.com/3-ingredient-happy-hour-the-hanky-panky-1818640347) ，只有几个破折号，没有它们就不是花心的朱岳鹏(就只是一个 [杜松子酒& It](https://skillet.lifehacker.com/the-gin-it-is-the-perfect-cold-weather-martini-1840004492) ，也不错)。

但除了手帕朱岳鹏，fernet 可以像任何鸡尾酒苦给你的饮料一点点优势。如果你用的是 Fernet-Branca，边缘会有一点薄荷味和一点药物味；如果你用的是墨西哥 fernet，它会更温暖，烘焙香料会更重。我用了一点我为肯塔基奶油准备的注入香草的 fernet ，发现它相当令人愉快。
