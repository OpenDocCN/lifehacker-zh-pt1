# 如何正确储存剩余的油漆

> 原文:[https://life hacker . com/how-to-rightly-store-left-paint-1847832661](https://lifehacker.com/how-to-properly-store-leftover-paint-1847832661)

尽管从技术上讲，一加仑油漆应该覆盖400平方英尺 ，但情况并非总是如此。另外，你可能正在画一个空间，却不知道它到底有多大。无论哪种方式，都很容易错误地估计一个项目需要多少油漆，当你完成后会剩下一些。

虽然把油漆桶的盖子打开储存起来可能很诱人，但有更好的方法。这里有一些这样做的技巧。

## 选择正确的容器

保持油漆新鲜的诀窍是不让空气进入。这使得 [罐装罐](https://www.truevalue.com/diy-projects/post/paint-and-stain/leftover-paint-storage) 非常适合这项工作，感谢它们的气密密封。此外，因为它们是透明的，你可以很容易地看到油漆的颜色。一旦你有了罐子，搅拌罐子里剩下的颜料，用一个漏斗把它转移到罐子里。

如果你手头没有罐装罐，需要马上储存油漆，你最好的办法是 [在罐子(或其他容器)的](https://www.truevalue.com/diy-projects/post/paint-and-stain/how-to-store-leftover-paint) 开口处放一张保鲜膜，盖上盖子，然后把盖紧。如果你用的是油漆的原罐，用橡皮锤轻敲盖子，这样盖子就形成了气密密封。

## 给容器贴上标签

接下来， [给罐子](https://www.truevalue.com/diy-projects/post/paint-and-stain/leftover-paint-storage) 贴上标签，这样你就能确切地知道你有哪种颜料，以及它是什么时候被打开的。考虑在标签上包括如下信息:

*   油漆品牌
*   颜色
*   类型
*   光泽/表面
*   房间/家具

## 把它存放在正确的地方

最好将油漆储存在阴凉干燥的地方，远离阳光。 处理任何一种颜料都是一个好主意，尤其是当颜料储存在透明的玻璃罐中时。此外，你要确保那里的温度保持在零度以上——理想情况下， [在华氏60-80度之间](https://www.hgtv.com/design/make-and-celebrate/handmade/how-to-store-leftover-paint) 。最后，一定要把颜料 [放在高架子](https://www.hgtv.com/design/make-and-celebrate/handmade/how-to-store-leftover-paint) 上，远离食物、饮料、小孩。
