# 什么是“COVID 护照”？

> 原文：<https://lifehacker.com/whats-a-covid-passport-1845323181>

航空旅行要恢复到新冠肺炎·疫情之前的样子还需要一段时间。在七个月的国际旅行限制后，许多人急切地等待登上国际航班的机会。这不仅仅包括因为 COVID 而不得不推迟梦想之旅的度假者，国际航班的限制也给那些商务旅行或去另一个国家探亲的人带来了挑战。

现在， [一项新的试点项目](https://www.afar.com/magazine/new-covid-passport-being-tested-at-newark-and-heathrow) 让乘客有机会上传他们与 COVID 相关的健康数据，正在由两家主要航空公司进行测试，作为一种潜在的促进疫情期间国际旅行的方式。以下是你需要知道的关于所谓的“COVID 护照”

### 什么是 COVID 护照？

从四月 开始，就有关于使用“豁免护照”进行国际旅行的讨论。最初的想法是向对新型冠状病毒病毒产生免疫力的人发放某种类型的文件，但随着我们对该病毒了解的更多，很明显它没有遵循典型的康复轨迹。在这一点上，T5T7】现在还不清楚是否每个感染了新冠肺炎的人都会获得某种类型的免疫力，如果是的话，会持续多久。对于这种类型的系统，也有很多法律和伦理上的挑战。

需要明确的是:新公布的 COVID 护照采取了不同的方法:它们不是提供免疫证明，而是充当“数字健康通行证”，乘客可以在抵达不同国家时向边境代理提供经认证的新冠肺炎测试结果， [阿法尔报道](https://www.afar.com/magazine/new-covid-passport-being-tested-at-newark-and-heathrow) 。它们被称为 [CommonPass](https://thecommonsproject.org/commonpass) ，由总部位于瑞士的非营利组织 [Commons Project](https://thecommonsproject.org) 和世界经济论坛开发。到目前为止，该组织已经向至少 37 个国家介绍了他们的想法。

美国海关和边境保护局(CBP) [现场运营副执行主任黛安·萨巴蒂诺告诉阿法尔](https://www.afar.com/magazine/new-covid-passport-being-tested-at-newark-and-heathrow) “由于乔维德·疫情，旅行和旅游已经全面下降”。“CBP 希望成为建立航空旅行信心的解决方案的一部分，我们很高兴帮助航空业和我们的联邦合作伙伴(启动)像 CommonPass 这样的试点。”

此时，联合航空公司和国泰航空公司已经开始测试 CommonPass，为伦敦和纽约之间以及香港和新加坡之间的特定航班的乘客提供选择。

### CommonPass 是如何工作的？

基本上，这需要很多步骤和国际合作。我们让远处的米歇尔·巴兰来解释一下 :

> 每个参与国必须决定两件事——第一，哪些新冠肺炎测试和实验室结果被认为是可信的。例如，在美国，等州要求的现行标准已经变成由 [临床实验室改进修正案](https://www.cms.gov/Regulations-and-Guidance/Legislation/CLIA) (CLIA)认证的实验室管理的 PCR 测试(也称为鼻拭子测试)。
> 
> 第二，每个政府必须确定其准入要求是什么。一旦这些都建立起来，CommonPass 就会创建一个拥有认证实验室的数字网络，这样旅行者的结果就可以上传到平台上。当旅行者进入他们的目的地时，他们会得到他们需要满足的入境要求。
> 
> 旅行者在经过认证的实验室进行新冠肺炎测试并将结果上传到手机后，他们可以完成目的地国家要求的任何其他健康问卷。CommonPass 确认合规，并生成一个二维码，航空公司工作人员和边境官员可以通过移动设备或打印出来的设备扫描该二维码。

目前的试验是这一过程中非常重要的一部分，让乘客、航空公司和边境代理有机会从头到尾了解他们是如何工作的。

### 如何知道 CommonPass 是否适合自己？

首先，除非你被安排乘坐美国联合航空公司和国泰航空公司的航班，否则这不是你今天必须做出决定的事情，因为我们还没有到这种标准做法的地步。目前，参加试运行的人不能免除政府的任何旅行限制，比如强制隔离。虽然将来可能会改变，但现在，你必须和其他人一样遵守同样的规则。

美国疾病控制和预防中心(CDC) [前首席战略和创新官布拉德利·帕金斯(Bradley Perkins)博士告诉阿法尔](https://www.afar.com/magazine/new-covid-passport-being-tested-at-newark-and-heathrow) “如果没有能力跨越国际边界信任新冠肺炎测试——以及最终的疫苗记录，许多国家将被迫保留全面的旅行禁令和强制检疫，只要疫情持续存在。珀金斯现在是 Commons 项目的首席医疗官，他指出，拥有 CommonPass 这样的国际数字健康信息共享平台可能是重新开放边境的关键。

#### 测试问题

在这个阶段，我们离这成为现实还有很多步，包括世界各国达成一项新冠肺炎测试和结果被认为可信的协议。同样重要的是要记住，目前，COVID 测试只能提供特定时间内关于一个人是否感染了病毒的信息。把它交给 [Lifehacker 的高级健康编辑贝丝·斯克沃里克](https://lifehacker.com/testing-alone-doesnt-stop-the-virus-1845278333) 我来解释一下:

> 一个人可能感染了冠状病毒，但测试结果仍然是阴性。这可能是因为他们得到了假阴性结果。其中一个可能的原因是:也许拭子没有获得足够的病毒 RNA，或者也许这个人处于感染的早期阶段，很难检测出来。FDA 在一份关于 PCR 检测 的 [说明书中指出，“阴性结果并不能排除新冠肺炎，也不应作为治疗或患者管理决策的唯一依据。一个否定的结果不排除新冠肺炎的可能性。”](https://www.fda.gov/media/134920/download)

使用 COVID 测试作为重新开放国际边界的手段，是对一个尚未完善的过程的极大信任。

#### 数据问题

您可能还想知道在 CommonPass 中收集和存储的所有个人健康信息会发生什么情况。根据公地项目 的说法，他们的框架将“允许个人访问他们的实验室结果和疫苗接种记录，并同意将这些信息用于验证他们的 COVID 状态，而不会透露任何其他潜在的个人健康信息。”

乘客将能够通过现有的健康数据系统、国家或地方登记处或个人数字健康记录(iOS 系统的 [苹果健康](https://www.apple.com/healthcare/health-records) 、安卓系统的 [公共健康](https://thecommonsproject.org/commonhealth) )来访问他们的实验室结果和疫苗接种记录。Commons 项目 [还声称](https://thecommonsproject.org/commonpass) “苹果健康和 CommonHealth 让个人在他们的手机上安全和私密地存储他们的健康记录，完全在他们的控制之下。”

虽然推出这样的系统可能需要一段时间，但理解它应该如何工作是决定 CommonPass 是否是您想要尝试的东西的重要的第一步。
