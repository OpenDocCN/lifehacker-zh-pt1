# 什么时候不使用魔法橡皮擦

> 原文:[https://life hacker . com/when-not-to-use-a-magic-eraser-1846575729](https://lifehacker.com/when-not-to-use-a-magic-eraser-1846575729)

神奇的橡皮擦是用途极其广泛的清洁产品，能够清除各种表面上最顽固的污渍和坚硬的粘糊糊的T4。但是有时候，他们也做得很好，除了粘稠物之外还会去掉一点油漆或清漆。这是因为魔法橡皮擦是一种研磨性清洁剂，并不是这样销售的。为了避免灾难，你需要记住这一点。

尽管它们看起来像海绵，但神奇的橡皮实际上是三聚氰胺泡沫块，这是一种多孔材料，并不像听起来那么可爱。根据HowStuffWorks.com的说法，三聚氰胺泡沫是由硬化的三聚氰胺树脂线包裹的微小气泡制成的。这些线完整时非常坚硬，断裂时非常锋利，就像微小的玻璃碎片。在表面上摩擦神奇的橡皮擦使其强大的功能组合发挥作用:三聚氰胺树脂将东西擦掉，气泡立即吸收它。如果有任何剩余，一点额外的压力会将三聚氰胺分解成微小的颗粒，这增加了更多的研磨力。这就像用砂纸来吸尘一样。

一旦你明白了魔法橡皮擦的工作原理，确定何时不用就变得容易多了。如果你永远不会在特定的表面上使用砂纸——即使是最好的砂砾——也不要在上面使用魔法橡皮擦。这消除了大多数油漆，搪瓷，清漆，密封，或以其他方式涂层表面，你可能会认识到最常见的受害者的魔术橡皮擦相关的犯罪。不锈钢也属于这一类，因为神奇的橡皮擦会使其光滑的表面永久变暗。基本上，如果有一个闪亮的涂层，远离魔术橡皮擦。

你还应该记住，就像砂纸一样，没有一点点的努力，魔法橡皮擦是不会起作用的。例如，如果你把一块神奇的橡皮擦切碎，扔进马桶里——出于某种原因，我看到有人推荐这样做——它不会在你冲水的时候神奇地把碗“擦洗”干净。气泡和三聚氰胺的混合物需要在物体上摩擦，才能洗掉并吸收证据；将它溶解在水中会违背初衷。你最好使用[WD-40](https://lifehacker.com/use-wd-40-to-break-down-tough-toilet-stains-5918339)[阿肯色](https://lifehacker.com/clean-your-toilet-with-alka-seltzer-tablets-5800315) ，甚至 [漱口水](https://lifehacker.com/mouthwash-works-pretty-well-as-an-improvised-toilet-cle-5977502) ，所有这些都会*实际上*溶解尘垢。

除此之外，大多数表面都是魔术橡皮擦的目标。先测试一小块看不见的地方总是一个好主意，尤其是在你以前从未使用过魔法橡皮擦的表面上。但是真的，只要你避免闪亮的东西——并且记住即使是“神奇的”清洁产品也必须遵守物理定律——你应该没问题。
