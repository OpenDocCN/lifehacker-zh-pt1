# 每个人都应该关注的 Twitter 机器人

> 原文：<https://lifehacker.com/the-twitter-bots-everyone-should-follow-1849630465>

自动化的 Twitter 账户是许多垃圾邮件的罪魁祸首，但也有例外。Twitter 实际上有一些很棒的机器人，它们添加了有用的功能，比如给照片添加颜色、解决谜题和抓取推文截图。这里是我们最喜欢的七个。

## Chessvision AI

如果你是个棋迷，你应该试试 Chessvision 的 [Twitter bot](https://twitter.com/ChessvisionAI) 。下次你在推特上看到棋盘的截图时，只需用**@ chesvisionai scan**回复推文，AI 机器人就会告诉你哪一方赢的几率更大。你可以用**扫白**或者**扫黑**来回复，如果你想告诉机器人你想让哪一方赢，机器人会分享一个解决方案，一个棋盘分析的链接，还有视频让你了解更多。

## 线程阅读器

病毒性 Twitter 帖子可能很难处理，所以 [ThreadReader](https://twitter.com/threadreaderapp) 从任何 Twitter 帖子中收集整个文本，并将推文转换成一篇简单的文章。只要用 **@threadreaderapp unroll** 回复任何一条推文，机器人就会完成剩下的工作。

## “登月”行动

对太空感兴趣？看看 [Moonshot Bot](https://twitter.com/moonshotbot) ，它每天四次在推特上发布来自阿波罗计划档案的图片。它还链接到 [Flickr 页面](https://www.flickr.com/photos/projectapolloarchive/) 以存档每条推文，所以你可以随时查看更多美国宇航局载人登月任务的照片。

## 提醒我这件事

无论是一场比赛的开始，还是黑色星期五销售的开始，一个好的提醒机器人可以帮助你记住任何重要的事情。 [@RemindMe_OfThis](https://twitter.com/RemindMe_OfThis) 是 Twitter 上最好的提醒机器人。用 **@RemindMe_OfThis in X days** 回复任何一条推文，其中 X 是一个数字，机器人会如期回复你。只要记得启用回复通知，以避免错过他们。你也可以让机器人在 X 分钟、小时或年后提醒你。

## 诗人 _ 这个

[@Poet_This](https://twitter.com/poet_this) 是一个你应该检查一下的抓取推文截图的机器人。用 **@Poet_This** 回复任何一条推文，机器人都会分享一张推文截图。你也可以在提到机器人的同时加上一些关键词来定制截图的外观。如果你想截屏一个完整的线程，提及机器人并使用关键字 **all** 。如果你正在寻找替代品，那么 [截屏](https://twitter.com/_screenshoter/) 也值得一试。

## 彩色机器人

下次你想让机器人给黑白照片添加颜色时，试试 [给机器人](https://twitter.com/colorize_bot) 上色。当你看到一条推文中有任何类型的媒体时，在回复中标记这个机器人，它会为你提供一个新的彩色图像。

## SaveVidBot

如果你想下载 Twitter 视频，可以在回复中标记 [SaveVidBot](https://twitter.com/SaveVidBot/) 。它会生成一个链接，你可以从那里下载视频。如果这个机器人达到了 Twitter API 的限制，你可以尝试一个替代方案，比如 [下载机器人](https://twitter.com/DownloaderBot) 。
