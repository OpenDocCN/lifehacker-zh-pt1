# 使用这个机器人，而不是用书签标记推文

> 原文：<https://lifehacker.com/instead-of-bookmarking-tweets-use-this-bot-1846330965>

很难跟踪每个人的情绪。据 TechJury 称，社交媒体平台平均每分钟有 528，780 条新消息*，*因此，只需更新一两次消息，这些有用的消息就会消失在数字的深渊中。当然，你可以将你最喜欢的一首歌加入书签，然后再去找，但是你真的会经常浏览你最喜欢的歌和书签吗？即使你这样做了，你也可能收藏了数百个甚至数千个，这让你很难在所有的收藏中找到你想在 2018 年保存的东西。



幸运的是，有一款名为 RemindMe_OfThis 的推特机器人可以帮你不费吹灰之力就记住 t weets。Twitter 用户[@ the salvah](https://twitter.com/theshalvah)创建了这个机器人，并对其进行编程，使其在被标记时自动提醒用户一个 t weet。它可以在你想要的任何时间间隔提醒你——几分钟，一周，一年，或者更久——是的，它真的有效。

### 如何创建此提醒的提醒

你可能见过其他用户在帖子中标记 RemindMe_OfThis 以备后用，但如果你不熟悉，我们会告诉你如何使用它。

1.  要设置自动提醒，请回复 t weet 并标记此提醒。包括你想被提醒的时间。例如:“@ RemindMe _ OfThis 周后提醒我这件事。”
2.  机器人将回复确认提醒是计划好的。你也可以再次回复取消。
3.  机器人会在指定的时间范围后给你发送一个 t weet。它会显示在您的 Twitter 通知和此订阅源的提醒中。请注意，提醒这个 weets 是公开的，所以其他人可以看到谁是这个机器人 weets 以及为什么。

### 将此提醒用作应用程序

虽然您可以通过在 Twitter 上与帐户互动来设置 RemindMe_OfThis，但如果您将 RemindMe_OfThis 添加为连接的应用程序，您将获得更多功能。你仍然可以用同样的方式设置提醒，但是你可以在应用程序的设置中更新或取消即将到来的提醒。你也可以在浏览器中启用推送通知，无论何时提醒你这个事件。

要将应用程序添加到 Twitter，请进入[remind meofthis . app](https://remindmeofthis.app/)并点击**“登录”**，然后使用您的 Twitter 帐户登录。这将允许 RemindMe_OfThis 访问某些个人资料信息，但如果你想知道它如何使用你的信息，可以在官方网站上查看该应用的隐私政策和源代码。

一旦应用程序连接到您的 Twitter 个人资料，您就可以在 Twitter 上的[【remindmeofthis.app/settings】](https://remindmeofthis.app/settings)或通过前往**设置&隐私>安全和帐户访问>应用程序和会话>已连接的应用程序> RemindMe_OfThis** 访问其设置。
