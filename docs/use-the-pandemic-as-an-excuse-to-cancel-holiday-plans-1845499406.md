# 以疫情为借口取消度假计划

> 原文:[https://life hacker . com/use-the-疫情作为取消假期计划的借口-1845499406](https://lifehacker.com/use-the-pandemic-as-an-excuse-to-cancel-holiday-plans-1845499406)

我们中的一些人喜欢旅游，或者在假期与家人、朋友和同事共度时光。有些人不知道。对于后一类人来说，我有一个好消息:你有一个完美的内在借口来退出任何和所有你不想参加的假期计划。(以防你不知道我说的是哪个借口:这是我们目前所处的疫情。)

Watch

首先，这不仅仅是一个“借口”。减少任何形式的聚会都是一个好主意，选择不参加晚宴和聚会——即使是社交距离较远的那种——总是最安全、风险最低的选择。

但这并不意味着你不能享受这个假期拒绝各种邀请。今年，我们有了一个铁一般的理由来避免乘坐飞机、火车和汽车去见那些你无论如何也不想与之共度太多时光的人，比如不相信戴口罩的大家庭，或者有令人讨厌的政治倾向的亲戚。

在较小的范围内，你可以避免参加你老板的节日聚会，你邻居的饼干装饰聚会，或者任何其他你可能不得不经常参加的聚会。

“哦，今年人们甚至都不会开派对了，”你可能会想。这是不正确的。如果我从过去的七个月中学到了什么，那就是喜欢派对的人会找到举办派对的方法，但社交距离较远、租用热灯、户外派对狂欢者无法与简单的“抱歉，我现在不太愿意参加任何形式的社交聚会”相提并论，因为即使是最狂热的派对人士也知道，试图让任何人感到羞耻而现在聚会是非常不合适的(或者至少我希望他们这样做)。

我喜欢和人在一起，所以我预测这个假期对我来说会有点艰难。(我唯一怀念在办公室工作的是办公室假日聚会。)但我可以为缺少有趣的节日聚会而悲伤，同时感谢这小小的一线希望，它让我可以礼貌地拒绝我不想做的事情的邀请，我鼓励你也这样做。 COVID 太可怕了。这个疫情太可怕了。但至少它让人们避免听到任何和所有波旁威士忌燃料的漫谈，你可能不得不坐着吃火鸡。
