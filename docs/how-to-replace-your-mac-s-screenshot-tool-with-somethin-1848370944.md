# 如何把你 Mac 的截图工具换成更好的

> 原文：<https://lifehacker.com/how-to-replace-your-mac-s-screenshot-tool-with-somethin-1848370944>

你的 [Mac 的标准截图工具](https://lifehacker.com/how-to-take-prettier-screenshots-on-a-mac-1828009545) 包含了大多数人捕捉屏幕上显示的图像所需的基本功能，但它还可以更好。它不允许你拍摄 [滚动截图](https://lifehacker.com/5-ways-to-take-long-screenshots-on-your-iphone-1847376761) (你捕捉整个网页的长截图)，其他工具——包括下面讨论的两个——更好地绕过一些网站阻止你截图的尝试。(最近尝试通过 Safari 截图网飞？祝你好运。)

## 浅盐湖

[Shottr](https://shottr.cc/) 是一款用于 Mac 的快速免费截图 app。像默认程序一样，它可以全屏截图或捕捉预定义的区域。它增加了一些必备功能，如滚动屏幕截图、OCR(从图像中复制文本的能力)，以及一个强大的编辑工具，允许你模糊文本或删除图像中的对象。

每次你用 Shottr 截图，它都会打开一个预览窗口，让你快速编辑截图，保存到你的电脑或者复制到剪贴板。如果你想使用延迟截图等功能，可以打开 Shottr 的菜单栏应用，它可以让你在截图 之前设置三秒钟的计时器 [。

如果你需要捕捉一个网站的整个图像，而不需要缩小和牺牲可读性，或者求助于屏幕录制，滚动截图是一个更好的选择。Shottr 包括这个功能，输出一个长的 JPG/PNG 文件，你可以按照自己的速度上下滚动，文件比视频小得多。](https://lifehacker.com/how-to-set-a-screenshot-timer-on-your-mac-1847582837)

编辑功能是这款应用真正区别于默认选项的地方。它们让您能够轻松定制您的屏幕截图，在您使用它们模糊文本或删除其中的某些内容(比如，桌面上可见的应用程序图标)后，无需求助于另一个照片编辑程序。该应用的 OCR 功能也值得一提，因为它在从图像中捕捉文本方面非常出色。当然，你可以使用 macOS Monterey 中的 OCR 功能来选择文本，但这需要你先截屏并在 Preview 中打开文件。Shottr 的 OCR 模式效率更高，允许你定义一个区域并拾取其中的所有文本，而根本不用截图。

您可以使用该应用程序为其主要屏幕截图模式配置键盘快捷键。比如滚动截图可以试试 **Command + Shift +8** 。只需打开 Shottr，进入**热键**选项卡进行设置。

Shottr 目前免费使用。默认情况下，该应用程序会收集一些诊断数据，不过你可以通过 Shottr 偏好设置菜单中的高级选项卡禁用这些数据。

## 干净镜头 X

对于超级用户来说，Cleanshot X 是最好的 Mac 截屏应用，尽管它是有成本的(目前许可费为 29 美元，之后的更新费为 19 美元/年)。它包括所有与 Shottr 相同的编辑功能，以及更多功能，所有这些功能都可以通过偏好设置中的快捷键标签进行定制，在这里，您可以为应用程序所做的基本上所有事情设置快捷键。你可以选择是否让鼠标指针出现在屏幕截图中，甚至可以在活动窗口的屏幕截图中添加一张干净的背景壁纸。

在其众多功能中，Cleanshot X 可以创建滚动截图；将您最近截图的缩略图放在桌面覆盖上，这样您就可以保存、下载、固定或编辑它们；从任何网站复制图像中的文本(有或没有换行符)；并录制您的屏幕。基本上，这就像有人想到了几乎所有的截图用例，并将该功能内置到应用程序中。

Cleanshot X 包括云存储，所以你可以自动将你的截图上传到它的服务器，并通过键盘快捷键将它们的链接复制到你的剪贴板。

您可以通过 30 天的免费试用期试用 Cleanshot X。该软件的许可费用为 29 美元，包括 1GB 的免费云存储空间和一年的免费更新。当一年结束时，你可以继续使用该应用程序，但未来的更新费用为 19 美元/年。

## **如何在不失去快捷访问的情况下替换 Mac 的屏幕截图功能**

如果你不喜欢使用 Mac 的屏幕截图工具，你可以从你的 Mac 上删除 [的快捷键](https://lifehacker.com/customize-your-mac-screenshots-with-these-hotkeys-1841481397) 来禁用它。您可以随时指定不同的键盘快捷键，或者重新指定原始快捷键以再次开始使用它。为此，进入**系统偏好>键盘**，打开**快捷键**标签。在这里，您可以从左侧窗格中选择**截图**，并取消选择右侧窗格中的所有选项。

这将禁用默认的截图工具，你现在可以安装上面提到的应用程序之一，用更好的东西替换它。
