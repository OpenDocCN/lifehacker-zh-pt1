# 和“雾居者”一起停留在神游状态

> 原文：<https://lifehacker.com/stay-awhile-in-a-fugue-state-with-the-fog-dweller-1847612900>

我现在正坐在我朋友红钩公寓的餐厅里，坐在一张庄严而漂亮的胡桃木桌子旁，这张桌子已经被我的“生产力”玷污了；笔记本电脑、报纸、我可能不会抽出时间阅读的书籍、一副坏了但仍能正常使用的耳机、皱巴巴的收据和一个大的部分使用过的纸巾套——甚至没有在穿孔的接缝处撕开——我拒绝扔掉这些东西，以防我需要它。我的头发有明显的未经清洗和蓬乱的发型。我穿着一件纽约尼克斯队的长袖衬衫，这是我的朋友借给我的，因为天气已经变冷了一点，当然，尽管我在旅途中带了超大的赫歇尔怪兽，我还是设法不根据天气的变化习惯打包。他告诉我，这是一件特别版衬衫。我默默点头。我对体育一窍不通。我以为他是大都会队的球迷。

我和桌子对面的这只塑料火烈鸟已经对视了 10 分钟了。漫不经心的观察者可能会注意到，这只无生命的鸟一眨不眨、布满灰尘的眼睛似乎在传达一种比它凝视的有生命、有呼吸的物体更热切的表情。火烈鸟没有名字(据我所知)，但它随意地栖息和倚靠在一个宽敞的花盆里，这是一棵名叫乔治的大而快乐的佛焰苞的家。

乔治已经活了相当长一段时间，看到了一些狗屎。十三年前从卡罗尔花园搬到红钩镇；飓风桑迪；我的朋友在盛夏外出三个星期时，隔壁邻居忘记给它们浇水。昨晚，当艾达在纽约发表毁灭性的恐怖演说时，乔治和红鹤似乎不知所措；他们的存在没有改变。我可以从他们的书里拿走一页。与此同时，现在，阳光灿烂，天空蔚蓝(天气和它的习惯正在改变)，一辆汽车开着它的扬声器超时工作；也许这是另一个值得我去做的提示。

虽然我也有被艾达独自留下的荒谬运气，但我不像我的同桌那样镇定，也不像外面那样活跃。今天，我宿醉未醒，迷雾重重。

## 雾切割器？一个雾- *居民*怎么样。

我相信你们大多数人都听说过 Fog Cutter:这是一种 Tiki 鸡尾酒，由贸易商 Vic 在二战期间发明，将奶油雪利酒漂浮在柑橘汁、橙汁和三种不同烈酒(通常是白色朗姆酒、白兰地和杜松子酒)的混合物上。但是，对我来说，假装有能力穿透任何类型的迷雾，都是不礼貌和不真诚的(超出了本专栏的范围)。盲人给盲人带路。

所以，这是另一种饮料。我们就叫它雾居者吧。我已经两年没喝过像样的陈年龙芽草了，所以我打算趁我还能喝的时候好好利用它。隔壁酒吧很有礼貌地给了我华丽的鹅卵石碎冰。我的红钩主人碰巧也是我最喜欢的 tiki 杯的主人，一个停产的熔岩杯，他不能被说服卖给我，但会让我在这里使用。一种由让我感觉好一点的小事组成的饮料。

如果你愿意，试试看:

*   1 盎司酸橙汁
*   一盎司吉法德的杏酒(我必须坚持你用吉法德的，因为它令人愉快)
*   盎司甘蔗糖浆(或简单的)
*   2 盎司陈年龙芽草(我不会坚持，但我强烈推荐 JM 龙芽草)
*   磨碎肉豆蔻。

将所有原料(除了肉豆蔻)放入平底玻璃杯中，充分搅拌，使原料融合。在提基杯中装满碎冰或碎冰，滤入四分之一的液体混合物，然后用平底玻璃杯的底部压实冰块。(液体会将冰融化一点，为你创造更多的空间，以获得最大的碎冰容量。)

加入更多的冰，将剩余的液体滤入盛满冰的杯子。在上面放更多的冰，然后在上面撒上大量新鲜磨碎的肉豆蔻粉。让它变成雪或者雾，这会给演示带来一些戏剧性，但也有助于巩固味道。
