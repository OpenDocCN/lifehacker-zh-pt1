# 你应该用刀子敲瓶子来打开它吗？

> 原文:[https://life hacker . com/should-you-bottle-by-beating-it-1845203900](https://lifehacker.com/should-you-open-a-bottle-by-hitting-it-with-a-knife-1845203900)

你有没有去冰箱里拿一杯装在玻璃瓶里的冰冻饮料，却发现你把可信赖的开瓶器放错了地方？嗯，就像这些天的大部分事情一样，抖音会罩着你的。

视频应用程序上最近的一个趋势是，用户用黄油刀以特定的节奏敲击瓶子，以瓶子顶部(和盖子)的摆动结束。显然，你所要做的就是轻击瓶身两次，然后轻击瓶颈，然后用力摇晃瓶盖，你就可以像对待一瓶Dom酒一样把它喝掉了。

因为这看起来像是到处都是碎玻璃的配方，包括在你的饮料中，我们决定测试#openbottlechallenge。我们有时在一天的早些时候拍摄这些视频，所以我选择了姜汁啤酒，而乔丹选择了令人愉快的粉红色覆盆子柠檬水。

但是，即使这种黑客抹杀了瓶盖，它仍然是最简单，最安全的方法来打开一个没有正常的开瓶器的瓶子吗？看看上面的视频就知道了。
