# 如何克服“变焦疲劳”

> 原文：<https://lifehacker.com/how-to-overcome-zoom-fatigue-1846347171>

在整个疫情，许多以前在办公室工作的人不得不粘在他们的电脑显示器前。随着工作转移到网上， 像 Zoom 和 Google Hangouts 这样的视频工具已经成为与同事定期见面的罕见渠道。但是由于缺少一种可以在没有屏幕的情况下看到你的同事的替代方式，所有这些视频会议导致了“变焦疲劳”的流行



根据斯坦福大学研究人员发表在《技术、思维和行为[](https://tmb.apaopen.org/pub/nonverbal-overload/)*杂志上的一项新研究，变焦疲劳基本上就像它听起来的那样——源于通过视频聊天保持远距离联系的压力增加——它导致工作倦怠、压力、和单调。但是有一些方法可以减轻视频会议对你精神的束缚。*

### *什么是变焦疲劳？*

*它并不专门适用于 Zoom，该公司的高管可能会认为这个术语对他们的营销努力有害无益。据斯坦福虚拟人类互动实验室的创始主任杰里米·贝伦森称，这个问题适用于所有的视频会议服务。G 总的来说，它描述了当你在浏览器窗口间切换以参加各种在线会议时，需要一直保持开机状态所带来的疲劳。这也是有道理的， [鉴于研究表明](http://sciencedirect.com/science/article/pii/S2211335517301316) 增加屏幕时间——尤其是与久坐的生活方式相结合——会增加你患上中度至重度抑郁症的几率。*

*如果你患有这种病，你可能通常会淹没在繁重的虚拟会议日程中，感觉自己几乎无法让 T4 保持清醒。*

### *是什么原因造成的*

*Bailenson 的研究指出了视频会议如此耗费精神的四个原因:*

*   *激烈的眼神交流令人疲惫。与你的同事对视以表示你在注意听会让人感觉很吃力。一天多次这样做会感到压抑。在会议的大部分时间里，如果缺乏一致的眼神交流，你的同事可能会认为你的注意力正在下降。*
*   *视频聊天时看着自己很累。在会议中观察自己只会加剧表现焦虑。在整个疫情生活的心理成本已经够沉重的了——为什么还要担心你在同事眼中的形象呢？* 
*   *视频聊天意味着我们走动得更少了。如果你一直被束缚在桌子上，你就不能像身体需要的那样四处活动。至少在传统的办公环境中，你可能不得不走到不同楼层的会议室。在不同的视频会议之间切换意味着我们坐得更多，动得更少， [这不利于我们的精神健康。](https://www.health.harvard.edu/blog/how-simply-moving-benefits-your-mental-health-201603289350)*
*   *非语言暗示更难理解。解读非语言暗示的挑战只会增加视频聊天给带来的压力。这可能会导致拜伦森所说的“认知超载”，你的脑袋可能会沉浸在对话中假设的潜台词中。*

### *对抗变焦疲劳的方法*

*幸运的是，Bailenson 没有在没有提供解决方案的情况下发现问题。*

*   ***对于眼神接触:**研究人员建议不要使用全屏设置。这样你的同事至少会看起来小一点，这样你就不会有压力去盯着他们的眼睛看。*
*   *为了自我意识:没必要每次开会都开着相机。如果你不是在呈现什么，那你拍自己有什么意义？如果你必须开着相机， Bailenson 建议调整你的设置，这样你就只能看到聊天中的另一个人，而不是双方都可以看到两个视频。与此同时，不要犹豫，关掉你的相机。*
*   ***对于移动性:** Bailenson 建议使用一个不同的相机，你可以连接到你的 feed，这样你仍然可以四处移动，如果你愿意的话，也许可以站着展示。另一个的方法是再次关闭你的相机，到戴上蓝牙耳机，这样你就可以在你的房子或公寓里走动了。*
*   *对于对非语言暗示的焦虑:关掉你的相机也很好，但是为了进一步加强这一点，研究人员建议在离开你的电脑时听会议。这样，如果你只使用音频，并且在参加会议时感到舒适，比如说，收拾碗碟，你就不会担心过度分析所有经常弹出的微提示。*
