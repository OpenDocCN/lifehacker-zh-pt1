# 试着用干库拉索酒代替甜苦艾酒

> 原文:[https://life hacker . com/try-using-dry-curacao-inst-of-sweet-苦艾酒-1846161412](https://lifehacker.com/try-using-dry-curacao-instead-of-sweet-vermouth-1846161412)

似乎我总是缺少什么。今天我本打算给你做一杯叫做 [的波旁鸡尾酒](https://www.diffordsguide.com/cocktails/recipe/3438/game-set-match) ，但是这不会发生。虽然我有必备的波旁酒和辛纳酒(每种都有很多)，但我(不知何故)完全没有甜苦艾酒了，这让我很难过。(就*等我查清楚*谁把它全喝了还不换掉。)

事实上，我已经摆脱了*所有*类似苦艾酒的修饰词。当我在我的酒吧推车里寻找的时候，我的目光落在一瓶积满灰尘的Pierre Ferrand干库拉索葡萄酒上，这种葡萄酒虽然明显不是苦艾酒，但看起来像是瓶装的波旁酒(我唯一喝过的波旁酒)和木质洋蓟酒的组合。

我是对的。橙色是一种非常令人愉快的味道，和其他的味道也很好。与朴实、森林般的西纳葡萄酒搭配时，感觉尤其美妙。当与波旁威士忌混合时，所有这三种成分创造了一种美味、强烈、细致入微的饮料。因为我们要处理两种利口酒，所以重要的是把波旁酒倒得重一点，保持另外两种量相当小。当有节制地使用时，干库拉索是可爱的，但太多会使你的饮料变得令人作呕，特别是当与另一种利口酒混合时。要制作这种酒，美味饮料，你需要:

*   2盎司波旁威士忌
*   1/2盎司辛纳
*   1/2盎司干库拉索

将所有材料加入一个有裂缝的玻璃杯中，搅拌至非常非常冷。如果你觉得必须保持主题不变，可以滤入一辆跑车，用一条桔皮装饰。
