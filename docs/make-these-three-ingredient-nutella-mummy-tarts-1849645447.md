# 做这些三种成分的果仁蜜饼

> 原文：<https://lifehacker.com/make-these-three-ingredient-nutella-mummy-tarts-1849645447>

由于其结构、丰富的风味和多功能性，酥皮点心是几十种精致甜点中令人印象深刻的主要成分。这也正是我做一份完全不复杂的万圣节甜点所需要的。这些木乃伊果仁馅饼只需要三种配料和一个烤箱——你甚至不需要任何艺术技巧或灵巧。最难的部分可能是用开罐的 Nutella 展示部分自制力。



木乃伊是我最喜欢的万圣节流氓画廊中的之一，不仅因为它让我想起了一个 [迷人的古埃及仪式](https://www.worldhistory.org/Egyptian_Burial/) 和布兰登·弗雷泽，还因为没有让它们看起来整洁的压力。任何畸形的肢体或松软的包装都会增加它们的魅力，如果你不是特别熟练的装饰师，还会给你一点犯错的空间。不像女巫、吸血鬼或弗兰肯斯坦的怪物，你甚至不用担心构造一张脸；最多，他们在腐烂的布条后面用两只小眼睛盯着你。幸运的是，几乎所有的圆形糖果都能让你有一双迷人的小眼睛。

从解冻的酥皮点心开始；商店里买的冷冻泡芙很好，甚至可能更好，但要确保包装上标明全黄油，因为味道比用起酥油做的更好。在你打算使用面团之前，至少在冰箱里解冻面团八小时。检查包装的解冻说明，因为必要时你可能可以把冷冻糕点放在柜台上一个小时—但是要小心这种方法，因为面包卷外面有过度解冻的风险。酥皮糕点由交替的超薄黄油和面团层制成，如果太软，就会变得脆弱。如果你在面团太软的时候处理它，你可能会把面团分层，有效地降低它的膨胀能力，所以试着提前考虑，在冰箱里解冻它。

展开一张酥皮饼，切下并留出矩形的三分之一作为包装纸(把它放在冰箱里保持凉爽，而你的工作】。将剩余的松饼切成六个大小相等的长方形，我立刻将它们放在铺有羊皮纸的烤盘上。(如果你把它们装饰在烤盘上，你就不用担心以后移动它们了，当它们变得更热并且充满果仁酱的时候。)

用一个小勺子的背面，在每个长方形上涂上一层薄薄的巧克力榛子酱，在外边缘留下四分之一英寸的裸露边缘。抵制过度的 N 乌特拉。我知道。这也违背了我的每一根纤维，但是你想要一层薄的不透明层。如果你做得太多，它可能会融化并从油酥面团中流出，导致一个令人伤心的烧焦的烂摊子。此外，边框也给了木乃伊条一个粘贴的地方。

一旦所有的长方形都涂上了果仁酱，从冰箱里取出分开的面团。用小刀或披萨刀，将油酥面团切成大约 1/4 英寸厚的长条。接下来你将自定义长度。切一条比每个馅饼宽度稍长的面皮。以一定的角度把它横放在馅饼上，这样它就可以接触到馅饼两面的边缘。使用这种最初的切割来帮助你隔开剩余的部分，用纵横交错的面团条覆盖每个馅饼，直到你在矩形的整个长度上形成交叉点。你应该能看到下面展开的果仁酱。为了让表面看起来更有光泽，在面皮上轻轻抹一点 [蛋液](https://www.thekitchn.com/egg-wash-recipe-23390924) 。在预热到 400 华氏度的烤箱里烘烤馅饼 12-15 分钟，或者直到它们膨胀成金黄色。

用糖果眼睛完成这些可怕的馅饼。我偏爱里斯牌的，但是你可以用 M & Ms、巧克力片、杏仁片或花生。要画瞳孔，用牙签在每只眼睛上沾一点 Nutella。
