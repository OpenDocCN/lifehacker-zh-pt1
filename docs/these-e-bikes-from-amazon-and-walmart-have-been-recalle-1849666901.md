# 这些来自亚马逊和沃尔玛的电动自行车已经被召回

> 原文：<https://lifehacker.com/these-e-bikes-from-amazon-and-walmart-have-been-recalle-1849666901>

美国消费者产品安全委员会 上周宣布，亚马逊、沃尔玛、易贝、Overstock 和西尔斯(以及其他零售商)在线销售的约 22，000 辆电动自行车被召回。电动自行车的制造商， [安彻尔公司表示，他们之所以主动召回](https://www.ancheer.shop/pages/recalls) ，是因为“在某些情况下，锂离子电池可能会点燃、爆炸或产生火花，对消费者造成火灾、爆炸和烧伤的危险。”



## 哪些自行车正在被召回？

此次召回影响了型号为 AM001907 的安彻尔电动自行车:根据美国消费品安全委员会 的数据，到目前为止，该公司已经收到了六起涉及骑手起火、爆炸或火花的事件报告 [。其中四起事件涉及烧伤。](https://www.cpsc.gov/Recalls/2023/E-Bikes-Recalled-Due-to-Fire-Explosion-and-Burn-Hazards-Distributed-by-Ancheer)

自行车的型号可以在包装和使用手册上找到，但不会在自行车本身上。知道你有被召回的型号的一种方法是，圆柱形电池像一个水瓶，就在自行车框架上的“Ancheer”白色字体标志的上方。这辆自行车也有 26 英寸的车轮。
