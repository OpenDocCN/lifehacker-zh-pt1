# 如何使用渐进霸王变强

> 原文:[https://life hacker . com/how-to-use-progressive-overload-to-get-strong-1847321428](https://lifehacker.com/how-to-use-progressive-overload-to-get-stronger-1847321428)

随着你变得越来越强壮，你就能举起越来越重的东西。这可能是显而易见的。但是初学者有时会忽略的是，这有点像先有鸡还是先有蛋的情况:你变得更强壮是因为你举起越来越重的重量。这就是渐进式超载的概念。

## 渐进霸王是怎么运作的？

在健身的任何领域，如力量、耐力或柔韧性，你都需要挑战你的身体，让它适应和提高。正如 [国家力量与训练协会解释的](https://www.nsca.com/contentassets/116c55d64e1343d2b264e05aaf158a91/basics_of_strength_and_conditioning_manual.pdf) ，这个有两部分:t 他“超负荷原理”和“渐进原理”

超负荷原则指出，“为了使一个人达到一定的训练适应性，身体必须承受比它所习惯的更大的刺激或负荷。”换句话说，你必须比你的身体所习惯的更加努力。

然后，循序渐进的原则让你可以在此基础上继续发展:“为了持续地达到某项活动或技能所需的训练适应性，训练刺激必须逐渐地、持续地增加。”

或者简单地说:如果你比平时更加努力，你会变得更健康，如果你想不断进步，你就必须比以前更加努力。

## 这在现实生活中是什么样子的？

如果你对永远越来越努力工作的想法感到沮丧，不要惊慌。从绝对角度来看，你会更加努力——比如说举更重的重量——但从相对角度来看，挑战在T2是一样的。你的锻炼将会达到你可能称之为“艰难但可行”的努力水平，你会注意到进步，因为你的数字在上升。(同样的方法也适用于耐力运动。据报道，正如自行车运动员格雷格·莱蒙德所说:“事情从来没有变得更容易，你只是跑得更快。”)

当我多年前开始举重时，65磅对我来说是一个相当有挑战性的卧推。我记得我为自己能在85磅的时候挤出一两个代表而感到自豪。现在，如果我要做一个长凳锻炼，我甚至都懒得把这些量放到横杠上。我的热身套装起价为95磅，一个重的单曲大概在150英镑左右。150磅和85磅一样重，但不可否认的是重量更大了。

我是如何取得进步的？嗯，我一直举着对我来说很重的东西。随着时间的推移，过去具有挑战性的相同重量开始变得容易，我需要在杠铃上增加越来越多的重量，以获得实际上感觉很重的东西。( [)我们这里有一个向导](https://lifehacker.com/what-does-it-mean-to-lift-heavy-1846309501) 来判断你是否在举“重物”)

大多数时候，我要么遵循一个告诉我举起多少磅的程序，随着时间的推移慢慢增加这个量，要么遵循一个告诉我在什么努力水平下举起的程序(一个叫做RPE的概念)，这允许我每天选择一个合适的重量。遵循一个计划通常会比即兴发挥获得更好的进步，但是只要你使用超负荷原则和进步原则，你*就会*变得更强。

## 每次都举不动更多怎么办？

没关系！你不必每次举重都增加杠铃的重量。

有各种各样的重量和代表范围可以有效地增强力量和肌肉。例如，如果我今天做了一次长凳锻炼，我可能会在110磅的时候做10组，或者在130磅的时候做5组，或者在150磅的时候做一些重量级的单人运动，或者这些运动的任意组合。如果我真的很累或者压力很大，我可能会决定在100磅的情况下做10组。如果我感觉很好，我也许能在115岁时完成。这就是我所说的广泛范围:所有这些都是足够艰苦的工作，以刺激我的肌肉适应并变得更强壮。(你可能会选择其中一种锻炼方式，这是有原因的，但我们目前不需要深入了解这些细节。)

什么*不会是渐进式过载？好吧，如果我只做10个一组的杠铃，那不会帮助我变得更强壮。如果我有一套迷你杠铃，它的最大重量是85磅，一旦我达到85磅不再是一个具有挑战性的重量，我的力量就会停滞不前。*

即使你的力量提高了，你也不必每次锻炼都做得更多，只要从长远来看你越来越强，并且你的锻炼仍然在对你有挑战性的范围内。

假设你正在用一个10磅的哑铃做二头肌弯曲。你可以用它做8次或10次。完美。但是在你的健身房，增加体重的唯一方法可能是拿起一个15磅的哑铃。如果这个重量对你来说太重了，没关系。坚持10磅的训练，最终你会为15磅做好准备。

## 您可以在多个指标上取得进步

虽然你可能渴望举起更重的重量，但杠上的重量并不是进步的唯一途径。有时候因为器材问题，或者只是因为力量提升慢，不能增加重量。(即使你的初学者进步神速，每个人的进度都会在某个时候慢下来。)

但是如果你很聪明，你可能不希望T1只在一件特定的事情上做得更好。很多初学者开始做深蹲、仰卧起坐和硬拉，每组五次，并试图在每次锻炼中增加重量。但是，如果你还知道如何举起沉重的单打和10或15组，你会是一个更全面的举重运动员。根据你的目标，你可以考虑除了后蹲外的前蹲，除了硬拉外的反屈或壶铃式摆动。在所有这些方面都有改进的方法，对于一个举重运动员来说，同时增加他们在辅助举重中的重复次数，增加他们在重的单人杠铃上的重量，以及增加他们花在训练上的时间是很正常的。

## 什么是双递进？

一种简单的结合目标的方法叫做双重渐进。这意味着你同时在两个指标上取得进步:重复次数和体重。下面是它的样子:

*   你的项目要求头顶压三组“8-12次”
*   你选择一个你能做8次的重量。
*   每次锻炼，你都要看看是否能增加额外的重复次数。(也许你能做9次、8次和8次。然后另一天你做9，9，9。很快你就达到10秒了。)
*   一旦你可以用那个重量做12组，12组和12组，你就可以换到更重的重量。
*   以8、8和8为一组的新重量重新开始，并重复这个过程。

一个稍微复杂一点的版本是添加集合:从3x8到3x12之后，你可以继续到4x12和5x12。

除了增加更多重量、更多重复次数或更多组，进步还意味着:

*   在两组之间休息得更少
*   做一个更难的练习
*   增加你的活动范围(例如，蹲得更深)
*   用慢动作做练习(例如，节奏深蹲，用三秒钟的时间把自己放下来)

关键是要确保你朝着对你有益的方向前进。例如，如果你能做20个俯卧撑，增加重复次数会让你的肌肉耐力更好，而减少重复次数的较重运动(如杠铃卧推)会让你更强壮。你可以两样都做，但是你应该优先考虑对你更重要的那一个。如果你的人生目标是做100个俯卧撑，那就多做几次！但是如果你想变得更强壮，你最好增加体重。

## 平台期是正常的

最后一件事，现在我们已经讨论了渐进式超载是什么样子。重要的是要记住，进展发生在长期中。一些竞技举重运动员可能不会在比赛之外测试他们的 [单代表最大](https://lifehacker.com/test-your-strength-to-get-an-end-of-year-benchmark-1845915681) ，这意味着他们一年只会发现一两次他们的硬拉增加了多少。这并不意味着他们在此期间没有进步。如果他们在做一个有效的项目，不断挑战自己，他们仍然在工作。

当你是一名举重运动员时，平台期是生活中的现实。有时候变强需要一段时间。有时候，你需要在技巧上多下功夫，才能表现出你新发现的力量。有时，压力、减肥或训练中的变化等因素会让你在短期内变得更弱，但如果你继续以挑战你的方式训练，你很快就会 [设定新的PRs](https://lifehacker.com/when-can-you-expect-a-personal-record-in-the-gym-1846765539) 。
