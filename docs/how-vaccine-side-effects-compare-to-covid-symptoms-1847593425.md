# 疫苗副作用与 COVID 症状相比如何

> 原文：<https://lifehacker.com/how-vaccine-side-effects-compare-to-covid-symptoms-1847593425>

新冠肺炎疫苗因其副作用而有些臭名昭著:疲劳、手臂酸痛，也许还有发烧。在少数罕见的情况下，会有更严重的并发症，如心脏炎症。那么，是接种疫苗风险更大，还是碰碰运气？



《新英格兰医学杂志》上的一项新的研究开始回答这个问题，将人们在服用辉瑞疫苗后经历的中度至严重症状与人们在 COVID 感染后经历的症状进行比较。(注射部位的发烧和疼痛等轻微反应不包括在内。)以下是他们的发现:

*   服用 COVID 的人患心律失常(心律不齐)、心肌梗死(心脏病发作)、深静脉血栓形成(腿部等部位有血凝块)、肺栓塞(肺部有血凝块)、心肌炎(心肌炎症)、心包炎(心脏附近组织炎症)、脑出血(脑部出血)和血小板减少症(血小板计数低)的风险大大增加。

*   接种疫苗的人患淋巴结病(淋巴结肿大)的风险大大增加。

这项研究是在以色列进行的医疗记录分析，它比较了 884，828 名接种疫苗的人和同等数量的未接种疫苗的对照，以及 173，106 名 COVID 感染的人和同等数量的对照。(每个人都与年龄、性别、家庭所在地和健康状况相同的对照组相匹配。)

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1430656144687706118&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1430656144687706118&autosize=1) 

所涉及的疫苗是辉瑞的，所以这一分析可能不适用于其他疫苗，如强生公司的疫苗。Moderna 疫苗与辉瑞公司的 T1 疫苗非常相似，所以如果对 T3 疫苗进行同样的研究，结果很可能是相似的，但是没有数据，我们无法确定。

“这些数据更令人信服的是疫苗对急性肾损伤、颅内出血和贫血等不良事件的实质性保护作用，这可能是因为感染被预防了，”斯坦福大学医学教授格雷斯·李(Grace Lee)写道，他为该研究撰写了一篇 [编辑评论](https://www.nejm.org/doi/full/10.1056/NEJMe2112543?query=recirc_curatedRelated_article#author_affiliations) 。

她还指出，比较风险的最佳方式包括接触 COVID 的机会；如果这是一种罕见的疾病，你更有可能从疫苗中获得严重的不良事件，而不是被感染，这可能会改变风险/收益计算。但是目前肆虐的[δ变种](https://lifehacker.com/how-contagious-is-delta-really-1847394390)，COVID 根本不是什么罕见病。

她写道:“然而，鉴于全球疫情的现状，暴露于新型冠状病毒(导致 COVID 的病毒)的风险似乎是不可避免的。”
