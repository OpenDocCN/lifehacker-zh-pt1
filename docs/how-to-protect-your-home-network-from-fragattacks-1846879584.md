# 如何保护您的家庭网络免受“碎片攻击”

> 原文：<https://lifehacker.com/how-to-protect-your-home-network-from-fragattacks-1846879584>

听到你的无线设备容易受到一种叫做“碎片攻击”的东西的攻击，并不能激发你的喜悦。然而，这个词比风险更可怕；没有证据表明任何人正在通过这些漏洞积极利用无线设备，即使有数百万人可能容易受到碎片攻击——即“碎片和聚合攻击”的简称



正如安全研究员 Mathy van hoef 所写:

> *“发现的漏洞影响了 Wi-Fi 的所有现代安全协议，包括最新的 WPA3 规范。甚至 Wi-Fi 的原始安全协议 WEP 也受到影响。这意味着，自 1997 年发布以来，新发现的几个设计缺陷一直是 Wi-Fi 的一部分！幸运的是，设计缺陷很难被滥用，因为这样做需要用户交互，或者只有在使用不常见的网络设置时才有可能。因此，在实践中，最大的担忧是 Wi-Fi 产品中的编程错误，因为其中有几个很容易被利用。”*

### 什么是碎片攻击？

“FragAttack”这个名称实际上指的是一组与帧或数据包相关的漏洞，这些漏洞可以被用来窃取某人在联网设备之间传递的信息，或者完全接管一台设备——无论是简单的物联网智能交换机还是你在家里用来浏览网页的旧笔记本电脑。攻击者可以将不需要的未加密帧注入网络，也可以利用帧的聚合方式(或聚合的拆分方式)注入并执行原本不存在的数据。

然而，正如 Vanhoef 指出的，攻击者需要在你的网络的无线电范围内才能引起混乱。这已经限制了你的潜在风险，因为这根本不是你可能在家里或公寓里经历的事情(除非你有一个狡猾的邻居)。

## 保护自己免受碎片攻击的简单步骤

让您的网络尽可能安全地抵御 FragAttack 漏洞的最佳方式是让您的设备保持更新——您会注意到，这也是我们针对每个安全漏洞给每个人的建议。确保您的路由器、智能设备、笔记本电脑、手机或任何其他设备都运行您能找到的最新固件和软件更新。如果你幸运的话，你的设备制造商将有办法自动更新它们。否则，您需要确保定期(比如每季度)检查可以修补此类漏洞的关键更新。

例如，Eero [已经更新了](https://blog.eero.com/fragattacks-fragmentation-aggregation-and-attacks-update-available-for-all-eero-customers/) 的路由器，以完全阻止任何 FragAttack 式的漏洞被利用:

> *“研究人员发现的许多漏洞不会影响 eero 网络，因为我们多年来对网络软件进行了定制更改。此外，eeroOS 6.2.1 和更高版本包含一个补丁，可保护您的网络免受“FragAttacks(碎片和聚合攻击)”漏洞的攻击，现已向所有 eero 客户提供。如果你在“设置”标签中看到的版本不是 6.2.1 或更高版本，你可以在移动应用中点击任何 eeros 的详细信息，并触发 OTA 更新。”*

除此之外，确保你在你的浏览器中到处使用类似[HTTPS](https://www.eff.org/https-everywhere)的扩展，这样你就可以一直连接到安全的网站(并且你通过你的设备传输的数据不会被拦截)。此外，我建议在您的路由器和/或设备中手动 [设置自定义 DNS](https://lifehacker.com/how-to-browse-faster-and-more-securely-with-cloudflar-1824256064) ，以帮助阻止任何试图将设备重新路由到恶意 DNS 服务器的攻击。

除此之外，不用太担心。是的，这些漏洞几乎存在于每一个联网的设备中，，但是它们(谢天谢地)足够隐蔽，也足够难以利用(需要足够的物理存在)只要你保持对安全和更新的掌控，你就应该没事——反正你也应该这样做。
