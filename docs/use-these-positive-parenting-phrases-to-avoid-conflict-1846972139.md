# 使用这些积极的育儿短语来避免与孩子发生冲突

> 原文：<https://lifehacker.com/use-these-positive-parenting-phrases-to-avoid-conflict-1846972139>

本周早些时候，我们谈到了如何成为一个更温柔的*父母，这听起来很甜蜜很可爱，但当你和爱发牢骚的孩子相处了一整天后感到疲惫时，很难做到。正如任何试图改变我们行为的尝试一样，如果我们能够创造新的惯例或习惯，它可以帮助我们在那些艰难的时刻自动驾驶——抖音的一位母亲正在教我们如何做到这一点，一次一个简短的视频。*

*我们 [后代脸书社区](https://www.facebook.com/groups/2018785615043946) 的一个成员给我介绍了 [Destini Ann 的一系列正面育儿视频](https://www.tiktok.com/@destini.ann?) ，我一直在看一个又一个，吸收着她的智慧和整体的正能量。但我特别喜欢她制作的四集“插曲”,突出了我们可以用在孩子身上的简单短语或咒语，来设定界限，让他们顺从，同时也能理解他们的感受。*

### *“每一种感觉都是可以接受的；每一个行为都不是。”*

*[在这段视频](https://www.tiktok.com/@destini.ann/video/6954457530995051781) 中，Destini Ann 提醒我们，我们的孩子可以疯狂。而且他们可以对*我们*发火(毕竟我们对*他们*发火)。他们可以随心所欲，但这并不意味着他们可以扔玩具或使用不尊重的语言。你可以验证他们的感觉*和*告诉他们他们的行为是不对的。*

### *“尊重我的‘不’”*

*Destini Ann 在 [这一集](https://www.tiktok.com/@destini.ann/video/6954824959449402630) 中谈到，她在家里没有很多武断的规则，但她确实有三个不容商量的领域:善良、安全和卫生。当她的一个孩子顶撞这三个孩子中的一个时，她会示范她如何同情他们的感受，同时也说这是他们需要“尊重她的‘不’”的时刻*

### *“这是一种取舍吗？”*

*在这段视频中，她谈到了当我们对某件事说“是”时，我们通常会对另一件事说“不”。例如，如果他们想去操场，这可能意味着晚饭前在家里一起玩的时间更少了。所以，当我们知道现在的*是*将导致以后的*否*时，我们可以问我们的孩子是否想“交换”这些东西，教他们如何掌控自己的时间预算。 [喜欢所以](https://www.tiktok.com/@destini.ann/video/6955131657150139654) :*

### *“你们营业吗？”*

*我特别喜欢 [这个](https://www.tiktok.com/@destini.ann/video/6955466645921860870) ，因为这是一个策略，在处理朋友和家人以及他们的孩子时，它会很好地服务于许多成年人。当你的孩子告诉你在学校的一个艰难时刻或向朋友发泄时，问他们:“你开放吗？”意思是，你愿意接受我的意见、反馈或建议吗？*

*Destini Ann 说，这有两个作用。首先，这会促使他们更容易接受你的想法，因为你已经通过询问他们是否想听你的想法来表达了对他们的尊重。其次，这给了他们一个说不的机会，因为这可能是他们更需要向某人发泄的时刻。*

*如果我们开始有意识地将这些积极的短语融入到我们与孩子的日常互动中，它可以帮助减少反复出现的冲突，或者帮助我们完全避免它们。*
