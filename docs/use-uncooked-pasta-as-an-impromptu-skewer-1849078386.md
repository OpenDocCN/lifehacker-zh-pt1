# 用未煮过的意大利面做串烧

> 原文：<https://lifehacker.com/use-uncooked-pasta-as-an-impromptu-skewer-1849078386>

有许多食谱要求人们用串肉扦或牙签“固定”某物。像填馅鸡胸肉、肉馅饼和培根包裹之类的东西——很多这样的小食品包裹需要用一点木头来刺，以帮助它们在烹饪时保持形状。但是如果你没有木头了呢？总是有意大利面条(或者意大利面)。



我在最近一期 [库克画报](https://www.cooksillustrated.com/) 的“快速提示”部分找到了这个提示。这是一位读者发来的，当你没有烤肉叉和牙签时，这个小技巧可以拯救你的屁股:

> 佛罗里达州塔拉西的李·萨特菲尔德(Lee Satterfield)在制作填充鸡胸肉时，突然意识到他没有任何串肉扦或牙签来在烹饪时将生鸡肉串在一起。然而，他手头上确实有一盒布卡蒂尼(意大利宽面条也可以)，长长的面条轻松地将家禽串起来。(此外，没有必要担心不小心咬到尖锐的串肉扦或牙签。

除了填馅鸡胸肉(甚至填馅鸡腿)，你还可以用小块未煮过的意大利面来固定各种各样的东西，比如烤火腿上的菠萝圈和樱桃，扇贝或虾周围的培根，或者一片鱼上的柠檬片。你甚至可以用面条来做开胃菜，比如羊肉串(樱桃番茄、马苏里拉奶酪球、罗勒叶)；或者稳固一个高耸的三明治(或者在三明治的顶部插一颗橄榄)。

然而，我不会用一块意大利面作为烤串。末端会烧焦，面条会在食物里变软，很难从烤架上取下烤串。(松软的烤串不是有效的烤串——我总是这么说。)
