# 制作这款鸡尾酒以纪念完美的桃子

> 原文:[https://life hacker . com/make-this-cocktail-in-memoriam-of-the-perfect-peach-1847390856](https://lifehacker.com/make-this-cocktail-in-memoriam-of-the-perfect-peach-1847390856)

桃子是我恐惧的一大来源。我确信我吃的最后一个好桃子是在我的童年，2001年或2002年，在马其顿斯科普里的Kisela Voda附近的某个地方。最有可能的是，那是我在从一个祖母家到另一个祖母家的路上从邻居的树上摘下来的——它被阳光亲吻着，温暖而毛茸茸的。我会把衬衫上的绒毛擦掉，因为我从惨痛的教训中知道，如果我不这样做，羽绒就会刺痛。(我第一次摘下枝头的桃子如此柔软，让我着迷，我把它贴在脸上，结果发现毛状体刺痛了我的脸颊。)当我咬着茂盛的天鹅绒外皮，享受着味蕾上的交响乐，果汁顺着下巴和双手流下时，我怀疑我没有再考虑这种水果即将面临的濒危状态。

在那个时代，完美的桃子算不上稀有商品。尽管我家住在首都，但我们的邻居更多的是果园，而不是大都市。在夏天，被当作街道的土路上经常散落着各种腐烂状态的核果。在我看来，真正从市场上买桃子的人通常是住在高楼里的朋友，或者是那些负责给老养老金领取者巴巴送水果的人，巴巴因关节炎而严重残疾，无法亲自前往泽伦帕扎尔。

我不知道桃子怎么了。我不知道这是否是西式超市的到来，迎合了新资本化(和资本主义)的劳动力对便利的需求。也许是邻居开始砍伐他们的果树，喜欢草地而不是小树林，花丛而不是食物。我父亲花了几年时间，试图让他自己修剪整齐的祖母绿盖茨比草坪成为现实，所有这些都是以酸樱桃、杏子和野玫瑰为代价的，它们在前院几乎没有任何进取心地存活了几十年。就这样，我童年心爱的桃子不见了。尽管如此，我仍然保持着希望。

希望一直持续到2018年夏天，那时我终于回到了斯科普里。我在市中心订了一个房间，离市场只有几步之遥。在时差神游中，我去买了桃子——为了安全起见，从几个不同的小贩那里买的——并把它们带到了我祖母的兵营。每一批的切片都摆出来试一试，但这些桃子不是。他们脸色苍白，水汪汪的，没有光泽。“今年雨下得太多了，”我的祖母摇摇头，啧啧说道，将她的注意力转移到电视上的土耳其肥皂剧上。对她来说向前看更容易——她已经有了几十年的好桃子。

从安大略的果园，到布加勒斯特的奥博市场，到全食超市的有机通道，再一次回到斯科普里。没有什么比得上我完全饱和的记忆。完美的桃子，带着芳香的光环，胎毛，刚刚泛红的色调，现在是一个难以捉摸的虚幻的球体。

对那些逝去的桃子的渴望往往不是因为它们的消失，而是因为每个季节都有传真来取代它们。在这一点上，我已经放弃寻找完美的桃子。我不能再忍受一次贫血和软弱的失望了。相反，当欲望来临时，我创造我自己的全息图。我知道这不是真的，但不知何故还是感觉更亲近。

本周的鸡尾酒是我对桃子记忆的颂歌。时令桃子提供了这种变戏法的基础，mezcal唤起了我眼泪的盐度(lol)。热量的冲击是可选的，但如果你想要毛状体激发的刺痛，这是必要的。要表达这种敬意，您需要:

*   3-5片桃子片(我用甜甜圈桃子，但是用任何成熟的和周围的)
*   3/4盎司柠檬汁(新鲜)
*   盎司蜂蜜糖浆*
*   2盎司麦斯卡尔(烟雾较少的品种)或布兰科龙舌兰酒
*   可选:少量哈瓦那苦啤，或者纯塔巴斯科辣啤也不错。

<small>**制作蜂蜜糖浆，只需在3份蜂蜜中加入1份热水，充分搅拌即可。*T3】</small>

将桃片和少量辣酱放入摇壶中，搅拌成糊状。加入柠檬、蜂蜜糖浆和麦斯卡尔。往摇壶里装满冰块，快速摇动；数到八应该足够了。滤入加冰的冰过的杯子，越新鲜越干燥越好。一个大冰块是理想的，但是要用你现有的。

(请随意将此视为您重复的模板。玩不同的修改器，和不同的精神。例如，杜松子酒几乎总是表现得很好。)
