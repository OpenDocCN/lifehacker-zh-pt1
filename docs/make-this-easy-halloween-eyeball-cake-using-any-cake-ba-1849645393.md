# 使用任何蛋糕面糊制作这个简单的万圣节眼球蛋糕

> 原文：<https://lifehacker.com/make-this-easy-halloween-eyeball-cake-using-any-cake-ba-1849645393>

试图为万圣节制作一个邪恶的怪物眼球蛋糕听起来像是被一个*搞定了！*灾难即将发生，但做一个适合季节的怪异装饰来吓唬你的朋友和家人并没有你想象的那么难。你不需要任何特殊的设备或者一套特殊的蛋糕雕刻技能来做这件事。你甚至不需要特别的蛋糕食谱。只需食用色素和简单的倒料技术，你就可以用任何自制面糊或盒装混合物制作令人毛骨悚然的条纹眼球蛋糕。



用这种方法做一个主题蛋糕是特别奇妙的，因为蛋糕已经*是*的装饰品了。T 何装饰不只是坐在外面(以软糖或糖霜的形式)，而是贯穿整个面糊，当你切蛋糕时，仍然可以看到，这对不知情的孩子(甚至成人)来说是一个令人震惊的事件。

这种方法使用一批浅色蛋糕面糊。你可以使用任何蛋糕面糊——刮制的、无麸质的或 Duncan Hines——只要你坚持使用香草、黄色的或白色的蛋糕，这样食用色素就会生动地显现出来。

按照指示制作面糊。取出两份——1 杯份和 1/2 杯份。这些少量的食物在不同的碗里与食用色素混合。最后你会得到总共三碗面糊，每碗都有不同的颜色。你可以使用任何适合你设计的颜色。我把大份的面糊保持原来的香草色，第二大份涂成红色，最小份涂成黑色。(邪恶的眼球必须是红色的，但是你可以添加绿色或黄色来制作一个可接受的怪物或看起来受感染的蛋糕。)

将最大量的面糊倒入准备好的蛋糕盘中。别耍花招，就像平常一样倒进去。将第二大份面糊——我蛋糕里的红色面糊——直接倒在中间；倒酒时不要四处移动。将面糊倒在中间的一堆中，这将使面糊自然地形成一个近乎完美的圆形，并通过将最初的面糊推向蛋糕盘的边缘来取代它。将最后的颜色再次直接倒入中心；如前所述，其他颜色将被推向边缘。结果是一个牛眼图案延伸到蛋糕的整个厚度。按照食谱或包装上的指示烘烤。

蛋糕冷却后，你可以随心所欲地装饰。我用糖霜增强了我的蛋糕的神秘感，做了一个诡异的万圣节怪物眼睛，但是你可以完全不加修饰；放在你剩下的节日点心旁边，它会让人毛骨悚然，因为它。如果要加糖霜(因为蛋糕没有一点糖霜算什么？)，用它沿着边缘画出令人不安的眼睑来突出你的设计。虽然现在是无实体眼球的季节，但你可以在一年中的任何时候尝试这项技术。使用多种颜色制作适合任何季节的迷人的条纹蛋糕。
