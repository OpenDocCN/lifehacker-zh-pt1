# 如何使用 50/15/5 规则来消费和储蓄

> 原文：<https://lifehacker.com/how-to-use-the-50-15-5-rule-for-spending-and-saving-1847857344>

有时候，坚持预算的最好方法是遵循一个简单的规则，这个规则不会规定每一分钱应该怎么花。这就是 50/15/5 规则背后的想法，它提供了管理你每月预算的指导方针，同时还能让你保持在你储蓄目标的轨道上。这条规则最适用于那些已经赚得足够支付开销的人*和*为退休存钱——下面看看它是如何运作的。

## **如何使用 50/15/5 法则**

[由富达推广，](https://www.fidelity.com/viewpoints/personal-finance/spending-and-saving) 每月 50/15/5 的预算规则帮助你在短期内保持财务稳定，同时确保你在退休后保持目前的生活方式。该规则细分如下:

*   你税后工资的 50%或更少应该花在基本开支上，比如住房、交通和食物。
*   你税前收入的 15%应该存进退休账户，比如个人退休账户或 401(k)账户(如果提供的话，包括匹配的雇主缴款)。
*   5%用于每月意外开支或建立应急基金。

当然，那只是你收入的 70%。剩下的 30%通常花在可自由支配的购买上，比如餐厅用餐、娱乐、服装或旅游。这种方法的好处是，你不必微观管理你所花的每一分钱——这种麻烦更有可能让你完全放弃预算(最近的一项调查 表明，20%的人根本不做预算)。

当然，如果你已经生活得有些舒适，这条规则最有效，因为如果你想改善退休后的生活方式，你必须增加退休储蓄(如果你被大量债务拖累，50/20/30 预算 或 [80/20 预算](https://lifehacker.com/how-to-set-up-an-80-20-budget-1844154593) 可能更适合你)。

要了解 50/15/5 的预算是如何运作的、、[请看这个计算器](https://communications.fidelity.com/pi/calculators/budgetcheckup/)、。记住，这条规则也是可以调整的——除了 15%指定用于退休储蓄之外，这条预算规则的其余部分更像是一条指导方针，而不是一条需要遵循的严格规则。许多人努力将开支控制在 50%，所以接近 65/15/5 可能更现实。
