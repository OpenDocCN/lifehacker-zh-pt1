# 我需要更换我的 Windows 7 电脑吗？

> 原文：<https://lifehacker.com/do-i-need-to-replace-my-windows-7-computer-yet-1846152289>

我从来没有想过我会再听到“Windows 7”这个词，但是一位 Lifehacker 读者最近提醒我，是的，有些人仍然在使用微软的生命周期结束的操作系统。我假设不是他们不想升级，而是有些人就是不会；他们可能尝试过，遇到错误信息，然后放弃了。



但是，执着于 Windows 7 会成为未来的安全问题吗？生活黑客读者**苏珊**问道:

> 我有一台运行 windows 7 的旧电脑，无法升级。我只用它来做一些事情，比如打印我的银行对账单、电子邮件、word 文档、存储照片等等。
> 
> 我担心我没有最安全的 WPS3，无法获得它。我目前有 WPS2 psk 或类似的东西。我有危险吗？我需要买一台新电脑吗？

### 我可能会尽快退出 Windows 7

事情是这样的:使用 Windows 7 [的普通人不会再获得任何操作系统的更新](https://www.microsoft.com/en-us/windows/windows-7-end-of-life-support-information) ，除了微软发现的极其关键和罕见的问题——一个我怀疑这些更新会通过 Windows Update 登陆。你必须听说他们，访问一些微软网站，自己下载并安装更新。大多数人不会这样做，也不会这样做我认为这是在未来年内有规律地发生的事情。

这是什么意思？你的个人电脑是一个定时炸弹。当然有可能一些未来的漏洞会被那些希望利用操作系统的人发现和利用，而这些操作系统将永远无法阻止他们；t 这里还有一个机会，你的系统将完全在雷达下飞行，你可以继续你的基本活动。

这是你应该冒的风险吗？我不这么认为。但我也不会责怪你继续下去，因为不是每个人都有时间、预算或兴趣购买一台全新的电脑来继续做基本的任务，这些任务只是在他们已付费的旧电脑上找到。

然而，在我得到真正的建议之前，这里有一个快速简单的答案:不要担心“wps 3”——实际上， *WPA3* ，是无线网络的安全标准。我认为，你用来加密无线网络的 WPA2 已经足够了，只要你有一个相当强的密码。我怀疑有人会坐在你的房子外面，或者拿着笔记本电脑站在你的公寓墙上，试图强行进入你的无线网络。请确保您使用的是 WPA2-AES，而不是 WPA2-TKIP 或 WPA2-TKIP+AES，以获得最佳的无线速度。

但是回到 Windows 7。我还有一个犹豫，建议你应该继续使用操作系统，也就是最终， [你的浏览器](https://cloud.google.com/blog/products/chrome-enterprise/extending-chrome-on-windows-7-to-support-enterprise-customers) 将停止接收安全更新。这只是增加了一个令人不快的媒介，攻击者可以利用它来扰乱你的数字生活。你可能有一年的时间。你可能有两年的时间——但是有一天，你的浏览器制造商将停止为 Windows 7 机器更新浏览器。这绝对是我建议升级到 Windows 10 的时候了——或者拿出一些其他的诡计。

让我们来谈谈那些狡猾的诡计。最近有没有尝试过 Windows 7 升级到 Windows 10？在你认输之前，我会给你最后一次机会，使用我之前描述的简单的“使用 USB key 作为安装介质”技术。你现在可能会有更好的运气。

如果没有，我会走第二条路: [备份你系统上所有重要的东西](https://lifehacker.com/back-up-and-clone-your-hard-drive-with-macrium-reflect-1825289970) ，即使这意味着要购买一个外部驱动器来做这件事——你应该已经准备好了，并用来定期备份你的数据。不要创建驱动器的完整映像，因为这将在以后造成一些破坏，而是将所有重要的文件复制到其他地方，要么使用好的 Windows 资源管理器，要么使用任何数量的免费备份工具。

一旦你这样做了，使用同样的安装介质(你的闪存盘)尝试 [全新安装](https://lifehacker.com/the-ultimate-guide-to-reinstalling-windows-from-scratch-1832897572#:~:text=The%20fastest%20and%20easiest%20way,10%20ISO%20directly%20from%20Microsoft.)Windows 10。这意味着你将在安装过程中清除硬盘上的所有数据，并在新的数字画布上安装 Windows 10。

如果那个管用，那太好了。如果你也使用旧的 Windows 7 密钥，你也许可以免费激活。如果它不工作，尽管，不要担心。

因为你不怎么在你的系统上*做*，你也可以探索安装 Linux——是的，Linux——这会给你安全更新*和*一个与你现有的桌面非常相似的体验。这是一个完全不同的操作系统，这意味着有些东西会有点奇怪(一些深奥的应用程序可能无法工作)。然而，它将适合你描述的所有基本任务。

我推荐用[**Linux Mint**](https://linuxmint.com/)，既 [易装](https://linuxmint-installation-guide.readthedocs.io/en/latest/) 又顺眼，或者类似[**Ubuntu Mate**](https://ubuntu-mate.org/)的。如果你已经清空了你的 Windows 7 系统，你需要用你的笔记本电脑(或者借一个朋友的)来制作 u 盘。你可以像安装 Windows 10 一样安装 Linux Mint，一旦它启动并运行，你应该能够拔出你的外部驱动器(或者你喜欢的云服务，或者其他什么)并恢复你的数据。

同样，这需要一点时间来适应——毕竟不是 Windows——但这将是一个比在未来几年坚持使用 Windows 7 安全得多的前景。如果这一切听起来像是太多的工作，你可以随时挑选一台 [新预算的 Windows 笔记本电脑](https://www.nytimes.com/wirecutter/reviews/best-laptop-under-500/#best-windows-laptops-under-500) 或更便宜的 [Windows 台式机](https://www.nytimes.com/wirecutter/reviews/cheap-desktop-pcs/#the-cheap-desktops-wed-buy) 。我不喜欢这种方法，因为你的旧电脑应该足以胜任你的工作，但这是一种选择。

无论如何，*在你开始摆弄你的 Windows 7 电脑之前，确保你已经备份了你的数据*——最好是在多个地方。我怎么强调都不为过。不要因为你瞎折腾而失去了你现有系统中最好的部分。事实上，你可能想准备一张 [Windows 7 安装盘](https://www.microsoft.com/en-us/software-download/windows7) (或闪存盘)以备不时之需，以防你举手投降，想要回到过去的样子。
