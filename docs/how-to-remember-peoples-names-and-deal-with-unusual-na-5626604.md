# 记住人们名字的 8 个技巧(以及如何记住不熟悉的名字)

> 原文：<https://lifehacker.com/how-to-remember-peoples-names-and-deal-with-unusual-na-5626604>

一个人的名字对他们来说是最重要的一个单词、以及你如何使用这个名字会对他们对你的印象产生深远的影响:想想当你钦佩的人以真诚的语气称呼你的名字时，你觉得自己很特别的时候，或者当有人不小心叫错了你的名字，或者更糟，在你面前恶意取笑你的名字时，你感到被轻视的时候。准确地记住名字是与人交往的最简单也是最重要的组成部分之一，所以这里有八个小技巧来更好地履行这一职责。

这里的第一个提示是最重要的。除非你是与世隔绝的 T2 隐士，否则在你的一生中，你将不得不为陌生人做成千上万次介绍，就像死亡和税收一样，个人介绍是不可避免的。不幸的是，我们大多数人一介绍完就忘了对方的名字。下面是一个典型的场景:

1.  对方对你说:*“你好，我是莎莎”*
2.  你报出自己的名字作为回应:*“很高兴见到你，萨莎。我是菲利普。”*
3.  一瞬间之后，你完全忘记了萨沙的名字，因为你的大脑太专注于思考你接下来要说的话，或者太专注于听萨沙说话。同样，萨沙也完全忘记了你的名字。
4.  对话可能会持续几分钟，然后当你和萨莎分手时，你们都不记得对方的名字，但你们都不好意思再问了。

这里的主要教训是，如果你没有在第一次某人告诉你的时候主动努力记住某人的名字，找到你的下一次机会就困难了。当然，你可以问一个朋友，但是第一次记得他们的名字会更好。让我们以更好的方式再次回顾我们相同的介绍场景:

1.  另一个人对你说:“你好，我是萨莎。
2.  一听到她的名字，就养成在脑子里重复它几遍:*萨沙，萨沙，萨沙*的习惯。如果你想练习大声说几次，问问他们的名字。"萨沙，拼写是 S-A-S-H-A？"或者“那个怎么拼？”目的只是为了多听几次这个名字，帮助你记忆。这一步最多只需要几秒钟，否则就会开始感觉别扭。
3.  现在自我介绍:*“很高兴见到你，萨莎。我是菲利普。”*
4.  除非你有非常紧急的事情要说，让萨莎说，当你听她说的时候，保持把她说的和她的名字联系起来。创造性地思考如何将这个名字和她的脸联系起来。如果你认识一个同名的人，试着把那个人和萨沙联系起来；或者，如果你知道一些聪明的记忆术或记忆辅助手段来帮助你记住她的名字，那么就使用它，不管它看起来有多荒谬。
5.  当你们最终分道扬镳时，向她提及她的名字。例如，*“很高兴和你聊天，萨莎。”*这有双重好处，让其他人感觉很好，因为你记住了他们的名字，同时也帮助在你脑海中强化他们的名字。

经过一些练习，你会发现你在记住别人的名字方面会变得更好，一点也不会显得尴尬。

you’你将不可避免地遇到许多名字不熟悉的人或者名字很难发音的人。有些习惯于人们忘记或搞糟他们的名字，所以如果你能准确地回忆起他们，你就能留下深刻的印象。

我在这里的主要建议是(抱歉地)请对方重复并澄清如何读他们的名字。他们可能已经习惯于重复或澄清他们的名字，所以他们可能不会介意，尤其是当你第一次见到他们的时候。不要担心如何准确拼写他们的名字，而是专注于使成为一个容易记忆的语音拼写。对于像 Chakravarty 这样的名字，你可以对自己说，*“CHALK-AHHH-VAR-TY，与 party 押韵”，* a 当你再次称呼他们的名字时，如果你发音正确，你可以问他们。至少，他们会欣赏你的努力。

每个有着非常规名字的人可能都记得小时候在学校被人取笑过他们的名字，如果你发表轻蔑或麻木不仁的言论，成年人会念错他们的名字——即使不是恶意的——会让你听起来无知、排外和偏执。

例如，永远不要说像*、*这样的亚洲名字太难记了，“或、这是一个古怪的名字，那是从哪里来的？如果你是多数群体的一员，也就是美国的白人，这一点尤其如此。

为了安全起见，在介绍自己时，总是称呼某人的真实姓名他们用的是。听到有人用未经批准的昵称或变体来称呼你可能会不舒服；更糟糕的是，当打电话的人在你面前修改你的名字时，这显示了他们的不尊重和傲慢。一个叫罗伯特的人可能不希望别人叫他罗布或鲍勃或鲍比男孩，或者可能只为亲密的朋友或家人保留这些变体。

当另一个人有一个对你来说不熟悉的名字时，这种情况会更加令人不快地着陆。例如，如果你遇到一个叫 Katsuyami 的人，你会自己选择将其缩写为“Kat”称呼他们自我介绍时用的名字，或者至少问一下。

一旦你记住了某人的名字，在对话中使用它可以帮助你发展更好的关系和更好地记住他们的名字。当然，在每句话前都提到名字是不必要的，听起来也很虚假，但是我发现你至少应该在说你好和再见的时候提到名字，这样才能给人留下好的印象。

听到你的名字被念错可能会令人恼火，但也是可以原谅的，尤其是如果很多人觉得你的名字很难发音，但听到有人完全叫错了你的名字会给留下对方根本不在乎你的印象。所以，如果你不能 100%确定你记对了某人的名字，最好不要称呼他们，而是*立即*找到你选择的重新学习的方法—像问一个朋友或者抱歉地要求那个人重新介绍他们自己。

然而，不要仅仅因为你第一次没有记住某人的名字，就放弃并不再尝试去记住他的名字。听到有人叫错你的名字是很可怕的，但知道有人很可能不知道你的名字，也不愿意知道你的名字也是令人恼火的。

当你写电子邮件或短信时，正确拼写 最重要的单词是他们的名字。一个人在书写中看到自己的名字的次数可能比任何其他重要的单词都多，所以任何拼写错误都会立即出现。

就我而言，人们经常把我的名字拼错为*菲利普*，甚至当我的电子邮件显示拼错为 *菲利普*时，我还是忍不住注意到每一次出现这种情况。把某人的名字写对的一个脑残简单的方法是简单地从他们写给你的前一封信中复制粘贴。别人把自己的名字拼错的可能性远远小于你把它拼错的可能性。

如果你能记住某人的另一半、配偶、孩子或父母的名字，你会和他们建立更好的关系。这样的话，就不会听起来像是，“那么，你妻子的艺术项目进展如何，“你可以变得更有风度，“黛博拉的*艺术项目进展如何？”不过，这是一个额外收获，所以不要像记住这个人的名字一样强调它。*
