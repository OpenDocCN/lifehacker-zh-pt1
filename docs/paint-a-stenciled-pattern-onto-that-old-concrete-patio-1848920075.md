# 在旧混凝土天井上画一个模板图案

> 原文：<https://lifehacker.com/paint-a-stenciled-pattern-onto-that-old-concrete-patio-1848920075>

今年春天，当你从你的家出来，评估你的户外空间时，你可能会发现你的旧水泥庭院需要稍微整修一下。如果你想在预算不多的情况下翻新一下庭院，你可以考虑在印花图案上绘画。你所需要的只是一些材料，一些有用的提示，一点时间和耐心。以下是如何在你的混凝土露台上刻下图案。



## 如何准备一个混凝土天井被模板

对于这个项目，你需要:

*   一张 [大幅面模版](https://www.cuttingedgestencils.com/wall-stencils/stencils-by-stencil-collection/stencil-patterns-new-stencils-wall-stencils/alhambra-tile-stencil-asulejos-spanish-tile-wallpaper.html)
*   一个 [大的蜡纸刷](https://www.cuttingedgestencils.com/stencils-stenciling-supplies-stencil-brushes/stencil-supplieses/2-professional-allover-stencil-brush.html?gclid=Cj0KCQjw4PKTBhD8ARIsAHChzRJ8lDEMUXBVjOve37eqJUKPTYKjA06LT4vrc04UfvcBNy8-q40kNtoaAtmoEALw_wcB)
*   一个油漆滚筒
*   E 外混凝土 [油漆](https://www.google.com/shopping/product/1?q=exterior+concrete+paint&client=ms-android-verizon&biw=393&bih=727&tbs=vw:l&sxsrf=ALiCzsaYmSz6Qdg0t2lNnxVVkaAB8WtJeA:1652378382259&prds=num:1,of:1,eto:3296477312362908216_0,prmr:1,pid:3296477312362908216,cs:1) 底色和模板色
*   [C 李尔密封器](https://www.acehardware.com/departments/paint-and-supplies/stains-and-finishes/varnish/1001296?x429=true&utm_source=google&utm_medium=organic-shopping&utm_campaign=organic-shopping)
*   油漆工胶带
*   一个装水的大容器和一块海绵来清洁你的模板

有很多模板选项，虽然传统的方法是使用仿瓷砖图案，但你可以用任何你喜欢的图案来刷新。

现在你可以开始为油漆准备混凝土了。使用来自软管或动力清洗机的强力喷雾去除任何苔藓、灰尘或其他松散的碎片。如果你的露台已经有一段时间没有打扫了，你也可以试着将洗碗精溶解在水中，用硬鬃毛扫帚或刷子擦洗表面的顽固污垢。检查混凝土是否有裂缝，并使用裂缝修补工具来封闭小裂缝和碎片。

一旦你的表面干净光滑，用你的滚筒和你的底色用室外油漆粉刷露台表面。根据你选择什么颜色作为底色以及它与你原来的颜色有多大的变化，可能需要涂两层才能得到均匀的颜色。一旦你的整个表面都涂上油漆，让你的底漆干一夜。

## 如何在混凝土上画出 模板

下一步是有趣的部分:从一个角落开始，最好是离门最远的那个，沿着混凝土边缘放置你的 ste ncil，并用油漆工的胶带将它粘牢。用你的蜡纸刷以轻轻的动作向下推动填充你的图案。这有助于避免漏印- 通过模板图案的边缘。如果你以前没有使用过模板笔刷，或者你只是需要刷新一下你的笔刷技能，在一块光滑的纸板上练习，了解一下你的笔刷和模板是如何工作的。

在天井的一边重复你的模式，作为你的第一排。这会给你一个很好的基础来设计你的模式。要估计布局，您可以剪下一张与模板大小相同的纸板或纸，用它来估计模板的位置。如果你的模板表面不适合角落或边缘，你可以用胶带把你不用的模板部分粘住，这样你就不会意外地把涂到其他表面。

## 一些让模版印刷更容易的技巧

打扫地板可能需要一段时间，但是 T2，你可以做一些事情让这个过程更顺利。首先，防止干颜料沿着模板图案的边缘堆积会使你的图案看起来更清晰 —u se 一桶或一盘水和一块大海绵定期擦去模板上的多余颜料以防止其出现。你也应该经常冲洗模板刷子，但是一定要在清洁工具后彻底擦干，这样你就不会不小心弄薄你的油漆。

如果你的图案中确实有小的瑕疵或错误，一旦模板层干燥，你可以用你的底色和一个小的油漆刷来润色。

在你的图案完成后，最后一步是使用透明的封闭漆作为面漆，以防止你的图案因行人交通或天气而磨损。在放置家具或在院子里行走之前，你应该让你的表面彻底干燥。
