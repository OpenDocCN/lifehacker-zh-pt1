# 使用宽松的“夹子”来避免简短、无意义的会议

> 原文：<https://lifehacker.com/use-slack-clips-to-avoid-brief-pointless-meetings-1847720527>

如果远程工作教会了我们什么，那就是我们的许多会议都是不必要的，过于复杂，尤其是如果团队分布在不同的时区。除非合作对议程是必不可少的，否则我们经常可以在自己的时间里只需几分钟就能得到我们需要的一切。Slack 一直在为我们共同的“这次会议可能是一封电子邮件”问题开发一个工具:它的解决方案——“clips”——现在正在向公众推出。

使用剪辑功能，您可以 [录制长达 3 分钟的音频或视频](https://slack.com/help/articles/4406235165587-Record-audio-and-video-clips-in-Slack) (带屏幕共享)，并通过 DM 或频道发送剪辑。视频剪辑自动转录与选项启用字幕。

虽然剪辑可能主要用于与工作相关的交流，但您也可以将其用于娱乐。只需知道剪辑与 Snapchat 或 Instagram stories 的*不同——它们的保留规则与你的公司对所有其他 Slack 消息的保留规则相同，不会自动消失或过期。* 

### 如何在空闲时间创建和发送视频剪辑

要在桌面应用程序中创建剪辑，请创建一条新消息，或者导航到您要将剪辑发送到的 DM 或频道。然后:

1.  点击**摄像头图标**并选择您想要共享的屏幕(如果您正在进行屏幕录制)。如果你想从一个普通的视频开始，你也可以添加屏幕共享中间记录。

2.  想要开始时点击**录制**，完成后点击**停止录制**。

3.  点击**下一个**。在此添加消息是可选的。

4.  点击**返回**或**纸飞机图标**发送您的剪辑。

如果您需要为您的录制选择不同的摄像机或麦克风，您可以在**设置**下找到它(单击齿轮图标)。

对于浏览器和移动设备上的 Slack，步骤是相似的。在 iOS 上，点击**摄像头图标**，选择**视频>录制**。要完成您的剪辑，请点击**录制>使用视频**，然后添加消息并发送。在 Android 上，点击**摄像头图标**，然后点击**摄像机图标**。点击**记录**开始，**记录**(再次)停止，然后**将**连接到添加信息并发送。

如果您想在视频剪辑上使用字幕，播放时选择 **CC 图标**。也可以看笔录(点击视频上的三条竖线)。

### 如何在空闲时间制作和发送音频剪辑

要在桌面上创建音频剪辑，请单击**麦克风图标**，开始录音。完成后，点击**勾号**，添加信息并发送。

在手机应用程序上创建音频剪辑的过程与制作视频几乎相同——只需在录制时按住**麦克风图标**。

如果您找不到摄像头和/或麦克风图标，您可能还不能访问剪辑。此外，该功能仅适用于付费计划。
