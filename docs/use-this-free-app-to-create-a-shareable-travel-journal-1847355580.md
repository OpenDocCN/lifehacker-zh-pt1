## 如何使用app制作旅行日记

这款应用名为[【Hoptale】](http://www.hoptale.com/)，它利用照片元数据来获取目的地的信息，并用它来编写旅行日志。除了照片和事实，你还可以记录你旅行的其他方面，包括你去过的地方的地图和你的行程——该公司表示，这可以在短短10分钟内完成。

Hoptale被设计用于三种不同的方式:

1.  用最少的努力创建一个旅行日志
    这是上面描述的部分，来自你的照片的元数据被用来勾画出一个旅行日志的基本组成部分(城镇的名字，你去过的地标，等等)，让你自己添加你自己的细节和评论。
2.  **分享你的旅行日志和行程**
    你不仅可以公开分享你的旅行(就像传统的旅行博客一样)，还可以向朋友和家人发送一个链接，让他们看到你的照片和你的近况。
3.  **将它用作旅行计划工具**
    当你准备好计划你的下一次旅行时，你可以查看应用程序的 [探索](https://hoptale.com/explore) 提要，找到其他人已经进行过的惊人旅行，并获得真实的、用户生成的照片和旅程。

该应用程序可以免费下载和使用，据Hoptale的发言人表示，“截至目前，没有计划向使用全功能应用程序的用户收费。”
