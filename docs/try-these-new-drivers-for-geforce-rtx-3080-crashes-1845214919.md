# 尝试这些新的驱动程序 GeForce RTX 3080 崩溃

> 原文：<https://lifehacker.com/try-these-new-drivers-for-geforce-rtx-3080-crashes-1845214919>

当你花几百美元买了一个顶级显卡时，你最不想让 T2 处理的事情就是在你使用它的时候无数次的崩溃。但这似乎是一些新的 GeForce RTX 3080 用户的情况；无论什么原因，Nvidia 的新显卡在某些游戏中崩溃。

Watch

让这个谜团更加棘手的是，崩溃问题的源头是难以确定的。正如最近 PC 游戏玩家 [所描述的](https://www.pcgamer.com/nvidia-rtx-3080-gpu-crashes-instability) :

> 就我们而言，我们在测试第三方版本的 RTX 3080 时遇到了问题，特别是工厂超频卡。有趣的是，只有当我们运行 1080p 基准测试时，驱动程序崩溃才会出现，而不是在 1440p 或 4K，而且也只是在特定的游戏中。我们已经测试了新的预发布固件，它禁用了工厂超频，但我们仍然遇到了同样的问题。
> 
> 然而，GPU 本身的欠频解决了稳定性问题，我们只需要稍微偏移-30%就可以实现。这确实说明了卡的电源子系统中的潜在稳定性问题。

虽然有很多假设被抛出，各种卡的电容器 可能在某种程度上导致崩溃，但我还不会归还你的 RTX 3080。首先，尝试安装 Nvidia 最新的 GeForce 驱动程序(456.55，在我们写这篇文章的时候)，看看它是否能修复你的系统崩溃问题。

根据 Nvidia 的说法，这次更新“提高了你的全新显卡的稳定性”——这是 Nvidia 昨天在 [的一个简短论坛帖子](https://www.nvidia.com/en-us/geforce/forums/geforce-graphics-cards/5/400499/rtx-3080-board-stability-new-driver-capacitors/?ncid=afm-chs-44270&ranMID=44270&ranEAID=kXQk6*ivFEQ&ranSiteID=kXQk6.ivFEQ-TzAxh_zO7PNce_As40mb9Q) 中愿意提供的尽可能多的细节，此外还暗示电容器索赔可能不是问题。现在说这是否能修复用户报告的每一个崩溃还为时过早，但为你的显卡运行最新的驱动程序永远不会有坏处。你当然可以手动下载它们，但是我推荐安装 Nvidia 的方便的[GeForce Experience](https://www.nvidia.com/en-us/geforce/geforce-experience)工具。

当新的驱动程序准备好的时候，你不仅会得到一个小小的通知，而且你甚至可以让它们在后台自动下载。这样，你所要做的就是点击“快速”或“定制”安装按钮，你就可以开始了。几分钟后，你就可以安装全新的驱动程序了(你甚至不需要重启系统就可以继续玩游戏了；我没有)。

但是，如果在更新驱动程序后，您仍然遇到崩溃，那么联系您的卡制造商来解释这个问题，看看他们是否能提供任何帮助可能是值得的。他们可能会给你送一个性能更好的替换显卡；尽管你也有可能遇到完全相同的问题。
