# 每个背包需要的物品，按用途排序

> 原文：<https://lifehacker.com/the-items-every-backpack-needs-ranked-by-usefulness-1849525349>

返校季意味着背包在脑中。不过，我相信拥有一个装满东西的背包也是成年人应该做的事情。除了我的电话、钱包和钥匙，我很少不带任何东西(T2)出门。当我外出时，我宁愿准备比最低限度更多的东西。

Watch

即使你不支持背包，也有很多被忽视的东西应该放在你的钱包、公文包或者至少是你的车里。以下是最聪明的物品，永远放在你的背包里(或你选择的背包里)，按照必要性从“必备品”到“嗯，你永远不知道”排列。

## 第一层:最基本的必需品

无论你是准备去远足、听音乐会，还是只是去上班，你都应该在包里准备好以下必需品:

*   手机充电器
*   耳塞
*   止痛药(如布洛芬)
*   卫生基本知识(卫生棉条、纸巾)
*   洗手液
*   额外现金
*   水瓶
*   除臭剂
*   面罩

## 第二层:你将是一个救生员

除了必需品，这一层准备意味着在你生命中的某个时刻，一个朋友或陌生人会告诉你“你是一个救星。”

*   便携式充电器
*   指甲钳/锉刀
*   镊子
*   口香糖/薄荷糖
*   无色唇膏
*   B 和-Aid
*   笔和拍纸簿
*   雨伞
*   可重复使用的购物袋
*   可重复使用的银器
*   遮光剂
*   帽子(御寒或防晒)
*   去污棒
*   一小卷胶带
*   附加 ID

## 第三层:“你永远不知道”

这一层是为那些喜欢提前做好准备的人准备的，并且在他们的背包里有足够的空间这样做。你永远不知道什么时候你可能会陷入困境，或者需要在一个意想不到的地方过夜。这个清单可以一直列下去，但这里有一些最基本的用品，可以合理地放在一个背包里(取决于你塞了多少)。

*   换洗衣服(或至少袜子和内衣)
*   牙刷和牙膏
*   游泳衣
*   微型手电筒(如果你的手机没电了)
*   书、一副扑克牌或其他娱乐
*   雨披
*   手套
*   垃圾袋
*   外置硬盘

最具争议的打包物品:你的护照。您可能需要备份 ID，但同时，您将增加丢失它的风险。这取决于你，取决于你把背包带去哪里，以及你乱放东西的倾向。
