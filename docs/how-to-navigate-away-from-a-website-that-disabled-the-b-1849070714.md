# 如何离开禁用“后退”按钮的网站

> 原文：<https://lifehacker.com/how-to-navigate-away-from-a-website-that-disabled-the-b-1849070714>

这是典型的一天:你启动你选择的搜索引擎，输入你感兴趣的查询，然后点击一个看起来有希望的链接。其实也没那么有用，你点后退按钮查看更多搜索结果。除了后退按钮什么也不做。你再试一次:什么都没有。这个网站让你陷入困境，莫名其妙地强迫你停留在页面上或完全关闭标签。然而，你并没有真的被困住——T2 的风景就在你的掌握之中。

虽然互联网不再是过去的蛮荒西部，但仍有大量可疑网站使用卑鄙的策略让你关注他们的内容。“诱捕”你是你会遇到的更具侵略性的策略之一，这种做法被谷歌称为“ [历史操纵](https://9to5google.com/2018/12/17/chrome-sites-wont-go-back/)”:T 有问题的网站会向你的搜索历史发送垃圾邮件，链接重定向回他们的页面，这使得后退按钮无用。对你来说，你最后访问的网站应该是你的搜索结果，比如说，谷歌或者 Ecosia。然而，由于历史操作，你的网络浏览器认为它应该回到你已经在的页面。

虽然这里的自然本能是简单地关闭浏览器标签或窗口，但这很烦人。你不应该被迫打开一个全新的窗口，输入和以前一样的搜索，都是因为一些秃鹫想让你在他们的网站上多呆一秒钟。幸运的是，你根本不需要这么做:与其徒劳地点击后退按钮，不如直接按住按钮。

无论你是在智能手机还是电脑上使用 Chrome、Safari 还是 Firefox，按住后退键都会显示该标签页或窗口的特定搜索历史。您可以选择列表中的任何链接跳回该站点，即使它不是该窗口中最后访问的站点。如果您发现自己在一个实现历史操作的网站上，您可能会在这里看到一些看起来很吓人的链接，或者您可能会看到有问题的网站被反复列出。

请仔细查看或滚动这些选项，直到你找到搜索结果页面，或任何你想访问的页面，选择它，你将立即逃离你的互联网捕获者。记住*永远不要*再访问他们的网站，并随时警告其他人不要访问这种浪费数字空间的网站。

据说，谷歌已经致力于解决这个问题有一段时间了，但是，显然，这个问题仍然存在。也就是说，如果你在桌面上使用 Chrome，你永远不需要被这些垃圾网站所困。您应该启用隐藏的搜索面板，而不是原样使用浏览器，这样您就可以查看搜索结果，而无需离开原始搜索页面。查看 [我们的完整指南，了解更多](https://lifehacker.com/google-chrome-has-a-hidden-search-sidebar-1849064065) 。
