# 为什么你需要独处的时间

> 原文：<https://lifehacker.com/schedule-yourself-some-alone-time-1846417163>

如果我们花太多时间独处，那种悲伤的感觉有一个名字:我们孤独。但是，没有花足够的时间独处会产生什么样的负面感觉呢？



由卡尔顿大学的罗伯特·科普兰领导的一个心理学家小组也为它提出了一个名字:孤独。孤独的反义词。我们需要平衡我们的独处时间和社交时间，正确的平衡可能因人而异。这可能是一个更有助于理解我们需求的框架，而不是试图将我们自己推入一个 [内向/外向二元](https://lifehacker.com/lets-quit-it-with-the-introvert-extrovert-nonsense-1713772952) 。

那么，你能对你的孤独做些什么呢？第一步是承认这个问题:如果你需要的独处时间超过了你能得到的时间，那你就是孤独的。Coplan 和他的同事发现孤独的主要工具是一个简单的问卷，询问人们是否同意类似“我希望我能更多地独处”这样的陈述。“通过一点自省，你可能会发现你实际上是否是孤独的。

《今日心理学》指出，如果是这样，你可能会感到“易怒、不知所措或精疲力竭”。如今，我们的心理健康面临着如此多的压力，试图将孤独产生的压力与焦虑、倦怠以及工作和生活中无数其他压力引发的压力区分开来可能会有所帮助。如果这个任务本身是难以承受的，考虑一下 [可能是时候去找一个治疗师](https://lifehacker.com/its-time-to-find-a-therapist-1846009669) 了，他可以帮你理清你大脑正在处理的所有事情。

与此同时，你可能会发现问自己独处时间多了是否感觉更好，然后安排和保护独处时间是有益的。例如，你不必同意每一次社交活动或变焦电话。你可以决定周五晚上呆在家里。

即使由于你目前的情况，你不能得到很多独处的时间，你也可以确保你得到的独处时间是和你自己在一起的时间。想想那些真正让你感到放松、快乐或精神焕发的事情——也许是读书或散步，而不只是滚动手机——只要你有一分钟的时间，就做其中的一项活动。
