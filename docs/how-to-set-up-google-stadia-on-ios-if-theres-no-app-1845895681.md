# 如果没有应用程序，如何在 iOS 上设置谷歌视距

> 原文：<https://lifehacker.com/how-to-set-up-google-stadia-on-ios-if-theres-no-app-1845895681>

正如你所记得的，苹果公司以铁腕手段经营其应用程序商店。在该公司允许游戏流媒体服务拥有常规应用之前，库比蒂诺将会是寒冷的一天——除非服务 [将每个可流媒体游戏提交给苹果](https://www.theverge.com/2020/8/6/21357771/apple-cloud-gaming-microsoft-xcloud-google-stadia-ios-app-store-guidelines-violations) 审查，这是不可能的。你可以打赌苹果会想从用户为流媒体订阅支付的任何费用中抽取的分成。

为了解决这个问题，流媒体服务正在转向你的网络浏览器。这正是今天你能够在 iOS 和 iPadOS 上通过 Safari 传输 Google Stadia 的方式。嗯，从技术上来说，通过一个在你的设备上运行的渐进式网络应用程序*而不是*，Safari UI 会把它弄乱，给人一种你在运行一个原生 iOS 或 iPadOS 应用程序而没有 Stadia 的错觉。

相反，你会希望启动 Safari——而且只能启动 Safari，因为你不能使用其他浏览器安装 PWA，比如 Chrome。调出 [谷歌 Stadia 的网站](http://stadia.google.com/) 并登录(如果需要)。点击浏览器底部的分享按钮，然后点击**添加到主屏幕**。

然后你会把一个类似应用程序的 Stadia 图标放到你的主屏幕上。它看起来会像这样:

我建议要么删除原来的 Stadia 应用程序，要么记住这个图标的颜色,这样当你在手机上搜索“Stadia”并看到两个选项时就不会感到困惑了:

从那里，你所要做的就是点击你的 Stadia PWA，在你的(不完全)Safari 浏览器体验中启动它。在那之后，你就可以在 Stadia 上玩游戏了，就像你使用任何普通的应用程序一样。

如果你在 iPhone 或 iPad 上遇到任何 Stadia 问题，要知道你需要*运行 [iOS / iPadOS 14.3](https://lifehacker.com/the-best-new-features-and-fixes-of-ios-14-3-1845888062) 才能完成 PWA 创建过程，所以相应地更新你的 iPhone 和/或 iPad。并确保您已经将 [稳固连接到您的 wifi](https://lifehacker.com/the-10-best-ways-to-boost-your-home-wifi-5931743) 。我不会*在手机连接上玩 Stadia(如果你可以的话)，因为如果你这样做太久，你的数据计划会很不满意。**

此外，我强烈建议将 Xbox 或 PlayStation 控制器与 iPhone 配对(通过蓝牙)来玩游戏。试图用虚拟界面做任何事情都将是徒劳的——如果我被困在一个有很好的 wifi 和没有控制器的荒岛上，我可能会使用这种设置，但仅此而已。

谢天谢地，这样做很容易。只需将你的 [Xbox](https://support.xbox.com/en-US/help/hardware-network/accessories/connect-and-troubleshoot-xbox-one-bluetooth-issues) 或 [PlayStation](https://www.playstation.com/en-us/support/hardware/ps4-pair-dualshock-4-wireless-with-pc-or-mac/) 控制器设置为配对模式，在你的 iPhone 或 iPad 上拉起**设置>蓝牙**，等待一段时间，当它出现时，点击你的控制器。一旦配对成功，Stadia 会立即识别出它，你就可以开始游戏了。

是的，Stadia *有两个免费游戏如果你不想立即购买游戏或注册订阅，或者不确定你的网络或互联网连接是否能处理所有的流媒体。试试 *Destiny 2* 或 *Super Bomberman R Online，*然后看看是否有必要进一步开发游戏流媒体服务。*
