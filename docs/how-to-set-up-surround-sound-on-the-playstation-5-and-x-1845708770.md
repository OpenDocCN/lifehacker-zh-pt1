# 如何在PlayStation 5和Xbox Series X上设置环绕声

> 原文:[https://life hacker . com/how-to-set-up-on-the-PlayStation-5-and-x-1845708770](https://lifehacker.com/how-to-set-up-surround-sound-on-the-playstation-5-and-x-1845708770)

PlayStation 5和Xbox Series X可能会提供更好的图形和更快的加载时间，但你有没有停下来想想你的下一代游戏体验听起来如何？两款主机都支持来自扬声器或条形音箱的5.1和7.1环绕声，以及通过耳机的虚拟环绕声。如果你家里有合适的设置，打开这些功能是轻而易举的。

### 在Xbox系列上使用环绕声

1.  从你的Xbox主屏幕，进入**设置>常规>音量&音频输出**。
2.  前往**“扬声器音频”**选择扬声器或条形音箱，或者如果您使用耳机或游戏耳机，选择**“耳机音频”**。
3.  **对于扬声器音频**:如果您的Xbox直接插入扬声器的A/V接收器或条形音箱，请选择**“5.1或7.1未压缩”**。(如果你的Xbox直接插入电视，然后将音频传递到你的接收器或条形音箱，你可能需要通过**“比特流输出”**选项使用DTS数字环绕或杜比数字。)
4.  **耳机音频:**选择**“耳机用Windows Sonic”**
5.  使用**杜比全景声**或 **DTS耳机:X** :选择音频选项，然后从微软商店下载杜比全景声应用程序，并按照应用程序内的设置说明进行操作。**注意:**杜比全景声对于扬声器或条形音箱的安装是免费的，但是对于使用和耳机需要15美元的许可费。同样，使用 DTS耳机:X附带 $20费用。
6.  **可选:**在**设置>设备&连接>蓝光**中设置您首选的蓝光电影流媒体比特率。默认选项对任何设置都适用，而选择“未压缩”选项可以提高质量，但需要更高的带宽*。*

一旦这些步骤完成，Xbox Series X上受支持的游戏、应用程序和电影应该会利用您喜欢的环绕声设置——或者您的新虚拟环境。

### **在PlayStation 5上启用环绕声**

#### **扬声器或条形音箱**

1.  按下控制器上的**“PlayStation”**按钮。
2.  进入**设置>声音>音频输出****T6】HDMI设备类型**
3.  将设备设置为“**AV Amplifier”**(不要选择“soundbar，”，即使您使用的是支持5.1或7.1的Sound Bar)。
4.  向下滚动并选择您的扬声器数量。
5.  选择**“调整扬声器位置”**并定制布局以匹配您房间中的扬声器位置。
6.  最后，设置与您的设置相匹配的音频格式(优先级)选项。如果您使用HDMI将PS5直接连接到接收器或条形音箱，请选择**“线性PCM”**。如果你将PS5直接插入电视，然后电视将信号传递给你的接收器或条形音箱，你可能需要使用**“杜比”**或**“DTS”**

#### **耳机**

一些耳机内置了环绕声功能，但索尼PS5上的3D音频可以在任何一对插入控制器3.5毫米插孔或通过USB或蓝牙连接到PS5的耳机上创建令人信服的环绕声舞台(索尼表示，它计划在未来的更新中为电视扬声器带来3D音频)。

1.  将耳机插入控制器或通过蓝牙连接到PS5(如果耳机连接到电视或外部音频设备，这将不起作用)。
2.  按PlayStation按钮打开菜单。
3.  进入**设置>声音>音频输出。**
4.  打开**“启用3D音频”**
5.  选择**“调整3D音频模式”**，然后按照的提示微调耳机的3D音频性能。

#### **打开蓝光光盘环绕声**

如果你正在PS5上观看蓝光电影，但不能享受到正确的5.1或7.1播放，请尝试调整蓝光音频格式设置。(注意:PS5目前不支持杜比Atmos，但使用杜比Atmos的蓝光仍然可以通过这些步骤输出到5.1或7.1设置)。

1.  观看电影时，按下控制器上的**“选项”**按钮。
2.  在弹出窗口中选择三点菜单图标。
3.  转到S **设置>音频格式**。
4.  将格式设置为**“比特流”**
5.  关闭选项菜单，您的媒体现在应该可以正常播放了。
