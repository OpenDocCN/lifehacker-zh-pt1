# 停电和限电的区别

> 原文：<https://lifehacker.com/the-difference-between-blackouts-and-brownouts-1849079269>

随着全国气温和湿度水平的飙升，许多人都在开大空调。虽然这完全可以理解，但这样做可能会导致停电。与此同时，到目前为止，强烈的夏季风暴也已经成为这个季节的固定节目，在闷热的天气中导致停电。



停电和限电在夏季更容易发生，所以了解两者之间的区别是个好主意。以下是需要了解的内容。

## 停电和限电的区别

停电和限电都会让你没电，但是有几个关键的区别你应该知道:

### 灯火管制

停电就是完全断电。虽然有些只持续几分钟，但其他的会持续几个小时，甚至几天。这在很大程度上取决于首先是什么导致了停电，以及解决这个问题有多困难。一些 [常见的停电原因包括](https://neccoopenergy.com/brownout-vs-blackout-whats-the-difference/#:~:text=at%20its%20peak.-,Brownouts,least%2010%20to%2025%20percent.) :

*   损坏的变压器或电线(由风暴、倒下的树木和/或事故造成)
*   增加的电力需求将电网推向极限
*   雷击电线杆
*   受干扰的地下电力线
*   电线上积冰

### 布朗宁顿

停电通常是由电力公司无法控制的情况造成的(如恶劣天气和事故)， [限电是故意的](https://neccoopenergy.com/brownout-vs-blackout-whats-the-difference/#:~:text=at%20its%20peak.-,Brownouts,least%2010%20to%2025%20percent.) 。限电发生在电力需求接近容量时(就像整个城市在同一天打开空调)，为了防止电网过载，电力公司限制某些地区的电力。

当一个地区的家庭停电时，他们仍然有电——只是电压比平时低得多。因此，虽然你可以打开一盏灯，但它会相对较暗。

### 滚动“停电”和“计划内断电”

有时，当电力公司知道电网将被推到极限时，它可能会选择实施有意的 [“滚动”停电。](https://www.directenergy.com/learning-center/difference-between-blackout-brownout) 这些通常在有限的时间内在多个地区发生，通常来自电力公司的最后一分钟通知。

但是轮流停电和 [“计划的停电是不一样的](https://www.directenergy.com/learning-center/difference-between-blackout-brownout) 那些往往要提前很长时间安排日常维护。
