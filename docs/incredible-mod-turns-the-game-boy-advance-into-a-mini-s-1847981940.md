# 令人难以置信的 Mod 将 Game Boy Advance 变成了一个带有 Joy-Con 和 Dock 支持的迷你开关

> 原文：<https://gizmodo.com/incredible-mod-turns-the-game-boy-advance-into-a-mini-s-1847981940>

用更好的屏幕和更大的电池等升级改造 18 岁的任天堂游戏男孩 Advance SP 相对来说没有什么痛苦。但是 [改装 GBA SP](https://hackaday.com/2021/11/01/modded-gba-sp-does-its-best-switch-impression/) 使其作为专用的 16 位交换机，支持 Joy-Con 和 TV-out？这是一个巨大的挑战， [男子玉米片制作](https://www.youtube.com/watch?v=zHOGTgf6qlk) 不知何故让它看起来很容易。



在坚持使用游戏机近二十年的 GBA SP 粉丝中，[Macho Nacho Productions](https://www.youtube.com/channel/UC4CsqctrGOn4NTz09sAhXwQ)可能最出名的是其 [承诺的升级套件](https://gizmodo.com/the-creators-of-that-spec-d-out-game-boy-advance-sp-wil-1847663541)[为游戏机](https://gizmodo.com/this-specd-out-game-boy-advance-sp-sports-next-level-fe-1847550018) 添加了现代设施，如更大的电池、无线充电、耳机插孔和无线音频流的蓝牙支持。这个想法是拿一个经典的主机，用更现代的手持设备来提升它的速度，但是如果有人反过来拿一个现代的手持设备，用复古的 16 位图形取代 3D 高清游戏，会怎么样？世界准备好接受 GBA 开关了吗？

GBA SP 已经有了屏幕升级套件，可以将 [手持设备的链接端口变成视频输出连接](https://retrogamerepairshop.com/products/game-boy-advance-sp-ips-backlight-tv-version-av-out-mod-kit?ref=6njgx4sufvs&variant=40667812298924) ，用于将其连接到电视，并在更大的屏幕上播放经典的 GBA 游戏。以此为起点，来自 Macho Nacho Productions 的 Tito 和 Kyle Brinkerhoff 仍然需要开发一个定制的 3D 打印后壳替代品，包括一对 Joy-Cons 可以连接的轨道，加上足够的空间来定制 PCB 和 Raspberry Pi Zero，以便 Joy-Cons 可以无线控制 GBA 游戏。除此之外，他们还需要在那里挤一块更大的 2000 毫安时电池，为所有额外的电子设备供电。

 [https://gizmodo.com/embed/inset/iframe?id=youtube-video-zHOGTgf6qlk&start=0](https://gizmodo.com/embed/inset/iframe?id=youtube-video-zHOGTgf6qlk&start=0) 

结果不再是可以放入口袋的 GBA SP，因此两人甚至创建了一个定制的类似开关的坞站，可以存放改装的手持设备，也可以很容易地将其连接到电视机。布林克霍夫已经在他们的 [GitHub 页面](https://github.com/philosiraptor/gameboyswitch) 上为任何想自己解决这个升级问题的人提供了所有你需要的文件，包括一个你可以购买 [定制 PCB 板](https://www.pcbway.com/project/shareproject/ground_boi_signal.html) 的链接。

只是不要期望 mod 的功能会像真正的任天堂 Switch 一样流畅。无线配对 Joy-Cons 是一个漫长的过程，涉及将 Raspberry Pi 连接到计算机并运行自定义代码，每次 GBA 交换机通电时都必须这样做。这看起来像是一个真正的痛苦，仅此一点可能就足以证明为任天堂 Switch 在线服务炮轰，祈祷有一天任天堂将 GBA 添加到受支持的复古游戏机列表中。
