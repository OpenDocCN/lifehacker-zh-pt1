# 学习可以拯救生命的谨慎手势

> 原文：<https://lifehacker.com/learn-the-discreet-hand-signal-that-could-save-a-life-1848017797>

“求救信号”是一个简单的手势——伸出你的手，手掌向外，然后将手指合拢在拇指上——但它可能会挽救你的生命。



就在上周，一名在北卡罗来纳州失踪的 16 岁女孩在肯塔基州被救出，她向一名路过的司机示意。根据劳雷尔县治安官办公室 的说法，这名未透露姓名的少年“在社交媒体平台‘Tik Tok’上做手势，代表家庭暴力——我需要帮助。”

一个开车的同伴认出了这个标志并给警察打了电话，然后尾随这辆车几英里，向当局传递信息。在一次交通拦截后，肯塔基州警方逮捕了 61 岁的詹姆斯·布里克，并指控他一级非法监禁，第二项指控涉及持有未成年人性表演材料，因为在他的手机上发现了材料。女孩与家人团聚了。

这个手势是加拿大妇女基金会 在疫情封锁之初创造的，对它的认识已经在抖音和其他社交媒体平台上传播开来。其目的是给家庭暴力受害者一种方式，在视频通话中无声地发出痛苦的信号，不会给施虐者留下任何数字证据。这里有一个 [的视频戏剧化的信号在行动](https://www.youtube.com/watch?v=AFLZEQFIm7k) 。

### 如果有人向你发出求救信号，该怎么办

如果一个随机的乘客在高速公路上向你示意，你可能应该报警，但是如果你从朋友或熟人那里收到这个信号，事情就有点棘手了。

根据加拿大妇女庇护所的说法，这个标志并不意味着“马上打电话给当局”意思是“安全地向我伸出手。”如果你认识的某个人在视频通话中发出求救信号，加拿大妇女庇护所建议你，“安全地与那个人联系，了解他们需要什么，希望你做什么。”这个想法是让发信号的人带头，而不是假设你知道他们想要什么或需要什么。

加拿大妇女庇护所还建议，如果有人在听另一半的对话，可以问一些可以回答“是”或“不是”的问题来回应这个信号。所以说，“你想让我打 911 吗？”或者“我应该寻找一些可能对你有帮助的服务，然后给你回电话吗？”而不是“告诉我发生了什么。”

他们还建议通过另一种交流方式联系对方，问一些看似无辜的问题，比如“你好吗？”或者“有空的时候联系我。”

### 如果施虐者也知道这个信号呢？

这些胁迫信号可能是一把双刃剑。他们依赖很多人知道他们的工作，但这意味着滥用者也更有可能知道他们。发出求救信号的人强调，信号只是一种工具，有时可以用来指示人们需要帮助。这不是一个神奇的解决方案。

知道还是值得的。在视频通话中，一个简单的、可识别的手势可以比一句口语更谨慎地使用，而且，与电子邮件或文本不同，当通话结束时，不会留下任何痕迹可供发现。
