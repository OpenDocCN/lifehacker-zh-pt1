# 使用 37%规则做出更好的决定

> 原文：<https://lifehacker.com/use-the-37-rule-to-make-better-decisions-1848916008>

在做决定的时候，你是那种立即知道他们想要什么，迅速做出决定，并且很少怀疑你的选择的人吗？或者你更像是那种“在陷入分析瘫痪绝望之前，去互联网的无底洞研究每一个可行的选择”的决策者？



如果你是前者，请接受我真诚的祝贺。你要么过于自信和禅，要么过于自律；不管怎样，你是幸运的。对于我们这些倾向于花一整天精心制作利弊清单的人来说，在怀疑和抛弃这些清单并重新开始之前，有一个迷人而有效的 37%规则。

## 什么是 37%法则？

37%法则来源于数学中的 [最优停止理论](https://www.americanscientist.org/article/knowing-when-to-stop) ，它决定了采取某一特定行动的最佳时间，以实现回报最大化和成本最小化(又名，停止寻求更多选项并扣动扳机的最佳时间)。根据数学家的说法，那个点就在你看到或探索了你的前 37%的选项之后。

当然，你需要先想出你的总宇宙，来计算你的 37%的选择会是什么。你可以通过设置最高限额或基于时间的最后期限来做到这一点。

例如:如果你正在购车，并决定在做出决定之前先看 10 辆车，你应该计划看前 3-4 辆，而不打算购买。探索完那些，你的探索期就到了一个收益递减点，比最初那三辆好的*下一辆车*就是守护者。

或者，你可以为你的搜索设置一个时间段，就像作家兼程序员 Brian Christian 在他的书 [*中描述的那样:*](https://brianchristian.org/algorithms-to-live-by/#:~:text=A%20fascinating%20exploration%20of%20how,a%20particular%20set%20of%20problems.)

> “如果你想获得最好的公寓，花 37%的时间(11 天，如果你给自己一个月的时间寻找的话)不明确地探索各种选择。把支票簿留在家里；你只是在校准。但在那之后，准备好立即承诺——存款和一切——你看到的第一个地方，胜过你已经看到的任何地方。这不仅仅是在观看和跳跃之间的一种直觉上令人满意的妥协。这是可证明的最优解。”

这条规则可以应用于任何性质的决定——从约会或选择度假地点,到购买房屋和[雇用秘书](https://www.linkedin.com/pulse/37-rule-shrirang-raddi/)T5(或任何其他求职者)。根据数学家的说法，遵循这条规则将使你免于陷入不必要的信息收集和数据分析的泥潭，让你行动起来，并最大限度地提高你的成功概率。

## 37%规则的局限性

当然，当涉及到重大的财务决策和内心问题时，这条规则并没有考虑到感觉、“直觉”、瞬间化学反应和推荐的力量。(例如，当有一个非常有魅力、非常有趣的人坐在你单身的对面，而这个人是由一个亲密的朋友审查和推荐的，即使他们*是你一年中的第一次约会。)* 

虽然它不会适用于所有情况，而且你不应该仅仅因为一个数学理论就放弃明显更好的选择，但如果你倾向于做出仓促但令人遗憾的决定，或者花太多时间研究所有的选择，这是一个有用的经验法则。下一次当你面对竞争选择时，请记住:大约你决策过程的前三分之一应该是信息收集，之后，选择你遇到的下一个伟大选择是最佳的。
