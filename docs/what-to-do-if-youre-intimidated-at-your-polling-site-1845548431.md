# 如果你在投票站受到威胁，该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-youre-intimidated-at-your-polling-site-1845548431>

尽管数百万人已经投票，但预计明天会有更多的人在当地投票站排队。诚然， [共和党人在 2016 年依靠选民压制战术](https://www.motherjones.com/politics/2017/10/voter-suppression-wisconsin-election-2016) ，但这一次，总统向他的支持者发出了多次行动呼吁，要求他们明天担任“投票站观察员”。虽然 [观察投票是选举过程中合法合理的](https://lifehacker.com/what-do-poll-watchers-do-1845230065) (阅读 [了解更多关于此处](https://lifehacker.com/what-do-poll-watchers-do-1845230065) )，但唐纳德·特朗普所建议的——他的追随者只是在投票站出现“观察】(从而恐吓)选民——并不是这样。

如果你发现自己被这些自封的选举大厅监督人之一骚扰或恐吓——或者官方投票工作人员、或任何其他人——记住你*有*投票权 [你可以采取](https://lifehacker.com/what-to-do-if-you-get-turned-away-at-the-polls-1829664675) 步骤来确保你行使这一权利。其中之一是拨打选民保护热线。这里有几个选择。

### 选民保护热线

如果你正在寻找一些关于最常见的选民压制骗局和技巧的信息我们可能会在明天看到，以及 [如果你被投票拒绝该怎么办](https://lifehacker.com/what-to-do-if-you-get-turned-away-at-the-polls-1829664675) ，我们已经广泛地讨论了这两个主题。以下是一些你想准备好的选民保护热线(或者保存在你的通讯录中):

#### 选举保护联盟

这个全国性的非党派组织由依法维护公民权利律师委员会管理，拥有多种语言的选民保护热线:

*   英语 : 866-OUR-VOTE / 866-687-8683
*   西班牙语/英语 : 888-VE-Y-VOTA / 888-839-8682
*   **亚洲语言/英语** : 888-API-VOTE / 888-274-8683
*   阿拉伯语/英语 : 844-YALLA-US / 844-925-5287

针对您所在州的信息， [使用此互动地图](https://866ourvote.org/state/) 。

#### 州和地方选举办公室

虽然这些不是专门的选民保护热线，但如果你发现自己需要与你所在州或地方选举办公室的人通话， [你可以在这里](https://www.usvotefoundation.org/vote/eoddomestic.htm) 查找这些电话号码。

#### 民主党

上周末，美国总统巴拉克奥巴马(Barack Obama)在 Instagram 上发布了一张方便的图片，提供了全国选民保护热线，以及各州的热线:

 [https://lifehacker.com/embed/inset/iframe?id=instagram-CG-biGqgOl3&autosize=1](https://lifehacker.com/embed/inset/iframe?id=instagram-CG-biGqgOl3&autosize=1) 

这个国家号码——1-833-DEM-VOTE(1-833-336-8683)——由 [民主党全国委员会](https://democrats.org/where-we-stand/the-issues/vote/) 的员工全天候监控，他们随时准备回答有关剥夺重罪犯选举权、选民清洗、投票工作人员不当行为、投票机、无障碍等问题。

其余的电话号码是各州民主党运营的选民保护热线。他们由 [Vote Save America](https://votesaveamerica.com/) 编制成这张图——同样是这个组织提供 [自带投票工具](https://lifehacker.com/how-to-byob-build-your-own-ballot-so-there-wont-be-an-1845403600) ，以及一系列其他信息，包括今天和明天你仍然可以志愿参加 的方式。

#### 共和党

[共和党全国委员会](https://www.gop.com/) 没有相当于民主党全国委员会的选民保护热线，一般也不提供太多联系方式。RNC 公开列出的电话号码是 **202-863-8500** ，尽管这看起来更像是一个办公室号码而不是热线。

你可能会更幸运地直接联系到你所在州的共和党总部；他们应该能给你指出正确的方向。例如，阿肯色州的 [共和党](https://katv.com/news/local/voters-calling-for-help-reporting-issues-to-voter-protection-hotline) 报告称，到目前为止，他们已经收到了“小问题，但没有超出预期的情况。”似乎没有每个州的共和党名单，但只要谷歌一下“[你所在的州] +共和党”，你应该就能找到。
