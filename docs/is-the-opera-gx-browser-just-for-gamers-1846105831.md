# Opera GX 浏览器只面向游戏玩家吗？

> 原文：<https://lifehacker.com/is-the-opera-gx-browser-just-for-gamers-1846105831>

Opera [最近收购了 YoYo Games](https://www.engadget.com/opera-buys-gamemaker-studio-2-developer-yoyo-games-160045897.html) ，游戏开发引擎 Game Maker Studio 的创造者，以建立其新的“Opera Gaming”部门。Opera 尚未宣布其与游戏制造商 Studio 的合作计划，但此次收购提供了一个很好的机会来检查 Opera 的另一款专注于游戏的产品:[Opera GX 网络浏览器。](https://www.opera.com/gx)



Opera GX 于 2019 年 6 月在“早期访问”中发布，因此技术上仍在开发中。乍看之下，它看起来也像是一款小众游戏应用。虽然它目前的形式最适合 PC 玩家，但即使你不玩游戏，也有充分的理由尝试 Opera GX。

最具游戏特色的是“GX 角”标签。点击 Opera GX 侧边栏顶部的控制器形状的图标，会打开一个窗口，显示最近与游戏相关的新闻报道、免费和打折 PC 游戏的链接以及即将发布的日历。该页面会自动更新，不需要进行配置，但是您可以自定义布局并指定您所在的区域，以使新闻提要更加相关。

Opera GX 还可以让你连接你的 Twitch 帐户、Discord 和其他消息应用程序。每个帐户都被添加到 GX 的侧边栏菜单中，这样你就可以直接从浏览器的界面上关注你的通知和消息，而不是将每个帐户作为单独的标签或应用程序打开。

一个单独的“GX 清理”功能可以让你快速清除你的历史和缓存——没有什么新奇的。然而，“GX 控制”可以让你随时监控 Opera GX 的一般网络、CPU 和内存，并设置浏览器使用多少资源的限制。还有一个打开的标签列表，显示每个标签使用了多少系统(并让你强制关闭任何使你的电脑陷入困境的标签)。

这些工具有助于保持 Opera GX 的系统使用率低，所以它不会拖慢你的电脑和干扰你的游戏。也就是说，即使是非游戏玩家也会发现内置的 messenger 应用集成和资源管理工具很有帮助——只要他们能够克服浏览器的视觉风格。

Opera GX 的默认主题唤起了陈词滥调的“游戏玩家”审美:黑暗的背景，高饱和度的霓虹灯口音，风格化的图标，动画用户界面元素，甚至环境背景声音和音乐。Opera GX 也有自己的一套默认壁纸，看起来大多像风格化的太空摄影，或者隐约暗示一种“黑客”美学。这些视觉元素本质上都不坏，但是我对我的浏览器看起来像游戏耳机的盒子不感兴趣。幸运的是，你可以上传自己的壁纸，并在浏览器的设置中更改或禁用视觉效果和声音效果。

另一个缺点是浏览器如何跟踪用户数据。Opera GX 跟踪某些数据(就像普通的 Opera 浏览器一样)，包括你的 IP 地址和浏览活动，加上一些来自 Twitch 等第三方账户的活动。Opera 与广告商和其他第三方合作伙伴共享这些数据，但 Opera 的跟踪感觉并不比谷歌 Chrome 差——用户可以从设置菜单中禁用(一些)数据共享。

只要 Opera GX 的数据跟踪不是一个交易破坏者，并且你确保根据主题设置进行调整，这样界面就不会花哨混乱，无论你是 PC 游戏爱好者还是普通用户，这款浏览器都有很多值得推荐的地方。如果你想亲自体验一下，可以在这里 下载。
