# 先抽一颗白菜，掌握温度控制

> 原文:[https://life hacker . com/master-temperature-control-by-smoking-a-cabbage-first-1847046161](https://lifehacker.com/master-temperature-control-by-smoking-a-cabbage-first-1847046161)

如果你的吸烟经历仅限于吸入兴奋剂，你可能会被一种同名的烹饪方法吓到。如果这听起来像你，我有一个建议:在你抽猪肩肉、排骨或任何其他种类的肉之前，先抽一棵卷心菜。

扔一颗大白菜进去会帮助你建立自信。如果你使用木炭烤架或任何其他没有精确温度控制的吸烟器，这尤其有用。如果你习惯了烤箱 (a )的数字转盘，锁定和稳定吸烟装置内的环境温度可能有点棘手，谁不是呢？)，如果你用的是韦伯水壶，甚至是一个简单的木炭吸烟器，你必须通过打开和关闭吸烟器顶部和底部的通风口来控制温度。这并不难，但确实需要一些时间来适应。

为什么是卷心菜？卷心菜很便宜。一整棵卷心菜的价格还不到任何一磅肉的价格——即使是打折最厉害的猪肩肉也比不上卷心菜的低价。你不太可能在第一次尝试吸烟时遇到这么多麻烦，以至于毁了你的卷心菜——甚至是猪肩肉——但是如果你碰巧毁了你的卷心菜，这也不是什么大损失。你所“浪费”的只是几个小时的时间和价值1.5美元的配料。(拆猪肩肉的刺痛？毁灭性的。)用卷心菜代替一块昂贵的肉来吸烟可以消除你第一次吸烟实验中的一定压力，这意味着你的大脑更有可能学习，你也更有可能享受学习过程。

切好的卷心菜也很好吃。它们柔软可口(显然还有烟熏味)，是完美的夏季配菜，尤其是搭配凉爽的奶油时。有很多熏卷心菜的食谱，所以选择一个对你来说听起来最好的。(我喜欢 [这个](https://barbecuebible.com/recipe/barbecued-cabbage/) ，里面塞满了黄油和培根。)然而，我会选择一个保持卷心菜完整的食谱——吸卷心菜的目的是让你习惯于长时间保持吸烟者的温度恒定，如果你的四分之一卷心菜在仅仅60分钟内就吸完了，你就会错过这一点。

我建议的另一件事——不，是命令——是你可以用某种探针温度计来监测温度。放在烤架或熏炉顶部的小刻度盘非常不准确，尤其是如果它正好放在煤的上方。(我的大概差了150度！)我有这个 [双探头体温计](https://amzn.to/3w6MHlq) ，让我可以同时测量食物和吸烟者的温度，很有帮助。无论你是在吸卷心菜还是一大块肉，温度控制都是等式中的一个重要部分，如果你的数字不正确，等式就不会那么好。
