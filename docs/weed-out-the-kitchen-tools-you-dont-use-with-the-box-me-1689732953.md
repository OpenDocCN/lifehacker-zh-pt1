# 用盒子法清除不用的厨房工具

> 原文：<https://lifehacker.com/weed-out-the-kitchen-tools-you-dont-use-with-the-box-me-1689732953>

有些厨房工具 [是不可或缺的](https://skillet.lifehacker.com/kitchen-tool-school-the-flippin-awesome-fish-spatula-1683871427) ，但其他的只是漂浮在你的厨房里，再也不会被使用。不过，整理厨房储物件只需一个简单的过程。

Watch

在 2015 年的一篇名为 [的帖子中，Kitchn 向读者征集了关于精简餐具、炊具和其他厨房相关杂物的最佳建议](https://www.thekitchn.com/what-are-the-most-essential-utensils-and-cookware-good-questions-216495) 。 [评论员 cook_at_home 建议的盒子法](https://www.openweb.com/share/1twjVv5nx632PU5T21KBBjdKkEv) ，既简单又有效:

> 把所有东西都拿走，放在一个盒子里。只在需要的时候拿出来。
> 
> 在你拿出某样东西之前，想想看有没有别的方法可以用你现有的东西来完成这项任务。
> 
> 一个月后，把盒子搬到地下室(或其他“遥远的”仓库)。6 个月后，捐献盒子。

盒子方法非常有效，因为它迫使你对自己的烹饪习惯变得现实。这使得从一堆杂物中分离出真正的必需品变得非常容易。给自己一个严格的时间框架也很关键:如果你在六个月内没有用过某样东西，你真的需要把它留在身边吗？

请随意调整这一过程，以满足您的需求。例如，如果把所有东西都放进一个盒子里听起来有点混乱，试着把你的厨房用品分类。从笨重的电器、超大的炊具和其他占用大量空间的东西开始。首先缩小大的东西可能会释放出很多空间，这样你就不需要如此无情地使用较小的工具了。你也可以根据你家有多少“远储”来调整等待期更长或更短。

我们都有买来的厨房工具，用来做那种*一样东西*那种*有一次*然后就再也不做了，尽管我们发誓我们会这样做。这是一个友好的提醒，提醒你不时地清理它们。所有这些不用的工具只是占用了你可以用来存放其他东西的空间——最好是你实际使用的东西。
