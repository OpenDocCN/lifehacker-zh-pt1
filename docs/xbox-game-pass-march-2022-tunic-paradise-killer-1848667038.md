# Xbox Game Pass 悄然增加两款惊艳游戏

> 原文：<https://kotaku.com/xbox-game-pass-march-2022-tunic-paradise-killer-1848667038>

除了少数几款热门游戏，Xbox Game Pass 这个月过得有点慢——直到昨天。在一个专注于独立游戏的展示期间，微软宣布又有两款游戏将登陆游戏通行证库:*祭服*和*天堂黑仔*。更好的是，它们都已经有了。



我知道，我知道…我已经收回我说的话了。不过嘿，至少 [字做一顿精致的早餐](https://kotaku.com/xbox-game-pass-march-2022-shredders-weird-west-1848654678) ！

由 Kaizen Game Works 开发的天堂黑仔是过去几年中比较独特的游戏之一。一部分是视觉小说，一部分是侦探游戏，三部分是怪异，你的任务是在阳光普照的袖珍宇宙中解决一起谋杀案，这个宇宙充满了古怪的人物，他们的名字像“爱情小姐死了”但是最吸引人的是巴里“新纪元”专辑中的那难以抗拒的朗朗上口的 synthwave 配乐。当你玩的时候，你解锁更多的音乐，慢慢地建立起一个点唱机菜单，你可以用它在任何时候切换音乐(全是喧闹声，没有铿锵声)。为了昨天在 Xbox 和 PlayStation 上发布，Kaizen Game Works 增加了额外的歌曲。

与此同时,是一款受*塞尔达*启发的冒险游戏，昨天刚刚上市，已经被誉为今年最好的游戏之一。你扮演一只小狐狸，在白色沙滩上空手醒来。从一个等距的角度来看，你在一个艺术幻想场景中掠夺地牢和砍杀敌人。你以前做过所有这些事情，这就是让*外衣*的核心品质脱颖而出的原因:它毫不含糊地信任玩家。菜单和游戏中的符号是用一种解释很少的北欧古字语言完成的，所以你不得不依靠直觉和从其他类似游戏中收集的内在知识。如果你还没有售出，请查看前半小时的游戏 。

但是，Tunic T1 在 Game Pass 上的出人意料的发布凸显了流行的游戏点播服务的增长战略。12 月，该服务见证了三款游戏——*镇民*、*零代*和*严肃山姆 4*——[突然增加了](https://kotaku.com/xbox-game-pass-is-starting-to-add-games-without-announc-1848150292)，打破了微软多年来在新游戏到来前几周宣布新游戏的惯例。

就在最近的一月( [通过时光倒流机](http://web.archive.org/web/20220106145459/https://twitter.com/FinjiCo/status/1479102833915936773) ), *的发行商 Finji 坚称这款游戏不会登陆 Xbox Game Pass，因此不清楚游戏通行证协议是何时签署的。当被联系到进行评论时，Finji 的一名代表告诉 Kotaku 公司不与媒体讨论业务往来。*

现在，一点也不像有什么邪恶的事情在发生——事实上，恰恰相反。直到昨天，你才可以购买到外衣，而这已经是游戏通行证库中的一部分了。没有办法预购游戏，所以玩家也没有办法早早买了然后觉得焦头烂额。但这是一家出版商的决定，并不能免除未来增加服务库的负担。现在的新常态是，显然任何东西都可以在任何时候击中游戏通行证。那么，当下一个惊喜发布游戏已经被成千上万的玩家预订时，会发生什么呢？

微软的代表没有立即回应置评请求。
