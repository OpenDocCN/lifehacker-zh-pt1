# 您每天都想使用的最佳 Microsoft PowerToys 快捷方式

> 原文：<https://lifehacker.com/the-best-microsoft-powertoys-shortcuts-you-ll-want-to-u-1848634350>

没有了 [微软 PowerToys](https://docs.microsoft.com/en-us/windows/powertoys/install) ，Windows 就不一样了——这是一套充满有用功能的实用程序，包括一个聚光灯般的搜索栏，可以快速打开应用程序或运行计算，一个快捷方式可以立即静音麦克风并关闭网络摄像头，还有一个工具可以让你将任何应用程序的窗口固定在屏幕上。所有的 PowerToys 工具都是免费的——以下是我们的最爱。

## 视频会议静音

没有什么比没有意识到你的网络摄像头和麦克风是开着的更尴尬的了，但是视频会议静音给了你一个键盘快捷键让你立即静音，另一个快捷键关闭你的网络摄像头，第三个快捷键同时关闭两者。由于这是一个全局键盘快捷键，它适用于所有视频会议应用程序，您只需记住一两个快捷键，而不是每个应用程序的一组单独的命令。

## 找到我的老鼠

给自己买一个或两个大显示器？这种设置的最大问题之一是找到微小的鼠标光标。我在超宽显示器上永远找不到我的光标，所以我最常用的工具就是找到我的鼠标。要找到光标，请按两次左 Ctrl 键。

## 总是在上面

作为一名作家，我经常喜欢将我的写作应用程序钉在屏幕上，同时参考截图或打开应用程序来准确描述我所写的功能。这就是为什么“永远领先”对我来说非常有用。如果你是一个喜欢从视频或 PDF 文档中做笔记的学生，你也会喜欢这个工具。启用或禁用总是在顶部的默认快捷方式是 Windows **键+ Ctrl + T** 。

## 梦幻组合

我们中的一些人对窗口管理非常挑剔。如果你是我们中的一员，那么 FancyZones 可以帮助你在屏幕上创建整洁的区域，并允许你在这些区域中安排多个窗口。在 FancyZones 中选择布局后，按住 Windows 键拖动窗口，直到它们与您选择的布局对齐。

## Image Resizer

图像缩放器允许您通过右键单击并更改为不同的分辨率来快速调整图像大小。您甚至可以在这里添加自定义分辨率来满足您的需求。Image Resizer 不能完全取代 Photoshop，但对于很多人来说，这是一种轻量级的免费方式，可以很好地执行任务。

## 电源重命名

PowerRename 是一个实用程序，可以让您批量重命名一组文件。在 Windows 资源管理器中选择一组文件，右键单击它们，然后选择 PowerRename。它有切换选择，如果你想大写每个单词或遵循大写或小写文件名。您也可以为每个选定的文件添加一个编号。

## 电动玩具跑

Mac 用户长期以来一直使用 Spotlight search 进行从打开应用程序到执行计算的各种操作。PowerToys Run 为 Windows 带来了这一功能。快捷键是 **Alt +空格键。**

## 醒着的

如果您想阻止电脑进入睡眠模式，请启用 PowerToys 中的唤醒实用程序。它做一件事，而且做得很好。您可以选择是否保持屏幕打开，并选择是永久还是暂时保持电脑处于唤醒状态。
