# 什么面包最适合做法式吐司？

> 原文：<https://lifehacker.com/what-breads-work-best-for-french-toast-1849175684>

我的生活建立在碳水化合物的房子上。我喜欢它们，也不打算停下来，但有时当我珍贵的面包和蛋糕不新鲜或变得单调时，我会尝试让它们重获新生——不仅仅是为了减少浪费，还因为法式吐司。它是甜的，松软的，令人满意的，和不新鲜的面包一起吃效果最好。*最不新鲜的*面包，那就是。为了找出你是否有最适合这项工作的面包、蛋糕或糕点——哦，是的，这远远不止是面包，伙计们——你需要问自己几个问题。



## 面包的面包屑(或质地)是什么？

如果你有 [不新鲜的热狗面包](https://lifehacker.com/the-best-french-toast-sticks-are-made-with-stale-hot-do-1848912873) 或旧的 [意大利面](https://lifehacker.com/transform-leftover-panettone-into-tasty-french-toast-1790452557) 盯着你看，那么你很幸运:这两种面包都可以成功地法式烘焙，尽管它们的外观和味道大相径庭。它们都是因为它们的“碎屑”或纹理而工作的。特定面包或蛋糕的面包屑是指其内部结构。如果你想到面包屑紧密的面包，那么它有均匀分布的小孔。具有开放面包屑或开放结构的面包具有更大、更不规则的气穴。没有面包屑或面包屑非常稠密的面包，就像玉米粉圆饼或皮塔饼。

法式吐司需要浸泡大量的甜蛋奶冻(或 [蛋奶酒](https://lifehacker.com/eggnog-makes-the-tastiest-three-ingredient-french-toast-1745325009) )。最好的浸泡面包应该有紧密的面包屑或开放的面包屑，但不是致密的。重要的是质地要轻盈透气，有足够的口袋让蛋羹渗透进去。抱歉，玉米饼要等返工日了。

## 你的面包有厚厚的外壳吗？

我见过一些用切片法式长棍面包做法式吐司的食谱。(考虑到它们的标题，这可能看起来合乎逻辑，但你不会尝试用炸薯条，是吗？)尽管法棍面包有敞开的面包屑，可以浸泡，但我并不因为它们耐嚼的外壳而偏爱它们。同样，如果你有一个剩余的乡村酸面包，你需要小心这个。面包皮基本上是面包外面的一层皮。有些面包有一层薄而细腻的皮，像沙拉，这是很好的法式吐司，但是有一层厚而硬的外皮的面包并不理想。不管你把面包浸泡多长时间，面包皮都不会变软。

虽然这不是一个交易破坏者，但它绝对打断了软，custardy 咬的纹理。除了叉子之外，我喜欢吃满满一盘，厚的面包皮需要刀子和两只手。然而，你可以通过使用锯齿刀，在浸泡前锯掉面包屑来减轻这种情况。

## 你的面包会碎吗？

当你知道有问题的碳水化合物即将被放入鸡蛋和乳制品的水坑中，然后被烤或甚至被烤时，结构是一个很大的因素。一个优质的法式吐司需要在面对液体和油炸时坚不可摧，同时保持其枕头般的声誉。L 寻找具有良好面筋结构的面包。不过，这并不一定意味着坚韧或耐嚼；rioche 是这种应用的最爱，因为它柔软、营养丰富，通常是一条面包，你可以切成你想要的厚度。它之所以有效，是因为它有一个坚固的结构——一个坚实的面筋框架，还有来自面团中鸡蛋的额外帮助。

## B reads 并不是唯一可以用来做法式吐司的碳水化合物

这就是为什么大多数软皮面包非常适合做法式吐司，它们可以处理这个过程而不会散开。但是这并不意味着面包是唯一有效的 T2 碳水化合物。一些蛋糕和糕点足够坚固，可以承受蛋羹的快速浸泡和煎锅中的飞溅。 [蓝莓松饼](https://www.tasteofhome.com/recipes/blueberry-muffin-french-toast/) 和面包蛋糕是有足够的结构可以托起的快速面包；只要在蛋羹中稍微浸泡一下，否则他们在看到烤盘之前就有分开的危险。

只要你的 [蛋羹](https://lifehacker.com/you-should-waffle-some-french-toast-1848807993) 是好的，只有三个理由让你不能在家里用法式吐司烘烤所有的令人厌倦的碳水化合物。使用你新的法式吐司配对问卷，哪些项目效果最好？硬皮酸面团、不新鲜的夏威夷卷、放了一天的羊角面包或天使蛋糕？请让我知道你是否法式烘焙的天使蛋糕，因为我非常想看。
