# 如果不能进行 Android 自动呼叫，该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-you-cant-make-android-auto-calls-1845968429>

今年早些时候，在用户更新到 Android 11 后，Android Auto 中的一个 bug 中断了谷歌助理的电话呼叫。谷歌在系统更新后不久修复了这个问题，但即使在安装了补丁后，一些用户仍然会看到这个错误。幸运的是，似乎终于有了解决办法。根据谷歌支持论坛上的一篇帖子，这个问题是由错误配置的应用程序权限引起的，至少对一些用户来说是这样。



修复挥之不去的 Android Auto 电话呼叫错误很容易，但首先你应该确认你是最新版本的 Android 和 Android Auto，以防万一。在你的手机上，进入**设置>系统>高级>系统更新**。点击**“检查更新”**然后运行安装，如果有可用的补丁。然后查看是否有新版本的 [安卓汽车应用](https://lifehacker.com/how-to-force-the-new-android-auto-update-1837371752) 。

同样，如果你通过汽车上的触摸显示屏使用 Android Auto，也要确保更新汽车的固件。更新你的汽车的固件取决于它的品牌和型号，但这些步骤应该很容易在制造商的支持页面上找到。

如果一切都是最新的，但谷歌助手呼叫仍然无法在 Android Auto 中工作，请尝试以下方法:

1.  使用 Android Auto 时，让 Google Assistant 开始打电话，或者轻触“电话”图标。 [当然不要在开](https://lifehacker.com/how-to-use-your-phone-in-your-car-without-killing-anyon-1843185336) 的时候这么做。
2.  你的安卓设备(或汽车的触摸屏)上会弹出一个窗口，询问你想用哪个应用程序打电话。选择您的手机应用程序，然后在出现提示时轻按“总是”。确保你点击**“总是”，**，而不是**“仅仅一次”**，否则你将不得不重复这些步骤。

看起来这个小小的调整就是让谷歌助手的电话呼叫在 Android Auto 中再次工作所需要做的，但它可能不会对每个人都有效。让我们知道调整这些设置是否对你有用，或者你是否知道其他的修复方法。
