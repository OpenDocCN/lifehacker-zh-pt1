# 使用“好、更好、最好”系统进行压力较小的假日锻炼

> 原文：<https://lifehacker.com/use-the-good-better-best-system-for-less-stressful-1848265236>

假期可能会有压力，尤其是如果你在旅行，招待客人，或者负责所有的圣诞购物。一方面是没时间运动。另一方面，如果你跳过常规锻炼，你会发现自己压力更大。



我们之前已经讨论过即使在休假中也要抽出时间锻炼对心理健康的好处。锻炼可以帮助你与你的常规时间表保持一致，让你的焦虑在你的大脑中少一些空间，并帮助你在一天的大部分时间里无所事事时感到更加灵活和有能力。

但是我们生活中繁忙的时间让我们很难像往常一样衡量锻炼。你可能没有那么多的锻炼时间，或者没有那么多的步数，或者你可能没有那么多的锻炼时间。不过，这个问题有一个绝妙的解决方案，我在《华盛顿邮报》关于抛弃你的健身追踪器 的文章 [中注意到了。(顺便说一句:我*完全*支持抛弃你的健身追踪器。)](https://www.washingtonpost.com/wellness/2021/12/21/fitness-activity-tracker-obsession-unhealthy/)

诀窍是摆脱要么全有要么全无的心态，要么你在锻炼中表现出色，要么你还不如呆在床上。提前决定当天什么是“好”、“更好”和“最好”的选择。然后做任何一个适合你的选择。

例如，我最近没有感觉在健身车上骑了整整一个小时，但我确保骑上自行车去做一些事情，即使我没有太多的时间。对我来说，半小时的轻松踩踏板是我的“好”锻炼，45 分钟是“更好”，我的“最好”是一个完整的小时，有专门的热身序列，一些高能量间歇，和一个不错的冷却。

你可以根据任何活动、任何时间承诺或强度水平来调整这个三层系统。也许你的“好”是在健身房做几组杠铃深蹲，然后回家，而“更好”和“最好”包括之后的配件和有氧运动。或者，“好”是去散步，“更好”是一个快速跟进的 YouTube 视频，“最好”是去健身房。你可以决定什么对你有意义。
