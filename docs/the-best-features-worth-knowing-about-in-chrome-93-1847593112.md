# Chrome 93最值得了解的功能

谷歌Chrome版刚刚推出桌面版和安卓版。这些每月更新可以给开发者带来各种新功能、调整和变化，但版本93有几个增强功能值得用户了解也是如此，包括桌面上Chrome网络应用的增强功能、Android应用界面的变化、有用的跨设备安全选项等等。

## 桌面:网络应用现在可以从URL链接启动，并支持多屏设备。

Chrome 93为桌面用户提供了两个新的网络应用功能。

首先，通过URL链接打开外部网络应用现在是无缝的。打开此功能后，如果您的设备上安装了web应用程序，单击Chrome中的链接将会打开该应用程序。

例如，如果你安装了带有Chrome 的 [YouTube PWA，点击YouTube超链接将在PWA中打开视频，而不是新的Chrome标签。这并不适用于每个web应用程序，因为它需要开发人员自己实现这一功能，但对于支持它的应用程序来说，这将是一个有益的改变。它在默认情况下也是禁用的，必须通过更改Chrome的实验标志菜单中的一些设置来打开:](https://lifehacker.com/how-to-install-youtube-as-a-progressive-web-app-on-wind-1846137659)

1.  打开Chrome，进入**Chrome://flags/# enable-desktop-pwas-URL-handling**
2.  从下拉菜单中选择**“启用”**。
3.  出现提示时重启Chrome。

另一个新特性是 [对web apps的多屏多窗口支持](https://web.dev/multi-screen-window-placement/) 。这个功能也需要开发者来实现，但是谷歌预测它会对许多类型的网络应用有帮助，包括:

> *   The multi-window graphic editor à la [gimp](https://www.gimp.org/release-notes/gimp-2.8.html#single-window-mode) can place various editing tools in the precisely positioned window. The virtual trading desk can display market trends in multiple windows, any of which can be viewed in full screen.
> *   The slide show application can display the speaker's notes on the internal home screen and the presentation on the external projector.

这些应用程序现在可以在一台显示器或多台显示器上同时打开多个窗口，这对多显示器用户来说应该是一个显著的改进。

## 跨设备双因素短信同步

从93版开始，如果你在多台设备上使用谷歌账户登录，Chrome将自动识别并同步短信登录码，进行双重认证——至少对于支持该功能的网站是如此。这个也需要开发者实现，但是将来很可能会有更多的网站和服务支持双因素同步。在那之前，你可以看到谷歌创建的 [模拟测试网站](https://web-otp-demo.glitch.me/) 的功能如何工作。

## Android:新的谷歌搜索界面和Android 12的材料支持

Chrome 93还为Chrome Android应用程序带来了一些UI变化。

首先，最近彻底检修的谷歌搜索结果的连续结果馈送已经被改进。新设计最初是作为一项实验性测试添加到Android Chrome 91中的，它让你可以从Chrome窗口顶部的滚动条打开页面并在页面间切换。该栏在搜索结果中不断滚动，因此您可以在页面之间快速跳转。

Chrome 93使卡片更容易阅读，据说比最初实现时反应更快。打开新的搜索结果栏仍然需要你在实验标志菜单中搜寻，然而:

1.  在Chrome Android应用中，进入**Chrome://flags #连续搜索**
2.  从下拉菜单中启用设置。
3.  出现提示时重新启动应用程序。
4.  应用程序重启后，进行谷歌搜索。
5.  从结果中选择一个网站。新的连续搜索导航栏将出现在搜索结果的顶部。

Chrome 93还支持Android 12的Material You接口到Chrome mobile，尽管这是另一个实验性功能。

你展示的材料改变了支持的Android 12应用程序的界面，以匹配你的手机背景，一旦新的操作系统更新在未来几个月推出，它可能会成为所有Android 12设备的默认功能。不过，现在你需要Android 12测试版。从那里，在Chrome的实验标志中启用该功能:

1.  打开Chrome应用，进入**Chrome://flags # dynamic-color-Android**
2.  从下拉菜单中选择**“启用”**。
3.  重新启动应用程序以启用更改。
4.  轻按提要顶部的一个URL信息卡以打开页面，然后向左或向右滚动以查看更多选项

这些是Chrome 93对普通用户的主要改变，但还有很多地方需要开发者去钻研。 Y 你可以在 [更新的发布说明](https://developer.chrome.com/blog/new-in-devtools-93/) 中详细了解这一切。
