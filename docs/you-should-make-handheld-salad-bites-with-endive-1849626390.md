# 你应该用菊苣做手持沙拉

> 原文：<https://lifehacker.com/you-should-make-handheld-salad-bites-with-endive-1849626390>

我们很快就要迎来节日聚会季，聚会需要可以拿在手里的食物。理想的聚会食物最多是几口，不需要任何器具，而且相当独立。沙拉不是很好的聚会食物，但是沙拉是很好的聚会食物。



如你所知，大多数沙拉是由一堆绿色蔬菜和许多其他材料组成的。其他的东西通常是一些其他的蔬菜，也许是一种水果，通常还有一些肉和/或奶酪。沙拉的味道改变了蔬菜和其他食物的比例，使之更有利于食物，这种食物包含在一个优雅的、令人惊讶的美味的小菊苣杯中。

对于外行人来说，生菊苣又脆又苦，是很多脂肪、咸味和甜味配料的完美陪衬。它提供了如此多的“叶子”味道，你根本不需要那么多叶子。(其实一片叶子就够了。)

把一份沙拉变成一口沙拉很容易。挑选你最喜欢的沙拉——在我的例子中，是一个楔形沙拉——从配料表中去掉生菜部分。将所有其他材料放入碗中混合(连同调料)，然后将混合物舀入单独的菊苣叶中。完成了。

如果需要沙拉的灵感， [看经典](https://lifehacker.com/make-better-salads-by-using-a-classic-recipe-as-a-templ-1837986043) 。一份科布沙拉，一份凯撒鸡肉，一份尼瓦兹沙拉，甚至一份【panzanella】——所有这些都会很棒。(对于 panzanella，将面包切成小面包丁。)你也可以在菊苣叶中加入不含生菜的沙拉，比如 [黄瓜和番茄](https://lifehacker.com/the-dressing-on-this-cucumber-salad-has-two-ingredients-1844149910) 、面条沙拉，或者任何熟食店的经典菜肴( [鸡肉](https://lifehacker.com/a-chicken-salad-manifesto-1843970579) 、 [金枪鱼](https://lifehacker.com/how-to-add-umami-to-tuna-salad-1838626322) 、 [鸡蛋](https://lifehacker.com/just-make-deviled-egg-salad-already-1836077324) 等等。).

弄乱沙拉杯的唯一方法是不要把馅料切成足够小的块。例如，如果被牙齿咬住，大黄瓜片可能会滑出杯子，剥夺你享受每一口黄瓜的体验(并可能毁了你漂亮的衬衫)。如果可能的话，将馅料切成小块，并确保将它们与调料混合在一起，给它们一些粘性——只有在你吃的时候不会把所有东西都洒到身上，大块的馅料才是有趣的。
