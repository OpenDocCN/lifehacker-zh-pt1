# 这个拉伸程序实际上让我更加灵活

> 原文:[https://life hacker . com/this-stretching-routine-actually-make-me-more-flexible-1848820923](https://lifehacker.com/this-stretching-routine-actually-made-me-more-flexible-1848820923)

尽管我开始这个伸展挑战的想法是想知道我是否能变得更灵活，但我不得不承认，我有点暗暗希望这个挑战不会*奏效*。毕竟，如果它不起作用，那意味着我可以毫无负罪感地退出。

Watch

但是，该死的，*成功了*，现在我不得不承受后果。继续拉伸，让自己踏上旅程，不断思考自己离成功还有多远？或者放弃，知道如果我花时间的话，我本可以得到我的分裂，但是我没有，因为我是一个放弃者？

我想，这是一个让我在自己的时间里思考的两难问题。现在，让我们来看看发生了什么:

*   在过去的6周里，我拉伸了20次，平均每周3次
*   大多数日子我都做了 [这种拉伸动作](https://lifehacker.com/im-finally-stretching-and-i-kind-of-like-it-1848700419) 或者类似的动作。大概花了15分钟。
*   在过去的两周里，我做了同样的训练，但是用了PNF(又名提桶和提杆)技术，这里描述的。

柔韧性中一个众所周知的现象是，你可以在以下任何或所有活动后进行更深入的拉伸:(1)热身，例如有氧运动或其他运动；(2)拉伸几分钟；和(3)使用先进的拉伸技术，如PNF。这意味着在每次训练结束时，我总是能够比开始时更深地劈腿或更低地触趾。

所以，为了给自己一个比较公平的依据，我决定把我的前后图*冷*。我没有先做任何有氧运动(我通常在我的动感单车上热身)，我在那天第一次进入每个位置的几秒钟内就拍了照片。如果由于我的拉伸技术，我可以在一次会议中进一步拉伸，谁会在乎呢？但是如果我能在寒冷的环境下伸展得更远，那么我知道我在现实世界中变得更加灵活了。

这些是我尝试的正面劈叉。很明显，我还远没有坐到地板上，但是在之前的照片(红色衬衫，最上面一排)和之后的照片(黑色衬衫，最下面一排)之间有明显的区别。我的前腿离地面更低，你可以通过比较它相对于我前臂的位置来判断。这是更好的腿筋灵活性的结果。我也能够保持我的躯干更加直立，这要归功于后腿臀部屈肌更好的柔韧性。

(几周前我意识到，如果我的最终目标是躯干直立的前分，那么用道具支撑自己更有意义，这样我就可以在拉伸过程中保持直立。所以这些天我把前臂放在两把椅子上，实际上是在做这张椅子。但由于我在“之前”的照片中没有这样做，所以我确保在“之后”的照片中摆了相应的姿势。)

还是那句话，很微妙。在之前的照片中，我需要用一个物体(这里是一件红色的衬衫)将我的两只手拉在一起。但是在我右侧(右臂向下)的后一张照片中，我能够将我的手靠得足够近，以至于我能够用食指和拇指接触。在我好的一面，我的左臂放下，我现在实际上可以不用t恤的帮助把我的手钩在一起。

差别只有一英寸左右。但是伙计，我第一次不穿衬衫就能摸到手指，我被震惊了。

好吧，再来一个。回想一下，我们用一段视频开始了整个冒险，这段视频让我在三分钟内 [从不碰脚趾到把手掌放在地板上](https://lifehacker.com/how-to-finally-touch-your-toes-maybe-even-today-1848668349) 。这是会话内适应一个例子。但是对于我们最后的前后对比，我比较了我之前的照片和我今天摸脚趾头的照片。

嗯，*操*。它工作了。我生气了。是的，拉伸后的差异也是可比较的。第一天，在拉伸之后，我能够用柔软的膝盖支撑我的手掌。这些天来，在一个疗程结束时，我经常可以用手掌撑着地板，用*锁住*膝盖。拉伸很有效，我可能会继续做下去，如果你愿意，你也可以。布拉格。
