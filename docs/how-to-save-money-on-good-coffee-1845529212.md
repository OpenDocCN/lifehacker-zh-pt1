# 如何在(好的)咖啡上省钱

> 原文：<https://lifehacker.com/how-to-save-money-on-good-coffee-1845529212>

听着，如果咖啡是给你带来真正快乐的日常小习惯之一，我们不是来告诉你削减开支或少花钱的。花在牛油果吐司和拿铁上的钱会阻碍你成为百万富翁的陈词滥调 [过于简单，嗯，愚蠢的](https://www.marketplace.org/2020/08/14/now-that-millennials-arent-buying-avocado-toast-and-lattes-can-they-afford-a-home) 。所以，喝你喜欢的，为你买得起的“最好的”咖啡买单。



但是如果你想在喝咖啡的习惯上省下一两块钱，这里有一些方法可以保持高质量和低成本。

### 在家酿造

如果你大多数早上都出去喝咖啡，首先，感谢你对当地企业的支持。但这可能会很快累积起来，尤其是如果你点浓缩咖啡的次数比点滴滤咖啡的次数多的话。虽然在家酿造可能涉及设备的初始投资以及持续的咖啡豆购买，但从长远来看，它可以为你节省一些现金 。

不过，偶尔让别人为你煮咖啡真的很好，所以没必要完全取消去咖啡店的计划。

### 买大袋整豆

当暴露在空气中时，整粒咖啡豆几乎立即开始变得不新鲜，所以一旦你打开一袋新的咖啡豆，你就要和时间赛跑了。这一点对于咖啡渣来说更加明显，这也是为什么一般建议你在酿造咖啡之前只研磨你需要的东西。

如果你想要最新鲜的咖啡，但又想省钱，那就买整粒咖啡豆，按杯研磨。这将使你不会扔掉不新鲜的食物，也不会担心在得到一个新包之前的最后几天。

如果你喝了 12-16 盎司。快速装袋——要么你喝了很多咖啡，要么你正在为几个人冲泡咖啡——考虑购买更大的数量。一些烘焙商会以每盎司轻微的折扣出售几磅装的真正好的咖啡豆。将它们储存在密封的容器或罐子里，尽可能保持新鲜。

### 精确测量

用秤称出你的豆子，而不是依靠勺子。这可以确保你得到正确的比例，没有浪费。当你在做的时候，定期清洁你的研磨机。这可能只是在某个地方省下几分钱，但随着时间的推移，这些钱会越积越多。

### 混合饮料，而不是单一来源

单一产地的咖啡来自特定的农场或地区，通常在 T2、T4 以高价出售，因为这种咖啡豆只在一年中的特定时间供应。有些人更喜欢单一产地，因为味道可能更独特，但混合高品质的豆子也一样美味。这可以为你每个包省下几美元，同样，随着时间的推移，这些钱会越来越多。

### 牛奶饮料用便宜的豆子

如果你在制作拿铁咖啡，或者在咖啡中加入大量的甜味剂或糖浆，你可以选择不太贵的咖啡豆，因为味道会被牛奶、奶油或糖掩盖。

### 寻找订阅折扣

如果你定期订购整粒咖啡豆，一些个体咖啡烘焙商会提供优惠。还有一些 [咖啡订阅服务](https://nymag.com/strategist/article/best-coffee-subscription.html) 以很小的折扣从不同的烘焙商那里进货。你也可以在社交媒体或烘焙网站上找到交易，尤其是如果你是第一次顾客的话。
