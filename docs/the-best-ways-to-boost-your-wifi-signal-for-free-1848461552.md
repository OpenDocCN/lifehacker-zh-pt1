# 免费增强 Wifi 信号的最佳方式

> 原文：<https://lifehacker.com/the-best-ways-to-boost-your-wifi-signal-for-free-1848461552>

如果你已经达到了互联网计划的最高速度，但速度仍然不够快，升级你的计划可能是你最好的选择。然而，如果你测试你的连接，发现你远没有达到你的计划*应该*能够达到的速度，你并不总是需要花钱来提高你的性能。有很多方法可以增强你的 wifi 信号，而且完全免费。



你的 wifi 问题有付费解决方案吗？绝对的。你可以花更多的钱在你的 ISP 上，或者你可以买一个新的改进的路由器。当然，那些方法会让你付出代价，所以首先你应该看看你是否能在不花一毛钱的情况下加快你的数字生活。

## 将您的路由器移到开阔的地方

为了提高 wifi 性能，你可以做的最好的事情之一就是把你的路由器搬到户外。把路由器藏在没人会看到的地方是很诱人的，因为谁愿意盯着一个无聊的、闪烁的盒子呢？但是，事实是，你的设备想要免费。

你的路由器负责发出你的 wifi 信号，那个信号只和它的环境一样好；如果遇到任何干扰，信号就没那么有效了。干扰可能来自家中的物理元件，如墙壁和门，或者来自微波和无线电话等设备的其他电子信号。

记住这一点，试着把你的路由器移到不受这些干扰的地方。如果能把路由器抬高，比如放在开放式的架子上，那就更好了。因此，与路由器隐藏在墙壁、门和微波炉后面相比，您连接的设备将接收到更强的信号。

## 靠近路由器

同样，尽量在靠近路由器的地方使用设备。没有必要直接坐在笔记本电脑上的路由器旁边，或者强迫你的 PlayStation 和路由器附在臀部，但如果你想让你的 wifi 信号尽可能好，你会希望留在视线范围内。

## 不要在网络上同时运行多个设备

你可能习惯于在网上一次做很多事情，比如一边和朋友玩游戏，一边在网飞网上观看 4K 的节目，同时把一个系列下载到你的 iPad 上。所有这些多任务处理都很棒，但它会给你的整体网速带来压力。

如果你想提高 wifi 的速度和性能，比如说尽可能快地下载文件，你不希望其他互联网密集型任务同时运行。在这些时候，尽一切努力暂停那些其他的任务。如果你和其他人住在一起，这可能需要礼貌地要求朋友和家人暂停一会儿他们最喜欢的节目，或者停止观看 Twitch(向粉丝道歉)。

## 经典的，关掉路由器再打开的动作

路由器遵循所有电子产品都遵循的规则:如果有什么不对劲，就关掉然后再打开。如果你的网络出问题了，而你不知道为什么，打开路由器的电源是一种恢复正常的方法。

## 使用更好的 DNS

DNS，或域名系统，通常被称为互联网的电话簿。当您访问网站时，您的设备会通过 DNS 查找来确认它试图访问的网站。毕竟你的电脑需要知道的是网站的 IP 地址，而不是名称本身。这种查找与数据库通信，以查看是否列出了您条目的 IP 地址。如果没有，搜索将会在更大的数据库中进行，最终结果会被发送回来，从而允许您连接到该站点。

如果你以前从未摆弄过你的互联网设置，你很可能使用你的 ISP 的 DNS，它通常比它需要的要慢。幸运的是，切换到更好的 DNS 并不十分困难。 [我们有一篇文章是关于用 CloudFare 的 DNS 服务](https://lifehacker.com/how-to-browse-faster-and-more-securely-with-cloudflar-1824256064) 设置你的设备的，该公司声称这比谷歌的更快。就谷歌而言， [希望通过其 DNS 连接](https://developers.google.com/speed/public-dns/docs/using) 来提高你的网速，所以如果 CloudFare 不适合你，你可以随意尝试一下。
