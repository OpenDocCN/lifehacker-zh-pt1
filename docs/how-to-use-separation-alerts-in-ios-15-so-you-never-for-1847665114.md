# 如何在iOS 15中使用“分离提醒”,这样你就再也不会忘记你的手机了

如果你像我一样，出门总是最后一刻上气不接下气的冲刺，因为你拼命避免迟到。因此，像钥匙、包、甚至MacBooks这样的东西经常被遗忘，等到想起来的时候已经太晚了。如果这对你来说是真的，你会想要利用iOS 15的新分离提醒，永远不会再偏离你的物品太远。

Watch

当然，分离警报不仅仅是为了把东西留在你身后。如果有人弄坏了你的设备，你的iPhone不会让他们知道。这是一个很好的第一预警系统，适用于任何你与你的物品分离的情况。此外，由于它与“查找我的”配合使用，您将能够看到设备最后一次连接到网络的位置。

## 哪些设备支持 **iOS 15** 分离提醒？

苹果公司没有一个完整的设备列表，但这里有一个列表，它目前可能支持:

*   运行iOS 15或更高版本的iPhone
*   运行iPadOS 15或更高版本的iPads
*   运行macOS Monterey或更高版本的MAC
*   运行watchOS 8或更高版本的Apple Watch
*   AirTags(目前还不知道具体的固件版本)
*   查找我支持的配件，如VanMoof的S3和X3电动自行车、Belkin的SOUNDFORM Freedom True无线耳塞以及Chipolo ONE Spot物品查找器

## 如何在您的 iPhone上设置分离提醒

一旦你知道去哪里找，分离警报就非常容易设置。首先，打开“查找我的”应用程序，然后轻按“设备”或“项目”，具体取决于您要为其设置离职提醒的对象。点击有问题的设备，然后向上滚动菜单。如果该设备与该功能兼容，您会看到一个“留下时通知”选项。轻点它。

在接下来的页面上，您会看到“**通知何时离开**”只需点击该选项旁边的滑块即可启用该功能。点击“完成”锁定您的设置。现在，无论何时你离开那个特定设备的附近，你都会在你的iPhone上收到提醒。

您还会注意到这个页面上的“**通知我，除了“**”选项。此选项允许您设置“查找我的”不会为其发送离职警报的特定位置。例如，你不希望每次出门时，你的iPhone没有带着你的“查找我的”自行车就对你大喊大叫。这就是为什么你可能会看到iOS推荐你的家作为选择的位置。

要选择您自己的位置，请点击“**新位置**”默认情况下,“查找我的”会调出您的当前位置。若要选取自定位置，请轻按该页面顶部的搜索栏，输入您选择的地址，然后轻按“**搜索**当“查找我的”调出您的位置时，您可以通过长按红色大头针并将其拖动到您喜欢的位置来进行微调。

此外，您可以使用蓝色圆圈来扩展此位置覆盖的半径；当你改变半径时，查找我的将会让你知道你的半径是多少英尺。如果你想让我帮你改变半径，你可以从下面的菜单栏中选择**小**、**中**或**大**，然后观察你的半径相应的变化。
