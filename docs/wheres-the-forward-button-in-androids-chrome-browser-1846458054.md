# Android的Chrome浏览器的前进按钮在哪里？

> 原文:[https://life hacker . com/where-the-forward-button-in-androids-chrome-browser-1846458054](https://lifehacker.com/wheres-the-forward-button-in-androids-chrome-browser-1846458054)

有时候，我会为《生活黑客》周刊的 [**科技911**](https://lifehacker.com/c/tech/tech-911) Q &专栏提出一些相当激烈的问题——黑客这个，安全那个，为什么我的设备上有色情之类的问题。真正的不朽之作。本周，我将通过帮助一位读者解决一个令人烦恼的Android问题来减轻语气，这个问题你可能会不时地问自己。

Watch

Lifehacker读者 **Loretta** 写道:

> *感谢您提供的关于Tech 911的精彩提示！我的问题是....为什么我可以在我的Android智能手机上返回，但不能转到我返回并希望返回的网站。相反，我不得不重新打一遍。*

啊，铬合金。因此，不管出于什么原因，谷歌已经决定，在Android的 UI中一个非常明显的位置放置一个专用的“后退”按钮，这比将任何像素专用于其同伴强大的“前进”按钮更重要。(至少，在安卓上。)

说实话，我不能怪他们。我整天都在网页上跳来跳去，但是我发现自己回去的次数比前进的次数多得多。毕竟，当你回到面包屑路径，也就是你的网络会话时，你总是可以通过简单地点击或轻触你之前点击的链接再次“前进”。然后你回到你刚才浏览的网站。不过，不可否认的是，如果使用“后退”按钮、滑动或快捷方式将你带回网页的开头，然后你必须四处搜寻才能找到带你到任何地方的链接，这就有点麻烦了。

不管怎样，这显然让你很烦，所以我有一些解决办法。首先，也是最重要的，这是你可以在Android的Chrome应用中找到前进按钮的地方。只需点击浏览器右上角的三点图标。寻找指向前方的箭头，你不会错过的。

不过，我会给你更好的。在你浏览器的地址栏中输入 *chrome://flags* ，搜索“历史”在结果中，寻找名为**“滚动历史导航”的选项**

启用它，重启你的浏览器，你现在应该可以从Chrome浏览器窗口的最左边和最右边滑动来前进和后退。我认为这对你来说甚至比点击一个按钮(或一系列按钮)更快，尽管我真的希望Chrome有一个简单的设置，在浏览器的右下角为那些想要它的人放下一个向前的箭头。
