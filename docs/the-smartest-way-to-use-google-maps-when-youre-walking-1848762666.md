# 独自行走时使用谷歌地图的最聪明方式

> 原文：<https://lifehacker.com/the-smartest-way-to-use-google-maps-when-youre-walking-1848762666>

在一个陌生的城市漫步会很有趣。它也可能令人畏惧，尤其是当你独自一人的时候，天黑后更是如此。没有人想让自己看起来不知道要去哪里，盯着智能手机上的谷歌地图经常会泄露秘密。如果你想自信地到达你想去的地方而不引人注意，谷歌地图有一个很好的解决方案。

Watch

当使用步行方向时，谷歌地图可以让你大声知道接下来该拐什么弯，但如果你不想让你的手机脱口而出你接下来需要做什么，或者通过戴耳机淹没你的周围环境，还有另一种方法:谷歌地图也可以无声地为你导航，使用振动。

使用不同的振动模式取决于下一个轮次。点击音量图标并选择**静音**。然后，简单地把它放进你的口袋，等待下面的一个振动来引导你:

**左转**:长蜂鸣，短蜂鸣，短蜂鸣

**右转**:短鸣、长鸣、长鸣

就是这样！记住这些振动模式，你将永远不需要看应用程序来知道你下一次应该什么时候和如何转弯。这个功能在 Android 和 iOS 上都可以使用，而且默认情况下应该可以使用，只要你选择**静音**模式。

[Reddit 用户 u/plzubblewrappmy 为我们强调了这个特别有用的功能](https://www.reddit.com/r/YouShouldKnow/comments/ttxwd4/ysk_if_youre_ever_in_an_unfamiliar_or_sketchy/) ，但是 t hread 也提供了其他有用的建议。如果你*习惯戴着耳塞四处走动，谷歌地图可以给你声音指引。但是，您需要从**静音**切换到**仅提醒**或**取消静音**。智能手表还可以帮助你解决一个微妙的问题:看一眼你的手腕比盯着你的手机要不那么显眼。*
