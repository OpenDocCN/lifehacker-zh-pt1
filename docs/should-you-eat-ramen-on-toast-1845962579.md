# 应该吃吐司拉面吗？

> 原文:[https://life hacker . com/should-you-eat-ramen-on-toast-1845962579](https://lifehacker.com/should-you-eat-ramen-on-toast-1845962579)

我不知道这是什么时候发生的，但在某个时候，我们——作为一个社会——认为食用一种碳水化合物是不礼貌的。也有一些例外，比如塔可钟的 [现已退役的](https://skillet.lifehacker.com/learn-how-to-make-taco-bell-potatoes-because-they-will-1844608145) 辣土豆软塔可和英国的薯条黄油，但在大多数圈子里，carb-on-carb是相当罕见的配置。

土豆是最容易逃脱的。我上面提到的taco和butty(在英国是“奇怪的小三明治”)都使用土豆作为“填充碳水化合物”，以小麦为基础的碳水化合物作为输送系统。另一方面，摄入两种以小麦为基础的碳水化合物感觉很尴尬——但这正是读者Kurt建议我做的。

然而，尴尬并不总是坏事。尴尬的时刻和情况可能会令人兴奋，如果不舒服的话。当你舒适的时候，没有什么真正令人兴奋的事情发生。虽然说库尔特的小吃“令人兴奋”可能有点夸张，但我确实有一会儿有点不舒服，结果完全不是我所期望的。

起初，似乎有太多的液体参与其中，但海藻很快就完成了工作，吸收了几乎所有的“调料”,制成了一种海鲜面条沙拉。这很令人惊讶，但没有透露太多。就连没有按照邮件中给出的说明行事的乔尔也对结果感到惊讶...食物(观看视频，看看我们有多惊讶)。如果你碰巧是一个错过螃蟹沙拉(或krab沙拉)的素食者，这种小吃可以用无蛋蛋黄酱来制作，以满足你的需求，我想你也会惊讶于自己有多么喜欢它。
