> 原文：<https://kotaku.com/keyboard-group-buys-rule-when-they-don-t-suck-1847468935>

机械键盘社区比我参加过的任何团体都更加狂热。经常出没于此的顽固的 keeb 怪胎们并不满足于等待某个大牌制造商做出好东西。这些人准备好并愿意设计、资助和资助他们自己的硬件和配件，如果你想自己获得一些不错的装备，你需要准备好跟随他们独特的流程。

大多数情况下，这种定制过程涉及到一种叫做团购的东西，在这种情况下，一群感兴趣的粉丝会凑钱来实现他们的键盘梦想。就像生活中的许多事情一样，等待是最难的部分。

你可能对团购这一更为主流的形式很熟悉。这真的是一种众筹形式，就像你在 [Kickstarter](https://www.kickstarter.com/projects/keyboardio/atreus) 或 [Indiegogo](https://www.indiegogo.com/projects/epomakerb21-retro-wireless-mechanical-keyboard#/) 上看到的项目。像这些平台一样，键盘团购从利益相关方那里收集资金，然后将这些资金用于产品的生产。与其冒险生产一堆没人想买的东西，不如从一个已经投资了你产品的专门客户群开始。这是一个很好的获得资金和生产产品的方法。

因此，团购是一种更草根的众筹形式。键盘爱好者不使用更大的平台，而是去类似[*geek hack*](https://geekhack.org/index.php)T5】或者我个人 go-to， [Reddit 的 r/Mechmarket](https://www.reddit.com/r/mechmarket/) 这样的社区论坛。他们将开始谈论他们对新键盘外形或新自定义按键集的想法。如果你去 r/Mechmarket，按 [IC(兴趣检查)flair](https://www.reddit.com/r/mechmarket/search?q=flair%3A%22Interest+Check%22&restrict_sr=on&sort=new&t=all) 排序，你会发现个人或小公司正在寻求启动的一系列项目。创作者将展示原型，分享制造细节和规格。

例如， [Redditor Cyberbay](https://www.reddit.com/user/cyberbay) 最近更新了他们的 Bubble 75 键盘的兴趣检查，这是一种可爱的小尺寸、垫圈安装键盘，具有独特的布局。Cyberbay 甚至上传了一个打字视频，这样人们就可以听到项目完成后的声音。

当创作者激起足够的兴趣时，就到了团购的时候了。键盘或附件的规格被锁定，制造时间被估计，钱从感兴趣的团体被收集。目标是支付生产的所有方面，从工具到运输，确保每个人的钱都用在正确的地方，团购发起人不会最终不得不从自己的口袋或其他地方拿出额外的钱来完成项目。

团购的一个问题应该是显而易见的，那就是先付钱，后收钱。最令人失望的事情莫过于手头有点闲钱，找到完美的键盘或按键或诸如此类的东西，然后发现几个月到一年都不会发货。对于新的机械键盘购物者来说，这是一个巨大的挫折，他们认为他们可以买到他们想要的东西，然后就完事了。

我说过等待是最难的部分吗？更多的是等待和希望一切按计划进行。我在团购中购买过一些零件，这些零件由于工厂错误而被推迟了几个月，比如钻错孔、电子设备损坏以及拉丝铝外壳上的随机划痕。我有过这样的按键集，其中某些按键不知何故偏离了中心，不得不被替换。我个人最喜欢的团购失误发生在 2017 年，当时组织购买小型键盘的人对参与者的电子表格进行了错误排序，以至于每个人的键盘开关选择都搞混了。幸运的是，这是一个热插拔板，所以我不需要重新出售任何东西，但请各位，学习如何电子表格。

从好的一面来看，一次成功的团购会产生一件高度受限的物品，如果处理得当，可以在售后市场上卖个高价。一旦团购键盘开始在顾客手中铺开，来自没有参与的人们的 FOMO 是超级强大的。作为一个额外的奖励，人们买下限量版的运动鞋或游戏机以荒谬的价格转售？看来他们还没发现机械键盘团购市场。

只是要小心，FOMO 是一把双刃剑。想象一下，找到了你梦想中的键盘，并意识到它是已经关闭的团购的一部分。当然，在某些时候可能会有临时演员，但是真的会有吗？(答案通常是否定的，你完蛋了。)

虽然看到一个很酷的新键盘硬件，然后意识到只有现在付钱，然后等几个月才能买到，这很烦人，但有时符合你所有标准的精心定制的对象是值得等待的。在我不断增长的收藏中，一些我最喜欢的键盘开始于我填写一个原始的谷歌表格，说是的，我有几百美元和一大堆耐心。

参加一个运营良好的团购会有令人难以置信的回报。做好你的调查，确保你不介意等待，然后抱最好的希望。
