# 无人的天空“完美星球”色情:20 个你可以参观的必看世界

> 原文：<https://kotaku.com/no-man-s-sky-planet-porn-ghgs-hello-games-1849655173>

《无人天空》的一大乐趣是发现令人兴奋的科幻柯达瞬间。该游戏自发布以来已经发生了巨大的变化，但漫游和发现这个程序生成的宇宙的奇迹的能力仍然是其众多魅力之一。

虽然游戏的更新美化了画面，给了玩家更多的活动和任务，但至少对我来说，新发现和美丽一直是这个游戏的核心。

因此，让我们利用一些卡尔·萨根式的对宇宙奇迹的热情，再来点摇滚后的音乐景观，让自己真正兴奋起来...这些精选的照片是在银河地质勘测中心的完美行星相册中找到的。如果你发现了自己的奇迹，你可以顺便加入 [并为](https://www.reddit.com/r/NMSGalacticHub/) 做出贡献。

所有这些世界都符合温室气体完美星球的标准:它们没有风暴，哨兵活动少，这里的天气不需要危险防护，不管一天中的什么时候。

诺姆伽马是穆里劳斯发现的一颗茂盛的生物群落行星。这里的突出是显而易见的；前景中奇妙的奇异植物，水的颜色，以及背景中看起来像这个星球的光环(或另一个星球的光环)的条纹，这一切都描绘了这种奇妙奇怪而美丽的科幻景观。

《无人区》的美丽并没有随着太阳的落山而结束。由 docteurshepherd 发现的 Saint Cloud 提供了一个完美的例子，说明为什么有生物发光植物群和草的行星应该在晚上探索。如果你没有独自在星光下漫步，那些生物发光的草地经常随着光波波动，使第一人称或第三人称的漫游变得迷人。 [看看这个视频](https://youtu.be/pn9O6GmA4j8) 是另一个星球上这种现象的例子。

这张由幽灵树懒发现的 Corneria 的照片是另一张精彩的夜景照片。随着更多的发光植物照亮道路，以及远处看起来像某种极光效果的东西，据说这颗行星的表面下还有漏斗状的洞穴。(遗憾的是，没有发现福克斯·麦克劳德的踪迹。)

这张由 Phaedrus29 发现的 Pezezui 的超酷照片有很多典型的 NMS 特质。其中最吸引人的可能是远处的那颗行星，可以通过它的大气层日冕看到。知道另一颗行星离我们只有一步之遥是在天空中发现其他行星的兴奋之一。它几乎看起来像一个设计好的天空盒。

这种游戏中的各种颜色总是一种享受。由马洛希皮发现的 Tecitu Tau 上这些植物的紫色奇异特性就是一个完美的例子。紫色就在这华丽的绿色画布上突然出现。如画廊所示，它们在晚上会发光！

由 MisterManix 发现的 Manixa，看起来就是自由主义者想要的未来。茂盛的绿草、泡泡和一条大彩虹。我可以在这里冷静几个小时。

自 2016 年 *NMS* 开播以来，野生动物已经走过了漫长的道路。这张奥拉星上一只动物的快照就是一个很好的例子。由 THompz 69 发现，这看起来是一个结交动物朋友的好地方。

TWC 杀手在欧拉雷，一颗非常深绿/蓝色的星球上，抓拍了一张类似三角龙的生物的漂亮照片。这里的比例感很强，巨大的恐龙占据了前景，远处的山和云点缀着风景。

在 NMS 发现行星和卫星彼此非常接近并不罕见。mathfreak2 发现的 Lutonbur Ondai 的天堂月亮就是这样一个例子，它展示了这种情况下可能产生的令人敬畏的美丽。这里的规模和范围令人印象深刻，因为你知道只要你愿意，你可以跳上一艘船去那里。

然而，我们又有了一个例子，在任何其他游戏中，这只是一个固定天空盒的一个方面。但是，由 dark discipled 发现的新费里塔，是一颗行星用一个横跨天空的环炫耀其程序生成的美丽的地方之一。我玩这个游戏的一些最有趣的事情就是跟着这样的环走，通常是步行。

“叠加 97”发现的卫星 Lisusi Beta 可能看起来非常简单和不起眼，但不要错过远处隐约可见的行星。像这样的远景只是让你在这个游戏中感受到的规模。

由 C0BaLt277 发现的 Thi Alpha 是另一个拥有发光植物生命的世界。这张夜景照片展示了星空下的灯光秀，背景是奇妙的渐暗的蓝色。

阿齐托邦 Bucolisphere 是由 LordAzurio 发现的一颗画面完美的行星。这张照片展示了你可以在那里找到的华丽的蘑菇状植物。

在这张由 zhunt69 发现的 Owst E45 的快照中，发生了很多事情。一、怎样才能忽略颜色？把任何东西都涂成紫色，我会爱上它，但这里也有很好的色彩光谱，世界之环扫过天空，另一颗行星在地平线上隐约可见。

由特法斯发现的魅魔适马是一颗完美的沉思星球。平静、中性的颜色和似乎永远模糊不清的地平线只会让你迷失在自己的思绪中。这颗遥远星球的日冕预示着在这个奇异的世界之外会有更大的发现。

这张由 Phaedrus29 发现的 Ohit J17 的照片，看起来像是可以被裁剪成适合一部旧的低俗科幻小说。大气层的薄雾轻轻地模糊了远处的星球，在这里传达出一种深度感和真实感。而且，再一次，知道这样一个东西不仅仅是一个天空盒子，而且如果你想，你可以飞到那个星球，这创造了一个完美的 NMS 时刻。

ApexFatality 发现的 Gardsome Beta 展示了*NMS*更有创意的植物生命。你怎么描述这样的事情？这种球根状、螺旋状的游戏算法创意表现实在太酷了。

如果我要把这些照片中的任何一张裱起来挂在墙上，那就是这张由洛萨-维京发现的照片。这张照片非常清晰，使它从其他光环和盘旋的行星照片中脱颖而出。能够从这个地点看到那个遥远世界的各个大陆有助于传达这个游戏能够达到的沉浸水平。

我们将以一个寒冷的音符结束，由 MercutioElessar 发现的 Neseycian Gamma 上的一个非常多云的日子。在这令人忧郁的风景中，一道美丽的彩虹横跨地平线。它只需要一个长的，低的，单调的环境片段，伴随着轻微的静电声在立体声音场中渐强渐弱。
