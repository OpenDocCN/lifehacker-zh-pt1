# 用速溶浓缩粉使暗淡的辣椒变得有生气

> 原文：<https://lifehacker.com/perk-up-lackluster-chili-with-instant-espresso-powder-1847790786>

周五晚上，我请了几个人来吃纯素辣椒。如果你不是素食主义者，纯素食辣椒可能听起来不那么令人兴奋，我可以告诉我的几个客人(除了纯素食者)对我选择无肉、无奶的选择不太感兴趣。(我为杂食动物提供了酸奶油和奶酪，但它们仍持怀疑态度。)

然而，一旦他们尝到了，一切都变了。“这实际上真的很好，”他们半耳语，用准歉意的语气咕哝。(为什么他们会怀疑我，我也不知道！)

至少有一半的功劳要归功于 A.A. Newton 的 [素食 Frito 馅饼食谱](https://lifehacker.com/this-soyrizo-frito-pie-will-please-the-whole-team-1841333300) (我用它作为模板，但用红酒和额外的西红柿进行了调整)，但乔氏品牌 soyrizo，这是一种非常好的假肉制品，做了很多繁重的工作。当与西红柿、豆子、洋葱和土豆一起炖时，它看起来几乎完全像碎肉，至少在质地上是这样。然而，辣椒尝起来有点太 T4 了。

好的辣椒应该有一种黑色——应该有一种烘烤过的、浓郁的味道来调和西红柿的水果甜味和酸味。鲜味很容易添加，甚至与素食菜肴一起添加——只需在其中加入一些味精——但结晶谷氨酸盐无法弥补你通常从美拉德反应中获得的风味化合物的缺乏。如果没有这些味道，辣椒对我来说就没有味道了。褐色肉类在这方面做得很好，但那是不在餐桌上的，而且 soyrizo 太粘了，以至于不能获得大量的褐色。

当我环顾我的储藏室时，我问自己，什么能贡献出通常由美拉德反应带来的美味？我没有足够的时间 [烧洋葱](https://lifehacker.com/transform-any-creamy-dressing-with-burnt-onion-1837851760) ，我开始慌了，但后来我看到了:一小罐梅达利欧浓缩咖啡粉，它已经在我的餐具室里放了至少六个月了。我在辣椒里加了一茶匙，搅拌直到它溶解，然后尝了尝。情况立即有了明显的改善。我又加了一点，又尝了一遍，终于放松了。辣椒尝完了*。*

*即使你的辣椒不是纯素食主义者或素食主义者，一点点浓缩咖啡粉或速溶咖啡可以给它更深，更黑，更完整的味道。你不需要很多，所以慢慢来——一次加入一茶匙或更少，搅拌直到完全溶解，每次加入后品尝，直到味道变好。你是否用奶酪和酸奶油来结束它取决于你——它不需要它，但辣椒是一种几乎总是受欢迎的食物。*
