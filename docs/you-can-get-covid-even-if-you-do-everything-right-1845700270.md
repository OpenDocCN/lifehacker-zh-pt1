# 即使你做的一切都是正确的，你也可以得到 COVID

> 原文：<https://lifehacker.com/you-can-get-covid-even-if-you-do-everything-right-1845700270>

例有 [成倍攀升](https://ronaviz.com) 。如果你还不知道谁患有 COVID，你很快就会知道了。随着病毒变得越来越常见，它也更有可能感染那些采取预防措施的人。



这部分是因为我们的安全措施不完善，但也是因为这个春天和夏天没有让我们在情感上为今年秋天我们生活的世界做好准备。因此，这里有一些提醒，告诉你即使你尽最大努力避免冠状病毒，你也可能会感染它。

### 你的泡沫比你想象的要大

如果你已经和几个家庭成员和亲密朋友建立了一个泡泡，你可能会认为你和他们在室内闲逛是安全的，并且不戴面具也能保持身体上的亲密。但是你怎么知道那些人真的没有 COVID 呢？

大多数情况下，你不会。你的泡泡不只是那些人；从某种意义上说，是那些人*加上*所有和他们有过密切交往的人。( [这张图](https://twitter.com/IBJIYONGI/status/1327736298329739264) 完美地说明了这个概念。)一旦你开始包括来自多个其他家庭的人，你的潜在携带 COVID 的联系网络就会扩大到几十人，或者更有可能是几百人。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1327736298329739264&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1327736298329739264&autosize=1) 

人们很容易认为，如果你很了解某个人，他们应该在你的泡泡里。但是，如果有人有一份与公众互动的工作，或者如果他们的孩子去了面对面的学校，或者如果他们有时与一个不像他们那样谨慎的朋友出去玩，那这个泡沫就有一个很大的空洞。真正拥有泡沫的人就像他们认为的那样安全，这种人少得可怜。

### 规则并不总是保护你

如果你做的每件事都是“正确的”，你如何定义它？在许多情况下，我们通过以下指导方针来做到这一点:如果你回到健身房、发廊、餐馆和你自己的工作场所，当它们重新开放时，你可能会认为所采取的预防措施足以保护人们。毕竟，的负责人做出了本该保护我们的决定。

但是这些规则并不总是足以保证我们的安全。通常，他们牺牲安全来支持其他目标，如允许企业重新开业，减轻州长和其他领导人的政治压力。一个教科书般的例子: [符合所有规则的室内自行车工作室](https://vitals.lifehacker.com/following-the-gym-rules-wont-always-protect-you-1845392720) 爆发。唯一的问题是，规则允许人们戴着面具在同一个封闭的空间里一起锻炼一个小时。人们当然会生病。

### 没有完美的安全措施

即使你对泡沫的大小保持警惕，即使负责人制定了合理的规则，即使你尽了最大努力，冠状病毒仍然可以通过。

另一个有用的例子是瑞士奶酪模式 ，每一层都降低你的风险。保持距离并不完美，但可以降低你的风险。口罩并不完美，但可以降低你的风险。在户外呆一段时间并不完美，但是可以降低你的风险。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1319901144836026368&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1319901144836026368&autosize=1) 

瑞士奶酪模式是一个好东西:即使其中一层干预缺失，其他层仍然可以保护你。将预防措施叠加在一起可以进一步保护你，就像你如何系好安全带一样*和*保持汽车刹车正常*和*小心驾驶。

但这也意味着不能保证。即使所有的层都堆叠在一起，每一层都只会降低你的风险；没有一个能完全消除它。你完全可以“正确”地做任何事情，但仍然会有一些病毒粒子通过。

### 你所在社区的人数比你采取的任何具体行动都重要

因为没有什么预防措施是完美的，你周围发生的事情很重要。如果现在是二月，而你所在的城镇只有一个人是 COVID 阳性，你不太可能在预防措施很少或很少的情况下感染冠状病毒。一旦出现几十个病例，预防措施就变得更加重要。

当病毒在你所在的地区肆虐时，它很可能会超出你的预防措施。想想用闪粉做手工:如果你只拿出闪粉几分钟，作为你项目的最后一笔，你可能能够控制住它，并且你可以用吸尘器清理掉你错过的零散部分。但是，如果你的孩子去一家闪光粉工厂实地考察，并在里面游泳，第二天*就会有*的闪光粉在你的袜子抽屉里，即使你尽了一切可能，从在她进屋前用软管冲洗她开始。

如果你生活在冠状病毒肆虐的地方，你可能对这个概念很熟悉。但是，如果你个人直到最近才知道谁感染了 COVID，你基本上在八个月的过程中自学了无论你采取什么预防措施都足够好。随着病例的增加，风险/收益平衡发生了变化。是的，即使你像以前一样遵循预防措施，你现在也可能感染冠状病毒。

最终，这意味着一些采取了所有可能的预防措施的人仍然会感染新冠肺炎。有些人尽了最大努力，却在某处出了差错，或者无法避免某些风险，也喜欢去工作。没有必要为跟随诊断而感到羞耻或耻辱，因为那会让我们更难坦诚地谈论它。尽可能遵循预防措施，尽量不要将阳性病例视为个人失败。有时候我们真的在尽力。
