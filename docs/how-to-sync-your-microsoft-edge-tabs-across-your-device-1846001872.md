# 如何在您的设备间同步您的 Microsoft Edge 选项卡

> 原文：<https://lifehacker.com/how-to-sync-your-microsoft-edge-tabs-across-your-device-1846001872>

Edge Chromium 终于可以让你在所有设备之间同步打开的标签页和浏览历史。这些承诺已久的功能预计将在 2020 年夏天推出，但在测试几个月后，他们终于推出了。浏览历史和打开标签同步可以在 Windows 10、macOS、Android 和 iOS 上运行，但你现在能否访问这些功能取决于你住在哪里。



如果你在英国，你很幸运——那里的用户是第一批获得新同步选项的，但他们应该很快会在其他国家推出。我在美国，在 Windows 10 和 Android 上 Edge 的 profile settings 菜单中可以看到选项，但暂时是灰显的。

如果你不住在英国，希望历史和标签同步早，你可以 [下载 Edge Chromium](https://www.microsoftedgeinsider.com/en-us/) 的测试版(标签和历史同步从 11 月开始提供)，但新功能应该很快就会在稳定版中准备好。

### 如何在 Edge Chromium 中打开历史记录和标签同步

一旦可用，设置打开标签和浏览器历史同步很容易，但你需要在至少两台设备上登录同一个 Microsoft 帐户才能工作，并且这些选项还需要在你使用的每台设备上切换。具体的方法因平台而略有不同。

**在桌面上(Windows 10 和 macOS):**

1.  开口铬合金。
2.  轻点**“…”**在浏览器的右上角。
3.  进入**设置>简档>同步**。
4.  确保同步已打开，然后从列表中选择**【历史】**和**【打开标签】**。

**在移动设备上(Android 和 iOS):**

1.  轻点**“…”从屏幕底部的菜单中选择**。
2.  选择**设置**
3.  向下滚动并点击您的 Microsoft 帐户名称。
4.  轻按**“同步”**
5.  在同步菜单中，确保打开了**“Sync”**选项，然后选择您想要在设备之间共享的数据。“历史记录”和“打开”选项卡应该对所有人都可见，但对于尚未提供该选项的国家的用户来说将是灰色的。
