# 你应该用剩余的咖啡(和其他饮料)来“浇灌”你的植物吗？

> 原文：<https://lifehacker.com/should-you-water-your-plants-with-leftover-coffee-and-1846877841>

当我们把一堆几乎空空如也的玻璃杯和马克杯从工作间搬到厨房时，我们都想知道:如果我把这些倒在我的盆栽植物里，会发生什么？



可能什么都不会发生——至少不会马上发生。虽然一些普通的饮料不应该用来“浇灌”植物，但其他的饮料是无害的，甚至是有益的。这完全取决于饮料中含有什么样的化合物。

## 萨尔茨:非常好

所有的饮料大部分都是水，但是苏打水——即使是有味道的——真的很接近普通的自来水。由于碳酸化和调味香精，它肯定更酸，但 [大多数植物在弱酸性土壤中生长最好](https://www.almanac.com/plant-ph) ，所以这不一定是件坏事。(盆栽土壤的 pH 值在 6-7 之间，通常更接近 6。)一两盎司的扁南瓜是无害的，尤其是如果你在把它倒进你的植物之前用自来水稀释的话。

## 纯茶和/或咖啡:实际上很好

我们最喜欢的含咖啡因的饮料只是死去的植物物质和水，这恰好是生长中的植物需要的三样东西中的两样。(光是第三。)说死去的植物物质含有少量的 [必需营养物质，如磷、](http://It’s not the same thing as using actual fertilizer, but u)和，这些使产生的饮料呈微酸性。基本上，剩下的茶或咖啡是一种非常弱的肥料。

不过，请记住，这只有在你喝黑茶和/或咖啡时才有效。出于很快就会变得显而易见的原因，甜味剂和奶精不应该出现在你的植物中——尤其是如果它们生活在你的房子里。

## 酒、果汁、奶制品和其他甜的或高脂肪的饮料:很难通过

如果你曾经害怕把酒瓶的残渣倒进你的榕树里，你的直觉是对的——即使你的推理不是。低剂量的乙醇对植物并没有那么坏，但是酒精饮料(和其他饮料)中的糖和其他碳水化合物肯定是有害的。

植物不需要使食物对人类有营养的碳水化合物、脂肪和蛋白质。虽然一点点糖不会立即被判死刑，但随着时间的推移，给植物提供它们不能也不会利用的养分会杀死它们——如果它们吸引的 T2 害虫 T4 还没有完成任务的话。昆虫喜欢糖、酵母、脂肪和任何它们可以清除的东西；将含有这些成分的饮料倒入少量的盆栽土壤中，一场 [蚂蚁](https://lifehacker.com/use-two-kinds-of-bait-to-solve-your-ant-infestation-1846716825) 或 [真菌小昆虫](https://lifehacker.com/how-to-kill-fruit-flies-and-their-lookalikes-1846592792) 的侵扰就要发生了。

如果用除了纯净水以外的任何东西给植物浇水的想法仍然让你紧张，你当然没有必要这样做。但是如果你经常在家里放一两英寸的咖啡，这是一个很好的利用这些液体的方法。只要记住剂量制造毒药:当有疑问时，用自来水稀释(糖/乳制品)饮料，并将其分散到多株植物上。
