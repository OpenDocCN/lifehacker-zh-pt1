# 先看剧透，这样你就可以看恐怖电影而不感到可怕

> 原文:[https://life hacker . com/read-spoilers-first-so-you-can-watch-a-horror-movie-wit-1847857010](https://lifehacker.com/read-spoilers-first-so-you-can-watch-a-horror-movie-wit-1847857010)

普遍的共识是 [大家都讨厌剧透](https://gizmodo.com/this-was-the-decade-that-spoiler-culture-changed-everyt-1838626676) ，如果你在网上剧透一部电影或电视剧，你就是一个坏人。我倾向于同意。在社交媒体或其他地方不小心泄露大的情节是一件小事(我关注了20年的一个电影博主决定在新的詹姆斯·邦德电影*上映前就用标题*毁掉它；更准确地说，他现在是我*用来*追随的电影博客作者。冒冒失失糟蹋东西，就像他们说的 [一个不折不扣的浑蛋举动](https://lifehacker.com/how-to-know-if-youre-an-asshole-1844857318) 。但是有一次我几乎每次都支持剧透，那就是我看恐怖电影的时候。不是我不喜欢恐怖；只是看着它让我感觉很糟糕(这种情况只会因疫情而加剧，在那段时间里，我的娱乐饮食主要是舒适的食物。但如果我先看剧透，我会感觉不那么糟糕，我可以专注于情节、节奏、摄影和音乐——你知道，电影的“电影”部分——而不是我内心焦虑的巨大鸿沟。这与 [的一项经常被引用的研究结果相吻合，该研究表明，如果人们在已经知道将要发生什么的情况下经历一个故事，他们会更喜欢这个故事。结果对被要求阅读三种类型故事之一的参与者来说是正确的——推理小说、欧·亨利式的讽刺性扭曲故事和“具有简洁分辨率的文学小说”；正如加州大学圣地亚哥分校的研究员尼古拉斯·克里斯腾菲尔德所说:“关键是，我们真的不是为了结局而看这些东西。我向怀疑者指出，人们不止一次愉快地观看这些电影，而且往往越来越愉快。”我确实意识到悬疑和惊吓是恐怖片的重点，也许比大多数类型都要多，当然我也看到了致命转折的价值(没有双关的意思)，但有些电影比其他电影更令人讨厌。在首映周末遭遇《第六感》的大揭露是一个伟大的电影记忆；当遗传性痉挛逐渐加剧时，试图避免压力性痉挛就不那么容易了。但在随后观看的两部电影中，我发现前者并不逊色，而后者更令人愉快。在每一种情况下，知道将会发生什么让我更容易欣赏电影正在做的事情*除了*反复打我的肚子。《遗传》尤其擅长被宠坏的或第二次观看:知道了即将发生的事情，我能够充分考虑这部电影真正的内容(悲伤、不正常的家庭、普遍的创伤)，因为我没有被不好的感觉消耗掉。](https://www.universityofcalifornia.edu/news/spoiler-alert-spoilers-make-you-enjoy-stories-more)

事实上，正是在我最初不愉快地观看导演阿里·阿斯特(Ari Aster)执导的《T2遗传》的后续电影时，我意识到我可以通过暂停电影并阅读维基百科上的情节摘要，立即将我第一次紧张的观看变成第二次更放松的观看。我就这么做了，一旦我知道了事情的结果，我就能够接受剩下的事情，而不会一直想吐。也许这不是导演的本意，但对我来说效果要好得多*。不是每个人都赞同这个计划。比如，Lifehacker的主编乔丹·卡尔霍恩(Jordan Calhoun)就反对我的策略；他声称，如果你和朋友一起看电影，未被破坏的人的体验会因他们与被破坏的人的接近而受到影响——前者能够根据后者的肢体语言和反应来判断什么时候会发生或不会发生可怕的事情。(这个论点对我来说似乎是可疑的，也许是因为我大多数时候只和我的妻子一起看电影，她通常都在看手机。)但是，如果你只看你的观众群中没有人看过的电影，这很可能是一个问题*和*你们彼此非常了解，足以抓住这些微妙的线索。(如果是这样的话，嗯，对你有好处，先生或女士或Mx。我有很多朋友。)

这不是一个每次都要从盒子里拿出来的工具。我发现它对更依赖于持久张力的电影最有用(见:Aster的作品，或令人惊讶的有效变焦恶魔附身电影*主持人*)。比方说，我可能不会在《惊声尖叫5》中使用它，因为这个系列太自我意识了，不会真的很恐怖。但如果我看了，那也没什么——当然，我已经看了足够多的其他四部《惊声尖叫》电影，而且在第二次或第三次观看时没有减少回报。都是我的享受程度，你看，最后如果有人问我 [“你喜欢恐怖片吗？”T21:我希望能够用一个强有力的回答“是的”。](https://www.youtube.com/watch?v=BCGh7CM-eRI)*
