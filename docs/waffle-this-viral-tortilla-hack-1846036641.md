# 华夫饼这个病毒玉米饼黑客

> 原文:[https://life hacker . com/waffle-this-viral-tortilla-hack-1846036641](https://lifehacker.com/waffle-this-viral-tortilla-hack-1846036641)

我发誓，我离开互联网大约九天，我差点错过了一个 [病毒式烹饪黑客](https://www.tiktok.com/@ellcarter1/video/6912385832364133634?lang=en) 。(我再也不请假了，我保证。)但是尽管我可能是最后一个*看到*这个流行的玉米饼折叠视频，我想我可能是第一个把玉米饼折叠好的人。

Watch [https://lifehacker.com/embed/inset/iframe?id=tiktok-6912385832364133634&autosize=1](https://lifehacker.com/embed/inset/iframe?id=tiktok-6912385832364133634&autosize=1) 

即使没有waffling，这也是一个相当不错的小技巧。它平均分配配料，所以你一口就能尝到所有的味道。它创造了一个整洁的小包装，馅料在吃的过程中一直留在玉米粉圆饼里。我强烈推荐它作为包裹配置。

你可以用无数种方法加热它。在锅里煎，烤，或者用意式面包压榨机碾碎。作为我，我用我的华夫饼干机做了我的，而且做得很好。

我从中心到边缘切了一刀来准备玉米饼。然后，我用剩下的 [水煮鸡肉](https://skillet.lifehacker.com/maximize-a-chickens-meal-potential-by-poaching-it-first-1822353539) (用一点 [西红柿酱](https://skillet.lifehacker.com/get-yourself-some-caldo-de-tomate-immediately-1844943389) 调味)、切片蘑菇(我先用鸡蛋羹)、切碎的马苏里拉奶酪和一小把芝麻菜填满我的包，将每种配料放在各自的小象限中。(如果我再做一次，我会加一些蛋黄酱或其他某种酱，可能会加到蘑菇和鸡肉象限中。)

然后我把它包起来，首先把鸡肉折叠在蘑菇上，然后把那部分折叠在奶酪上，最后把整个东西折叠在芝麻菜上。然后我刷上黄油，用中高温的华夫饼干烤模搅拌，直到它看起来像这样。

这是一顿令人愉快的小午餐，也是清理冰箱剩菜的绝佳方式。我对这种包装方式的唯一疑虑是，它减少了暴露在直接加热下的玉米粉圆饼的数量，这反过来又减少了T2松脆的不动产的数量，但这不是最大的交易。此外，确保不要把象限装得太满，否则你的小包将不再整洁，这是这种准备方法的吸引力的一部分。

在馅料方面，没有规则和限制。像抖音视频中那样的早餐卷饼是一个显而易见的选择(只是不要煮过头了你的鸡蛋)，但不要害怕变得奇怪，用剩下的肉酱面、咖喱或炒饭来填充你的早餐。你也可以在把你的配料放入指定的象限之前把它们华夫饼干化，就像我对蘑菇做的那样。我尝试了一个SPAM/bologna/American cheese的配置(两种肉都是waffled)，但没有设法得到它的照片。不过味道很好——你必须相信我的话，T2。
