# 如何像个十足的疯子一样设置家里的 Wifi

> 原文：<https://gizmodo.com/how-to-set-up-your-home-wifi-like-a-complete-maniac-1847362678>

我的房子目前正在被掏空，我们暂时居住的地方没有互联网或有线电视，所以我已经在我的商店空间工作了大约一个月。上周，我正在完成一个编辑工作，当互联网断了的时候，这个工作耗费了我一个上午的时间。我在我们的 CMS 中编辑，最近没有保存，所以我跑到房子里，看看我在房子的旧办公室里设置的路由器是否有问题。



一个打扫房子的人不小心拔掉了路由器。我把它插回去——危机解除了。但是还有一个更大的问题:他告诉我，很快就没有路由器的安全空间了，我必须想出别的办法。我认为这将是一段时间，甚至可能足够长的时间来制定一个计划。大约一天后，我被告知时机已到。办公室就要变成非办公室了。我试着把所有东西都搬到地下室，但令人惊讶的是，把网络设备放在离我办公桌 100 英尺的地下混凝土块结构中最终没有成功。

是时候用我手头的东西解决问题了。我们正在搬家，所以我有很多可怕的易碎的消毒箱。现在，你必须知道这是怎么回事。

房子的壁板将被回收，所以我毫不犹豫地抓起几个甲板螺丝，把一个小垃圾箱固定在外墙上。我把延长线和同轴电缆穿过垃圾箱的地板，把所有东西都插到电源板上，几秒钟后，把我的手机连接到我的 Eero 路由器，不出所料，它工作正常。这是一个很好的路由器，非常容易操作。我点击手提袋上的盖子，在 Jalopnik Slack 里放了几张照片来展示我的作品。Raph 说我应该把照片分享给 Gizmodo 的工作人员，于是我就这样做了。

Andrew Couts 很有帮助地建议我在手提袋的底部增加一些散热孔，因为调制解调器和路由器会变热。这看起来很聪明，我已经意识到我需要一种密封盖子的方法来防雨，所以我去五金店买了一些东西。

我有:

*   几种不同的胶水
*   大猩猩胶带
*   一些弹性橡皮筋
*   几个拧入式挂钩/孔眼
*   一些网筛
*   小风扇
*   一个用来连接落水管和檐槽的小塑料适配器，碰巧与风扇的直径大致相同

我的计划是回去工作，然后在下班后组装这个装置的第二个版本。不幸的是，当我回到我的发帖站时，我的互联网几乎完全瘫痪了——一项速度测试显示，我的网速远远低于 1 Mbps，即使对 Jalopnik 来说也太慢了。 于是，我把整件事拆开了。

我没有放弃显然不是解决这个问题的好方法，也没有花一分钟去思考这是否是我真正需要做的，而是开始构建一个更复杂的外壳版本。我已经买了所有的东西。

我不知道速度下降是否是由过热引起的，或者这甚至是会发生的事情，但盒子里很热，安德鲁提到过通风，所以我决定增加一个风扇。我在手提袋上剪了一个很大的方形孔，用来放水槽适配器，然后用胶带粘住，让它面向地面。然后，我用胶带将风扇固定在盒子内部，这样它就可以将空气向下吹出。我在组件的出料端贴了一些网以防止虫子进入。

在盒子的另一端，我钻了一些 1 英寸的洞，用网覆盖，并做了一个小塑料罩来防止雨水进入。我用双面胶将调制解调器和路由器固定在风扇和进气孔之间的盒子地板上，并将手提包拧到房子外面。一旦我插上所有的东西，盖上盖子，我就能感觉到冷空气流入，暖空气流出。最后，我在盒子的边缘加了一些挡风雨条，进行了一次速度测试，，等待着迎面而来的暴雨。

雨停后，我检查了一下自己的表现。我在盒子的下边缘发现了大概一茶匙的水，远离电子设备所在的凸起区域。我在积水的地方钻了一个小排水孔，并添加了几条弹性绳来绑住盖子，压缩挡风雨条以获得更好的密封。

我已经使用这个设置大约两周了。我们遇到过一些雨，包括一些较大的雷暴，盒子一直保持干燥。风扇还在运转，里面没有虫子，我的设备也不会受到屋子里挥舞大锤的家伙的攻击。速度在 120-130 Mbps 范围内，超过了可接受的速度。

我建造了一些我非常自豪的东西，也建造了一些丑陋的、暂时的、功能性的东西。这种糟糕的 wifi 情况显然属于后一种情况，尽管我有点反常地为它的简易性感到自豪。这有点像我和一些朋友在那种意义上建造的 [24 小时柠檬车](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=&cad=rja&uact=8&ved=2ahUKEwjEtdmosfrxAhWDZM0KHRSODggQFjAAegQIBBAD&url=https%3A%2F%2Fwheels.blogs.nytimes.com%2F2010%2F12%2F03%2Ffrom-jalopnik-joke-to-24-hours-of-lemons%2F&usg=AOvVaw1_sv1c3Iey0EXLU5PPrSm4)——显然很糟糕，考虑不周，但很有趣，而且出奇地可靠。

几个月后，壁板就会开始倒塌。希望到那时，我们将有一个完成的房间，我可以正确地设置这些东西。如果没有，我相信我能想出更糟糕的东西。
