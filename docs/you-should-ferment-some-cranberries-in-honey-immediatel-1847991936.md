# 你应该立即在蜂蜜中发酵一些蔓越莓

> 原文：<https://lifehacker.com/you-should-ferment-some-cranberries-in-honey-immediatel-1847991936>

我曾经住在沼泽里。

某个秋天，科德角的沼泽充满了水。巨大的轮式机器搅动着葡萄藤，蔓越莓漂浮在顶端。令人惊叹是不足以形容这种颜色的词。

我对蔓越莓的欣赏永无止境。我很喜欢不加糖的蔓越莓汁，那种会让你满脸皱纹的果汁。我在夏天沏越橘冰茶，还做越橘食品。因为所有这些，我怀疑我有赛马般的泌尿健康。(实际上我不知道赛马是否有健康的泌尿道，但它听起来像是很好的柔韧的 T2。

诚然，你可以用蔓越莓做的事情有限。可以榨汁。你可以卡住它。但是我很少看到真正有创造性的应用—y 你不会在任何菜单上找到油炸蔓越莓，也不会在苹果中找到它们作为火锅的一部分。然而我总是在寻找新的和有创意的方式来庆祝感恩节餐桌上最小的英雄。

好消息，我发现了一个:在蜂蜜中发酵蔓越莓。

为什么要发酵它们？发酵食品对你有好处，发酵蔓越莓增加了这种被低估的水果对健康的益处。蜂蜜还会使果仁变甜，给它一种像糖果一样的感觉——把它们放进嘴里是非常令人愉快的。蜂蜜可以单独使用，但我更喜欢吃蜂蜜中的发酵蔓越莓酱，而不是我体验过的任何蔓越莓酱。

现在，如果你同意我的观点，你需要今天去农贸市场或商店*，把这些吸管倒进罐子里。这个食谱需要的一件事是时间，而你离感恩节只有几周时间了(离圣诞节还有八周)。* 

这个过程是非常简单的。这就像清洗你的小红莓一样简单，把它们塞进罐子里——压紧但不要压碎——然后在里面装满蜂蜜。盖上，每天左右翻一翻，隔几天打个嗝。它很快就会开始冒泡。

生蜂蜜是你在这里的最佳选择，特别是流动的生蜂蜜。如果你的不是流质的，不要用微波炉加热——你会杀死发酵所需的细菌。在炉子上慢慢加热，*只要*足够让它运转，然后把它和小红莓一起倒入罐子里。你绝对*必须*用蜂蜜覆盖*所有的*小红莓。只要它们在蜂蜜里，你就是金色的。你可以用一个重物来保持它们浸没在水中。我喜欢 [这些。](https://www.amazon.com/Ball-Fermentation-Lids-Springs-Metal/dp/B084CZ7FHM/ref=sr_1_6?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/you-should-ferment-some-cranberries-in-honey-immediatel-1847991936&asc_source=&dchild=1&gclid=Cj0KCQjw5oiMBhDtARIsAJi0qk1JImvRKeggm6UPTkIpYvuffE9V2n8N9lft-Goo7SivRh_UCFYpBjAaAsjCEALw_wcB&hvadid=445096834988&hvdev=c&hvlocphy=9061081&hvnetw=g&hvqmt=e&hvrand=6676911614464675845&hvtargid=kwd-919913817270&hydadcr=20228_11121809&keywords=ball jar fermentation kit&qid=1635968462&sr=8-6&tag=kinjalifehackerlink-20)

你应该只使用形状好的浆果——它们不能太软，也不能有任何坏点。坚实、成熟的浆果是你的目标。可以加入柑橘皮、肉桂棒、丁香、生姜等。，但我保持它简单:蔓越莓和当地蜂蜜。

我通常会让这种酵素发酵至少两个星期，然后才费心去品尝它；这是一种随着年龄增长而改善的享受，这就是为什么从今天开始很重要。冒泡是一个好现象，但是即使没有冒泡，在 3 到 4 周之后，蜂蜜将会渗透到蔓越莓中。到了感恩节，它会变得美味可口，到了圣诞节，它会成为节目的主角。

如果你是一个容易被发酵食品的风险吓到的人，买一些 pH 试纸，并在品尝前确保 pH 值在 3.5 到 4.6 之间。

将蜂蜜加入节日鸡尾酒中，或者淋在薄煎饼、蛋糕或果馅饼上。你也可以 将发酵后的 crans 脱水，成为更有趣的小吃。这种蜂蜜在茶中也是不可思议的，在柠檬水中是非凡的，在热棕榈酒中是卓越的。(现在是炎热的棕榈季节。)
