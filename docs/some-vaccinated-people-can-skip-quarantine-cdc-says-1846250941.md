# 疾控中心说，一些接种疫苗的人可以跳过隔离

> 原文:[https://life hacker . com/some-vaccined-people-can-skip-quarantine-CDC-says-1846250941](https://lifehacker.com/some-vaccinated-people-can-skip-quarantine-cdc-says-1846250941)

根据来自T4疾病预防控制中心的最新指导意见，如果你完成了两剂新冠肺炎疫苗，你可能不必在接触测试呈阳性的人后进行隔离。这是一个政策上的改变，而且是基于一些有希望的新数据。

隔离的目的是防止某人在等待观察是否出现症状时传播冠状病毒。(一旦你知道你生病了，你是在隔离，而不是隔离。)在隔离期间，你现在应该知道，你需要呆在家里，采取预防措施保护他人。

这是来自疾病预防控制中心的新建议:

> **但是，与疑似或确诊新冠肺炎患者有接触的已接种疫苗人员，如果符合以下所有标准**<sup>**:**，则无需 [**隔离**](https://www.cdc.gov/coronavirus/2019-ncov/if-you-are-sick/quarantine.html)</sup>
> 
> *   **完全接种疫苗(即在接受2剂系列疫苗的第2剂后≥2周，或在接受一剂单剂疫苗后≥2周)**
> *   **在接受系列中最后一剂后的3个月内**
> *   **自目前接触新冠肺炎病毒后一直无症状**
> 
> **不符合上述所有3项标准的人员在与疑似或确诊新冠肺炎患者接触后，应继续遵循当前的检疫指南。**

**注意3个月的时间框架。疫苗的保护作用可能会持续三个月以上，但目前我们还不知道。如果你是第一批在12月接种疫苗的人，那么从3月左右开始，你可能需要在暴露后再次开始隔离。**

**也就是说，如果我们得到的数据显示保护期比这更长，三个月的期限可能会延长。**

### **为什么这样可以？**

**如果你感到困惑，因为你认为疫苗仍然允许人们传播疾病，让我们花一分钟来澄清。**

**虽然我们几个月前就知道COVID疫苗(辉瑞和Moderna)可以预防严重疾病，但临床试验*没有测试*接种疫苗的人是否能够传播疾病。这就产生了疫苗不能预防感染的神话，但事实是我们不知道它是否能预防，或者能预防多少。医师梅根·兰尼 [在Twitter帖子](https://twitter.com/meganranney/status/1359671364291035141) 中指出，我们实际上有几项研究暗示疫苗可能会停止或大大减少传播。**

**疾控中心说:**

> **虽然新型冠状病毒病毒从接种疫苗的人传播给他人的风险仍不确定，但接种疫苗已被证明可以预防有症状的新冠肺炎；有症状和症状前的传播被认为在传播中比纯粹的无症状传播具有更大的作用。此外，避免不必要的隔离对个人和社会的好处可能超过潜在的但未知的传播风险，并有助于将公共卫生资源用于将新型冠状病毒病毒传播给他人的高风险人群。这一免除疫苗免疫人群检疫的建议与 [自然免疫人群](https://www.cdc.gov/coronavirus/2019-ncov/hcp/duration-isolation.html) 的检疫建议一致，便于实施。**

**这有点像赌博，但每一个高风险的决策都是如此。要求接种疫苗的人仍然隔离也有其不利的一面:比如错过工作。**

**如果你接种了疫苗，并能够跳过隔离，CDC建议采取一些措施以防万一:你仍然应该确保在他人周围戴上口罩，遵守所有其他安全协议(如保持距离)，并确保注意自己的症状。疫苗并不是100%有效，所以你还是有很小的可能会生病。**

**随着我们对疫苗和新型冠状病毒变异体的了解越来越多，预计政策会有所发展。以澳洲为例， [仍要求接种疫苗的人入境前对](https://www.travelandleisure.com/travel-news/australia-covid-vaccinated-international-travelers-quarantine-requirement) 进行检疫。他们看到了和疾控中心一样的证据，但做出了不同的判断。我们都还在学习一些东西关于这些抗病毒疫苗的强度，所以现在要做的事情是遵守任何适用于你的规则，并等待政策如何变化。**
