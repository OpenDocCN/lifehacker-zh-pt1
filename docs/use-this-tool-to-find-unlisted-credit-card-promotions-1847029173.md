# 使用此工具查找未列出的信用卡促销活动

> 原文：<https://lifehacker.com/use-this-tool-to-find-unlisted-credit-card-promotions-1847029173>

如果你在市场上寻找一张新信用卡，但不想浪费时间在信用卡公司网站上寻找预先批准的交易，可以考虑查看一下[card match](https://www.creditcards.com/cardmatch/)——这是一个可以比公开获得的注册奖金更高的工具。虽然这个工具不是一个全面的信用卡一站式商店，但它是一个开始购物的好地方。它是这样工作的。

## 我的牌局是什么？

CardMatch 是一个在线工具，它根据你的信用评分为你匹配信用卡优惠(你可以在这里找到)。如果你的信用评分更高，你可能会看到签约奖励(奖励积分或航空里程)比通常在公共渠道获得的更高。

例如，通过 CardMatch，我提前获得了 AmEx daily 信用卡的批准，注册奖励为 15，000 奖励积分。如果没有这个工具，网络搜索只能得到 10，000 点奖励积分，相差 5，000 点——根据美国运通的积分计算器，这相当于 30-50 美元。

## 使用 CardMatch 有什么好处？

CardMatch 的代价是你必须提供个人信息，其中一些可能会被用于营销垃圾邮件和短信。您必须提交电子邮件、地址、电话号码、工资和您的社会安全号码的最后四位数字，因为该工具将运行一个“”信用检查，以便生成报价(与硬查询不同，软查询不会影响您的信用评分)。当然，你可以选择不接收那些营销短信和邮件，但它们仍然很烦人。

在安全性方面，该工具由 Bankrate 运行，Bankrate 是一家消费金融服务公司，它使用银行级别的安全性来确保您的个人信息安全，根据 point Guy 的说法是 [。同样，如果你不想冒垃圾邮件的风险或者不想分享那么多个人信息，那也没问题。但是如果你已经货比三家了呢？和直接在信用卡网站上申请预审批没那么大区别。](https://thepointsguy.com/guide/using-the-cardmatch-tool/)

另一个考虑是该工具不全面——由于该工具是基于与信用卡公司的合作关系，你不会看到市场上的每一张卡。使用 CardMatch 是开始购物的一个好方法，但是您可能还想在工具之外做一些研究。

## **预先核准不等于*实际*核准**

CardMatch 只是根据对你信用的软查询，告诉你你有可能符合什么卡的条件。如果你真的申请了信用卡，贷方会对你的信用运行一个 [硬查询](https://www.investopedia.com/terms/h/hard-inquiry.asp) ，这是对你信用历史的更深一层的探究，可以暂时降低你的信用评分几分。在这种情况下，该工具可能会推荐一张您实际上不符合条件的卡。也就是说，除非你的信用记录中有任何隐藏的意外，否则你应该有资格申请一张预先批准的信用卡。
