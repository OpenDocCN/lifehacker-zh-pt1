# 使用这些网站进行实时 COVID 旅行限制

> 原文：<https://lifehacker.com/use-these-sites-for-real-time-covid-travel-restrictions-1845168227>

让我们首先承认，如果你现在不必旅行，你绝对应该呆在家里。但是如果你一定要去旅行，了解你计划的目的地新冠肺炎的最新情况是很重要的。

Watch

这可能会令人困惑，因为疫情的情况和相关的旅行限制似乎每天都在变化——但有一些工具可以显示国内和国际旅行的风险、限制和要求的实时信息。这里有几个选择。

### 屠龙者

《屠龙记》是一款的订阅服务，主要面向旅行社，但 [也面向普通大众](https://thepointsguy.com/news/this-new-tool-shows-you-real-time-covid-19-requirements-for-international-and-domestic-travel) 。它扫描所有 50 个州和 124 个国家的新冠肺炎限制的实际实时更新，并根据个人风险承受能力调查给每个位置以颜色编码的分数。

你也可以获得官方的政府指导和关于企业如何处理检疫和限制的信息。

这项服务有 14 天的免费试用期，之后每月要花费 9.99 美元。但是如果你是在最后一刻出发，这可能是值得的。Android 用户可以在 Google Play 商店 中找到一个 [独立 app，而 iOS 用户目前被限制为](https://play.google.com/store/apps/details?id=com.dragonslayertravel.ds&hl=en_US) [一个浏览器版本](https://safe-t.dragonslayertravel.com/auth/signup) 。

### 旅游动态数据库

[TravelVitals 是一个可搜索的网络数据库](https://travelvitals.amexgbt.com) 来自美国运通全球商务旅行。您可以通过特定的旅行因素来缩小范围，包括位置、航空公司、火车站或酒店名称。你会得到一长串相关信息，从准入限制到屏蔽要求，再到一家公司的奖励计划。

搜索结果还会显示最后一次验证的日期，所以你会知道你正在阅读的信息是最新的。我们这边的一些快速搜索显示当天验证日期。

### 国际航空运输协会互动地图

国际航空运输协会维护着一张彩色编码的全球地图 ，上面有各国对新冠肺炎入境的限制。界面非常简单:只需点击国家并阅读弹出窗口，它还会显示该国家信息发布的最后日期。

国际航空运输协会指出，该地图经常更新，但实际上可能不是实时的。

### 联合航空公司和 Wanderu 互动地图

[美联航新冠肺炎地图](https://www.united.com/ual/en/us/fly/travel/restrictions-map.html) 是国际航空运输协会全球工具的国内版本，也可作为移动应用程序使用。它 [向你展示一个地方是开放还是关闭](https://lifehacker.com/stay-up-to-date-on-covid-travel-restrictions-with-this-1845011555) 用于休闲旅游以及任何测试和检疫要求。还可以看商家(餐馆等。)是开放的，以及任何相关的限制。当您单击一个特定的州时，您会看到信息最后被查看的日期—大多数都是在最近几天内。

我们还介绍了 [Wanderu](https://lifehacker.com/use-this-map-to-find-out-about-bus-and-train-travel-res-1845114125) ，它有一个交互式的各州地图，对公共汽车和火车的限制特别有帮助。虽然不完全是实时的，但这是一个起点。
