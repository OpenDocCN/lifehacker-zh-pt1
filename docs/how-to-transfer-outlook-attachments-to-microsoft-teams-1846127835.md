# 如何将 Outlook 附件传送给 Microsoft 团队

> 原文：<https://lifehacker.com/how-to-transfer-outlook-attachments-to-microsoft-teams-1846127835>

最后，您可以将 Outlook 中的电子邮件附件拖放到 Microsoft 团队中。当然，这个功能花了将近五年才出现，但它已经出现了。



要将电子邮件附件从 Outlook 移动到团队，首先在 Outlook 中打开一封电子邮件(当然是带附件的)。单击附件突出显示该文件，然后将其拖到团队的对话中进行共享。就是这样。

对于不使用 Outlook 或团队的人来说，这似乎不是一个重大变化，但对于用户来说，这是一个很大的烦恼。在新功能出现之前，您必须保存电子邮件附件，在桌面上打开它的位置，然后单击并将其拖到团队中，或者通过团队的文件浏览器打开它。

团队和 Outlook 之间新的拖放文件共享只是最近出现在微软应用程序中的几个新的跨应用程序功能之一。

该公司目前正在测试新版本的 Outlook，该版本将简化应用程序在所有桌面和网络平台上的外观，包括默认的 Windows 10 邮件应用程序。这种简化的一部分包括更容易找到的快捷方式以及与其他微软应用程序和服务的集成，特别是 Office 365 套件。随着微软推出新的 Outlook，更多的功能可能会出现。

新的 Outlook 仍在测试中，直到 2021 年才会全面实施，但你可以现在就 [试用该应用的早期版本](https://lifehacker.com/test-drive-microsoft-outlooks-future-look-1846021518) 。

Edge Chromium 中也有新的 Outlook 集成。该浏览器最近的 88 版本更新在 Edge 的新标签页中增加了一个部分，可以预览来自 Outlook 收件箱的最新电子邮件。Edge 88 还为您的微软帐户提供了新的密码安全工具，侧栏搜索，可视化升级，使其与微软的其他应用程序更加一致，等等。你可以在这里 [了解](https://lifehacker.com/the-best-new-features-in-edge-chromium-88-1846113006) 的所有变化。
