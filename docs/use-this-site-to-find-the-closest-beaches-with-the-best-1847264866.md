# 使用此网站寻找最近的海滩，那里有最好的设施

> 原文：<https://lifehacker.com/use-this-site-to-find-the-closest-beaches-with-the-best-1847264866>

旅行的最大乐趣之一就是发现新的地方。当然，在猫途鹰“要做的事情”部分的顶部，你会发现一些众所周知的景点，但是那里还有更多——如果你知道去哪里看的话——海滩也不例外。



虽然在大多数沿海地区，找到水并不困难，但找到最好的海滩并不总是容易的。也许有一个挤满了游客，小贩以 5 美元的价格出售小瓶水。也许几分钟后还有另一家，那里有充足的停车位，人少得多，还有干净的浴室。很明显，那是你想要的海滩。

幸运的是，有一个网站可以帮助你找到完美的海滩，无论你去世界的哪个地方。以下是需要了解的内容。

## 如何找到你附近最好的海滩

无论你是在家附近寻找新的海滩，还是想探索你第一次去的某个地方，一个名为[Beach nearly](https://beachnearby.com/)的网站都能帮上忙。使用众包数据，Beach Nearby 允许用户通过特定海滩或位置的名称进行搜索，或者使用交互式地图找到海滩。

网站上的每个海滩都有一个个人资料页面，里面有各种信息和照片(取决于人们的贡献)。除了提供满分为 100 分的海滩评分，这些档案还包括一般的提示和数据，如海滩是岩石还是沙滩，是公开还是私人的，入场费(如果有的话)，以及最近一次评级的日期。

页面的其余部分向用户介绍停车和/或公共交通情况、安全功能(如救生员值班时)以及该地区的天气信息。还可以选择根据这些(和其他)特征搜索海滩。
