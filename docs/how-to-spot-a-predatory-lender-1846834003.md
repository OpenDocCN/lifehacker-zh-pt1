# 如何识别掠夺性放贷者

> 原文：<https://lifehacker.com/how-to-spot-a-predatory-lender-1846834003>

大多数人都熟悉掠夺性贷款的概念，这是一种将不公平或欺骗性条款强加给借款人的贷款做法，通常采取发薪日贷款或利率异常高的产权贷款的形式。但是掠夺性贷款和高利率的合法贷款有什么区别呢？以下是你如何发现不同之处。

Watch

### 这个提议好得令人难以置信

如果一个潜在的贷款人对检查你的信用评分毫无兴趣，快速批准一笔远高于其他地方的贷款额，或者一般来说承诺慷慨的条款，很少问问题，那么要警惕。如果贷方似乎对你是否真的能还清贷款不感兴趣，这意味着他们可能不希望*你还清贷款——他们想尽可能长时间地向你收取利息(可能高于平均水平)。*

### 三位数利率

掠夺性贷款者的一个明确标志是，他们将提供三位数的短期贷款利率，尽管信用评分最差的人有的替代选择，可以确保他们以接近 30%的年利率获得现金(这仍然很高)。掠夺性贷款人也不太愿意告诉你贷款的总成本和利息，通常会把它藏在小字的某个地方，或者在你签署文件后再掩盖。

### **虚增的费用和罚款**

合法贷款提供了一条清晰的还款之路，毫无意外。另一方面，掠夺性贷款人想让你永远上钩，他们这样做的一种方式是收取额外的滞纳金，滞纳金是贷款总额的一个百分比，而不是固定费用。即使你是一个负责任的借款人，并且提前还清了贷款，你也可能会面临提前还款罚金，罚金也可以按你贷款的百分比来计算。过多的 [垃圾费](https://www.investopedia.com/articles/mortgages-real-estate/08/mortgage-closing-costs-fees.asp) 也很常见。

### **贷方不会向征信机构报告你的支付情况**

适当的贷款人依靠信用评分来决定是否给你贷款，这就是为什么他们向计算你信用评分的三大信用机构报告你的付款(或未付款)情况——Experian、Equifax 和 TransUnion。自然，如果不守信用的贷款人更想让你继续负债——并保持低调——他们就不太可能这么做。在你申请贷款之前，确保任何潜在的贷款人都会报告你的付款情况。

### **在**之前，你从未听说过贷款人

除非是你已经信任的大银行或信用合作社，否则对你潜在的贷款人做一些调查。一个简单的谷歌搜索就会发现其他消费者的任何投诉，检查消费者金融保护局的投诉数据库 也是如此。

### 贷方想直接进入你的银行账户

虽然许多贷款人会要求进入你的银行账户，以处理自动付款，提供这些信息不应该是一项要求。 [根据信用因果](https://www.creditkarma.com/personal-loans/i/what-is-predatory-lending) ，一个掠夺性的贷方可能会把你的账户当成自动提款机，如果你的账户资金短缺，甚至在你收取银行透支费的时候，还会重复要求付款。另外，请注意，您应该能够随时取消自动付款。
