# 最简单的方法来跟踪你最喜欢的即时锅煮时间

> 原文：<https://lifehacker.com/the-easiest-way-to-keep-track-of-your-favorite-instant-1847637673>

如果你有一个速溶壶——或者我继母喜欢称之为“一锅”——你可能已经注意到互联网上贴满了速溶壶的图片和备忘单。这些时间和水比例的表格以层压的表格、磁铁和简单的打印输出的形式出现，但是它们经常相互冲突，大多数厨师都有他们的最爱。



我没有这些小抄中的任何一个，但我有几个最喜欢的即时锅食谱，我总是忘记，不得不重新搜索。把它们写下来是一个简单的方法，但是我非常擅长丢失纸片(不管有没有层压！)，所以我决定用干擦记号笔把这些比例和时间直接写在一个锅上。(我知道这在技术上不正确，但我认为我继母给这种电器起的名字实际上更有意义。)

我只是刚刚开始做这件事——我在森林里静静地坐着，等待从来没有来过的麋鹿时想到的——到目前为止，我已经把我喜欢的鸡蛋和米饭的时间和比例——我在速溶锅里煮得最多的两种东西——直接写在了设备上。如果我能记住它们，我会擦掉记号笔，为其他我还没记住的菜写下新的时间和比率。(尽管更有可能的是，我会继续添加时间和比率列表，直到闪亮的金属看起来像纹身大会上的水手。我从来不擅长记数字。)

如果你想用我的简写，请随意。“HP”和“LP”代表“高压”和“低压”，而“NR”和“MR”代表“自然释放”和“手动释放”。我也为“水”写“H₂O”得利用一下化学学位。
