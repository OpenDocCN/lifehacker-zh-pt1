# 暗网和深网的区别

> 原文：<https://lifehacker.com/the-difference-between-the-dark-web-and-the-deep-web-1848654878>

每当你听到黑暗网络的时候，你可能会想到毒品、死亡和堕落。或者说，是*深*网？两者都是吗？你可能会发现自己将这两个术语混为一谈，假设暗网和深网都是非法和一般不正当活动的相同场所。虽然暗网是深网的一部分，但深网并不是真正的暗网。



## 黑暗之网到底是什么？

让我们从你可能会想到的“黑暗网络”和“深度网络”这两个词开始说起。你可能会想象一个地下的、秘密的网站网络，在那里非法活动猖獗，肆无忌惮地传递毒品、违禁品和非法媒体。这些网站肯定存在，它们是所谓“黑暗网络”的一部分。

然而，黑暗网络并不仅仅是邪恶者的游乐场。虽然这些网站吸引了所有的注意力，但黑暗网络实际上并不是由其网站的内容来定义的(即使我们大多数人是这样定义的)。相反，暗网只是一个无法通过传统方法访问的私有网络的集合。

## 什么是“表面网”？

我们大多数人所知道的互联网被称为“表面网络”本质上，它是由搜索引擎索引的网站的集合。如果它出现在谷歌搜索中，它就是 surface web 的一部分。暗网就不是这样了。你不能打开 Chrome，输入“毒品，请”就指望找到这些网站。事实上，你根本不能使用 Chrome，也不能使用任何传统的网络浏览器。

## 如何进入黑暗网络

如果你想进入黑暗网络，你需要特殊的工具。(你可以在这里 找到更多关于访问暗网 [的信息。)需要一个特定的浏览器，比如 Tor，才能入门。就像你使用 Chrome 访问脸书或 Lifehacker 等公共网站一样，你也可以使用这些浏览器之一访问黑暗网页。](https://lifehacker.com/how-to-browse-the-dark-web-if-youre-shady-or-just-curi-1848643880)

然而，使黑暗网络独一无二的不仅仅是浏览器:因为你需要特殊的协议来访问这些网站，这种交易通常是私人的和匿名的。这就是为什么黑暗网络对非法活动有吸引力——网站活动不会追溯到个人用户账户。像比特币一样，加密是黑暗网络的货币，因为它也在交易中保护你的隐私。

也就是说，这并不全是坏事。虽然黑暗网络最常见的使用案例可能是违法的，但任何有理由匿名的人都可以利用这些网络。黑暗网络上“好”的常见例子是告密者，他们需要一个地方来泄露他们的信息，而不让负责这些数据的政府和组织知道他们是谁。

## 那么到底什么是“深度网络”呢？

暗网仅仅是深网的一个分支(一个很小的分支)。deep web 也被称为“隐藏的”web，是没有被搜索引擎索引的站点的集合。虽然这个定义包括了黑暗网页，但也包括了完全无害的网站。这些网站大多隐藏在登录页面之后，范围从银行和电子邮件到流媒体等付费内容。当然，你可以通过谷歌搜索找到网飞，但你不会找到《T2 最好打电话给索尔》第一季第五集的球员，除非你先登录这个网站。

它还包括协议页面，负责在您登录网站时识别用户帐户，在您购物时进行支付，以及其他您永远不需要看到的网站。简而言之，它既是互联网的主干，也是你经常看到的互联网的一部分。深层网页没有索引，但它们通常有可以直接链接的 URL，可以从 Chrome 或 Firefox 等传统浏览器访问。

简而言之，深网不是一个可怕的地方。暗网*可以是*，但不一定。
