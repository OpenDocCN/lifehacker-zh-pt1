# 如何减缓你的汽车贬值，这样你就可以卖更多的钱

从你把新车开出停车场的那一刻起，它们就失去了相对于购买价格的价值。虽然确切的数量因年份而异，但品牌、和型号、、、[凯利蓝皮书估计](https://www.kbb.com/car-advice/how-to-beat-car-depreciation/)、、、的汽车在第一年将损失20%或更多的价值，在前五年将损失60%左右。

没有办法完全避免贬值。事实上，这种情况会持续到汽车基本上变得毫无价值，尽管速度会在你达到100，000英里时减慢。然而，如果你打算在几年内转卖你的车，或者只是想最大化它的价值，你可以做一些事情来最小化折旧。

### 作为买家:选择转售价值高的车

有些汽车比其他汽车更能保值。例如，丰田 s往往比其他品牌的T4贬值更快。(有趣的是，我们已经看到行驶了60，000英里的二手Tacomas以接近一辆新卡车的价格售出。如果你打算在你的车报废之前就把它转卖，你可能会选择一个以高转卖价值而闻名的品牌和型号。

另一个选择:买二手的。再说一次，一些车辆保持其价值，所以你可能不会节省太多的前期取决于你在寻找什么。但是如果你买了一辆车龄五年或者已经跑了很多英里的车，大部分的折旧已经发生了。只要确保车辆得到良好的保养(见我们的下一点)。

### 作为车主:保养你的车，不要过度定制

这个很明显。爱护你的车，当你卖掉它时，它更有可能值钱。这包括执行定期维护和任何必要的修复，以及保持清洁。记录你所做的一切，这样你就能证明你是一个负责任的主人。最后，每年驾驶 [低于平均里程数](https://www.caranddriver.com/research/a32880477/average-mileage-per-year/) 可能有助于你的车保持相对于同一车型年的同类车的价值。

虽然你可能喜欢一辆装饰华丽的车，但定制你的车会让它更难卖出去。这包括车窗着色、添加字样、以及任何其他华而不实的售后升级。

### 作为卖家:自己卖

把你的车折价或者卖给经销商会减少你的利润。如果你能私下处理销售，你也许能更接近真实的市场价值。当你卖掉你的汽车时，你得到的现金越多，你的折旧就越少。

### 奖金:从你的税收中扣除折旧

如果你把你的车用于你的生意或副业，你也许可以在你的纳税申报表 中 [扣除的折旧。](https://www.irs.gov/taxtopics/tc510)
