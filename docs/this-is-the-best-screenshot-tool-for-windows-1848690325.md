# 这是 Windows 最好的截图工具

> 原文：<https://lifehacker.com/this-is-the-best-screenshot-tool-for-windows-1848690325>

Windows 提供了多种截图方式，但如果你经常截图，内置工具可能就不够用了——比如滚动截图、固定时间间隔和云上传。

Watch

虽然有一堆可供选择的截图工具，但对于 Windows 来说，[【ShareX】](https://getsharex.com/)是最好的。ShareX 完全免费且高度可定制，所以你可以设置工作流程，比如上传截图到你自己的服务器(或任何云存储服务)并自动复制链接。然而，ShareX 的界面可能会令人生畏，所以下面是设置它的最佳方法。

## 如何在 ShareX 中设置热键

使用 ShareX 的最佳方式是设置键盘快捷键。安装应用程序时会提示您这样做，但如果您想添加更多快捷方式(或更改它们)，您可以导航到**热键设置...**在左窗格中。

该窗口将在左侧显示各种屏幕截图操作，如捕获整个屏幕、捕获区域等。键盘快捷键显示在右侧的同一行中。

最重要的是右手边方框的颜色。如果它是红色的，那么您必须更改键盘快捷键，因为它已经被分配给 Windows 中的一个不同的任务。如果是绿色的，你就可以走了。

要添加新的操作，单击左上角的 **Add** 按钮，并从右侧的第一个下拉菜单中选择任务。例如，我选择了**屏幕捕捉>捕捉最后一个区域**，并为它设置了一个自定义的键盘快捷键。下一次我在电脑上截图自定义区域时，我可以使用**捕捉最后一个区域**动作再次截图相同的区域。

## 如何在 ShareX 中设置截图文件夹

接下来你要做的是为你所有的 ShareX 截图建立一个文件夹，让你的截图有条理，而不是把它们都存储在默认的 ShareX 文件夹里。

为此，导航至**应用程序设置...>路径**。在这里，您可以选择**使用自定义截图文件夹**并点击**浏览...**选择您喜欢的任何文件夹。

您还应该返回到 ShareX 的主窗口，并在捕获任务>将图像保存到文件后转到**，以确保每个截图都被发送到正确的文件夹。如果您想将所有截图复制到剪贴板，您可以在捕获任务**后选择**下的**复制图像到剪贴板**。**

请记住，在捕获任务之后，您可以在**下设置多个操作，并且您应该检查所有选项以创建适合您需求的工作流。**

## 如何在 ShareX 中将图片上传到云端

ShareX 支持 Google Drive 和 OneDrive 等常见服务，以及亚马逊 S3 和谷歌云等更高级的选项。您可以前往 **ShareX >目的地>目的地设置进行设置...**。选择您需要的服务，然后将其连接到您的在线帐户。

完成后，请务必访问 ShareX 中的热键设置，以设置一个键盘快捷键，直接将图像上传到您首选的云存储服务。完成后，你可以在上传任务后导航到 **ShareX >并选择**将 URL 复制到剪贴板**以立即获得一个可共享的链接。**

## 如何在 ShareX 中编辑和注释截图

ShareX 还允许你编辑截图和添加注释。最好的方法是在图像编辑器中打开捕获任务>后进入**，这将立即在内置图像编辑器中打开捕获的截图。您可以将鼠标悬停在任何工具上来查找其名称，但最重要的是:**

*   **步骤**:这个工具有一个看起来像 001 的图标。它会在你的截图上添加编号的圆圈。如果你试图在一个图像中显示多个步骤，选择步骤工具并按正确的顺序点击，使圆圈标签 ed 1、2、3 等出现在图像上。
*   **箭头**:让你在图像上画箭头，指出截图中的重要区域。
*   **形状工具**:矩形和椭圆形工具让你在截图上画出形状。您可以使用右边的填充颜色工具用纯色填充这些形状。如果你想要 [隐藏截图](https://lifehacker.com/stop-blurring-your-sensitive-information-in-screenshots-1848626457) 中的敏感信息，这是一个比模糊工具更好的选择。
*   **光标**:将鼠标光标放在 屏幕上。

## 其他值得了解的 ShareX 特性

除了截图，ShareX 还可以通过进入**工作流**选项卡，然后进入**开始/停止屏幕录制**选项来捕捉屏幕录制。还有一个选项可以让你将屏幕录像保存为 GIF 格式。

如果你想在 ShareX 中对截图做更多的事情，点击左窗格中的**捕获**菜单。你会发现很多有用的选项，如自动截图，添加截图定时器，滚动截图，截图一个或多个显示器，等等。
