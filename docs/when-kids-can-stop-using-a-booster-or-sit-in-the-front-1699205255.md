# 当孩子们可以停止使用助推器或者坐在汽车的前座时

> 原文:[https://life hacker . com/when-kids-can-stop-use-a-booster-or-sit-in-the-front-1699205255](https://lifehacker.com/when-kids-can-stop-using-a-booster-or-sit-in-the-front-1699205255)

最终摆脱增高座椅是许多孩子(和他们的父母)期待的一个里程碑。但是匆忙的过渡可能会让孩子在车祸中面临更大的受伤风险——这可能是违法的。

各州法律规定何时儿童停止乘坐汽车安全座椅或助推器是合法的，这些法律因年龄、身高和/或体重而异。(这里有 [一个很好的列表](https://www.verywellfamily.com/us-car-seat-laws-by-state-4082814) 如果你想查你所在州的汽车座椅法。)

然而，即使这些法律也可能不是最安全的指导方针。

[美国儿科学会](http://www.healthychildren.org/english/safety-prevention/on-the-go/pages/car-safety-seats-information-for-families.aspx) 建议，体重或身高超过汽车安全座椅前向限制的儿童应使用安全带定位辅助座椅，直到车辆安全带合适为止。这通常发生在8到12岁之间，当他们长到4英尺9英寸高的时候。

你怎么知道安全带什么时候合适？使用SafeKids.org的 [五步安全带测试](https://saferide4kids.com/blog/take-the-5-step-seat-belt-test/) :

1.  安全带的肩部是否位于胸部中间、肩部中间？
2.  这孩子能一直坐到椅背上吗？
3.  孩子的膝盖在座位边缘弯曲，脚放在地板上吗？
4.  安全带的膝部是否穿过儿童的大腿上部和臀部？
5.  孩子能在整个旅途中保持这个姿势吗？

他们已经离开了助推器，并请求坐在前面吗？没那么快！仅仅因为他们已经从加高座椅毕业并不意味着他们坐在前排就安全了。

在大多数情况下，13岁是最安全的赌注，因为大小*和*年龄都是儿童身体能够更好地承受碰撞冲击的因素。你可以在这里 阅读更多关于前排安全的 [。](https://lifehacker.com/when-can-kids-ride-in-the-front-seat-of-the-car-1838975741)
