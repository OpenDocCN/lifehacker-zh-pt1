# 你在约会时发现了什么可爱，但在一段感情中却令人恼火的事？

> 原文：<https://lifehacker.com/what-did-you-find-cute-when-dating-but-irritating-in-a-1849473654>

在一段关系的开始，当你的玫瑰色眼镜安全就位时，新伴侣的怪癖是可爱的。但是随着约会的早期阶段变成严肃关系的严酷现实，某些行为从可爱变成了彻头彻尾的令人讨厌。



我，，亲眼目睹了无数恋人在我的红发开始覆盖他们家中的每一寸土地后，对我的红发不再抱有幻想。或者我已经意识到，我一直在看的那个极其滑稽的人实际上是无法停止开玩笑的。当然，还有一连串的*噪音*，它们很快从“可爱”变成“难以忍受”:持续不断的嗡嗡声。临床打鼾。连续打喷嚏一百万次。

所以现在我请求你，Lifehacker 的读者们，分享一下你的玫瑰色眼镜最后一次失去玫瑰色的时候。有哪些你在约会初期觉得可爱，到后来在一段感情中觉得气人的事？

在他们开始 [展示自己恶心的一面](https://lifehacker.com/when-to-show-your-gross-side-in-a-new-relationship-1848839196) 之后，当某些习惯从可爱恶心变成恶心恶心的时候，发生了什么吗？当你们两个不得不做决定的时候，某人随和的氛围真的令人沮丧吗？或者也许有人的自发性开始演变成鲁莽？或者——但愿不会如此——*你的伴侣会模仿婴儿的声音吗？*

我喜欢把这种想法浪漫化，随着关系的发展，你只会越来越喜欢你爱人的小怪癖。事实上，我们中的许多人都曾经说过这样一句话:“【to】嗯，一开始我觉得它很可爱...“很多时候，在一段长期的关系中，最初可爱的东西变得越来越难以忍受。

在下面评论一下在你们的关系中，什么样的事情从可爱变得令人讨厌。然后，我将在下周的另一篇文章中总结最佳答案。
