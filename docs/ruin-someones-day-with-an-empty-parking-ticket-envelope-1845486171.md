# 用一个空的停车罚单信封毁了某人的一天

> 原文：<https://lifehacker.com/ruin-someones-day-with-an-empty-parking-ticket-envelope-1845486171>

没有什么比走到你的车前，看到一个愤怒的橙色停车罚单信封卡在你的雨刮片下面更让人心烦意乱了。当然，还有比得到一张罚单更糟糕的事情可能发生——希望 25 美元、40 美元或 65 美元(我最近在布鲁克林打扫街道时忘记移动我的汽车而被兑换的金额)不会从你紧张的每月预算中挤出支持——但这总是很糟糕。但是如果你把你的门票看作是给你众多敌人中的一个带来糟糕的一天的代价，它可能会稍微差一点。假如你住在一个用信封交付停车罚单的城市(因为某种原因，你不想只在网上支付罚单，因为这是 2020 年)，你需要做的就是保存好这个信封(希望你能保持原样)，直到下一次你的邻居、同事或一个匿名但极其恶劣的停车者以如此的方式冒犯你(也许是恶劣地停车)。

然后，简单地放上信封——没有违章停车罚单，希望你已经付过了！—在他们的挡风玻璃下，这样当他们回到自己的车里时，他们也会感受到不得不向停车神献上礼物的痛苦。此外，如果他们认真缴纳税款和费用——或者只是偏执——当他们打开信封发现引文丢失时，他们的痛苦不会结束。在接下来的几周里，他们将不得不在网上数据库中核对自己的车牌号，以确保不会因为迟交那张根本不存在的罚单而被处以双倍罚款。(在纽约，有效机票有时需要两周或更长时间才能出现在数据库中。)

这个方案不会对你的敌人造成真正的伤害，但是它*可能*会毁了他们的一天。啊，就是这张票。
