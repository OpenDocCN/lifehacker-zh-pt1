# 这是激励自己打扫房间的最有回报的方法

> 原文：<https://lifehacker.com/this-is-the-most-rewarding-way-to-motivate-yourself-to-1848145391>

六年前，克里斯·弗莱明(Chris Fleming)发布了小品《 [公司来了](https://www.youtube.com/watch?v=GBwELzvnrQg&t=22s) 》，作为他热播网络系列[*格利*](https://www.youtube.com/watch?v=ZdPdiQNWDeY&list=PL6B0B7F79AE5B948B&t=0s) 的一部分。在影片中，一个疯狂的盖尔用吸尘器吸了吸空气，转了一圈，然后大喊“大卫，把贝壳放在门把手上！、“抖抖枕头，你们这些失败者！”和“我们能把同性恋植物弄走吗？?"



YouTube 上有超过 1300 万的浏览量，可以肯定地说这个视频是可信的(尽管它只展示了盖尔和弗莱明所能提供的一切的一小部分)。我从来没有用吸尘器吸过空气，但是如果我知道除了我自己或我男朋友之外的其他人会从前门进来，我会很有动力去打扫我的住所。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-GBwELzvnrQg&start=22](https://lifehacker.com/embed/inset/iframe?id=youtube-video-GBwELzvnrQg&start=22) 

这并不是说我只在有人来的时候打扫卫生，但是打扫卫生是不同的。如果你住在房子、公寓、公寓、船或改装过的储物容器里，你必须定期做一些清洁工作——实际上是每天都要做，事实上是每天都要做。拥有一栋房子意味着你的待办事项清单永远不会完成，人们很容易对肮脏的踢脚板之类的东西念念不忘，这些东西几乎没人会看。(嗯。那不完全正确。我的继母曾经称赞我的踢脚板有多干净，这毁了我的生活，因为我把“干净的踢脚板”变成了我个性的一部分。)

我的观点是邀请朋友来家里会让你集中注意力。它给你一个目标。你突然有了灵感，让你的空间，嗯，不是一尘不染，而是宜居和迷人。公共区域——书房、厨房和浴室——成为你的优先选择。垃圾被清理出去，杂乱的东西被从交通繁忙的地方移走，溢出物和粘滞点被清除，厕所被打扫干净。你甚至可以擦掉浴室镜子上的干牙膏。

如果你是那种不以自我舒适为动力的人——你是否值得拥有一个有序的住所？—为别人打扫卫生可能会让你干劲十足。然后，一旦你的客人离开，你可能会注意到你的心情会因为一个更干净的客厅而明显改善，并且你可能会从此把打扫卫生视为你可以为自己做的一件好事。即使打扫房间的很大一部分包括把要洗的衣服搬进卧室，拥有一个感觉和外观都不错的房间可以激励你去处理其余的。

这种策略最适合你仍想给其留下深刻印象的新朋友。如果你真的喜欢和这个人在一起，这也会有所帮助，因为这次拜访可以作为对你所有辛勤工作的额外奖励。(主要的回报是一个更干净的空间，让我们面对现实吧，这只是暂时的。)
