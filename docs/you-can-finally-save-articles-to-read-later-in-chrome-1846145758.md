# 你终于可以在Chrome中保存文章以供“稍后阅读”

> 原文:[https://life hacker . com/you-can-finally-save-articles-to-read-la ter-in-chrome-1846145758](https://lifehacker.com/you-can-finally-save-articles-to-read-later-in-chrome-1846145758)

谷歌正在测试一项新的稍后阅读功能，让用户保存他们遇到的文章和其他链接，并希望稍后阅读。保存的链接出现在Chrome书签菜单的一个新的“稍后阅读”部分，但不会像普通书签那样永久保存。

Watch

还有其他应用程序和浏览插件做同样的事情，但是Read Later内置在Chrome的界面中，所以你不必依赖第三方解决方案——或者 [用过多的打开标签](https://lifehacker.com/close-all-your-tabs-just-close-em-1845775727) 和书签堵塞你的浏览器。

现在阅读已经是iOS用户的默认功能，但你可以在Android，Windows和Mac上尝试它，方法是在Chrome的稳定版本中打开一个实验标志。

要找到并启用标志，打开一个新的Chrome标签，然后进入**Chrome://flags/# read-la ter**。使用下拉菜单启用设置，然后在提示时重启Chrome。

或者，如果你不想弄乱浏览器的实验设置，你可以下载Chrome Canary。从版本90开始，Chrome Canary默认开启“稍后阅读”。请记住，Canary是用于测试未完成的功能，因此不太稳定，所以您可能会遇到错误和稳定性问题。结果是，你可以尝试一些新功能，比如稍后阅读和在屏幕共享时将Chrome通知静音的。

### 如何使用Chrome的稍后阅读功能

**桌面上:**

1.  点击Chrome导航栏中的星形图标(你可以用这个图标来标记网站)。
2.  选择**“稍后读取”**页面将关闭。
3.  当你准备好补上你保存的文章时，点击浏览器右上角的**“阅读列表”**。**注意:**如果没有看到阅读列表，打开一个新的标签页或者右击导航栏选择**“显示书签栏”**
4.  单击列表中的链接将其打开。单击复选按钮标记为已读，或单击“X”将其删除。

**在手机上:**

1.  点击并长按链接或URL，直到快速选项菜单出现。
2.  点击**“稍后阅读”**保存到Chrome的阅读列表中。
3.  要找到你的阅读列表，点击三点**“更多”**图标，然后前往**书签>阅读列表**。
4.  在这里，您可以打开保存的链接，将它们标记为已读，或者从列表中删除它们。
