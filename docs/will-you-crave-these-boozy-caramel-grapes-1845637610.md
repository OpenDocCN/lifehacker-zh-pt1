# 你会渴望这些香甜的焦糖葡萄吗？

> 原文:[https://life hacker . com/will-you-dream-these-boozy-caramel-grapes-1845637610](https://lifehacker.com/will-you-crave-these-boozy-caramel-grapes-1845637610)

我最喜欢的秋季传统之一是和我的新英格兰男朋友争论焦糖苹果。他——一个错误的人——认为来自康涅狄格州的给了他对什么是*的特殊洞察力。*”何坚持说，苹果糖——又难吃又笨——是上等的。

然而，我取笑他，因为他不知道如何做苹果糖，而且我拒绝做除了焦糖以外的任何东西；就这样，，这座房子依然是焦糖苹果屋。不过最近，我考虑扩大我的房子，为一些浸过伏特加、蘸过焦糖的绿葡萄腾出空间。它们是甜的，是酸的，外面裹着花生。我爱他们。

是的，看起来好像是我泄露了这个视频的结局，但是许多问题仍然存在:伏特加能进来吗？所有形式的焦糖都有效吗？你应该先洗葡萄吗？这些只是这里将要解答的几个谜题。

我想这份食谱是设计来作为派对点心的，但是我们这些天没有太多派对。幸运的是，它和“独自坐在沙发上”零食、“周一晚上”零食或“躲在桥下”零食一样有效。你甚至不用等焦糖凝固。J 只要用牙签扎一颗葡萄，蘸上焦糖，压成一堆碎花生，塞进嘴里。我不会称之为“优雅”，但它直截了当，它是美味，它是*而不是*包裹在过于坚硬、过于甜腻的糖果外壳中。
