# 你应该做这些芥末鸡蛋饼干

> 原文:[https://life hacker . com/you-should-make-these-deviled-egg-cracker-1849560253](https://lifehacker.com/you-should-make-these-deviled-egg-crackers-1849560253)

我有 [很多想法和感受](https://lifehacker.com/13-smarter-ways-to-make-killer-deviled-eggs-1847167171) 关于炒蛋[正确的做法](https://lifehacker.com/how-to-make-perfect-deviled-eggs-1821471027) 来制作它们。但最近，我变得半痴迷于“解构”和“重新想象”的魔蛋。我是 [未加调料的鸡蛋](https://lifehacker.com/un-devil-your-eggs-this-easter-1848778518) 和 [加调料的鸡蛋沙拉](https://lifehacker.com/just-make-deviled-egg-salad-already-1836077324) ，的忠实粉丝，看起来我将把玛莎·斯图尔特的 [骗子的加调料的鸡蛋](https://www.marthastewart.com/1527149/cheaters-deviled-eggs) 加入我的未加调料的鸡蛋剧目中。

像这个流派的其他食谱一样，没有捣碎，混合和管道。相反，鸡蛋被水平切成片(而不是垂直切成两半)，放在涂有蛋黄酱和芥末的饼干上，然后用一些新鲜莳萝和甜熏辣椒粉结束。

玛莎有完整的食谱，但我更愿意把它当作一个模板。芥末酱，第戎酱可以换成经典的黄色或者加一点辣根。你也可以在加点野生的配料，就像我们在 [做的未加调料的鸡蛋](https://lifehacker.com/un-devil-your-eggs-this-easter-1848778518) 。腌制肉、鲑鱼(或任何其他)鱼子、腌制蔬菜、冷烤蔬菜、薯片，甚至黄油都可以。(黄油和萝卜搭配使用效果尤其好。)然而，有一些策略可以确保你的芥末鸡蛋饼干成功。

## 不要把鸡蛋煮过头

通过将鸡蛋浸入沸水中，然后在煮10分钟后在冰浴中电击它们来避免可怕的灰色环。这会给你一个完全煮熟的蛋黄，但不是干的，这是非常重要的，因为干的蛋黄感觉很恶心。

## 给你的鸡蛋调味

辣椒粉不错，但盐是绝对必须的。在任何情况下都不要忘记它。每个鸡蛋片上的一小撮犹太盐是可爱的，但是像马尔东这样松脆的片状盐真的会提升你的T2美食。

## 用绿色的东西结束你的鸡蛋

鸡蛋和饼干的组合可能会很难吃，所以要确保上面有一些新鲜的东西，比如玛莎推荐的莳萝，或者一些欧芹或香葱。这不仅让T2的咬合力更平衡，也有助于T4的展示。

## 如何制作芥末鸡蛋饼干

#### 配料(12-15份，取决于你把鸡蛋切成多薄):

*   3个鸡蛋
*   2汤匙蛋黄酱
*   1汤匙芥末
*   可选:调制山葵
*   12-15块饼干(我喜欢里兹饼干，但玛莎推荐米饼)
*   盐
*   辣椒粉
*   某种绿色草本植物
*   还有其他有趣的配料让你喜欢吗

拿一壶水烧开，轻轻地把鸡蛋加入水中(我用蜘蛛来做这个)。将鸡蛋煮10分钟，然后在冰浴中震荡15分钟。鸡蛋去皮，横着切片。

将芥末和蛋黄酱混合在一起，然后加入辣根调味，如果用的话。将你的混合物涂在饼干上，并在每片饼干上放一片鸡蛋片。用盐和辣椒粉调味，顶部用新鲜香草装饰。如果你愿意的话，可以加入香脆的培根或烟熏三文鱼(我经常这么做)。
