# 你应该制作自己的“Spotify 馅饼”图

> 原文：<https://lifehacker.com/you-should-make-your-own-spotify-pie-chart-1849035508>

从 Spotify 的年终总结到对你的音乐库进行分类的众多应用程序，炫耀你最喜欢的音乐艺术家和流派是使用流媒体音乐平台的一半乐趣。



## 什么是 Spotify Pie？

最新的 Spotify 收听统计趋势是" Spotify Pie。“由加州大学洛杉矶分校学生达伦·黄(Darren Huang)创建的简单 web 应用程序分析你在过去一个月里听的所有歌曲，将歌曲整理成特定的流派，并将所有数据变成饼状图，你可以在社交媒体上与朋友分享。它类似于 Spotify Wrapped，但更简单，你可以在任何时候使用它，而不是等到年底才能看到哪些艺术家和流派给你排名最高。

Spotify Pie 是一个非官方的第三方应用程序，但它可以在几乎所有桌面和移动浏览器上运行。请注意，你需要使用 Spotify 帐户登录才能看到你的结果，但你的收听历史是该应用程序访问的唯一数据，否则应该是安全的。

## 如何制作自己的 Spotify 馅饼

如果你看到其他 Spotify 听众在社交媒体上分享他们的 Spotify 馅饼，并想亲自尝试一下，以下是方法:

1.  首先，在手机或桌面上打开 Spotify Pie 网页。
2.  点击**“登录 Spotify”**，使用您想要创建饼图的账户登录。
3.  登录后，Spotify Pie web 应用程序将生成一个流派转盘。从那里你可以分享或截图你的结果。

我亲自尝试了 Spotify Pie，虽然我对过去几周我听得最多的乐队并不感到惊讶，但我对该应用程序在音乐流派方面的高度专一感到震惊。这与 Spotify 如何分类歌曲关系更大，而不是 Spotify Pie 如何分类，但我看到一些用户对他们饼状图中的流派感到惊讶甚至困惑。

抛开流派标签不谈，Spotify Pie 仍然是一种有趣而简单的方式来跟踪和分享你的音乐收听统计数据。一定要尝试一下——但是如果你的一些派片是你以前从未听说过的类型，不要感到惊讶。
