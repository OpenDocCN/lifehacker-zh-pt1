# 购买复古服装和配饰时如何识别假货

> 原文:[https://life hacker . com/how-to-spot-fakes-when-shooting-on-1846162736](https://lifehacker.com/how-to-spot-fakes-when-shopping-for-vintage-clothing-an-1846162736)

任何花时间购买复古服装和配饰的人都知道，这可能是非常偶然的。当然，你需要了解整个事情，这些衣服是旧的(或者至少它们应该是旧的)，这意味着它们可能已经被穿过，并且可能不是最完美、一尘不染、完整无损的衣服。

但是它们真的是真实的吗？嗯，这取决于很多事情，包括在这种情况下你认为什么是“真实的”——比如它是真的旧的还是试图通过的新的复古风格的物品。

然后还有假货的问题。虽然这可能不会成为19世纪中期90年代左右的基本版t恤的问题，但当你与更知名的奢侈品牌和设计师打交道时，这种事情就会发生。那么，你怎么知道一件复古服装是真货，还是试图让你多付钱呢？以下是需要了解的内容。

### 确保这些作品物有所值

只是澄清一下:我们不是在谈论旧货店里的物品，那里的价格相对较低，有时它们附属于一个非营利组织。我们说的是真正的古董店和网站，有精心策划的服装系列和明显更高的价格。因为你为这些作品支付了额外费用，所以你想确保你的钱花得值，不会被骗。

### **如何辨别一件古董是不是赝品**

好吧，那你怎么识别假货呢？洛杉矶服装设计师吉利安·克拉克说，这完全取决于假货的质量。以下是她在采访真正简单的 时得到的三点建议:

#### 检查徽标

克拉克告诉Real Simple:“大多数假货很容易被识别出来，因为它们模仿的是任何一种标志或设计。”"通常情况下，设计会被稍微改动一下，所以不关心的消费者几乎完全不会注意到。"

#### 检查标签

当谈到复古设计师服装时，克拉克说，大多数商品都会有某种“真品标签”或其原始标签，这不容易复制。

#### 向值得信赖的古董零售商购物

最后，克拉克建议，坚持与你认识和信任的古董服装经销商打交道，不要一开始就试图卖给你假货。
