# 你应该华夫饼干一些沙拉三明治

总之。看到女孩们点了一份沙拉三明治华夫饼——我决定称之为“fawaffles”——这是我最终做出来的动力，它们很好吃。

如果你有自制沙拉三明治的食谱，欢迎你使用，但是搅拌加水的混合物也有令人满意的地方。我肯定每种混合在盘子里的表现都有所不同，但是乔氏混合在不到两分钟内就变成了美味的华夫饼干，里面很嫩，外面有点脆。(每块华夫饼还含有12克蛋白质，如果你的新年决心包括得到华夫饼 ，这是很棒的 [。)](https://lifehacker.com/the-best-way-to-cook-seafood-if-youre-skittish-about-co-1848334399)

不同品牌的说明会略有不同，但你要加入足够的水，让面糊可以舀，但仍然很软。(对于乔氏混合，那是1杯混合和3/4杯水，盖上盖子休息半小时。)

沙拉三明治面糊比许多其他华夫饼干更粘，所以你可能想在华夫饼干之前用一点油打你的华夫饼干。(只要 [避免含有卵磷脂](https://lifehacker.com/the-one-thing-you-should-never-put-in-your-nonstick-pan-1848332657) 的喷雾，如果你的华夫饼机不粘的话；随着时间的推移，它会堆积在涂层上，让你以前不粘的华夫饼- 制作机变得非常粘。)

将华夫饼烤模加热至中火，给两个盘子都倒点油。一旦热了，把1/2杯面糊放在华夫饼烤模的中心，用力压下。让它煮1 1/2-2分钟，直到fawaffle很容易从炉栅中释放出来。(你可以用筷子把它夹到锅铲上。)

搭配一个鸡蛋作为富含蛋白质的早餐，或者搭配通常的沙拉三明治。(我做的时候tzatziki没了，所以用了酸奶油。这很好，但如果有tzatziki会更好。)
