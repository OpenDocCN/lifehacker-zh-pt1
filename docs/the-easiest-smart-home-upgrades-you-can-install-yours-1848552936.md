# 你可以自己安装最简单的“智能家居”升级

> 原文:[https://life hacker . com/the-easy-smart-home-upgrades-your-can-install-yours-1848552936](https://lifehacker.com/the-easiest-smart-home-upgrades-you-can-install-yours-1848552936)

让你的房子“更智能”——通过自动控制和语音命令——可以简化家务*和*为你节省能源成本。如果你曾经梦想过像杰特森一样的生活，但认为智能技术对你的预算来说太复杂或太昂贵，这里有一些你可以自己做的升级。

## 智能锁

智能锁可以让你安心，可以让你远程锁门或开门，或者让客人进入。最好的这种锁是可以通过应用程序从手机或平板电脑上操作的，价格从150美元到300美元不等。一些智能锁还能够使用面部识别或读取您的指尖进行访问。市场上大多数智能锁都配有一个无线集线器，可以插入你的墙上插座，锁本身靠电池运行。它们 易于安装，只需一把螺丝刀即可完成的工作。

## 智能恒温器

智能恒温器可以是一个很好的升级，帮助你节省供暖和制冷成本。智能恒温器的价格从100美元到250美元不等。有几类具有“学习”功能的恒温器，根据你的习惯使用数据，根据活动自动改变你家的温度，还有节能恒温器，调节温度，帮助你省钱。

一旦你为自己选择了正确的恒温器，安装和使用这种装置就相当简单了——你需要一把用于低压电线的小型平头螺丝刀、一把电钻和一把螺丝枪。开始安装前，确保关闭HVAC系统的电源。遵循制造商的布线安装说明，用提供的螺丝将其安装到墙上。这是一个简单的项目，，但它确实需要一些基本的布线。

## 智能灯泡

普通的s mart light 灯泡起价约为8美元，T2的价格高达50美元，带着各种铃铛和哨子的高档型号也是如此。智能灯泡的优势在于，你可以对它们进行预编程，通过手机上的应用程序对它们进行控制，在某些情况下，它们会对语音命令做出反应。有些型号具有颜色变化和度假模式等功能，即使你不在家也可以控制你的灯。它们不需要任何额外的硬件，只需通过应用程序将其拧入与您的设备配对的灯插座即可。这个项目不需要工具，而且你可以通过关掉旧灯泡来节省电费。

## 智能水传感器

如果你担心漏水或洪水，智能水传感器是让你安心的好方法。它们的价格从40美元到400美元不等，取决于你想监控的区域。将传感器放在你的地下室或担心漏水的电器或水槽附近可以帮助你避免灾难。除了感测水，一些模型还可以感测温度的极端变化，以帮助防止像冻结管道这样的事情。它们非常适合洗衣机、热水器和其他可能漏水的电器，而且安装简单。有些型号需要插在插座上，但也有电池驱动的型号可以在没有插座的地区使用。这个项目不需要任何工具，但你可能需要安装一个应用程序来从你的传感器设备获取警报。

## 智能插座

智能插座是监控能源使用和远程开关的好方法。您可以将它们直接安装在现有插座上，也可以选择具有语音控制和远程功能的产品。这些都很方便，而且对节省你的电费来说很重要。它们不需要任何安装工具，价格从12美元到60美元不等。
