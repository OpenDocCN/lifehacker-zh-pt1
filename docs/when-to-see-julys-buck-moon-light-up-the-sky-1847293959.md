# 什么时候看七月的“巴克月亮”照亮天空

每个月都有满月，但只有七月可以声称月亮是以一种长着巨大鹿角的动物命名的:巴克月亮。这个特殊的月份已经成为天体事件的重磅炸弹，英仙座流星雨 处于其关键时刻。但是这个月的满月提供了另一个让你抬头仰望天空的理由——幸运的是，对于任何决定外出冒险(或向窗外瞥一眼)的人来说，这几乎是不可能错过的。

## Buck Moon是什么时候？

7月的满月将在23日日落后升起，并在美国东部时间晚上10点37分达到顶峰。根据Space.com的说法，它会在前一天晚上和后一天出现，所以不熟悉月亮八个相位的人真的无法分辨。

这个月亮被称为雄鹿月，因为“公鹿的鹿角在这个时候正处于完全生长的状态。根据《农民年鉴》[的记载，“雄鹿每年脱落并再生鹿角，随着时间的推移，鹿角会变得更大，更令人印象深刻。”。月亮是为了纪念那些巨大的鹿角而命名的，尽管这个月亮有几个其他的昵称(一般满月都是这样)。几个世纪以来，美国土著部落给七月的月亮起了不同的名字，下面的农民年鉴列出了这些名字:](https://www.almanac.com/content/full-moon-july#)

> **浆果月**(Anishinaabe)**樱桃成熟时的月亮**(达科他州)**成熟玉米月**(切诺基)，以及**覆盆子月**(阿尔冈昆，奥吉布韦)。

在南半球，现在是冬天，七月的满月有不同的名字，更能代表寒冷的天气。正如 [地天注](https://earthsky.org/astronomy-essentials/full-moon-names/) 所说，这些名字分别是:狼月、旧月、冰月。

## 如何看巴克月亮

你将不得不看向东南方的地平线，看着月亮升起，并保持你的眼睛在那里训练，看着它在23日达到顶峰。《农民历书》提供了一个月出计算器 来计算你与月出的直接坐标，但是你可以一直看着天空，就像几千年来人们一直做的那样。
