# 使用此网站查看M1 MAC电脑上运行的软件

> 原文:[https://life hacker . com/use-this-website-to-see-what-software-runs-on-m1-Macs-1845750628](https://lifehacker.com/use-this-website-to-see-what-software-runs-on-m1-macs-1845750628)

苹果从英特尔处理器转向在Mac电脑和MacB上使用自己的M1芯片，这意味着iOS应用程序现在可以直接在macOS上运行。不幸的是，这也意味着一些为英特尔硬件开发的桌面应用程序现在在新的M1 MAC电脑上不兼容。

虽然许多流行的Mac应用程序都是原生支持的，但其他应用程序需要一个名为Rosetta 2的程序。Rosetta 2是一个仿真器——基本上，它使用软件来模拟英特尔的硬件，因此您可以在Mac上运行仅支持英特尔的应用程序，即使它们本身还不支持。Spotify的桌面应用就是一个例子。

然而，有相当大一部分应用程序不能在M1 MAC电脑上运行*和*不支持Rosetta 2，比如流行音乐制作软件Pro Tools。此外，一些应用程序可以在Mac上运行——无论是本机运行还是通过Rosetta 2运行——但存在漏洞和性能问题。Adobe的许多创意套件应用程序——如InDesign和Illustrator——都属于这一类。

为了帮助用户确认他们新买的M1苹果电脑可以运行什么软件，程序员阿卜杜拉·迪亚 创建了一个名为 [的便捷网站，苹果芯片准备好了吗？](https://isapplesiliconready.com/) “这显示了哪些应用程序可以在M1的MAC电脑上运行。您可以使用页面顶部的搜索栏来查找特定的应用程序，或者逐页浏览整个列表。

有过滤器只显示“M1本地支持”、“仅Rosetta 2”或“不工作”的应用程序，你可以将列表缩小到特定类别，如视频制作、生产力、摄影等。

该表的图标表明某个应用程序是否受支持(无论是本机支持还是Rosetta 2支持):

*   绿色勾号表示它运行正常。
*   一个小小的感叹号意味着这个应用程序可以工作，但是有一些错误。
*   红色禁止符号表示该应用程序不兼容。

该表还列出了运行应用程序所需的M1固件版本。你可以点击该应用程序，查看其性能的更多详细信息，以及是否计划为尚不兼容的程序提供M1支持。

如果您没有看到列出的您正在寻找的应用程序，或者您是一名开发人员，想要将您的软件添加到网站或更改列出的支持，请单击**“请求一个应用程序**”并填写提交表单。
