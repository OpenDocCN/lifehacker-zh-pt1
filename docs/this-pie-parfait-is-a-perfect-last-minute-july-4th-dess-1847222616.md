# 这个馅饼冻糕是7月4日最后一分钟的完美甜点

作为一个美国人，我并不感到特别自豪，这并不是什么秘密，但我是一名生活在美国的美食作家，这意味着某些编辑决策会受到某些美国假期的影响。(这就是资本主义，宝贝。)本周日是独立日，这一天通常以食用“国旗蛋糕】、以及其他红色、白色和蓝色甜点来庆祝。

我不打算在7月4日打开我的烤箱，但即使是我那颗共产主义铁石心肠也有点被主题甜点迷住了，而且碰巧我最喜欢的两种水果——蓝莓和樱桃——符合节日的配色方案。用一些白色鲜奶油使它变圆，你就有了一个解构的国旗甜点，如果不是一个完整的国旗蛋糕的话。

这种款待的形式很简单:在一个漂亮的玻璃杯中，将樱桃派片与鲜奶油和蓝莓分层，然后用勺子吃。如果你愿意，你也可以用蓝莓派和新鲜樱桃来调味。用商店买的馅饼；自制馅饼太珍贵了，不能弄碎。

鲜奶油可以是新鲜的，也可以是罐装的，或者你可以融入整个美国主题，拿一桶凉爽的鲜奶油。想用樱桃派馅料和饼干代替完整的馅饼吗？去吧！这种甜点应该是自由和简单的——非常适合忘记我们甚至有一个漫长的周末即将到来的人(我)，或者讨厌烘焙的人(也包括我)。它的味道也很好，因为它是馅饼、鲜奶油和蓝莓，人们会喜欢它，因为它是*馅饼*、*鲜奶油*和*蓝莓*。

额外加分:准备一罐或一碗或鲜奶油，这样你的客人就可以根据自己的喜好添加额外的奶油。甜点的每一种成分都很重要，但鲜奶油可能是最重要的。
