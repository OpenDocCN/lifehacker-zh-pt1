# 如何抚养一个优雅的失败者

我10岁的女儿有很多优秀的品质。他风趣、大方，对世界充满好奇。然而，他不喜欢输掉比赛。公平地说，这并不稀奇。我认为，如果可以选择的话，我们大多数人更愿意赢得我们玩的任何游戏——否则，记分有什么意义呢？但这是我们在我家已经努力了多年的事情，我在这个过程中学到了一些技巧。

### 从合作博弈开始

记分的目的是为了弄清楚谁赢了这场比赛，但是赢真的不是*玩*的目的。孩子们不会在六岁时开始踢足球，因为他们想击败对手；他们玩是因为好玩。因此，在你开始向他们挑战糖果王国决斗之前，向他们介绍合作游戏。

学龄前儿童是开始玩合作游戏的黄金年龄，这种游戏不会让*你*对抗*他们*。相反，你们一起朝着一个共同的目标努力，同时仍然获得练习的所有好处，如轮流、按照指示、和磨练精细的运动技能。我们家最喜欢的一款是 [喂狗](https://www.amazon.com/Peaceable-Kingdom-Woozle-Preschool-Builder/dp/B00L3DQ1B0?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-raise-a-gracious-loser-1846160639&asc_source=&tag=kinjalifehackerlink-20) ，不过其他流行的合作桌游还有 [轰赶猫头鹰](https://www.amazon.com/Peaceable-Kingdom-Winning-Cooperative-Matching/dp/B004HVKAAI/ref=sr_1_8?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-raise-a-gracious-loser-1846160639&asc_source=&crid=21WVS3UTWIZSM&dchild=1&keywords=cooperative board games for kids&qid=1611949693&s=toys-and-games&sprefix=cooperative,toys-and-games,169&sr=1-8&tag=kinjalifehackerlink-20)[人鱼岛](https://www.amazon.com/Peaceable-Kingdom-Mermaid-Cooperative-Adventure/dp/B004HVKABC/ref=sr_1_6?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-raise-a-gracious-loser-1846160639&asc_source=&crid=21WVS3UTWIZSM&dchild=1&keywords=cooperative board games for kids&qid=1611949693&s=toys-and-games&sprefix=cooperative,toys-and-games,169&sr=1-6&tag=kinjalifehackerlink-20) 和 [竞逐宝藏](https://www.amazon.com/Peacable-Kingdom-Race-Treasure-Game/dp/B006S9MUPU/ref=sr_1_4?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-raise-a-gracious-loser-1846160639&asc_source=&crid=21WVS3UTWIZSM&dchild=1&keywords=cooperative board games for kids&qid=1611949693&s=toys-and-games&sprefix=cooperative,toys-and-games,169&sr=1-4&tag=kinjalifehackerlink-20) 。

### 教他们玩长线游戏

最近，当我和儿子开始玩一轮 [跳波](https://www.mattelgames.com/en-us/cards/skip-bo) 时，我突然明白了一些事情，他甚至在完成第一轮之前就已经开始毛发倒竖。虽然我知道在那场游戏中，运气会迅速而广泛地波动，但我意识到，他把每手牌都当成了自己的迷你游戏。连续四手坏牌对他来说就像是四次失败。难怪这一点都不好玩。

当我用这些话向他提出这个问题时——“试着不要把每一个回合都看成是它自己的游戏，而是一个更大的拼图的一部分”——他产生了共鸣，“我不明白为什么你已经开始不安了；我们*刚刚*开始玩“从来没有。

如果你的孩子没有很快产生共鸣，那就在你玩的时候指出你自己的不幸，这样他们就能从两个角度看到游戏是如何进行的。你可能会说，“哦，哇，我以前远远领先于你，现在你就在我的后面！”或者“我以为我能追上你，但是我觉得你现在已经远远超过我了！”

年幼的孩子通常只从他们的角度看比赛，让你讲述一些你的经历(特别是当你输了的时候)可以帮助他们建立对对手的同情。目标不是让他们对胜利感到难过，而是提醒他们，在任何给定的时间，如果有人赢了，其他人就输了。

### 做一个优雅的失败者(和成功者)

有一次，我和丈夫以及两个好朋友去亚利桑那州北部的一个小木屋度周末，有人指责我。我们四个人围坐在小屋的餐桌旁，喝着啤酒，玩着令人兴奋的猜谜游戏——就像成年人做的那样——我的一个朋友无意中说出了他认为我是我们小组中最有竞争力的一个。

我知道这显然是错误的，于是我列出了一个详细的论据，准确地说明我是如何*而不是*成为团队中最有竞争力的(这根本不是一件有竞争力的事情)。从那以后，我不得不审视自己的内心深处，承认我有可能有点好胜心，这主要是在我玩棋盘游戏或翻杯游戏时表现出来的(但只是因为我两者都很擅长)。然而，真的想赢并对赢*或*输表现得很差劲并不一定要去做，这是你应该为你的孩子做的榜样。

自从我儿子很小的时候，我们就有一个固定的惯例，当一场比赛结束时，我们握手并说，“好比赛。”对他来说，这是一种带着微笑看我输球的好方法——不一定是因为输了而高兴，而是因为参加了比赛而高兴。这也是练习和示范优雅取胜的好方法。不要幸灾乐祸。

最重要的是，当我们失去一只手的时候，或者说，当你的足球队输掉一场重要比赛的时候(我是布朗队的球迷，所以...)。如果你让它毁了你的一天，他们会意识到胜利是如此重要，以至于失败意味着它会对你的情绪产生负面影响。告诉他们如何让损失，无论是大的还是小的，尽可能地从你的背上滚下来。

### 如果你需要，把它收起来

游戏应该是有趣的，所以如果游戏之夜每次都以一堆眼泪结束，那可能是时候从游戏中休息一下了——或者至少从一个似乎让他们毫无理由地沮丧的游戏中休息一下(我正在看着你， [《大富翁游戏玩家](https://www.amazon.com/Hasbro-C1815-Monopoly-Gamer/dp/B01MR4UXGC?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-raise-a-gracious-loser-1846160639&asc_source=&tag=kinjalifehackerlink-20) )。一旦过了一段时间，紧张局势有所缓解，他们准备再试一次，你随时可以再试一次。
