# 在 Queso 中融化碎奶酪制作下一级玉米片

> 原文：<https://lifehacker.com/make-better-nachos-by-melting-shredded-cheese-in-queso-1845889338>

通常，当一个人做玉米片(或奶酪薯条)时，会有一个点，你必须在奶油、美味的奎索和非常有弹性的奶酪拉之间做出选择。奎索不像融化的碎奶酪那样有弹性，而且碎奶酪容易出油并很快凝固——但是，结合它们的力量，你可以拥有两种奶酪的精华。



将切碎的奶酪融化在一碗从商店买来的奎索里，你会得到一种奶油状的、美味的奶酪，有着惊人的弹性。我是说，看看这奶酪。看着它，你会惊叹于它的壮丽。

融化得很好的碎奶酪在这里效果最好——想想瓦哈卡奶酪、胡椒杰克奶酪，甚至是马苏里拉奶酪——但即使是碎切达奶酪也非常棒(但是，不要吃像帕尔马干酪或陈年高达干酪这样的超硬奶酪)。jarred 奶酪酱中的融化盐和乳化剂使一切都保持美味和流动性。即使在室温下，这种组合仍然是可蘸的和可舀的，这是你用有些易碎的玉米片送到嘴里的东西的两个非常理想的品质。

为了达到最佳效果，使用一份碎奶酪和两份罐装 queso(按体积计)，然后在一个小调味汁锅中用中低火加热，或者在微波炉中的一个耐热碗中加热 30 秒，直到所有东西都变热、流动和有弹性。当你能像上图一样成功时，你就成功了。将你的奶酪混合物倒在一堆碳水化合物上，做玉米片、巧克力或薯条，或者只是蘸着吃。(薯条是显而易见的选择，但你考虑过蔬菜吗？柿子椒特别好看。)
