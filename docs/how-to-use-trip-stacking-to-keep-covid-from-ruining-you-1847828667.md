# 如何使用“旅行堆积”来防止 COVID 破坏你的假期计划

> 原文：<https://lifehacker.com/how-to-use-trip-stacking-to-keep-covid-from-ruining-you-1847828667>

随着旅行禁令，航空公司取消航班的增加，以及与达美变体相关的普遍不确定性，人们正在预订重叠的假期，以确保他们能够旅行——这就是所谓的“旅行堆积”基本上，这是一个后备计划，确保你仍然可以在已经取消的日子里旅行，即使你最初的旅行计划失败了。下面来看看它是如何工作的。

## **什么是“行程叠加”？**

旅行叠加是指你在同一时间预订了多个假期，除了一个假期外，其他假期都打算取消。通常包括预订一次雄心勃勃的大旅行(海外旅行)和一次较小的旅行，要么在美国，要么在离家更近的地方，比如墨西哥或加勒比海。

原因很简单:COVID 时代的旅行限制是不可预测的，这使得很难确定地计划假期，尤其是因为大多数人提前几个月就计划好了大型旅行。例如，在欧盟最近将美国从入境旅游国家的“安全名单”中删除后，许多欧洲国家实施了旅行限制。如果你从今年夏天早些时候就开始计划旅行——当时大多数人都预计边境会在秋天开放——你可能会被迫取消计划。

问题是，许多航空公司和酒店都有灵活的取消政策，允许你以最低或零成本取消或改变计划，这使得他们更有吸引力。在这种情况下，真正的问题是大多数旅行者已经预订了时间，突然取消迫使他们在飞行中改变他们的计划。当然，这些假期可以重新预订，但这并不总是容易做到的，尤其是如果这些日子不能延期，必须在年底前休完。

这就需要一个预先预订的后备计划了:如果你雄心勃勃的旅行计划失败了，你仍然有另一个假期作为后备(例如，你仍然可以去可取消的佛罗里达旅行，而不是西班牙)。毕竟，通常情况下，在家度假可能是好的，但如果你已经预订了三周的国际旅行，那可能就没什么意思了。

## **确保你真的提前取消**

通过预订更多你想去的假期，你从其他想去同样旅行的人那里抢走了一些时间。出于这个原因，如果越来越多的人开始取消预订，航空公司和酒店最终可能会开始缩减灵活的取消政策。如果你计划安排多次旅行，试着尽快取消你所有的后备预订，不仅仅是因为这是礼貌的做法，还因为你不太可能忽略任何最后一分钟的取消费用。
