# 这个扩展让你最终可以在手机上后台播放 YouTube

> 原文：<https://lifehacker.com/this-extension-lets-you-finally-play-youtube-in-the-bac-1848143437>

移动端的 YouTube 从来都不是理想的体验。与在电脑上观看 YouTube 不同，在电脑上你可以灵活地让视频在后台播放，而手机上的 YouTube 通常被锁定在你用来观看它的应用程序或网站上。并且如果你想要所有的花里胡哨的东西——比如无广告流媒体——你需要支付 YouTube Premium，不管*用的是什么*设备。也就是说，除非你走另一条路。



## 醋解决了你在 YouTube 上的大部分问题

介绍醋([【App Store 上 1.99 美元](https://apps.apple.com/us/app/vinegar-tube-cleaner/id1591303229) ) ，你不知道自己需要的 YouTube 扩展。D 由 [和一只恐龙](https://andadinosaur.com/launch-vinegar) 开发，扩展适用于 iPhone、iPad 和 Mac，并修复了视频流服务的许多主要问题。

他最直接、最明显、最受欢迎的表演是正在移除视频广告。虽然广告是 YouTube 上许多内容创作者的重要收入来源，但它们也令人沮丧、咄咄逼人，而且经常成为攻击目标。虽然你可以使用许多可用的广告拦截器中的一个来处理广告，但醋方便地内置了这一功能。

说到这，醋还阻止 YouTube 跟踪你的播放、暂停、寻找活动。如果，读完那句话，你在想“嗯，YouTube 跟踪我的播放，暂停，寻找活动？”你并不孤单。我想我们中的许多人并不知道这一点，也不一定感到惊讶，但尽管如此，还是对 YouTube 和谷歌令人毛骨悚然的间谍活动感到沮丧。

这一扩展还将画中画带回了移动 YouTube。这项服务通常将 iOS 上的画中画仅限于 YouTube 高级用户(T2)，即使你在 Safari 上使用网页版的 YouTube。醋解决了这个问题，允许你在浏览其他应用程序时保持你的 YouTube 窗口不动。

我最喜欢的功能？醋还可以让你只播放音频，因为它允许视频继续播放，即使 Safari 在后台。这意味着你终于可以在 YouTube 上听音乐，或者在应用程序之间快速切换，而无需暂停视频的音频。这是我们最有可能在电脑上复制观看 YouTube 的体验的方式。

虽然你可以在 Safari 标签页之间跳转，并保持视频播放，但你不能简单地切换应用程序或锁定手机，并期望播放继续进行；你需要从播放窗口顶部的视频质量选项的最右侧选择“音频”，这样才能工作。

## 用醋做 YouTube 有什么弊端吗？

是的。只有一个，但此时:不幸的是，使用醋时没有 4K 或 HDR 的支持。根据你的设备，这可能不是一个交易破坏者，因为非 4K 设备在 YouTube 上的最高价格是 1080 便士。然而，对于 Mac 或兼容的 iPhones 和 iPads 来说，这是一个遗憾。但愿，一只恐龙能在未来为 4K·HDR 带来支持。
