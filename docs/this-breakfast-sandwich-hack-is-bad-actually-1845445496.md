# 事实上，这种早餐三明治很难吃

> 原文:[https://life hacker . com/this-breakfast-sandwich-hack-is-bad-actually-1845445496](https://lifehacker.com/this-breakfast-sandwich-hack-is-bad-actually-1845445496)

我住在西海岸，这意味着当我起床的时候，我的大多数同事已经比我早了个小时，我经常有几条消息在等我。昨天早上，其中一条是《每日邮报》文章 的链接，是我们的副主编乔丹·卡尔霍恩发给我的。

Watch

这篇文章吹嘘了一个“秘密培根和蛋卷烹饪技巧”，它将“永远改变你吃它们的方式。”诀窍？在硬皮面包卷上挖一个洞，放一些培根，在培根上打一个鸡蛋，在培根上撒上奶酪，然后合上一起烘烤。《每日邮报》声称，这种配置将“阻止蛋黄从面包中渗出。”

“这样真的好吗？，”乔丹问道。“我可以看到这个工作，”我回答，几乎没有醒来，“虽然如果你把它(三明治)切成两半[蛋黄]仍然会渗出来，哈哈。”但后来我喝了咖啡，又想了想，真的读了这篇文章。然后疑问出现了。

首先，我很确定阻止蛋黄渗出的唯一方法是把它煮到不再流动——这是物理问题！——但《每日邮报》似乎确信这是最简洁的解决方案。他们解释说:“这个简单的技巧将阻止蛋黄渗出整个面包卷，或者在吃的时候流到你的手上。”"相反，面包会吸收所有的蛋黄."

这不是我的经历。

我按照指示做了三明治。我从面包卷的顶部切下一片非常薄的面包，在底部挖了一个小洞，在里面放上咸猪肉。(我用了熏火腿，因为那是我吃的，而且看起来它能更好地防止生蛋白渗入面包。)然后，我往肉窝里打了一个鸡蛋，上面浇上奶酪，盖上盖子，355℉烤25分钟。(《邮报》的文章建议烹饪温度为190℃，但视频却建议烹饪温度为180℃；我看了录像。)

它看起来非常好，我会很高兴用叉子和刀子吃它(用一些辣酱)，就像我吃鸡蛋一样(因为它基本上就是这样——烤鸡蛋)。我通常会把早餐三明治切成两半，但那样会破坏“蛋黄会跑”实验的全部意义，所以我保持三明治完整，小心翼翼地咬了一口。

天气非常热。我又咬了一口(很烫)，然后又咬了一口。我大概吃了五口才吃到蛋黄(也有点热)，蛋黄从三明治里喷出来，溅到我脸上，我不喜欢。我没有照片给你，因为我不想在网上分享一张我脸上有鸡蛋的照片。面包并没有真正有机会吸收蛋黄，因为蛋黄喷出来了，还因为两片面包上都覆盖着火腿或奶酪，这两种脂肪成分可以防止水分(和液态蛋黄)进入。

除了脏乱之外，蛋黄和其他三明治配料的比例也不合适。我的前五口没有蛋黄，我不喜欢这样。松软的蛋黄的全部意义在于，当你切开三明治时，它会渗出来，让你在吃的时候蘸着三明治，享受每一口蛋黄般的美味。

我决定再做一个，看看我是否更喜欢切成两半。我想，这很好，但就像其他鸡蛋三明治一样乱，如果不是更乱的话。一半的蛋黄实际上从三明治的一侧弹出，可能是因为被油腻、滑溜的肉和奶酪包裹着。

再一次，我可以看到它作为刀叉工作得相当优雅，但它并不是对普通早餐三明治配置的改进。我的建议是拥抱蛋黄固有的混乱本性。或者，如果你不想吃乱七八糟的三明治，试试炒鸡蛋。炒鸡蛋不渗。
