# 如何快速判断你的啤酒杯是否真的干净

> 原文:[https://life hacker . com/how-to-quick-tell-if-your ' s beer-glass-is true-clean-1798734099](https://lifehacker.com/how-to-quickly-tell-if-your-beer-glass-is-really-clean-1798734099)

几年前，我决定获得导游资格——品酒师的啤酒版本。在学习期间，我学会了什么构成了一个“ [啤酒杯](https://www.craftbeer.com/craft-beer-muses/coming-clean-about-dirty-beer-glassware) ”玻璃杯。老实说，这是我以前从未考虑过的事情。干净的杯子就是干净的杯子，对吧？不对。

Watch

像清洁剂残留物和残留的油之类的东西在玻璃器皿上很常见，它们会改变啤酒的香气和味道——它们的存在意味着玻璃器皿并不真正干净。好消息是，有一个非常简单的方法来判断你喝的那个杯子是不是“干净的啤酒”

这种方法有一个缺点:如果你经常去酒吧，或者喜欢在普通的餐馆或机场酒吧喝一两杯啤酒，我建议你不要继续阅读。一旦你知道如何检测脏玻璃杯，你就无法避免这样做，我可以从个人经验告诉你，脏玻璃杯几乎无处不在。

好吧，你确定你想知道这个？真的确定吗？

好吧，我警告过你。如果你往啤酒杯里倒啤酒，气泡粘在杯壁上，你就能看出啤酒杯不“干净”。它看起来会像下面这张照片:

在不干净的杯子里，啤酒的泡沫也会消散得更快，而且/或者当你开始喝的时候，泡沫不会附着在杯子的边缘

该视频(针对调酒师)还解释了如何确保你有一个干净的啤酒杯——如果你是清理它的人——向前迈进。你可以通过把家里的啤酒杯浸入水中来判断它们是否干净。如果水均匀地覆盖在里面，那么你就很好。如果你的杯子里有水滴，它可能会更干净。

这些我都是几年前学到的，从那以后我想我已经退回了一瓶啤酒(杯子里满是气泡，味道真的很糟糕)。除非你的啤酒真的很难喝，否则我不会建议你大吵大闹。然而，如果你注意到这种情况在你经常去的酒吧发生了不止一次，你可能会想去别的地方喝酒。
