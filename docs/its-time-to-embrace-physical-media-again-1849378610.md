# 是时候再次拥抱物理媒体了

> 原文：<https://lifehacker.com/its-time-to-embrace-physical-media-again-1849378610>

终于到了承认流媒体应用和数字发行已经毁掉了大多数创意媒体产业的时候了，也许实体媒体一直是正确的选择。

好吧，这有点戏剧性。但是这并不完全是错的。

公平地说，流媒体应用并不都是坏事。流媒体服务和数字商店使我们几乎可以在我们拥有的任何设备上轻松访问我们最喜欢的节目、乐队和视频游戏。他们给了我们合法的方式来支持传统媒体，而不会屈服于贪婪的黄牛或可疑的盗版网站。但是，这种一开始是“切断绳索”并将其与有线电视公司和唱片公司捆绑在一起的方式只会催生一个新的企业霸主——一个不尊重其客户、其发行的媒体或其制作者的企业霸主。

## 流媒体和数字媒体的问题

人们可能会争辩说，像办公室里的 T2 离开网飞这样的烦恼，或者第一部人体模特 T4 的电影没有数码版本，而移动中的人体模特 2 已经有了 T7，这只是新的数码景观的一部分。

但是这种短暂性开始看起来更像是一个缺陷，而不是一个特征。在过去的一周里，我们了解到 [华纳兄弟发现毫不客气地从 HBO Max 应用](https://www.vulture.com/2022/08/hbo-max-movies-removed.html#:~:text=The%20six%20films%20in%20question,Anne%20Hathaway%20(what's%20Warner%20Bros') 中删除了电视节目，原因仅仅是它想停止向其创作者支付剩余款项——如果你每月支付 14.99 美元，期望 HBO Max 原创内容在 HBO Max 上实际可用，对不起。与此同时，数字视频游戏定期从数字货架上退市，使它们无法购买或重新下载，即使不可避免的服务器关闭导致多人模式——甚至整个游戏——[无法玩，即使在你已经购买并下载了游戏](https://lifehacker.com/how-to-back-up-your-old-playstation-games-locally-and-1846576681) 。

不仅仅是音乐、电影和游戏——由于流媒体和全数字平台，就连电子书和漫画也岌岌可危。看看 [对亚马逊最近漫画革新](https://www.ign.com/articles/comixology-update-controversy-amazon-digital-comics) 的反弹就知道了，这使得某些用户几乎不可能购买新漫画，并且由于不必要的布局变化，一些漫画和漫画变得不可读。

所有流媒体应用，不管是哪种媒体，如果你取消订阅，他们会取消你对他们免费图书馆的访问权，他们会经历突然的服务中断，或者他们会永久离线——更不用说 [流媒体商业模式臭名昭著地破坏了艺术家](https://lifehacker.com/which-music-streaming-service-is-the-most-ethical-1848508243) 和通过这些应用分发他们作品的创作者。

这些问题和更多的问题使得消费者越来越难以享受他们的购买，并使得媒体保存几乎不可能。

你知道什么*没有*那些问题吗？物理媒体。

这并不是说物理媒体不会出现问题。流媒体确实让媒体变得更容易获取，而且看似“无限”供应(只要还在供应)，而实体媒体只能以有限的数量存在，这就为公司和经销商的剥削创造了条件。此外，物理介质像所有物理物质一样会变质，可能会丢失、损坏或被盗。然而，就所有权、客户代理和媒体保存而言，物理媒体在几乎所有方面都优于数字媒体和流媒体。

## 流媒体有问题，但物理媒体不是已经死了吗？

我承认，购买硬拷贝可能不适合每个人或每次购买，但实体媒体对更多人来说是值得的，而不仅仅是业余爱好者、历史学家或小贩。不要让任何人愚弄你，让你认为物理媒体已经过时或无用——大多数媒体仍然接受完全物理发布(T2)。主要的电影都有蓝光版本，大多数整季的电视节目仍然在那些大的多光盘盒子里出售。他们可能只是更难找到。

电子游戏更棘手。许多新的“实体”游戏仍然需要在线连接，即使它们在技术上是离线的单人游戏，但许多新游戏完全可以在光盘上玩(或者在任天堂 Switch 的情况下，可以在卡带上玩)。购买旧游戏、 [、老式游戏机](https://lifehacker.com/the-best-ways-to-play-vintage-games-on-your-modern-tv-1849021233) 和稀有版本可能会非常昂贵，但像限量版游戏、超级稀有游戏和严格限量游戏这样的公司会印刷通常不会盒装发行的独立游戏的实体副本，在某些情况下甚至会重印旧游戏。实体视频游戏通常比数字游戏降价更快，有时在发布后几天或几周就大幅降价(任天堂游戏除外)。

CD 仍然是收集和聆听高质量音乐的绝佳方式，由于独立唱片公司和分销商以及收藏家市场，黑胶唱片 和盒式磁带等“过时”格式再次受到欢迎。虽然这些复古复古可能是作为潮人的信用货币开始的，但它们正迅速成为比流媒体更可靠的访问音乐的方式。此外，直接从艺人或唱片公司购买实体音乐是一种更好的财务支持方式，而不是通过流媒体赚取一小部分钱。

## 那费用呢？

实体媒体的另一个问题是成本:购买每一块新媒体的成本会不会超过你每月的流媒体账单？

当然，尤其是如果你买了很多新东西来看、玩或听——但这不一定是件坏事。就我个人而言，我发现我更多地使用我购买的媒体，而不是我流媒体或租赁的东西。流有一种趋势，让艺术作品像一次性的“内容”一样被我们的大脑加工成多巴胺。也就是说，流媒体或购买数字化的东西绝对有它的位置，当你不一定关心你正在观看/播放/阅读的内容，也不介意它突然消失时，这是一个值得的选择。但是，即使你看不出拥有某样东西和流媒体之间有什么区别，事实仍然是，物理媒体总是比数字媒体更可靠。

数字内容来来去去，服务器离线， [用户如果迁移到不同的应用](https://lifehacker.com/transfer-your-music-library-and-playlists-among-any-str-1849125094?rev=1656529558342) 就会丢失内容，但你客厅里的那一架子电影不会去任何地方。此外， [只要你有合适的设备](https://lifehacker.com/how-to-make-your-retro-console-games-look-better-on-a-m-1847855422) ，你就可以永远享受它，只要你想，而不必订阅新的服务，下载应用程序，或为你的无线连接大惊小怪。
