# 没有活塞时如何疏通马桶

> 原文：<https://lifehacker.com/how-to-unclog-a-toilet-when-there-is-no-plunger-1820623969>

这真的是 [噩梦](https://lifehacker.com/how-to-recover-from-a-nightmare-1837255377) 的东西:面对别人家的厕所情况，你做你的生意，冲水...什么都没发生。或者更糟的是，你冲水的时候，马桶里的水(以及里面的水)开始慢慢上涨，就像超大洪水一样。光是这一点就足以引发焦虑——但是你很快扫了一眼浴室，发现看不到一个活塞。不要惊慌或跳出窗外。请尝试这些策略。



### **在水槽里放热水**

这个技巧可以解决别人家或你自己家厕所堵塞的问题，但对于我们这些不在家时容易尴尬的人来说，它绝对是最有用的。为了让事情继续进行，YouTuber Aaron Bjorn 说你 [需要一些热水倒入马桶](https://www.youtube.com/watch?v=y2-MnmO_K3o) 。水池可能需要加热 T5，然后给你足够热的水以供使用。所以让它以最高温度运行。

### **伸手去拿洗手液或洗发水**

当水加热时，四处寻找一些液体肥皂。洗碗精是最理想的，但是你被困在浴室里，所以你可能在水槽下面找不到任何洗碗精。幸运的是，洗手液就足够了。即使是 s 汉普也能在紧要关头发挥作用。当你找到一些的时候，倒很多到马桶里。它将起到润滑剂的作用，并有望帮助去除导致堵塞的任何东西。

如果主持人是那种只独立制作过燕麦棒肥皂之类的东西的人，公寓治疗中心的苔丝·威尔森建议你去找 T4 的泻盐，或者甚至是浴弹。他们是那种把清洁用品放在水槽下或浴室壁橱里的人吗？试着找些漂白剂和洗衣粉。威尔逊说用两杯漂白剂和一杯洗衣粉。真的，任何能润滑或起泡的东西都可以。

### **将热水倒入马桶，然后等待**

现在你已经准备好润滑剂了，找一个杯子或容器——如果有必要的话，用他们的牙刷架——开始把热水从水槽倒入抽水马桶。同样，这里越热越好。但是先别冲！

一旦你添加了一加仑左右(或者在不冒溢出风险的情况下尽可能多)，YouTube 频道清洁我的空间说 是时候玩等待游戏了。坚持住，让肥皂和热水发挥它的魔力。

### **再做一遍，然后冲洗**

几分钟后，肥皂会渗入堵塞物，你会看到马桶水位开始下降。加入更多的液体肥皂和热水，然后再等几分钟。现在你可以做出判断，决定什么时候冲水。希望，当你这样做的时候，一切都会顺利。

### **最后一招:全部压下**

如果你尝试了热水加肥皂的方法，但马桶仍然堵塞，你可能不得不亲自处理堵塞。伸手去拿马桶刷，或者找一个可以解开的衣架——如果你真的绝望了，你甚至可以把手伸进垃圾袋——然后开始往排水孔里戳。是的，这会很恶心，但是你可以在堵塞物消失后清理干净。

也就是说，如果你已经到了用热水和洗手液冲几次都不起作用的地步，那就咬紧牙关，悄悄地去找你的主人要一个活塞。这可能会有点尴尬，但是你们两个和最终可能会更好。没有什么比满溢的马桶更能破坏派对了。
