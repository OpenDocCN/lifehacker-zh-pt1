## 什么是人性化的害虫控制？

基本上，人道的害虫控制——有时被称为“清除野生动物”——包括在不杀死它们的情况下将不想要的动物从你的家或院子里弄出来，以及首先防止它们进入。

根据国际人道协会， 的说法，一旦你意识到你有害虫问题，最好马上采取行动。首先要确定最初把小动物带到你院子里的食物来源，然后把它们移走。(这包括保护您室外的任何垃圾、回收和/或堆肥容器。)然后，使用不杀方法将动物驱逐出你的家，并密封任何洞或其他入口点让它们再次进入。

## 人道害虫控制是如何工作的？

虽然在防止不受欢迎的动物进入你的家或院子方面，你可以自己做很多事情，但清除组件有点棘手。例如，设置不杀动物的陷阱，然后重新安置动物可能看起来很有同情心，但根据不列颠哥伦比亚防止虐待动物协会(BC SPCA) 的说法，这既不是永久的也不是人道的解决方案。

在许多情况下，最好请专业人士来处理移除事宜。但这本身又提出了另一个挑战:确保你选择的公司使用的方法不会在过程中伤害动物。国际人道协会 [整理了一份选择野生动物搬迁服务的指南](https://www.humanesociety.org/resources/choosing-wildlife-control-company) ，包括确定该公司的做法是否真正人道和道德的问题。
