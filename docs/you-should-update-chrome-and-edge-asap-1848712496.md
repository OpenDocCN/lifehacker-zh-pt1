# 你应该尽快更新 Chrome 和 Edge

> 原文：<https://lifehacker.com/you-should-update-chrome-and-edge-asap-1848712496>

生活中有三件事是不可避免的:死亡、税收和软件更新。像 Chrome 这样的程序永远不会完美，安全漏洞不可避免地会被发现——而的目标只是在坏人利用这些漏洞之前，通过更新来粉碎这些漏洞。今天，Chrome 和 Edge 用户 [发现自己处于这样的境地](https://www.bleepingcomputer.com/news/security/emergency-google-chrome-update-fixes-zero-day-used-in-attacks/) 。



## 谷歌 Chrome 有什么问题？

3 月 25 日星期五，谷歌 [发布了 Chrome](https://chromereleases.googleblog.com/2022/03/stable-channel-update-for-desktop_25.html) (版本 99.0.4844.84)的新更新，并附带了一个变更日志，详细说明了一个安全漏洞。零日漏洞 CVE-2022-1096 被确定为高严重性，是 Chrome V8 JavaScript 引擎中的一个类型混淆漏洞。

当代码在没有提前检查的情况下读取或写入错误类型的内存时，会出现类型混淆缺陷。虽然这通常会导致软件失败，但它可以让坏人利用这个过程来运行他们自己的代码。

然而，关于这次更新有趣的是，谷歌缺乏关于漏洞利用的细节。该公司证实了利用 CVE-2022-1096 在野外存在攻击，但除此之外没有详细说明。谷歌显然希望有足够多的用户在分享更多关于这个零日漏洞的信息之前更新他们的浏览器，以防止更多黑客利用它。

## 为什么微软 Edge 用户需要更新？

虽然 Chrome 用户更新浏览器是显而易见的，但 Edge 用户也这样做也很重要。这是因为微软使用 Chromium 构建了他们的浏览器，Chrome 是由开源代码库驱动的。因为都是 Chrome 浏览器，Chrome 和 Edge 一样受到这个零日漏洞的威胁，需要立即更新。

## 如何更新 Chrome 和 Edge

幸运的是，检查两种浏览器的更新很容易。只需点击浏览器窗口右上角的三个点，前往“帮助”(Chrome)或“帮助和反馈”(Edge)，然后选择“关于谷歌 Chrome”或“关于微软 Edge”。在这里，您将能够看到是否有可用的更新要安装。如果是这样的话，那就尽快去做吧。
