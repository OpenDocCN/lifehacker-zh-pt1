# 使用此工具找出您可以在团队日程中远程工作的地方

> 原文：<https://lifehacker.com/use-this-tool-to-find-out-where-you-can-work-remotely-o-1848363514>

现在，许多在疫情期间可以选择远程工作的人被召回办公室，越来越明显的是，并非所有的雇主都同意这种安排。最初是为了帮助防止员工感染新冠肺炎病毒而制定的，一旦疫苗变得广泛可用，在家工作的政策就开始失效。

但是一些雇主已经走上了不同的方向，允许人们长期远程工作。(在某些情况下，因为他们在疫情期间停止租用办公场所。)

如果你也有这样的选择，你可能会考虑搬到一个新的城市，甚至是一个国家。无论是因为天气，离家人更近，还是生活成本更低(或者是多种因素的结合)，可能性似乎是无穷无尽的。但是如果你仍然被要求和你团队的其他人一样工作，你会想要选择一个可行的地方。以下是如何识别这些地点，并根据它们的时区缩小选择范围。

## 如何找到远程工作的新地点

与其做几个小时的研究，你可能想试试 Kayak 的新互动“寻找时区”工具 来帮助缩小潜在的搬迁地点。要使用它，请从下拉菜单中选择您工作的时区，然后浏览您的选项。

除了全球地图，你的选择还有列表形式的，(莫名其妙地)从最差到最好排列，首先基于他们的时区，以及你是否能在那里生活并保持你目前的工作时间表而没有太多麻烦。名单上的目的地包括时区/时差、边境状况和新冠肺炎疫苗接种率等信息。

这个工具中总共包含了 111 个国家——选择这些国家或者是因为它们是 Kayak 网站上搜索次数最多的目的地，或者是因为它们已经宣布了针对远程工作者的强化签证政策。
