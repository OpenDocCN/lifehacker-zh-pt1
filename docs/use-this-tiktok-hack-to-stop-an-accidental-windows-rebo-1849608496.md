# 使用这个抖音黑客来阻止意外的 Windows 重启

> 原文：<https://lifehacker.com/use-this-tiktok-hack-to-stop-an-accidental-windows-rebo-1849608496>

当我们的电脑做我们不希望它们做的事情时(看着你，突然的软件更新)，这令人沮丧。然而，唯一比流氓电脑更令人讨厌的是，当问题是我们自己该死的错误时。当您意外重启计算机时，就会出现这样一个自我强加的问题。当我试图关闭我的电脑，而是点击“重启”按钮时，有时会发生这种情况。但是，下一次我将使用这个抖音提示来中止这个过程，而不是等待计算机重新启动，这样我就可以关闭它。

Watch

[创造者 jackstechtopia 最近发布了一个抖音](https://www.tiktok.com/@jackstechtopia/video/7133361248724389162?_r=1&_t=8WB9EzvUoxS&is_from_webapp=v1&item_id=7133361248724389162) 分享了一个快速取消重启的方法。如果您的 PC 刚刚开始重新启动，包括显示带有加载轮的蓝屏，您可以按住 ESC 键来停止它。几秒钟后，您会看到蓝色的重启屏幕消失，取而代之的是您的主屏幕。从这里，你可以继续使用你的电脑，或者，如果你打算完全关闭它，关闭它。

这种方法对我来说是新的，所以我在运行 Windows 11 的笔记本电脑上进行了测试。你瞧，按下 escape 键取消了重启，就像 jackstechtopia 一样。从视频上的评论来看，我不是唯一一个不知道这个小技巧的 Windows 用户。(我猜叫“逃”键是有原因的。)
