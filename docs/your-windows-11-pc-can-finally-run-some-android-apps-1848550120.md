对于Android用户来说，这是一个同步他们的智能手机和PC应用程序的好方法，但并不是所有的东西都会交叉。亚马逊应用商店仍处于预览阶段，目前只有少数应用得到官方支持。同样，由于Windows 11使用亚马逊应用商店而不是谷歌Play商店来下载安卓应用，你下载的任何高级应用、或你进行的任何应用内购买、都将绑定到你的亚马逊账户而不是你的谷歌账户。当然，大多数安卓设备使用谷歌a 账号。

也就是说，Windows 11的亚马逊应用商店预览版包括1000多种应用和游戏，包括亚马逊Kindle、Audible、Comixology、华盛顿邮报应用等。这些应用程序经过测试，证实可以在运行英特尔或AMD硬件的Windows 11电脑上运行，未来还将增加英特尔或T2的应用程序。

## 如何通过亚马逊应用商店在Windows 11中安装和运行安卓应用

要在Windows 11 PC上安装Android应用，首先要更新Windows 11和微软商店。

1.  在你的PC上，打开**设置>更新&安全> Windows更新。**
2.  点击**“检查更新”**
3.  如果有可用的更新，请按照屏幕上的说明进行安装。
4.  一旦你的电脑处于最新状态，请在你的电脑上打开Microsoft商店。
5.  转到**“库”**选项卡，选择**“获取更新”**
6.  让更新安装。

安装了所有必要的更新后，你现在可以下载Windows 11的亚马逊应用商店预览版了。

1.  打开微软商店
2.  搜索**“亚马逊应用商店。”**
3.  打开亚马逊Appstore预览登陆页面，选择**“安装”**
4.  如果出现提示，请安装Windows的Android子系统。
5.  安装后，启动Amazon Appstore并登录您的Amazon帐户，或者创建一个新帐户。
6.  你现在可以在亚马逊Appstore预览版中浏览Android应用。一些应用程序也在微软商店中列出。

要在Windows 11中运行Android应用程序，只需像打开任何其他程序一样，从桌面或开始菜单中打开它。该应用程序将在自己的窗口中打开。

## 在Windows 11上安装Android应用的其他方式

通过亚马逊Appstore在Windows 11上下载Android应用程序很容易，但这不是在Windows上安装Android软件的唯一方式——你也可以在Windows 11上 或 [通过BlueStacks](https://lifehacker.com/how-to-run-android-apps-on-windows-11-1847819230) 或其他仿真软件运行apk。这些替代方法甚至可以让你安装没有官方支持的应用和游戏(尽管它们是否能正常运行是另一回事)。
