# 孩子们整个夏天都可以免费用餐

> 原文:[https://life hacker . com/kids-can-get-free-footprints-all-summer-1846448101](https://lifehacker.com/kids-can-get-free-meals-all-summer-1846448101)

美国农业部 [延长了夏季](https://www.usda.gov/media/press-releases/2021/03/09/usda-extends-free-meals-children-through-summer-2021-due-pandemic) 的疫情免费餐计划，至少持续到2021年9月30日。如果你还没有参加，你可能会惊讶地发现这些餐食是为所有学龄儿童准备的，而且是100%免费的，不需要任何证明文件。

正如美国农业部所说:

> 今天延长的豁免允许安全膳食分发网站免费为所有儿童服务，无论收入如何。此外，弃权:
> 
> *   允许通过[【SFSP】](https://www.fns.usda.gov/sfsp/summer-food-service-program)和 [无缝夏季选择(SSO)](https://www.fns.usda.gov/sfsp/seamless-summer-and-other-options-schools)——统称为“ [夏季膳食计划](https://www.fns.usda.gov/sfsp/summer-meals-toolkit)”——在所有地区免费提供膳食；
> *   允许在正常要求的团体环境和用餐时间之外用餐；和
> *   允许父母和监护人为他们的孩子取餐，包括批量取餐，以支付多天喂养儿童的费用。

联系你的学区，了解他们如何提供或分发膳食。例如，在我们当地的学校，家长每周都要填写一份在线表格，说明他们为多少孩子要求用餐，以及这些孩子是否完全在偏远地区，或者他们是否已经在学校吃了一些饭。然后，在约定的那一天，我们开车去学校，自助餐厅的工作人员通过窗户递给我们一个大包。饭菜包括每天典型的学校午餐和早餐的所有组成部分:小盒牛奶、袋装胡萝卜、可用微波炉加热的主菜等等。

通常情况下，家庭必须有资格享受免费或低价餐，学校需要根据特定的规则和用餐时间提供膳食，但在疫情，美国农业部已经放弃了一些规则，以消除障碍，帮助更多的孩子获得食物。豁免因州而异， [如果你好奇的话可以在这里查找](https://www.fns.usda.gov/ffvp/covid19-waiver) ，但是最好的办法是联系你孩子的学校。

不仅是学校，教堂和社区中心也会提供或分发暑期餐。前几年，这些夏季餐饮地点只能设在低收入地区，但豁免允许更多的地点。美国农业部有一个 [夏日大餐寻找器](https://www.fns.usda.gov/meals4kids) ，它会帮你找到你附近的位置。
