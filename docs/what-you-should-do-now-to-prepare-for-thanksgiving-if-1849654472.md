# 你现在应该为感恩节做些什么准备(如果你焦虑的话)

> 原文：<https://lifehacker.com/what-you-should-do-now-to-prepare-for-thanksgiving-if-1849654472>

人们说财富偏爱勇敢的人，但我认为它偏爱焦虑的人(像我一样)，尤其是在活动策划方面。感恩节不就是一个事件吗？我们当中“非常注重细节”的人倾向于提前“太远”地计划事情，但是额外的一点准备通常最终会有回报。如果我说的让你点头，我有一些建议，你可以现在就采取行动——在十月中旬——确保你的感恩节如你所愿。



## 现在是检验食谱的时候了

这是一个显而易见的问题，但是火鸡日的早晨(甚至是前一天)不是尝试新的馅饼食谱、给你的经典面包卷“旋转”或者更换馅料的时候。现在就尝试所有这些吧。你会带着额外的自信走进这个大日子，你会在研究的幌子下吃到今天的馅饼。你也可以多做一些馅饼皮，然后把它们扔进冰箱，因为你永远不会有太多的馅饼皮，尤其是如果你像我一样，喜欢弄乱馅饼皮。(我就是不尊重他们！)

## 发酵某物

发酵需要一段时间，但我们有整整六(6)周的时间，直到火鸡时间，这给你足够的时间来制作一些 [发酵蔓越莓蜂蜜](https://lifehacker.com/you-should-ferment-some-cranberries-in-honey-immediatel-1847991936)[发酵大蒜蜂蜜](https://lifehacker.com/make-fermented-garlic-honey-with-just-three-ingredients-1792567207) ，或者一些 [盐曲](https://lifehacker.com/shio-koji-helps-your-food-be-its-best-self-1826640401) 来涂抹在 [节日烤肉](https://lifehacker.com/brine-your-holiday-meats-in-shio-koji-1848133750) 或蔬菜上(作为一种时髦的味觉清洁剂)。你也可以利用这段时间制作季节性的康普茶——也许是蔓越莓、苹果或南瓜。(南瓜香料红茶菌是个东西吗？我打赌是的。)

## 去旧货店买餐具吧

快，在别人想到之前，去你当地的 Goodwill 或类似 Goodwill 的机构 看看架子上的盘子、餐具、肉卤盘、便宜的酒杯、砂锅菜、大汤锅、桌布和咖啡杯(用来盛咖啡和馅饼)。我在 Goodwill 找到了很多非常漂亮的东西，价格很低，所以在支付全额零售之前，至少检查一下你当地的旧货店。

你还应该去 [餐厅供应店](https://lifehacker.com/why-the-restaurant-supply-store-will-be-the-hero-of-you-1847982785) 找刀具、砧板、磨刀器和珩磨杆、烘焙设备，还有 [外卖容器](https://lifehacker.com/store-all-your-leftovers-in-soup-containers-1845301624) (用来装剩菜)，但那不太可能被挑走。

## 储备受欢迎的不易腐烂的东西

我不是一个关注“供应链”的人，但它看起来充其量是不稳定的。如果你知道你将需要罐装南瓜，或 Jiffy 烘焙混合物，或冷冻糕点，或法式油炸洋葱，或任何其他不易腐烂的感恩节主食，那就去你平常的商店买，然后藏起来。这将有助于分散大餐的成本，你不必担心讨厌的供应链在最后一刻破坏你的菜单。
