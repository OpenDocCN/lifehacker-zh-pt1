# 用这个免费程序升级你的园艺技能

> 原文：<https://lifehacker.com/upgrade-your-gardening-skills-with-this-free-program-1846538654>

园艺是关于经验的，这使得开始不仅仅是有点吓人。如果你以前没有做过，即使是像种西红柿 这样简单的事情也会带来一百万个问题:我需要一个高架床吗？种子应该什么时候入土？这是太亮了，还是不够亮？等等——这种气候下西红柿还能生长吗？

如果这些问题让你想起了什么，你应该熟悉一下当地大学的推广服务。合作推广服务，也称为推广计划，基本上是农业的社区推广和教育计划。美国农业部(USDA)和公立州立大学之间的合作，这些项目自 1914 年以来一直以这样或那样的形式存在。根据古老的《农民年鉴》 记载，美国农业部希望“为全国农民提供当地专家关于农业和耕作的建议”，并认为州立大学是实现这一目标的最佳途径。他们是对的——这个项目立即取得了成功，现在所有 50 个州都有推广服务。

你不必是一个农民，甚至不必住在你所在州的大学附近，就可以利用它的推广服务。大多数项目在多个县都有物理办公室，你可以使用美国农业部的赠地大学网站目录 轻松定位。一旦你找到你当地的办公室，看看他们的网站，看看他们正在做什么。

一开始可能会有些难以接受。推广服务网站，作为一个规则，绝对塞满了有用的信息，使他们成为你能想到的任何项目的完美参考。但它们不仅仅是当地园艺技巧的维基百科——它们主要是活动日历。记住，推广服务被发明的唯一目的是教社区中的人们如何做酷的事情。在互联网出现之前，他们专门通过讲座、课程、研讨会和一般的社区拓展来做这件事，所有这些他们今天仍然在做。这意味着有一个推广服务班，涵盖你想学的一切，从 [建造鸡舍](https://lifehacker.com/build-a-chicken-coop-for-your-free-range-flock-1686218913) 到当地农业历史，再到获得你的园艺大师证书，所有这些都是免费的。

列出你能从推广服务中学到的每一件事是不可能的，但是可以说，如果你对你所在地区的种植、饲养、收获、烹饪和保存食物有任何疑问，它们应该是你的第一站。在那里工作的人都是经验丰富的当地人，知道所有的答案，他们会很乐意与任何询问的人分享。
