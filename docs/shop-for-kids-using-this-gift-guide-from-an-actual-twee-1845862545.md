# 使用这份来自真实青少年的礼物指南为孩子们购物

> 原文:[https://life hacker . com/shop-for-kids-using-this-gift-guide-from-an-an-actual-twee-1845862545](https://lifehacker.com/shop-for-kids-using-this-gift-guide-from-an-actual-twee-1845862545)

你购买节日礼物的时间已经不多了。如果你一直犹豫不决，不知道该给你单子上最后几个孩子买什么，我儿子可以帮你解决。因为有谁能比一个把玩具、娱乐和送礼物都看得很重的孩子更好地建议你给你一年没见的侄子或你最好朋友的女儿买什么呢？

我10岁的儿子按年龄分类挑选，从你生命中最小的开始。

他知道他需要从《小爱因斯坦》音乐剧开始，因为这是他小时候的最爱，也是朋友或家人有了孩子时我们的必备礼物。(亚马逊告诉我，我们已经买了这个家伙11次，我们很久以前就称他为“奥托”。)

孩子说:“章鱼奥托的创造者知道孩子们喜欢什么。这有助于孩子们学习颜色，它有一张看起来很傻的脸，这很神奇。孩子们喜欢和他互动。”

我儿子为一个蹒跚学步的小孩挑选的是 [伟易达的强光足球](https://www.amazon.com/VTech-Bright-Lights-Soccer-Ball/dp/B0765ZPXMC/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/shop-for-kids-using-this-gift-guide-from-an-actual-twee-1845862545&asc_source=&dchild=1&keywords=VTech Bright Lights Soccer Ball&qid=1607715489&sr=8-1&tag=kinjalifehackerlink-20) ，这是他去年为他表弟的第一个生日挑选的，并且一炮而红。他是这样说的:

“这是一件有助于教幼儿如何在生命早期玩一种非常流行的游戏的事情。它非常独特，有一张脸，而且会发光。”

有什么不喜欢的？

我儿子说，每个学龄前儿童需要的是一大袋T2和T3。这是一个不容忽视的经典选择。正如他所说:

“我记得在幼儿园时，我总是喜欢玩这些东西，因为它们有点像乐高玩具，但它们没有那么危险，也没有那么难。你可以和他们一起建造许多不同的东西，如果他们有一个哥哥或表弟，他们可以一起建造更大的东西，比如一座城堡。”

我儿子记得五岁到六岁是他和我的侄女们(她们都和他同龄或者稍大一点)第一次真正开始骑着他们的 [电动滑板车](https://www.amazon.com/Razor-Scooter-Green-Frustration-Packaging/dp/B01EN6LY3I/ref=sr_1_7?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/shop-for-kids-using-this-gift-guide-from-an-actual-twee-1845862545&asc_source=&crid=3F2SIPBE23KJ8&dchild=1&keywords=razor+scooter&qid=1607716155&sprefix=razor+sc,aps,170&sr=8-7&tag=kinjalifehackerlink-20&th=1) 到处跑的年龄，他相信这个年龄的每个孩子都应该有这样的经历。

“每当我去我奶奶家，我和我的堂兄弟会在附近骑着我们的滑板车，”他告诉我。“骑起来很有趣；感觉就像你在天空翱翔。”

如果你礼物清单上的孩子大约7到8岁，他们还不喜欢神奇宝贝，现在是时候了，我经验丰富的训练师宣布:

“那是我第一次接触神奇宝贝卡片。在那个年纪，人们真的会觉得有特异功能的神兽真的很酷，然后你会一辈子都钻进去。我会从太阳和月亮系列开始，因为它有很多独特的神奇宝贝，但也很好。”

我建议你从 [这个训练套件](https://www.amazon.com/Pok%C3%A9mon-TCG-Trainer-Lycanroc-Alolan/dp/B01N6TC9MD/ref=sr_1_7?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/shop-for-kids-using-this-gift-guide-from-an-actual-twee-1845862545&asc_source=&crid=3L09O9P98S8UQ&dchild=1&keywords=pokemon sun and moon starter deck&qid=1607716634&sprefix=pokemon sun and moon starter action figures,aps,158&sr=8-7&tag=kinjalifehackerlink-20) 开始，它包括副牌、一张游戏垫、伤害指示物、条件标记和一枚游戏币。

考虑到这基本上是我儿子在过去两年里想要的一切，对于这个年龄范围，他坚定地建议你选择乐高套装，我并不感到惊讶。

如果你知道孩子喜欢什么，你可以根据他们的兴趣和/或他们还没有的东西来选择一套。如果你在黑暗中拍摄，他认为乐高星球大战不会出错。具体来说，选择一艘看起来很酷的船，比如这艘价格合理的 [抵抗A翼星际战斗机](https://www.amazon.com/LEGO-Star-Wars-Wing-Starfighter/dp/B07Q2TQ48C/ref=sr_1_3?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/shop-for-kids-using-this-gift-guide-from-an-actual-twee-1845862545&asc_source=&dchild=1&keywords=lego star wars&qid=1607717823&sr=8-3&tag=kinjalifehackerlink-20) 来自*天行者崛起。*

“飞船是一件大事，”我的孩子说，“因为孩子们喜欢通过发挥他们的想象力来进行虚拟的太空战。他们甚至可以改变他们的游戏方式；他们不必把它仅仅当作星球大战的东西来玩。他们可以把它用于任何与《星球大战》无关的太空战。”

我儿子说，11-12岁的孩子想要电子游戏。当然，你选择哪种游戏将取决于他们拥有什么样的游戏系统以及他们还没有什么。他有一个任天堂Switch，所以他特别推荐 [*超级粉碎兄弟终极*](https://www.amazon.com/Super-Smash-Bros-Ultimate-Nintendo-Switch/dp/B01N5OKGLH/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/shop-for-kids-using-this-gift-guide-from-an-actual-twee-1845862545&asc_source=&crid=2P36SQ1IPHD1U&dchild=1&keywords=super smack bros ultimate&qid=1607718245&sprefix=super smac,aps,167&sr=8-1&tag=kinjalifehackerlink-20) ，这是一个他和他的那些堂兄弟可以一起玩几个小时的游戏。(你也会想玩的；挺好玩的。)

“在这一点上，这个年龄的孩子唯一想做的事情就是一直玩电子游戏，”他说。“他们没有太多其他选择，尤其是在2020年。他们目前能做的其他事情不多。”

当你是对的，你就是对的。

他希望我选择最后一个，因为我比他“更有青少年经验”，但我鼓励他提前考虑几年后他可能会喜欢什么，我认为他的建议实际上是正确的:

“嗯，到这个时候，他们并不真正喜欢玩具，”他说。“所以我会给他们买些东西来装饰他们的卧室。我的意思是，很多父母现在可能正在为他们的青少年寻找礼物，他们会说，“唉，我不知道该给他们买什么——他们现在不喜欢任何东西。”所以为他们的房间准备些东西会很好。也许是一盏新灯或者某种墙壁装饰品，比如贴纸和海报。"

我将通过建议这些装饰灯来帮助他完善这个想法，这些装饰灯将为任何一个青少年的卧室增添一点点光彩。
