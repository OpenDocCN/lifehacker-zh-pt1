# 不，夜间模式没用

> 原文：<https://lifehacker.com/no-night-mode-wont-help-1846830398>

当天色开始变暗时，你可能会注意到你的手机屏幕变成了更橙色的色调，这有助于你为睡眠做准备。该理论认为，我们的身体预计在白天只能看到蓝光，所以晚上阻挡蓝光应该有助于我们放松，并在上床睡觉时感觉更累。

Watch

睡前使用手机可能会干扰你的睡眠，这是真的，但是颜色的改变真的能逆转这种影响吗？不太可能，根据 [的新研究](https://www.sciencedirect.com/science/article/abs/pii/S2352721821000607?dgcid=coauthor) 的说法，这支持了专家们多年来的说法。

### 我们最近学到了什么

在这项新的研究中，大学生被分配到三种情况中的一种:在睡前一小时内收起 iPhone，在启用夜班的情况下使用 iPhone，或者在没有夜班的情况下使用 iPhone。通过活动跟踪器测量，这三组人的睡眠质量没有差异。

研究人员写道:“综上所述，我们的研究结果与之前的文献相结合，不支持蓝光过滤器(如夜班)作为改善睡眠的干预措施，无论是通过睡眠特征(如睡眠持续时间，睡眠开始潜伏期，褪黑激素开始或快速眼动睡眠时间)来衡量。”

然而，大学生睡眠不足是出了名的，所以研究人员想知道参与者是否太累了，以至于无论他们用手机做什么都没有睡着的问题。当他们观察睡眠时间超过平均水平的人时——换句话说，那些不太可能被剥夺睡眠的人——他们发现睡前使用手机的人和不使用手机的人之间的睡眠质量有所不同，但是夜班的使用似乎又一次没有造成差异。

### 我们已经知道的

夜班(及其在机器人和计算机上的等价物)是基于科学的，这是有道理的，但当它被引入时，还没有在现实世界中经过严格的测试。 [【这里】这是一篇来自 2016 年](https://lifehacker.com/will-night-modes-on-my-smartphone-or-tablet-actually-1766261703) ，的 Lifehacker 的文章，例如当这些模式刚刚开始流行的时候，we 指出，虽然科学支持蓝光可能干扰睡眠的观点，我们不知道将你的显示器染成橙色是否真的能解决问题。

今天，蓝光已经得到了更坏的名声，但支持它的科学还没有真正出现。验光师和眼科医生 [一直试图告诉我们](https://vitals.lifehacker.com/does-blue-light-actually-affect-your-health-1839035806) 来自屏幕的蓝光没有任何严重的有害健康影响，他们不建议将夜间模式作为干扰我们睡眠的智能手机的万灵药。
