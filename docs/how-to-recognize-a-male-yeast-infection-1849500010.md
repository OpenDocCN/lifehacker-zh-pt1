# 如何识别男性酵母菌感染

> 原文：<https://lifehacker.com/how-to-recognize-a-male-yeast-infection-1849500010>

你可能不知道有阴茎的人会感染酵母菌，因为非处方救济的广告很容易针对没有阴茎的人。 P 预防建议通常针对阴道，酵母菌感染治疗通常被归类为女性健康。公平地说，阴道感染了大量的酵母菌，但是值得知道的是任何人都可能感染，以及如何识别它。



## 什么是酵母菌感染？

不管你是谁，你带的是什么设备，你都可能有一些常见的真菌，叫做白色念珠菌。 [据 Web MD](https://www.webmd.com/men/can-men-get-yeast-infections) 报道，现在你的口腔和消化道里可能就有一点，更不用说你皮肤的其他湿润部位了。这种真菌通常也存在于阴道中。虽然它在大多数情况下不会引起问题，但是如果它在任何一个地方生长过多，就会导致酵母感染。

当然，这可能发生在真菌存在的任何地方，比如你的嘴——当这种情况发生时，它被称为口腔鹅口疮。当它发生在阴茎的尖端时，它被称为龟头炎。

## **阴茎酵母菌感染的症状有哪些？**

根据 [梅奥诊所](https://www.mayoclinic.org/male-yeast-infection/expert-answers/faq-20058464#:~:text=Yes%2C%20men%20can%20get%20yeast,on%20skin%2C%20especially%20moist%20skin.) 和 [网站 MD](https://www.webmd.com/men/can-men-get-yeast-infections) ，如果你认为自己可能患有酵母菌感染，请留意以下症状:

*   阴茎上潮湿的皮肤，可能有厚厚的白色物质积聚在皮肤褶皱中
*   阴茎皮肤上闪亮的白色区域
*   阴茎顶端发红、发痒或烧灼感
*   类似农家干酪的分泌物，甚至闻起来不舒服，像“面包”
*   阴茎顶端肿胀
*   拉回包皮有困难
*   排尿困难
*   难以勃起或保持勃起

当然，这些症状中的大多数也很容易是其他疾病的症状，所以如果你正在经历其中的一些症状，请密切关注，并记住以下风险因素。

## 你的阴茎是如何感染酵母菌的？

龟头炎在未割包皮的阴茎上更常见。据 Web MD 称，还有其他一些风险因素:

*   长期服用抗生素
*   患有糖尿病
*   超重
*   免疫系统弱的
*   清洁自己有问题
*   对肥皂、香水或化学物质敏感的

最后，最迫切的问题的答案:你会通过与患有酵母菌的人发生性关系而感染酵母菌吗？是的，你可以。所以，如果你和患有阴道酵母菌感染的伴侣发生性关系，真菌的过度生长也可能会传播到你的鸡鸡上。

## **龟头炎的诊断和治疗**

诊断阴茎酵母感染是相当容易的，医学博士:你的医生会询问你的症状，看一看，并可能拭子尖端进行实验室测试。重要的是，你要避免试图诊断自己，因为酵母感染的许多症状与其他性传播感染的症状相似。

你的医生可能会推荐一些非处方抗真菌霜剂，但他们也可能会开一种或类固醇霜剂或口服药物。在反复感染的情况下，他们可能会建议包皮环切术、更好的卫生习惯、更好地管理糖尿病、减肥或避免使用某些肥皂或化学品。

感染应该在开始治疗的三到五天内消失。为了帮助防止这种情况再次发生，确保每次淋浴或做爱时清洁和干燥阴茎，使用无味的卫生产品，穿宽松的棉质内衣。与你的伴侣交流他们的健康状况，如果他们有酵母菌感染，确保在性交时使用避孕套，直到他们的感染被清除。
