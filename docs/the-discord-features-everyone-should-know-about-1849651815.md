# 每个人都应该知道的不和谐特征

> 原文：<https://lifehacker.com/the-discord-features-everyone-should-know-about-1849651815>

Discord 不再仅仅是游戏玩家的天下，它正在成为被 Slack 甚至 Instagram 等其他平台挤出的社区的避风港。Discord 的免费计划给了你足够的权力来运行一个大型社区，有节制，有规则，等等。但 Discord 最糟糕的一点是用户界面:Discord 的用例可能已经进化了，但界面仍然停留在玩家的土地上。如果你觉得黑暗的背景、自动播放的表情符号、动画和小文本令人讨厌，这里有一些辅助功能设置，你可以调整它们，让眼睛看起来不协调。



Discord 在设置中有一个单独的辅助功能部分，但较小的生活质量功能分布在整个设置应用程序中。

## 从 Discord 的辅助功能设置开始

在 Discord app 中点击**档位**图标，打开**设置**。从边栏中，选择“**辅助功能”。**"以下是您应该更改的设置:

*   饱和度:降低 UI 元素的颜色饱和度，让它们看起来更酷。
*   **角色颜色**:如果你愿意，你可以选择完全不显示角色颜色。
*   **启用减少运动**:用这个开关处理所有的嗖嗖声动画。当您启用此功能时，Discord 还会在 GIF 被聚焦时自动禁用 GIF 播放，并且动画表情也被关闭。
*   **贴纸**:为贴纸选择“**永不动画**”选项，稍后感谢我们。

## 调整背景和文本大小

辅助功能之旅的下一步是**设置**中的**外观**部分。

*   **主题**:如果不喜欢深色背景，切换到主题的**灯光**选项。“与电脑同步”或“自动”是任何希望自己的 Discord 应用程序自动匹配当前系统主题的人的好选择。
*   **深色侧边栏**:启用此功能，当 Discord 的其余部分为浅色主题时，侧边栏保持深色模式。这个设置会让 Slack 用户有宾至如归的感觉。
*   **信息显示**:确保设置为“**舒适**，这样线程更宽敞。
*   **聊天字体缩放**:您可以在这里精确调整文本大小、消息组之间的间距以及整体缩放级别。为了更好的阅读体验，即使你没有视力障碍，也要将所有东西放大至少 20%。对我来说，18px 的字体和 110%的缩放比例效果最好。

## 选择文本和图像的显示方式

转到“**文本和图像**”部分，调整 Discord 处理图像、链接和自动嵌入媒体的方式。如果你不想看图片、视频和搞笑猫，你可以禁用“**当发布为聊天链接**功能来禁用预览。

如果您有视觉障碍，我们建议您启用“**带图像描述**”选项。这将提供在 Discord 服务器中共享的映像的描述。从**嵌入和链接预览**部分，您可以完全禁用媒体嵌入。

## 控制您的语音和视频设置

最后，转到“**语音和视频**部分。Discord 有一个名为**的语音活动**的功能，它会在你说话时自动取消你的静音。如果您认为不和谐的声音在您不希望的时候出现，您可以使用滑块调整输入灵敏度。但是如果你觉得这太麻烦，尝试切换到“**按键通话**”选项，当你按下快捷键时，它会解除静音。

在高级设置中，您可以启用“**噪音抑制**”、“**回声消除**”和“**噪音降低**”等功能来改善您的语音聊天。
