# 如何将洋葱切片以保持其形状

原来你切洋葱的方向很重要。你破坏的细胞壁越多，洋葱就会变得越软，尤其是在烹饪后:

> 当你切开这些层时，你将它们分成小块，煮熟后变软——非常适合做成光滑的汤或酱。然而，当你沿着纤维的方向切片时，你保留了它们的结构。这意味着即使烹饪很长时间，洋葱也会保持其形状，直到变成深棕色，味道鲜美，这使它们成为法式洋葱汤或焦糖洋葱的完美选择。

如果你生吃洋葱，比如汉堡或沙拉中的洋葱，或者将它们融化成令人舒服的汤，这并没有太大的区别，但如果你想用可辨别的洋葱片制作美味的汤或炖菜，你应该垂直切片(从一极到另一极)。

这样做并不难。从切掉洋葱的两端开始，然后将洋葱从一极到另一极切成两半。如果你看到同心圆(环)，你已经在错误的方向切片。剥下洋葱皮，然后沿着洋葱的自然纹路切片。这样做可以减少洋葱受到的细胞损伤，这将有助于洋葱保持更大的弹性和更好吃。
