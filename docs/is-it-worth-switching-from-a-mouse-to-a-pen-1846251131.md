# 从鼠标换成笔值得吗？

> 原文：<https://lifehacker.com/is-it-worth-switching-from-a-mouse-to-a-pen-1846251131>

从我记事起，我就是一个鼠标爱好者——从未动摇过。然而，如果你想完全抛弃你的鼠标呢？这是 Lifehacker 读者 **James** 为本周的 [**Tech 911**](https://lifehacker.com/c/tech-911) 提出的问题。他写道:

> 我尽我所能用我的罗技鼠标签下我的名字。我已经 80 多岁了，觉得写信很难。我渴望尝试 Pentype 鼠标，以便在签名或绘制草图时为我提供更好的控制。
> 
> 我的系统是 Windows 10，64 位。你能推荐一款鼠标或者给我一个关于这种鼠标的评论吗？



### 钢笔 m ouse 还是 ddrawing pad？

我承认，我已经很多年没有看到对真正的笔式鼠标的评论了。它们总是给我一种廉价的感觉，30 美元的小玩意你可能会在百思买的深货架上找到，也许是上面贴着“在电视上见过”的标签。它们不是一种非常传统的使用系统的方式，看起来非常笨重，而且你必须一直处理无线充电。 *Eeech* 。

我说的是一个非常具体的产品类型，尽管、就像这个:

很多人用笔与电脑互动，但方式略有不同。我建议你尝试一款全功能的绘图板，一款你可以用手写笔进行交互的绘图板，尽管一开始用笔鼠标可能会更便宜(T2)。

专业人士？触控笔永远不会没电。它仅仅是你的手与平板电脑互动的延伸，这是手术的真正内容。像其入门级产品 [Wacom CTL4100](https://smile.amazon.com/dp/B079HL9YSF/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/is-it-worth-switching-from-a-mouse-to-a-pen-1846251131&asc_source=&tag=kinjalifehackerlink-20) 这样的绘图平板电脑获得了大量好评，而 Wacom 是一个值得信赖的品牌，所以你知道笔和 PC 之间的任何关系都可能得到解决，这比我能说的一些 20 美元“笔鼠标”的随机制造商要多

缺点是什么？好吧，你会点击，而不是点击，所以可能需要一点时间来适应。也就是说，我甚至听说有些人采用混合方法——将他们非主要的手放在鼠标上并使用它来点击，同时使用笔+绘图板组合来浏览窗口、移动应用程序等等。

这也是价格。前面提到的 CTL4100 售价高达 80 美元。这只是我看到的各种笔鼠标价格的四倍左右； h 然而，我认为这次购买就像购买一把优质的菜刀。当然，你可以从亚马逊买到 20 美元的刀来完成这项工作——就像我过去做的那样——但是质量通常很差，它们的长期预测也不是很好。(当然，尽管如此，当你丢失、损坏或生锈时，没什么大不了的。)

我假设你和你的电脑互动相当频繁，而且鉴于你特别提到签名和素描，我认为 CTL4100 是完美的选择。我之前也没有提到，它有很多内置的压力敏感性，以防你想探索它作为你涂鸦的一部分。笔式鼠标？不确定它是否会有那种内置功能，因为它更像是一个输入设备，而不是一个理想的绘图工具。

所以这是我的建议:带着绘图板去。我(和许多其他人)喜欢 Wacom，但我相信还有更好的选择。不过，考虑到你的需求，我不会买几百美元的东西——除非数字艺术是你日常生活的重要部分。

如果你购买了一个绘图板，请确保你调出了 Windows 10 的**笔& Windows Ink** 设置，在那里你还应该看到一个你想要启用的关键选项:“**允许我的笔在传统应用程序中充当鼠标。”**我自己没有笔/绘图板，所以我不能确定这是它的名字(或者它是否仍然存在)，但我知道这是以前确保你可以使用手写笔与系统范围内的应用程序进行交互的一种方式。还有一种可能是，你的绘图板附带的任何应用程序都会有类似的设置(确保你已经安装了它，以及平板电脑制造商的驱动程序，以确保最佳体验)。
