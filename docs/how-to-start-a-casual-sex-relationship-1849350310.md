# 如何开始一段随意的性关系

> 原文：<https://lifehacker.com/how-to-start-a-casual-sex-relationship-1849350310>

一项的新研究表明，约会- 应用程序的用户更有可能随便约会和随便做爱。惊喜！虽然我们可能不需要一项研究来说明这个显而易见的事实，但是随意的性行为仍然会得到不好的评价，尤其是如果那是你唯一追求的。我们的性需求随着生活的变化而起伏不定，有时性是一种你想做的没有任何附加条件的愉快的活动。但是你如何在保持尊重和安全的同时进行随意的性行为呢？首先，实际定义一下随意性对你来说意味着什么可能是个好主意。

性健康教育家、顾问克里斯汀·马克博士说:“随意的性行为可以作为一个总括术语来定义那些与他们的性伴侣没有那么熟悉、依恋或承诺的人之间的性活动。“这可以包括从一夜情到有利益关系的朋友，但通常不在浪漫关系或依恋的范围内。”

由于随意性行为的参数可能有点微妙，所以在你下次勾搭之前要记住以下几点。

## **随意的性行为本身并不可耻**

一旦你理解了随意性对你意味着什么，同样重要的是接受这是你现在生活中正在寻找的，不要因为追求它而让自己(或他人)感到羞耻。

“追求随意的性行为总是可以的，如果这是你真正想要的，”马克说。“有很多重要的需求可以通过性来满足——比如性快感、压力释放、满足感、联系——这些不一定要发生在浪漫关系或依恋的背景下。”

但马克提醒说，同样重要的是要审视自己，确保自己的需求得到满足，而不是为了别人而牺牲自己。“如果你发现自己处于随意的性关系中，但你正在寻找更认真的东西，在这一点上要对自己真实，否则随意的性可能会成为不符合你需要的东西。”

## **开放的交流和安全是最重要的**

任何类型的关系，包括也许还有 *，尤其是*像这样的独立关系，都在交流中茁壮成长。如果你正在寻找一根火柴，并且你知道你只对一些不经意的东西感兴趣，尽快让它为人所知。“直接简单地传达这一点，”马克说。“你可以直接说，‘我只是在这里寻找随意的性爱，我想坦率地说出来，这样我们就能达成共识了。’就这样——保持简单、直接和诚实。从另一方面来说，如果你不是寻求随意性爱的人，那就简单直接地交流吧。

虽然在 COVID 的最初几个月，美国报告的性传播疾病(STDs)病例有所减少，但其中大多数，包括淋病和梅毒， [到 2020 年底](https://www.cdc.gov/media/releases/2022/p0412-STD-Increase.html) 又死灰复燃。这就是为什么，如果你经常与你不确定其性传播感染状况的伴侣进行无屏障保护的性行为，马克建议对每个新伴侣进行测试(特别是当伴侣与多个其他人发生性行为时)，或者在发生流体结合(没有屏障方法如避孕套)时进行测试。

马克说:“确保如果你在谈论性传播感染，如果涉及到可能导致精子与卵子相遇的身体部位，你谈论的是预防意外怀孕。”。“对你的意图、需求和安全保持透明。这一点至关重要，通过与伴侣交流这一点，你向他们表明你也关心他们的健康，”

如果你在与新伴侣发生性关系后发现自己患有性病，情况也是如此。

“性传播感染很常见，”马克说。“大约四分之一的人在一生中会被诊断为性传播感染。因此，知道了这一点，就试着直接接近它，甚至可能以这种方式来框定它。没有屏障保护的性行为有内在的风险，每个人都可以权衡这种风险和对自己的好处。这种风险的一部分是潜在的性传播感染，所以直接面对这个事实，这是你必须面对的。”

她建议与你最近的性伴侣联系，让他们知道他们需要接受测试。“避免责备或羞辱。简单地陈述事实，让他们知道你想关心他们和他们伴侣的健康，所以他们需要接受测试和治疗。”

无论是一夜情还是利益相关的朋友，马克说她最大的建议是永远对你的性伴侣保持诚实和明确，并经常检查。

“不管你有什么样的性生活，做一次脉搏检查以确保你和伴侣意见一致总是一个好主意，”她说，“简单地通过说‘太棒了，你怎么样？“我们在这里做得好吗？对你想要的东西保持透明和直接。不要为了勾搭而给人留下你想要什么的错误印象。做一个有同理心和开放的沟通者。"
