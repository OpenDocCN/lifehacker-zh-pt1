# 如何解锁 iOS 14.5 中隐藏调试模式

> 原文：<https://lifehacker.com/how-to-unlock-that-hidden-debug-mode-for-airtags-in-ios-1846837065>

苹果的 AirTags 有一个隐藏的调试模式，你可以在你的连接设备(理想情况下，是 iPhone)上非常容易地访问它。虽然其中的功能不会给你神奇的追踪能力，但你至少可以享受一些与 AirTag 追踪相关的数据。那，你也可以改变一些颜色。



为了开始使用 [调试模式](https://www.reddit.com/r/apple/comments/n5vtle/frustration_lead_to_airtag_hidden_dev_mode/) ，你将需要一个连接到你的设备的 AirTag。接下来，打开**查找我的**应用程序，选择要定位的 AirTag，点击绿色的**查找**图标，打开 [**精确查找**](https://support.apple.com/en-us/HT210967) 。一旦你这样做了，你应该看到传统的“哪里是我的东西”屏幕，否则你可以使用它在你的环境中导航，以查看上述 AirTag 跑到哪里:

要访问 AirTag 的隐藏调试菜单，只需轻按四次您要查找的项目的名称。这应该会弹出一个看起来相当狂野的屏幕:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1390332693188157440&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1390332693188157440&autosize=1) 

在你开始瞎搞之前，你可能需要截图，记下你做了什么调整，这样你就可以手动恢复它们。这是一个有点令人困惑的屏幕，因为用户可能从来没有打算看到它，所以你要确保你不会调整任何你可能失去跟踪的东西。

此外，你*能做的事情*也不多。调试屏幕——只要 Apple 允许您访问它——就可以让您调整 Precision Finding 屏幕的渲染方式，但您可能不需要(或不想)改变屏幕上出现的点的数量。同样，您可以使用顶部的两个滑块来改变 Precision Finding 的摄像头输入，以修改您在四处寻找 AirTag 时看到的模糊效果。同样，更多的设置你*可能*不想调整。

最重要的是，底部的四个滑块是您在锁定丢失的 AirTag 时，如果您想要调整将获得的颜色时可以使用的。当然，这只是一个表面上的改变，但如果你对 iPhone 的所有东西都有一种特定的风格，你现在也可以把它带到你的 AirTag Precision Finder 上。

仅此而已。没有太多可以玩的东西，但是看看你的 iPhone 的引擎盖下面总是很有趣。这将是一个很棒的派对戏法，至少——假设苹果保留了这个功能。
