# 如何写一份更好的资助提案

> 原文：<https://lifehacker.com/how-to-write-a-better-grant-proposal-1846292936>

写作是一种独特的艺术形式，与撰写有说服力的文章或信息性的博客文章有很大的不同。资助提案中的内容必须迎合其目标资助者，简明详细地描述资金将资助的工作，*和*在数百个应用中脱颖而出，如果不是数千个，其他应用中脱颖而出。



写作也是一种技能，你做得越多，就会做得越好。在过去的五年里，我撰写了自己的资助申请，并审核了其他人的资助申请。一路上，我收集了一些技巧来帮助初学者有一个良好的开端。

### 找到适合你工作的合适人选

在我早期与 [财政资助者 Fractured Atlas](https://www.fracturedatlas.org/) 的合作中，我发现许多赠款申请人认为存在一个适合他们提议项目的潜在资助者名单，他们所需要的只是获得它。不幸的是，这样的汇编清单通常不存在，寻找资金来源是一个艰巨的过程。在你开始之前，确保你已经准备好了。

大多数个人都可以使用 Candid(前身为基金会中心) [提供的基金会资源，这是一个在线名录](https://fconline.foundationcenter.org/) 列出了从小型家庭基金会到大型政府机构的资助者。如果你有多余的钱，你可以购买这项服务的 [【基本】会员](https://shop.candid.org/Subscriptions/FDOEssentialPlan?biennial&_gl=1*19wfq4t*_gcl_aw*R0NMLjE2MTM1OTEwNDcuQ2p3S0NBaUFtck9CQmhBMEVpd0FybjNtZk5iNldWUWpZdWJpOGJqbHpmOXRnaU1DcS1mcGkteU5iME1mdndtZC1maFEyZDdleW9ES2l4b0NqUzRRQXZEX0J3RQ..*_ga*MTQzOTk5Njk2OS4xNjEzNTkxMDQ3*_ga_5W8PXYYGBX*MTYxMzU5MTA0Ni4xLjEuMTYxMzU5MTA1Mi4w&_ga=2.149349942.1804954281.1613591047-1439996969.1613591047&_gac=1.247107190.1613591048.CjwKCAiAmrOBBhA0EiwArn3mfNb6WVQjYubi8bjlzf9tgiMCq-fpi-yNb0Mfvwmd-fhQ2d7eyoDKixoCjS4QAvD_BwE) ，每月 49.99 美元，两年 758 美元。否则——一旦疫情结束——你可以进入 candy 的任何一个官方 [现场](https://candid.org/find-us) 免费浏览他们的数据库。寻找与自己工作最匹配的机会。每个资助者都希望支持不同的东西:例如， [哈克尼斯基金会](https://harknessfoundation.org/) 只支持艺术——特别是舞蹈项目。其他的侧重于人文学科，如国家人文捐赠基金，而其他的，如福特基金会，倾向于将他们的关注点分散到更广的范围。 [骨折图册建议](https://blog.fracturedatlas.org/6-tips-to-improve-your-art-grant-applications) :

> 看看他们的网站，他们的使命宣言，还有他们过去的 990。看看他们过去资助过谁。通过研究资助者，你将能够真正了解你的项目是否真的是他们可能支持的那种东西。

有些机会可能与你的工作不是 100%匹配，但仍然符合你试图实现的基本目标。最重要的是要记住，你不应该试图改变你的工作来适应任何一个资助者的模式。那永远不会成功——对你的项目不会，对投资者也不会，他们总是能知道你的项目什么时候真的不适合他们的组织。合适意味着你的使命和愿景与他们的相匹配，并且你符合机会的资格标准。例如，我曾经为纽约艺术基金会的一个网络系列写了一笔赠款，该系列以镜头前和镜头后的女性为特色，瞄准了媒体中女性的一个特定机会。这个申请是成功的，因为该组织的使命与我的使命相匹配，我的工作与资助者试图支持的东西直接相关。

### 保持内容清晰简洁

即使你已经找到了合适的机会，写自己的作品仍然会令人紧张。首先也是最重要的，告诉投资者你到底是做什么的，放弃华丽的语言或行话。如果你经营一个青年合唱团，请在你的计划书中特别注明。在说你管理一个唱诗班之前，你不会想立刻开始谈论“歌唱在这个现代时代的重要意义”;这只会混淆读者，模糊你的信息。尝试用通俗易懂的语言解释事情——拨款审查人员可能在你的领域工作，但只有你有你独特的经验。“解释一下*为什么*有一个你需要解决的问题，并提供支持你的主张的证据，*然后*将它与你的项目联系起来，然后回到资助者的优先事项，”费城亚洲艺术倡议前发展总监瓦妮莎·拉马尔霍建议道。“你需要为评审者把这些点联系起来，以证明他们为什么应该资助你。”

我喜欢从指导一个从未做过三明治的人如何将花生酱、果冻和面包变成午餐的角度来思考这个问题。你不能假设任何事情:你必须告诉他们在哪里可以得到那罐果冻和花生酱，使用什么器皿，以及如何将它们组装成三明治。如果他们的经历发生在其他地方，你可能会得到一个看起来非常奇怪的三明治。

### 要求准确的资金数额

在要求你真正需要的东西和理解资助者愿意付出的东西之间有一个微妙的平衡。一般来说，一个资助者只愿意支持你全部项目预算的 30%左右。资助者通常不想成为一个项目的唯一资助者，他们更希望知道你的工作得到了其他来源的支持。例如， [纽约艺术委员会](http://www.nysca.org/downloads/guidelines/FY2021/FY21_IND_Guidelines.pdf) (NYSCA)在电影类个人艺术家指南中指出，申请人应该“注意，NYSCA 不会支付超过项目总成本 50%的费用。”

了解这一点有助于你制定一个准确的预算，并设定一个合理的要价，既满足你的需求，又符合出资者的指导方针。这就是搜索资助者过去 90 年的资料的地方——非营利组织必须向国税局提交的税单——过去的捐赠信息就派上用场了。你可以看到一个资助者通常给了多少，并可以根据他们过去的行为得出你的数字。要求太多是一个问题，但要求太少也可能引发危险信号。如果你去找一个通常给更大金额(比如说 50，000 美元)但你的要价很低(10，000 美元)的资助者，他们会认为你的项目规模对他们的资助来说太小了。重要的是尽可能诚实。为你实际需要的东西做一个预算，估计你能从你申请的所有资助者那里筹集到多少。基于他们的投资习惯和你的需求，你应该能够确定最合适的要价。

### 重读并重写，关注细节

完成提案后，大声重读一遍，并在语言听起来笨拙或不清楚的地方做必要的修改。也让另一个人通读一遍，因为这将有助于你从外部的角度看待你的写作，并找出你的支持信息中的任何漏洞以及任何打字错误或语法错误。

拉马尔霍就初稿完成后需要考虑的问题提供了一些额外的见解:

*   是作家太接近项目和叙事，以至于不能客观地评估它？
*   叙述读起来是清晰还是混乱？
*   可以扫描吗？
*   你能找到评估标准和资助者优先顺序吗？
*   它流畅/读起来好吗？

“平均来说，在准备提交之前，我会将一份提案至少经过 4 到 5 次草稿，”拉马尔霍说。当你为你的申请过程计划时间表的时候，要考虑到这一点。第一稿很可能不是你的最后一稿，所以给自己足够的时间思考和编辑。不要害怕寻求帮助。
