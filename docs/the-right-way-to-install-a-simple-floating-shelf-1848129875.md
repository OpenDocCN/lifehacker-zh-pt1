# 安装简易浮动搁板的正确方法

> 原文：<https://lifehacker.com/the-right-way-to-install-a-simple-floating-shelf-1848129875>

如果你的房间中有一个壁龛、角落或其他尴尬的空白空间，需要一些额外的储物空间或装饰，最好的解决方案可能是一个简单的浮动搁板。浮动搁板的制作成本低，T5 和 T6 安装起来也很简单，T8 还可以完成一个需要一点东西的空间。

## 制作浮动搁板所需的工具

你需要准备一些工具——还有你自己的材料——准备好去工作。对于工具，g 集合在一起一把锯子，螺丝，一个带预钻头的钻，和一个 P hillips 头钻头，一把水平尺，一把锤子，和一些抛光钉(或气动抛光钉枪)，以及一把卷尺。鞋钉探测器也会派上用场。

对于你的搁板材料，选择一块不超过 英寸厚的木板，根据你想要的搁板深度而定。此外，你还需要一个一英寸宽两到三英寸宽的架子边缘装饰条，以及一块窄一点的木头用作你架子的钉条(不比一英寸宽大)。

## 如何测量你的货架零件

首先，测量想要悬挂搁板的空间。对于三面空间，，,比如壁橱或壁龛，你需要将钉条固定在你想要放置搁板的高度的墙钉上。量出你的身高，用铅笔在墙上的三面都标上。

接下来，测量你的钉条，使其比你的架子的边和宽度短大约两英寸半，在架子的两端各剪一根，在架子的后边缘各剪一根。对于转角搁板，搁板的每一侧都需要一个钉条。测量你空间的宽度，并根据需要裁剪你的搁板。对于转角搁板，你需要把搁板剪成 45/45/90- 度的三角形。要检查你的架子是否水平，把它放在你的钉条上，，用你的水平仪在顶面上。

## 安装您的饰件

现在，您已经准备好测量和裁剪您的修剪。测量搁板的边缘，根据需要裁剪。如果你要做一个转角搁板，你需要以 45 度角切掉切边，以匹配搁板的边缘。确保你的装饰沿着边缘是光滑的，这样一旦安装好，它看起来就会很干净。最后，用钉条把你的架子钉牢，，把你的装饰钉在架子边缘。你的架子现在可以用油漆或染色剂完成了。

## **使用支架**

如果你要从墙上直接安装一个搁板，侧面没有支撑，你可以使用一些硬件来简化安装。 支架有很多种类型，但一个坚固而简单的选择是单钉 [金属支架](https://www.amazon.com/Floating-Brackets-Shelves-Bathroom-Decoration-Screws/dp/B08N119J1R/ref=asc_df_B08N119J1R/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-right-way-to-install-a-simple-floating-shelf-1848129875&asc_source=&hvadid=475689979807&hvdev=c&hvdvcmdl=&hvlocint=&hvlocphy=9061081&hvnetw=g&hvpone=&hvpos=&hvptwo=&hvqmt=&hvrand=5805698726374308473&hvtargid=pla-1110087019673&linkCode=df0&psc=1&tag=kinjalifehackerlink-20) 它带有连接它所需的螺钉和锚。将你的支架安装到墙上有螺栓的地方，确保它们是水平的，然后测量并切割钉条，使其适合支架之间，然后将其安装到墙上，使钉条的顶部边缘与支架的顶部齐平。

然后，将搁板放在支架顶部，用钉子固定在钉条上。要测量你的装饰，先把你的侧边裁成架子的深度，然后钉上。然后，测量搁板前部的总宽度(包括侧边装饰)、并裁剪前部装饰，使其匹配。将前装饰钉在搁板边缘，你就有了一个浮动搁板。
