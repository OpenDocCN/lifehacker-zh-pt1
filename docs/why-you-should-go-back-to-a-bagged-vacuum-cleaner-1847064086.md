# 为什么你应该回到袋装真空吸尘器

> 原文：<https://lifehacker.com/why-you-should-go-back-to-a-bagged-vacuum-cleaner-1847064086>

老派的袋装真空吸尘器现在很少见了。这在一定程度上是有道理的:如果你能以不到 200 美元的价格从亚马逊订购一台全新的、轻便的罐式真空吸尘器，为什么要为一台要求更换袋子的更重、噪音更大的机器费心呢？



说白了就是因为袋装吸尘器他妈的棒。去年夏天，我在 Craigslist 上花 80 美元买了一台翻新的 Kirby Sentria II，它立刻改变了我的生活。是的，它很重，也不是特别灵活，但它在灵活性方面的不足完全可以用纯粹的功能来弥补。(这个东西像 800 美元的戴森棒真空吸尘器一样吸走猫毛——绝对管用。)

但是，据我所知，袋装真空吸尘器提供的不仅仅是实用的功能。除了最花哨的无袋款，它们至少有三个明显的(也是巨大的)优势。如果你没有几百美元来买戴森，那么在你的价格范围内，几乎任何袋装吸尘器都是下一个最好的选择，我很乐意告诉你为什么。

## 袋装真空吸尘器是过敏症患者的救命稻草

在我成年后的大部分时间里，我都在使用无袋真空吸尘器，我认为打喷嚏是交易的一部分。运行真空吸尘器总是会扬起一些灰尘，清空罐子会使灰尘滚滚到空气中。如果你对真空中的任何东西过敏——灰尘、尘螨、宠物毛发——整个过程可能会比它的价值更令人恼火。

袋装真空吸尘器不会出现这种情况。只要连接头部和袋子的管子密封良好，所有的灰尘、头发和其他细小颗粒物质都会直接进入一个专门用来收集细小颗粒物质的袋子。更换袋子可能会释放出一小股松散的灰尘，但与倒空罐子产生的漩涡相比，这根本不算什么。如果你或任何和你住在一起的人患有室内过敏症，你应该检查一下袋装真空吸尘器。如果你需要的话，你甚至可以买 HEPA 级别的包。

## 袋装真空吸尘器比你想象的更环保

我知道你在想什么:真空吸尘器对环境极其有害。每次你换袋子的时候，你都是在把大量的灰尘和头发装进一个不会坏掉的袋子里送到垃圾填埋场。

这里有两件事。首先，虽然许多真空袋——尤其是那些符合 HEPA 标准的真空袋——是由不能生物降解的合成布制成的，但其他许多真空袋确实可以。你可以购买由纸质材料和其他可堆肥材料制成的袋子，这些材料适合大多数型号的袋装真空吸尘器。第二，你认为哪一种污染更大:越来越模糊的真空袋，其中一些是可生物降解的——还是全球工业每年生产更多的塑料真空罐，当它们破裂时不可避免地被扔进垃圾填埋场？

## 你真的可以修理袋装真空吸尘器

这让我想到了最后一个优势，这是一个很大的优势:与绝大多数便宜的罐式真空吸尘器不同，袋装真空吸尘器是为维修而设计的。经常磨损的零件(尤其是皮带)非常便宜，很容易更换，看几个 YouTube 视频就可以自己动手了。对于更难的东西，总有真空修理店，它们并不像听起来那么晦涩难懂。几乎美国的每个大都市都至少有一个，因为商业清洁公司需要保持他们的吸尘器处于工作状态。如果你住在有很多酒店或办公楼的城镇或城市的车程范围内，很有可能你也会找到一家真空修理店。

能够根据需要维护和修复你的真空听起来像是一件很小的事情，但这种事情出奇的少。使用同一台吸尘器几年来保持你的地毯清洁，保持你口袋里的钱，保持你的吸尘器远离垃圾填埋场，这是 T4 的伟大目标。
