# 这些航空公司让你免费“携带”

> 原文:[https://life hacker . com/these-airlines-let-you-carry-on-free-1849106506](https://lifehacker.com/these-airlines-let-you-carry-on-for-free-1849106506)

随着机票价格的持续上涨，消费者正在寻找控制机票总成本的方法。如果你做得好，你可以避免的一项费用是支付把你的行李带上飞机的荣誉。每个航空公司都有自己的一套行李规则和定义。什么是严格意义上的随身包？什么是私人物品？哪个是免费的，哪个是要付费的？

当你出现在你的门口时，提前知道什么可以让你免于支付(很可能是不合理的)费用。以下是美国最大的航空公司的分类，以及它们对随身行李和个人物品的政策。

## 允许携带手提行李和个人物品的航空公司

[**美国航空公司。**](https://www.aa.com/i18n/travel-info/baggage/carry-on-baggage.jsp) 美国航空公司不会对你的随身行李或个人物品收费，只要它们符合规定的尺寸。手提行李不得超过22 x 14 x 9英寸，您的个人物品必须能够放在您前面的座椅下面(18 x 14 x 8英寸)。尿布包、软边母乳冷却器、婴儿车和医疗或移动设备不算个人物品。

[**达美航空**](https://www.delta.com/us/en/baggage/carry-on-baggage) **。**达美航空允许购买任何机票的乘客免费携带一个随身行李包和个人物品。对于随身行李，其尺寸不得超过22 x 14 x 9英寸，但不受任何重量限制(只要您的航班是国内航班)。经批准的个人物品包括钱包、公文包、相机包、笔记本电脑包或尿布包。此外，你在候机楼买的雨伞和免税商品不算个人物品；你也可以把它们带上飞机，没有任何麻烦。

[**夏威夷航空公司。如果您乘坐夏威夷航空公司的航班，您不仅可以免费获得一个列伊，而且您也不必为您的随身行李付费。夏威夷航空公司的规定比他们的一些免费随身行李更严格，要求25磅的重量限制，但执行标准的22 x 14 x 9英寸尺寸规定。如果您的随身物品是乐器，夏威夷航空公司允许例外，只要它能放在头顶的行李舱里。对于个人物品，夏威夷航空公司允许携带钱包、公文包、笔记本电脑包或小背包，但它们必须放在您前面的座位下面。**](https://www.hawaiianairlines.com/legal/domestic-contract-of-carriage/rule-17)

[**西南航空**](https://www.southwest.com/html/customer-service/travel-fees.html) **。如果你曾经带着一大堆行李飞行，你最好的选择是西南航空公司。他们不仅不会对您的随身行李或个人物品收费，您还可以免费托运两个包(前提是它们的长度、宽度和高度不超过50磅或62英寸)。西南航空公司允许随身行李比同行的大，规定尺寸为24 x 16 x 10英寸。对于个人物品，它们必须适合16.25 x 13.5 x 8英寸的尺寸。** 

## 只允许携带个人物品的航空公司

[**联合航空公司。**](https://www.united.com/ual/en/us/fly/travel/baggage/carry-on.html) 如果您购买了美国联合航空公司的普通经济舱机票，您只能携带一件个人物品登机。你的个人物品必须适合9 x 10 x 17英寸的尺寸。如果你决定带一个手提包，你需要支付25美元的费用。您的包的尺寸不得超过22 x 14 x 9英寸。

[**捷蓝航空。**](https://www.jetblue.com/help/adding-bags) 如果你要乘坐捷蓝航空，请准备好尽可能多的东西放在你的个人物品中，否则你会为此付出代价。对于他们的基本票价，捷蓝航空公司收取65美元将一件随身物品带到登机口，但仍然要接受检查，因为该航空公司不会为他们的基本票价客户分配头顶置物箱空间。

[**精神航空公司。**](https://www.lugless.com/airline-baggage-fees/spirit-airlines-baggage-fees/) 如果你乘坐的是Spirit航空公司的飞机，在登机前，确保你所有的事情都准备好了。虽然你可以免费携带一件私人物品，但一个随身携带的包的价格在26美元到65美元之间，取决于付费的方式和时间。个人物品的尺寸不得超过18 x 14 x 8英寸。

[**忠实航空公司。**](https://www.allegiantair.com/popup/changed-bag-fees-policy) 如果你带着随身行李在忠实的天空中飞行，请准备好为这份奢华买单。Allegiant Airlines每件行李收费35美元，较重的行李额外收费(最高75美元)。您的随身行李尺寸不得超过22 x 14 x 9英寸，否则将收取50美元的“机场行李费”。
