# 你应该冷冻鹰嘴豆液

> 原文：<https://lifehacker.com/you-should-be-freezing-chickpea-liquid-1849193948>

当谈到鹰嘴豆和蚕豆时，液体和豆类是同一个罐子里的两个非常不同的居民。仅仅因为你在沙拉中用了鹰嘴豆，并不意味着你在同一天就想吃这些松软的煎饼 (但你可能会在周六)。如果你吃剩了 aquafaba— 或者任何时候你吃鹰嘴豆，发现自己要把液体倒掉了— 停下你的手。它冻得非常好。

自从 2014 年发现 Aquafaba 作为搅打蛋白 的惊人有效替代品 [(点击](https://lifehacker.com/use-chickpea-liquid-as-an-awesome-replacement-for-eggs-1712243751) 了解更多历史)以来，它已经获得了很大的吸引力。然而，它有更多的应用，而不仅仅是作为一个巴芙洛娃的聚会戏法。这是一种很好的纯素食成分，可以使汤变稠，或者使烘焙食品结合和透气，比如松饼和前面提到的松饼。

我不是素食主义者，在我的烘焙中，我大多使用鸡蛋，但我确实经常使用 aquafaba 来增稠汤和酱料。它有一些类似鸡蛋的成分，像 [白蛋白和球蛋白](https://www.seriouseats.com/science-of-aquafaba-meringues-5185233#:~:text=Aquafaba%20is%20rich%20with%20substances,useful%20in%20creating%20culinary%20foams.) ，它的淀粉给汤料增加了粘度。我花了一些时间练习停止吸干我的鹰嘴豆，但是我已经养成了冷冻剩余的蚕豆的习惯。每当我碰巧在做汤，而且肉汤可能需要一点*什么的时候，*我可以很容易地把转到我的冰箱里。

美国的测试厨房建议冷冻 [冰块托盘](https://www.americastestkitchen.com/guides/vegan/what-is-aquafaba#:~:text=Buy%20on%20Amazon-,How%20to%20Store%20Aquafaba,as%20well%20as%20fresh%20aquafaba.) 里的东西，但我通常一次只冷冻一罐，而略少于半杯不足以指挥一整托盘。相反，我将 aquafaba 冷冻在一个小塑料容器中，然后将砖块放入冷冻袋中长期储存。有时我会一次放两三块糖，为下一次喝汤或做调味汁做准备。如果你决定把它分成冰块托盘，一定要在倒之前搅拌或轻轻摇动鹰嘴豆液体。Aquafaba 不是天然同质的，因此这有助于在冷冻前均匀分布白蛋白和球蛋白颗粒。

Aquafaba 最多保持 [四个月](https://www.freezeit.co.uk/can-you-freeze-aquafaba/) 。如果你担心 aquafaba 在冰柜中放置一段时间后表现不佳，那就打消你的担心吧。F 冷冻不会破坏或损害淀粉，淀粉是其优良搅打能力的原因，根据美国测试厨房的说法，冷冻和解冻版本和以前一样有效。将冰块放入碗中，放入冰箱过夜，或者用微波炉加热一两分钟，根据冰块的大小来解冻 aquafaba。如果你要把它放在汤或酱里，只需把一块冷冻的厚片和其他东西一起直接扔进锅里。
