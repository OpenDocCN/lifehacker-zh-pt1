# 挑战经典回合制 RPG 模式的 10 款神奇宝贝游戏

> 原文：<https://kotaku.com/pokemon-legends-arceus-nintendo-snap-colosseum-masters-1848414016>

谁准备好观看由神奇宝贝主演的不是传统回合制 RPG 的最新游戏了？《神奇宝贝传奇:阿尔宙斯》本周上映，这是一长串让我们以新的和不同的方式玩口袋妖怪的故事和衍生产品中的最新一个。*例如，神奇宝贝传奇:阿尔宙斯*将卡普空*怪物猎人*系列的元素与之前从未在*神奇宝贝*游戏中出现的动作 RPG 元素混合在一起。

因此，当我们准备开始这个全新的冒险时，让我们来回顾一下其他 10 款混淆了怪物收集公式的神奇宝贝游戏。

让我们从该系列最早的衍生产品之一开始，哈尔实验室的*让我们一边拍照一边看口袋妖怪做事情*。*神奇宝贝 Snap* 轨道上的偷窥游戏与漫游*神奇宝贝红色*和*蓝色*的像素化牧场相去甚远，但基本想法是一样的——抓住他们。

早在 1999 年，神奇宝贝 Snap 就在任天堂 64 游戏机上首次亮相，距离最初的游戏男孩 RPG 风靡北美仅一年。虽然为时尚早，但任天堂和 Game Freak 已经计划让*的神奇宝贝*远远超出玩家的口袋。Snap 是未来事物的一种尝试，它尝起来相当不错。

嘿，孩子们，你想在你的电视上玩你的 Game Boy*pokémon*RPG 的完整控制台版本吗？太糟糕了。在 2000 年向西方观众发布的*神奇宝贝体育场*给了粉丝们一个诱人的体验，一个 3D 控制台*神奇宝贝*游戏可能没有实际交付任何接近*红色*和*蓝色*的冒险。能够使用转移包用你的游戏机交易你的口袋妖怪是很棒的，但是所有没有情节的战斗让*体育场*变得无趣。

尽管如此，这个游戏还是有希望的。也许这些华丽的 3D 神奇宝贝最终会在任天堂的下一台家用游戏机上得到一个真正的 RPG。

嗯，嘿，你知道吗，一个真正的任天堂游戏机上的神奇宝贝角色扮演游戏，只是不是粉丝们想要的那种。任天堂没有采用 Game Boy 中的一款游戏，而是用 3D 图形更新它，而是利用 Genius Sonority，一家由前*Dragon Quest*dev 组成的日本工作室，来制作一款全新的冒险游戏。

*神奇宝贝斗兽场*及其续作*《神奇宝贝 XD:黑暗之风》*将该系列带向了更黑暗的方向。游戏设定的奥勒地区是一片稀疏的土地，通常没有漫游的口袋妖怪。作为主角，你的任务是从一个城镇游荡到另一个城镇，净化被黑暗腐蚀的口袋怪物。你没有收集野生神奇宝贝，而是从你与之战斗的驯兽师那里偷来的。这两款游戏都有这种令人毛骨悚然/酷的氛围，这让它们在粉丝心中赢得了一席之地。它们可能不是我们想要的游戏机 RPG，但它们肯定是别的东西。

还记得任天堂 DS 的早期吗，当时任天堂真的很喜欢那些以酷和有创意的方式使用手写笔和触摸屏的游戏？正是在那个令人兴奋的时代，Hal Labs 为我们带来了 2006 年的《神奇宝贝游侠》( pokémon Ranger )( T1 ),这是一款游戏，你可以在口袋怪物周围画圈来捕捉它们。手写笔能做的真的就这么多。

你不用在神奇宝贝中心治疗你的神奇宝贝，而是给你的手写笔充电。与不同类型的怪物合作给了你的手写笔不同的力量，这使得快速画圈比简单地在纸上画圈更有趣和更有活力。

《神奇宝贝游侠》不仅比听起来有趣得多，它实际上还衍生出了两部续集，《T2》《神奇宝贝游侠:阿尔米娅的阴影》和《T4》《神奇宝贝游侠:守护标志》使其成为了一部多产的手持《神奇宝贝》衍生游戏。

说到多产的手持神奇宝贝衍生产品，这里是其中最多产的。日本开发商 Chunsoft，前五个*龙之旅*游戏的开发商(感觉到这里有一种模式)，从 2005 年起就开始将口袋妖怪引入其 roguelike 系列。与其炫耀最初的一对 Game Boy/DS 游戏的像素图形，我们不如在这里展示最近为 Switch 重拍的*神奇宝贝神秘地牢:营救队 DX* ， [的迷人手绘视觉效果。](https://kotaku.com/being-the-one-human-in-a-world-of-pokemon-is-pretty-coo-1842084962) 

*神秘地牢*系列没有抓住神奇宝贝，而是让你扮演一个变成口袋怪物的人类，在一个充满神奇宝贝的世界里与另一个神奇宝贝合作解决神奇宝贝的罪行。这是一个有趣的公式扭曲，即使罗格列式基于网格的战斗并不适合每个人。

神奇宝贝大师是任天堂试图以最古怪的方式利用移动游戏 gasha 的热潮。玩家不用在神奇宝贝上花游戏币，而是用他们辛苦赚来的(或买来的)宝石来购买整个系列历史中的训练器。来自各种手持 RPG 的角色带着他们标志性的神奇宝贝加入玩家的队伍，在一个全新的地区进行回合制战斗。

是 [一款漂亮的手机游戏](https://kotaku.com/new-mobile-game-pokemon-masters-is-more-about-people-th-1837673897) 太快就没电了。

万代纳木错让我们看了"*神奇宝贝*格斗游戏。"在升级到 Wii U 和 Switch 之前，起源于日本游戏厅的*扑克锦标赛*将那些迷人的回合制战斗转变为一款全面的战斗游戏。有点自由漫游的 3D，有点侧身的 2D， [和所有优秀的](https://kotaku.com/pokken-tournament-the-kotaku-review-1764966156) 。

唯一的缺点是，这款游戏没有囊括数百个神奇宝贝中的每一个。Switch 版本只有 23 架战斗机，包括两架皮卡丘和两架 Mewtwos。我仍然希望在未来的 DLC 中加入其他数百个。

[*PokéPark Wii:皮卡丘的冒险*](https://en.wikipedia.org/wiki/Pok%C3%A9Park_Wii:_Pikachu%27s_Adventure) 最酷的功能之一是能够将你的冒险截图并分享到 SD 卡或发布在 Wii 留言板上。开发者生物公司确实走在了时代的前面。

这是一个关于刷牙的游戏，你可以收集帽子，并赢得长期彻底刷牙的奖励。 [根据我们的评论员](https://kotaku.com/pokemon-smile-the-kotaku-review-1844242342) 卢克·普兰科特(Luke Plunkett)的说法，任何时候你嘴里都至少有一个神奇宝贝我告诉自己，我会尝试并坚持与这个列表中的标准 RPG 有一些相似之处的游戏，但你嘴里有神奇宝贝。那会让你的嘴变成一个口袋。震惊了。

神奇宝贝频道尽可能远离最初的*神奇宝贝 RPG* 格式。这是一个和你的皮卡丘一起看电视的游戏，而奥克教授轮流看着你。它部分是数字宠物模拟，部分是冒险游戏，大部分是无聊的。这是一款 GameCube 游戏，它的存在主要是为了让制作*神奇宝贝*衍生产品和辅助游戏清单的人可以指着它大笑。我的义务已经完成了。
