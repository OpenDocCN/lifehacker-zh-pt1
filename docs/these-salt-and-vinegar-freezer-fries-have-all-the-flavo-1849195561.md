# 这些盐和醋冷冻薯条有所有的味道，没有潮湿

薯条是我最喜欢的预制马铃薯——甚至是杂货店里的冷冻马铃薯。当它们新鲜出炉，滚烫，外脆里嫩时，它们达到了顶峰。这就是为什么用醋浇我家做的油条是一种内心斗争:保持它们的简单和基本？或者事后把它们酱了，味道很好但是感觉很难过很湿。原来，我一直都是反着做的。获得咸醋潘趣酒而不需要忍受浸泡的薯条的最好方法是在你烤薯条之前彻底调味。

我一直很喜欢英国人和加拿大人对噘嘴、高钠法式炸薯条的喜爱。薯片版本同样受欢迎(甚至可能更多)，但我认为油炸食品中厚厚的马铃薯部分更好地平衡和分配了那些强烈的味道。然而，将水状物质淋在油炸和松脆的东西上的想法是一种烹饪矛盾，每次都会让我犹豫片刻。结果每次都是一样的:W et fries。潮湿的薯条。我一直很后悔。

我突然想到，美味的薯条不一定是这样的。与其把盐和醋作为调味品，为什么不把它作为一种有目的的配料，像腌泡汁或擦剂？我满怀信心地开始测试这种盐醋法式油炸食谱，但在我的内心深处有一个小小的疑问。如果醋必须悬浮在水中才能变得浓烈呢？也许味道会“煮掉”毕竟，你可以用醋来擦拭台面和清洁其他表面，一旦变干就不会有持久的酸味。我希望得到的好处是干而脆的薯条，没有潮湿、和明显的咸味、醋味，它们会附着在每一根薯条上，而不是沉淀在锅底。

## 如何给薯条调味

我买了一包奥瑞达薯条，特别脆，但没有调味，还有一瓶亨氏麦芽醋和精制海盐。我做了三批:一批放在醋和盐的溶液中大约一分钟，然后立即铺在箔衬烤盘上，一批在盐和醋中浸泡15分钟，然后铺在箔衬烤盘上(那个 我撒了额外的盐)，还有一批控制，以确保我没有想象出虚幻的醋味。

结果是惊人的。我非常喜欢这两套薯条(实际上是三套)。快速折腾烤批次是一个令人愉快的醋暗示，即使盐在预调味过程中溶解在液体中，它是很好的平衡和美味。这种风格非常适合那些喜欢吃整平的冷冻油炸食品，但不一定能猜到他们吃的是醋的人。浸泡15分钟的薯条也很棒。正如我所希望的那样，腌制让醋味渗透得更深，最终的味道是可以辨别的醋味。在烘烤第二批之前额外撒的盐肯定会粘在每根薯条上，这感觉更像是我正在寻找的扑鼻盐水的攻击。

两个批次的薯条都很脆，比控制批次的更脆。醋中的酸还会产生额外的褐变，这是一个受欢迎的副作用。制作百吉饼和椒盐卷饼时也会出现类似的现象。根据素食物理学家在 [这篇文章](https://veganphysicist.com/pretzels-and-ph-hacking-the-maillard-reaction/#:~:text=Or%20in%20(almost)%20plain%20English,characteristic%20browning%20and%20flavour%20development.) 中的说法，加入沸水中的碱液或小苏打(这里是或醋)会以一种显著的方式引发 [美拉德反应](https://en.wikipedia.org/wiki/Maillard_reaction) ，即使在这些薯条的烘焙时间相对较短的情况下。

这些盐醋薯条的配方可以根据你是想要稍微调味还是更浓来调整。我想你可以将它们浸泡更长时间，以获得更多的醋味，但过了某一点后，我不确定它们会吸收更多。用你自己的手试试这些- 切薯条，让我在评论中知道结果如何。

## 盐和醋冷冻薯条

成分:

*   4盎司薯条
*   1汤匙麦芽醋
*   茶匙精制海盐(烘烤前额外洒一点)

把烤箱预热到400华氏度。

用锡箔纸或羊皮纸在一个薄烤盘上做衬里。没必要给箔片上油。

在一个大的宽底碗中(即使是煎锅也可以)，将醋淋在薯条上，撒上盐。将薯条拌入盐和醋中，直到均匀覆盖。要想味道更浓，将它们放在溶液中15分钟，不时搅拌和翻动。准备好之后，用钳子或叉子将薯条移到平底锅上，将多余的醋留在碗里。给它们多撒些盐。如果想要口味清淡一点的话，在加入盐和醋后，将薯条平铺在烤盘上。

烤20分钟。轻轻地将薯条从锡纸或羊皮纸上揭下，立即享用。
