# 我们需要谈谈丹麦的“口罩不起作用”研究

> 原文：<https://lifehacker.com/we-need-to-talk-about-that-danish-masks-dont-work-study-1845716261>

丹麦最近一项关于戴口罩的研究正在被分享，好像它证明了口罩并不能保护我们。这项研究实际上并不是这么说的。



我将在下面详细说明，但现在我想指出这篇论文中的几行文字:

> 最重要的限制是研究结果是不确定的...

和

> 需要强调的是，这项试验并没有将口罩作为源头控制[保护他人免受佩戴者伤害]或在社交距离和其他公共卫生措施无效的环境中作为保护措施。

所以，如果你分享这项研究是为了支持你之前的观点，即口罩是愚蠢的，你不想戴口罩，请停止。现在，让我们看看这项研究到底是关于什么的。

### 这项研究旨在回答什么问题？

这项新发表的名为 [的研究于今年 4 月和 5 月在丹麦进行，该研究名为在其他公共卫生措施中增加口罩建议以防止丹麦口罩佩戴者](https://www.acpjournals.org/doi/10.7326/M20-6817) 感染新型冠状病毒病毒的有效性。

研究环境和我们现在生活的世界不太一样。病例很少，大多数人没有戴口罩。该国依靠社交距离作为减缓新冠肺炎传播的主要方式，从 3 月开始封锁，从 [持续到 5 月](https://www.reuters.com/article/us-health-coronavirus-denmark/fast-in-first-out-denmark-leads-lockdown-exit-idUSKBN22U1TC) 。

问题是:如果我们给人们邮寄一盒口罩，并告诉他们戴上口罩，戴口罩的人被感染的风险会大大降低吗？这项研究没有设盲，因为很明显你知道自己是否戴着口罩。参与者对自己进行了抗体测试，并报告了他们是否按照说明佩戴口罩的调查结果。

### 他们发现了什么？

调查人员没有看到他们所希望的感染率的大幅下降。两组人——收到一盒口罩的人和没有收到口罩的人——最终的感染率大致相同。

他们是否看到了风险的小幅降低？不可能说。这项研究被赋予了统计学意义上的，以检测口罩组和对照组之间的感染减少了 50%。如果口罩降低的风险更低——25%？10%?——这项研究没有发现这一点。

这就是为什么这项研究的主要发现被报道为从不到 50%的减少到甚至可能增加感染。这是一个很大的可能性范围，因为这项研究的目的不是告诉他们之间的区别。

“底线是——你戴口罩可能不会使你感染的风险减半。但它可能会减少 20%，而在经历大规模流行病的整个国家，这一数字加起来，”卧龙岗大学的流行病学家 Gideon Meyerowitz-Katz 说。

### 它没有发现的

这项研究没有问如果你周围的大多数人都戴着口罩，而你选择不戴，会发生什么。(当时戴面具在丹麦并不常见。)

这项研究没有问如果你不戴口罩去生活、购物和社交会发生什么。(在研究的大部分时间里，丹麦都处于封锁状态。)

这项研究没有测试全国范围的口罩要求(同样，没有)。

这项研究没有测试口罩是否能防止你生病时传播冠状病毒。(它只研究了佩戴者的感染风险。)

这项研究没有直接测试口罩，只测试了当你建议人们戴口罩时会发生什么。(正如 Vinay Prasad [在 MedPage Today](https://www.medpagetoday.com/blogs/vinay-prasad/89778) 中写道，“一个蛋糕既是面糊也是烤箱的温度，一个公共健康建议既是建议本身也是人们是否遵循它。”这项研究没有将两者分开。)

正如疾病预防控制中心最近澄清的那样，我们有其他证据表明口罩可能对佩戴者有益 。丹麦的研究丰富了我们的知识，告诉我们不要期望口罩对佩戴者有很大的好处。这也提醒我们，人们并不总是遵循建议；这项研究的参与者必须报告他们是否按照建议佩戴了口罩，只有大约一半的人说他们完全按照指示做了。

Meyerowitz-Katz 说:“根据这项研究，说口罩失效或无用是不正确的。”他指出，即使对任何个人的益处很小，口罩的强制规定仍可能有“对社会的巨大益处”。我们已经知道了，不是吗？继续戴着你的面具。
