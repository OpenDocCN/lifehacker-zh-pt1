# 如何计划今年10月恢复你的学生贷款

> 原文:[https://life hacker . com/how-to-prepare-the end-of-student-loan-nuance-1846776468](https://lifehacker.com/how-to-prepare-for-the-end-of-student-loan-forbearance-1846776468)

暂停联邦学生贷款支付和利息 [将于10月](https://www.cnbc.com/2021/01/20/biden-plans-to-extend-pause-on-student-loan-payments-until-october.html) 结束，但是你准备好了吗？考虑到 [55%的借款人](https://studentloanhero.com/featured/student-loan-relief-covid-2021/#more) 没有信心在今年晚些时候继续还款，你可能想现在就制定一个计划，因为没有保证学生贷款延期会继续或债务豁免会提供。

## 评估你现在能支付多少

自从学生贷款延期实施以来，很多事情都发生了变化，这可能包括你的收入和你如何花钱。例如，你可能会挪用学生贷款的专款来填补你的应急基金，或者像许多美国人一样，用 [来偿还高息信用卡债务](https://www.cnbc.com/2021/03/08/americans-paid-off-a-record-83-billion-in-credit-card-debt-in-2020.html) 。

现在的挑战是，在不让其他高利率债务失控的情况下，再次为最低学生贷款还款腾出空间。看看你的银行账户，列出你的支出和收入，然后使用[StudentAid.gov贷款模拟工具](https://studentaid.gov/loan-simulator/) 重新评估你的还款策略(关于如何使用它的更多信息， [查看这篇Lifehacker帖子](https://twocents.lifehacker.com/improve-your-student-loan-strategy-with-this-free-tool-1844299139) )。

如果你的情况发生了巨大变化，你没有赚到任何钱，你仍然可以从你的贷款人那里获得非疫情选项，例如继续延期或 [收入驱动的还款计划](https://studentaid.gov/manage-loans/repayment/plans/income-driven) ，这些计划将贷款支付与你当前的收入挂钩(你可能希望推迟这些选项，直到拜登总统在贷款豁免方面的意图更加明确，即使获得1万美元或5万美元的债务豁免是一个很大的机会)。

### **考虑为你的贷款再融资**

现在贷款利率很低，所以如果你能利用私人贷款机构的固定利率再融资来降低你的APR和月供，这可能是一个不错的选择。 [正如Nerdwallet解释的](https://www.nerdwallet.com/article/loans/student-loans/should-you-refinance-federal-student-loans) :

> 例如，假设你欠了30，000美元，利率为7%，还款期限为10年。以3%的利率进行再融资——这大概是你能想到的最好利率——将为你节省近7000美元。

然而，这应该非常谨慎，因为向私人贷款人再贷款会使你失去联邦学生贷款保护的资格，就像上面描述的收入偿还计划一样。 [根据《消费者报告》](https://www.consumerfinance.gov/ask-cfpb/should-i-refinance-my-federal-student-loan-into-a-private-student-loan-with-a-lower-rate-en-1687/#:~:text=If%20you%20refinance%20your%20federal,get%20with%20federal%20student%20loans.) ，私人贷款未担保的其他联邦保护包括基于您的就业、公共服务或永久残疾的债务豁免。

也就是说，如果你财务安全，已经在偿还贷款的轨道上，有稳定的工作，只是想还清剩余的债务，再融资是一个很好的选择。

### **考虑推迟提前付款**

正如我们在以前的帖子中指出的，如果你的财务状况稳定，在10月暂停期结束之前推迟提前还款可能是一个更好的选择。考虑这些选项:

*   [将每月付款存入储蓄账户](https://twocents.lifehacker.com/should-you-plan-for-50-000-in-student-loans-forgivenes-1846632320) ，然后在学生贷款延期到期时一次性付清。这样，你就有了紧急情况下的现金，如果有任何贷款豁免，你就不会多付欠款(例如，如上所述，如果拜登总统后来豁免了5万美元的余额，那就浪费了这笔钱，尽管这不太可能)。
*   考虑投资一部分钱到退休计划中，特别是如果你没有退休计划的话，前提是当你的学生贷款重新开始时，你仍然可以支付最低还款额。由于退休储蓄会随着时间的推移而复利，而学生贷款通常是低息贷款，你实际上可以通过将钱投资于这两者来提前获利( [这位生活黑客详细介绍了这一点](https://twocents.lifehacker.com/should-you-pay-student-loans-or-save-for-retirement-whi-1846717247) )。代价是你要花更长的时间来还清学生贷款。
