# Airbnb 的最佳度假租赁选择

> 原文：<https://lifehacker.com/the-best-vacation-rental-alternatives-to-airbnb-1849088885>

上周，喜剧演员 Caleb Hearon 在 Twitter 上直言不讳地表达了他对 Airbnb、[proclaiming](https://twitter.com/calebsaysthings/status/1536559315460337664)T8】的不满，“我受够 airbnb lmao 了，这些混蛋现在有宵禁、安静时间和家务清单。我会在一家酒店。”这条推文引起了一点争议(33K 转发和攀升)，其中 [一条 ctress 茶色纽森齐明，说](https://twitter.com/TrondyNewman/status/1536559836942548992) ，“我住的最后一家(Airbnb)，店主拒绝了我的送货员，理由是她的‘不接待客人’政策。作家凯茜·圣·昂格讲述了一次，她的 Airbnb 主人让她 [帮她抓住她逃跑的猫。](https://twitter.com/Caissie/status/1536561795590082562) 



Airbnb 的系统性批评(及其对当地房地产市场 的 [影响)并不新鲜，但越来越多的是，隐藏的预订费用、不合理的规则和清洁要求使得这项服务对租房者来说不再像过去那样有用——它的主要竞争对手也是如此，比如 Vrbo，它也承担着类似的高额附加成本。尽管 Airbnb 及其同类公司无处不在，但它们并不是城里唯一的游戏。这里有三种预订选择，下次你安排旅行时可以考虑一下。](https://www.curbed.com/2022/05/new-york-more-airbnb-listings-apartments-rentals.html)

[](https://www.bookdirectvacationrental.com/frequently-asked-questions/)****预订直接度假租赁。**预订直接度假租赁是一家第三方供应商，它将旅行者与度假租赁物业的所有者联系起来，这样他们就可以直接向他们预订，其想法是双方都可以通过消除中间人来获得经济利益。美国消费者新闻与商业频道写道:“直接预订网站为房东提供了一个渠道，他们可以以更低的价格将房产出租给客人，同时比有 Airbnb 这样的中间人参与时获得更高的利润。”大多数 Airbnb 的客人会为他们的预订支付大约 14%的服务费，业主通常会支付 3%。 通过直接与产权人打交道，还可以用 Airbnb 不允许的方式协商费率、组织物流。重要的是在签署之前仔细阅读与业主的租赁协议，因为每个家庭仍然有自己的取消政策和房屋规则。**

**[**漏斗。**](https://www.hopper.com/deal-drops) 如果你的年龄足够大，还记得 Airbnb 存在之前的某个时期，trav elers 曾经住在这些被称为“酒店”的疯狂建筑里。如果你对 Airbnb 的附加费用和清洁政策感到失望，也许是时候考虑回归传统了。如果你确实选择了酒店，那就去看看 Hopper 吧，这是一款将旅行者与打折酒店房间联系起来的应用。**

**根据众多 R eddit 用户的说法，Hopper 的最佳用途实际上不是通过应用程序本身进行预订，而是使用它来查看哪些酒店提供折扣，然后直接打电话给这些酒店进行预订。这样，如果出现问题，你可以确保有竞争力的价格，而不必通过 Hopper 客户服务(通常不像酒店本身提供的服务那么强大)。最大的好处:总的来说，酒店的取消政策比航空旅馆更好。你可以节省费用，甚至有人会每天早上为你整理床铺。**

**[**民宿。**](https://www.homestay.com/) 如果你想省钱，并在外国感受当地文化，你是寄宿家庭的理想顾客，这种家庭将你与对接待游客感兴趣的家庭或联系起来。。如果这听起来不像你的速度，考虑一下:通过和当地人呆在一起，你将接触到了解你所访问的社区的人，他将能够推荐最好的餐馆，并且告诉你旅游陷阱以避免。寄宿家庭确保他们所有的主人都经过当地合作伙伴的审查，以确保所列的个人资料与描述相符。你也可以通读主持人的评论，看看过去的嘉宾都说了些什么。此外，在 [巴塞罗那](https://www.homestay.com/homestays/search?utf8=%E2%9C%93&search_type=search_box&latitude=&longitude=&country_code=&ne_lat=&ne_lng=&sw_lat=&sw_lng=&radius=&type=&order=&location_id=32&google_place_id=&price_filter_nights=&location=Barcelona%2C+Catalonia%2C+Spain&check_in=09+Jul+2022&check_out=12+Jul+2022&guests=1) 、 [悉尼、](https://www.homestay.com/homestays/search?utf8=%E2%9C%93&search_type=search_box&latitude=&longitude=&country_code=&ne_lat=&ne_lng=&sw_lat=&sw_lng=&radius=&type=&order=&location_id=178&google_place_id=&price_filter_nights=&location=Sydney%2C+New+South+Wales%2C+Australia&check_in=09+Jul+2022&check_out=12+Jul+2022&guests=1) 和其他国外大城市有几十种预订选择，每晚价格不到 50 美元，这个价格你很难在 Airbnb 上看到。每次预订都包括 15%的费用，但考虑到价格要低得多，它通常会比 Airbnb 的固定费用便宜很多。**
