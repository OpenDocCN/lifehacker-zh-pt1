# 你应该现在就下载 Wordle，免费玩几年

> 原文：<https://lifehacker.com/you-should-download-wordle-now-to-play-it-free-for-year-1848473800>

[Wordle](https://www.powerlanguage.co.uk/wordle/) 已经成为我们很多人的日常困扰；每天花几分钟时间尝试在六次或更少的猜测中破译 Wordle 单词是很好的(今天花了我五次，让我感到非常羞愧)。虽然这款游戏到目前为止都是免费的，但《纽约时报》最近对它的收购引起了人们对 Wordle 可访问性未来的担忧。



[T 何次已说](https://www.nytimes.com/2022/01/31/business/media/new-york-times-wordle.html) 字儿将“最初”继续自由发挥。但是“最初”是相当主观的，而且《纽约时报》已经提供了游戏应用的付费订阅，所以我们可能都对 Wordle 的未来感到有点紧张。幸运的是，现在有一种简单的方法来保护这个游戏。

正如 Aaron Rieke 在 Twitter 上展示的那样，Wordle 是一款轻量级游戏，完全可以在你的浏览器上玩。因此，游戏的单词库，包括过去、现在和未来的答案，完全存储在 HTML 中。如果你看一下这些代码，你可以看到明天、下周、明年的答案；这都是板上钉钉的事。

所以，你需要做的就是把 HTML 文件下载到你的电脑或手机上；当你打开该文件时，它会在你的浏览器中启动一个新的游戏，并像实时网站一样运行。不管有没有互联网都可以，所以如果(或当)第次锁定了付费墙后的网站，你可以简单地禁用 wifi 并从你下载的文件中播放 Wordle。而且，由于游戏的答案键 中有[2135 个单词，理论上你可以玩这个游戏 7 年。](https://gizmodo.com/how-to-play-the-original-wordle-offline-1848463004) 

不幸的是，缺点是下载 Wordle 不会延续你的优势。然而，对于 Wordle 内心的平静来说，这只是一个小小的代价。

## 如何把 word 下载到我们的 T2 电脑上

如果开发者增加了下载按钮，在你的桌面上下载 Wordle 会变得更加容易。 [只需打开游戏](https://www.powerlanguage.co.uk/wordle/) ，在窗口点击右键，然后将文件保存到你的电脑中。就是这样！如果你需要使用游戏，双击文件在新的浏览器窗口中打开它。

## 如何下载 Wordle 到我们的 iPhone 或 Android 上

在智能手机上下载 Wordle 也很容易。如果你有安卓设备，打开你的浏览器(比如 Chrome)，然后点击右上角的三个点。现在，点击这个菜单顶部的下载图标，Chrome 就会把 Wordle 下载到你的手机上。要在未来访问该文件，再次点击三个点，选择“**下载**”，然后点击 Wordle，这将在离线窗口中打开游戏。

iPhone 或 iPad 上的情况类似，但是， [正如 CNET 的尼尔森·阿吉拉尔强调的](https://www.cnet.com/how-to/download-wordle-to-your-computer-or-phone-to-keep-playing-for-free/?UniqueID=FC493DD0-84ED-11EC-93FE-40CF4744363C&ftag=COS-05-10aaa0b&ServiceType=twitter&PostType=link&TheTime=2022-02-03T12:37:01) ，你将无法使用 Safari 或 Chrome 来玩 Wordle。虽然你可以用 Safari 保存游戏，但出于某种原因，似乎只有微软 Edge 能打开 Wordle 如果你在 iOS 或 iPadOS 上没有这款浏览器，你可能想下载它作为一个专用的离线 Wordle 工具。

首先在 Safari 中打开 Wordle，点击共享图标，然后点击小的“**选项**按钮。在这个页面上，选择**网页存档**，然后**搞定**接下来，选择“**保存到文件**”，点击要保存文件的文件夹，然后点击“**保存**”

在你的 iPhone 或 iPad 上安装 Edge 后，进入你的文件应用，长按 Wordle 文件，然后选择“**分享**从这里，点击“Edge”，然后选择“**在微软 Edge** 中打开”来玩游戏。让苹果来做一些比它需要的更复杂的事情吧。

虽然这种方法可以确保你在未来的几年里跟上时代的步伐，但是不要害怕回到过去 。Wordle 档案可以让你随时玩过去的游戏；让足够的时间过去，旧词突然又变成新词。
