# 让我们跑 200 米短跑吧

> 原文：<https://lifehacker.com/lets-run-some-200-meter-sprints-1845182400>

欢迎回来，跑友们！(记住， [我们都是这个月的跑者](https://vitals.lifehacker.com/go-somewhere-new-for-a-run-or-hike-1844954732) 。)在本周的 [健身挑战](https://lifehacker.com/c/lifehacker-fitness-challenge) 中，我们将跳过 [俯卧撑](https://vitals.lifehacker.com/break-up-your-run-with-pushups-for-extra-fun-and-suffer-1845105210) 和 [丘陵](https://vitals.lifehacker.com/tackle-a-hill-head-on-1845026700) 并专注于跑步的有趣部分:在平坦的跑道上快速前进，全程看着终点线。



这是我最喜欢的跑步锻炼:200 米重复跑。如果你附近的高中或中学有跑道，很可能是 400 米。这意味着我们只能跑半圈。就是这样。(我的意思是，我们将多次这样做，但不要操之过急。)

如果你知道起跑线和终点线在赛道上的位置，那就去使用它们。(终点将在直道的尽头，在对面的弯角交错发车。)但其实并不重要；随便挑个地方。

如果你没有可用的赛道，就找一条平坦的道路或停车场，在你的手表或手机上设置一个一分钟计时器。

现在你该这么做。去跑道散步或慢跑几圈来热身。( [)任何跑步的前 10 分钟都是最糟糕的](https://vitals.lifehacker.com/the-first-mile-always-sucks-let-it-go-1752457839) ，所以在你要求你的身体做任何艰难的事情之前，先把这些事情做完。)然后，只跑半圈。

如果你一直对 [跑慢是跑多](https://lifehacker.com/you-need-to-run-slower-1844984642) 的关键这一事实感到恼火，现在是你大放异彩的时候了。忘了慢吧——你要尽可能快地跑完那半圈*。如果你愿意，你可以全力以赴。如果你想有战略眼光，稍微收敛一点，比你的最高速度低一两级。我觉得 200 米的速度很有趣。很快。你感觉你在飞。*

*穿过终点线后，慢慢地走。从你的水瓶里喝一口。然后走完剩下的一圈，或者沿着停车场原路返回，直到到达起跑线。到那时，你应该已经恢复得很好了，可以再次出发了。(如果不是，就多休息一会儿。当你只是为了好玩而跑步时，没有教练会对你大喊大叫。)*

*试试这个训练，看看你是否和我一样喜欢它。如果你不经常做间歇跑，就做几次这样的短跑——也许四次——并在一周的剩余时间里坚持轻松跑。以一些更轻松的慢跑结束锻炼，并享受应得的休息。*
