# 千万不要忽视这些类型的背痛

> 原文：<https://lifehacker.com/never-ignore-these-types-of-back-pain-1849447675>

通常感觉好像过了某个年龄，背痛就成了生活中的事实。这种不适不一定是不可避免的:你可以通过 [最佳伸展](https://lifehacker.com/try-these-stretches-for-an-achy-back-or-neck-1848222633) 和 [瑜伽动作来缓解背痛](https://lifehacker.com/these-yoga-moves-will-actually-relieve-back-pain-1796335626) 来抑制疼痛。不幸的是，因为背痛被视为如此普遍，许多人很快就忽略了可能有更严重的事情发生的迹象。这里有一些迹象表明你的背痛不寻常，应该去看医生或理疗师:

*   **背部疼痛延伸到你的侧面/上腹部:**如 [AICA 骨科](https://aica.com/stomach-and-back-pain-at-the-same-time/#:~:text=Pancreatitis,by%20sharp%20movements%20like%20coughing.) 解释说，虽然你的背部和腹部同时疼痛可能是巧合，但它也可能是更大的潜在疾病的迹象。如果你的背痛辐射到你的前部或侧部，这可能表明胰腺炎，阑尾炎或肾结石等疾病。为了减轻你的痛苦，需要进行适当的诊断。
*   **颈部和腿部的上背部疼痛:**颈部和背部疼痛经常相伴而生，可能是一些轻微疾病的迹象，如不良姿势。另一方面，根据约翰霍普金斯医学的说法，从颈部传来的背痛可能是神经受损的迹象。脊椎神经受压的其他迹象包括虚弱、麻木或从背部传到腿部的剧烈刺痛。
*   **伴有“发麻”的疼痛:**类似于上一点，如果你的背部疼痛伴有任何麻木或刺痛(那种“发麻”的感觉)，那么你应该检查一下，以确保你的脊髓没有问题。
*   **失去膀胱控制的疼痛:** [根据 Healthline](https://www.healthline.com/health/overactive-bladder/back-pain-and-incontinence) 的说法，因为你的背部肌肉和神经离你的膀胱如此之近，尿失禁和背部疼痛经常联系在一起。如果你的膀胱有问题，你可能会感到背部疼痛；同样，如果你的背部有问题，你可能会尿失禁。这是不容忽视的，因为失去膀胱控制可能意味着医疗紧急情况，如硬膜外血肿或严重的椎间盘突出。
*   **发烧带来的疼痛:**发烧可能是更严重的疾病的迹象，如感染，大卫·安德森，一位在 [北卡罗来纳](https://www.orthocarolina.com/physicians/david-t-anderson-md) ， [的脊柱外科医生告诉赫芬顿邮报](https://www.huffpost.com/entry/back-pains-never-ignore_l_5c87fa72e4b038892f47fee6) 。幸运的是，安德森说这是罕见的，发烧和背痛没有必然联系。然而，持续的背部发烧是去看医生的原因。
*   持续时间超过十天的疼痛:一般来说，你不想忽视任何一种持续时间超过十天的疼痛——尤其是如果疼痛只会越来越严重。你等待解决疼痛的时间越长，最终治愈它的时间就越长。

Watch

这已被正常化，以推动日常背部疼痛，但任何似乎不寻常的事情都可能是你需要医疗护理的迹象。如果你感到麻木或刺痛、虚弱、膀胱失控、发烧或疼痛从背部蔓延到身体其他部位，请去看医生。不要等到疼痛自己消失，否则你可能会忽略一个更严重的潜在问题。
