# 如何计划虚拟新娘送礼会

> 原文：<https://lifehacker.com/how-to-plan-a-virtual-bridal-shower-1845753323>

结婚已经是压力最大的事情之一你可以花上几千美元让自己承受——就像生孩子一样。婚前新娘送礼会应该给你一个短暂的喘息，暴风雨前的平静，帮助你记住你所做的一切都是为了爱，婚礼真的应该是一个大派对。但是仅仅因为没有被完全取消的婚礼已经在疫情期间 [被缩减规模或者在网上](https://digiday.com/marketing/the-biggest-theme-is-uncertainty-how-coronavirus-has-changed-the-wedding-industry/) 举行，这并不意味着你必须放弃新娘送礼会的乐趣。以下是如何计划一个庆祝虚拟新娘送礼会。

### 保持传统的活力

传统是许多新娘送礼会的重要组成部分，但在网上进行时，它们需要调整。通常情况下， [伴娘计划淋浴](https://www.marthastewart.com/7964646/bridal-shower-hosting-rules#:~:text=The%20maid%20of%20honor%20traditionally,guests%20are%20taken%20care%20of.) 活动，这是为你的数字淋浴保持有益的做法。让一个人负责协调将会给你的社交活动带来团结一致的感觉。此人可以发送邀请；寻找、收集和计划淋浴活动；并回答与会者的问题。他们还可以管理缩放或视频聊天，包括允许人们进入房间，协调屏幕共享，以及在出现任何技术问题时进行故障排除。

### 淋浴游戏和活动是关键

让你的沐浴者参与进来实际上比面对面更重要。人们可以面对面交流，嘘嘘，祝贺新娘，或者只是闲逛。在视频聊天中，所有人的眼睛都面向前方，注意力都集中在小盒子里的场景上。没有结构，这可能会非常尴尬。提前计划一些游戏和话题，让客人参与进来，把注意力集中在主宾身上。这里有几个让你开始:

#### 鸡尾酒混合

这可以在你伴娘的指导下完成。为活动选择一杯官方鸡尾酒，提前发给所有客人，这样他们就有机会在活动开始前收集配料。如果调酒师不是伴娘的强项，可以考虑邀请一位调酒师来参加庆祝活动，他可以指导客人如何制作招牌饮料。(别忘了提供不含酒精的替代品。)

#### 玩几轮新娘的 Libs

还记得 [狂 Libs](https://www.madlibs.com/) 吗？在游戏中，通过用玩家选择的随机单词替换名词、动词、副词和形容词来构建有趣的故事，这种游戏可以根据您的在线活动进行修改。使用相同的格式来创建一个新娘或情侣主题的故事，以此来吸引你的与会者。你可以下载 [样本新娘库](https://modernmoh.com/bridal-shower/bridal-shower-mad-libs/) 来帮助你开始。

### 为演讲和祝酒留出时间

在婚礼的某个时刻，你会想给聚集在一起庆祝新娘的朋友和家人一个机会...庆祝新娘。列出几个想要分享几句话、一句祝酒词或一个难忘的故事的人，或者只是提供良好的想法和良好的祝愿的人，让他们提前足够长的时间了解，以计划他们想说的话 。你可以通过事先采购视频来使它变得特别，从照片和视频拼贴画到远道而来的客人的录制亮相，这些客人通常不会在喧闹的新娘送礼会上站起来说话。一定要提前计划好淋浴的这一部分，让事情有条不紊地进行。

### 让新娘炫耀她们的礼物

确保提前提供 [新娘送礼会登记处](https://www.blueprintregistry.com/guides/sample-registries/sample-wedding-registries/bridal-shower-sample-wedding-registry) 的链接，以便有时间购买礼物并及时到达在线活动现场。让每个人都把礼物包好，这样新娘就可以在镜头前打开——看到新娘对礼物的反应总是令人满意的。

### 以一个伤感的时刻结束

一个典型的新娘送礼会将留给尊贵的客人一堆照片来浏览以记住这个夜晚，但笔记本电脑屏幕上一堆面孔的照片可能不会感觉特别。事先计划一张主题集体照——每个人分享给新娘的信息，他们可以举着相机——这可以作为这个非常怪异，但仍然非常特别的事件的纪念品。
