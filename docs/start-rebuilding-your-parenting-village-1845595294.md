# 开始重建你的育儿村

> 原文：<https://lifehacker.com/start-rebuilding-your-parenting-village-1845595294>

最近有很多关于许多父母感受到的深深的孤立的文章，特别是那些在家全职工作的父母，他们的孩子也在家学习。我一直避免点击其中的大部分，因为纠结于此似乎是徒劳的。当然，我们是孤立的，感觉就像没有什么可以做的，直到疫情结束。

Watch

但我忍不住读了《纽约时报》的一篇文章，标题是“ [”育儿从来不意味着这种孤立的](https://www.nytimes.com/2020/10/07/parenting/childcare-history-family.html) ，”这篇文章回击了《纽约时报》另一篇文章上的某些评论者，他们基本上是这样的，“嗯，你们决定要孩子了。那是你的选择，现在忍着点，别抱怨了。”杰西卡·格罗斯写道:

> 这种批评没有抓住的是，基本上在整个人类历史上，父母从来没有像他们在 2020 年的大部分时间里那样在孤立的核心单元中抚养孩子，几乎没有或根本没有家庭或社区的支持。华盛顿州奥林匹亚常青州立学院的历史和家庭名誉教授斯蒂芬妮·孔茨说，单个家庭对孩子完全负责“绝对是闻所未闻的，除非在完全紧急的情况下”。，也是《我们从未有过的样子:美国家庭和怀旧陷阱》的作者

作为独生子女的父母——除了我们是养父母的那几年——我和我丈夫不需要太依赖一个大村庄。我们的人数比我们的孩子多，可以很容易地在需要时放弃工作或休息。即便如此,《纽约时报》的这篇文章还是让我想起了在非疫情时代，我错过的所有关于为人父母的小事:我们会与儿子朋友的父母协调，轮流拼车去参加放学后的活动。两个孩子的父母会把一个孩子丢给我们，这样他们就可以一起去看比赛或者和另一个孩子约会。我们把儿子交给他们，这样我们就可以——气喘吁吁地——一起出去吃午饭或晚饭。

我们必须不断提醒自己，我们正在经历的事情是艰难的，因为正如《泰晤士报》标题所指出的，为人父母从来不应该是这样的。人类是一个合作的物种——我们注定要一起生活和工作，一起抚养我们的孩子。我们曾经为彼此做了那么多，但现在感觉我们根本做不到。

然而，我们面前还有一个漫长的冬天，在这个冬天里，甚至所有我们设法在孤立中生存下来的户外、远离社交的工作场所都将消失。我们必须找到一种方法，成为彼此情感上的支持，即使我们不能成为一个身体上的支持系统。

所以我的建议是，我们今天都试着花几分钟——明天再花几分钟——开始重建我们今年失去的一些养育之村。如果你习惯了在去游乐场的路上为另一位妈妈买咖啡，那么今天就买一杯她最喜欢的饮料，放在她的前门。如果你去杂货店，问问你的单亲爸爸邻居是否需要什么。如果你附近的天气允许的话，提议监督一次户外活动。

与你的其他家长朋友创建一个群文本，并将其称为“反 Instagram”群。对于这个群，你将*而不是*发送你当前生活中最可爱、最温暖、*最不具代表性的*照片。不，你将只发送 Zoom 崩溃和成堆的脏衣服的照片以及任何其他可以说明这一点的照片，是的，我们现在仍然生活在地狱中，但我们在一起，即使感觉我们相距如此之远。

在这个疫情，每天的生活感觉就像是每个人都在为自己着想，但是我们仍然需要彼此。所有关于这是我们“新常态”的笑话(或辞职)都不能改变这个事实。我们现在能做的任何一件小事都将帮助我们度过未来的岁月。
