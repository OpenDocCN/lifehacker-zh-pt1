制造商的新虚拟队列让你注册一个候补名单，保证你有机会购买RTX 3080一旦有更多的可用。你不需要在电脑或智能手机上打开网页，也不需要不断刷新浏览器。当最后轮到你时，你会收到一封电子邮件通知，RTX 3080有货，EVGA会为你预留一台。然而，如果你没有在八个小时的窗口内买下它，预约就会移到下一个排队的人身上。

你可以 [在这里](https://www.evga.com/products/productlist.aspx?type=0&family=GeForce+30+Series+Family&chipset=RTX+3080) 注册EVGA的虚拟队列，尽管任何注册了EVGA库存提醒的人都已经在等待名单上了。您用于虚拟队列的电子邮件必须与您的EVGA商店帐户匹配，才能在结账时购买GPU。

预订是按照注册的顺序进行的，所以你越快加入虚拟队列，就能越快买到你想要的RTX 3080。虚拟排队系统比大多数零售商使用的免费模式好得多，但EVGA警告说，你可能至少要等到今年年底才能有机会购买RTX 3080。

如果你没有耐心，RTX 3080可以从其他几家零售商那里买到，但如果没有保证预订，在售罄之前成功购买一台将会困难得多。以下是一些有用的提示:

*   用 [Nowinstock](https://www.nowinstock.net) 注册补货通知，加上你可以直接从你使用的网上商店得到的任何提醒。
*   使用 [Visualping.io](https://visualping.io) 监控网站，并在商店列表更新时接收通知。
*   为您使用的每个商店创建一个帐户，并尽可能多地保持登录状态。
*   将您的所有结账信息保存在浏览器的自动填充或密码管理器中，以使购买过程尽可能快。
*   想想快乐的事情。总有一天你会得到那个显卡的。
