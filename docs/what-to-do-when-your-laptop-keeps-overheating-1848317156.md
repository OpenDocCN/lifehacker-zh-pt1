# 笔记本电脑持续过热时该怎么办

> 原文：<https://lifehacker.com/what-to-do-when-your-laptop-keeps-overheating-1848317156>

如果你的笔记本电脑倾向于过热，你可能很熟悉这些迹象:风扇旋转的声音比喷气式发动机还大，电脑太热，不能放在腿上使用，以及你的机器正在为生存而战的一般感觉。笔记本电脑持续过热是无法忍受的——事实上，你可以使用几种不同的方法来帮助它降温。



## 为什么笔记本电脑过热(以及为什么这是一个问题)

过热的笔记本电脑不仅仅是舒适的问题。如果你的笔记本电脑变得太热，它实际上会阻止机器正常工作。这是因为 CPU、充当电脑的“大脑”、和 GPU(控制电脑图形的芯片)不允许过热。如果让这些处理器不受控制地变得越来越热，将会永久性地损坏或毁坏芯片。

一旦组件达到某一温度(通常高于 90°C ),系统就会开始降低其峰值速度和性能，以试图冷却下来。这对保存那些计算部件很有好处，因为否则它们可能会被烤焦，但这对你完成任务的能力不是很好。

如果你只是发送电子邮件和做一些轻松的浏览，你可能不会注意到节流太多。但是，如果你依靠电脑的速度来完成工作，这种速度变慢可能会很痛苦。

## 如何让你的笔记本保持凉爽

有很多方法可以让你的笔记本电脑保持凉爽；虽然不是所有的方法都适用于所有的机器，但是这些步骤可以帮助降低计算机的温度，同时提高整体性能。

### 检查风扇

许多笔记本电脑保持组件冷却的方式之一是通过风扇。当风扇高速运转时，你的电脑可能会特别吵闹。这里的想法是从房间中抽取冷空气，并从机器中抽出热空气。然而，随着时间的推移，灰尘在风扇上堆积，最终削弱了风扇将空气进出笔记本电脑的能力。风扇超时工作，组件不变冷，所以你的机器节流以减少热量。

第一步应该是用一罐压缩空气清理笔记本电脑的通风口。如果灰尘和碎片不太严重，这种非侵入性的程序可以快速帮助您的笔记本电脑再次开始冷却。

如果事情仍然太热，你的笔记本电脑很容易打开，你可以使用风扇。从这里开始，你可以用刷子、压缩空气或真空吸尘器轻轻*推开灰尘和碎片。如果你不知道如何正确打开你的笔记本电脑，网上可能有关于如何打开你的特定型号的教程。*

*当然，并不是所有的笔记本电脑都有风扇。如果你有一台 MacBook Air、Chromebook 或其他轻薄笔记本电脑没有风扇冷却功能，你会想在这里寻找其他替代品。*

### *注意你的表面*

*您的笔记本电脑渴望平坦、凉爽的表面以获得最佳性能。如果它有风扇，这有助于气流，但当笔记本电脑接触这样的表面时，热量会更容易消散。*

*然而，柔软、不平整的表面，如床单、枕头或地毯，不利于热量管理。柔软的材料会堵塞通风孔，阻碍风扇使空气通过机器的能力。这些材料在散热方面也很差，这意味着热量会在你的笔记本电脑中积聚。*

*一般来说，把你的笔记本电脑放在坚硬的表面——桌子、柜台、书桌等。你也可以将笔记本电脑悬挂在支架上，或者使用 [笔记本电脑冷却垫](https://www.amazon.com/s?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/what-to-do-when-your-laptop-keeps-overheating-1848317156&asc_source=&crid=1PLAK9WY0JYL8&k=laptop cooling pad&ref=nb_sb_noss&sprefix=laptop cooling pa,aps,75&tag=kinjalifehackerlink-20) 来帮助促进空气流通。*

### *导热膏和导热垫*

*导热膏允许热量从 CPU 和 GPU 更有效地转移到散热器，从而保持温度更低。另一方面，散热垫有助于将热量排出散热器，因此这两种散热工具可以很好地配合使用。让我们从导热膏开始:虽然所有的笔记本电脑从一开始就带有导热膏，但随着时间的推移，导热膏会变弱。不仅如此，有时在制造过程中使用的浆糊并不理想，或者没有被有效地应用，所以你从一开始就处于不利地位。*

*重新贴上热敏贴可不是件容易的事，，尤其是如果你不熟悉打开笔记本电脑和在里面修修补补的话。一旦你打开你的笔记本电脑，访问 CPU 和 GPU，你需要移除先前的热粘贴层，并应用一个新的外套。*

*您可以将热粘贴应用到几乎任何带有 CPU 和 GPU 的设备上。然而，这些装置越难打开，过程就越棘手。有些笔记本电脑设计得很容易操作；其他的，比如 MacBooks，是出了名的难以打开。更糟糕的是，大多数 MacBooks 的 CPU 和 GPU 都背向背板，这意味着你需要拆开整个机器，然后翻转主板，才能接触到处理器。*

*另一方面，使用热垫通常要容易得多。Linus Tech Tips 通过简单地在设备的散热片上添加散热垫，提高了无风扇 M1 MacBook Air 的性能。虽然这样做大大提高了 MacBook Air 底座的温度，但在压力测试中，它的风扇冷却系统实际上胜过了 M1 MacBook Pro。*

*无论您是想添加导热垫、使用导热膏，还是两者兼而有之，您最好的选择是查找适用于您特定笔记本电脑的教程，就像任何维修一样。*

### *少做*

*对我们来说，推动电脑是很自然的事情，尤其是当我们同时在做多件事情的时候。上述方法应该会自然地提高你的设备性能，所以你也许可以运行更多的应用程序。然而，即使有了这些冷却策略，您的笔记本电脑也可能无法处理您在任何给定时间点运行的大量程序。*

*例如，如果你打开了太多的 Chrome 标签，试着只打开你真正需要的标签。如果你在工作的时候不需要来让你的*模拟人生*游戏在后台运行，那就让它一直关着，直到下次再玩的时候。如果你想编辑视频，但过热会导致速度变慢，请尝试关闭所有应用程序，但编辑项目所需的程序除外。*

*注意你如何使用你的笔记本电脑可以大大有助于保持凉爽，尤其是你保持你的机器时间越长。*
