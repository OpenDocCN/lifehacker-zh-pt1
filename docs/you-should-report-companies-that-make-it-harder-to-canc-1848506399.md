# 你应该举报那些取消比注册更难的公司

> 原文:[https://life hacker . com/you-should-report-companies-that-make-that-that-that ' s-that-hard-cancer-1848506399](https://lifehacker.com/you-should-report-companies-that-make-it-harder-to-canc-1848506399)

很少有公司会让注册他们的服务变得困难。事实上，许多公司让注册变得太容易了，向潜在的客户提供免费试用、折扣和其他诱人的额外待遇。然而，*真正卑鄙的是那些公司让取消你的会员资格变得几乎不可能。* 

Watch

这不仅仅是我的观点；联邦贸易委员会同意如果一家公司提供一种注册服务的方法，他们也应该提供同样的取消方法。你不应该只点击几下就能订阅一个平台，然后不得不在电话旁坐上几个小时，等待销售代表试图说服你留下来。那是错误的。

在美国，很多公司都使用这种策略，尤其是新闻媒体和出版商。根据美国新闻学会 的调查，只有41%的受访出版商提供了在线取消订阅的简便方法。《纽约时报》是这种做法的罪魁祸首，T4的许多地方和全国性报纸也是如此。

幸运的是，时代在变。联邦贸易委员会去年宣布将打击那些取消服务比注册服务更难的公司。正如美国联邦贸易委员会消费者保护局局长塞缪尔·莱文所说:

> “今天的执行政策声明明确指出，欺骗消费者注册订阅程序或在他们试图取消时诱骗他们是违法的...采用黑暗模式和其他肮脏伎俩的公司应该引起注意。”

通过这一声明，联邦贸易委员会明确表示，公司必须遵守三大支柱:

*   弄清楚所有相关的财务信息，例如正在销售的产品的总成本以及在客户被收费之前取消的最后期限。
*   在向客户收取上述服务费用之前，确保客户已经同意。
*   对于本文来说，最重要的是，公司必须为客户提供简单易行的取消步骤。取消方法应该和注册产品的方法一样简单。

即使有了这些规定，许多公司仍然在使用卑鄙的手段来使取消服务变得困难。当你遇到这种行为时，你可以做点什么:R 向联邦贸易委员会举报公司。

为此，负责人向【https://reportfraud.ftc.gov/#/】和[填写报告。你做了之后，](https://reportfraud.ftc.gov/#/) [不仅会和FTC](https://vm.tiktok.com/TTPdkvbqor/) 共享，还会和全国3000多名执法者共享。

现在说这些报告在改变企业行为方面有多快或多有效还为时过早。目前还不清楚联邦贸易委员会是否会针对特定行业，而不干涉其他行业。出版商是明显的违规者，但许多健身房也是如此，他们提供简单的注册套餐，同时让顾客陷入无法取消的会员资格。希望联邦贸易委员会能像这样轻易地取消所有类型的订阅和会员资格，迫使公司寻找其他(可能是T2的)道德方法来留住顾客。
