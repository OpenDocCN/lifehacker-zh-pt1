# 报告:PlayStation 规划 Xbox 游戏通行证式订阅服务

> 原文：<https://kotaku.com/report-playstation-planning-xbox-game-pass-style-subsc-1848157416>

索尼正在敲定新的 PlayStation 订阅，旨在为该平台提供自己的 Xbox 游戏通行证式服务，暂定于明年春天推出，[*报道*](https://www.bloomberg.com/news/articles/2021-12-03/playstation-plans-new-service-to-take-on-xbox-game-pass)*。* 

*Watch*

*目前的计划显然是将 PlayStation 基于云的游戏服务 PlayStation Now 与 PlayStation Plus 的每月免费游戏产品相结合，逐步淘汰前者，支持后者更突出的品牌。根据彭博看到的索尼内部文件，这一新订阅(代号为斯巴达克斯)的月费将为玩家提供无限制的现代和经典 PlayStation 游戏库。*

*Kotaku 联系索尼寻求评论，但在出版前没有得到回复。*

*就像 Xbox Game Pass 一样，斯巴达克斯订阅也将根据用户愿意花多少钱来提供各种口味。最便宜的一层应该看起来很像当前的 PlayStation Plus 服务，第二层将添加 PlayStation 4 和(最终)PlayStation 5 游戏，第三层将提供扩展的演示和游戏流媒体以及 PlayStation、PlayStation 2、PlayStation 3，甚至 PlayStation Portable libraries。*

*预计《斯巴达克斯》也将在 PlayStation 4 和 PlayStation 5 上推出，这意味着那些仍未能获得索尼难以找到的下一代游戏机的人不会被冷落。*

*抛开关于时机的扶手椅分析，这对索尼来说是显而易见的。Xbox Game Pass 于 2017 年首次推出，是微软视频游戏部门几十年来制作的最佳产品，以其网飞风格的游戏方式吸引了超过 1800 万名用户。随着各代游戏机的现有安装基础越来越大，以及对这种订阅模式的明显渴望，如果索尼以与微软和 Xbox 相同的热情对待其 PlayStation 计划，Spartacus 肯定会受到欢迎。*

*老实说，这个行业的每一家大公司都应该给我们更多的方法，以合理的价格合法地玩经典游戏。*
