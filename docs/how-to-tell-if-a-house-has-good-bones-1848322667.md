# 如何辨别一所房子是否有“好骨头”

> 原文：<https://lifehacker.com/how-to-tell-if-a-house-has-good-bones-1848322667>

不太新建造的房屋通常被房地产经纪人描述为“好骨头”，这意味着其核心是坚固的结构和高水平的质量和工艺。当然，找房子的时候，你看不到油漆和干墙下面的东西，所以你怎么知道你看中的房子是否有这个令人满意的骨架呢？这里是寻找的地方，寻找什么。

## 屋顶

房子最重要(也是最昂贵)的部分之一是屋顶。在你的研究中，找出你未来新家的屋顶是用什么材料建造的 ( [)石板、水泥瓦和金属比木头或沥青瓦](https://www.aboveroofing.com/how-often-should-you-replace-your-roof-the-complete-guide/) )以及它最后一次被更换的时间。寻找可见的磨损迹象，如下垂、生锈、泄漏、发霉、真菌以及任何 [翘起、缺失、破裂或卷曲的木瓦](https://www.inchcalculator.com/new-asphalt-shingle-roof-warning-signs/) 。随着时间的推移，无人照管的屋顶问题会慢慢影响到房屋内部的损坏，所以保持警惕是值得的，或者至少确保你不会对维修的需要感到惊讶。

## 基金会

基础是字面上的意思保持一个房子在地面上，水平，和绝缘。一个糟糕的基础将会在整个房子里表现出来。留意下垂的地板或天花板，发霉的气味，地板或墙壁上的裂缝，倾斜的烟囱，卡住或关不上的门，以及卡住或关不上的窗户。还要确保地下室的任何柱子或混凝土支撑完全直立——不要倾斜。

## 布局

如果你计划翻新(即使你不打算)，你会想要额外注意平方英尺和目前的平面图。它是开放的还是封闭的(每个单独的区域都被自己的墙围起来)？考虑楼层、浴室和卧室的大小和位置。你希望主卧室在单独的楼层，还是需要靠近小孩的房间？确保洗衣房的位置方便——并评估厨房和浴室中电器和厕所的位置，因为管道和电气工程的改变成本很高。

## 自然光

这栋房子的位置能让充足的自然光进来吗？它可以有一个华丽的厨房，但它被附近有盖的天井或有纱窗的门廊遮住了吗？花点时间弄清楚房子面向哪个方向，以及从日出到日落光线将如何穿过它。(朝南的窗户通常比朝北的窗户采光更好；朝东的窗户享受大量的晨光，而朝西的窗户在下午被阳光亲吻。)

## 潮湿和结构腐烂的迹象

湿气会对房屋结构造成严重破坏。注意家中可见的湿气影响——腐烂的木头、霉菌或霉变、潮湿的爬行空间、天花板上的水渍、起泡的墙纸、扭曲的地板、白蚁损害和发霉的气味都是令人不安的迹象。如果你看到任何墙壁裂缝、天花板翘曲或混凝土腐蚀的迹象，要格外注意，因为它们可能表明结构性问题，修复起来会非常昂贵。

## 请专业人士帮忙

虽然如果你知道要寻找什么，你可以自己收集一些好的信息，但在购买房屋时，没有什么可以替代专业检查。你所在地区的认证检查员将对房屋的地基、屋顶、供暖和制冷系统、阁楼、可见隔热层、天花板和地板以及其他结构元素进行全面分析。您可能希望增加氡、霉菌、害虫和铅的测试。如果检查员没有涵盖电气问题，考虑雇用电工指出潜在的电气问题或建议的升级。
