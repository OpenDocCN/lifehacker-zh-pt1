# 如何在 Windows 11 新的文件浏览器标签发布前试用它

> 原文：<https://lifehacker.com/how-to-try-windows-11s-new-file-explorer-tabs-before-it-1849045449>

Windows 用户认为他们的操作系统拥有比 macOS 的 Finder 更好的文件管理系统。虽然这种说法仍有争议，但 Mac 用户可以在 windows 上使用的一个文件管理功能是 Finder 标签:这些标签可以让你像在网络浏览器中浏览网站一样，在文件窗口之间快速切换，这是 Windows 无法做到的方便高效的事情。至少，直到现在。

[早在 2013 年，随着 Mac OS X Mavericks 的发布，苹果在 Finder](https://lifehacker.com/split-or-merge-finder-tabs-in-os-x-with-a-couple-clicks-1686823201) 中引入了标签。现在，微软在近十年的大部分时间里都没有忽视这个功能:大约四年前，该公司测试了文件浏览器的标签，但最终决定不在操作系统中实现它。然而，在三月份，[《the Verge》报道了](https://www.theverge.com/22970473/microsoft-windows-11-file-explorer-tabs) 微软确实在为未来版本的 Windows 11 恢复该功能。虽然它还不是一个官方特性，但是它已经被一些 Windows 用户所使用，更多的用户也有望使用它。

该公司目前正在测试文件浏览器标签，作为 Windows Insider 预览版的一部分。这个最新版本的 [Build 25136](https://blogs.windows.com/windows-insider/2022/06/09/announcing-windows-11-insider-preview-build-25136/) ，让文件浏览器看起来更像一个网络浏览器:当你打开一个新窗口时，你会在左上角看到它的活动标签，旁边是一个(+)你可以点击打开另一个标签。如果你是一个同时在多个文件浏览器窗口中工作的人，这个标签系统绝对是一个游戏改变者，让移动文件和切换窗口变得轻而易举。

不幸的是，微软还没有向 Insider Preview 项目中的用户全面推出文件浏览器标签——该公司希望在向更多测试人员发布之前收集反馈，但没有时间表。也就是说，现在尝试使用新功能的最佳方式是在 Windows Insider 预览版中注册您的计算机。

然而，提醒一句:注册内幕预览版是要求微软用未完成的软件更新你的电脑。你可以在其他 Windows 用户之前利用新功能，但要冒继承错误或其他问题的风险。您应该考虑在主计算机以外的设备上注册 Insider Preview，或者使用虚拟机来保护您的主要数据。当然，没有什么可以阻止你注册你的电脑，因为现在它只是重要的是要了解风险。

## 如何在 Windows Insider 预览版中注册您的计算机

记住这一点，下面是你注册的方法:进入**开始>设置>Windows Update>Windows Insider 计划**，然后点击“开始”选择“链接帐户”，点按您的 Microsoft 帐户，然后点按三个内部渠道之一进行选择。微软推荐 Beta 频道，但是 Dev 频道是你尽早获得文件浏览器标签的最佳选择。请记住，虽然，这个渠道有最大的风险为错误和其他软件问题。

做出选择后，点按“继续”，然后再次点按“继续”。最后，点击“立即重启”，等待你的电脑重新启动；返回**开始>设置> Windows 更新>立即更新**安装最新软件。如果你没有马上看到这些标签，请耐心等待:希望微软会尽快在你的电脑上推出这项功能。
