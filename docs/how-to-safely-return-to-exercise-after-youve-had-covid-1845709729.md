# 在你得了新冠肺炎病后，如何安全地恢复锻炼

> 原文：<https://lifehacker.com/how-to-safely-return-to-exercise-after-youve-had-covid-1845709729>

通常，生病后恢复锻炼的过程很简单:相信你的身体，放轻松，不要想太多。但是如果你已经感染了冠状病毒，就需要格外小心。

运动员和休闲锻炼者最终会出现挥之不去的并发症，包括动脉血栓和心脏炎症。一些运动员也报告了疲劳和呼吸问题，这些问题在他们从新冠肺炎康复后仍然存在。

纽约特殊外科医院的一组医生公布了一套恢复锻炼 的 [指南。(作者之一乔丹·梅兹尔也在《纽约时报》上](https://link.springer.com/article/10.1007/s11420-020-09777-1) [写了这些指导方针背后的推理。)](https://www.nytimes.com/2020/11/17/well/move/exercise-covid-recovery-complications-clots-heart-lungs.html)

这些建议包括:

*   如果你认为你只是感冒或流感，无论如何，如果可能的话，做新冠肺炎测试。
*   如果你有心脏或肺部疾病，请在恢复锻炼前咨询医生。
*   如果你之前身体健康，并且有轻微的病毒感染，你可以考虑在无症状七天之后恢复锻炼。
*   预计需要一个月的时间才能回到你的完整训练计划中。
*   如果你的症状再次出现，特别是胸痛、发烧、心悸或呼吸困难，停止锻炼并去看医生。

对恢复锻炼的担忧大多与有氧运动和调节活动有关。逐渐增加运动量的建议是从你正常运动量的一半开始(所以，如果你是跑步者，这可能是你正常运动量的一半)。这些指导方针并不具体针对举重，但作者指出了一套非针对 COVID 的指导方针 ，同样建议当你回到举重房时，从轻松开始，然后逐渐增加。
