# 你应该和你的孩子一起开始这个装饰传统

> 原文：<https://lifehacker.com/why-you-should-start-this-ornament-tradition-with-your-1848131382>

在我的成长过程中，我们家没有很强的圣诞装饰传统。或者根本没有，真的。没有手工制作的具有情感价值的小饰品，也没有世代相传的精美纪念品。关于我们的圣诞树，我记得的主要事情是它那滑稽的 80 年代人造的稀疏、僵硬的树枝，以及我在晚饭前偷偷摸摸的糖果“装饰品”。我努力回忆起其他装饰品的样子。

Watch

这就是为什么今天，作为一个有 3 岁、6 岁和 8 岁孩子的家长，当我了解到每年给你的孩子(任)买一件纪念品这一美妙而迷人的传统时，我必须抑制住恐慌和恐惧的感觉。虽然在过去的两年里，我确实有先见之明，让我的孩子从我们购买圣诞树的圣诞商店的小选择中挑选装饰品，但我错过了几年创造个性化、基于童年记忆的装饰传统的关键时间。但是如果对我来说现在开始还不晚，对你来说也不晚。

## 为什么你应该给你的孩子买纪念品？

创建纪念品传统的目的有两个:不仅你，作为父母，每年都可以享受温暖人心的时刻，因为你看到特别的装饰品反映了你孩子生活中的主要事件、主题、经历和兴趣，*他们*可以收集 18 个或更多来开始他们的成年生活！—为他们自己的树做有意义的装饰。(这样，当他们搬出去装饰他们的第一棵树时，他们就不必像我们其他人一样，依赖 Target 的一堆无聊的银球了。)

## 如何开始传统

年度饰品可以自制也可以购买——事实上，为了达到怀旧的效果，我们推荐两者兼而有之。当他们还小的时候，有什么能比用一张甜美、胖乎乎的照片(容易在 [Shutterfly](https://www.shutterfly.com/ornaments/?icid=HOL%7CSub%7CC2%7C100521%7COrnaments%7CHeader) 或 [Snapfish](https://www.snapfish.com/custom-christmas-ornaments) 上制作)或一个 [经典的粘土婴儿手印/脚印](https://www.amazon.com/Baby-Handprint-Footprint-Ornament-Keepsake/dp/B08D68F4WY?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/why-you-should-start-this-ornament-tradition-with-your-1848131382&asc_source=&gclid=Cj0KCQiAkZKNBhDiARIsAPsk0WgO46bPSIDGUf5sQMJtbWye-77NndxjyNtl7uRwhsoAsmY7L3AWapYaAv7REALw_wcB&maas=maas_adg_A0CAF374D21277D84FCD3895820D53EA_afap_abs&ref_=aa_maas&tag=kinjalifehackerlink-20) 制作婴儿的第一件圣诞装饰品更好呢？其他创意包括:任何带有他们名字和年份 的个性化物品，或者与重大事件或他们的兴趣相关的装饰品。它可以是一辆房车或一架飞机，象征着搬家；一架钢琴，象征着音乐课的开始；或者是达斯·维德，象征着一个痴迷于《星球大战》的反派影迷。唯一的先决条件是，它必须与积极的童年记忆联系在一起。

## 如果你的孩子长大了怎么办？

如果你的孩子已经过了婴儿/蹒跚学步的年龄，但还没有走出家门，而你想开始，不要害怕。刚才，我惊慌失措地跑到地下室去清点可能属于我每个孩子的装饰品，我发现的比我预想的或记得的要多。如果你的孩子上了日托所或幼儿园，很有可能，你已经收到了一些雪花，中间有他们的照片(谢谢你们，幼儿园老师)，或者是他们三岁时做的一只邋遢但可爱的眼睛瞪得大大的泡沫驯鹿。

如果他们年龄稍大一点，有什么理由让这个孩子今年不能选择一个足球和一个我们中间的装饰品来纪念呢？绝对不行。在任何给定的年份，你都可以随意增加一倍来充实你的收藏，或者一起回忆他们六岁时的生活，尽可能地填补这些空白。
