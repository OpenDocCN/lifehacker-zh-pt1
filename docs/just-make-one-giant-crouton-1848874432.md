# 就做一个大面包丁

> 原文：<https://lifehacker.com/just-make-one-giant-crouton-1848874432>

我不知道是谁发明了油炸面包丁，但他们做到了。将一束面包放在一束生蔬菜上，不仅让这堆植物味道更好，口感更有趣，还减少了面包浪费，因为最好的油炸面包丁是由不新鲜的面包制成的。



有些人把面包切成方块做成面包丁，有些人把它撕开(以增加质地的变化)。第一种然后是烹饪书作者 [阿里·斯拉格](https://alislagle.com/) 喜欢的方法，他制作一个大面包块，然后把它撕成一口大小的小块(为了*更大的*质地变化)。来自 [Food52](https://food52.com/blog/27302-why-ali-slagle-chicken-caesar-salad-is-genius) :

> 阿里在煎锅里开始做油炸面包丁，切成大块的油炸吐司，然后切成一口大小。这让我们摆脱了不可避免的小面包屑，它们会在剩下的面包屑之前开始燃烧，并给了我们一个新的油炸面包丁范例:脆脆的边缘，是的，但也有温暖蓬松的中间。

认识到这个智慧，我决定用一条在杂货店引起我注意的三奶酪粗面粉面包制作我自己的大面包块。阿里用橄榄油煎她的面包，这是一个不错的计划，但我决定用培根油，因为它味道很好，从冰箱里出来就像一场梦一样蔓延。

我从面包上切下一片一英寸长的面包，在每一面从边到边涂上大量的油脂，然后在铸铁锅中煎几分钟，直到每一面都变成深金黄色，有一些烧焦的斑点。我把它从锅里拿出来，让它冷却一分钟，然后把煎好的面包撕成粗片。

油炸面包丁味道很好，就像大多数用熏肉油煎的东西一样。我想他们会在家吃科布沙拉。他们确实是外脆内松，就像承诺的那样；不新鲜的美国 T2 会给他们更多的咀嚼。

你可以用符合你心意的纽约面包制作一个巨大的油炸面包块，尽管我建议你选择内部蓬松或耐嚼的 T2 面包，而不是 T4 蛋糕般的面包。煎炸油脂的选择也由你决定，但是培根油脂很难错。
