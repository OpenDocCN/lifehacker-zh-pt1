# 为什么您应该升级到 Windows 11 2022

> 原文：<https://lifehacker.com/why-you-should-upgrade-to-windows-11-2022-1849562445>

当去年 [Windows 11](https://lifehacker.com/how-to-install-windows-11-right-now-even-if-you-dont-h-1847799922) 推出时，那是一次彻底的更新。它用一个新的中央任务栏和更新的用户界面元素改变了几十年前的操作系统的面貌。但是 [它并没有完全完成](https://lifehacker.com/7-frustrating-new-features-in-windows-11-and-how-to-fi-1848626828) : 它的黑暗模式缺乏，任务栏令人沮丧，新的开始菜单留下了一大堆有待改进。Windows 11 的第一次重大更新(2022 年)是为了弥补这些缺点。



这是一个更新，有很多修复(包括功能和 UI 元素)，以及一些很棒的新特性。如果你已经在运行 Windows 11，这个很容易推荐。进入**设置> Windows 更新**开始该过程。在你等待的时候，尝试所有新的东西。

## 快照布局将所有其他内容都留在后面

Windows 在窗口管理方面一直很棒，Windows 11 的新快照布局功能是一个很好的更新。将鼠标悬停在最大化按钮上，可以看到一系列不同的窗口配对，您可以立即将屏幕分成几个部分。

现在快照布局功能比以往任何时候都更容易使用。将任何窗口拖到屏幕顶部，您将看到这些预置。将它拖到一个窗口配对上，放开它，它就会吸附到位。非常直观。此外，还有将窗口对齐到不同位置的键盘快捷键。

## 文件夹进入“开始”菜单

新的开始菜单开始朝着正确的方向发展，它能够创建文件夹并自由地固定它们。将应用程序拖到彼此之上以创建一个文件夹，并将它们放在您想要的任何位置。固定应用程序区域现在也更大了。

## 在任务栏中拖放(最后)

对 Windows 11 最大的批评之一是任务栏设计。虽然你仍然不能将它移动到屏幕上的任何地方，但是一些旧的任务栏功能已经出现了。拖放终于起作用了，所以你可以拖动文件或标签，并使用任务栏图标在不同的应用程序和窗口之间移动它们(就像你在 Windows 10 中一样)。时间和日期也显示在多个显示器上，所以欢呼吧。

## 新的触摸屏手势让二合一设备更加有用

2 合 1 Windows 笔记本电脑的用户会非常喜欢这次更新:你现在可以从屏幕底部向上滑动来查看任务栏和正在运行的应用程序。从右下角向上滑动现在可以调出快速设置。

## 有新的游戏更新

新的更新也为游戏玩家带来了一些生活质量的改善。顶部有一个新的 Xbox 控制器栏，可以很容易地使用控制器在游戏之间导航。微软还为窗口游戏增加了自动 HDR 和可变刷新率的支持。以前，这些功能只有在全屏播放时才可用。DirectX 10 和 11 中以窗口模式运行的旧游戏的延迟也得到了改善。

## 智能应用程序控制带来新的安全功能

你从微软商店下载的每个应用程序都经过了 Windows 11 的审查。但是你从一个随机网站下载的应用程序呢？它们可能包含病毒或恶意软件，尤其是当坏人提高他们的隐身能力时。新的智能应用程序控制功能使用新的人工智能模型来防止恶意软件安装在您的 PC 上。企业所有者和管理员可以选择打开或关闭它，或者只让它处于评估模式。

[微软 Defender SmartScreen 在这里也是隐私大玩家](https://blogs.windows.com/windowsexperience/2022/09/20/available-today-the-windows-11-2022-update/) 。当您在已知的恶意网站中输入凭据时，该功能会向您发出警告，这样您就不会意外地将您的个人详细信息透露给骗子。

## **任务管理器终于得到了 Windows 11 的待遇**

任务管理器是 Windows 11 中常见的眼中钉，因为它仍然使用旧的 Windows 10 风格。微软最终用 Windows 11 的流畅设计对其进行了更新，使其与文件资源管理器等其他原生应用更有家的感觉。更好的是，它还有一个新的黑暗模式。

## 全系统实时字幕

[Windows 11](https://lifehacker.com/windows-11-accessibility-features-everyone-can-use-1849163481)中的直播字幕已经相当不错了。虽然他们以前仅限于媒体，但现在他们在整个操作系统中发挥的作用。按下 **Windows + Control + L** ，任何正在播放的音频都将激活实时字幕。还有一个麦克风选项，可以将房间里的任何内容转换为屏幕上的实时文本。所有这些处理都发生在设备上，这意味着没有任何东西被发送到微软的服务器上。

## 以及更多功能

随着 2022 年的更新，微软继续推动添加新的第一方应用程序。这一次，他们增加了一个新的录音机应用程序，轻松编辑视频的 ClipChamp，以及一个新的家庭安全应用程序。

文件资源管理器看到一些小的，值得注意的变化。主页部分被重新命名为快速访问——但是右边面板上你可以锁定文件和文件夹的组，也称为快速访问，仍然存在。文件夹预览，文件夹图标中的小缩略图，卷土重来。你现在可以使用**控制+右键**T3】点击快捷方式打开更长的原始上下文菜单。

这一新的更新也使你的电脑更加环保。Windows 现在将尝试通过在能源网格中使用低碳能源时安排更新来减少您的碳足迹。

最后，将在 10 月推出一些新功能，包括在文件浏览器中添加标签，对照片应用程序进行全面修改，复制时的建议操作，任务栏溢出菜单，以及向更多设备共享的能力。
