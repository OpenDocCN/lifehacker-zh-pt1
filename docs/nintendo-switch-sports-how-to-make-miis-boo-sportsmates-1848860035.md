# 如何为任天堂Switch体育制作一个Mii

RIP *Wii运动*。保龄球只有十个球瓶。拳击没了。最令人期待的模式——高尔夫——甚至还没有出现。但是也许由*任天堂Switch体育*犯下的最令人震惊的罪行是它如何完全排挤Miis，这位可爱的古怪玩家创造的化身定义了任天堂账户多年。然而，在任天堂的新体育游戏中扮演Mii在技术上是可能的，我们可以向你展示如何做到这一点。

*任天堂Switch体育*，今天在任天堂Switch举行，事实上是 *Wii体育*的复兴，可以说是电子游戏史上最大的文化现象之一。像它的祖先一样，*任天堂Switch体育*是一款体育sim卡，你可以在其中使用动作控制——在这种情况下是娱乐游戏——来参与各种活动。现在，有六种不同的迷你游戏可供选择:保龄球、羽毛球、网球、排球、足球和chambala(或击剑)。

任天堂Switch体育的主旨是与其他玩家竞争，以赚取积分，解锁定制物品，并打扮你的游戏角色，称为“运动伙伴”与miis相比，运动伙伴更驯服、更时尚、更团结。(有些人甚至会用“无聊”和“鼾声如雷”这样的词。)如果你更喜欢Miis的超现实主义，你需要做两件事。

## 如何在*任天堂Switch体育报*创建一个信息产业部

1.  首先，从你的交换机的主屏幕设置，向下滚动到“Mii”菜单，然后点击“创建一个新的Mii。”您将看到从头制作一个或者使用程序生成的一个作为画布的选项。一旦完成，它们应该出现在“Mii列表”菜单中。(您最多可以节省100英里。)
2.  接下来，开机*任天堂Switch体育*。当你创建一个角色的时候，这个选项有点被埋没了:在“身体”选项下面，你会看到一个“Miis”按钮你可以从你的开关中选择任何MII，并将它们导入到*开关运动*中，尽管注意它只带来头部。(你说得对，那么这个选项在“身体”下面*就*奇怪了。)

对于一些长期的任天堂粉丝来说，选择离开Miis而转向运动伙伴是有争议的。
