# 为什么你不应该把蛋壳放回纸箱里

> 原文：<https://lifehacker.com/why-you-should-never-put-eggshells-back-in-the-carton-1848215758>

每当你用鸡蛋烹饪或烘焙时，良好的食品安全实践是最重要的。不过，有时候，懒惰会占上风:在某个时候，我们大多数人都会把用过的蛋壳放回纸箱里，而不是给 T2 找一个更合适的容器。即使你从来没有用这个动作送过任何人去医院，这仍然是一个你应该立即改掉的坏习惯。



现在你几乎肯定知道，鸡蛋可能被沙门氏菌污染，这会导致严重的食物中毒。我们大多认为生的和未煮熟的 T2 鸡蛋是主要的风险因素，但是重要的是要记住鸡蛋的外壳也会被污染。幸运的是，这很少是一件大事:工业消毒程序会处理绝大多数的表面污染，所以只要所有的鸡蛋都完好无损，纸箱环境通常是你最不用担心的。

当你把空壳放回纸箱时，一切都变了。你突然 [将另一个潜在的细菌来源](https://eggsafety.org/faq/is-it-safe-to-keep-used-egg-shells-in-the-carton-with-intact-eggs/)——生蛋残渣——引入你储存鸡蛋的地方。如果你认为任何剩余的蛋是安全的，仅仅因为它们的壳是完整的，那就再想想吧。蛋壳不是诺克斯堡； [*沙门氏菌*可以穿透它们](https://academic.oup.com/femsre/article/33/4/718/583518) ，这使得交叉污染的可能性很大。即使你只是把壳放在纸箱里一分钟，然后把剩下的敲碎，它们也一定会留下一些残渣，这些残渣可能足以占据整个纸箱。冷藏并不能杀死细菌——它只是减缓了细菌的速度。

底线:把空壳和完整的生鸡蛋放在同一个容器里会让沙门氏菌有更多的机会繁殖，并有可能使人生病，所以不要这样做。相反，把你的蛋壳收集在碗里，然后马上处理掉。这确实需要你在开始吃之前记得把碗拿出来，但这肯定比食物中毒要好。
