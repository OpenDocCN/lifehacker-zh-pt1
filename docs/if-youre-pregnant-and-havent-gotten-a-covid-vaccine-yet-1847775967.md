# 如果你怀孕了，还没有接种COVID疫苗，你真的应该接种

> 原文:[https://life hacker . com/if-you-treaty-and haven-a-covid-vaccine-yet-1847775967](https://lifehacker.com/if-youre-pregnant-and-havent-gotten-a-covid-vaccine-yet-1847775967)

在美国上市的所有三种COVID疫苗(Pfizer、Moderna和J & J)被认为在怀孕期间使用是安全的，并且数据 [强烈支持如果您再次怀孕、最近怀孕或打算很快怀孕，就接种疫苗](https://lifehacker.com/new-data-shows-not-getting-vaccinated-in-pregnancy-is-t-1847475082) 。但是只有31%的孕妇接种了疫苗，所以美国疾病控制中心最近发布了一份“紧急健康咨询” 以确保每个人都知道风险和好处。

当疫苗首次被批准时，是基于排除怀孕人群的试验。专家没有任何理由说明人们不应该在怀孕期间使用疫苗，因此2020年12月的官方建议是，如果你怀孕了，由你和你的医生决定你是否应该接种疫苗。与此同时， [关于疫苗影响生育](https://lifehacker.com/heres-your-covid-vaccine-rumor-roundup-1845935618) 的毫无根据的神话在社交媒体上爆发。

从那以后，有了更多的安全数据，而且非常清楚。接种疫苗后怀孕的人(反之亦然)没有增加流产、生育问题或妊娠并发症的风险。(我们在这里说明 [几个关键研究的结果](https://lifehacker.com/new-data-shows-not-getting-vaccinated-in-pregnancy-is-t-1847475082) 。)

COVID对怀孕的人和他们的婴儿有额外的风险，这使得疫苗特别重要。如果你一直在想“嗯，我年轻健康”，怀孕改变了COVID的等式。根据疾病预防控制中心的数据，迄今为止，已有125，000人在怀孕期间检测出病毒呈阳性，其中22，000人最终住院治疗。也就是说，每六名患有COVID的孕妇中就有一人住院。到目前为止，已经有161人死亡，但是死亡率正在上升；其中22例死亡发生在2021年8月。

更不用说:如果你患有严重的COVID，你的宝宝的预后并不好。

> 有症状的孕妇感染新冠肺炎病毒后，进入重症监护室的风险增加一倍，死亡风险增加70%。患有新冠肺炎病毒的孕妇发生不良妊娠结局的风险增加，可能包括早产、死产和新生儿同时感染新冠肺炎病毒而住进重症监护室。

如果您对接种疫苗的安全性或风险和益处有疑问，请与您正在接受产前护理的提供者交谈，或致电 [疾病预防控制中心的COVID信息热线](https://www.cdc.gov/cdc-info/ask-cdc.html) 。
