## 器皿摆放的艺术

我们大多数人都知道叉子放在盘子的左边，勺子和刀子放在右边。当这些器具的*数量*增加时，我们的困惑也随之增加。但是，正如礼仪顾问 [帕梅拉·希林斯](https://www.wcspeakers.com/speaker/pamela-hillings/) 告诉 [玛莎·斯图尔特](https://www.marthastewart.com/276333/how-to-set-a-formal-dinner-table?slide=661524a0-c9be-413e-840a-5e1ac7121c85#661524a0-c9be-413e-840a-5e1ac7121c85)“餐桌的摆放是基于逻辑的。”吃饭时，把没用过的餐具放在离我们的盘子3英寸远的地方是没有意义的。这就是为什么餐具是按照从外到内的顺序摆放的；离盘子最远的餐具先用，离盘子最近的最后用。

在盘子的左边，从左到右依次是:鱼叉、沙拉叉、餐叉。(注意:在欧洲，沙拉通常在主菜后吃，这就是为什么你有时会看到沙拉叉放在离盘子最近的地方。如果你觉得特别新奇，你可以自己动手。)在盘子的右侧，从左到右依次放置餐刀、鱼刀和汤勺。(后面是牡蛎叉，如果需要的话。)

器皿的底部应该与盘子的底部边缘对齐。(玛莎·斯图尔特说，这个盘子应该是一个“充电器”，一旦客人把餐巾放在膝盖上，就可以拿掉——但一个基本的餐盘也可以起到同样的作用。)

所有与甜点相关的东西都放在盘子上方，中间。甜品叉应该放在离盘子最近的地方，叉齿朝右，而甜品勺位于餐叉上方，转向相反的方向(手柄朝右)。

关于餐具的最后几句话:根据食物的不同，可以有许多不同的摆放方式。例如，如果在沙拉和主菜之间上汤，勺子应该放在两把刀之间。避免在桌子上堆满你不需要的餐具。(I 如果没有汤或鱼，就把那些餐具从桌上拿走。)你的正式餐桌可能只需要两把餐叉、一把小刀、和一把勺子、就可以了。

## **面包和黄油盘子放在哪里？**

虽然这是正常用餐时的例行公事，但正式的用餐场合可不是用餐刀给面包卷涂黄油，然后把它放在烤羽衣甘蓝旁边的地方。不，高级餐桌有单独的面包盘子，放在盘子的左边，叉子的上方。将黄油刀斜放在面包盘上，以获得最大限度的餐桌时尚。

## 高脚杯放在哪里

高脚杯位于盘子的右上方(在刀子和勺子的上方)。当容纳三个玻璃杯时，将水杯放在餐刀上方，白酒杯直接放在其右侧，红酒杯形成一个整洁的玻璃三角形的顶部中心。另一种选择是把所有的杯子从大到小排成一条直线:水杯、红酒、白酒、香槟杯(水杯正好放在盘子中线的右边)。

## 餐巾放在哪里？

小时候摆桌子的时候，我妈妈教我把折叠好的餐巾放在叉子下面。然而，这是一个时髦的餐桌失礼。当客人试图将不起眼的餐巾从枷锁中挣脱出来时，餐巾纸放在叉子下面会产生噪音，可能会导致银器叮当落地。为了达到极致的精致，把餐巾放在叉子的左边，或者最好放在盘子的中间——折叠好或者卷上装饰性的餐巾环。

## 最后，创建一个座位图

如果预计有四位以上的客人，建议分配座位。考虑提前制作座位卡，放在每位客人的餐盘上方。然而，玛莎·斯图尔特也说，不要把你的座位卡放在你自己的位置上。我们无法找出为什么这是不礼貌的行为。在这件事上相信玛莎，好吗？
