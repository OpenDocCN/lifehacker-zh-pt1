# 灵活性和移动性之间有什么区别(为什么要关注)？

> 原文：<https://lifehacker.com/whats-the-difference-between-flexibility-and-mobility-1847275695>

柔韧性曾经是一个很容易理解的概念:如果你能做劈叉或者把手放在肩胛骨后面，你就是柔韧的。如果你不灵活，你必须做一些伸展运动。但是最近几年，术语*移动*变得更加流行，它的意思并不完全相同。

Watch

移动工作不仅仅包括拉伸。你可以使用泡沫滚轴，或者做一些增强力量的运动，因为这些运动可以让你的关节有更大的活动范围。如果你在锻炼你的灵活性，你可能会做更多的动态拉伸而不是静态拉伸。这是因为灵活性是移动性的一部分，但不是一回事。

专家的定义略有不同，但共识似乎是这样的:

柔韧性是指你可以在不受伤的情况下将关节移动多远。被动拉伸，比如双腿坐在地板上试图触摸你的脚趾，将测试你柔韧性的极限，特别是你的腿筋可以被动伸长多少。大致相当于你的被动活动范围。

相比之下，机动性就是你能移动一个关节多远*。想象一个舞蹈演员站起来，在她面前抬起她的直腿离开地面。如果她能把脚抬得离地很高，她就有很好的机动性。但是不仅仅是腿筋的灵活性:S 他也有力量，平衡，和技巧让她的腿采取那个位置。*

## 为什么移动性很重要

生活中的大多数事情都需要机动性，而不仅仅是灵活性。无论你是举着杠铃蹲着，还是蹲着抱起蹒跚学步的孩子，良好的灵活性意味着你可以弯曲你的脚踝、膝盖和臀部，足以让你的脚在地面上保持稳定，同时安全地保持重量(杠铃或蹒跚学步的孩子)。

小腿肌肉紧张的人通常很难 [深蹲到合适的深度](https://lifehacker.com/how-deep-should-you-squat-1841473363) ，更好地深蹲的第一步是增加小腿的灵活性，以改善脚踝的活动范围。但是不仅仅是灵活性。其他一些会影响你行动能力的因素:

*   **你的关节结构**。比如 [有些人的臀窝更偏向骨盆前方，有些更偏向侧面](https://themovementfix.com/the-best-kept-secret-why-people-have-to-squat-differently/) 。肌肉的确切位置及其附着点并不总是因人而异。你的身体结构决定了可能的运动范围，甚至在考虑柔韧性之前。
*   **稳定性**。有时肌肉需要收缩到 [来稳定](https://www.acefitness.org/fitness-certifications/ace-answers/exam-preparation-blog/1189/joint-mobility-and-stability/) 一个关节，因此，它不能放松太多，同时仍然保持你的身体安全并在控制之下。尽管我们有时认为力量和机动性是对立的，但变得更强有时有助于机动性。
*   **更有实力**。有时我们可能不够强壮，无法将身体移动到某个位置。在我们之前的例子中，舞蹈演员仅靠柔韧性无法达到那个位置；她还必须加强臀部屈肌，以便能够举起她伸直的腿的重量。
*   **无论你是否热身**。温暖的肌肉可以比寒冷的肌肉伸展得更远，所以在锻炼开始时感到僵硬并在最后具有更好的灵活性并不罕见。

## 如何获得更好的移动性

提高机动性的第一步是确定你需要做什么动作，以及为什么。然后，寻找一个常规或向教练寻求提高特定类型的灵活性的步骤。

例如，我们这里有一些建议 [用于改善下蹲](https://lifehacker.com/how-to-improve-your-ankle-mobility-for-squats-1842315361) 时脚踝的灵活性。第一个一点也不夸张；使用高跟鞋或盘子是一个提示。尽管从技术上来说，这不会让你的脚踝在其运动范围内进一步移动，但它让*你*在控制下移动，这有效地增加了你臀部的灵活性，让你可以蹲得更深。我们的其他建议包括拉伸和使用泡沫滚筒。

泡沫滚轴在流动性方面很受欢迎，因为它们在微观层面拉伸你的肌肉，而不需要你进行特定的静态拉伸运动。在揉搓需要更柔韧的肌肉后，你可能会发现你可以更容易地移动它们。出于这个原因，这通常是热身的一部分。

动态拉伸也有助于灵活性，作为热身的一部分，它们也很有效。你不是坐下来强迫肌肉伸展，而是用你的身体在一系列运动中主动移动。 [受控关节旋转，或 CARs](https://www.physioinq.com.au/blog/controlled-articular-rotations) ，是一种动态拉伸，让你在运动范围内慢慢移动像肩膀这样的关节。当你在越来越大的范围内移动手臂时，你正在热身相关的肌肉，同时锻炼力量和柔韧性。

静态拉伸在日常运动中也有一席之地。尽管灵活性只是移动性的一部分，但却是非常重要的一部分！为了你的运动目标，你可能需要也可能不需要静态拉伸，但是融入静态拉伸可以有效地增强你肌肉的伸展能力。
