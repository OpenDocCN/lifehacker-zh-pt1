# 用湿纸巾防止微波干扰

> 原文：<https://lifehacker.com/prevent-microwave-messes-with-a-wet-paper-towel-1845808034>

如果你的微波剩菜在整个重新加热过程中爆裂飞溅，造成一片混乱，就像上图所示，这意味着你跳过了一个关键步骤。你需要的，当然是一张湿纸巾。

Watch

在盛剩菜的碗或容器上盖一张湿纸巾有几个好处。最重要的是，它包含食物并阻止食物溅满微波炉的内壁和天花板。弄湿纸巾是很重要的，因为水把它压下来，帮助它粘在你的容器上，形成一点密封，防止它被风吹走，就像干纸巾或餐巾一样。

作为一个额外的好处，它还增加了一点水分，这可以帮助蒸你的剩菜，防止它们变干。

你不需要在毛巾或餐巾纸上洒很多水来达到效果。只需在水龙头下运行几秒钟，挤出多余的水分，然后覆盖在食物上。

这对于有一定深度的容器或碗来说效果最好，所以纸巾实际上并没有接触到食物；在某些情况下，将它直接挂在盘子里的食物上会造成食物粘在纸巾上的混乱局面。尽管这可能比把垃圾溅得满微波炉都是要好。
