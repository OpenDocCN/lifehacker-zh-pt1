# PS5 获得漂亮的新图形功能

> 原文：<https://kotaku.com/ps5-playstation-5-vrr-variable-refresh-rate-graphics-vi-1848840146>

你们当中的任何 PC 游戏玩家，甚至更认真的 Xbox 用户(他们已经拥有这项技术多年了)，都将熟悉一种称为可变刷新率(VRR)的技术，这种功能是 [允许](https://en.wikipedia.org/wiki/Variable_refresh_rate) “一种可以连续无缝地动态变化的动态显示刷新率。”该功能将于本周晚些时候在 PS5 游戏机上推出。

Watch

如果你珍惜游戏中的视觉保真度(和一致性)，这比听起来要重要得多。通过将控制台和显示器同步到相同的刷新率，两者之间不太可能出现延迟或质量下降，这意味着像屏幕撕裂这样的事情可以减少甚至消除

虽然索尼的公告只包含了 几个字，但如果你想要一段视频来解释它是如何工作的，还需要一些例子，这里有 Nvidia 为他们自己的 G-Sync 技术准备的东西:

 [https://kotaku.com/embed/inset/iframe?id=youtube-video-oT8f_1oRsLU&start=0](https://kotaku.com/embed/inset/iframe?id=youtube-video-oT8f_1oRsLU&start=0) 

现在谈谈技术细节。首先，这项更新是本周晚些时候更广泛的 PS5 更新的一部分，但只能在兼容 HDMI 2.1 VRR 的电视和显示器上工作，所以你需要先检查你的显示器是否支持它。其次，这不会只是扳动一个神奇的开关；游戏需要单独打补丁才能使用 VRR，所以这种支持的时机(和最终可用性)取决于各个工作室和发行商。

索尼今天早些时候公布了一份列表，显示了将在“未来几周”获得补丁的游戏:

你玩的游戏不在列表上？或者是，你想知道支援什么时候会到？索尼表示，“这只是获得 VRR 支持的 PS5 游戏中的一小部分，我们要感谢他们才华横溢的开发团队。请继续关注他们的频道更新，因为一旦他们的游戏补丁上线，你将拥有与 VRR 最好的体验。”
