# 用浴缸溢流盖享受更奢华的沐浴

> 原文：<https://lifehacker.com/the-right-way-to-use-a-bathtub-overflow-cover-for-a-mor-1846665546>

坐在快速冷却的浅浴缸里是放松的反义词。你不必在齐颈深的热水池里尽情享受——天哪，那是*梦*——你被迫选择你身体的哪一部分想要浸入水中，有策略地交替它们以防止暴露的部分变得太冷。

解决这个问题的唯一长久之计是用一个专门为浸泡而设计的浴缸来替换你的小而悲伤的浴缸，但是由于显而易见的原因，这对大多数人来说是不可能的。在你能够实现你所有的浴缸梦 之前，你可以用一个便宜的溢流排水盖给你已经有的浴缸增加几英寸的浸泡空间——只要你正确使用它。(如果你不这样做，它什么也不会做。)充分利用下水道井盖有两个秘诀，谢天谢地，这两个秘诀非常简单。

## **仅用于洗澡**

当吸盘和被吸盘吸附的表面都干净干燥时，效果最好。为此，请在每次使用后取下溢流盖，以便在下次使用前有时间完全干燥。洗澡或手洗衣物时不要开着；它会被肥皂泡沫覆盖，然后停止工作。

在这一点上，你也想在粘上它之前清除浴缸和盖子上的任何残留，特别是如果你喜欢泡泡浴(或浴弹)。肥皂和水可以解决这个问题，但是如果你不耐烦，用酒精快速擦拭两面即可——酒精可以溶解几乎所有东西，并迅速蒸发，留下一个干净、完全干燥的表面。

## **正确定位**

为了获得最佳效果，你应该始终将盖子的顶部(有大孔的部分)与你的溢流排水管的顶部对齐，而不是底部。这在排水管的商业端产生了一个漂亮的气泡，这是保持上升的洗澡水的关键。用力按压每个小吸盘，确保其牢固。想要一个视觉指南，可以看看这个来自(名字令人难以置信的)bathsofdistinction YouTube 频道的视频:
