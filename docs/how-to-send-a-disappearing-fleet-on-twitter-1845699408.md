# 如何在Twitter上发送一个正在消失的“舰队”

> 原文:[https://life hacker . com/how-to-send-a-distanting-fleet-on-Twitter-1845699408](https://lifehacker.com/how-to-send-a-disappearing-fleet-on-twitter-1845699408)

经过几个月的测试后，Twitter的“舰队”功能现已向所有iOS和Android用户推出。尽管Twitter的营销舰队是“消失”的推文，但它们实际上是大多数社交媒体平台上流行的“故事”功能的另一个版本(包括 [LinkedIn，出于某种原因](https://lifehacker.com/will-linkedins-big-redesign-help-you-land-a-job-1845207929) ) ，其特色内容只在周围停留24小时。

车队可以包含推文(你的或其他人的)、图片、gif或视频剪辑，以及附加的叠加文本或表情符号。就像其他社交媒体应用上的故事帖子一样，拥有新车队的账户会在普通Twitter feed上方的滑动菜单中突出显示。轻按个人资料图片进行观看，并轻按屏幕边缘跳到您订阅的下一个舰队。

张贴舰队——我想这可能是“转瞬即逝”，一个永远不会流行的短语——也将为任何以前使用过故事的人所熟悉:

1.  在Twitter应用程序中，点击您的个人资料图标
2.  轻按**“分享一个转瞬即逝的想法..”**
3.  用表情符号、文本等自定义舰队，然后轻按**“舰队”**即可发布。舰队将出现在你的个人资料中，并保持24小时可用。

您还可以将推文作为车队共享:

1.  在你想要的推文上点击**“分享”**，呃，舰队。
2.  选择**“共享车队”**
3.  添加您想要的任何反应或额外文本，然后轻按**“舰队”**

定制选项目前有限，但Twitter表示，未来将为车队增加贴纸和直播选项。不过，用户可以回复车队——就像Instagram、脸书和其他任何地方一样；车队回复以DMs的形式私下发送。

在该公司的 [公告博客文章](https://blog.twitter.com/en_us/topics/product/2020/introducing-fleets-new-way-to-join-the-conversation.html#:~:text=To%20share%20a%20Tweet%20in,will%20be%20available%20in%20Fleets.&text=Your%20followers%20can%20see%20your,top%20of%20their%20home%20timeline.) 中，Twitter称许多用户报告说他们对发推感到不舒服，“因为这感觉如此公开，如此永久，就像有太多的压力去转发和点赞。”舰队的设计是为了帮助用户“更舒适地分享个人和随意的想法、观点和感受。”

如果这听起来像你在网站上的经历，也许可以尝试一下舰队，而不是为每条推文或 [羞愧地删除你的旧帖子](https://lifehacker.com/how-to-delete-your-old-tweets-1844010380) 而苦恼。请注意，你的追随者仍然可以截屏你的舰队，目前没有办法判断是否有人保存了或分享了你的舰队。换句话说，即使24小时后这些不好的经历从你的个人资料中消失了，它们也可能仍然困扰着你。
