# 如何用谷歌搜索召唤 AR 幽灵

> 原文：<https://lifehacker.com/how-to-summon-ar-ghosts-with-a-google-search-1845530277>

万圣节是我最喜欢的节日，但就像 2020 年的其他事情一样，今年对我们大多数人来说不会那么喜庆。如果你渴望一些善意的恐怖氛围，你可以试着通过召唤一个鬼魂或复活的骷髅为你跳舞来驱除这些废话——至少是数字化的。

谷歌在万圣节前夕向其 3D 模型库中添加了一些适合季节的角色。这些模型可以在大多数 iOS 和 Android 设备上的增强现实中看到，但你~~需要念出正确的咒语~~在谷歌搜索应用程序中搜索正确的关键词，或者通过你的手机浏览器上的谷歌网站召唤它们。

[根据 TechCrunch](https://techcrunch.com/2020/10/29/google-brings-halloween-to-life-using-augmented-reality) ，“万圣节”是最可靠的搜索词，应该调用和舞动的幽灵，但其他可能的命中包括:

*   骨骼
*   南瓜
*   杰克灯
*   《人鬼情未了》
*   服装
*   狗
*   猫
*   海盗
*   热狗

基本上，如果你搜索一些与万圣节有关的东西，你应该能找到模特。下面是如何在增强现实中实现它们的快速演练:

1.  执行搜索，然后滚动搜索结果。
2.  谷歌会问你是否想**“召唤一个 3D 幽灵。”**
3.  点击**“3D 视图**”确认并等待模型加载。您也可以从底部菜单中选择其他型号。
4.  根据您使用的设备，您可能需要使用设备在房间内四处寻找幽灵，或者点击**查看您的空间**

你的网络睡眠伴随着背景音乐，这增加了一些怪异的气氛。你也可以给你的亡灵和/或装扮的动物朋友拍照或录像，与你现实生活中的人类朋友分享。
