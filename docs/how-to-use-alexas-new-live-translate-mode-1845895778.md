# 如何使用 Alexa 新的实时翻译模式

> 原文：<https://lifehacker.com/how-to-use-alexas-new-live-translate-mode-1845895778>

由于 Alexa 新的实时翻译模式，亚马逊 Echo 设备现在可以作为你的私人翻译工作。与其他数字助理 提供的 [翻译功能一样，Live Translate 是一种人工智能功能，可以识别多种语言，并为两种说话者翻译。Live Translate 背后的技术听起来令人印象深刻，你可以在亚马逊的公告帖子](https://lifehacker.com/use-google-assistants-interpreter-mode-to-talk-to-someo-1840397512) 中了解更多信息。



Live Translate 易于使用，我们将向您展示如何使用，但需要记住一些限制。

首先，该功能仅在美国的 Echo 设备上可用。具体来说，亚马逊表示，用户必须将他们的 Echo 设备的位置和系统语言设置为“英语美国 S. ”才能使用实时翻译。如果需要，你可以在 Alexa 移动应用程序 中 [轻松更改两个](https://www.amazon.com/gp/help/customer/display.html?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-use-alexas-new-live-translate-mode-1845895778&asc_source=&nodeId=GVBCTNK6EC45CKBE&tag=kinjalifehackerlink-20) [，但不是在所有国家，这样做可以更改 Alexa 的口音、以及时间、天气和你收到的当地提醒。](https://www.amazon.com/gp/help/customer/display.html?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-use-alexas-new-live-translate-mode-1845895778&asc_source=&nodeId=G96ABGB2ASX5M2NK&tag=kinjalifehackerlink-20) 

英语系统语言要求也意味着 Alexa 一次只能在英语和其他支持的语言之间进行翻译——至少目前是这样。发布时，可识别语言的完整列表为:

*   英语
*   法语
*   德国人
*   印地语
*   意大利的
*   巴西人葡萄牙人
*   西班牙语

实时翻译可以在所有 Echo 设备上运行，但 Echo Show 用户获得了一个额外的功能:Echo Show 的屏幕上显示了每种语言的两位发言者的实时翻译，这样你就可以回读最近说过的句子。也就是说，Echo 扬声器是唯一支持 Live Translate 的*Alexa 设备，这限制了你实际上可以使用 Live Translate 的地方。如今，我们很少有人去旅行，我敢打赌，在美国旅行时，更少人会随身携带回声扬声器作为翻译。*

*希望 Live Translate 将来会出现在其他 Alexa 驱动的设备上(以及其他国家),但现在，它可以让国际客人更容易地访问。* 

### *如何在 Echo 设备上使用 Alexa 的实时翻译*

1.  *当你准备好使用实时翻译时，说:**“Alexa，请将英语翻译成[第二语言]。”这个短语的其他变体应该可以使用；确保你告诉 Alexa 要翻译哪些语言。***
2.  *像平常一样继续你们的谈话。Alexa 会监听并根据需要进行翻译。请注意，您需要靠近回音扬声器，以便它能够清楚地听到并解释两个扬声器。*
3.  *对话结束后，说出**“Alexa，停止”**关闭直播翻译。*
