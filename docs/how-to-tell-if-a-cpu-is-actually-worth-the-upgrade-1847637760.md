# 如何判断 CPU 是否真的值得升级

> 原文：<https://lifehacker.com/how-to-tell-if-a-cpu-is-actually-worth-the-upgrade-1847637760>

有很多关于一个设备的 CPU(也就是中央处理器)有多少个内核的宣传。经常听到人们说更多的内核意味着更强的处理能力，虽然这在技术上是正确的，但这并不是全部。实际上有几个因素决定了处理器的能力和速度...即使理论上 CPU 在某些方面“更差”，但它可能比其他的更适合某些任务。



这可能会令人困惑，尤其是如果你正在寻找一台新的电脑，并希望找到合适的 CPU。为了提供帮助，我们整理了这份关于 CPU、其规格以及如何测试的快速指南。这不是一个全面的分类，但它应该有助于您理解 CPU。

## 如何理解 CPU 的 pecs

首先，这里有一个最重要的 CPU 规格的快速纲要:

*   **核心:**“核心”是处理器的别称。因此，一个四核 CPU 实际上有四个独立处理器，一个八核有八个，以此类推。CPU 拥有的内核越多，它能够可靠管理的任务就越多。然而，默认情况下，更多的内核并不一定*使 CPU 更好或更快；这仅仅意味着它可以同时处理需要多个内核的任务。*
*   ***时钟速度:**CPU 的时钟速度(以 Hz 和 GHz 为单位)表示它每秒可以处理多少数据。一个 GHz 相当于一个亿条指令。比如一个 2.9 GHz 的 CPU 每秒可以处理 29 亿条指令，一个 4 GHz 的 CPU 可以处理 40 亿条指令。更快的时钟速度几乎总是意味着更好的性能。*
*   ***超频:**一些 CPU 支持一种称为“超频”的功能，这种功能让用户可以将 CPU 的时钟速度提高到超过其附带的基线。超频可以提高处理器的性能，特别是对于游戏或媒体渲染等资源密集型任务，但代价是更高的热量输出。由于过多的热量会损坏 CPU 或其他没有适当冷却的 PC 组件，CPU 制造商通常会包括桌面应用程序，帮助您安全地超频处理器并监控内部温度。*
*   ***缓存:**一个 CPU 的缓存就是它有多少内存。虽然较大的高速缓存更适合多任务处理，但在某些情况下，高速缓存较小的 CPU 运行速度更快，所以高速缓存较大并不意味着 CPU 更好。*
*   ***超线程和多线程:**线程是一种在一些 CPU 上发现的技术，它允许核心同时执行多个进程，有效地将 CPU 的核心数量增加一倍。英特尔称之为“超线程”，而 AMD 称之为“多线程”，但它们是一回事。像多核一样，更多线程意味着更好的多任务处理，但它不会影响单核任务的速度。*
*   ***热功率设计:**热功率设计(TP D )衡量一个 CPU 的功率需求。虽然这个数字对于 CPU 的性能没有太大的意义，但知道你是否正在构建自己的 PC 是很重要的，因为你将需要一个 PSU(电源单元)来为你的 PC 中的每个组件供电——CPU、显卡等。*
*   ***硬件代:**一个 CPU 的硬件代是指设备的微技术。制造商反复发布 CPU，对新一代的增强功能进行调整。虽然新一代产品通常比以前的版本运行得更好，但情况并非总是如此。例如，拥有更多内核的旧 CPU 可能会与拥有更少内核或更慢基本时钟速度的新一代 CPU 竞争。*
*   ***类型:**最后，还有不同类型的 CPU:具体来说，移动 vs 桌面。移动 CPU 通常具有集成图形处理功能，因此您不需要单独的 GPU，但由于它们运行较小，并且通常针对较低的 TPD，因此它们的性能可能不如桌面版本。*

*许多规格可以通过 [阅读 CPU 的产品名称](https://www.newegg.com/insider/understanding-processor-names-cpu-naming-schema-guide/) 简单收集，但现在你可能已经注意到一种模式:CPU 的规格可能在纸上看起来更好，但有一些警告和矛盾，使得很难一眼就比较 CPU。*

*CPU 是电脑的核心部件，但我们使用电脑的原因各不相同。一个视频编辑可能想要一个具有多线程的高端处理器，而一个 PC 游戏玩家可能会选择具有更好时钟速度的 CPU，即使是 [这个“更便宜”的选项。](https://gizmodo.com/this-is-the-best-budget-cpu-1844673665) 那么，人们怎么知道一个 CPU 是否值得升级呢？*

## *如何理解 CPU b 基准测试*

*衡量 CPU 性能的最佳方法不是浏览规格表，而是基准测试。制造商、评测人员和发烧友级用户使用基准测试来实时查看 CPU(和其他 PC 组件)的性能，并将其与其他 CPU 进行比较。*

*当然，如果你想和其他 CPU 比较，你需要知道你自己的 CPU 的名字和型号。幸运的是，有一个简单的方法可以找到这个:*

1.  *在 Windows PC 上，打开开始菜单，进入**设置>系统>关于。***
2.  *查找列在**“设备规格”下的处理器***

*如果你想要一个更详细的读数，你可以在任务管理器中找到你的 CPU 的型号、内核、时钟速度和其他规格:* 

1.  *按下 **Ctrl + Shift + Esc** 打开任务管理器窗口。*
2.  *单击“性能”选项卡。*
3.  *在这里你可以找到 CPU 的名称和型号。左下方还列出了时钟速度、内核和逻辑处理器(也称为“线程”)。*

*您也可以在设备管理器应用程序中找到 CPU 信息:* 

1.  *在任务栏或 Windows 开始菜单中搜索**“设备管理器”**，然后从结果中点击应用将其打开。*
2.  *在设备管理器窗口中，向下滚动并单击**“处理器”**旁边的箭头，查看您的 CPU 的名称、型号和基本时钟速度。CPU 的每个核心都将被列为自己的设备。请注意，多线程/超线程 CPU 会列出更多内核(例如，四核可能总共显示八个处理器)。*
3.  *双击处理器以查看更多信息，如驱动程序详细信息。*

*一旦你有了你的 CPU 的型号和规格，它将更容易与预期的升级进行比较，看看它是否适合你。有一些 [基准测试](https://www.geekbench.com/) [程序](https://www.passmark.com/) 你可以用它们来运行你自己的测试，但是查找产品评论和基准测试比较图表更容易。几个好的资源是、像[Tom ' s Hardware](https://www.tomshardware.com/reviews/cpu-hierarchy,4312.html)和[CNET](https://www.cnet.com/topics/processors/products/)这样的产品评论网站以及像 [Newegg](https://www.newegg.com/CPUs-Processors/Category/ID-34) 这样的在线商店。*
