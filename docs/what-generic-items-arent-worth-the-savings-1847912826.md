# 哪些“普通”产品不值得节省？

> 原文：<https://lifehacker.com/what-generic-items-arent-worth-the-savings-1847912826>

总的来说，我认为一般的项目得到一个坏名声。你买的几乎所有东西都可以在商店品牌的替代品中买到，而且几乎总是，它基本上与价格更高、获得所有荣耀的名牌没有什么区别。当然，包装可能看起来有点奇怪，但里面的东西才是最重要的——通常，选择低端产品来节省一些钱是值得的。

除了奶酪饼干。Cheez-Its 是唯一可以接受的选择。所有的商店品牌都是垃圾——我知道，因为我都试过了。

凯洛格肯定也知道这一点，因为就你所得到的而言，Cheez-Its 的价格高得离谱。即使你货比三家，小小的 7 盎司。在纽约的大多数杂货店，一盒(老实说，基本上是一份)大约要花你 4 美元。21 盎司的。一盒 Aldi“Savoritz”的价格差不多，但是 1/3 的价格，你*肯定*得到 1/3 的满足感。(我其实是好心；萨沃里兹简直糟透了，当我们真的买下它们时——我是土生土长的中西部人，从小家境贫寒，不可避免地会买到便宜货——我通常会把它们浸在辣酱里，让它们更有味道。这可能很奇怪。)

所以我很想知道:有哪些非通用的物品对你来说永远值得溢价？也许是因为舒适的卫生纸(实际上，我会把我关于 50 美分的单层纸卷如何更好的文章留到以后再说)。也许你真的能分辨出幸运符和棉花糖伴侣。又或许你的保险只覆盖了 Pepper 医生(他上了四年汽水医学院不是为了被 Pibb 先生抢风头)。在评论中分享你的偏好，我们会在以后的帖子中总结它们。(辣&辣 Cheez——它的永远！)
