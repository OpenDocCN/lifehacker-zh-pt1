# 被动投资应该很无聊

> 原文:[https://life hacker . com/passive-investing-should-be-boring-1846255446](https://lifehacker.com/passive-investing-should-be-boring-1846255446)

如果你是一名被动投资者，希望长期积累你的储蓄，你应该多久检查一次你的投资组合？既然许多人都有一种 [损失厌恶偏见](https://en.wikipedia.org/wiki/Loss_aversion) 会导致对负面消息的过度反应，那么你应该幸灾乐祸地不知道，还是每天关注你的股票？答案取决于哪个选项最无聊。

### **将情感从你的投资组合中剔除**

比你多久看一次你的投资组合更重要的是你如何对投资组合的变化做出情绪反应。由于许多人 [感觉损失金钱比他们享受赚钱](https://money.com/stock-market-losses-gains-graphic/) 更多，这可以产生一种偏见以避免未来的损失。这是过于频繁地检查你的投资组合会导致糟糕的、冲动的决定这一观点的基础。

出于这个原因，通常是[推荐](https://investorplace.com/2020/01/how-often-check-investment-accounts/) 你要么每个季度，要么每个月不要查看你的投资组合超过一次。距离可以帮助你更理性地看待投资组合的表现。你将摆脱短期噪音带来的 [压力](https://www.betterment.com/resources/high-frequency-monitoring/) ，可以专注于实现长期目标。

换句话说，如果市场波动把你逼入绝境，每天查看可能不是一个好主意，除非你自律。保持常规，理性，和 [无聊](https://www.nytimes.com/2021/02/04/upshot/stock-market-winning-strategy.html) 。

### 好吧，但是你应该多久平衡一次你的投资组合？

平衡你的投资组合更多的是确保你感到舒适的 [预定风险水平](https://www.investopedia.com/terms/r/rebalancing.asp) 仍然反映在你积累的股票、债券和其他资产类别的组合中，因为它们的风险和回报都不同。例如，如果你在股票上表现出色，你投资组合的整体价值就会增加，因此，与债券等其他资产相比，股票的权重会更高。因此，再平衡包括买卖投资，以维持适当的资产配置。 [正如Investopedia所说](https://www.investopedia.com/investing/how-renew-and-adjust-your-portfolio/) :

> 投资组合再平衡只不过是对你的投资进行定期维护，就像去看医生做检查或给汽车换机油一样。

这就是为什么很多人每月结算 [或者每季度结算](https://www.investopedia.com/articles/stocks/11/rebalancing-strategies.asp) 的原因。尽管如此，当意外的市场事件可能使他们的投资组合失去平衡时，投资者也可能会启动资产重新配置，比如当 [股市上涨或下跌5%或更多](https://www.moneyunder30.com/rebalance-your-portfolio) 。再平衡的理想频率应该基于实用性、交易成本和投资组合中的 [允许漂移](https://ycharts.com/glossary/terms/drift) 。

尽管如此，有证据表明，从长远来看，你重新平衡投资组合的速度并不那么重要。 [据Investopedia](https://www.investopedia.com/investing/how-renew-and-adjust-your-portfolio/) 报道，2009年Vanguard的一项研究显示，无论是每月、每季度还是每年重新平衡投资组合的被动投资者，其平均年化回报率和波动性在三组中几乎相同。这就是为什么Vanguard只建议每半年或每年检查一次你的投资组合。

### **底线**

由于投资组合分配不需要太多的干预，所以真的没有必要每天检查你的长期投资。尽管如此，许多人喜欢观察他们投资的日常变化，他们这样做的同时保持情感上的距离，而其他人会因为波动而感到压力。如果短期波动影响了你的判断，那么每月或每季度检查一次的不干涉方法对你来说可能是更好的计划。
