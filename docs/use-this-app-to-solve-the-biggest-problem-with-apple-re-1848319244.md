# 使用此应用程序解决苹果提醒的最大问题

> 原文:[https://life hacker . com/use-this-app-to-solve-the-maximum-problem-with-apple-re-1848319244](https://lifehacker.com/use-this-app-to-solve-the-biggest-problem-with-apple-re-1848319244)

提醒应用程序让我们越来越喜欢。随着最新的 [iOS 15](https://lifehacker.com/36-of-the-best-new-ios-15-features-for-iphone-1847674175) 更新，你现在可以访问 [标签和智能列表](https://lifehacker.com/you-should-definitely-be-using-smart-lists-on-your-ipho-1847657547) ，这使得它可以用作 [电影或阅读列表跟踪器](https://lifehacker.com/the-best-way-to-organize-your-tv-and-movie-watchlist-is-1848250529) 。

Watch

提醒现在也变成了一个功能强大的任务管理应用，以至于你可能不需要像Todoist或TickTick这样的第三方解决方案。“提醒”是免费的，可以和Siri一起使用，默认情况下，它会安装在你所有的苹果设备上。

但提醒有一个明显的问题:虽然这款应用在管理任务方面很棒，但在添加提醒时却有点糟糕。你需要选择列表并点击**加**按钮来记录任务，即使这样，添加细节如注释、截止日期和时间也是的一件苦差事。

[提醒我快一点](https://apps.apple.com/us/app/remind-me-faster/id985555908) 可是解决麻烦的好办法。这是一个简单、免费的应用程序，只有一个目的——尽可能快地在提醒应用程序中获取您的想法。

应用程序打开时，键盘已准备就绪，默认列表已经选定(但您可以使用左上角的下拉菜单进行更改)。

写下您的提醒，添加备忘录，选择您想要的时间(它有预设的日期和时间)，然后轻按“发送”按钮。一秒钟后，提醒将被添加到您选择的列表中，并包含所有相关信息。

该应用程序还允许你创建有截止日期的任务，没有截止时间。在到期日期部分向上滑动以切换到日模式。在应用程序的设置中，您还可以创建预设位置，以便为任务快速添加基于位置的提醒。只需点击位置旁边的点图标，即可在三种优先模式之间切换。

尽管这款应用速度很快，但它确实遗漏了一些东西。没有对添加日期的自然语言处理的真正支持。您也不能给提醒事项添加标签或照片。

但如果你经常使用提醒应用程序，你应该将提醒我更快的应用程序放在你的Dock或iPhone的第一个屏幕上。在一段时间内，尝试使用这个应用程序来记录你所有的提醒。一旦你这样做了，你会发现越来越少的任务从你的脑海中溜走。
