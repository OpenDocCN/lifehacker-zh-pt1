# 你应该空气油炸一些 Shishito 辣椒

> 原文：<https://lifehacker.com/you-should-air-fry-some-shoshito-peppers-1847012341>

shishito 辣椒是一种非常有趣的辣椒。大多数都是温和而明亮的，但偶尔你会有一个非常辣的，正是这种紧张感使 shishitos 充满诱惑。那，他们是优秀的浸渍者。



烹饪 shishito 辣椒从来都不是一个伟大的烹饪挑战。你可以烤、烧、烤或者在热锅里泡。一旦你看到烧焦的小泡泡，它们就可以吃了。但是，虽然烹饪 shishito 辣椒从来都不难，但在空气油炸锅中烹饪它们几乎太容易了。

你在上面的照片中看到的十几个辣椒在我的小空气油炸锅完成加热后一分半钟就准备好了。我把它们扔进凉的、没有加热的篮子里，放入刚刚够涂的橄榄油，然后把温度设置到 400 华氏度，让它们用微型对流烤箱加热。炊具达到目标温度仅一分钟后，除了几个辣椒之外，所有的辣椒都奇妙地起泡了，另外 30 秒就搞定了最后两个。

我现在准备和吃 shishito 辣椒的速度几乎是危险的，但幸运的是，它们是一种蔬菜，很难吃“太多”蔬菜。

真正的挑战在于决定用什么来蘸它们。任何含有大豆和生姜的食物都很好，但是我喜欢在附近放一些奶油和冷却的东西，以防万一我碰到舌头。我一直在用额外的辣椒粉做这个杜克式蛋黄酱 ，我认为它会与这些辣椒搭配得非常好，但我随时欢迎更多蘸酱的建议。
