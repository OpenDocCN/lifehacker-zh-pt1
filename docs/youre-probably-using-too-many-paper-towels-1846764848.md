# 你可能用了太多的纸巾

> 原文：<https://lifehacker.com/youre-probably-using-too-many-paper-towels-1846764848>

纸巾只是另一种形式的[不可回收纸垃圾](https://www.businessinsider.com/dont-recycle-napkins-paper-towels-2016-2) ，你可能比你需要的更多地使用它们。根据美国人口普查数据和 NHCS 西蒙斯全国消费者调查，平均来说，大约有 4530 万美国人每月使用 8 卷或更多的纸巾。作为一次性产品，纸巾本质上是浪费的(尤其是吸水性更强的多层品牌),并且会导致森林砍伐。它们也很贵——根据 At 共享的数据，美国占了消费市场的近一半，[—除非你把它们制成堆肥，当你不需要的时候使用它们，只会给我们的大规模垃圾填埋场](https://www.theatlantic.com/family/archive/2018/12/paper-towels-us-use-consume/577672/) 增加一件垃圾。



当你可以使用海绵的时候，使用餐巾或纸巾来清理溢出物或擦拭水槽可能是你的习惯。在我姐姐的纸巾之家度过了感恩节后，我个人被迫重新评估我的纸巾瘾。当我用海绵或洗碗巾而不是纸巾清理一个又一个的溢出物或垃圾时，我开始问自己我到底为什么要用纸巾。然后，当火腿开始漏得到处都是时，我想起来了，是的，有时候你确实想把脏东西拖干净，然后扔到垃圾桶里。我并不是建议我们废除纸巾，但是我们都可以少用它们。这里有一些例子说明纸巾是值得的，还有一些例子说明你最好还是拿些可重复使用的东西。

### 毛巾或海绵在粗糙的表面上更有效

在粗糙或有纹理的 T2 表面上使用纸巾是浪费时间。纸巾会撕裂或留下颗粒，增加你试图清理的混乱。对于这些任务，使用海绵或刷子；它将承受清洁粗糙表面所需的压力，你可以一次又一次地使用它。

### 用毛巾吸干地毯上的溢出物

擦地毯也是一样。毛巾或面巾比纸巾更容易弄脏地毯。纸巾不具备毛巾的吸水性，如果你将它们压入地毯中(或者，但愿不会，试着擦洗污渍)，你只会将 [碎纸巾纤维](https://www.apartmenttherapy.com/paper-towel-use-mistakes-36717182) 加入混合物。取而代之的是投资更多的吸收性清洁布 ，它们将更好地吸收溢出物并且不留痕迹。

### 在玻璃、镜子和电子产品上使用微纤维

我们的电子产品通常需要比粗糙纸张更柔软的触感——同样的情况也出现在或玻璃和镜子上。纸巾中的微小纤维会划伤触摸屏和眼镜之类的东西。用超细纤维布代替擦拭你的 [游戏控制器](https://lifehacker.com/how-to-clean-your-gaming-controllers-without-damaging-t-1845783130) 、平板电脑、手机和其他表面。

### 用布擦拭小的溢出物和污渍

很难让自己放弃轻率地使用纸巾来清理小的脏乱和溢出物，但是这些时候你最好选择可重复使用的布来代替。纸巾它们可以重复使用，吸收数倍于自身重量的物质。

### 擦你的手

你很可能不会在家里的浴室里用纸巾擦手(那将是对纸张的巨大浪费！ ) ，所以当你在 [厨房做饭](https://www.rd.com/list/shouldnt-clean-with-paper-towels/) 的时候，没有理由那样使用它们。水槽就在那里；把手上的食物弄掉，然后用抹布擦干。说到这里:绝对不要用纸巾擦盘子。

### 你用纸巾擦油脂吗

另一方面，清理油污可能完全值得一两张纸巾。 [油脂](https://www.thekitchn.com/why-being-paper-towel-free-in-87955) 像橄榄油、黄油、熏肉脂肪都会弄脏衣服，使它们很难再利用。使用纸巾清除多余的油脂或其他物质，这些物质可能会使清洁布无法使用。你仍然可以改变你的习惯，减少使用次数(例如，把油腻的熏肉放在金属架上，而不是放在纸巾上)。

### 用纸巾清理宠物的粪便

当你的小狗出了事故，或者猫咳出了一团毛时，想抓住毛巾，然后再用它来清洁你的工作台面是一个挑战。用纸巾就好了，整个扔掉。我们希望在这里减少。你可以想怎么用力就怎么用力——只是要记住，虽然你可以经常清洗和消毒毛巾，但这样做所用的水量可能会抵消掉避免使用纸巾带来的一些环保好处。
