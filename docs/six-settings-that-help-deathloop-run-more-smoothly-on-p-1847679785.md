# 帮助 Deathloop 在 PC 上更流畅运行的六个设置

> 原文：<https://kotaku.com/six-settings-that-help-deathloop-run-more-smoothly-on-p-1847679785>

如果你一直在 PC 上玩*死亡循环*，那么你很有可能遇到过一些困扰刚刚发布的 游戏的 [的诸多问题。好消息是，在我们等待补丁的时候，你可以做一些事情来平滑你的体验。](https://kotaku.com/even-beefy-rigs-arent-saving-deathloop-from-pc-issues-1847676264)

关于是什么导致了这些问题还没有官方说法，如果我们问他们，贝塞斯达也不会回答。但共识似乎分成了两个部分，一个是通常令人厌烦的问题，几乎伴随着每一款搭载毫无意义的 Denuvo 反盗版软件的游戏，另一个是某种复杂形式的内存泄漏。无论哪种方式，结果都是如此多的人在 PC 上玩 Arkane 的最新沉浸式 sim，有一些非常不愉快的口吃，帧速率下降到 0.1 FPS。

我肯定被后者所困扰，还有游戏抱怨我的 3070 没有足够的 VRAM 来正常运行，这是...不对。但是如果你愿意修改设置，你应该能够让它平静下来，防止口吃。

### 垂直同步

为了提高性能，你首先要做的是关闭垂直同步。那是在选项-视觉效果-视频设置里。然而，如果你不想玩屏幕撕裂的游戏，你可以转到你的 GFX 卡自己的设置，并在那里强制执行。NVidia 的控制面板有管理 3D 设置下的选项。

### NVidia 反射

同样在*死亡循环*中的选项-视觉-视频设置下是 NVidia Reflex。这是一个可以减少延迟的工具，所以希望也可以防止一些可怕的口吃。如果你打开这个，游戏建议你也打开低延迟，选项如下。

### 扩大规模

接下来在选项-视觉-视频设置中要改变的是升级。可供选择的选项是 AMD FidelityFX 超分辨率 1.0。这与 AMD 或 NVidia 显卡一起工作，并升级较低分辨率的图像以提高帧速率。然后你可以使用下面的设置，AMD FSR 模式，来选择你想要多高的提升。在中等钻机上平衡，但如果你足够幸运，有一张更新的 GFX 卡，你可以把它提高到超高质量。

### FPS 限制

将其设置为 60 到 120 之间的任何值。为什么？我假装我知道。但是大一点的男孩说要这么做。

### 环境遮挡

这个隐藏在选项-视觉-高级设置中，似乎默认为电脑不能光线追踪的玩家打开光线追踪。所以一定要深入了解，看看发生了什么。如果你没有花式卡，设置为 FidelityFX CACAO 性能，不要问问题。或者你可以把它提升到平衡或者质量，然后提升到 NVidia HBAO+，或者以不同的质量进行光线追踪。没有选项可以完全关闭它。

### 阴影

同样在高级设置中，阴影可以让你从很低到很高。把这个降低应该会看到你的每秒帧数上升，所以这是值得下降到中等。

奇怪的是，像纹理细节这样的常见问题似乎对这里的性能没有太大的影响，所以不要担心将所有这些设置为低，然后对此感到愤怒。模型细节也是如此。当你在高级设置中时，你也可以关掉糟糕的相机运动模糊。游戏:别这样。

其中一些设置需要重启游戏才能生效，所以在诅咒我的名字之前请记住这一点。然后你可以再次享受从按下“Play”到游戏真正开始所花的时间。

显然，在*死亡循环*中有很多修复工作要做。举例来说，现在它甚至不能理解屏幕分辨率。将游戏设置为窗口和 2560 x 1440，你会得到 2480 x 1440，这只是普通的*怪异的*。(3440 x 1440 得到了更奇怪的 3312 x 1387，这甚至都不接近。)
