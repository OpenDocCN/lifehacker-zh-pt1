# 现在，您可以从 Windows 上的图像中复制文本

> 原文：<https://lifehacker.com/you-can-now-copy-text-from-images-on-windows-1849504993>

微软的 PowerToys 有一个新的实用程序，可以让你从图像中复制文本:如果有人给你发送了他们的名片照片或他们银行账户详情的截图，你可以使用 PowerToys 在 Windows 上快速复制。新的实用程序叫做文本提取器，与[PowerToys 0.62 版本更新](https://github.com/microsoft/PowerToys/releases/) 一起提供。可以从 GitHub 下载最新版本的 PowerToys 安装完成后，打开 PowerToys 并点击左侧窗格中的**文本提取器**。

## 如何在 Windows 上使用文本提取器

现在，您将在 PowerToys 的右窗格中看到文本提取器的键盘快捷键。默认情况下，这个快捷键是 **Windows 键+ Shift + T** 。下一次当你在图像中遇到文本时，或者如果你在一个阻止复制粘贴的网站上，使用键盘快捷键激活文本提取器，然后点击并拖动文本来复制它。你也可以按住 **Shift** 将原来的选择拖动到屏幕的其他部分，如果你在选择时犯了一点小错误，这很有帮助。一旦文本被复制，打开任何允许你输入文本的应用程序，按下 **Ctrl + V** 将其粘贴到那里。

如果您不喜欢默认的键盘快捷键，您也可以自定文本提取器。转到 PowerToys 并在左窗格中单击文本提取器，然后在右窗格中单击**设置**。C 点击**激活快捷键**旁边的**铅笔图标**。在新窗口中，按下要绑定到文本提取器的组合键；点击**保存，**就大功告成了。请注意，此键盘快捷键必须以 Windows、Ctrl 或 Alt 键开头。

## 尝试其他新的 PowerToys 工具

PowerToys v0.62 还带来了一些其他有用的工具。快速重音可让您更快地键入重音字符，屏幕标尺可让您测量屏幕任何部分的像素数量。

快速口音需要通过 PowerToys 手动启用。一旦你这样做了，你可以按住键盘上的任何字母，并立即按下左箭头键或右箭头键。你会看到一个整洁的工具栏，上面有与该字母相关的所有重音字符——选择你需要的任何一个。

屏幕尺也好用。PowerToys 更新到 v0.62 后，按 **Windows + Shift + M** 启动。从工具栏中选择**边界**工具，点击并拖动您想要测量的区域。您将在屏幕上看到该区域的像素数量。要让它留在屏幕上，按住 **Shift** 同时点击并拖动。
