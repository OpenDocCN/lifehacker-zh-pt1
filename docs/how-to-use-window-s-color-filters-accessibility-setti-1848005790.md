许多现代操作系统和应用程序都包含一些设置，让色盲(或色盲*)的用户更容易使用这些设置——以下是你如何在Windows的四个最新版本中设置这些选项: 11、10、8和7。*

## *如何为色盲用户设置 Windows 11或Windows 10*

*在微软仍然支持的所有Windows版本中，Windows 11和10都是最容易为你的电脑配置色盲滤镜的。*

*在这两个版本的Windows上，色盲支持是通过“颜色过滤器”功能运行的。您可以通过两种方式 中的一种 [访问该功能，或者打开开始菜单并搜索“颜色过滤器”，或者进入设置并选择**易用性**(在Windows 11中为**易用性**)，然后选择**颜色过滤器**。](https://support.microsoft.com/en-us/windows/use-color-filters-in-windows-43893e44-b8b3-2e27-1a29-b0c15ef0e5ce)*

*一旦颜色过滤器被激活，六个可用颜色过滤器中的一个将被自动应用。如果您以前从未使用过该功能，Windows将默认激活“灰度”滤色器。*

*除了灰度之外，您还可以选择两种额外的颜色滤镜或三种色盲滤镜中的一种，总共有六种滤镜选项。目前，Windows 10和11都提供了红绿(绿色弱/暗红色)、红绿(红色弱/暗红色)和蓝黄(三色)滤镜，此外还有三种颜色滤镜选项。*

*合适的过滤器将取决于您的个人需求。大约在 [时，8%的白人男性患有某种形式的色盲](https://blinkux.com/ideas/seeing-red) ，这可能会削弱他们区分红色和其他颜色的能力。对于色盲的这部分人来说，鱼精色或弱红绿滤镜效果最好。*

*单击任何过滤器选项都会自动为您的计算机设置它。如果您想要预览滤镜在使用时的外观，可以使用Windows 10中颜色滤镜页面底部的色轮(或Windows 11中出现在页面顶部的色轮)，以及与色轮颜色匹配的调色板和图像。*

*在Windows 10中，你也可以在这个页面上激活颜色过滤器的快捷方式，如果你有一个自己喜欢的色盲设置的应用程序，这将非常有用。默认情况下，快捷键将是 **Windows键+ Ctrl + C** 。*

## *其他版本的Windows中的辅助功能选项*

*如果您运行的是早期版本的Windows，您仍然可以选择；然而，您可能只能更改Windows主题的颜色。【Windows 8和Windows 7都有增加屏幕对比度的高对比度模式，使不同的颜色更容易区分。*

### *Windows 8*

*在Windows 8中，你可以通过导航到 **设置** > **易用性** > **高对比度**来打开或关闭这种模式。您可以使用快捷键**Alt+left Shift+Print Screen**随时打开或关闭该模式。*

### *Windows 7*

*在Windows 7中，这个过程看起来很相似。从**控制面板**中，选择**易访问中心>**T5】使计算机更容易看到。使用此菜单，您可以激活高对比度模式并选择高对比度主题。和Windows 8中一样，你可以通过快捷键 **Alt +左Shift +打印屏幕**随时激活或停用**高对比度**。*

### *为您的电脑设置色盲的其他选项*

*如果可用的高对比度主题不能提供您正在寻找的内容，您可能会发现第三方应用程序可以提供类似于Windows 10提供的过滤功能。像独立应用程序 [Visolve](https://www.ryobi.co.jp/products/visolve/en/) 或[Firefox extension ColorBlindExt](http://hrmoh.ir/clipmarks-backup/clips/LQjRgUaCuVPq5h6PbmecOmp3YT6h3CpmzSue/index.html)这样的工具可以提供滤镜，使缺乏色彩的用户更容易区分不同的颜色。*

*越来越多的应用程序和视频游戏也开始实现色盲功能。通常，如果这些功能可用，您可以通过应用程序设置菜单的视频或辅助功能部分来访问它们。*
