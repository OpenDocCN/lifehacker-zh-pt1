# 重命名 Chrome 窗口，这样你就不会丢失它们

> 原文：<https://lifehacker.com/rename-chrome-windows-so-you-stop-losing-them-1846714841>

选项卡过载是真实的，正如我拥挤的 Edge 和 Chrome 浏览器栏可以证明的那样。但是关于什么...窗口超载？如果你是那种试图通过为每个选项卡或选项卡组分配不同窗口来组织浏览器的人，谷歌现在允许你重命名这些窗口，这样你就可以更轻松地在它们之间跳转。



警告？一旦你完成了一天的浏览，你需要确保你总是使用**三点菜单>退出来关闭 Chrome。**这个 会立刻切换你所有打开的窗口，但是只要你在 Chrome 的**设置>隐私和安全**中启用**继续你离开的**，当你再次启动浏览器时，它们都会重新出现——自定义名称和一切。

为每个 Chrome 浏览器窗口设置一个自定义名称很简单。首先，确保你运行的是最新版本的 Chrome(通过**三点菜单>帮助>关于谷歌 Chrome** )。在那之后，你所要做的就是在你的标签栏的任何空白区域点击鼠标右键，弹出一个带有新的**“名称窗口”的上下文菜单**选项。

永远关闭特定的 Chrome 浏览器窗口会让你的自定义名字消失。这很有意义，因为你已经基本上消除了浏览器的“实例”。如果你希望保持永久的、独立的浏览器窗口，你真的应该在你的台式机或笔记本电脑上设置 [独立的 Chrome 配置文件](https://lifehacker.com/create-a-separate-chrome-user-account-to-reduce-distrac-1694477575) ，这将为你提供完全独立的 Chrome 版本，你只需点击几下鼠标即可调用。

例如，我有一个“个人”Chrome，我默认使用它来浏览大部分内容，还有一个单独的“工作”Chrome，它被配置为可以处理我所有的 Lifehacker 内容。它有我工作中使用的所有标签，包括我的电子邮件、我正在写的故事和我正在研究的的故事，它不会被我许多打开的标签阻塞，比如说，掌握*星谷*。这些游戏标签也没有以任何方式与我的工作账户绑定，这可能是我和我的老板更喜欢的。)

如果你发现在标签栏上点击鼠标右键很难调出正确的上下文菜单，你也可以进入那个完全相同的**Name 窗口...**将鼠标悬停在 Windows 10 任务栏中的 Chrome 图标上和右键点击任何一个打开的 Chrome 窗口。在 macOS 上，如果你想要一个超级简单的方法来重命名你当前的 Chrome 实例，只需点击**窗口>命名窗口**。
