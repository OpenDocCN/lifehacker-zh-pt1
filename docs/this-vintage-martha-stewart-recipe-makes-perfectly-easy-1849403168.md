# 这种老式玛莎斯图尔特食谱非常容易蜜饯水果

> 原文：<https://lifehacker.com/this-vintage-martha-stewart-recipe-makes-perfectly-easy-1849403168>

我喜欢主持晚宴，但我讨厌主持焦虑。为了克服这一点，我一直在寻找美丽的，令人印象深刻的，秘密简单的食谱。艾娜·加滕(Ina Garten)和玛莎·斯图尔特(Martha Stewart)是两位厨师，不可否认是出色的晚宴主持人，他们掌握了这些类型的食谱(严格来说，我还没有参加过他们的晚宴，但请给我时间)，自从我妈妈最近发现了她那本《T2》[*【娱乐】*](https://www.abebooks.com/servlet/BookDetailsPL?bi=30733496718&searchurl=an%3Dmartha%2Bstewart%26fe%3Don%26sortby%3D17%26tn%3Dentertaining&cm_sp=snippet-_-srp1-_-title5) (1982)，这是一本玛莎关于举办完美聚会的经典书籍，我向它寻求建议，并找到了玛莎制作诱人水果盘的简单秘诀——给你的水果加糖霜。



## 什么是冰糖水果？

糖衣水果本质上是涂了一层糖釉的生水果，这给了它一种光彩夺目的反光。虽然这个术语是法语，但这种技术被用来制作一种等效的中国街头食品，称为，并且可能在许多文化美食中使用。将这种糖果技术应用到新鲜的季节性水果上会产生优雅的效果，点缀在奶酪板上看起来特别好。一个额外的好处是:糖衣的甜味可以帮助过季水果调味。

给你的水果浇糖霜，做一个融化的糖浆，小心翼翼地把水果浸入其中，时间要足够长。结束了。薄薄的一层糖在几分钟内就冷却下来，闪闪发光地附着在水果表面。很简单。

## 玛莎是怎么做的

玛莎的食谱使用草莓和绿葡萄，但我尝试用新鲜的樱桃，因为我碰巧有很多。因为你要把水果浸在超过 250 华氏度的融化的糖里，所以强烈推荐有可抓茎的水果。樱桃、葡萄串和草莓都很容易拿着蘸着吃。任何没有茎的东西都可以串起来蘸着吃。无论你喜欢哪种水果，都要注意把大部分的糖都榨干，因为嚼一大块糖一点都不好玩。

第一步是准备好你的水果。洗干净晾干，确保在室温下。这样做的目的是确保糖能顺利地附着在水果的表皮上。当水果浸入超热的糖中时，任何水分(包括冰箱冷藏水果的冷凝物)都会迅速蒸发，并可能导致潜在的危险蒸汽飞溅，降低糖浆温度的奇怪冷点，或者(在最好的情况下)轻微破坏糖衣光泽的小气泡。从室温水果开始，好吗？

当你的水果干了，准备好了，做热糖浆。使用一个小而深的罐子，至少一杯糖和几汤匙水。在玛莎的食谱中，她用两杯糖盖住几串葡萄和一品脱草莓。你不会用光所有的糖浆，但你希望有一个足够深的水池来快速方便地蘸水果。我不知道我有没有告诉过你，但是糖太烫了，你需要快点，这样你就不会把水果煮了。

用中高火煮沸混合物，直到蒸汽停止(因为所有的水都蒸发了)，气泡开始减速并变大。如果你不确定，用糖果温度计来检查温度。食谱建议，一旦糖浆达到 265 华氏度，你就可以浸泡水果，这在糖果术语中被称为“硬球阶段”将每个水果蘸上足够长的时间，使其覆盖在外面，然后放在抹了油的烤盘上冷却。玛莎的食谱到此为止，但我做了一些调整。

我不喜欢我的水果蜜饯表面的样子。硬球阶段最适合粘稠、耐嚼的糖果，如焦糖。我喜欢焦糖，但不是这个。当你咬水果时，糖会立刻粘在每颗牙齿上。当我试着装盘时，樱桃也粘在了一起，这在晚宴上是很可怕的。相反，我发现把糖煮到“软脆-硬脆”阶段对我的樱桃更好。当我咬它的时候，薄薄的糖衣粉碎了，在柔软成熟的水果旁边创造了一个令人愉快的并列结构。糖是酥脆的，而不是耐嚼的，在我的嘴里溶解了。电镀工艺也有所改进；硬脆饼干里的糖没有那么粘——除非它变湿了。为了防止这种情况，在你打算使用水果的一两个小时内给水果涂上一层，并把它放在一个密封的容器里。

我建议的最后一个变化是浸泡后在涂油的晾衣架上冷却水果。尽管我尽了最大的努力，当放在羊皮纸内衬烤盘的上时，一些糖仍然聚集在每个樱桃的底部。晾衣架会让我快速抖掉多余的糖，尽可能保持涂层薄而光滑。

有了这些小小的编辑，这种准备技术令人眼花缭乱:一种诱人的方式来呈现丰富的夏季水果。如果把水果完全裹在糖果里的想法让你牙疼，你可以只把糖浆洒在水果上来代替。只需将水果排成一行放在羊皮纸上，然后将糖浆一点点撒在上面即可。结果将是闪闪发光的食物，看起来像是在一个带着露水的早晨刚刚收获的。
