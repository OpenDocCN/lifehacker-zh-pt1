# 如何在 Xbox 云流媒体上玩游戏(以及为什么它如此重要)

> 原文：<https://lifehacker.com/how-to-play-on-xbox-cloud-streaming-and-why-its-such-a-1847197043>

经过几个月的限制，仅限受邀者的封闭测试后，Xbox 云游戏现已公开发售。这项服务在技术上仍处于测试阶段，但现在包括了所有 Game Pass Ultimate 订户，为可以说是游戏 中最好的服务 [增加了更多价值。](https://lifehacker.com/which-pc-gaming-subscription-service-is-best-1845019353) 

Watch

[Xbox 云游戏测试版](https://www.xbox.com/en-US/xbox-game-pass/cloud-gaming) 支持运行最新版本 Android、iOS/iPadOS、macOS 或 Windows 10 的设备。微软还计划在未来将云流媒体技术引入 Xbox One 机型，并将支持即将推出的 Windows 11。

Xbox 云流媒体是一件大事:虽然存在许多其他云流媒体游戏服务，如 Stadia 和亚马逊，但 Xbox 云流媒体是第一次一家老牌游戏公司推出自己的云流媒体服务。如果它运行良好，它可能标志着 Xbox——以及整个游戏——的一个重大转折点，如果你玩最新游戏如 *Halo Infinite* 所需要的只是一部智能手机、一个稳定的互联网连接和 Game Pass Ultimate。

## Xbox 云流媒体升级了云流媒体基础设施

Xbox 云流媒体有潜力，但如果这项服务要像微软希望的那样取得成功，它需要正确的技术。为此，微软升级了其流媒体技术性能，同时扩大了测试版的可用性。游戏现在可以在相当于 Xbox Series X 的硬件上运行。

流质量仍然限制在 1080p 分辨率和 60fps，所以你不会实现实际 Xbox 系列 X 可以实现的 4K 120fps 游戏，但其他技术增强，如光线跟踪和(潜在的)更快的加载时间现在是可能的。输入延迟和整体性能取决于您的互联网连接，但您可以通过 wifi 或移动连接进行流式传输，包括 5G(如果可用)。

我们将向现有的 Game Pass Ultimate 用户展示如何在他们的首选设备上设置 Xbox Cloud Streaming。

## 如何使用 Xbox 云流媒体

你需要一些东西来访问 Xbox 云流媒体:

*   **一个游戏通行证终极订阅。对于那些还没有订阅的用户，微软提供前三个月每月 1 美元的特别介绍价格。在那之后，价格会一个月涨到 15 美元，但对于使用 Xbox 和 PC 游戏的旋转库来说，这仍然是一个不错的价格。**
*   **支持的设备**运行最新版本的 Android、iOS、iPadOS、macOS 或 Windows 10。查看此页面以获取受支持设备 的 [完整列表。](https://www.reddit.com/r/xboxinsiders/wiki/xgspreview/gettingstarted#wiki_xbox_cloud_gaming_verified_devices)
*   **一个蓝牙或 USB 控制器** [与您的设备](https://lifehacker.com/how-to-connect-xbox-and-playstation-controllers-to-your-1842640968) 配对。你可以使用 Xbox 控制器或支持的第三方选项，如智能手机的主干或 Razer Kishi。(有些游戏也支持触摸控制，但使用实际的游戏手柄会舒服得多。)
*   高速互联网连接。微软建议在 5Ghz Wi-Fi 上使用至少 10Mbps 速度的网络。

最后，你需要合适的网络浏览器。目前支持的每个平台都通过 web 访问服务，但浏览器支持因操作系统而异。

**安卓:**

*   铬

**iOS 和 iPadOS:**

*   旅行队

**苹果电脑:**

*   旅行队
*   边缘
*   其他 Chromium 浏览器(如 Brave、Opera、Vivaldi 等。)

**Windows 10:**

*   铬
*   边缘
*   其他 Chromium 浏览器(Brave、Opera、Vivaldi 等。)

一旦您配对了控制器并安装了正确的浏览器，只需在您的设备上加载[【xbox.com/en-US/play】](https://www.xbox.com/en-US/play)，使用您的 Game Pass 帐户登录，然后从您的游戏库中选择一个游戏进行游戏(注意:目前只有部分 Game Pass 游戏支持云流媒体。随着时间的推移，将会添加更多的标题。)

## Xbox 云流媒体也有家庭流媒体

Xbox 云流媒体对 Game Pass 的订户来说是一项巨大的福利，但这不是当你离开电视时玩 Xbox 游戏的唯一方式。如果你拥有一台 Xbox Series X/S 或 Xbox One，你也可以直接从你的主机[【Android】](https://lifehacker.com/how-to-stream-your-xbox-one-to-your-android-phone-or-ta-1841064651)[iOS](https://lifehacker.com/how-to-stream-xbox-games-on-your-iphone-or-ipad-1845428424)或 [Windows 10](https://lifehacker.com/windows-10s-xbox-app-is-awesome-even-if-you-dont-have-1723617404) 设备连接到同一个网络。
