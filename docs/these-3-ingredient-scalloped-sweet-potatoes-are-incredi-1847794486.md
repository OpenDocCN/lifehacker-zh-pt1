# 这些三种成分的干贝甘薯非常美味

> 原文：<https://lifehacker.com/these-3-ingredient-scalloped-sweet-potatoes-are-incredi-1847794486>

由于我的工作性质，我(有点不情愿地)已经开始考虑感恩节了。我很想抽一只火鸡，但是菜单的其他部分还没有解决。有一件事是肯定的:这些扇贝美人要上餐桌了。



当然，没有理由等到 11 月的第四个星期四来享用这一堆丰富的奶油红薯(或山药)。这份食谱正是我想要的秋季配菜。它便宜、容易，而且非常富有和舒适。

如果你已经阅读 Lifehacker 有一段时间了，你可能会注意到它与我从法国带回来的 [这道脆皮土豆食谱](https://lifehacker.com/the-best-potatoes-au-gratin-dont-contain-any-cheese-1825751769) 非常相似。比起山药，我几乎总是更喜欢白薯，但这是山药胜出的一个例子。橙色块茎的薄片*在它们的奶油浴中融化*，而奶油脂肪呈现出来，厚重的奶油起泡并变成褐色，形成一层起泡的外壳。所有这些浓缩成浓稠的奶油蛋羹，注入了山药的甜味——或者红薯，不管是什么——它主宰一切。

虽然这看起来像是对第三种成分的一种事后补充，但我敦促你不要在盐上吝啬。所有的脂肪和(自然产生的)糖使盐成为一种非常重要的成分。没有它，这道菜会呈现出一种调味不足的甜点那种黏腻腻的甜味，这可不是你想要的。这道菜几乎很难加盐——我对每一个山药都用了不到 1/4 茶匙，一点也不“太咸”

我喜欢这个食谱的另一点是，你可以根据需要放大或缩小它。你可以用一个小的铸铁锅，用一根山药来做，或者你可以做足够一群人吃的。我这个周末做的那个含有三个山药和一品脱重奶油，所以用它作为指导比例。

如果你想在火鸡日为她打扮一番，你可以随时准备一份磨碎的硬奶酪，一些面包屑，或新鲜的香草——或者将这三者结合起来做一份美味的脆皮面包，但在调整之前先以最纯净的形式旋转这道菜。这真的是难以置信的美味，品尝是可信的。

## 三种配料的干贝甘薯(或山药)

**配料:**

*   甘薯或山药，切成约 1/8 英寸厚的片
*   重奶油(每 3 个红薯至少 1 品脱，如果土豆大的话更多)
*   盐(每个土豆少于 1/4 茶匙)

把你的烤箱加热到 375 华氏度。用一把薄刀或曼陀林切片机(使用该死的防护装置)将红薯去皮切片，然后在砂锅或其他形状相似的烤箱器皿中放置单层切片——两端稍微重叠。倒入浓奶油，直到刚好盖住马铃薯片，然后在整个层上撒上一两层盐(不要留下未加盐的切片)。重复上述步骤，将红薯、奶油和盐分层，直到吃完红薯。

把整个东西放进烤箱，烤到顶部变成棕色，起泡，奶油在侧面冒泡，大约 30-45 分钟，取决于你有多少红薯。从烤箱中取出，冷却 10 分钟，然后切成美味的小方块，即可食用。剩菜可以很好的再加热，所以要做一大份。
