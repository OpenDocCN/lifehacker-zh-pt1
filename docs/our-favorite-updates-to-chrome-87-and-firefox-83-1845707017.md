# 我们最喜欢的 Chrome 87 和 Firefox 83 的更新

> 原文：<https://lifehacker.com/our-favorite-updates-to-chrome-87-and-firefox-83-1845707017>

谷歌的 Chrome 和 Mozilla 的 Firefox 浏览器的新版本昨天发布，如果你还没有更新，现在是最好的时机:访问浏览器的“关于 Chrome”或“关于 Firefox”部分，这两个部分都在每个浏览器的帮助菜单下，它会自动开始更新。在此期间，以下是你将在 [Chrome 87](#chrome) 和 [Firefox 83](#firefox) 中经历的主要变化。

### **Chrome 87 的新功能**

首先，也是最重要的一点，谷歌在几个关键领域做了一些调整来提升 Chrome 的速度。您不必启用任何东西来从这些调整中受益，只需更新到 Chrome 87，然后让我们开始吧。正如 [谷歌](http://e/) 所描述的:

> *   Chrome now prioritizes your active tabs over all open tabs-reducing CPU usage by 5 times and extending battery life by 1.25 hours (based on our internal benchmark).
> *   Now Chrome has a 25% faster startup speed and a 7% faster page loading speed, all of which use less power and memory than before.
> *   Chrome browser on Android can now load pages almost instantly when you navigate forward and backward, which makes these common tasks super fast.

我不会逐字逐句地复制谷歌的博客文章，但它强调的下一个功能对于像我这样倾向于同时打开太多标签页的人来说也是一个游戏规则改变者。如果你用的是 Chromebook，你现在可以通过点击 Chrome 最小化按钮右边的向下箭头来搜索打开的标签页。

### **火狐 83 的新功能**

Mozilla 也加入了“我们让浏览器变得更快”的游戏。下面这个词， [直接来自他们](https://www.mozilla.org/en-US/firefox/83.0/releasenotes/) ，关于火狐 83 中的性能提升打击:

> 由于我们的 JavaScript 引擎 SpiderMonkey 的重大更新，Firefox 变得越来越快，您现在将体验到页面加载性能提高了 15%，页面响应速度提高了 12%，内存使用减少了 8%。我们已经替换了 JavaScript 引擎的一部分，帮助您编译和显示网站，同时提高了引擎的安全性和可维护性。

接下来，同样重要的是一个新的 [**HTTPS 专用模式**](https://blog.mozilla.org/security/2020/11/17/firefox-83-introduces-https-only-mode/) ，Mozilla 将在 Firefox 中首次亮相。在 Firefox 的选项中启用它，你将不再需要像 HTTPS 这样的扩展；只要加密的 HTTPS 连接可用，您的浏览器就会默认使用这些连接，从而使您在浏览网页和使用信用卡信息时更加安全。

火狐也在调整它的搜索面板。像 Chrome 一样，你可以通过打开的标签页进行搜索，重新访问你已经忘记的任何东西。你也可以点击搜索面板底部的图标，将你的搜索重定向到特定的服务——谷歌、亚马逊、维基百科、易贝等。—或者专门在书签、标签或历史记录中搜索特定关键字。

Firefox 83 还为那些运行 MacBooks 或 Windows 笔记本电脑的触摸屏用户提供了新的**捏放功能**，以及新的键盘快捷键 用于观看画中画视频:

> *   **Ctrl** + **↓** :静音
> *   **Ctrl** + **↑** : cancel the mute.
> *   **↓** : the volume decreases.
> *   **↑** : increase the volume.
> *   **: Look backward for 15 seconds.**
> 
> *   **Search forward 10%**
> ***   **Home page** : Search video*   **at the beginning** : the video is searched.*   **Ending space** : pause/continue**

**Mozilla 还调整了 **Firefox 的 PDF 浏览器**的外观——无需设置即可启用。更重要的是，您可以直接在浏览器中填写 PDF 表单，并将其内容(包括您的输入)保存到您的台式机或笔记本电脑中。**
