# 你可以用这个隐藏的 Face ID 设置锁定你的秘密隐藏标签

> 原文：<https://lifehacker.com/you-can-lock-your-shady-incognito-tabs-with-this-hidden-1847160329>

谷歌正在为 iPhone 和 iPad 上的 Chrome 测试一项新功能，当它们在后台打开时，你可以用面部识别锁定匿名标签。(不幸的是，该功能在 Android 上不可用——至少目前还不可用)。

Watch

Chrome 的匿名模式阻止你的设备保存你的浏览历史，这在你需要偷偷查找一些东西时会很有帮助。然而，除非你每次停止使用 Chrome 时都关闭你的匿名标签，否则有人可以很容易地打开你的背景标签来查看你在做什么。不过，通过启用新的隐私功能，你可以安全地在 Chrome 中打开匿名标签，而不用担心有人在你不注意的时候窥探你隐藏的内容。

该功能在稳定的 iOS Chrome 应用程序中可用，但它隐藏在实验标志菜单中，在应用程序设置中不可用。谷歌可能会在未来的应用程序更新中将其作为一个标准功能，但现在如何打开它:

1.  在 iPhone 或 iPad 的新 chrome 标签中打开 **chrome://flags** 。
2.  使用搜索栏找到**“匿名设备验证”**
3.  从下拉菜单中设置标志为**“启用”**。
4.  当提示保存并应用您的更改时，点击以重新启动 Chrome。
5.  重新打开 Chrome，然后进入**设置>隐私。**
6.  启用新的**“关闭 Chrome 时锁定匿名标签”**选项。

启用该功能后，您可以放心地在后台打开隐名标签，因为任何试图访问您的活动隐名标签的人都需要提供匹配的 Face ID 来解锁它们。

它是这样工作的:

1.  在 Chrome 中重新打开一个背景隐藏标签。
2.  出现提示时，轻触**“用 Face ID 解锁”**。
3.  让您的设备执行面部扫描。如果匹配，你的标签将解锁，你可以从你离开的地方继续浏览。

虽然锁定你的匿名标签会阻止其他人进入，但请记住，Chrome 的匿名模式 [不是匿名浏览模式](https://lifehacker.com/what-incognito-mode-can-and-cant-do-to-protect-your-dat-1843888437) 。网站和互联网服务提供商仍然可以在匿名模式下跟踪大量数据，因此它不应该用于替代其他隐私工具，如 [跟踪拦截器](https://lifehacker.com/block-chromes-floc-tracking-with-this-duckduckgo-extens-1846667051) 和[VPN](https://lifehacker.com/how-to-find-a-trustworthy-vpn-1833045522)。
