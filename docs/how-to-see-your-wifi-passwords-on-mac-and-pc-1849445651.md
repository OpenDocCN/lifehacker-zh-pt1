# 如何在 Mac 和 PC 上查看您的 Wifi 密码

> 原文：<https://lifehacker.com/how-to-see-your-wifi-passwords-on-mac-and-pc-1849445651>

wifi 密码是一种既让人渴望又让人害怕的力量:每个人都想知道它，但忘记它，你就完了。为什么微软和苹果使得在 Windows 和 Mac 上很难找到 wifi 密码还不清楚(毕竟这是*有价值的*信息)。但是，请放心，如果您准备点击一些菜单，这些密码是可用的。

不仅仅是你的电脑隐藏了 wifi 密码:你也可以在你的 iPhone 和 Android 设备上 [看到这些密码](https://lifehacker.com/how-to-unhide-your-wifi-password-1849151134) 。下次有人问你要无线网络时，只要你的一个设备连接到它，你就可以提供帮助。



## 如何在 Windows 上查看您的 wifi 密码

微软把你的 wifi 密码藏在各种设置屏幕后面。然而，如何到达那里，根据你的平台不同， [略有不同，](https://support.microsoft.com/en-us/windows/find-your-wi-fi-network-password-in-windows-2ec74b2e-d9ec-ade1-cc9b-bef1429cb678) 。如果你运行的是 Windows 11，点击**开始**，键入**控制面板**，然后前往**控制面板>网络和互联网>网络和共享中心**。如果你在 Windows 10 上，点击**开始**，然后前往**设置>网络&互联网>状态>网络和共享中心。**

从这里，在任何平台上，点击**连接**旁边的特定 wifi 连接，然后，在 **Wi-Fi 状态**中，点击**无线属性**。现在，打开**安全**选项卡，然后确保**显示字符**框被选中。完成后，您将看到您的网络密码。你可以和朋友分享或者写下来放在冰箱上。

## 如何在 Mac 上查看您的 wifi 密码

苹果做的事情和 Windows 有点不同。您不必查看当前连接的网络的密码；相反，操作系统会将您连接的每个 wifi 密码保存在设备的钥匙串中，以便随时随地访问。你可以把你旧公寓的 wifi 密码分享给转租人，或者透露那个咖啡店的密码，那个咖啡店*真的*不喜欢分发 wifi 。

要找到它，请在 Mac 上打开“钥匙串访问”(最快的方法是使用**命令+空格键**来搜索它)。在搜索栏中键入您的 wifi 网络名称，然后当它出现在列表中时点按它。突出显示网络后，单击窗口顶部的(I)，或者右键单击网络并选择**获取信息**，然后单击**显示密码**旁边的复选框。输入你 Mac 的登录密码，钥匙串访问会依次显示 wifi 密码。
