# 这些是 Mac 上 TweetDeck 的最佳替代品

> 原文：<https://lifehacker.com/these-are-the-best-replacements-for-tweetdeck-on-mac-1849132982>

截至 7 月 1 日，Mac 应用 [的 TweetDeck 已经停产](https://twitter.com/TweetDeck/status/1531994955555676163) 。该应用于 2008 年由独立开发者 Iain Dodsworth 首次发布(2011 年被 Twitter 收购)，在 Mac 上拥有一批用户的忠实追随者。对他们来说，这很痛苦。

Watch

TweetDeck 以多栏布局、实时推文和快速提醒而闻名。这是一个简单的应用程序，但它做得很好。如果你在哀悼损失，这里有一些其他的选择可以尝试。

## [翠儿](https://tweetenapp.com/)

你可以把 Tweeten 看作是 TweetDeck 的精神继承者。这款应用基于 TweetDeck，目前正在积极开发中。TweetDeck 用户将会有宾至如归的感觉，它拥有可定制的多栏布局和强大的通知功能。

就像 TweetDeck 一样，Tweeten 的速度非常快，无论是刷新提要还是滚动提要。TweetDeck 的键盘快捷键也可以使用，你可以使用 GIF 搜索、预定推文和活动跟踪等功能。Tweeten 是一款适用于 Mac 和 Windows 的应用程序，此外，还有一个 Chrome 扩展。

## 把 TweetDeck 的网站变成一个应用程序

TweetDeck 还没有完全消亡——Twitter 仍将运营网站。我们知道，我们知道；该网站的速度和功能远不如这款应用。另外，它会在你所有的浏览器标签中丢失。

如果你不喜欢在浏览器中使用该网站，你可以尝试将 TweetDeck 变成它自己的应用程序——有多种方法可以做到这一点。像 Chrome 和 Edge 这样的浏览器都支持这个功能(从三点**菜单**按钮，进入**更多** > **更多工具** > **创建快捷方式**)。也可以使用类似 [流体](https://fluidapp.com/) 或 [Unite](https://www.bzgapps.com/buyunite) 的独立应用。

## [Tweetbot](https://tapbots.com/tweetbot/mac/)

tweet bot(9.99 美元)是 Mac 最好的 Twitter 客户端之一。它设计得很好，很快，可定制，*和*它有像 TweetDeck 一样的多栏布局。将鼠标移到右下角，你会看到一个蓝色的浮动图标。向右拖动按钮，您就有了一个可以自定义的新列。
