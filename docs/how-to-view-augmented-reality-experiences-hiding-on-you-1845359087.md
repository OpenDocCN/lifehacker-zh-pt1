# 如何查看藏在亚马逊盒子里的增强现实体验

> 原文:[https://life hacker . com/how-to-view-augmented-reality-experiences-hiding-on-you-1845359087](https://lifehacker.com/how-to-view-augmented-reality-experiences-hiding-on-you-1845359087)

亚马逊正在改变其包装以使用更少的材料，但它也在新的盒子上添加了古怪的二维码，让你可以使用亚马逊的AR Player应用程序在手机上扫描和玩3D模型。增强现实体验可以在iOS和Android上运行。

Watch

亚马逊的第一个二维码是一个AR南瓜，当你在上面画画时，它会改变外观，就像雕刻一个数字杰克灯。南瓜会出现在从现在到万圣节之间运送的包裹上，所以你在黄金日购买的一些东西可能会出现在一个新的盒子里。

也可以下载、绘制、扫描AR南瓜，无需盒子。

1.  从 [苹果应用商店](https://apps.apple.com/us/app/id1519505870#?platform=iphone) 或 [谷歌Play商店](https://play.google.com/store/apps/details?id=com.Amazon.AmazonAR) 下载亚马逊AR播放器应用。
2.  **如果你没有盒子可以扫描:**打开智能手机上的 [这个链接](https://amazonar.app/halloween) 。向下滚动到**“没拿到箱子？”**并点击**“遵循这些步骤”**链接。
3.  **如果你在一个实体盒子上使用二维码:**用黑色记号笔在空白南瓜上画一张脸。准备好之后，扫描AR播放器中的二维码。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-UTnjUxQNZsc&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-UTnjUxQNZsc&start=0) 

扫描后，3D南瓜会出现在ar玩家的相机模式中。然后，你可以用不同的颜色选项和配件定制“雕刻”南瓜，如蝙蝠翅膀、帽子、眼镜等。该模型通常只是漂浮在取景器中，但新的iPhones可以使用3D南瓜作为自拍过滤器，跟踪你的头部运动。在应用程序中拍摄的照片可以与你的联系人分享或发布在社交媒体上(哦，别担心——亚马逊不会访问或存储AR播放器的数据)。

南瓜灯是亚马逊的第一个AR实验，但很可能不是最后一个。亚马逊的宣传材料显示了该应用程序如何将一个盒子变成一辆小型AR汽车，变出了一只数字柯基犬，等等。

当你用完这些盒子时，记得回收或重新利用它们成为其他东西——比如你孩子的堡垒，紧急引火物，甚至是在家 [种植平菇的基质](https://www.mushroom-appreciation.com/mushroom-spawn-cardboard.html) (这就是我用它们的原因)。
