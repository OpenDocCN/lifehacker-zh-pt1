# 如何送一份不破坏惊喜的节日礼物

> 原文：<https://lifehacker.com/how-to-send-a-holiday-gift-without-spoiling-the-surpris-1845888942>

我家的包裹一到就被打开了，所以这个假期把节日礼物和标准包裹分开是很重要的。圣诞节的一半乐趣是在圣诞节早上打开礼物，你不希望有人打开你给他们的全新平板电脑，让他们以为这是他们订购的除臭剂。不用担心，你可以用这个小小的黑客工具来帮助你的礼物在圣诞节打开合适的包裹。



### 订购该商品作为礼物

这听起来像是一个显而易见的答案，但它值得一说:一旦你选择了你的在线商品，去你的购物车，寻找一个选项来选择“这是一个礼物。”应该会弹出一个表单，允许您指明礼物是给谁的，并包含一条消息。礼物收据(和便条)将向接收者表明这确实是一份礼物，这样他们就可以避免他们的惊喜被破坏。(他们可以保留隐藏了你支付的价格的礼物收据，并允许他们退货。)

### 在地址栏写下你的信息

为了防止丢失礼物收据，享受圣诞惊喜，在地址行写上“礼物送给…”。这是为您的假日套餐提供的额外预防措施，您仍应在退房前选择“这是一份礼物”。当礼物到达时,“礼物送给…”会印在盒子外面的运输标签上，这样收件人就可以很容易地认出这个包裹是礼物，并把它扔在树下，留到圣诞节那天。在地址行中加上“来自圣诞老人”或其他有趣的字符，不仅仅是给孩子——来自圣诞老人的礼物会让我成为一个成年人，这是一种很好的节日快乐。

### 确保你的礼物是可邮寄的

根据商家和地点的不同，有些商品不能作为礼物发送——如果你是通过亚马逊全球商店这样的方式进行国际运输，请注意。 [正如亚马逊注明的](https://www.amazon.com/gp/help/customer/display.html?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-send-a-holiday-gift-without-spoiling-the-surpris-1845888942&asc_source=&nodeId=201936750&tag=kinjalifehackerlink-20) ，“这些选项不适用于亚马逊全球商店。如果这些选项对其他项目不可用，您将看到一条提示，上面写着“我们很抱歉”。这件物品不能包装成礼品，也不能附便条寄出。这也适用于他们的 [礼品包装选项](https://www.amazon.com/gp/help/customer/display.html?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-send-a-holiday-gift-without-spoiling-the-surpris-1845888942&asc_source=&nodeId=201937150&tag=kinjalifehackerlink-20) 。有些物品由于尺寸或形状的原因无法包装，所以如果你想走礼品包装的路线，请记住这一点。如果你在和一个较小的夫妻店交谈，询问他们对邮件安全包装和标签的选择——他们中的许多人会额外多做一点，这是购物小的许多好处之一。
