# 用两种乔氏产品做一份时令汤料

> 原文：<https://lifehacker.com/make-a-filling-seasonal-soup-with-two-trader-joes-produ-1847964471>

Trader Joe's 非常擅长两件事:在他们的冷冻食品区储备永无止境的新奇有趣的方便食品，并在每个假期推出新的——有些人可能会说是不必要的——感恩节- 和圣诞节- 主题商品。

但是他们最普通的冷冻食品之一，冷冻有机茉莉香米，最近赢得了我的心。米饭应该放在微波炉安全的袋子里蒸，但我通常会撕开袋子，直接把米饭加入煨肉汤、汤或炖肉中，让它变得蓬松，让它更加饱满。这可能不是往我的汤里加米饭的最经济的方法，但是，作为一个一个人吃很多顿饭的人，这是最方便的。(如果我不需要一整包米饭，我会把剩饭放在冷冻袋里，然后再放回冰箱。)

然而，茉莉香米并不是唯一可以做到这一点的冷冻米饭。他们的任何冷冻米饭，包括调味的“炒饭”等，都可以加入到他们的汤里，制成一顿几乎即时的饱餐。他们的 Chimichurri 米饭就是这样(我已经有一段时间找不到这种米饭了)，他们最新的季节性米饭也是如此:火鸡和香肠馅的油炸 R 冰。你所要做的就是拿一杯冷冻米饭混合物，把它和一杯半你最喜欢的肉汤或汤(鸡汤、火鸡肉汤、和番茄&烤红辣椒汤在这里效果很好)一起放入调味汁锅中。将它煮沸，转小火慢炖，煮到小香肠完全变热，不断搅拌以防止米饭粘在调味汁锅的底部。

这种组合美味可口，令人满意，有小火鸡肉丸、胡萝卜、切成薄片的青豆和玉米。所有常用的馅料口味都在这里发挥作用(迷迭香、百里香、欧芹等)。).如果你喜欢去年的所有东西，除了剩菜混合调料，你会喜欢这个炒饭，然后，这个汤。

一旦你结合并享受了这种方便的搭配，你可能会开始注意到其他预先包装的冷冻米饭、谷物、饺子、和面食混合物，它们可以在热汤中煨成令人满意的一餐。当搭配秋收汤时，红薯汤团是非常好的，任何 gyoza 在他们的味噌姜汤中都很好。只是不要依赖任何一个特定的组合。乔氏也很擅长处理掉所有我喜欢的物品。(我会从停止销售草莓先生中恢复过来吗？我高度怀疑。)
