# 这是幼儿最好的视觉共同育儿日历

> 原文：<https://lifehacker.com/this-is-the-best-visual-co-parenting-calendar-for-young-1846771037>

[过渡是大多数幼儿家长的痛点](https://offspring.lifehacker.com/how-to-avoid-transition-based-tantrums-1846363670) 。从一项活动到另一项活动，即使是最微小的动作也会让一个蹒跚学步的孩子因为不知道接下来会发生什么而焦虑不安。所以只有才说得通，小孩子可能经历的最大转变之一——从妈妈家到爸爸家，然后再回来——会给他们带来一些压力。这就是为什么一位家长制作了一个视觉日历，清楚地向他们的女儿说明她什么时候来，什么时候走。

Watch

[金赛·莫朗](https://www.tiktok.com/@skywalkeraa16_18?) 贴在抖音上的日历，采用了数字和圆点的颜色编码系统，就像这样:

 [https://lifehacker.com/embed/inset/iframe?id=tiktok-6945878124437572869&autosize=1](https://lifehacker.com/embed/inset/iframe?id=tiktok-6945878124437572869&autosize=1) 

她和她爸爸在一起的日子用黑色记号笔写着；她和妈妈在一起的日子是蓝色的。他们还在日期框的左上角或右上角包含红点，以指示孩子何时回家或离开家。

正如 [早安美国](https://www.goodmorningamerica.com/amp/family/story/moms-clever-parenting-schedule-love-parents-tiktok-77323996) 报道:

> 由于 Dylann 是一个视觉学习者，Morlang 想创造一些东西来简化每周日程的工作方式。

> “我发现当她问‘我什么时候能再见到爸爸？’我会说，“两天后。”或者“有一天”，她只是没有抓住它，”Chino 的居民 Morlang 告诉“早安美国”。”“它在我脑袋里种下了一颗种子，我说，‘好吧，我要去彩色编码了。"

一天结束后，莫朗会擦掉当天的数字和圆点，这样他们的女儿就能更容易地看到不久的将来会发生什么。如果他们的女儿想知道离她再次回到爸爸身边还有多少天，她可以简单地数出方块，直到她看到下一个黑色数字。

对于学龄前儿童来说，这个日历系统是一个特别好的解决方案，他们已经大到可以自己数日子了，但是还太小，不能自己记住时间表或者一周中的每一天。
