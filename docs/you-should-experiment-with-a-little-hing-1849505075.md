# 你应该尝试一些小东西

> 原文：<https://lifehacker.com/you-should-experiment-with-a-little-hing-1849505075>

说到令人垂涎三尺的饭菜，有时候最辛辣的食材才是最本质的。如果没有发酵鱼酱，亚洲美食会是什么样子？或者没有各种酸菜的德国或者韩国料理？而我们都知道 [芫荽如何能分科](https://lifehacker.com/you-should-eat-cilantro-roots-1849491217) 。单独来看，这些配料可能会让不习惯的人转身离开，但是当与成品菜肴中的补充配料搭配时，即使是最挑剔的食客也不会感觉到它们的存在，只会感觉到味道的和谐。兴也不例外，一旦你熟悉使用这种强大的粉末，你就能制作出自己不可抗拒的味道组合。

Hing(也叫 [【阿魏】](https://en.wikipedia.org/wiki/Asafoetida) )是一种原产于阿富汗和伊朗的山 [阿魏](https://en.wikipedia.org/wiki/Ferula) 的根的干燥和磨碎的汁液。虽然你可能会忘记这一切，但你永远不会忘记那股强烈的气味。它经常在印度菜肴中以极小的剂量使用，但极小并不意味着不重要。这是我尝过的最独特的味道之一，完全是独一无二的。对我来说，很难准确地描述或举出与新来者相关的口味，但我会试一试。大蒜和洋葱的味道和气味就好像是披着硫磺斗篷的超级英雄，他们的超能力正在把他们芳香的灵魂添加到你的食物中。那有帮助，对吧？其他人用完全负面的术语来描述它，但是我有一种潜在的怀疑，他们在“嘲笑别人的美味”闻起来是什么味道:干燥、粉碎的植物树脂。我承认它确实很强大，而且有点古怪。但是对于那些不能吃真正的大蒜或洋葱的人来说，荤腥是这些口味的可靠替代品。最重要的是，当添加到包含六种以上不同香料的菜肴中时，它可以统一一切，并添加那种*我说不出的味道*。

因为 hing 很结实，而且纯的东西有粘性(毫无疑问这是树脂的一个特点)，它通常会在出售时加入大米或小麦淀粉，以保持 T2 的松散和干燥。当你购买一个罐子时，确保你知道它是否包含任何其他淀粉，像 L.G Hing 的 [这个](https://www.walmart.com/ip/LG-Hing-Compounded-Asafoetida-3-5-oz/741214287?wmlspartner=wlpa&selectedSellerId=101122055) ，或者如果它更纯，像这个来自 [Best Hing 的](https://store.177milkstreet.com/products/pure-indian-food-best-hing-ever?currency=USD&variant=39302088785977&utm_medium=cpc&utm_source=google&utm_campaign=Google%20Shopping&utm_source=google&utm_medium=cpc&utm_campaign=17132908520-jump_lp:store_ch:performancemax_mb:bill_ob:mix_tt:shopping_ca:store-multipleproducts_ge:seasonings&spices_c1:_c2:_c3:&utm_content=&utm_term=gadid=https://store.177milkstreet.com/products/pure-indian-food-best-hing-ever%3Fcurrency%3DUSD%26variant%3D39302088785977%26utm_medium%3Dcpc%26utm_source%3Dgoogle%26utm_campaign%3DGoogle%2520Shopping?utm_source=google&utm_medium=paid&utm_campaign=17132908520&utm_content=&utm_term=&gadid=&gclid=Cj0KCQjw39uYBhCLARIsAD_SzMTsERRj2eVZ9O7Y555o-xjbF-3qsYnHRMV9suNW84WYqw7MS66ubVcaAswEEALw_wcB) 。仔细阅读配料表来帮助你决定在你的食物中添加多少。我有过最好的东西，我第一次用它的时候，仅仅是⅛茶匙，它就压倒了我盘子里的其他东西。吸取教训。现在，我只有一点钱。

在鲜味包装的菜肴中，Hing 与豆子以及任何你可以看到自己添加洋葱或大蒜的地方搭配得非常好。如果你想从屡试不爽的食谱开始，看看 [马苏尔·达尔](https://www.indianhealthyrecipes.com/masoor-dal-recipe/) ，或者 [拉杰马·达尔](https://www.thespruceeats.com/rajma-red-kidney-bean-curry-1957982) 。只需撒一点点香料，就能带出扁豆的麝香、泥土的味道，以及温热香料的醇香。如果你已经处于实验阶段，请记住洋葱和大蒜的味道，并从那里开始探索。当我想到一碗丰盛的、令人舒适的木豆时，我的思绪会飘向炖菜、辣椒或肉砂锅。这些可以用一点葱头来突出它们的其他美味。

请注意，当在烹饪过程中有机会开花时，它与菜肴的融合最好。这通常有两种方式:在热油中与其他香料一起烘烤，或者撒在切碎的西红柿等清淡的配料上，或者在炖菜加水后。如果你用的是更纯的东西，一定要洒在周围。如果你把它们都倒成一堆，热油的高温可能会导致树脂颗粒再次粘在一起，，它会变成一个超级小精灵，而不是分布在你的盘子里。就像醋之于黄瓜，烟之于三文鱼——用辛做一点实验，你会发现这种辛辣的成分正是你让你的食物开花所需要的。
