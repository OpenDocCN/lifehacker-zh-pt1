# 什么是“睡眠离婚”,什么时候应该考虑离婚？

> 原文：<https://lifehacker.com/what-is-a-sleep-divorce-and-when-should-you-consider-on-1846758141>

当你真正停下来想一想的时候，为了睡觉而和另一个人同床共枕是一个很糟糕的想法。当然，你可能会习惯睡在伴侣旁边，这样会感觉更安全，但是就睡眠质量和干扰而言，很难打败拥有自己的床。

每个人都长期疲惫不堪，你可能会认为我们会尽一切努力获得最好的睡眠。然而，情侣睡在同一张床上仍然被认为是一种规范——以至于不与浪漫的伴侣同床共枕会带来一些耻辱。但不一定非要这样。问问那些选择“睡眠离婚”的夫妇就知道了。这是什么，什么时候你可以考虑一个。

### 什么是睡眠离婚？

简而言之，睡眠离婚是指通常同床共枕的人意识到，他们可能会更适合单独睡觉。这可能意味着*我爱露西*风格的双人床在同一个房间，独立的卧室，或其他完全不同的东西。

这个想法是采取措施来确保每个人都能得到最好的睡眠。如果现在还不清楚的话，睡眠离婚并不是你真实关系的反映。事实上，它 [可以在许多方面帮助改善你们的关系](https://www.nytimes.com/2019/07/31/fashion/weddings/is-it-time-for-a-sleep-divorce.html) ，包括在白天能够对彼此有更多的耐心，因为你休息得很好。

### 何时考虑睡眠离婚

如果睡眠中断或其他问题让你们的关系变得更糟，你和你的伴侣可能要讨论睡眠离婚。《纽约时报》*2019 年的一篇 [文章深入探讨了睡眠离婚的概念，提出了一些为什么它可能是一些夫妇的可靠选择的原因。其中包括:](https://www.nytimes.com/2019/07/31/fashion/weddings/is-it-time-for-a-sleep-divorce.html)*

*   *按照不同的时间表睡觉*
*   *打鼾和其他破坏性呼吸*
*   *不再有能力舒适地同床共枕*
*   *有不同的睡眠(和个人)卫生标准*
*   *对你的伴侣怀恨在心，因为你没睡好是他们的错*
*   *为睡觉和/或同床而争吵*
*   *有不同的睡眠偏好，比如室温、毯子的数量、是否开着电视睡觉等等。*

*此外，不要忘记，当谈到同床共枕这样的话题时，人们并不总是开诚布公的。也许你不满意你的睡眠安排，但认为你的伴侣喜欢它，所以你什么也没说——然后事实证明你的伴侣也在寻求改变。一次简短的谈话可能会产生一个对每个人都更好的局面。*
