# 带孩子旅行时，如何更好地打包行李

> 原文：<https://lifehacker.com/how-to-pack-better-when-traveling-with-kids-1849169551>

为什么每次你离开家，孩子们似乎比成年人需要更多的东西——尤其是在旅行的时候？虽然你可以轻松地把你的个人物品装进一个随身携带的 ，但你需要多个超大行李箱来装下你孩子所有的衣服、玩具和装备。

当然，当你带着孩子旅行时，有些必需品是你绝对不能留在家里的，但是 [可以削减开支，用更少的](https://www.scarymommy.com/parenting/how-to-travel-with-kids-not-overpack) 来凑合。这里有一些为下次家庭度假打包的策略。

### 如何在旅行时少带些儿童用品

当你为你的孩子打包行李时，考虑一下你如何为自己打包更轻的行李。问问你真正需要的是什么，你能留下什么，你带的每样东西如何能起到双重作用。

**少选(多用)档位。**带上你习惯的所有装备，尤其是婴儿和幼儿的装备，可能会让你的旅行更加舒适，但是*需要*它吗？一辆婴儿车在机场可能不错，但一个手提袋或背包也可以。也许一张毯子可以有多种用途。如果你一定要带所有的东西，考虑投资那些可以折叠、放气或以其他方式紧紧打包的轻便物品。

在目的地租赁或借用装备。大多数酒店根据客人的要求提供婴儿床，你可以在 Airbnb 上搜索也有婴儿床的酒店(并询问你的主人他们家还有什么适合孩子的，比如玩具和加高座椅)。家庭友好型餐厅通常有高脚椅。当拜访有孩子的家庭成员时，你们可以分享一些物品。

使用拼车服务时，你可以选择要求一个汽车座椅，这样就不需要带了。Lyft 和 [优步](https://www.uber.com/us/en/ride/car-seat/) 都提供汽车座位，每次收费 10 美元。租车公司 [通常也会提供汽车座椅](https://blog.autoslash.com/child-safety-seats-and-how-to-avoid-excessive-fees-on-car-rental/)——如果加上每天的附加费，这些会很贵，但是你可以从一些 AAA 会员的公司 [免费得到。](https://lifehacker.com/psa-you-can-borrow-a-free-child-car-seat-from-aaa-1765244524)

如果你需要你不想带的笨重装备，试试婴儿装备租赁服务。这些公司会将婴儿车、高脚椅、汽车座椅、婴儿床等送到你的酒店或出租屋。

到达后购买标准物品。你不需要打包足够的尿布来覆盖整个旅程。保证。同样的道理也适用于任何你可以在杂货店或药店轻易买到的东西，比如额外的零食和化妆品。另外，出租屋和旅馆通常都有洗发水之类的标准用品。除非你的孩子有特殊需要，或者你要去一个偏远的地方，你可以在目的地重新进货来节省空间。

不要(一定)为所有的突发事件做计划。如果恶劣天气的可能性极小，你可能不必为每个家庭成员都准备好全套雨具。如果你绝对需要的话，可以买雨披或者额外的几层。

提前准备好行头。一点点的计划对你带来你需要的东西大有帮助。一天一天地摆放衣服——甚至可以单独打包并贴上标签——这样你就不会有一堆乱七八糟的东西或者只有 t 恤没有裤子。将鞋子限制在大多数活动中必不可少的范围内。考虑更深的颜色和图案，不容易显示污垢和污渍，以防你需要重新穿的东西。

根据你孩子的年龄、你的计划和你使用洗衣设施的情况，多带一些足够的衣物。

选择带洗衣机/烘干机的房间。如果你要在某个地方呆很长时间，找一家 Airbnb 或有洗衣机和烘干机的租赁房屋。你可以给每个人带更少的衣服，你也不需要一整个箱子的脏衣服带回家。

给孩子一个空间限制。如果你的孩子已经大到可以自己收拾行李了——或者至少可以帮忙——给他们一定的空间。一旦他们的背包或打包箱装满，他们就不能携带任何额外的个人物品。当然，检查他们收集的东西可能是个好主意，这样就不会全是玩具和内衣了。类似地，如果你为他们打包，试着将每个人限制在一个单独的包装立方体内。

## 带着孩子旅行时如何更聪明地打包行李

所以你已经删除了所有无关的东西——现在是打包的时候了。

将所有东西打包并贴上标签。我们已经提到了整理服装和限制每个孩子的分配空间。拿一个包装立方体或大的 Ziploc，在上面贴上你孩子的名字。当你到达时，所有的立方体或袋子会被装进一个大的家庭手提箱或他们各自的背包里，并被放入各自的抽屉或房间角落里。

带额外的塑料袋或塑料袋。多扔几个拉链袋，用来装旅游纪念品或其他物品。这使得一切都包含在内，井井有条。此外，带上一把超市塑料袋，用来收集脏东西，并把它们与你的其他物品分开。

打包一个共用的洗漱用品和急救包。同样，使用拉链包:在一个地方收集基本的急救物品，所以如果需要创可贴、Neosporin 或 OTC 药物，你可以拿一个包，而不是翻遍每个洗漱包。

带一个弹出式篮子或洗衣篮。如果你是自驾游，并且有足够的空间，考虑带一个弹出式篮子，你可以在度假时用来收集脏衣服，这样你就可以洗衣服或者在最后把所有东西都拖回家，而不是重新打包进行李箱。篮子或洗衣篮也可以用来存放枕头、毛巾或其他在单独的袋子里没有位置的共享物品。
