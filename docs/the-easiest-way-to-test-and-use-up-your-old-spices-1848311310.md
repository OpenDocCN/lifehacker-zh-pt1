# 测试(和用完)你的旧香料的最简单的方法

> 原文:[https://life hacker . com/the-easy-way to-test-and-use-up your-old-spices-1848311310](https://lifehacker.com/the-easiest-way-to-test-and-use-up-your-old-spices-1848311310)

一月通常是清理、巩固和清洁的月份，但是这个一月似乎特别适合它。奥米克隆几乎感染了我认识的每一个人，而且天气很糟糕——没有什么理由离开房子；还不如打扫干净。

你冰冷的冰箱 是一个很好的开始，你的调料架、橱柜或抽屉也是。减少你的香料可能是一个挑战，特别是如果你有一小部分一茶匙大小的量在玻璃罐里。

快速闻一下你的香料会告诉你一些有用的信息。如果香料有很浓的香气，那就好！像平常一样使用它。如果闻起来什么都没有？扔过来。没用的。(除非是姜黄。可以用无味姜黄 [染面料](https://lifehacker.com/use-old-turmeric-to-make-a-brilliant-yellow-fabric-dye-1846336290) 。)

但是如果你的香料介于两者之间呢？如果它只运行了一半容量呢？这种香料仍然有一些味道，最好用在一大碗爆米花中。

效力不确定的香料在食谱中不太管用。你可以尝试通过加倍(或三倍)来补偿褪色的味道，但不能保证这样做会产生正确的味道。

爆米花要宽容得多。这是一个完美的空白画布，没有任何强烈的竞争味道来掩盖仍然存在的味道。(另外，它非常便宜，所以如果你的香料实验失败了，也不是完全失败。)把它撒上去，直到你能尝到它的味道，或者把一大堆不新鲜的粉末混合在一起。(为了获得最佳效果:在食品加工机中加入几茶匙盐，把所有的东西都碾碎，这样它们就会粘在果仁上。)

爆米花也是全新香料的完美载体，尤其是那些你可能不熟悉的。你圣诞节收到香料套装了吗？在尝试其他菜肴之前，在爆米花上尝试粉末，以掌握它们的味道。(这让你更好地理解香料本身的味道，但这也是晚餐吃爆米花的一个很好的借口。)
