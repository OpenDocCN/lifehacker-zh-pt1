# 你会想一整年都做这道辛辣的蔓越莓芥末酱

> 原文:[https://life hacker . com/youll-want-to-make this-spiral-cranberry-芥末-all-yea-1845620398](https://lifehacker.com/youll-want-to-make-this-spicy-cranberry-mustard-all-yea-1845620398)

我一直觉得蔓越莓酱 [比感恩节](https://skillet.lifehacker.com/we-should-eat-cranberry-sauce-all-year-1840055543) 大。虽然它在火鸡日起着重要的作用——确保我们的味觉不会被盐、脂肪和鼠尾草饱和——但这种香甜扑鼻的调味品可以用来平衡全年丰富可口的食物。当搭配纯芥末粉时，口感尤其干净。

正如我们之前在 中了解到的，制作超辣芥末最简单的方法是将干芥末粉与少量水混合，等待10分钟。就是这样——水分激活了种子中的酶，让它们释放出带来痛苦的刺激性化合物。你可以让芥末保持原样，或者你可以 [把它和一些甜的东西](https://lifehacker.com/make-a-hot-and-sweet-mustard-with-three-ingredients-1845050707) 混合在一起，创造出一种可以完美搭配咸肉的调味品。

“甜的东西”由你决定。可以用蜂蜜。你可以用枫糖浆。或者，你可以在假日季节吃一点蔓越莓酱。辛辣的蔓越莓芥末酱绝对会出现在你的剩菜三明治中，是的，但它同样会出现在火腿和奶酪中，出现在软椒盐卷饼中，或者作为热的小奶酪配菜。这是一种令人痛苦又美味的甜味、扑鼻味和鼻窦闭塞的混合物，而且很容易制作。热蔓越莓芥末在新鲜的时候是最辣的，所以根据你的需要做一点，以获得最大的辣味。

### 又辣又甜的蔓越莓芥末酱

**配料:**

*   1份芥末粉(我喜欢 [S & B](https://www.sbfoods-worldwide.com/products/search/031.html) )
*   1份水
*   3份蔓越莓酱(果冻是最光滑的芥末，但整个浆果可以提供一个很好的对比)

**指令:**

在一个小碗中混合水和芥末粉，搅拌成糊状。盖上盖子，放置10分钟，让那些刺鼻的化合物醒过来。加入蔓越莓酱，搅拌并用叉子捣碎，制成光滑的芥末酱。(远离你的脸，因为芥子气有点痛！)尝一尝，如果需要的话，加入更多的蔓越莓(或一点蜂蜜或枫树)。涂抹在火鸡三明治、火腿三明治、奶酪三明治或任何其他富含盐分的三明治上。
