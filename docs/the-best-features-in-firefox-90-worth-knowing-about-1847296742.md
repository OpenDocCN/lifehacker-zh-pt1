# 火狐 90 最值得了解的特性

> 原文：<https://lifehacker.com/the-best-features-in-firefox-90-worth-knowing-about-1847296742>

火狐 89 的 [设计大修后，90 版本更新比较窝囊。但它专注于两件重要的事情——隐私和追踪器屏蔽。如果你是火狐的粉丝，并且喜欢它保护你的隐私和自动屏蔽追踪器的一贯使命，你会喜欢这次更新的。](https://lifehacker.com/the-biggest-changes-in-firefox-89-1847028614) 



要更新到最新版本 sion，o 打开火狐应用，进入**设置>常规>火狐更新>检查更新**。火狐也在为安卓系统推出 v 版本 90 更新，你可以从的 [谷歌 Play 商店](https://play.google.com/store/apps/details?id=org.mozilla.firefox&hl=en_IN&gl=US) 获得。以下是火狐 90 的新功能。

## SmartBlock 2.0 适用于您的脸书登录

Firefox 90 更新改进了 SmartBlock，这是一种针对隐私浏览和严格模式的跟踪器阻止功能。您现在可以使用脸书以私人模式登录网站，同时仍然可以阻止跨网站跟踪。如果你使用的是以前版本的 Firefox，浏览器会阻止使用脸书登录的请求，因为它会通过跟踪服务器。

现在，当你尝试使用脸书登录时，Firefox 会识别出这一点，并暂时允许它(仅在登录过程中)。一旦完成，跨站点跟踪将再次被阻止。

## Windows 的后台更新

此次更新还为 Windows 电脑引入了一个后台更新功能，即将在后台检查、下载和安装任何未完成的更新(它每七个小时检查一次)。

Firefox 正在逐步推出这一功能，你可以从设置中启用它。

1.点击**菜单**按钮，进入**设置**。

2.在**通用**部分，向下滚动到 **火狐更新**。

3.在这里，启用**自动安装更新(推荐)**选项，当火狐没有运行、时，根据需要勾选或取消勾选、。

## 添加和同步信用卡

Firefox 现在还允许你添加信用卡并同步到你的 Firefox 账户(在桌面和 Android 应用上)。此信息在您的帐户中加密，并将在您的设备上提供。

## Android 版 Firefox 的变化

在安卓系统方面，火狐已经在设置上调整了界面。你现在可以在顶部看到一个大横幅，要求你将 Firefox 设置为默认浏览器，还有一个用于添加信用卡的新区域。

Firefox 90 还为支持的网站添加了后退/前进缓存功能，所以你可能会发现在网页之间来回切换更快了。

## 技术变化

这个更新完全删除了 FTP: 以前，你可以使用一个标志来启用文件传输协议，但这是不可能的了。这意味着你需要使用一个专用的浏览器，比如 [FileZilla](https://filezilla-project.org/) 来访问 FTP 服务器。

您还应该注意到，有一个新的“ [关于:第三方](https://support.mozilla.org/en-US/kb/identify-problems-third-party-modules-firefox-windows) ”页面来帮助您管理与第三方应用程序的兼容性问题。 [仅 HTTPS 模式的例外](https://support.mozilla.org/kb/https-only-prefs) 可在“关于:首选项#隐私”中管理
