# 垃圾冰咖啡指南

> 原文：<https://jezebel.com/the-dirtbags-guide-to-iced-coffee-1847350675>

抖音有大量的 java 爱好者制作漂亮而精致的拿铁，充满了主要用于审美的装饰。如果我们相信抖音是福音，似乎每个需要或渴望咖啡因的人现在都是自学成才的咖啡师。想想这个玫瑰牛奶拿铁，它的特色是心形的、有玫瑰香味的燕麦牛奶冰块；牛奶起泡剂；还有一台 Nespresso 机。结果是美丽的，但对于一杯可能尝起来令人作呕的早餐来说，这实在是太小题大做了。虽然我有一部分希望我有精力用同样的崇敬来对待我的早餐，但我了解我自己。方便胜过美观，每次都是如此。如果可以的话，我会一直选择垃圾选项，我这样做的决定通常是正确的。



如果世界上一切都好，并且我一直积极主动地去杂货店购物和完成周日的小任务，那么到了周一上午 9 点，我冰箱里的冰镇啤酒罐就满了，可以喝了。总的来说，这是一个很容易完成的任务，并确保我早上有咖啡喝——这对每个人都是一个胜利。然而，有时生活会阻碍我们。最近，我发现自己陷入了一个困境,我的水壶里没有咖啡，也没有时间或欲望跑到价格昂贵的咖啡馆去买通常是我周末享受的东西，一杯冰咖啡和一份丹麦点心。幸运的是，我的早期版本已经为这种精确的情况做好了准备。

早期隔离的 [达尔戈纳咖啡微趋势](https://jezebel.com/embracing-the-distraction-of-tiktok-coffee-1842664897) 给我留下了一罐速溶咖啡，它已经在我的茶水间坐了一年多，积满了灰尘。互联网向我保证，我可以用布满灰尘的晶体制作冰咖啡，这是我以前一直避免的——不是出于对咖啡的势利，而是因为我根本不知道我能做我将要做的事情。两到三茶匙的咖啡粉倒入一个玻璃杯的底部，用水龙头流出的热水注满。产生的污泥闻起来很好，类似于浓缩咖啡；我在杯子里装满了冰块和冷水，然后通过摇晃瓶子“起泡”了一些奶精。用金属吸管快速搅拌，结果真的比我通常做的冷泡茶好，因为它尝起来很浓，不像咖啡水。消费后 10 到 15 分钟，我的眼睛从我的头上振动出来，这种方式并不完全令人不快——这明显偏离了我的正常习惯。感受新事物是多么美好。

当我把这个方法分享给一个朋友时，他摇摇头告诉我，是的，当然，速溶咖啡和冰咖啡一样管用。“我只是把它和冷水混合，”他说。“直接从水龙头。每次都奏效。”
