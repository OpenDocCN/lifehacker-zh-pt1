# 这些 Mac 提示让日常任务变得更快

> 原文：<https://lifehacker.com/these-mac-tips-make-everyday-tasks-faster-1849386855>

电脑使做任何事情都更快(这是关键)。但是，即使你的 Mac 已经可以快速处理你扔给它的任何东西，也有办法计算得更快。无论是为了工作、娱乐，还是介于两者之间的事情，这些建议可以让你在 Mac 上更快地完成几乎任何任务。



## 悬停时更快地打开文件夹

当你想把一个文件放入一个文件夹时，你通常用光标拿起它，把它拖到文件夹上，然后等待文件夹弹开。这很好，但是感觉花费的时间比应该花费的时间要长一些。事实证明，你可以大大加快这个过程。

直到偶然发现 [这个 Reddit 线程](https://www.reddit.com/r/MacOS/comments/wg6p2t/is_there_a_way_to_increase_the_speed_of_this/?utm_source=share&utm_medium=web2x&context=3) 我才知道这一招。简而言之，在**系统偏好设置>辅助功能>指针控制**中使用 b ，你会发现当你将文件悬停在文件夹上时，可以改变文件夹弹开的速度。将“弹簧加载延迟”旁边的滑块移动到“短”，你几乎不用等待。

## 搜索是你最好的朋友

Spotlight search 可能是你的 Mac 上最有用的功能。有了它，你可以搜索电脑上的任何文件，打开应用程序，查看来自互联网的结果，做简单的数学问题，查看字典定义，等等。我几乎不再使用 Finder，因为 Spotlight 非常方便。

Spotlight 如此有用的部分原因是它的键盘快捷键: **Command + Space** 。在任何时候，你都可以使用这个快捷方式来访问任何你需要的东西。这使得它成为你的 Mac 工作流程中必不可少的一部分(至少对我来说是这样)。

## 实际使用 Siri(结果可能有所不同)

Siri 可能不是最可靠的智能助手，但当它工作时会很有用。在我的 Mac 上，我倾向于用它来运行 macOS 应用程序和任务，类似于我在 iPhone 上使用的那些程序和任务，比如设置提醒、查看天气、打开应用程序或文件。如果键盘上有麦克风图标，您可以随时长按它来激活 Siri。如果没有，你的 Mac 很可能默认为一个**命令+空格**长按调出 Siri。

当然，正如你可能从你的 iPhone 中知道的那样，Siri 并不完美。虽然让 Siri 设置提醒通常比自己完成动作更快，但如果它搞砸了，肯定会花更长时间。有一些耐心，为一些失误做好准备，Siri 可以成为你日常 Mac 任务的一个值得的工具。

## 快捷方式不仅仅适用于你的 iPhone 和 iPad

iOS 和 iPadOS 上的 Shortcuts 应用程序使您的移动设备能够自动完成任意数量的任务。然而，那些相同的快捷方式也可以在你的电脑上使用， [通过 macOS 快捷方式 app](https://lifehacker.com/you-really-need-to-set-up-shortcuts-on-your-mac-1848064831) 。

你可以使用快捷方式完成一个 *lot* ，包括下载 YouTube 视频，自动启用低功耗模式，访问常用应用时自动进入分屏。 [探索我们的其他用例列表以获得一些最初的灵感](https://lifehacker.com/9-of-the-most-helpful-shortcuts-to-automate-tasks-on-yo-1847982949/) ，或者浏览快捷方式库以查看其他用户发现哪些是有效的。

## 下载剪贴板管理器

如果你还没有在你的 Mac 上使用剪贴板管理器，让你自己到最近的下载链接。对于大多数 Mac 用户来说，复制粘贴一次只能完成一项任务:你复制一些东西，然后粘贴到其他地方。如果你想粘贴你以前复制的东西，你需要再次复制它。

然而，剪贴板管理器大大地解放了你:管理器保存了你复制的项目的列表，使得任何和所有的项目在你需要的时候都容易被访问。我一直依赖这个特性。它让我可以从一个页面复制多个项目，然后粘贴到另一个页面，而不需要不断返回到原始来源。我还可以快速粘贴我几小时前或几天前复制的内容，而不需要翻遍搜索历史或其他应用程序。

macOS 上有大量的第三方剪贴板管理器供你选择， [我们在这里](https://lifehacker.com/how-to-copy-more-than-one-thing-to-your-clipboard-at-a-1848711373) 列出了几个我们最喜欢的。

## 下载窗口管理器

同样，如果你没有窗口管理器，就去找一个。它们给你的 Mac 带来了窗口捕捉功能——不可否认，这是你在 Windows 操作系统中会发现的功能。也就是说，在窗口管理方面，macOS 比以前更好。如果你将光标悬停在窗口中的绿色按钮上，然后按住 Option，你会看到一个将窗口移动到显示屏左侧或右侧的选项。然而，它远没有专用的 windows 管理实用程序所能带来的功能方便。

例如， [磁铁](https://lifehacker.com/magnet-for-mac-is-the-window-management-tool-ive-desper-1793489282) 可以让你通过键盘快捷键将窗口吸附到显示器的特定部分。您还可以通过将这些窗口拖到特定的角落来将它们对齐。

## 退出和切换应用程序的最佳方式

一个 macOS *必须*是**命令+ Tab** 。此键盘快捷键会弹出一个菜单栏，其中包含所有打开的应用程序的图标:当您按下 tab 键时，光标会移到应用程序图标上。当你徘徊在一个你想切换到，只是放手。你也可以按相反的顺序，点击 **`** 而不是 tab 键——那是 tab 键正上方的键，上面也有波浪号(~)。

你也可以使用这个快捷键立即退出任何打开的应用。按住 tab 直到光标悬停在你想要关闭的应用上，然后，按住 command 键，点击 **Q** 。

## 利用选项键

Option key 是 macOS 最保守的秘密。当您按住它时，您可以访问“隐藏”菜单和功能，而您并不知道这些功能对您可用。例如，在点击左上角的苹果菜单时按住 Option，你就可以选择立即重启或关闭 Mac，而不必先处理弹出窗口。类似地，退出应用程序的菜单选项变为“退出并保留窗口”，这将在下次启动应用程序时保存您打开的窗口。

如果你有兴趣了解所有最好的“选项”快捷方式， [在这里查看我们的指南](https://lifehacker.com/10-smart-option-key-shortcuts-youre-not-using-on-your-m-1847599777) 。

## 微调亮度和音量

如果你觉得 Mac 上的亮度和音量键改变了太多东西，有一个简单的方法来微调变化。在按下有问题的键之前，按住 **Option + Shift** 。这些键会告诉 macOS 将显示器或键盘亮度或音量改变四分之一步，而不是一整步，让您可以快速调整到最佳位置。

## 上网浏览快一点

无论你使用 Chrome 还是 Safari，你大概都知道一些经典的键盘快捷键(比如 **Command + T** 打开新标签页)。然而，有一些不太为人所知的键盘快捷键可以帮助你更快地移动:

*   **Command + Option +箭头键**:在标签间移动。
*   **Command+1–8**:浏览前八个标签。
*   **Command + 9** :移动到最后一个标签页。
*   **Command + L** (仅限 Safari):高亮显示网址栏。

## 跳过垃圾

如果你知道你想要一个文件*消失*，并且不希望处理清空垃圾，h 高亮显示该文件，然后按 **Command + Option + Delete** 将它从你的电脑中立即清除。

## 安装固态硬盘

这条建议可能适用于 2022 年的一小部分 Mac 用户，但它是必不可少的:如果你的 Mac 仍然使用物理硬盘，升级到 SSD。旋转磁盘在速度上无法与固态硬盘竞争，更换它是你能做的最好的事情之一，以 [提高速度和效率，并延长旧笔记本电脑或 iMac](https://lifehacker.com/8-ways-to-make-your-old-laptop-feel-new-again-1847931000/slides/7) 的寿命。

如果你有一台带硬盘的老款 Mac，谷歌一下在你的特定设备上安装固态硬盘的说明。这并不太难，也不太贵，而且它将会给 T2 的生活质量带来巨大的提升。
