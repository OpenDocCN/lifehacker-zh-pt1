# 当你觉得生活中一切都不顺心时，该从哪里开始

> 原文：<https://lifehacker.com/where-to-start-when-it-feels-like-nothing-is-going-righ-1640250197>

糟糕的一天可能会不时发生，但更长的时间可能会变成几周或几个月的感觉 [没有动力、失落、或无望](https://lifehacker.com/how-to-do-the-impossible-task-when-youre-depressed-1828942797) 。生活可能会越来越多，但这里有一些起点，可以帮助你爬出洞穴，回到正轨。



这是一个非常大的话题，所以我们 [向你询问个人经历](http://lifehacker.com/how-did-you-or-someone-you-know-recover-from-a-rut-in-1639099594/all) ，你提供了。有这么多伟大的贡献，所以感谢所有讲述你的故事的人。这些都是来自像你一样的读者的精彩故事和建议——混合了我们自己的建议和策略，以及研究——表明如何有可能摆脱你生活中的任何常规。

### **做点什么**

你感到停滞不前可能是因为...嗯，你是。你醒来，工作，，上床，，然后重复。有时候，当你觉得自己哪里也去不了的时候，你只是需要一些东西来放松一下。不管是什么——你们中的许多人都推荐志愿服务、参加一个课程、或者做一些社区服务——但是如果你尝试一些超出你常规的事情( [尤其是如果你学到了一些东西](https://lifehacker.com/top-10-highly-desired-skills-you-can-teach-yourself-5905835) )，那会有很大的帮助。你可以发现你喜欢的东西，或者认识新的人，给你的日常食谱带来新的味道。

很多人可能会感到不快乐，因为他们的生活中没有让他们兴奋的事情，所以到处寻找快乐是很重要的。有可能你的舒适区正是让你沮丧的原因。你永远不知道事情会把你引向何方。如果你对某样东西感兴趣，就试一试。 [肯雷诺兹](http://lifehacker.com/for-most-of-my-20s-i-hopped-from-dead-end-job-to-dead-1639290305) 正是这么做的:

> 我决定积极主动，而不是被动反应。我一直对编程和软件工程很着迷，但过去总是放弃追求它，因为我说服自己，我没有足够的智力去做这件事。我没有再次放弃，而是注册了 Code A cademy 的账户，学习如何编写 JavaScript……在另一位密友的鼓励下，我申请了 IBM 的软件工程师职位。我得到了这个职位，我比以前学到了更多，成长了更多，并且实现了我最大的梦想之一，这是我以前从未梦想过的。你能做的最好的事情就是做点什么。任何事。一旦你做了一件事，就不要放弃去做。永远不会。

你所做的并不总是很大。可以是 [只是一件小事](http://lifehacker.com/pull-yourself-out-of-a-rut-by-doing-just-one-small-thin-5838682) 。想出一件困扰你很久的事，并处理它。修理一个摇晃的门把手，取消你的网飞订阅，或者预约一个你已经推迟的约会。成就感会让你再次动起来。

重要的是你开始行动了。无论事情看起来有多小，你都在掌控，并向自己证明你有能力改变自己的感受。开始钩针编织，或者尝试一种你从未吃过的食物。选择一些东西——无论你有什么感觉——并努力去做。

### **寻找支持**

你可能认为你是一块石头，但事实是没有人能独自做每件事。伸出手去，找一个会听你说话、给你建议，甚至强迫你走出家门去公园散步的人。不管你认为自己有多孤独，你都不是。Lifehacker 评论者 [Piccles](http://lifehacker.com/people-its-all-about-the-people-in-your-life-i-guaran-1639147046) 说这是让你的生活感觉更好的关键:

> 人们。都是关于你生命中的人。我保证在某个地方会有人关心你。最后，我和几年前没怎么见面的大学朋友重新联系上了，这些友谊给了我继续前进的力量。让你周围的人合理地关心你，想花时间和你在一起，想看到你成功，会有很大的不同。

找家人、朋友，甚至同事聊天。当你需要帮助的时候，寻求帮助没有错。如果你认为自己可能会抑郁，那么寻求帮助就更重要了。抑郁症是一种比任何常规都更能让你沮丧的斗争，但它不必一直这样。 [你的感觉完全正确](http://lifehacker.com/five-lessons-i-learned-from-dealing-with-depression-1595249546) ，但是当你沮丧时，不仅仅是简单的不开心。

寻求专业帮助 ，记住检查你的心理健康应该和检查你的身体健康一样重要。总有办法跟某人说， [即使你负担不起](http://lifehacker.com/how-to-find-someone-to-talk-to-when-you-cant-afford-th-1185844420) 。

当然，在话语中也总能找到力量。也许一本书的某一段或一句引语会给你一个你需要的跳跃。 [Fiona2393](http://lifehacker.com/at-one-point-in-my-teens-i-got-really-interested-in-na-1639139547) 语出惊人:

> 在轻度抑郁/发情期，我看到了 Gene Kranz，一位备受尊敬的任务控制高层，在发射台火灾中失去三名宇航员后对他的团队的回应……我已经数不清这两个段落有多少次了。承认我在生活中的某个地方搞砸了，我有很多工作要做来“修复”它，这是成功的一半。

无论适合你的是什么，你都需要时不时的支持。要求不容易，但是你生活中的人不想看到你不开心。所以，向某些东西伸出手——人、宗教、鼓舞人心的话语——紧紧抓住它，开始前进。

### **做些运动**

有很多很好的理由去锻炼 ，但是没有比当你墨守成规的时候更好的时机了。你会增强自信，缓解疲惫的大脑，获得更好的睡眠，你会有更多的精力去做你真正关心的事情。翻译:你会更快乐。

开始的时候不一定要疯狂。像每天散步 25 分钟这样简单的事情就能让你动起来，改变你的节奏。评论者 [pdxwhy](http://lifehacker.com/i-was-in-a-horrible-rut-a-dead-fiance-a-shitty-job-wi-1639264966) 决定开始散步只是为了有一段时间思考:

> 我开始散步是为了健身，后来成为竞走爱好者。这样做让我能够真正思考，只是活在当下。散步让我能够冥想和祈祷，这是迈向美好生活的重要的第一步。
> 
> 我总是告诉人们出去走走，把事情想清楚。

如果你在寻找更大的转变，为自己设定一个锻炼目标。 [斯科特](http://lifehacker.com/id-gained-a-fair-amount-of-weight-and-had-slipped-happi-1639144441) 瞄准了一个大家伙:

> ...背景是一部名为“骑行分水岭”的电影，这是一部关于名为“旅游分水岭”的活动的纪录片，这是世界上最艰难的山地自行车比赛之一。我说“听起来是个不错的目标。”两年来，我一点一点地进行训练，并在此基础上不断改进(实际上，我在 GTD 运用了这些原则，并将它们应用到山地自行车运动中)。
> 
> 2013 年 6 月 14 日，我开始了与来自世界各地的 160 多人的比赛。不用说，我不再墨守成规了...

当然，你不必参加世界上最艰难的自行车比赛。只要你让自己动起来，你会感觉更有控制力，并准备好处理那些挤压你生活的事情。找一项你想坚持的运动、爱好或常规，你很快就会感觉更好。

### **走到外面**

在户外呆上几分钟(或者至少走出家门)会彻底改变你的生活吗？可能不会，但这是感觉更好的第一步。事实上， [最近的研究发现，](https://lifehacker.com/improve-a-bad-mood-by-going-outside-for-5-minutes-1831927148) 即使在大自然中呆上五分钟也能帮助你改变心情:

> 研究人员进行了两项独立的测试，参与者被要求在不同的时间里坐在房间里或户外，所有人都没有电子设备让他们有事可做。
> 
> 虽然在这两种情况下，负面情绪在短短几分钟后就会降低，但置身于大自然中往往会激发积极的情绪。

所以，走出去，绕着街区散散步，去公园，甚至只是在外面的某个地方坐几分钟。换个环境可能会有帮助。

### 对你的生活做出或大或小的改变

对于许多感到停滞不前的人来说，他们生活中总有一些事情让他们感到压抑。可能是你住的地方，你的工作，甚至是某个特定的人吸走了你的快乐(甚至是三者都吸走了)。花一点时间思考你生活的方方面面，并 [确定哪些可能是需要改变的消极来源](https://lifehacker.com/give-your-negative-inner-voice-a-name-1833910237) 。

如果你的工作让你不开心， [你不一定要辞职](http://lifehacker.com/how-to-keep-going-when-youre-demotivated-at-work-1631681320) 。设定工作时间和个人时间界限的简单改变会有所帮助。评论者[angrygreycatreads](http://lifehacker.com/i-found-that-what-was-the-mainstay-of-my-dissatisfactio-1639125462)对工作感到不满意，但找到了改善工作的方法:

> 我不能辞掉我的工作，但是我发现，通过设定一些明确的界限，不让工作像以前那样渗入我的个人生活，做更多我喜欢的事情(尤其是下班后)“不，准时离开去上瑜伽课”有很大的帮助。

有了界限，你会有更多的空闲时间。你可以利用这段空闲时间四处闲逛充电，或者——就像评论者 [scarlet.pirate](http://lifehacker.com/i-started-scheduling-my-free-time-with-the-stuff-i-want-1639184407) 建议的那样——把你的空闲时间当成第二份工作来安排:

> 我必须努力。所以和朋友出去玩就像安排会议或拜访客户——你最终会变得非常积极，而不是被动。与其说“我想他们没回我短信”，不如说你开始主动。什么时候对你最合适？嘿，我还没收到回复，我们还继续吗？(人怪怪的；说，“你想找个时间一起出去玩吗？”这永远不会发生。说:“这周五下班有空吗？”你开始讨论。)

有时一个简单的改变可以扭转局面，但其他时候需要更大幅度的改变。也许你生命中有一个人除了消极什么都没有。尽管很难，但把那个人从你的生活中剔除是必须的。你的环境也可能会影响你。评论员 [edenn1423](http://lifehacker.com/moved-out-of-the-destination-city-that-had-a-crap-job-1639498147) 急需一场足以与核选项相匹配的变革:

> 搬出了那个工作和关系都很糟糕的“目的地”城市。放下一切离开了。3 年后，我以 90%的工作量赚了两倍的钱，买了房子，有了孩子，不再每天晚上酗酒。有史以来最好的决定。

有时候，你需要做出的改变就在你的脑海中，在你的思维方式中。你有没有想过，也许事情并没有你想象的那么糟糕？有可能你只是把一切都看得太“半空”评论者 [重启](http://lifehacker.com/i-went-through-a-pretty-rough-patch-in-my-late-20s-div-1639133380) 决定对发生的事情负责:

> 我感觉到并说出了“我有一片乌云笼罩着我。”有一天，我意识到坏事不仅仅发生在我身上。是我造成的。当我对自己的生活负起责任，不再找借口，事情就有了转机。这并不容易，但我让自己回到了正轨。我做了导致“坏事”的选择，一旦我决定重新考虑这些选择，坏事就会消失。算是吧。坏事还是会发生。但你掌控着自己的幸福。如果你让生活发生在你身上。不知不觉就结束了。

没有比当你 [感觉自己失控](https://lifehacker.com/adjusting-your-mindset-may-work-best-when-you-have-no-c-1454878786) 的时候更好的尝试调整心态的时机了。记住，除非你采取行动，否则什么都不会改变。在某种程度上，你可以控制和谁在一起，做什么，住在哪里，如何思考事情，甚至在哪里工作。这些事情中的任何一个简单的改变都能给你所需要的视角调整。

* * *

走出困境没有正确的答案，因为每个人的情况都完全不同，但希望这些想法可以成为你的出发点。如果有的话，把这些故事和建议作为证据，证明即使是生活中最低谷的时候也是有可能克服的。
