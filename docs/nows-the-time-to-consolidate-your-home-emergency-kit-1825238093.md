# 立即整合您的家庭应急包

> 原文：<https://lifehacker.com/nows-the-time-to-consolidate-your-home-emergency-kit-1825238093>

还记得你买了一个新的开罐器，因为你认为你失去了旧的吗？你打算怎么处理你车库里积满灰尘的那卷用了一半的胶带？你上次用厨房垃圾抽屉里的手电筒是为了假装绝地武士，是吗？这些都是你可能到处都有的物品，你可以把它们合并成一个家庭应急包。这可能会救你和你家人的命。



目标是 [创造一个 72 小时应急包](https://lifehacker.com/seven-emergency-preparedness-tips-you-may-not-know-1787063633) ，它应该能够让你和你的亲属在没有电力或任何外界帮助的情况下连续三天保持强壮。你不必为此购买新的供应品。下一次你清理或整理家里的任何地方时，留意有用的应急物品。首先，很有可能你会发现一个空的塑料箱，或者一个已经很久没人用的背包或行李袋。抓住它，开始用这些东西填充它，如果你在清理的时候碰到它们的话:

*   72 小时内不会腐烂的食物(你一直保存的罐头或干货，但只是占据了你橱柜的空间)
*   手电筒

*   电池(手电筒用)
*   一声口哨

*   垃圾袋和塑料带(用于个人卫生)

*   基本工具，如钳子和几把扳手

*   开罐器

*   镊子
*   剪刀
*   管道胶带
*   比赛
*   漂白剂(当用九份水对一份漂白剂稀释时，可用作消毒剂)
*   额外的手机充电器(确保它是为你当前的手机准备的)，或者太阳能电池

*   72 小时所需的药物

*   婴儿配方奶粉和尿布

*   宠物的食物和水

*   额外的一副隐形眼镜或眼镜(老处方总比没有好)
*   额外的衣服(从你要送人的一堆衣服中整理出一套衣服)
*   一些让每个家庭成员感到舒适的私人物品，比如糖果、玩具、游戏、书籍、咖啡或茶

这里还有一些你应该买下来放在应急包里的东西，这些东西你不太可能在家里随意找到

*   防尘口罩
*   一个额外的急救箱(这应该与普通的家庭急救箱分开)
*   超湿巾或婴儿湿巾
*   电池供电或手摇收音机
*   NOAA 气象电台
*   72 小时的水(密封的瓶子、罐子或盒子——不是水龙头里的水)
*   当地地图(防水，如果可能的话)
*   聚酯薄膜保温毯

在你的工具包里放一张写有信用卡和借记卡号码(有安全码)的纸，还有一些现金，这是个好主意。卡系统在紧急情况下可能会瘫痪。一旦你的工具包放在一起，找一个容易拿到的地方，把它放在那里。
