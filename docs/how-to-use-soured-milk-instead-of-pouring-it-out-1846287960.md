# 如何使用酸奶而不是倒掉

> 原文：<https://lifehacker.com/how-to-use-soured-milk-instead-of-pouring-it-out-1846287960>

如果你看过 [这个视频](https://skillet.lifehacker.com/should-you-try-these-retro-soda-pop-concoctions-1846089449) ，你已经知道我是那种变态的人——很像一个孩子——喝全脂牛奶只是为了尝一尝。我喜欢它，多年的 [对我的身体有益](https://www.youtube.com/watch?v=InOuPYCsUjs) 帮助我培养了一种精致的“乳制品口味”如果我的牛奶有一丁点儿酸，我都能尝出来。



然而，这种轻微的异味，甚至更强烈的酸味，并不一定意味着牛奶是无用的。根据 2015 年 NPR 采访自然资源保护委员会的与 [达纳·冈德斯](http://www.nrdc.org/about/staff/dana-gunders) 的报道，变质的巴氏杀菌奶不太可能让人生病:

> 冈德斯写道，因为随着牛奶的老化，它变得更酸，创造了一个“对可能导致疾病的微生物不友好”的环境

生(未经高温消毒的)牛奶就不是这种情况。不要喝酸的生牛奶(新鲜的生牛奶 [是危险的，因为它是](https://www.fda.gov/food/buy-store-serve-safe-food/dangers-raw-milk-unpasteurized-milk-can-pose-serious-health-risk) )。

但是一点点刻薄不会伤害你。只要你的牛奶没有长霉，也没有令人作呕的腐臭，稍微过了保质期的酸性牛奶可以用来做很多美食。

烘焙是最明显的选择，如果你想掩盖酸味，这是最好的选择——像“普通”牛奶一样一杯一杯地使用。(冈德尔斯非常喜欢把它用在煎饼里。)就在昨晚，我用一些牛奶做了一个柠檬橄榄油蛋糕，它已经过了“最佳食用”日期两个星期，我还会再做一次。(“最佳食用日期”是制造商对牛奶何时尝起来“最新鲜”的最佳猜测——这几乎是一个无用的日期)。我不想喝这种有问题的牛奶——它有点发黄，闻起来有点酸，但它很好地融入了丰富的、已经有点酸的面糊中。( [品尝台](https://www.tastingtable.com/dine/national/sour-milk-recipes-leftover-food-hacks-baking) 声称，酸奶实际上比新鲜的牛奶更好，额外的酸有助于发酵，但这种效果很难明确量化。)

如果你有很多酸奶，酸奶可以用来腌制 [家禽](https://skillet.lifehacker.com/this-buttermilk-brine-will-turn-you-into-a-turkey-breas-1830318684) 和猪肉，效果非常好——只需加入一点盐，让你的肉在调味的酸性牛奶中浸泡一夜，然后像往常一样烹饪。你也可以把整个“变酸”的事情引向它的逻辑结论，做奶酪。乳清干酪( [腌制](https://skillet.lifehacker.com/will-it-sous-vide-fresh-homemade-cheese-1797317672) 或 [否则](https://lifehacker.com/the-easiest-way-to-make-your-own-ricotta-cheese-at-home-1784142814) )从来都不是一个坏选择。
