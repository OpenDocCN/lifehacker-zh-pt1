# 你应该油炸一些鲑鱼

> 原文:[https://life hacker . com/you-should-air-fry-some-salmon-1847299642](https://lifehacker.com/you-should-air-fry-some-salmon-1847299642)

我来自一个吃鲶鱼的家庭，在我搬到太平洋西北部之前，我从来没有怎么吃过鲑鱼。这两种鱼在质地、味道或烹饪风格上有着天壤之别，但最终，在我将近35岁的时候，我想出了我最喜欢的制作三文鱼的方法，其中包括空气油炸锅。

Watch

我跟你说实话，空气油炸锅并不一定比普通的大烤箱做得“更好”——它只是做得更快，耗能更少。如果你要为一个五口之家做三文鱼，你最好坚持使用全尺寸烤箱，但如果你想为一个人做一些三文鱼，这是适合你的方法。

然而，让这种三文鱼真正美味的是高效空气油炸锅和快速预煮干盐水的结合。我之前已经谈到过 [这种方法](https://lifehacker.com/fix-your-dull-fish-with-a-salt-and-sugar-cure-1846734923) ，但是仅仅在糖和盐的混合物中停留15分钟，就可以给拳头注入味道，并带走多余的水分，使它更结实，更有弹性。这让你的准备时间增加了一刻钟，但这是不干涉的准备时间，所以谁在乎呢。

## 如何用你的空气油炸锅烹饪三文鱼

你所要做的就是抓一块带皮的三文鱼柳，涂上一层由两份盐和一份糖混合而成的闪亮混合物。让它保持15分钟，然后在冷的流水下冲洗鱼并拍干。在拳头的两面擦一点橄榄油——刚好够涂——然后在385度的空气油炸锅里煮7-10分钟，这取决于菲力的大小(末端应该很容易剥落，顶部应该稍微变成棕色)。

这似乎太简单了，但是干盐水就是你需要的所有调料。如果你想的话，你可以加入一些香料，但要确保在放油之前先放入，因为这层脂肪可以防止它们燃烧。我通常用干盐水保持它的简单，然后通过新鲜香草、柠檬汁或 [这种复合黄油](https://lifehacker.com/how-to-make-steakhouse-butter-at-home-1847102251) 来增加额外的味道，这就是我昨晚完成我的鱼的原因。实际上，我会坚持让你先用复合黄油试一下，然后再做别的。(尽管我最初开发这种带刺山柑斑点的肥油是为了搭配牛排，但我认为它搭配一块鱼更好。)
