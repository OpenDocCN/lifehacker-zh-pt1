## 如何在没有链接的情况下拨打谷歌聊天电话

这一 [功能正在全球](https://workspaceupdates.googleblog.com/2021/12/start-meetings-or-audio-calls-from-one-on-one-google-chat-messages.html) 为谷歌工作空间账户和个人账户推广，b 但谷歌实现这一功能的方式可能会有点令人困惑。首先，它是一个谷歌聊天功能，实际上在谷歌聊天应用程序中不起作用。它在Gmail应用程序的聊天选项卡中工作。如果你点击谷歌聊天中的新会议按钮，你会被转到Gmail应用程序来拨打电话。

但是一旦你在Gmail应用程序中找到聊天标签，你就可以开始了。如果你看不到聊天标签，它可能会在设置中被禁用(可能像我们一样，你在几年前就禁用了Hangouts标签)并且从未回头。您可以从**设置** > **账号** > **聊天**中启用。

当你打开一个对话时(更新到最新的Gmail应用程序后)，你会在顶部工具栏看到两个新图标。您可以使用它们来拨打音频或视频电话。

一旦通话激活，您将在顶部的横幅中看到时间戳。从t 这里，您可以继续做其他事情，而t 点击横幅将带您回到通话屏幕。就像其他流行的聊天应用一样，未接来电也会显示在同一个聊天窗口中。
