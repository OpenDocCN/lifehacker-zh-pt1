# 你可能一直以来都没有去核桃子

> 原文：<https://lifehacker.com/youve-probably-been-pitting-peaches-wrong-this-whole-ti-1847295033>

我喜欢桃子——我可以一连几个小时吃一个桃子！—但是我不喜欢从水果的中心去除顽固的、几乎粘在一起的果核。幸运的是，我在社交媒体上关注了一些非常聪明的人，这些人通常有解决这些日常烦恼的办法。(老实说，我讨厌说“你做错了！”所以我只会说"*我一直*做错了，"因为我不了解你的生活。)

幸运的是，来自 Kitchn 的 Naomi Tomky 发现了它，尝试了它， [写了关于它的](https://www.thekitchn.com/best-way-to-cut-a-peach-23197878) ，并在推特上发布了关于它的消息，这使我尝试了它，爱上了它，并写了这个关于它的小帖子。

秘诀很简单:抓起一个桃子，从中间切开，垂直于桃子侧面的锯齿状线，将它切成两半。轻轻拧动两半，直到其中一个脱离。哇哦。那很容易。

接下来，将两半切成四分之一，确保与那条线成 90 度角。再次扭转，然后从四分之一处弹出坑。

我对桃子和油桃都尝试过这个小技巧，它对两者都很有效。我甚至用那些扁平的甜甜圈桃子试了试，它对那些很有效。如果你直接从果核中咀嚼水果，这看起来可能不会令人难以置信地改变你的生活，但对于喜欢制作和食用桃子馅饼、桃子蛋糕、桃子冰淇淋和其他核果甜点的人来说，这肯定会让夏天变得容易得多。
