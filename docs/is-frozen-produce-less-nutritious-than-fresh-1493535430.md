# 冷冻农产品不如新鲜的有营养吗？

> 原文：<https://lifehacker.com/is-frozen-produce-less-nutritious-than-fresh-1493535430>

乍一看，很容易认为冷冻蔬菜不如新鲜蔬菜健康，但事实并非总是如此。事实上，冷冻蔬菜通常也同样富含营养。在某些情况下，它们甚至能更好地保留营养。

### 冷冻产品通常和新鲜产品一样营养丰富

一般来说，我们倾向于认为冷冻产品不如新鲜产品有营养，因为它通常与冷冻披萨和微波炉晚餐一样出现在杂货店的过道里。这与其说是一个实际问题，不如说是一个形象问题，冷冻产品通常和你在杂货店买的新鲜产品一样有营养。

一般来说，农产品从农场到杂货店需要几天到两周的时间。在运输过程中，由于暴露在热、光和氧气中，营养物质从农产品中逸出。当农产品不是当季的时候，这个时间甚至会延长，这就是为什么浆果在冬天通常更贵更难找到。这个视频提供了一个很好的解释:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-zjsOOT347cA&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-zjsOOT347cA&start=0) 

相比之下，冷冻产品通常在成熟时采摘，然后立即冷冻。这锁住了营养成分，有时会使冷冻产品比新鲜产品更有营养，但这真的取决于蔬菜。 [华尔街日报讲解](http://online.wsj.com/news/articles/SB10001424052702304483804579282520637682840) 研究的过程和结果来自 [冷冻食品基金会](http://www.frozenfoodfacts.org/) :

> 采摘后不久，准备冷冻的蔬菜会受到热水或蒸汽的快速冲击——这被称为热烫，它会杀死一些营养成分，但也会阻止冷冻后的褐变和营养成分的损失。这一步骤中损失最大的是水溶性维生素，如 C 和 b。然后蔬菜被快速冷冻，锁住了大部分营养成分，便于长期储存....

> 冷冻西兰花、草莓和青豌豆都比冷藏样品含有更多的维生素 C。然而，冷冻菠菜的维生素 C 含量低于新鲜或冷藏的样品。佩格博士在论文中写道，这可能是因为切碎的菠菜表面积大，在热烫过程中会浸出更多的维生素 C。

[研究](http://bfff.co.uk/wp-content/uploads/2013/09/Leatherhead-Chester-Antioxidant-Reports-2013.pdf) 经过研究表明同样的事情:冷冻农产品和新鲜农产品具有相似的营养价值。当农产品不合时令时，通常最好买冷冻的。如果是这样，在杂货店或农贸市场寻找当地农民，从新鲜农产品中获取最大的营养。

也就是说，冷冻产品不像生产品那样能立即获得营养。因为它是包装好的，这意味着它经常和奶酪、糖和其他不太健康的成分密封在一起。当你购买冷冻产品时，水果或蔬菜应该是配料表上的唯一项目。此外， [健康饮食建议](http://www.eatingwell.com/nutrition_health/nutrition_news_information/fresh_vs_frozen_vegetables_are_we_giving_up_nutrition_fo) 购买包装上标有美国农业部“美国 A 级(也称为“花式”)”标志的冷冻产品，这表明使用了高质量的产品。

### **如何让冷冻农产品味道更好**

冷冻产品的耻辱不仅仅来自它的营养价值——冷冻产品尝起来和新鲜的不一样，所以很难想象它是健康的。这个问题的根源是冷冻产品的质地。

为了解决这个问题，厨师推荐 调味，避免简单的菜肴，最重要的是，蒸蔬菜而不是煮:

> 使一批冷冻蔬菜变味的最快方法是将它们煮熟。如果你没有时间或者没有兴趣去打开蒸笼烧水，那就用一点水把它们放进微波炉里。只要确保你的水只到蔬菜表面的 1/3 处，然后煮到完全变热。也就是说，如果你有时间，拿出炉子蒸锅，好好招待他们！

味道是个人喜好的问题，但不可否认的是，冷冻产品的味道比新鲜的有点不同。对某些人来说， [比其他人](http://www.bbc.co.uk/news/health-12982663) 更重要，但是一个完全不科学的排名超过了《赫芬顿邮报》 [分解了一些最好和最差的生产](http://www.huffingtonpost.com/2012/05/15/frozen-vegetables_n_1515098.html#s971662&title=Brussels_Sprouts_) 来购买冷冻食品。最差的包括抱子甘蓝、胡萝卜、花椰菜和芦笋。玉米、甜豌豆和毛豆等主食通常和新鲜的一样美味。无论哪种方式，当用正确的方法烹饪时，大多数冷冻产品都可以很好的保存下来。
