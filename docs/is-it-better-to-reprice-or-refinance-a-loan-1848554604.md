# 贷款重新定价好还是再融资好？

> 原文：<https://lifehacker.com/is-it-better-to-reprice-or-refinance-a-loan-1848554604>

大多数人都知道你可以为贷款再融资，这包括以更好的条件获得新贷款，并还清旧贷款。再融资通常与房屋抵押贷款有关，但 [你可以再融资任何贷款](https://www.nerdwallet.com/article/loans/personal-loans/how-to-refinance-a-personal-loan) 只要条款允许，你可以找到更好的交易。抵押贷款再融资是如此普遍，这几乎是给新房主的标准建议:关注利率，如果利率低于你现在支付的水平，就再融资。

但是还有另一个选择来改善你的债务状况:重新定价。虽然类似于再融资，但重新定价有几个明显的优势，如果你想降低成本或缩短当前贷款期限，这是值得研究的。

## 什么是 reprice？

当你重新贷款时，你通常会去找一个新的贷款人，以更好的条件发起一笔新的贷款。你用新贷款偿还旧贷款，把你的生意转移到另一家银行。任何曾经为抵押贷款进行再融资的人都知道，这可能是一个漫长而复杂的过程。当你申请原始抵押贷款的时候，你不得不费力地去查阅所有的文件？你支付的所有费用和签署的表格？你必须从头再做一遍。

另一方面，重新定价将这一过程保留在同一个贷方。基本上，你会去你的银行说，嘿，我注意到这些天你的贷款条件比我当初贷款时好得多——我能转换到那些新的低利率吗？你没有把业务交给竞争对手的银行，而是把一切都放在内部。这可能会令人困惑，因为许多贷款人将这称为再融资，这两个术语经常互换使用。

虽然这听起来和再融资没什么不同——因为它*和*本质上是一样的——但对你当前的贷款重新定价比再融资有一些明显的优势:

*   **成本。**大多数贷款重新定价都会从您的银行收取简单的固定费用，通常在 500 美元到 1，000 美元之间，具体取决于贷款的规模。另一方面，再融资会引发一系列额外的成本，因为你要再次经历整个贷款过程。当你呆在你现在的贷方时，所有的工作都已经完成了。再融资可能会花费你几千美元，如果你还在“锁定”期，除此之外，你可能还要支付罚金。
*   **时间。**如果你有抵押贷款，你会知道银行在打开资金龙头的时候会像冰川一样移动。再融资也是一样——可能需要几个月才能完成这个过程。另一方面，重新定价通常是银行的一个简单的内部流程，通常在几周内完成。

这是两个非常大的优势，所以值得你花时间去研究重新定价而不是再融资。如果你在另一家银行找到了更好的利率或条款，也值得把这些带给你现在的贷款人，看看他们是否会为你现在的贷款重新定价，以保住你的生意。你最糟糕的情况是被告知“不”,不得不经历更艰难的再融资过程。

这不仅仅适用于抵押贷款:信用卡经常在没有咨询你的情况下对你的贷款重新定价，根据“”基于行为的重新定价来改变你的利率，这通常是惩罚性的(你的利率上升，通常是由于错过付款)，但并不总是如此。只要条款允许，而且你的贷款人愿意，你可以对你的任何贷款重新定价。需要注意的是，你的贷款人没有*和*对你的贷款重新定价——你的银行可能会坚持全额再融资，包括所有相关的费用和延迟。但很可能值得调查。

底线:如果利率对你有利，你想为你的贷款再融资，先考虑重新定价。它可以节省你的时间和金钱。
