# 重新加热土豆泥的唯一方法

> 原文：<https://lifehacker.com/the-only-way-to-reheat-mashed-potatoes-1846345736>

与汤、炖菜、红烧菜和辣椒不同，土豆泥在冰箱里放置一夜后，味道会更好。冷土豆又硬又干——仅仅是它们以前的影子。但是用微波炉加热或者在热火炉上猛烈搅拌只会让它们变得更干——如果你搅拌得太多的话——更油腻。对于真正豪华的剩余捣碎器，你必须温柔地用爱加热它们(更多的脂肪)。

事实上，我已经成功地用微波炉加热了一些冷土豆泥，但是我是在 15-20 秒内完成的，在每次加热之间，轻轻加入一半 T3，一半 T20，牛奶和/或大块黄油 T4。为了这些目的，一个 350 度的烤箱也可以。不管用什么加热方法，添加额外的水分和脂肪是没有商量余地的，因为它们会把你僵硬、干燥的土豆转变成香甜的奶油土豆。对于水分，你可以使用奶油、牛奶，甚至是高汤，但要确保里面也有一些脂肪，无论是黄油还是 [纯素食替代品](https://skillet.lifehacker.com/these-fake-butters-are-good-actually-1834313812) (如果那是你的包)。

你对土豆做得越多，它们就会变得越厚越粘，但是轻微的折叠不会造成太大的伤害，尤其是如果土豆一开始就没有过度劳累的话。如果你正在用微波炉加热少量的土豆，将它们放入碗中，加入少量黄油(我每份用大约一汤匙)，并加入少许奶油，一半&一半，牛奶或肉汤。加热 20 秒，用叉子轻轻搅拌，再加入一点奶油，再次加热。直到你的土豆变热呈奶油状。

如果你在烤箱里重新加热更大的一份，你可以稍微放手一点。将土豆均匀地摊放在烤箱安全的托盘上，涂上黄油(每杯大约半汤匙)，倒入 1/4 - 1/2 杯奶油，一半&一半，牛奶，或肉汤 ( 或肉汤和奶制品的混合物)在上面——你希望土豆刚好被盖住。用箔纸盖紧，在 350 度的烤箱中加热大约半个小时，直到土豆达到 165 华氏度。用叉子叉起，上桌。
