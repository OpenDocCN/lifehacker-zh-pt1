# 透明胶带奇怪但令人惊讶的实际用途

> 原文：<https://lifehacker.com/weird-but-practical-ways-you-arent-using-scotch-tape-1848662336>

我们都知道透明胶带在包装礼物、连接错误的纸片和修补破损的书页时的优点。但事实证明，我们可能睡在这种胶带的一些鲜为人知，但同样有用的应用上。这里有一些创造性的方法来使用它，不仅仅是基本的。

清洁你的键盘。如果你手头没有一罐压缩空气，一条胶带几乎可以同样有效地快速带走碎屑和灰尘——尤其是当它们卡在笔记本电脑键盘和屏幕之间令人讨厌的缝隙中时。

捡起碎玻璃。当你扫完大块的碎玻璃或罐子时，小块的可能会留下来，等着下次你赤脚走过房间时卡在你的脚后跟上。所以用胶带包一卷纸巾(有粘性的一面朝外)，用它来捡起任何残留的小碎片。

修补磨损的鞋带。如果你的鞋带末端破损严重，无法穿过鞋带孔，用一条胶带将它重新变窄变整齐——这应该可以持续到更换鞋带。

自己制作棉绒滚筒。是，T3】实际皮棉辊存在 T5】。但有时，即使作为正常的成年人，当我们需要时，我们身边也没有一个。输入透明胶带:将它大量缠绕(粘性面朝外)在毛刷或空卫生纸卷筒的手柄上。瞧，DIY 去绒效果最好。

**整齐地插花。**将一条条胶带以十字交叉的方式重叠在一起，在花瓶的顶部 [制作一个均匀间隔的网格](https://www.scotchbrand.com/3M/en_US/scotch-brand/tips-uses/tape-hacks/) 。将花放在胶带之间的空白方格中，以确保它们的间距合适，并且直立美观。

**将保鲜膜边缘固定到位。**在两次使用之间，用一条胶带将塑料包装的边缘固定住(当它不可避免地被盒子吞没时，可以帮助你找到它)。或者，在纸卷的末端贴上一张纸，对折，形成一个(或两个)标签，便于拉出。

**防止你的猫爪造成伤害。**再多的胶带也无法防止*所有*与卡特彼勒相关的磨损，但它有助于将爪子对家具的损坏降至最低。在你想让你的猫避开的地方贴上双面胶，很有可能它们会避开。

自己做苍蝇纸。用胶带把空的卫生纸或纸巾卷包起来，挂在天花板上(或房子里最容易被窃听的地方附近)来抓苍蝇。

检查农产品的成熟度。好吧，你实际上不用胶带检查*的成熟度*。但是，如果你好奇一个水果是否可以食用，剪下一个小三角形，看看里面，如果不是，用胶带将三角形封回原位，不要打乱成熟过程。

**装上备用电池以备后用。**如果事先准备很困难，可以用胶带将备用电池固定在挂钟、儿童《我的世界》、灯或其他小设备后面，以备需要更换旧电池时使用。

给小孩子惊喜。如果你需要在短时间内给一个小孩留下深刻印象，带上一个气球、一条胶带和一个图钉。将胶带放在气球的中间，将大头钉穿过胶带，看着他们的眼睛亮起来，因为它(希望)不会爆开。口袋里放根棒棒糖以防万一。

**打造完美无瑕的猫眼。**对于可能没有足够经验徒手画出“猫眼”的化妆爱好者来说，可以用两小块胶带在你的外眼角旁边做一个三角形。用眼线笔填充胶带片之间的空隙。在你离开房子之前，不要忘记拿掉胶带。

**防止标签弄脏。**涂上一层薄薄的“你猜对了”,防止种子、处方标签、寄信人地址和文件夹标签上的深色印记或湿记号笔被弄脏...豆子。开玩笑的。磁带。用胶带保护他们。
