# 不同种类的酵母以及如何使用它们

> 原文：<https://lifehacker.com/the-different-kinds-of-yeast-and-how-to-use-them-1846153945>

我收到的最常见的烘焙问题之一是，当食谱需要速溶酵母时该怎么办，但你只有活性干酵母。它们到底有多不同，你用哪种有关系吗？

Watch

我很高兴地告诉大家，不，没有。所有三种类型的商业酵母——速溶、 [、活性干](https://lifehacker.com/how-to-make-a-single-packet-of-yeast-last-1842923653) 和蛋糕——做同样的事情，稍加调整就可以互换使用。你只需要知道你有哪种类型，是什么让它滴答作响。

速溶干酵母和活性干酵母非常常见，几乎完全相同。两者都是通过干燥保存的，这使它们具有相同的超细卵石质地。流行的观点认为，你应该将速溶酵母与干配料混合，而活性干酵母应该溶解在湿配料中。这是一个可靠的经验法则:速溶酵母的设计是为了在混合过程中激活，而活性干酵母最好先启动一下。但是，如果液体成分的温度合适，这两种技术对两种类型的酵母都非常适用，这可能会让你感到惊讶。

根据 [红星酵母的使用说明](https://redstaryeast.com/yeast-baking-lessons/yeast-types-usage/) ，以下是每种方法的最高液体成分温度:

*   **添加到干配料中的酵母:** 120-130 华氏度——摸起来明显温暖，但不烫手。
*   **酵母因潮湿的成分而开花:**110-115 F——勉强温热。

酵母在这些范围内最活跃。稍微冷一点没关系——只是酵母需要更长的时间来完成它的工作——但是更高的温度会杀死酵母。

这比什么都更能让人放心。食谱通常不会包括这两种酵母的说明，所以你可能会在没有意识到的情况下，在面粉中加入速溶酵母或活性干酵母——一旦你这样做了，你就会吓坏了。不要慌！只要你的液体原料不太热，酵母是活的，你如何把它们加入到你的面团中并不重要。

在这一点上，有一种情况下，开花总是正确的举动，即使是速溶酵母。如果你完全不知道那罐酵母有多老了，那就先把它发酵一下。在温水中撒几撮，搅拌一下，静置 5-10 分钟。如果它没有变成泡沫和冒泡，它就死了，你需要一个新的罐子。

这样就剩下蛋糕酵母，也被称为“新鲜”或“压缩酵母”它根本没有干燥，它的高含水量带来了一些挑战。首先，它不耐储存，所以祝你在商店里找到它。你需要比干酵母多得多的蛋糕酵母才能达到同样的效果——红星酵母推荐 [的重量比](https://redstaryeast.com/recipes/how-to-convert-dry-yeast-to-cake-yeast/) 多三倍不到——而且很微妙。有多精致？它非常脆弱，温度超过 90-95 华氏度的 [液体会杀死它](https://redstaryeast.com/yeast-baking-lessons/yeast-types-usage/cake-fresh-yeast/) 。但除此之外，它可以像干酵母一样使用:在干原料上碾碎后就可以了。

那么，应该用哪一种呢？可能不是蛋糕酵母——偶尔摆弄一下很有趣，但经常使用就太不切实际了。选择即时干燥还是主动干燥取决于个人喜好。我用速溶酵母是因为我经常烘焙，而且很容易大量找到。如果你是一个零星的面包师，活性干酵母包就不那么重要了，效果也一样好。
