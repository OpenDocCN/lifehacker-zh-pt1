# 这是唯一“快速”催熟鳄梨的方法

> 原文:[https://life hacker . com/this-is-the-only快速成熟鳄梨的方法-1849574036](https://lifehacker.com/this-is-the-only-fast-way-to-ripen-an-avocado-1849574036)

鳄梨有一个讨厌爱它们的粉丝俱乐部。它们味道很好，可以切片、捣碎或混合，可以用来制作从 [沙拉酱](https://www.recipetineats.com/creamy-avocado-salad-dressing/) 到 [冰淇淋](https://www.foodnetwork.com/recipes/alton-brown/avocado-ice-cream-recipe-1945087) 的任何东西。但是他们喜欢装模作样。鳄梨看似几分钟就完全成熟了，但大部分时间都放在台面上，要么坚如磐石，要么已经腐烂。这一事实导致了大量“快速成熟”鳄梨的出现。如果你很想听完所有的故事，享受这种感觉。那很好。无论如何，那些黑客不工作。事实证明，只有一种方法。

除了让水果自然成熟几天之外，还有一种独特的方法可以加速鳄梨的成熟。它比快了*，但还是不快。将鳄梨放入纸袋，或任何封闭的空间，理想情况下，与另一个已经成熟的【跃变水果】放在一起(比如苹果或香蕉)。建议用纸，因为它可以吸收可能导致霉菌的水分，然而它只有一天，所以霉菌不太可能在鳄梨的皮肤上生长。(如果你没纸了，你可以把餐巾或纸巾扔进塑料袋里。根据鳄梨在“暂停”前的当前成熟状态，它将在12-24小时内成熟，其他水果也将提前成熟。我遇到的每一个“成熟”的黑客都是软化的黑客。你必须考虑你真正想从你的绿色小女神那里得到什么——成熟，或者这该死的东西已经变软了。*

[马里兰大学分校](https://extension.umd.edu/resource/ethylene-and-regulation-fruit-ripening) 详细描述了一个水果成熟时会发生许多特征变化，软化只是其中之一。

> 果肉的硬度通常会变软，含糖量上升，酸度降低。芳香挥发物被释放出来，水果的真正风味形成了。水果的颜色通常变暗，果皮和果肉变软，绿色背景颜色变淡。

真正影响成熟的唯一方法是用乙烯 化学引发生物过程 [。乙烯是一种指示成熟过程的植物激素，不，你不需要从亚马逊上买。这就是纸袋的用武之地。跃变型果实是可以从植物上成熟的果实。它们可以在成熟前收获，并在储存中继续成熟，因为它们会自然释放乙烯。未成熟的水果释放低浓度的乙烯，但随着它的成熟，乙烯，除其他外，将提醒其他发育过程，并开始释放越来越高浓度的乙烯气体。因为鳄梨已经在释放乙烯，所以把它放在一个封闭的空间(如纸袋、塑料袋或纸箱)里会锁住乙烯并包围水果，发出信号让它释放越来越多的乙烯。如果放在柜台上，这种重要的催熟气体就会流失到房间里。](https://extension.umd.edu/resource/ethylene-and-regulation-fruit-ripening)

如果鳄梨特别绿，加入一个已经成熟并释放出高级乙烯气体的水果，会给鳄梨一个跳跃的开始。请注意，水果也会受到乙烯容器的影响，很可能变得*过熟*。鳄梨受益于其他水果的高级生命阶段，达到其游戏的顶端，而无辜的助手水果萎缩并变成糊状。

把你的鳄梨放在袋子里，每八到十个小时检查一次，看看它有多熟，然后拿出来减缓成熟的速度。这个乙烯容器的把戏对任何跃变期的水果都有效(我跟你说的是硬桃子)，而且实际上会成熟水果的质地、颜色，最重要的是味道。如果你曾经点了一道包括鳄梨片和未熟的鳄梨片的菜，你就会知道这有多令人失望。在我目前的生命阶段，我可以咀嚼坚硬的食物，但未成熟的鳄梨尝起来像水一样。

鳄梨软化技巧(如在上面倒盐，放入烤箱，或简单地在食品加工机中消灭它)，可能会让你捣碎它，但味道将保持不变:缺乏。

如果你组织了一次足球周日聚会，并且你负责鳄梨酱，我最好的建议是给自己至少24小时的鳄梨袋时间。如果你没有那么多时间，你最好拿几把Sabras，偷偷把它舀进你自己的特百惠保鲜盒里。
