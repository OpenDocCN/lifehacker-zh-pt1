# 当你需要钱的时候，如何使用汽车股权贷款

> 原文：<https://lifehacker.com/how-to-use-an-auto-equity-loan-when-you-need-the-money-1849318889>

过去的几年对大多数人来说都是一段疯狂的旅程，我们生活的方方面面都发生了或大或小的变化。当你现金充裕时，这种混乱已经够难的了，但当资金紧张时， [通货膨胀飙升，](https://www.marketwatch.com/story/almost-all-the-economic-numbers-line-up-a-u-s-recession-is-likely-11658336680) ，紧急情况突然出现，你要支付巨额账单，，情况会变得很糟糕。

如果你没有可靠的应急基金来帮助你度过紧急时刻，最常见的解决办法是借钱。使用信用卡处理突发债务是一个简单的解决方案，但这些利率通常会让你后悔这个决定——任何类型的发薪日贷款都会有同样的结果。你需要的是基于抵押品的担保贷款，比如房屋净值贷款——但如果你是租房的话，这一点帮助都没有。这就是你的车可以进来的地方:就像房屋净值贷款一样，你也许可以获得汽车净值贷款，，即使你还欠着。

以下是汽车股权贷款的工作方式。

## 汽车 t itle 贷款和汽车 equity 贷款有什么区别？

首先要明白的是，有两种方式来贷款对你的车:汽车股权贷款和汽车产权贷款。你会像躲避瘟疫一样想要避免后者，因为它基本上是一种高息 [发薪日贷款](https://lifehacker.com/how-to-protect-yourself-from-predatory-payday-loans-1822295550) 恰好对你的汽车所有权设置了留置权。它们往往是非常短期的，更容易获得，这就是人们上当的原因，但它们是一笔糟糕的交易，如果你拖欠还款，你可能会失去你的汽车。

另一方面，股权贷款通常由传统的贷款机构提供，比如你的银行。这是一种担保贷款，用你的汽车股权作为抵押，所以利率是合理的，付款将是明确和固定的。

## 如何计算你在汽车中的权益

获得汽车股权贷款的第一步是弄清楚你能借多少钱。这是一个非常简单的过程:

1.  确定你还欠这辆车多少钱。如果你已经还清了贷款(或者一开始就是用现金买的)，这个数字显然是零。
2.  通过检查 [凯利蓝皮书](https://www.kbb.com/whats-my-car-worth/) 或 [Autotrader](https://www.autotrader.com/car-values/) 或其他一些资源来确定汽车的当前价值。(准备好失望吧——汽车贬值*快*。)
3.  从第二个数字中减去第一个数字。这既是你在汽车中的权益，也是你汽车权益贷款的潜在价值。这并不意味着这就是银行或其他贷款人实际上会提供给你的——他们会有自己奇怪的数学方法来计算他们愿意承担多大的风险。

例如，如果你有一辆状况良好的 2018 款福特金牛座，其目前的估计价值约为 18500 美元。如果你欠了 5000 美元的贷款，你可以*潜在地*从你的股权中借 13500 美元。虽然有些贷款人会让你借 100%的汽车股本，但许多人不会愿意借给你那么多，但这是一个好的起点。

获得汽车股权贷款的过程类似于任何其他贷款。你找到一家提供汽车股权贷款的贷款机构(不是所有的贷款机构都提供汽车股权贷款——大多数大银行都不提供，所以你可能需要寻找较小的本地银行或网上银行，如 [Mariner Finance](https://www.marinerfinance.com/car-loans/auto-refinance/) )，填写申请，并完成贷款机构要求的任何其他步骤。只要一切正常，这个过程往往会很快。有了网上银行，你通常可以在一天之内获得批准和现金，但是如果你有时间的话，为了获得最好的利率，稍微推迟一下可能是值得的。

## 不利方面

虽然汽车股权贷款比发薪日贷款更好，是解决短期现金短缺的好办法，但也有一些不利因素需要考虑:

*   **风险。**你用你的汽车来担保贷款，这意味着如果你不能偿还贷款，你可能会失去汽车。如果你已经还清或接近还清汽车贷款，这可能会特别令人恼火。
*   **隐性成本。**一些贷款机构收取额外费用，因为汽车股权贷款并不常见，被认为比其他贷款风险更高，所以请务必阅读所有细则。由于贫穷的代价很高，贷款人可能会要求你购买综合汽车保险来保护他们的资产，所以除此之外，你的月供会更高。

底线呢？如果你有短期的现金需求，并且你的汽车有很多资产，汽车资产贷款是一个相对稳定的弥补缺口的方法。但是首先探索其他的选择可能是有意义的，你应该始终把风险记在心里。
