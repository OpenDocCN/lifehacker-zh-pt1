# 用盘子检查你的肉汁是否“熟了”

> 原文：<https://lifehacker.com/use-a-plate-to-check-if-your-gravy-is-done-1846529748>

如果你曾经做过肉汁，或任何添加了增稠剂的酱料或类似食物，你就会知道，一旦你把它从热腾腾的煨锅里拿出来，放到盘子里，它的作用就完全不同了。肉汁、果酱和许多酱料在冷却时都会变稠，当它们冒泡时，很难判断它们何时“完成”。不要依赖于他们如何涂抹勺子，而是在一个冷的或室温的盘子上蘸一点，看看它的表现。(如果你对勺子戏法感到满意，那很好，但是我总是纠结于勺子是否有足够的涂层，结果我盯着勺子看的时间太长了。)



当选择你盘子的温度时，想想你打算如何储存或端上你正在烹饪的食物。果酱制造商用一个冷盘测试他们的产品，看他们如何凝胶，这是有意义的，因为果酱通常储存在寒冷的环境中。将一点果酱放在一个放在冰箱里五分钟的盘子上，然后将盘子垂直放置。如果堵塞 [下降缓慢，则](https://www.mountainfeed.com/blogs/learn/15722885-how-to-find-the-perfect-jam-setting-time#:~:text=Pop%20a%20plate%20in%20the,it%20runs%20slow%2C%20it's%20set!) 完成。(你也可以把果酱放在盘子里，然后 [把盘子放进冰箱](https://nchfp.uga.edu/how/can_07/jelly_point.html) 。如果它在几分钟后落下，你应该是好的。)

对于酱料和肉汁，你不需要盘子这么冷。两者通常都放在室温(或稍暖)的盘子里，所以在盘子上测试比放在冷冻的盘子上更有意义。一旦你认为你的肉汁或酱汁达到了“终点”，取一勺倒在室温的盘子上，静置几秒钟，观察它的表现。如果它比你想要的稍浓一点，煮久一点，然后重复，直到它变浓到你喜欢的程度。
