# 什么是‘悲伤债务’，我们怎样才能摆脱它？

> 原文：<https://lifehacker.com/what-is-grief-debt-and-how-can-we-get-rid-of-it-1846420211>

正如我们在过去的一年中所熟知的那样，悲伤不仅限于失去所爱的人(尽管那是一种特别痛苦的悲伤)。无论是失去工作，长时间见不到家人和朋友，还是仅仅哀悼我们在疫情之前的生活，我们仍然不知道这种集体悲伤和创伤在未来的岁月里会对我们产生多大的影响。



这种悲伤不仅会随着时间的推移而积累，而且在经历了一年不间断的种族不公正、自然灾害和前总统政府造成的所有损害之后，我们永远没有时间在下一次创伤发生之前处理好一次创伤。这给我们留下了一种叫做“悲伤债务”的东西——就像金钱债务一样，很难还清。这里有一些关于悲伤债务的知识以及如何处理它。

### 什么是悲伤债务？

正如艾米丽·劳伦斯为《井+好 所写的那样，悲伤债务“发生在我们从情感银行中提取能量来处理每一次冲突的时候，这些情况彼此复合，进一步耗尽能量，直到没有任何东西可以提取。”

处理一个创伤已经够难了，但是同时处理多个创伤又是一个完全不同的挑战。这是因为如果我们没有机会完全度过第一次创伤，“往往会发生的是，你在度过第一次创伤之前就开始关注下一次悲剧，”临床心理学家 [卡希纳·a·路易斯](https://www.strengthsandsolutions.com/about-kahinalouis-strengths-and-solutions) 博士告诉 [好+好](https://www.wellandgood.com/grief-debt/) 。

因此，为了应对生活，我们压抑创伤，“当你压抑创伤时，你可能会压抑某些情绪，但它们不会消失，”路易斯补充道。"它们会浮现在你的思想和行动中，而你甚至没有意识到."

### 如何摆脱悲伤债务

虽然处理和度过悲伤当然是可能的(也是明智的),但这里的想法是学习应对策略来管理它向前发展——而不是期望它在某段时间内消失。但是当我们处于悲伤债务的情况下，最重要的是至少做点什么来度过它，而不是让创伤继续堆积。

每个人都有自己喜欢的技巧，无论是写日记，正念练习，与治疗师和/或悲伤咨询师交谈，还是只是让自己尖叫和哭泣，精神病学家 [詹姆斯·s·戈登](https://cmbm.org/about/founder/)博士告诉 [好+好](https://www.wellandgood.com/grief-debt/) 。重要的是找到适合你的方法，并给自己时间和空间去做，而不要关注下一件坏事。
