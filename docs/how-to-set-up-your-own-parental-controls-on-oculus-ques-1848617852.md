# 如何在Oculus Quest上设置自己的家长控制

> 原文:[https://life hacker . com/how-to-set-up-your-own-parental-controls-on-oculus-ques-1848617852](https://lifehacker.com/how-to-set-up-your-own-parental-controls-on-oculus-ques-1848617852)

Oculus Quest— 现在被称为Meta Quest— 非常有趣，但是没有屏幕你可以和他们一起看，很难知道你的孩子是如何使用它的。虚拟现实缺乏家长控制，但你*仍然可以*使用其他功能和设备设置一些限制。

## 知道没有Quest家长控制

设置Quest家长控制的第一步是了解没有家长控制。虽然Quest需要一个脸书账户才能使用，但从技术上来说，仅限13岁及以上的用户使用，它没有任何内容过滤器。

当然，在让你的孩子购买或玩游戏之前，你可以查看特定游戏的内容分级。然而，如果你正在寻找一个可以打开来限制某些内容的过滤器，你不会找到一个。Quest 有一个内置的浏览器，也没有家长控制。

尽管如此，这并不意味着VR父母控制没有选项。

## 转换到其他设备

虚拟现实家长控制最大的问题之一是缺乏一个屏幕来看孩子们看什么。您的孩子可以在耳机上 [播放电影和在线视频](https://lifehacker.com/how-to-stream-movies-and-tv-in-virtual-reality-1848340674) ，而无需向任何人展示。你可以通过 [将耳机视图](https://support.oculus.com/articles/in-vr-experiences/oculus-features/cast-with-quest-2/) 投射到另一个设备来解决这个问题。

首先，确保耳机和您想要将投射到的设备在同一个wifi 网络上。然后，下载Oculus应用到你的手机上，打开右下角的 **设置**。轻按“**配对新耳机**”，然后按照说明连接耳机和应用程序。

在Oculus打开的情况下，点击手机右上角的图标，该图标看起来像一个带有wifi信号图标的耳机。在的**演员表下，选择您的耳机，点击**开始**从那里，你可以通过手机上的耳机获得现场直播。**

## 保存文件

您还可以将耳机设置为与移动应用程序同步保存的文件，以跟踪您的孩子正在下载的媒体。在耳机中，按下Oculus按钮调出主菜单，然后选择**应用程序**按钮。选择**文件**应用，点击右上角的云图标。

现在你可以在Oculus移动应用的**设备**下的**同步媒体**下看到任何保存的文件。这样，您就可以随时关注您的孩子保存在设备上的内容种类 —j 只需知道此媒体 [将在14天后自动消失](https://www.engadget.com/oculus-quest-update-media-mobile-090046369.html) 。

## 设置路由器限制

虽然可能没有内置的Quest家长控制，但您可以限制wifi路由器上的浏览器内容。大多数路由器 [都有一些家长控制](https://lifehacker.com/how-your-kids-are-outsmarting-all-your-parental-control-1848249586) ，通常让你过滤某些内容，监控流量，屏蔽特定网站。

这取决于您的具体路由器。请参考路由器的说明，了解您的选项以及如何使用它们。然而，有时这些控件并不适用于每个浏览器，所以先在耳机中试用一下。

## 要考虑的其他安全因素

记住，即使有内置的VR家长控制，也没有完美的解决方案。结合使用这些步骤并考虑第三方网络安全应用程序将会产生最佳效果。

即使您不担心内容，出于一般安全考虑，最好还是使用这些控件。像任何设备一样，VR头戴设备也容易受到黑客攻击，因此关注您的任务历史和内容是一个好主意。
