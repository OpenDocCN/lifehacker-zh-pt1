# 用这个小技巧来阻止自己呕吐

> 原文：<https://lifehacker.com/use-this-trick-to-stop-yourself-from-throwing-up-1849070937>

我们都经历过恐慌、不舒服的时刻(或者很多次)，当那种可怕的、完全消耗的“哦，妈的，我要吐了”的感觉占据我们的身体，让我们向最近的容器、水槽、草地或任何不是我们自己的衬衫和裤子的地方冲刺。虽然我们不能保证让你的胃里的东西无限期地下去(如果他们需要出来，他们最终会出来)，但这种方法可以帮助你在短时间内阻止呕吐，直到你可以找到一个更好的地方释放它。

Watch

当抖音的创作者 [@onlyjayus 在社交媒体上发布了一个视频](https://www.tiktok.com/@onlyjayus/video/7043537716394462470?is_copy_url=1&is_from_webapp=v1) 描述了一些预防疾病的技巧——其中一个是哼歌来抑制呕吐的冲动——作为大多数健康相关的黑客，我持怀疑态度。但这一说法随后被伦敦帝国理工学院的英国国民健康保险制度外科医生 Karan Rajan 博士证实为,他利用自己拥有近 500 万追随者的平台给出医学建议，并将事实与医学虚构区分开来。

为什么这一招能防止或延缓呕吐？根据 Rajan 的说法，哼唱抑制了呕吐反射。正如 [罗斯·佩里](https://www.cosmedics.co.uk/doctors-surgeons/dr-ross-perry/) 博士，医学博士美容皮肤诊所的主任也向每日邮报解释说，“当呕吐反射发作时，经常会引起各种压力和焦虑的感觉。通过控制和调节你的呼吸，它帮助你放松。要同时插科打诨和哼哼实际上非常困难。”

佩里补充了其他一些加强你呕吐反射的建议，比如刷你的舌头，或者在舌头上放盐来激活味蕾。“对一个人有效的方法可能对另一个人无效，但最终还是要把注意力放在其他事情上，而不是呕吐，”他补充道。

(顺便说一句，原始视频中另一个经医生验证的方法是在感到恶心时嗅闻——而不是摄入——外用酒精。)

作为一个在我的赤手空拳中抓到孩子呕吐，并沮丧地从汽车座椅、地板垫和昂贵的地毯上清理它的父母，我一定会把这颗智慧的珍珠传授给我的小家伙们。他们是否有足够的情感资源在时机成熟时实施黑客攻击，这仍是一个不确定的 TBD，但我计划在未来的公路旅行前进行彩排。

但对于我们成年人来说，这种黑客行为似乎很容易被起诉。所以下一次，当你克服了净化的冲动时，试着哼哼——只要时间够到去厕所。
