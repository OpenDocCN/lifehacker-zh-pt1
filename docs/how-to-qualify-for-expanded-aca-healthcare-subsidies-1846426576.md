# 如何获得扩大的ACA医疗补贴

> 原文:[https://life hacker . com/how-to-qualify-for-expanded-ACA-health care-补助费-1846426576](https://lifehacker.com/how-to-qualify-for-expanded-aca-healthcare-subsidies-1846426576)

民主党最新的刺激方案 [法案](https://www.congress.gov/bill/117th-congress/house-bill/1319/text#toc-H3E33A2EFC9B94CB3B5DACF351A51F835) 包括一项360亿美元的临时扩大奥巴马医改税收抵免，使中产阶级在2021年和2022年更容易获得每月保费补贴。该法案还提高了这些补贴的金额，并废除了一项可能导致纳税季令人不快的意外的规定。以下是你需要知道的。

### 等等，每月都有医疗补助？

是的，确实如此。数百万美国人已经不利用 [保费税收抵免](https://www.investopedia.com/irs-form-8962-understanding-your-form-8962-4845835) (PTC)适用于通过 [健康保险市场](http://healthcare.gov) 购买的平价医疗法案计划。PTC是一个显著的月折扣— [【平均512美元】](https://www.cnbc.com/2021/02/11/how-to-avoid-tax-surprises-with-the-health-insurance-premium-credit.html)—使您有资格获得更好的保险。

目前，资格取决于你的收入是否在贫困线的100%到400%之间(2020年个人收入略高于51，000美元)、你的年龄、家庭规模和你的居住地。这种信贷是浮动的，如果你挣得少，就要付出更多，而且是可退还的，这意味着如果你的信贷超过了你的应税负债，你将获得现金退款。更多详情可以在 [这里](https://www.investopedia.com/premium-tax-credit-2021-definition-5115090) 找到。

### 那么，有什么变化呢？

补贴将在每个收入水平上增加，从而减少你的收入比例，以符合给定的计划(医疗保健新闻网站KFF在这里有更多细节)。为了大致了解利率是如何增加的， [看看这张对比2020年和2021年](https://www.healthinsurance.org/wp-content/uploads/2021/03/H.R.-1319-effect-on-subsidies.pdf) 利率的图表。这张图表中的大多数人每个月会节省大约50-100美元，尽管补贴会根据收入、年龄或地点而有所不同。

另一个大的变化是消除了“补贴悬崖”，它描述了一旦你的收入超过贫困线的400%，你的PTC补贴会突然被切断。由于报税人可以选择每月获得抵免，作为基于他们估计的年收入的预付款，当他们的实际收入在一年中攀升时，许多人会措手不及，从而取消他们的PTC资格。在这种情况下，借款人只有在纳税的时候才知道他们欠下了美国国税局所有的每月信用额(T2)——总计可能达数千美元。

这一修正消除了400%的突然截止，将超过门槛的人可支付医疗保险费的收入比例限制在8.5%，同时仍允许他们有资格。这一修正旨在防止家庭 [支付高达其年收入](https://www.healthinsurance.org/obamacare/beware-obamacares-subsidy-cliff/) 一半的健康保险费。

[Per MedPage Today](https://www.medpagetoday.com/infectiousdisease/covid19/91512) ，该法案还包括适用于2020纳税年度的“还款假期”。这又一次摆脱了补贴悬崖，因为收入高于预期的人不必偿还额外的税收抵免。

最后，一项特殊规则允许任何人在计算税收抵免资格时，在领取至少一周失业保险的情况下，可以忽略高于贫困线138%的任何收入。
