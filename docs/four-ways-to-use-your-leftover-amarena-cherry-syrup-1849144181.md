# 使用剩余阿马雷纳樱桃糖浆的四种方法

> 原文：<https://lifehacker.com/four-ways-to-use-your-leftover-amarena-cherry-syrup-1849144181>

从一罐 Amarena 樱桃中捞出最后一个梗是非常令人失望的——如果没有樱桃，那就更令人沮丧了。这些苦樱桃味道浓郁，有一种浪漫的深红色，你一打开盖子就会被它深深吸引。它们是马拉希诺樱桃的升级版，真的可以为圣代或鸡尾酒增添一些光泽。但是当最后一颗樱桃被吃完时，不要沮丧地倒出剩下的五盎司糖浆。这种东西是甜樱桃的魔力，是你把从鸡尾酒到华夫饼的所有东西都做成樱桃的秘密原料。(加上罐子真的很可爱。)

## 糖衣和釉料

做一份美味的樱桃糖衣，放在面包蛋糕、肉桂面包或一些自制的甜甜圈上会很棒。大多数釉料是由糖粉、液体(通常是水或牛奶)和偶尔的黄油混合而成。要做樱桃糖霜，找一个简单的糖霜配方，像云杉吃的这个，用 Amarena 樱桃糖浆代替液体。按照食谱的指示混合。更简单的樱桃釉配方是，将两杯糖粉和四分之一杯阿玛瑞纳糖浆搅拌在一起，然后用 p our。

## 煎饼和华夫饼

当你意识到你没有对每一罐后总理樱桃都这样做时，准备好踢自己吧。Amerena 樱桃糖浆已经是糖浆了，你知道谁喜欢被糖浆覆盖吗？煎饼和华夫饼(不评价你的其他非食品糖浆应用)。你可以在薄煎饼上淋上几汤匙樱桃糖浆，立即添加一股酸樱桃味和一层枫糖浆样的涂层。

你也可以把糖浆直接加到煎饼或华夫饼干面糊里。这种方法将产生一个可爱的，玫瑰色的早餐。制作时，在搅拌面糊时，用等量的 Amarena 樱桃糖浆代替配方中的糖。尽管它是一种液体代替了干的成分，但大多数食谱要求相对少量的甜味剂，并且它不会影响稠度。请记住，樱桃味会很淡(这种方法大多是为了可爱的粉红色)，所以我建议在这些热蛋糕上也浇上一点樱桃糖浆。

## 增加新鲜樱桃水果馅

当新鲜樱桃上市时，我疯狂地冲向我的馅饼盘。完全成熟的水果提供了无与伦比的味道，但不知何故，我总是得到至少一袋不那么美味的樱桃。如果你发现自己处于类似的情况，或者现在是冬天，你所能得到的只是冷冻樱桃，剩余的 Amarena 糖浆可以为你的水果增加一些“成熟度”。准备好樱桃馅料后，在每杯馅料中加入一汤匙樱桃糖浆。在倒入烹饪容器之前，将它搅拌到混合物中以覆盖樱桃。像往常一样烘烤。

## 鸡尾酒

一旦你这样做几次，你可能会开始称自己为调酒师，我不会阻止你。Amarena 樱桃糖浆是许多鸡尾酒中简单糖浆的绝佳替代品。它(很明显)是甜的，但与简单的糖浆不同，它带来了酸、微苦的樱桃味，为威士忌或黑朗姆酒等喜怒无常的烈酒增添了一点趣味，但它足够明亮，可以为西瓜鸡尾酒增添活力。要获得微妙的樱桃口味，在鸡尾酒中加入半盎司阿马雷纳樱桃糖浆，就像 o ld f ashioned，s ide c ar，n egroni 或 Manhattan。为了让樱桃更有明星气质，在鸡尾酒中加入一盎司，比如 c osmopolitan、m argarita 或 perol s pritz。一般来说，任何使用甜苦艾酒或黑樱桃的地方都是 Amarena 糖浆加入派对的安全场所。

这种鸡尾酒配方是水果味和柑橘味的。这是一个休闲的下午，招待一些朋友，搭配水果和可涂抹的奶酪的绝佳选择。这无疑给夏日增添了一点生气，但也可以根据你是否需要在不久的将来走下一段楼梯来修改。也可以将前两种食材分批，提前放入冰箱保存。当你准备好了，在注入香槟酒之前，倒三盎司在冰上。

## 醉人的樱桃柠檬水

**配料:**

*   2 盎司 [柠檬酒](https://lifehacker.com/the-two-best-ways-to-make-limoncello-at-home-1849089119)T3】
*   1 汤匙 Amarena 樱桃糖浆
*   4 盎司普罗塞克(对于酒精含量较低的版本，用苏打水来代替)
*   柠檬轮装饰

用冰块装满岩石杯或高球杯。在鸡尾酒摇壶中，加入冰，柠檬和樱桃糖浆。*摇晃几秒钟。在冰上过滤内容物。填充普罗塞克或苏打水。

*如果你没有摇壶，你可以将这两种原料倒入一个空玻璃杯中，用力搅拌，然后加入冰块。
