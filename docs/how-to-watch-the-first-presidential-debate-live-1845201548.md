# 如何现场观看第一场总统辩论

> 原文：<https://lifehacker.com/how-to-watch-the-first-presidential-debate-live-1845201548>

明晚，唐纳德·特朗普和乔·拜登将在克利夫兰进行 2020 年大选季的首场总统辩论。无论你纯粹是为了看好戏而观看还是不知何故仍然不确定谁会得到你的投票，这场辩论很可能至少会成为不寻常的 90 分钟电视。以下是你需要知道的关于辩论的内容，以及如何现场观看。



### 关于第一次总统辩论，你应该知道些什么

川普和拜登之间的第一场总统辩论将于明天，9 月 29 日(T0)星期二，美国东部时间(T2 时间)晚上 9-10:30 在俄亥俄州克里夫兰的凯斯西储大学举行。《T4》福克斯新闻频道周日的主持人克里斯·华莱士将担任主持人。华莱士选择了以下六个主题，每个主题大约 15 分钟， [NPR 报道](https://www.npr.org/2020/09/28/916614846/trump-and-biden-debate-tuesday-heres-what-you-need-to-know) :

*   特朗普和拜登的记录
*   最高法院
*   新冠肺炎（新型冠状病毒肺炎）
*   经济
*   美国城市中的种族和暴力
*   选举的公正性

据 NPR 报道，这些话题已经引起了争议 : [36 名民主党参议员签署了一封信](https://www.npr.org/2020/09/23/916197984/democrats-criticize-upcoming-debate-for-not-including-climate-change) 要求将气候变化加入辩论议程。还有人对将种族和暴力混为一谈表示担忧，指出这样做会混淆问题。

特朗普最近的言论表明，如果他输掉选举，他不会承诺和平移交权力，关于选举诚信的辩论的最后一部分很可能是...有趣，退一步说。

### 如何现场观看第一场总统辩论

观看这场辩论应该非常简单，即使对那些没有有线电视的人来说也是如此。它将在大多数主要网络和有线新闻频道现场直播，包括 ABC、CBS、Fox、NBC、CNN、福克斯新闻频道、MSNBC 和 C-SPAN。你也可以在 YouTube 或 [上听或看 NPR](https://www.npr.org/2020/09/28/916614846/trump-and-biden-debate-tuesday-heres-what-you-need-to-know) 上的 [直播辩论。](https://www.youtube.com/watch?v=wW1lY5jFNcQ)
