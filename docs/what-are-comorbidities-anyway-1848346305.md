# 到底什么是“共病”？

> 原文：<https://lifehacker.com/what-are-comorbidities-anyway-1848346305>

疾病预防控制中心主任罗谢尔·瓦伦斯基最近说，大多数死于新冠肺炎的人有多种合并症，这是“令人鼓舞的消息”。虽然我不同意“令人鼓舞”的部分，但我认为重要的是看一看共病到底是什么。



## 共病到底是什么意思？

发病率这个词指的是疾病和健康状况。它是死亡率的对应词，意思是死亡。

为什么我们不直接说呢？这些是统计学家和公共卫生人员在分析大量数据时使用的通用词汇。(有趣的事实:CDC 的公共健康杂志被称为 MMWR，用于*发病率和死亡率周报*。)

那么，共病是指同时患有多种疾病或健康状况。如果死于 COVID 的人有“四种共病”，这意味着除了 COVID 之外，他们还有其他四种健康状况。

## 哪些合并症会增加死于 COVID 的风险？

那么什么算共病呢？看你问谁了。当瓦伦斯基发表她臭名昭著的引用时，她指的是(来自，是的，MMWR)。让我们看看这篇论文是如何分解共病的。

首先，他们没有考虑所有可能的共病。相反，他们列出了八种他们认为是危险因素的情况:

> 严重后果的危险因素包括年龄≥65 岁、免疫抑制和其他六种潜在疾病。所有有严重后果的人至少有一个危险因素；78%的死者至少有四个孩子。

术语“风险因素”比“共病”更准确，因为 65 岁不是一种疾病。文件中列出的风险因素包括:

*   65 岁及以上
*   免疫抑制
*   慢性肺疾病
*   慢性肝病
*   慢性肾脏疾病
*   慢性神经(脑)疾病
*   糖尿病
*   慢性心脏疾病

有关哪些健康状况会增加您的死亡或严重疾病风险的更多信息，请参见 COVID，[CDC 在这里有一个更完整的列表](https://www.cdc.gov/coronavirus/2019-ncov/need-extra-precautions/people-with-medical-conditions.html) 。例如，它们包括神经系统疾病下的阿尔茨海默氏症和痴呆症。

这些风险因素相当普遍，通常会一起出现。例如，[65 岁以上的成年人中有 17%患有心脏病](https://www.cdc.gov/aging/publications/coronary-heart-disease-brief.html)[27%患有糖尿病](https://www.cdc.gov/diabetes/pdfs/data/statistics/national-diabetes-statistics-report.pdf) 。我们可能都至少有几个邻居和家庭成员有清单上的四种危险因素。

这就是为什么正如瓦伦斯基所说的那样,*而不是*令人鼓舞的是，“超过 75%的绝大多数死亡发生在至少患有四种共病的人身上。”是的，如果 COVID 以同样的比率针对健康人和患有合并症的人，情况会更糟*, b 但是如果你听到这个消息松了一口气，你可能会忘记许多美国人都有这些疾病，包括你的一些亲人。*
