# 跑步时如何防止皮肤发炎

> 原文:[https://life hacker . com/how-to-prevent-champion-when-you-run-1846496462](https://lifehacker.com/how-to-prevent-chafing-when-you-run-1846496462)

跑得足够久足够远，有些东西会开始摩擦。也许是你的大腿，也许是你的胸罩带子，也许是一些随机的斑点，这些斑点在10英里的跑步中从来不是问题，但在12英里的跑步中却是致命的。以下是处理方法。

### 知道什么导致皮肤发炎

擦伤是由摩擦引起的皮肤刺激。如果你想一想你每分钟走了多少步，你跑了多少分钟——这真的累加起来了。

包括汗水在内的水分会软化皮肤，使其更容易因摩擦而受损。这就是为什么皮肤发炎在夏季会更严重。如果你习惯把水倒在头上降温，或者如果你在小路上跑过小溪，注意不要弄湿任何容易发炎的地方。

任何东西都可能会擦伤，但往往是你最意想不到的东西，比如衣服上的带子或接缝，在短途旅行时还好好的。你计划在长跑中穿的任何东西都应该是你以前在长跑或至少是中长跑中穿过的，这样你就不会有任何突然的惊喜。

### 润滑

最快的解决办法是给自己买一根 [BodyGlide](https://www.bodyglide.com/product/body/) 或者一桶凡士林，在你容易发炎的地方使用。(凡士林会弄脏一些衣服，如果你碰巧喜欢三项全能运动，凡士林不适合做潜水服，所以BodyGlide是更好的全能产品。)

出门前申请，如果是长跑，可以考虑带上一些在旅途中重新申请。马拉松通常会在后半程的加油站或医疗救护站提供凡士林(志愿者会在冰棒棍上递给你一点凡士林)。即使在跑步过程中，你也可以把它涂在受影响的皮肤上，这样可以防止皮肤进一步发炎。

### 穿着讲究

如果你的大腿在短裤里摩擦，买一条长点的短裤。(自行车短裤或压缩紧身衣最适合这种情况。)

如果你被接缝或带子擦伤了，那就去找无缝的、没有标签的衣服吧。贴身的衣服不容易摩擦，因为没有太多的机会摩擦。排汗面料有助于保持干爽，还能防止皮肤发炎。

### 使用胶带，但要仔细考虑

胶带可以保护皮肤，但有几点需要注意。它最适合用在乳头上，乳头是男性常见的发炎部位。(运动文胸对于有胸的人来说，往往会避免这个问题。如果你戴了胸罩仍然会有乳头发炎，一个更合身的胸罩可能会解决这个问题。)

在已经粗糙的皮肤上贴胶带会进一步刺激皮肤，所以可以考虑乳头胶带或乳头防护物，它们是为此目的而制造的，通常具有更温和的粘合剂。

胶带不仅不能防止皮肤发炎，还会引起皮肤发炎，特别是当胶带变湿，然后打滑或起皱时。并且胶带的表面会刺激附近的皮肤。例如，有一年我在半程马拉松之前完成了一次长跑，我的上臂有些擦伤，我以为是腋窝的胡茬引起的。所以在比赛日，我拍了一段kinesio胶带(从我的旅行包里免费的！)到擦伤的地方。我的手臂很好，但比赛中胶带摩擦了我的腋窝，最后我的情况更糟了。事后看来，我应该坚持BodyGlide。
