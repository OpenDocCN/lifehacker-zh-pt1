# Xbox Game Pass正在失去一些最好的游戏

> 原文:[https://kotaku . com/Xbox-game-pass-2022年2月-阵容-对比-crossfirex-1848459347](https://kotaku.com/xbox-game-pass-february-2022-lineup-contrast-crossfirex-1848459347)

世界上的Xbox硬盘最终需要休息一下。2022年2月，Xbox Game Pass不再是最近记忆中最拥挤的发布名单。不仅是即将到来的阵容相当单薄，而且微软的游戏点播服务中一些最好的和最大的将很快无法使用。

Watch

下面是2月15日一切都半途而废:

*   *控制*(云、控制台、PC)
*   *码脉*(云、控制台、PC)
*   *最终幻想十二:黄道时代*(主机，PC)
*   *媒介*(云、控制台、PC)
*   *Project Winter* (云、控制台、PC)
*   *猎鹰者*(云、控制台、PC)

一年后——或者说大概有足够的时间让游戏完成10.7%——巨大的*最终幻想XII*reduxT5离开服务。Remedy Entertainment的*控件*也有类似的时间。尽管 *The Medium* 和 *The Falconeer* 并没有掀起什么波澜，但这两款游戏都是Xbox系列X发布期间的必玩游戏，因此它们与当前游戏机服务的背离无疑令人惊讶。

相比之下(抱歉)，下面是即将发生的事情:

**2月3日**

*   *对比*(云，控制台)
*   *Dreamscaper* (云、控制台、PC)
*   *说谎*(云、控制台、PC)

**2月10日**

*   *围攻*，通过游戏预览(云、主机、PC)
*   *交叉火力*(控制台)
*   *永恒的边缘*(云、控制台、PC)
*   *Skul:英雄杀手*(云、主机、PC)
*   *地球上最后的孩子和毁灭之杖*(云、控制台、PC)

**2月14日**

*   *方舟:终极幸存者版*(云，控制台，PC)
*   *阴魔*(云、控制台、PC)

是的， [*Skul:英雄杀手*是一款令人愉快的罗格式](https://kotaku.com/an-action-platformer-perf-for-those-too-chicken-for-hal-1847966338) 游戏，具有整洁的变形元素，但它并不是那种你会输掉整个周末的游戏。*对比*是一个不可抗拒的时尚平台，拥有杀手级的配乐，但它也是在八年前首次问世(当时它可以免费向PS Plus的订户提供)。

这里的避雷针是 *CrossfireX* ，一款以Remedy开发的战役为特色的第一人称射击游戏。在两年前的一次预演活动中， [*Kotaku* 的伊恩·沃克指出](https://kotaku.com/crossfirexs-campaign-doesnt-seem-like-a-remedy-game-yet-1844486510) 似乎足以胜任一部军事射击片，但缺乏工作室的标志性繁荣。看看这个游戏从那以后是如何发展的，如果有的话，将会很有趣。如果没有，好吧，这个月还有很多其他的可以玩。
