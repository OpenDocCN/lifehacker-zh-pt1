# 当 COVID 限制不断变化时如何计划去维尔京群岛的旅行[更新]

> 原文：<https://lifehacker.com/how-to-plan-a-trip-to-the-virgin-islands-when-covid-res-1847034449>

由于美属维尔京群岛(USVI)是美国领土，在疫情期间，它一直是的热门度假目的地，因为旅行规定不如 [国际旅行](https://www.cdc.gov/coronavirus/2019-ncov/travelers/international-travel-during-covid19.html) 严格。一些地方仍然要求大陆游客入住 [授权的检疫酒店](https://www.canada.ca/en/public-health/services/diseases/2019-novel-coronavirus-infection/latest-travel-health-advice/mandatory-hotel-stay-air-travellers.html) 或别墅，或者要求 [定期检测](https://www.forbes.com/sites/alexledsom/2021/06/01/europe-is-opening-june-eu-travel-restrictions-covid-19-test-requirements-quarantine-by-country/?sh=113832852104) 。但是随着疫苗接种和不断变化的 [疾病预防控制中心的建议，](https://www.cdc.gov/coronavirus/2019-ncov/travelers/map-and-travel-notices.html) 计划一次旅行是相当复杂的，所以这里有一些建议来计划从大陆到 USVI 的假期。



## 提前两个月开始计划你的海岛度假

餐馆和短途旅行提前几天(有时几周)就被预订一空，所以在最后一刻冒险通常是不可能的。例如，受欢迎的 Oceana 餐厅&小酒馆可能会得到的好评，但游客和工作人员会敦促你提前预订。在我最近的一次旅行中，这家著名的餐厅在那个周四开始的周末被预订了。

这些岛屿也有日落鸡尾酒巡游和水下海上远足，但它们都很快填满。没有必要过度计划你的行程，但是在去这个岛的几周前，试着预定热门活动。

像汽车租赁这样的事情也很快被预订，虽然租赁可能仍然开放，但价格会上涨。我提前买了机票，但迟迟没有租车，留给我的是每天 100 美元的租金。相反，如果你打算租车的话，可以同时预订机票和租车。

对于航班，你可以预计直达往返票价在 245 美元到 489 美元之间，如果你同意中途停留，票价会便宜到不到 200 美元。Qtrip 的 2021 年机票研究 发现，最佳购买时间是旅行日期前 64 天左右，这样可以获得最优惠的价格，所以提前两个月左右计划购买机票。

## 计划你的新冠肺炎测试，因为它仍然是必需的

USVI 要求每个旅行者，无论是否完全接种疫苗，在抵达前不超过五天的时间里，新冠肺炎病毒检测结果为阴性。可接受的检测包括 PCR、抗原、和阳性抗体检测(抗原和抗体检测都接受快速检测)。 ) [在美国，测试已经显著下降](https://www.npr.org/2020/08/17/903175184/why-are-testing-rates-going-down) ，周转时间通常会更快，但仍可能需要 24 到 48 小时才能得到结果，因此请做好相应的计划——您需要将结果上传到 [USVI 旅游门户](https://usvitravelportal.com/) 。尽管没有要求，我还是带了一份疫苗接种卡的复印件。

## 如何使用 USVI 旅游门户网站

在你启程之前，你会希望你的测试结果得到批准。一旦你收到你的阴性新冠肺炎测试结果，进入[USVI 旅游门户](https://usvitravelportal.com/) 并点击阅读说明。系统会提示您输入您的旅行日期、航班和座位号，以及您的住宿信息。他们会问一些常规的新冠肺炎安全问题，然后指导你上传你的阴性新冠肺炎结果。大多数结果是以数字形式发送的，但是如果你收到一份硬拷贝，拍张照片或者扫描一下。

表格提交后，您将收到一封确认电子邮件。在一到两天内，你会收到通知，你已经获准抵达。传送门送你一个绿色二维码，下飞机时随手可得。你需要这个许可才能进入这个岛，所以在起飞前要确保你有这个许可。
