# 如何查看任何 iPad 应用程序的所有打开的窗口

> 原文：<https://lifehacker.com/how-to-view-all-open-windows-for-any-ipad-app-1847713845>

当你在 iPad 上使用 Safari 这样的应用程序时，单个窗口是远远不够的。虽然在 iPad 上创建新窗口和窗口配对比以往任何时候都容易，但是*找到*窗口完全是另一个问题。但这一切都很好——ipad OS 15 中有一个新功能可以让你更容易做到这一点。这叫书架。

## 如何使用工具架查找 iPad 应用程序的所有打开的窗口

你可以把 Shelf 想象成 Dock 的一个版本，但是仅限于一个特定的应用程序。当您为一个应用程序打开多个窗口时，工具架会自动显示在屏幕底部。它停留在那里几秒钟，然后消失——但你可以把它带回来:点击屏幕顶部的新的 [多任务菜单按钮](https://lifehacker.com/every-multitasking-feature-in-ipados-15-worth-knowing-a-1847664365) (三个点)。(这只在你当前正在查看全屏窗口时有效。)

然而，使用 Dock 是查看所有打开窗口的一种更可靠的方式。从屏幕底部向上滑动以显示 Dock，并点击屏幕上已经打开的应用程序的图标。或者，您可以点击并按住应用程序图标，然后选择“**显示所有窗口**”选项。

一旦货架打开，你可以点击任何窗口切换到它。在窗口预览上向上滑动将关闭它。

## 如何使用应用程序切换器找到应用程序的所有打开的窗口

虽然搁板功能很棒，但它并不直观——而且你只能在几秒钟内看到窗口的预览。找到一个应用程序所有打开的窗口的更可靠的方法是使用可靠的旧应用程序切换器。在你的 iPad 上，从底部的主工具栏向上滑动并按住一秒钟以打开应用切换器。

在这里，您将看到所有打开的应用程序和窗口的列表。如果一个应用程序打开了多个窗口，你会在它旁边看到一个小小的**窗口**图标(看起来像两个层叠的方块)。

当你点击它，你现在只能看到给定应用程序的所有窗口。这将包括拆分视图对、滑动窗口和全屏窗口。

你可以点击一个窗口切换到它。如果你想关窗，只需向上滑动。
