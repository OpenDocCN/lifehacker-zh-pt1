# 您可以用一个命令摆脱 Windows 膨胀软件

> 原文：<https://lifehacker.com/you-can-get-rid-of-windows-bloatware-with-one-command-1848707156>

Windows 一直有一个臃肿软件的问题(即使最近三星的 [在那个部门摘得桂冠](https://lifehacker.com/how-to-turn-any-android-smartphone-into-a-google-pixel-1833921883) )。膨胀软件是一个总称，指的是你不需要的，也从来没有要求过的应用程序、服务和程序，Windows 预装了一大堆。



随着 Windows 11 的推出，情况有所改善，微软已经开始削减其安装介质的脂肪。但这仍然是一个问题，像 Skype，3DBuilder，Asphalt 8，Microsoft News，Feedback Hub 等应用程序都是默认安装的(并占用系统空间)。

移除膨胀软件在两个方面帮助你:你将恢复一些存储空间*和*加速你的计算机。如果你使用的是低端机器或旧电脑，或者你只是想运行一个更轻便的操作系统，本指南可以帮助你用一个命令解决这个问题。

传统的移除臃肿软件的方法需要追踪那些不需要的应用程序，并逐个卸载它们。在某种程度上，这仍然有效，只要你能找到在**设置** > **应用** > **应用&功能**中列出的应用。在这里，找到一个膨胀软件 app，点击三点**菜单**按钮，选择“**卸载**选项。确认并遵循流程。

但是这对于微软潜入操作系统的隐藏应用程序、系统应用程序和后台进程不起作用——这就是自动化脚本可以救命的地方。开发者 Sycnex 的[Windows 10 解封脚本](https://github.com/Sycnex/Windows10Debloater) 在 Windows 10 和 Windows 11 上都可以工作，并给你一个点击选项来删除所有膨胀软件，或膨胀软件列表的定制子集。

在开始之前，您应该了解从 GitHub 或其他来源运行脚本的内在风险，尤其是那些请求提升权限的脚本。这个特殊的脚本需要管理权限，如果这个脚本是恶意编写的或者写得不好，可能会对您的系统产生负面影响。

在撰写本文时，还没有任何关于这个大约四年前的脚本被破坏或产生意外影响的报告。也就是说，潜水前保持谨慎总是很重要的。在你的电脑上运行脚本之前通读它们，看看它是否真的执行了它所宣传的过程。如果你看到任何奇怪或可疑的行为，比如更改隐私和安全设置或添加新用户，最好避免运行该脚本。

要开始，进入**开始**菜单，搜索“ **Windows PowerShell、**，右键点击结果，选择“**以管理员身份运行**”

在 PowerShell 窗口中，键入下面的命令，然后按 Enter 键。它会自动从 GitHub 库下载文件，并创建一个系统还原点，以防出现问题。

`iwr -useb` [`https://git.io/debloat|iex`](https://git.io/debloat|iex)

输入这个命令将启动一个新程序，在那里你会看到禁用 Cortana、One Drive 等选项。

点击“**自定义阻止列表**按钮，选择要删除的膨胀软件。保存文件，然后点击**中的**按钮删除带有自定义阻止列表的软件。

该脚本将运行，并将卸载选定的应用程序。一旦完成，重启你的电脑，你将会运行得更精简，更有效率。
