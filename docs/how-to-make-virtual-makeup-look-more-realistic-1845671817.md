# 如何让虚拟化妆看起来更逼真

> 原文：<https://lifehacker.com/how-to-make-virtual-makeup-look-more-realistic-1845671817>

虚拟滤镜在社交媒体上早已流行，但在加速的数字世界中，化虚拟妆是为了 [而不是自拍](https://techcrunch.com/2020/11/12/loreal-rolls-out-a-line-of-virtual-makeup/) 。从数字聚会到专业会议，在视频应用程序中找到皮肤平滑功能或使用滤镜来改变你的外表变得越来越常见。如果你想改变自己的外表，像疫情时代之前那样经常出门，这里有一些让虚拟化妆看起来尽可能真实的建议。

### 选择正确的过滤器样式

就像常规化妆一样，你想找到适合自己 [肤色](https://www.colorescience.com/blogs/blog/how-to-determine-your-skin-tone-before-buying-face-products) 和 [面部结构](https://hair-and-makeup-artist.com/facial-anatomy-proportions/) 的颜色。一些过滤器可以覆盖整个头部，改变你脸部的形状和颜色，这可能会泄露你的秘密。如果你在寻找一个完整的面部和干净的外观，可以考虑使用实际的遮瑕膏或粉底霜搭配简单的滤镜。

### 考虑你的环境

你要去参加一个专业会议吗，在那里你可能想要一个更微妙的外观？或者你正在参加一个虚拟的婚礼或聚会，你想更喜庆？选择滤镜时请考虑您的设置。

例如，避免在专业设置中使用 [增强眼睛](https://www.google.com/imgres?imgurl=https%3A%2F%2Fsnaplensfinder.com%2Fassets%2Flens%2Fbig-eyes-1%2Fbig-eyes-1_meta.jpg&imgrefurl=https%3A%2F%2Fsnaplensfinder.com%2Flens%2Fbig-eyes-1%2F&tbnid=DUg2m_sz5o4nbM&vet=10CNUBEDMoxwFqFwoTCICNitvtgO0CFQAAAAAdAAAAABAC..i&docid=EMm8rQ2iwltinM&w=220&h=220&q=filters%20that%20make%20your%20eyes%20big&ved=0CNUBEDMoxwFqFwoTCICNitvtgO0CFQAAAAAdAAAAABAC) 的滤镜。参加聚会可能会很有趣，但你可能不想在求职面试时看起来像个卡通人物。一个简单的眼影，一个小睫毛，和一点红润的脸颊看起来很自然，适合任何员工会议或聚会；你可以负担得起一个更大胆的颜色鲜艳的过滤器，就像一个虚拟的聚会。如果你的目标是一个真实的外观，那么也不要穿任何有光泽或金属质感的，或者任何看起来有点像贴纸的东西。

### 找到你的角度

试用新的滤镜时，请注意滤镜是如何随着面部移动的。笑得很开心，眯着眼睛，上下左右摇头。有延迟吗？你不想点头，然后发现你的睫毛在你的额头上，所以在表演之前测试一下你的角度和动作，这样你就能知道会发生什么。

此外，尽量和相机保持一段距离:你想坐在一个 [中特写和一个](https://www.bhphotovideo.com/explora/video/tips-and-solutions/filmmaking-101-camera-shot-types) 特写之间。再靠近一点，滤镜的瑕疵就会变得清晰，幻觉也就消失了。

考虑一下你会在聚会或会议上呆多久。如果是三个小时的演示，使用数码化妆可能很难管理。如果揉眼睛或者捂嘴打哈欠，手指就会像手偶一样沾满口红和眼影。就像你会在意化真实的妆一样，记住你化的是数码妆，而且会变。
