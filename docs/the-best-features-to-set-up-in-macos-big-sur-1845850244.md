# 在大苏尔马科斯设置的最佳功能

> 原文：<https://lifehacker.com/the-best-features-to-set-up-in-macos-big-sur-1845850244>

好像你*刚刚*安装了最新的操作系统版本，然后他们又发布了一个新版本。随着更新而来的是新的设计、工具和一张与自然相关的背景照片。当然，还有一些很酷的新功能。如果你刚刚更新到 macOS 大苏尔，这里有最好的新功能可以玩。

最大的变化是新的控制中心，它与最新 iOS 中 iPhones 和 iPads 上的控制中心非常相似。使用右上角的切换开关快速访问 wifi、蓝牙、勿扰模式等控制。您现在还可以添加小部件(或从第三方应用程序下载小部件)来进一步定制您的控制中心。与最新 iOS 相似的还有对 Messages 的更新，现在可以让你在群聊中添加线索、大头针和照片。

Safari 也进行了重大改革。它现在可以让你用背景照片定制你的主页，并增加了一个新的隐私报告，让你可以轻松地看到哪些追踪者在网上跟踪你。
