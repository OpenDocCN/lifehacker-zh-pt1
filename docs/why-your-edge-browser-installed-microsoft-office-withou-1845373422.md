# 为什么你的 Edge 浏览器没有经过询问就安装了微软 Office

> 原文：<https://lifehacker.com/why-your-edge-browser-installed-microsoft-office-withou-1845373422>

edge Chrome[最开始是](https://lifehacker.com/our-favorite-tips-for-microsofts-new-edge-chromium-brow-1841041653) 作为 [可敬的替代品](https://lifehacker.com/the-best-browsers-for-ad-blocking-that-arent-chrome-1835130559) 到 [谷歌 Chrome](https://lifehacker.com/yes-you-should-quit-using-google-chrome-1844408232) 在 Windows 上，但是没多久微软就把它变成了一个累赘。更重要的是，看起来 Edge 现在是在你的电脑上安装(甚至更多)微软软件的载体——当然，不需要你主动要求。

我们不喜欢臃肿的软件，或者你的电脑或智能手机上的那些预装应用。其中一些应用程序是值得的，但大多数只是占用空间，在某些情况下无法完全删除。一些公司在软件膨胀方面比其他公司更糟糕，但是微软因在 WindowsT3 中植入额外的软件而臭名昭著。现在，Windows 内部人士在测试最新的 Edge Chromium 预览版时，发现浏览器未经许可安装了 Microsoft Office web apps。

到目前为止，这些报告仅来自 Windows 内部人士，但这些后门安装不太可能是早期发布的错误。这不仅仅是微软的问题。例如，Chrome 也可以在没有任何通知的情况下安装谷歌文档和其他 G Suite 应用程序。

然而，与 Chrome 不同，Edge 是作为操作系统的默认浏览器预装的。这意味着一些用户可能最终会使用一些他们没有要求的 Office 应用程序。好消息是，Microsoft Office 的这些渐进式 Web 应用程序版本可以轻松删除:

1.  打开控制面板应用程序(使用开始菜单搜索找到它)。
2.  点击**下的**【卸载程序】**【程序&功能】**
3.  滚动列表并单击选中您要删除的应用程序，然后选择**“卸载”**
4.  按照屏幕上的说明完成该过程，然后对每个要删除的应用程序重复该过程。

移除 Edge 秘密安装的 pwa 很容易，但这不应该是唯一的选择。希望微软重新考虑，至少给人们说不的选择。否则，Edge Chromium 是谷歌 Chrome 等资源密集型浏览器的一个非常可靠的替代品。我们能不能不要那些多余的东西？
