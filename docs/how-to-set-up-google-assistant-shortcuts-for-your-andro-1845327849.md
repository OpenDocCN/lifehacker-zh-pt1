# 如何为你的 Android 应用程序设置谷歌助手快捷方式

> 原文：<https://lifehacker.com/how-to-set-up-google-assistant-shortcuts-for-your-andro-1845327849>

谷歌助手现在可以与一些第三方应用程序配合使用，这很好。所谓“配合”，我的意思是你现在可以在你的 Android 数字助理上发出几个命令，以调用该平台一些顶级应用程序的各种功能。这是一个功能的有限亮相，应该看到广泛的吸引力，哦，几年前。

Watch

尽管如此，生产力就是生产力。假设该功能的推出让你大吃一惊——确保你已经将你的 Android 更新到了最新版本的操作系统，以及谷歌 Play 商店中的任何应用程序(这不会有什么坏处)——一些最受欢迎的 Android 应用程序现在可以通过快捷方式进行设置。

哪些应用？很高兴你问了。现在没有一个有限的列表，所以假设我们在谈论的应用程序有数百万人使用，并且已经存在了很多很多年。Twitter、脸书、YouTube 和 Gmail 等家喻户晓的应用。

幸运的是，有一个简单的技巧，你可以用来查看支持这些快捷键的应用程序，什么快捷方式*是*，以及你需要说出什么语音命令来激活它们。简单地捏一下你的手机，启动谷歌助手(或者你启动它的其他方式)，然后说**“我的快捷方式。”**当你这样做时，你将被带到一个如下所示的屏幕:

向下滚动一点，你会看到一个名为**“你的应用程序的所有快捷方式**”的部分。**这是金矿。轻按任何应用程序，你就会看到你可以通过谷歌助手召集的行动的简短列表。例如，以下是 Twitter 的部分列表:**

**点击你想要激活的任何快捷方式旁边的**加号图标**，然后你会在每个应用列表的**“你的快捷方式”**部分找到它们。点击这些快捷方式旁边的**铅笔图标**，将触发短语调整为您想要的:**

**既然你有了新的触发短语来启动应用程序中的动作，你甚至可以将这些绑定到谷歌的例行程序中，比如说，你需要立即每天发一条推文，作为你早上醒来过程的一部分。(真希望那不是个东西。)**
