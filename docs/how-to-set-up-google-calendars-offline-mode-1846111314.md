# 如何设置谷歌日历的“离线模式”

> 原文:[https://life hacker . com/how-to-set-up-Google-calendars-offline-mode-1846111314](https://lifehacker.com/how-to-set-up-google-calendars-offline-mode-1846111314)

谷歌正在为其基于网络的谷歌日历版本恢复“离线模式”——至少对工作区用户是这样。假设你之前已经将谷歌日历连接到了众多可用的智能手机或桌面应用中的一个，那么大多数人都不应该需要这个，但为了以防万一，你需要知道你的行程，备份一下也无妨。

Watch

我将带你完成启用离线模式的过程，但要知道谷歌仍然像往常一样向谷歌日历用户推出这一功能。如果你现在还不能安装，一两周内再试一次。如上所述，以下是如何在谷歌日历中设置离线模式。

首先在桌面浏览器中导航到 [**谷歌日历**](https://calendar.google.com) ，点击右上角的齿轮图标。点击**设置**，看看**快捷键**下是否有**离线**选项，像这样:

点击该选项，然后点击**“打开离线日历”**即可。您的日历将开始同步到您的桌面，您可以通过谷歌日历主界面右上角的图标查看进度。完成后，您将看到一个状态，显示“准备脱机”

如果你在失去互联网连接的情况下尝试访问谷歌日历，你会看到一个小提示，告诉你你正在离线模式下工作。你可以查看你的活动，但你的活动会受到一些限制，比如你不能创建或编辑活动，不能给客人发电子邮件(显然)，也不能访问你的任务或提醒列表。同样， [谷歌](https://support.google.com/calendar/answer/1340696?hl=en) 写道:

> 如果您清除电脑的缓存图像和文件，也将清除对Google日历的离线支持。再次打开Google日历脱机版以脱机使用日历。

我们希望谷歌向谷歌日历的普通用户推出这一有用的功能，而不仅仅是那些将谷歌日历作为工作区设置的一部分的用户。还不知道这是否会发生，或者什么时候会发生，但是看起来这对于谷歌来说是一个足够容易的改变。
