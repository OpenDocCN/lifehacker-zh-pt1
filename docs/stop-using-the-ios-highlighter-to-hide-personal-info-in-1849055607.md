# 停止使用iOS荧光笔在照片中隐藏个人信息

> 原文:[https://life hacker . com/stop-using-the-IOs-highlight-to-hide-personal-info-in-1849055607](https://lifehacker.com/stop-using-the-ios-highlighter-to-hide-personal-info-in-1849055607)

你不应该遵循抖音在你的订阅中给出的每一条建议，但最近流传的这条技术建议值得注意:不要使用iOS荧光笔来屏蔽截图和其他图像中的重要信息。

Watch

正如抖音账号 [lifehack.america](https://www.tiktok.com/@lifehack.america/video/7102823287310568750) (无关系)所展示的那样，其他iOS用户可以撤销你对图像所做的某些编辑。例如，假设您使用股票iOS照片应用程序中的荧光笔工具来屏蔽信息。另一个用户可以在照片中打开标记的图像，，然后编辑曝光和亮度来“撤销”荧光笔标记，看看你隐藏了什么。这个技巧是可行的，因为荧光笔工具实际上并不是直接在图像上绘制不透明的像素，而是改变已经存在的像素的色调和暗度。

 [https://lifehacker.com/embed/inset/iframe?id=tiktok-7102823287310568750&autosize=1](https://lifehacker.com/embed/inset/iframe?id=tiktok-7102823287310568750&autosize=1) 

虽然这一惊人的发现可能会让用户在下次使用荧光笔工具阻止电话号码、用户名、银行卡详细信息和其他敏感信息时犹豫不决，但这并不一定会排除通过绘制图像来保持隐私的方式。

正如lifehack.america的一些用户在评论中指出的那样，只有当下面的信息被荧光笔工具的一条线隐藏时，辉煌和曝光技巧才有效。多次敲击更容易混淆你想要隐藏的东西。注意，“多笔画”并不意味着一气呵成地乱涂乱画——你必须通过触摸屏幕、拖动手指，然后一遍又一遍地抬起手指来重复绘制线条，以创建多个笔画。

## 轻松可靠地隐藏iPhone上的敏感信息

然而，还有更可靠的方法来隐藏你的信息，如 [照片应用程序的矩形工具](https://lifehacker.com/9-useful-features-for-anyone-who-takes-screenshots-on-t-1848264725/slides/3) ，甚至用表情符号来阻止东西。您还可以裁剪照片中的图像，这样就不会出现个人信息。这里还有像 [MaskerAid](https://apps.apple.com/app/maskeraid/id1590163828) 或 [Signal](https://lifehacker.com/how-to-quickly-blur-faces-in-photos-1843908720) 这样的第三方应用程序，可以让你直接在图像上模糊或绘制(而不是在新的“图层”上)，以安全地遮挡你试图隐藏的任何东西。当然，你也可以将照片上传到图像编辑工具，如Photoshop或GIMP，绘制或裁剪敏感信息，然后保存为JPEG或PNG，以合并图层，防止其他人撤销你的编辑。

无论你使用什么工具，只要确保你掩盖的任何东西都被完全涂抹，因为有 [工具可以很容易地显示部分遮挡的文本和“未模糊”的图像](https://lifehacker.com/stop-blurring-your-sensitive-information-in-screenshots-1848626457) 。或者，你知道，首先不要发送或上传任何有重要数据的东西。
