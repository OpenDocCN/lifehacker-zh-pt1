# 如何在微波炉里做出完美的土豆泥

土豆泥就像你希望的那样复杂。想剥皮吗？太好了。想吃米饭吗？也很棒。不做那些事？你猜对了——仍然很棒。我想我从来没有做过两次同一碗土豆泥。从 烹饪方法，到插件，到土豆本身，有太多的变量，调整这些变量是很大一部分乐趣。

不过最近，我非常喜欢用微波炉煮土豆。它很快。这很容易。它不会把我的厨房——一天比一天暖和——变成蒸汽房。土豆在自己的水分中蒸，所以也不涉及排水。唯一的限制因素是你能放进微波炉的土豆数量。

相反，这是为几个人做少量土豆泥的完美方式，尤其是如果你只有几个土豆的话。(拜托。我求求你:不，谁会想要一小份土豆泥？？？下面评论下来。我太累了。)

一些微波捣碎器的食谱让你融化黄油并加热任何乳制品。我从不这样做。油炸土豆的热量完全融化了黄油，我加入的牛奶或奶油的量不足以使它们冷却。老实说，从微波炉里出来的土豆非常热——稍微冷却一下是可以的。你在下面看到的食谱是裸骨(或裸土豆)，但那是设计好的。你可以随意放大或缩小，并根据自己的需要进行调整和重复。(如果你需要灵感，试试 [一点洋葱蘸](https://skillet.lifehacker.com/just-add-store-bought-french-onion-dip-to-mashed-potato-1839621708) 然后从那里开始。)

### 两人份的微波炉土豆泥

**配料:**

*   大约一磅土豆，切成1英寸大的块(我通常用4个洗净的、未去皮的尤康丝)
*   两大撮盐
*   2汤匙黄油(每个土豆大约1/2汤匙)
*   1汤匙一半一半(或奶油或牛奶)
*   2茶匙蛋黄酱(可选)

将土豆放入微波炉安全的碗或盘中，盖上保鲜膜，在顶部开一个半英寸的口子。将土豆用微波炉加热9-12分钟(取决于你有多少土豆)，直到它们可以很容易地被叉子刺穿。将它们从微波炉中取出，用一把大叉子、一把大木勺或一个马铃薯捣碎器粗略捣碎，让多余的水分蒸发掉。

加入你的黄油并捣碎直到它融化，然后加入你的那一半&和蛋黄酱并再次捣碎。即使你不喜欢蛋黄酱，我也推荐它——它真的有助于乳化疏水性(脂肪)和亲水性(水状)。尝一尝你的土豆，然后根据你的喜好调味和装饰。
