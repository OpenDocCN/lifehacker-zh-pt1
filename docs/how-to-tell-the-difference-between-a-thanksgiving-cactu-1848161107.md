# 如何区分感恩节仙人掌和圣诞节仙人掌，以及为什么这很重要

> 原文：<https://lifehacker.com/how-to-tell-the-difference-between-a-thanksgiving-cactu-1848161107>

尽管圣诞仙人掌有着喜庆的名字，但它是一年四季都很棒的室内植物，即使在七月中旬也不会显得格格不入。它们还能活得很长，如果得到适当的照顾，它们可以代代相传。



但是如果我们告诉你，你认为是圣诞仙人掌的肉质植物实际上可能是一种完全不同的植物呢？不是一般的植物——T2 感恩节仙人掌？是真的。它们都是在深秋或初冬开花的仙人掌，并以年末假期命名，但园艺中心经常将感恩节仙人掌误标为圣诞仙人掌。

《农民年鉴》 的一篇文章解释了感恩节仙人掌和圣诞节仙人掌之间的区别，以及为什么知道哪一个更重要。以下是需要了解的内容。

## 感恩节仙人掌大战圣诞仙人掌

我们先从外观上的一些快速差异开始，承蒙 [农民年鉴](https://www.farmersalmanac.com/thanksgiving-cactus-christmas-cactus-73148) :

### 感恩节仙人掌

*   **叶**:形似蟹爪，每边有二至四个锯齿状的节段，节段上有尖刺。最后一段的末端略凹。
*   **花**:从叶尖或叶段连接处生长；看起来像另一朵花里面的长管；大多是柔和的颜色，包括红色、粉色、桃色、紫色、橙色或白色。

### 圣诞节仙人掌

*   **叶子**:边缘较圆，呈扇形；尖端略微弯曲。
*   **花**:颜色通常为粉色或白色。

## 为什么它很重要

你可能已经猜到了，感恩节仙人掌会在感恩节前后开花——大约比圣诞仙人掌开花早一个月。知道你手上的仙人掌是哪种类型将有助于你确定什么时候你可以期待花朵的出现。

嗯，至少在大部分情况下:感恩节和圣诞节仙人掌偶尔会在三月和五月之间开花，尽管 T2 农民年鉴指出春天的花不会像年底那么多。
