# 如何为每个房间挑选最好的灯泡

> 原文:[https://life hacker . com/how-to-pick-the-best-bulb for-every-room-1713809049](https://lifehacker.com/how-to-pick-the-best-light-bulb-for-every-room-1713809049)

房间的照明不仅仅是装饰的一部分；它可以影响从你的 [睡眠时间表](http://lifehacker.com/sleep-better-at-night-by-increasing-your-daytime-light-1697399439) 到你的 [脑力](http://lifehacker.com/blue-light-may-be-better-than-coffee-for-boosting-brain-1472598981#_ga=1.187625000.1268082208.1431441811) 的一切。所以，你想让你的家——这个你可能要在生活中度过很长一段时间的地方——有良好的照明是可以理解的。但是有这么多不同的选项可供选择！

Watch

某些类型的照明有特定的用途，对于你的家来说，你需要合适的类型，这取决于你房间的用途。为了优化你家的照明，首先考虑你如何使用每个房间。

### **考虑每个房间的功能**

一般来说，照明功能分为三类:环境照明、任务照明和重点照明。

*   **普通照明或环境照明**作为房间的整体照明。它照亮了整个房间，被认为是房间的“自然光”你可以使用枝形吊灯、吊灯、轨道灯或壁灯来营造充满房间的环境光线。
*   **任务照明**照亮工作或阅读区域。您希望这种照明比环境照明更亮，因此对比度将光线聚焦在指定区域。台灯和橱柜下的厨房灯是常见的工作照明选择。但是吊灯和轨道灯也可以用于任务照明，但这取决于你如何在房间里分层照明，以及你的灯泡有多亮(我们稍后会谈到)。
*   **强调** **照明**突出一个特定的区域，比如一件艺术品或者一个书架。它通常会在物体周围产生阴影以获得戏剧性的效果。壁灯和景观灯是常见的重点灯。

要使用这些技巧为你的房间提供合适的照明，考虑一下你将如何使用每个房间，以及房间里是否有你想要强调的东西。然后，开始分层。 [HGTV推荐](http://www.hgtv.com/remodel/mechanical-systems/3-basic-types-of-lighting) 你从环境照明开始，然后考虑任务和重点照明:

> 罗得岛州普罗维登斯的照明设计师Markus Earley说:“我喜欢在规划房间照明时从一般到特殊。”然而，对于以任务为主的房间，如家庭办公室，一些设计师首先关注任务照明。在兼作照片或艺术画廊的走廊里，重点照明可能是第一个考虑的因素。

然后想想房间里的光线会射向哪里。先不要担心灯具，只要想想你希望不同的灯光落在房间的哪个位置。如果你不确定从哪里开始，考虑这些一般性的、逐个房间的建议:

*   **客厅**:除了环境光， [真实简单建议](http://www.realsimple.com/home-organizing/decorating/decorating-tips-techniques/interior-lighting) 在房间的一个角落使用重点灯。专注于一个物体，比如一件艺术品或者一把椅子。
*   **厨房**:在头顶上增加你的环境光，然后增加较低的工作照明来照亮你工作的柜台空间。如果可能的话，水槽也是增加工作照明的好地方。
*   **卧室**:在你卧室的床头柜上安装任务照明是很常见的。Real Simple还建议让光线远离床。他们建议将头顶的环境光从床上斜向更衣区。
*   浴室:浴室照明可能很棘手。您需要为镜子提供任务照明，但是头顶上的任务灯会产生阴影。考虑点亮镜子的任意一侧。然后，用头顶上的环境灯充分照亮房间。

当然，如果你是一个租房者，你可能无法对你家或公寓的一些照明位置做太多的调整。但是这些通用的指导方针可以给你一个如何使用你所拥有的东西的想法。

### **选择合适的灯泡**

你的灯泡就是你的光源，所以灯泡的类型决定了灯光的外观。不同的灯泡性能不同，有四种基本类型:

*   白炽灯:这些是我们大多数人已经使用了几十年的传统灯泡，它们已经开始被淘汰，取而代之的是更节能的选择。它们发出温暖的光芒。
*   紧凑型荧光灯(CFL):这种灯比白炽灯 节能75%[。它们的寿命也更长。它们通常发出冷色调，但你可以在一系列亮度水平和温度中找到它们。值得注意的是，节能灯确实含有汞，尽管数量很少，但仍然需要更加小心地处理和处置，国家地理杂志说。](http://www.earthsfriends.com/cfl-vs-incandescent/)T9】
*   led灯:这种灯和节能灯一样节能，但是寿命却是节能灯的三倍。它们过去主要用于任务照明，因为它们只能提供刺目的直射光，但像节能灯一样，它们已经从 [发展到](http://lifehacker.com/the-cree-warm-white-is-a-great-led-bulb-feels-like-an-476462606) 。他们现在提供了和白炽灯一样的外观，但是他们更有效，摸起来不那么热，并且持续很长时间。由于这些原因，它们也可能更贵，但 [有实用折扣](http://www.howtogeek.com/220220/psa-you-can-save-lots-of-money-on-led-light-bulbs-with-utility-rebates/) 。
*   卤素:它们发出明亮的白光，类似于自然光。非常适合任务照明。他们还 [比白炽灯泡](http://pressroom.gelighting.com/facts/cfl-vs-halogen-vs-led-232394) 节省10-20%的能源。

还有其他类型的灯泡，但这些是你在家里最常用的。在这篇文章中，我们只关心灯泡看起来如何。如果你想了解更多关于T2如何工作的信息，看看美国照明协会的这篇有用的帖子。美国联邦贸易委员会现在要求灯泡包装包括亮度信息，灯泡将持续多久，它使用多少能源，以及它是否符合能源之星的要求。(这里的 [是来自NRDC](http://www.nrdc.org/energy/lightbulbs/files/lightbulbguide.pdf) 的一个样本标签)。)

当讨论外观时，我们关心的是亮度——亮度是用流明来衡量的；和浅色外观——这是用开尔文来衡量的。

#### **‘流明’等于亮度**

流明越多，灯泡越亮。一个典型的家用灯泡大约产生800流明，相当于60瓦。那么每个房间需要多少流明呢？这取决于你的房间有多大，你的墙壁是什么颜色，当然，还有你喜欢的灯光强度。 [使用这个计算器](http://www.charlstonlights.com/led-light-requirement-calculator) 根据你的家庭和喜好得出一个具体的数字。但是这里有一个一般的细目， [via HouseLogic](http://www.houselogic.com/home-advice/lighting/room-lighting-design-tips/) :

*   **厨房**:5000-10000总流明
*   **浴室**:4000-8000总流明
*   **卧室**:2000-4000总流明
*   **客厅**:1500-3000流明
*   **餐厅**:3000-6000流明
*   **家庭办公室**:3000-6000流明

请记住，这些都是粗略的估计，并考虑了每个房间不同类型的灯泡和照明选项的使用。例如，厨房通常更明亮，包括环境光和作业照明的混合。卧室和客厅通常不太明亮。

如果你知道如何用瓦数来照明你的房间，这里有一个瓦数到流明的小抄 。

#### **‘开尔文’等于光的出现**

除了亮度，你还要考虑光线的色温。几年前，节能灯还不太好，因为它们大多只能发出非常蓝的冷光。但是它们已经走了很长的路，你现在可以找到它们更温暖、更黄的色调。包装应该告诉你光的色温，从暖到冷，用开尔文测量。开尔文越高，光线越冷。照明博客 [电池+灯泡解释了](http://blog.batteriesplus.com/2013/seeing-things-in-a-different-light#sthash.JbtOpl1u.dpuf) 灯泡盒通常指的是不同的灯泡温度。还补充了这些温度最适合在您家中使用的地方:

*   **柔白/暖白(2700开尔文):**最适合卧室和客厅；给他们一种传统的温暖舒适的感觉。
*   **亮白/冷白(4100开尔文):**厨房、浴室或车库最佳；让房间看起来更白，更有活力。
*   **日光(5000-6000开尔文):**浴室、厨房、地下室最佳；适合阅读、复杂的项目或化妆-提供最大的色彩对比。

你也可以试试这个来自能源之星的 [互动工具](https://www.energystar.gov/productfinder/product/certified-light-bulbs/results) ，它会为每个房间的不同照明选项建议使用哪种灯泡。

对灯泡的工作原理有一个基本的了解会有所帮助。这样，你可以挑选你喜欢的灯泡。此外，如果你想改变照明强度，调光器是一个很好的选择。我们已经讲述了如何自己安装调光开关。

### **挑选你的固定装置**

既然你已经知道了你的照明功能，你想要的亮度，你喜欢的温度，是时候选择最佳类型的灯具来优化所有这些因素了。以下是一些常见的灯具，以及它们通常是如何(以及在哪里)使用的:

*   **天花板固定装置**:非常标准的环境照明。 [房屋设计师说](http://www.thehousedesigners.com/articles/choosing-right-lighting-fixture.asp) 它们是门厅、走廊、卧室、工作区、楼梯的理想选择。在走廊里，他们建议每隔8到10英尺放置一个灯具，以获得足够的照明。
*   吊灯:当用于一般照明或环境照明时，它们最好用在餐厅或卧室。
*   壁挂式灯具:这些通常是壁灯。它们可用于任何房间的环境照明、任务照明或重点照明，具体取决于你的摆放位置和使用的灯泡种类。
*   **吊灯**:用于任务或一般照明，它们悬挂在天花板上，并配有灯罩以避免眩光。它们最适合放在餐桌、工作台面或其他工作区域。
*   **嵌入式照明**:同样，嵌入式照明可用于任何地方的普通照明、任务照明或重点照明。这完全取决于它们的亮度和位置。
*   轨道照明:你也可以用轨道照明做几乎任何事情。它的用途非常广泛，因为你可以经常移动单个灯具，将它们指向你想要的任何方向。这可能是为了突出一些艺术品，或者你可能只是用它们来照亮整个房间。
*   **台灯**:非常适合客厅的重点照明或卧室的任务照明。

[照明研究中心提供了一些附加灯具的详细信息](http://www.lrc.rpi.edu/patternbook/equipment/index.asp) ，包括如何安装它们以及它们有什么样的照明效果。记住:不同的灯具需要不同类型的灯泡。所以当你挑选灯具时，考虑一下它需要什么类型的灯泡。

这与其说是照明规则，不如说是设计规则，但在选择合适的灯具时，你也要考虑尺寸。太小或太大的灯具会让你房间的比例看起来很奇怪。 [LightsOnline列出了一些选择合适尺寸夹具的指南](http://www.lightsonline.com/how-to-choose-the-right-sized-fixture/) ，但这里有一些亮点:

*   **台灯**:一个很好的经验法则是，台灯的高度不应该超过台灯所在位置的1.5倍，灯罩直径不应该比桌面宽。
*   **吊灯和吊灯**:测量你桌子的宽度或直径。然后从这个数字中减去12”。这是吊灯宽度或直径的最大限制。请记住，设计复杂的灯具实际上会显得更大，所以如果这是吸引你眼球的地方，你需要稍微缩小你的最大宽度。假设你有8英尺高的天花板，灯具的底部应该悬挂在桌面上方30到36英寸之间。但是如果你的天花板更高，建议是每增加一英尺的天花板就在桌子上方增加3英寸。
*   烛台:你越靠近烛台所照亮的东西，烛台就应该越小。例如，在靠近镜子的浴室里，选择9-10英寸左右的小镜子。在浴室里，安装壁灯，间隔36到40英寸，在镜子的侧面，距离水槽的中心线18英寸。如果壁灯有灯罩，把灯罩的底边放在眼睛水平线以下一点(离地面60到68英寸)。

对一些人来说，家居设计是自然而然的，装修时很容易注意到你的照明。对于我们其他人来说，可以遵循一些规则和指导方针，这些应该会让你朝着正确的方向开始。
