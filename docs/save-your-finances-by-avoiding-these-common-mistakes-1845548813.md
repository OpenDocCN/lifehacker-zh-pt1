# 通过避免这些常见的错误来节省你的资金

> 原文：<https://lifehacker.com/save-your-finances-by-avoiding-these-common-mistakes-1845548813>

当你不太注意你的财务规划时，很容易犯错误，而且有些错误比其他错误代价更大。以下是一些常见的财务失误以及如何避免它们。



### **没有预算**

[预算是你最重要的个人理财工具](https://twocents.lifehacker.com/you-really-do-need-a-budget-1843440246) : I 这是一个根据你的收入列出支出清单的支出计划，帮助你避免超支和负债。预算能让你控制现金流，让你更容易为应急基金或其他长期财务目标(如买房)存钱。通过坚持预算，你会更快地还清债务，避免未知开支的压力，并且能够毫无愧疚地挥霍。

### **没有按时支付信用卡账单**

错过或延迟信用卡付款可能会导致更高的利率，滞纳金，甚至失去你的信用卡。错过付款还会损害你的信用评分，这对于获得贷款或信用卡至关重要，尤其是那些利率较低的贷款或信用卡( [你的分数的 35%由你的付款历史决定](https://twocents.lifehacker.com/the-5-factors-behind-your-credit-score-1797277397) )。你的信用评分也被房东、雇主、保险公司用来确定你的可靠性。为你自己省下许多麻烦，让你的付款自动化 。

### **没有得到保险**

财务规划的基石是最小化风险，这包括保护你已经拥有的资产。幸运的是， [有了保险](https://lifehacker.com/what-types-of-insurance-do-i-really-need-1488536904) 你就不用担心突发事件或自然灾害会损坏或摧毁你的财产或影响你未来的收益。许多人放弃保险来省钱。然而，跳过保险是一个糟糕的主意。一辆撞坏的汽车、被盗的财产或终身残疾可能比你每月较少的保险费要昂贵得多。

### **未计入非月度费用**

每月和每天的日常开销很容易追踪——电话账单、房租和汽车付款都是定期的，很难错过，尤其是如果你已经注册了自动取款。另一方面，一次性支出——生日、节假日、假期——会悄悄降临到你身上，让你负债累累。做预算时，回头看看你以前的开销，列出所有不定期的经常性开销，比如每季度或每年一次，并将它们纳入你的预算。

### 用退休基金还债

退休储蓄通常在破产时受到保护，但如果已经提取，它们就失去了这些保护。提前退出是有惩罚的。举例来说，如果提前从 401(k)计划中提款，会有 10%的罚金；y 你提取的任何东西都必须缴税，美国国税局会根据你当前的税级自动从这笔钱中扣缴税款(通常当你停止工作和退休时，你的税率会更低)。最重要的是，你还失去了长期投资的复利，这严重限制了你退休时能提取多少钱。无论你做什么，在没有首先寻求债务减免之前，不要提取你的退休储蓄。
