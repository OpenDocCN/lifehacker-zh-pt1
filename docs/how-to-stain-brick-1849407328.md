# 如何给砖着色

> 原文：<https://lifehacker.com/how-to-stain-brick-1849407328>

砖作为一种色彩丰富的建筑材料的好处是，如果你喜欢那种颜色，就不需要处理油漆、墙纸或其他墙面材料。但是，如果你家里面或外面的砖的颜色不完全是你自己挑选的，有一个相对简单——也是永久——的方法来改变这一点。下面是染砖要知道的。



## 染色、粉刷和粉刷砖块之间的区别

在你开始给你的砖头上色之前，确保这是你真正想要做的——而不是粉刷或油漆它。虽然这三种产品都可以在室内和室外使用，但相似之处仅此而已。下面是染色、刷白、画砖的主要[:](https://757brick.com/stained-brick-whitewashed-brick-or-painted-brick/)

### 染色砖

*   染色剂被吸收到砖块中(以及所有的角落和缝隙)，有效地将它染成不同的颜色
*   给砖上色看起来不像是被粉刷过，而是让它看起来像是最初用这种颜色的砖建造的(或者至少这是目标)
*   在没有大量修补的完整或大部分完整的砖块上效果最好

### 粉刷砖

*   最终产品是改变或变白砖的自然颜色，你可以根据粉刷的厚度来控制
*   熟石灰、盐和水的混合物
*   保留砖的原始表面和纹理

### 粉刷砖

*   目标是改变砖块表面的颜色(或者至少是可见的部分)
*   通常使用 100%乳胶漆
*   油漆不会融入砖的纹理，所以你最终会在上面涂上一层光滑的厚涂层
*   因为它不像染色剂或石灰水那样粘附在砖块上，所以需要更频繁的维护

如果你真的想给你的砖头上色，我们将带你完成下面的过程。

## 如何给砖着色

你在这个项目上投入的时间和精力很大程度上取决于砖块的状况，以及你希望污渍有多深。接下来该怎么做:

### 清洁砖块

为了让污渍能够完全渗透砖块，砖块必须是干净的。

如果你正在粉刷室内砖，在桶里装满温水和少量的 T2 茶匙或 T4 温和清洁剂，用尼龙鬃毛刷擦洗这个区域，从顶部开始往下刷。

然后，尽可能少用水，尽最大努力冲洗砖块。在你开始之前，用塑料覆盖所有的家具和地板，以减少混乱。让砖块干燥 24 小时后再继续。

使用设置为 1000 PSI 的 [压力清洗机清洁**室外砖块**。小心不要损坏砖块。](https://www.thisoldhouse.com/painting/22740402/staining-brick)

### 修复任何损坏的砖块

清洁砖块时，你可能会碰到一些需要小修小补的地方，比如随着时间的推移已经变质的砂浆。如果是这样， [对](https://www.thisoldhouse.com/painting/22740402/staining-brick) 进行必要的修补，然后让砂浆干几天。

### 保护周围地区

一旦你准备好染色，花时间覆盖任何附近的表面——地板、车道、家具、植物和灌木丛等。你想要保护的东西。如果你染色的砖墙上有窗户、壁炉或其他无法移除的东西，用塑料布和胶带盖住它们。

### 混合并涂抹染色剂

首先，按照制造商的说明混合你的染色剂——这可能包括在罐子里混合颜料和溶剂(它们会随着时间的推移而分离)，然后(可能是 ) 用一些水稀释染色剂(取决于你使用的类型)。

现在，是时候染色了。室内项目用画笔**，蘸上颜料，涂在 [的砖块上，光滑、重叠、均匀的涂层](https://www.thespruce.com/how-to-stain-brick-tutorial-5115258) 。根据温度和湿度——以及污渍涂层的厚度——它应该在大约一天内干燥并固化。**

对于较小的**室外区域**，你可以使用上面的方法(如果有意义的话，加入一个滚轴)。但是对于较大的外墙，使用喷漆器可以节省很多时间。

如果你走那条路，用喷雾器装上着色剂混合物， [从大约 12 英寸远的](https://www.thisoldhouse.com/painting/22740402/staining-brick) 处，以一个小角度把它涂到砖块上。用从左到右将喷在砖块上，使其平滑、均匀。用油漆刷修补任何凹凸不平的地方，让墙壁变干。

理想情况下，你会想再做一遍——这一次，从上到下喷洒——这样你就能在两个方向都覆盖了。但是如果这是不可能的，或者，如果你不想让颜色变得更暗，你可以就此打住。否则，继续在一个方向涂抹污渍，让它干燥，然后在相反的方向涂抹另一层，直到你得到你想要的颜色。
