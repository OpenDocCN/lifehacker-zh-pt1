# 欢迎来到洛奇

> 原文:[https://lifehacker.com/welcome-to-lodge-1846454548](https://lifehacker.com/welcome-to-lodge-1846454548)

欢迎来到Lodge，Lifehacker的最新栏目探索家庭生活的方方面面——从大问题(买一套房子到底要花多少钱？)到小(每个房间最好的灯泡是什么？).

Watch

我们已经讨论这些话题很多年了。(浏览这一部分，你会发现我们的档案中已经有了成吨的故事。)我们只是还没有，嗯，给它们一个家，现在是时候了。不管是好是坏，疫情让我们的家变得比以往任何时候都重要。对我们中的一些人来说，他们已经成为我们的办公室；对另外一些人来说，他们是我们孩子的教室。我们通过视频电话与朋友交谈，在烛光下吃外卖，期待着有一天我们可以安全地回到餐馆。我们比以往任何时候都更加意识到需要一个避难所，以及一个可以进行多任务处理的高效空间。

在Lodge中，我们为那些没有一两个毛茸茸朋友的家庭提供了宠物内容。(或者一个有鳞/有羽/无毛的朋友，如果你是这么滚的话。)所以，如果你想知道为什么你的猫喜欢打翻东西，或者什么植物对你的宠物来说可能是危险的，这就是你要去的地方。(哦，如果你家里和周围有不需要的小动物，我们也会处理这些问题。老鼠最讨厌什么气味？什么能把浣熊从你的车库里引诱出来？别走开，我们会让你知道的。)

我们的发行由美国家庭保险公司赞助。也就是说，Lodge完全是一个编辑倡议——我们报道什么以及如何报道完全取决于我们自己。我们很想知道你有什么问题或者你想看什么故事，请在评论中联系我们。脱掉你的鞋子，呆一会儿。这是你的小屋。
