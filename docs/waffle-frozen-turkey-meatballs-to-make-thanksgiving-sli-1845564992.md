# 华夫饼冷冻火鸡肉丸制作感恩节滑块

我不是一个大的火鸡三明治消费者，但是感恩节三明治是一个不同的故事。当你把 [蔓越莓酱](https://lifehacker.com/we-should-eat-cranberry-sauce-all-year-1840055543) 牵扯进来的那一刻，我就对它了如指掌。不管是冷盘、烤火鸡还是火鸡汉堡，只要符合主题，我都会津津有味地狼吞虎咽。

在最近一次去杂货店的路上，我拿了一袋冷冻火鸡肉丸，不知道该怎么处理它们(它们才5美元！)。然后，昨天下午，当我从我的电器柜里拿出我的沉浸式循环器时，灵感来了。我的华夫饼干制作机坐在那里，请求加入感恩节的讨论。"我想知道我是否能把那些肉丸打碎成小块？"我问用具。他说他认为值得一试。

我的华夫饼机(其想象的个性和声音只是我自己的延伸)是正确的。我确实能够把冷冻的火鸡肉丸捣成多汁的小馅饼，有着极好的外皮，而且它们尝起来非常好，还有点蔓越莓酱(当然是放在夏威夷卷上)。

冷冻肉丸需要一点哄骗——毕竟，它们是坚硬的冷冻肉球——,所以摇晃它们不像我们在这一类别中的其他产品那样。你必须轻轻摇动这些家伙*，每隔几分钟旋转它们，直到它们变软变平(总共需要大约10分钟)。不要强迫你的华夫饼机关闭；给它时间让热量发挥作用。*

*一旦你有了一个热乎乎、多汁的小火鸡馅饼，把它滑到涂有少许蛋黄酱、少许芥末和一勺蔓越莓酱的面包卷上，加入一小片布里干酪。*

### *华夫饼干感恩节滑块*

***配料:***

*   *冷冻全熟火鸡肉丸*
*   *罗尔斯(我喜欢国王的夏威夷)*
*   *蛋黄酱*
*   *芥末*
*   *酸果曼沙司*
*   *布里干酪*

***指令:***

*把华夫饼干机调到中火，让它升温。将几个肉丸放入华夫饼机中，关闭，让顶部的盘子放在肉丸上。让它们热两分钟，轻轻按压顶板，让肉丸再煮一分钟。一旦你看到一个外壳开始发展和变暗，将肉丸旋转90度并重复。继续这支小曲和舞蹈，直到球变平，并在外面形成一层漂亮的金棕色外壳。*

*如果你想的话，可以烤一下你的面包卷，然后在下半部分涂上少许蛋黄酱和一些黄色芥末，把火鸡肉饼放在下半部分的上面，在肉饼上面舀一勺蔓越莓酱，用一片布里干酪和面包卷的上半部分盖住。食用并重复，直到你没有肉丸(或蔓越莓酱)。*
