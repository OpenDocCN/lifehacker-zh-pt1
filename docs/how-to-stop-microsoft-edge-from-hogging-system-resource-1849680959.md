# 如何阻止微软Edge霸占系统资源

> 原文:[https://life hacker . com/how-to-stop-Microsoft-edge-from-hogging-system-resource-1849680959](https://lifehacker.com/how-to-stop-microsoft-edge-from-hogging-system-resource-1849680959)

微软Edge已经赢得了声誉良好的浏览器的地位，成为大多数Windows用户的一个很好的默认选项。然而，它并不完美。如果微软Edge让你的电脑变慢了，在切换到不同的浏览器之前，你可以尝试一些修复方法。我们已经梳理了该应用的性能设置，以帮助您加快浏览器的速度，并最大限度地减少其在PC上的资源使用。

## 启用效率模式

效率模式专为那些在笔记本电脑上使用Edge的人设计，可减少浏览器的资源使用，并帮助您从您的机器中挤出更多的电池寿命。这一改变将使非活动标签处于睡眠模式，并减少Edge的整体资源占用。

要启用此功能，请打开Microsoft Edge，然后单击右上角的三个点。选择**性能**，然后从弹出窗口中启用**效率模式**。

## 禁用启动升压

Startup Boost允许Microsoft Edge在后台运行助手服务，这有助于浏览器更快地启动。当Edge打开时，启动增强对其资源使用几乎没有影响。但是，当Edge不运行时，它可以继续在后台获取系统资源。

如果这种资源使用困扰你，c 点击浏览器窗口右上角的三个点，然后选择**设置**。在左侧窗格中，点击**系统和性能**，然后禁用**启动增强**。

## 禁用后台扩展和边缘应用

当您关闭Edge时，浏览器允许扩展和Edge应用程序在后台运行。如果你有像微软团队安装了 这样的 [边缘应用，这个功能就很有用，因为它们可以继续独立于浏览器运行。](https://lifehacker.com/how-to-stop-microsoft-teams-from-slowing-your-mac-to-a-1849198990)

但是，如果此后台活动影响系统性能，请尝试禁用该功能。该选项位于Edge设置的**系统和性能**部分。当微软Edge关闭时，禁用**继续运行后台扩展和应用**。

## 修复新的选项卡页面

默认情况下，Edge的新标签页看起来很像昔日的MSN，展示了一堆新闻、广告和推广链接。这种混乱可能是Microsoft Edge性能下降的一个潜在原因。

在Microsoft Edge中打开一个新选项卡，然后单击此页面右上角的齿轮图标。现在，使用以下设置:

*   布局:**聚焦**
*   快速链接:**关**
*   背景:**关**
*   显示问候语:**关**
*   Office侧边栏:**关**
*   新标签提示:**关**
*   内容:**内容关闭**

这些变化应该会清理新标签页中的混乱。最后一步，你可以点击搜索表单下方的三点图标，然后选择**隐藏最近的Bing搜索**。

## 更新Windows和Microsoft Edge

如果这些选项都不能解决您的性能问题，那么是时候更新您的计算机和Microsoft Edge了，因为微软在每个新的更新中都包含了错误修复和稳定性补丁。请确保在更新后重新启动计算机，以使这些更改生效。

最后一个调用端口是恶意软件。你应该买一个好的反恶意软件应用程序，比如来检查你的电脑是否有病毒。如今，加密挖掘恶意软件足够聪明，当你打开Windows任务管理器时，它会让你的计算机慢下来。快速扫描和恶意软件清除可以突出显示并删除这些进程。
