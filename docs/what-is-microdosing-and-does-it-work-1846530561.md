# 什么是微剂量，它有作用吗？

> 原文：<https://lifehacker.com/what-is-microdosing-and-does-it-work-1846530561>

微剂量——服用少量通常是迷幻药的行为——在某一类人中很受欢迎，他们寻求比咖啡因和时间管理等标准生活技巧所能提供的更多的生产力。这是否有效还不得而知，但是 [的一项新研究](https://elifesciences.org/articles/62878) 表明，这种被吹嘘的好处可能一直要归功于安慰剂效应。

所涉及的药物，通常是 LSD 或裸盖菇，在大多数地方都是非法的，因此讨论如何微剂量的社区通常是地下的，非官方的，充满了轶事而不是数据。 [这幅 2018 年的《T2》](https://www.thecut.com/2018/05/microdosing-guide-and-explainer.html)详细描述了这一现象。

然而，基本的想法是，通过服用极少量的致幻剂——通常是娱乐剂量的 10%或更少——你不会感到兴奋，,但你会感到*某种东西*。一个 [reddit 指南](https://www.reddit.com/r/microdosing/comments/jopryk/you_shouldnt_feel_anything/) 描述了一些常见的好处，如感觉专注、有创造力、“开放”、平静、警觉、富有同情心、感激或处于心流状态。

有了如此微妙的好处，个人可能无法知道他们的经历是来自药物还是他们自己期望的表现。换句话说，这可能是安慰剂效应。

“安慰剂效应”一词最初是指这样一个事实，即临床试验中的人即使没有接受药物测试，也往往会有所改善。现在它经常被描述为一种精神超越物质的现象，在这里你将你的感知变成现实。但是 [也可以是不太令人兴奋的因素组合](https://vitals.lifehacker.com/the-placebo-effect-is-no-reason-to-buy-junk-1774878033) ，比如把好的变化归因于药物，把坏的变化归因于我们自己或外部障碍。我们也很少孤立地经历一件事；如果你用微剂量来帮助你集中注意力，你可能也会放一些好音乐，在门上挂一个“请勿打扰”的牌子。

因此，为了解决这个问题，一组科学家想出了一个“自我致盲”方案，微剂量者可以在家里进行。参与者被要求准备含有和不含有他们常用的微量迷幻药的胶囊。他们将胶囊放入带有二维码的信封中，然后洗牌并选择信封用于研究，以确保他们获得四周的微剂量，四周的零剂量，或者一半一半。

参与者对他们的感受进行了评级，并参加了各种在线测试来衡量他们的认知功能。最终，接受和未接受微剂量的人在“所有心理结果”上都有所改善。得到真正微量剂量的人改善稍微多一点，这似乎表明一个小的真实效果。

但有一个转折:调查人员要求参与者猜测他们是服用了真剂量还是空剂量。服用微量药物时感觉和表现最好的人与正确猜测他们正在服用微量药物的人是同一批人。所以有可能这种影响是由于人们知道他们在吃什么。

研究人员写道:“研究结果表明，微剂量的轶事益处可以用安慰剂效应来解释。”进一步的研究也许能够梳理出具体的好处，如果它们确实存在的话。也许某些类型或剂量比其他的更有效。但到目前为止，微剂量的好处似乎非常小。
