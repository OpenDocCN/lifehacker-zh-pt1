# 孩子们不需要总是分享

> 原文：<https://lifehacker.com/kids-dont-always-have-to-share-1847378137>

在我大儿子打开生日礼物几分钟后，我两岁的儿子开始哭了起来。他想在哥哥的新玩具一从盒子里拿出来就玩，但他的弟弟坚决拒绝了他的要求。虽然我对我最大的孩子不想分享感到失望，但我也不能责怪他。对于一个孩子来说，没有什么比在收到你的玩具后几分钟就被迫把它们送给别人更让人恼火的了。

然而，不和朋友或家人分享违背了我小时候学到的一切。如果我认识的人想用我正在用的东西，我明白我应该毫无疑问地给他们一个机会。当我看到这种情况在我的孩子们之间发生时，我问自己，“孩子们总是要分享吗？”

我带着这个问题去见了 [育儿心理学家、畅销书作家、四个孩子的母亲希瑟·维滕贝格博士。](http://babyshrink.org/) 她解释说，与其让*强迫*孩子分享，不如让*逐渐教会*这种行为。但她承认，对许多父母来说，这说起来容易做起来难。

“这实际上比如厕训练更难，因为这是终身的，”她说。“分享是最复杂的人类行为之一，许多人从未掌握它。”

培养考虑他人福祉的年轻人，是维滕贝格相信教会孩子分享的原因之一。她就父母如何将这一做法付诸行动提供了一些建议。

## **为什么教孩子分享很重要**

分享属于基本社会情感技能的范畴，儿童需要学习这些技能来帮助管理自己的情绪，同情他人，结交和保持朋友。虽然分享有助于促进儿童的移情行为，但维滕贝格说，这偶尔会与儿童保护自己界限的需要相冲突。几个世纪以来，这是一项被父母忽视的技能，因为它可以教会年幼的孩子，尤其是女孩，取悦他人是很重要的。

“你不能真正‘强迫’一个人‘友善’或关心或同情，”她解释道。“你可以强迫他们放弃界限，坚持分享，即使他们觉得这样做很不对。但这是错误的教训，不是吗？”

因为孩子们仍在发展和学习如何在社交中处理自己，强迫他们展示像分享这样的高级技能会让他们陷入失败的境地。这可能会导致发脾气和流泪，告诉他们分享只会让事情变得更糟。

“相反，当他们在操场上或生日聚会上处于潜在的分享情境中时，应该一起指导和鼓励，”维滕贝格说。

### 家长如何“教”分享？

维滕贝格强调，在教孩子如何分享时，父母应该知道，这不是像如厕训练那样在一个周末就能学会的技能。如果父母想让他们的孩子知道学习这种移情行为有多重要，他们需要表现出耐心、鼓励、重复— [并愿意在自己身上表现出来](https://lifehacker.com/want-to-raise-empathetic-kids-show-em-how-its-done-1831050481) 。

“这是一种极具挑战性的社会情感行为，需要几个发展阶段，大量的尝试、错误、大量的鼓励和父母的良好榜样，”她说。

在我教我的两个儿子如何分享的情况下，维滕贝格说，教兄弟姐妹可能更令人厌倦，因为他们把自己最好的和最差的行为留给了彼此。此外，在这种情况下，父母更容易失去耐心，因为我们希望我们的孩子能够和睦相处。

她强调父母有必要对他们的孩子坦诚相待，并承认分享对他们来说有多困难。但同时，让他们知道学会如何做是多么重要。此外，树立一个榜样，在孩子面前展示你希望看到的行为。父母也可以安排一个你可以监督和指导的练习分享会。

“当你看到它时，大声喊出‘在野外分享’，”她补充道。“这不一定是什么大事——只是一个平静但真诚的承认:‘嘿，我看到你和你哥哥一起轮到你了。我知道那一定很难。但你已经是个大孩子了，我看到你关心你的哥哥。干得好。"
