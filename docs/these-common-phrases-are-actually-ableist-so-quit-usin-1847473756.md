# 这些常用短语实际上很简单，所以不要用了

> 原文:[https://life hacker . com/these-common-phrase-is-actually-ableist-so-quit-usin-1847473756](https://lifehacker.com/these-common-phrases-are-actually-ableist-so-quit-usin-1847473756)

我们可能没有意识到，但是我们一天中使用的许多单词和短语可以不经意地贬低其他人。我们的许多习惯用语和常见的陈词滥调可以准确地描述为对残疾人的歧视。随着越来越多的人意识到它们有问题的来源，这些短语中的一些已经变得不那么常见了(例如，今天的孩子不像几十年前那样喜欢“r-word ”),但是它们中的许多我们仍然一直在使用。

根据美国疾病控制中心的数据[，美国目前有六千一百万成年残疾人。美国残疾人口的庞大数量可能意味着你知道有人有残疾。(比如我弟弟，就有一条腿。)在我们的言语中变得干练——有意无意——显然是我们应该努力避免的。它证实了无意识的偏见，并加深了不同社区的人们之间不必要的分歧。为此，这里有一些你绝对应该从你的词汇中抹去的常用短语和单词，希望它们最终能完全从词典中消失。](https://www.cdc.gov/ncbddd/disabilityandhealth/infographic-disability-impacts-all.html#:~:text=61%20million%20adults%20in%20the,is%20highest%20in%20the%20South.)

### “那太逊了。”

对令你失望的人或事说“蹩脚”是很容易的。但是这个词被牛津词典 定义为“(特别是动物)由于腿部或脚部受伤或生病而无法毫无困难地行走。”用它来形容一个人是的侮辱——[然而多年来，人们通常就是以这种方式使用](https://www.washingtonpost.com/posteverything/wp/2015/05/13/lame-stand-up-and-other-words-we-use-to-insult-the-disabled-without-even-knowing-it/) 。所以今天不要用它来表明你对百思买ps5售罄的沮丧。

### "别这么傻了。"

你可能还记得你在操场上的时候。医学上，根据 [国立卫生研究院](https://www.ninds.nih.gov/Disorders/All-Disorders/Spasticity-Information-Page) 的说法，“痉挛”是一个过时的术语，与一个人不受控制的身体运动有关，通常由脊髓损伤引起。在英国，这个词变成了一个普通的侮辱，相当于美国中的“r字”..这个词本身是“笨拙”的一个不那么沉重的代名词，但这两种用法对那些经历这种医疗状况的人来说都是非人性化的。所以不要用它来形容你家狗狗的怪异行为。

### “那是疯狂的/疯狂的/疯狂的。”

考虑到我们通常对他人的问题是多么的不了解，称某人为疯子是极其不屑一顾的。正如 [我们今年早些时候写的:](https://lifehacker.com/avoid-using-these-triggering-phrases-to-describe-people-1846388865) “往往是一个词扯出 [来形容女性，而男性常用](https://www.washingtonpost.com/posteverything/wp/2014/07/09/men-really-need-to-stop-calling-women-crazy/) 。如果一个朋友或同事的行为让你觉得不愉快或具有破坏性，找一个准确的方式和他们谈论这件事。不要说他们疯了。”另外，不要叫他们“疯子”不要用“疯狂”来形容汉堡王的排队。

### 残废，只是一般

“残废”一词在历史上描述了由于事故、出生时的残疾或其他医疗条件而导致的身体残疾。你不必引用一个边缘化的社区来准确描述另一个人(或事物，或系统)正经历这样或那样的不足。

### "他患有[插入医疗状况]. "

你可能会认为某个残疾人在“受苦”，考虑到他们的生活可能与你的不同。但是残疾人士并不像作家埃里卡·莫内斯今年早些时候为流行糖写的那样，对他们的生活持如此悲观的看法。“人们经常用‘痛苦’来描述残疾，因为他们把残疾理解为一种状况，一种只会导致疼痛和痛苦的状况，”莫恩斯写道。"然而，残疾人可以像其他人一样过着充实的生活."总之，不要以残疾来定义一个人；那不关你的事。

### “充耳不闻”

这是一个特别容易避免的问题，因为不听或不愿意理解与完全听不见是两码事。你描述的这个人不是聋子，所以最好避免将他们的冷漠与美国 的100万人社区相提并论。

### "我整个周末都在打扫卫生，我太强迫症了."

有许多其他的方式来描述某人的挑剔或倾向于干净整洁，而不涉及这种真正的人格障碍。说你认识的某个人“强迫症”是因为他们把自己的桌子整理得比你的更有条理，这是难以置信的简单化。

### 形容某人是“坐轮椅的”

虽然这个短语经常被使用，但没有冒犯的意图，就像上面的“痛苦”,用这个短语来描述一个使用轮椅的人最终会以他们的残疾来定义一个人的整体，而不是把它仅仅视为他们生活的一个方面。理解一个人不仅仅是身体能力或局限，这很好。

### "你真是个疯子/疯子/神经病。"

一般来说，当有人让你心烦意乱或行为与给定的情况不相称时，用通常与边缘化和排斥神经非典型的人相关的词来描述他们是不合适的。这些话“[捕捉]他们在瞬间成为这种危险的、不可预测的、不同的灵魂，”西北大学教授克里斯多夫·莱恩 [告诉BBC](https://www.bbc.com/news/magazine-17997413) 。患有心理疾病的人不应该被排斥，这就是为什么你应该避免使用“疯子”或“神经病”这样明显伤人的词来描述他们——更不用说在战斗中把他们扔向你的朋友了。
