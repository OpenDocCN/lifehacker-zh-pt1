# 为什么我的 Wifi 网络没有出现在我的所有设备上？

> 原文：<https://lifehacker.com/why-doesnt-my-wifi-network-appear-on-all-my-devices-1846644997>

无线网络是生活中最大的谜团之一。你可能已经在你自己的设置中遇到了很多令人沮丧的问题:为什么我的下载比慢，当我有很好的连接时？为什么当我在家里走动时，我的连接很快就中断了？为什么虽然花了很多钱买硬件，我的 wifi 还是很糟糕？诸如此类的事情。

对于本周的[**Tech 911**](https://lifehacker.com/c/tech/tech-911)T5】Q&A，生活黑客读者乔治想要一个常见的问题的答案——为什么他的 wifi 给他带来如此多的悲伤？—有趣的转折:

“这个问题已经有一段时间了。起初，我的 wifi 网络只显示在我妻子的安卓平板电脑和手机上。我婆婆在一个街区外的无线网络显示出比我们自己的网络更强的连接。我的设备，包括 Android 和 iPhone，不仅开始显示她的 wifi 更强，甚至当站在我的路由器和调制解调器旁边时，其他几个也是如此。当其他人的设备尝试连接时，除了我自己的设备外，没有其他设备显示任何可用的 wifi。这是怎么回事？

### 如有疑问，恢复出厂设置

乔治，这真是一个古怪的问题。我没有一个好的解释，为什么你的设备，只有*你的*设备，在一个街区外的无线网络显示出很强的信号。这不正常。典型路由器的信号不会到达那么远；如果你的整个房子都充满了强烈的信号，那你就很幸运了。

我们的设备变得很奇怪。其他人的都行动正常，因为没有远处的 wifi 信号可供他们追踪。至于你看到的据称比你自己的更强大的其他无线网络，这似乎是合理的。奇怪的是，当你就在你自己的路由器旁边时，会发生这种情况；可能是某种无线干扰造成的。

拿出一台笔记本电脑——理想情况下至少有 wireless-n 连接，所以有一台是在过去十年左右购买的——使用 [一款专用应用程序](https://lifehacker.com/see-how-bad-your-wifi-situation-really-is-with-wifi-ana-1826534324) 来检查你的 wifi 信号。你看到了什么？如果有很多 wifi 干扰，你会看到你的无线网络(和其他人的)都试图使用相同的频道。如果是这种情况，进入路由器的设置，手动选择一个空闲的频道(如果设置为自动选择)，或者设置为自动选择一个频道(如果您之前手动选择了一个繁忙的频道。

如果它处于“自动”状态，尝试关闭路由器，等待大约 15 秒钟，然后再次打开。理想情况下，它不会回到为你的无线网络使用超级拥挤的信道。如果有，您可以进行手动校正。

如果应用程序显示你的路由器的 wifi 信号实际上非常强，只是你的手机显示奇怪，我不会担心。也许你的设备有某种缺陷或硬件问题。(U 为了确保万无一失，请使用最新版本的 Android 或 iOS。)

接下来，继续将路由器重置为出厂默认设置，通常通过点击设置中的按钮或使用回形针在路由器上戳一个小硬件按钮来完成。为什么要用核能？从头开始更容易，尤其是如果您只需键入 SSID 和密码即可重新开始。

一旦你完成了，更新它的固件——路由器用来工作的软件。如果你的路由器没有内置的自动完成这项工作的方法，你可能不得不前往其制造商的支持网站，键入你的路由器的*确切的*型号，然后自己下载。U 把它加载到你的路由器上，重启它，你就可以开始了。

接下来，再次检查通道情况。如果你需要做任何调整，让我们开始吧。否则，您可以尝试的另一种故障排除技术是对您的 2.4Ghz 和 5Ghz 网络使用不同的 SSID(如果您对两者使用相同的 SSID)。

你现在看到了什么？如果您的设备只显示您的 2.4GHz 网络，而不是 5Ghz 网络，这有助于查明问题。也许存在故障诊断无法解决的硬件问题。试试拧紧你的路由器天线？至少你可以暂时接入*的一个*无线网络。

这并不能很好地解释为什么你的手机没有无线网络，而你妻子的手机却有。我敢打赌你正在使用的设备有一些奇怪的地方，但我不希望它们都有类似的问题，唯一的统一特征是你拥有它们的事实*。*

如果更新你的设备不允许你连接到你的 wifi ，但是其他人*可以*连接到你的 wifi ，有理由认为你的设备有问题。为什么他们都受到影响，问题是什么，我不完全确定。也许你的设备很旧，你的路由器正在使用他们不支持的加密(例如，WPA2，如果你的设备只兼容 WPA)。或者，也许你正在为你的无线网络使用你不应该使用的旧标准(如 WEP)，切换到更现代的标准(如 WPA2)将解决这个问题。

仍然是，你的设备应该仍然能够*看到*无线网络，即使它不能连接。如果你的手机允许，试着手动输入你的无线网络信息，看看是否有帮助。(我怀疑不会。)M 确保一切都在运行最新的更新，然后工厂重置它们以确保你已经消除了任何软件变量。如果您在设备上仍然看不到您的 wifi 网络，则是时候更换新路由器或新设备了。
