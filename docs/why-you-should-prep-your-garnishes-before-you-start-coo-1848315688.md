# 为什么你应该在开始烹饪前准备配菜

> 原文：<https://lifehacker.com/why-you-should-prep-your-garnishes-before-you-start-coo-1848315688>

让我给你描述一下:你在做早餐。今天，你决定用棕色黄油做软炒鸡蛋。你计划用一把健康的细香葱片来结束这场争夺战，或者如果香葱吃完了，可以用大葱。你将吃鸡蛋和烤面包。

Watch

鸡蛋已经 [摇匀成蓬松状](https://lifehacker.com/you-should-scramble-eggs-in-a-jar-1844164046) ，黄油在锅里，刚刚开始起泡。面包片在烤面包机里。韭菜还在冰箱里。你马上就能抓住他们。(它们毕竟是最后一步。)哦，黄油颜色变深了。该加鸡蛋了。鸡蛋做好了。吐司烤好了。韭菜还在冰箱里。

现在是做决定的时候了。你会花一分钟时间冲洗和切细香葱，还是坐下来吃滚烫的鸡蛋和烤面包？我通常选择后者，因为此时我很饿，而且我不喜欢室温炒鸡蛋。(如果我已经做了几个小时的菜，我更有可能选择不加配菜的路线。到那时，我已经准备好走出厨房去吃那该死的东西了。)

这太糟糕了，因为 [完成你的菜](https://lifehacker.com/what-it-means-to-finish-a-dish-1797691879) 是重要的最后一步。装饰品不仅仅是为了好看；它们增加了额外的风味、新鲜度或质地，补充和增强了你已经美味的作品。一盘没有配菜的菜就像一个没有句号的句子——你需要的一切技术上都在那里，但它还没有*完成*。

我想说的是，你的配菜不应该是事后想起的。这是食谱的重要组成部分，也应该如此对待。通过清洗、切、切或品尝你的香草、柑橘等，为你的成功做好准备。在 T2 之前，你甚至开始做饭。让它成为你工作的一部分，当你精神饱满，对烹饪充满激情的时候，准备这些收尾工作。你更有可能小心翼翼地准备你的配菜，你的叶子、皮等等更有可能真的到盘子里。
