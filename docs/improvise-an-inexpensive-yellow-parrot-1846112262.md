# 用 Pernod 近似一只黄色鹦鹉

> 原文：<https://lifehacker.com/improvise-an-inexpensive-yellow-parrot-1846112262>

我喜欢尝试新的鸡尾酒，没有什么比*几乎*拥有制作一种鸡尾酒所需的所有原料更让我恼火的了。可悲的是，这正是我本周发现自己的处境，当时我的朋友丹给我发了一份黄鹦鹉的配方，这是一种极其酒精含量高(而且不太划算)的三种成分的酒。

Watch

我有一个相当满的酒吧推车，但上面没有制作这种饮料所需的瓶子。然而，我确实喝了类似的酒，如果不是完全一样的话。我没有喝苦艾酒，而是喝了 [Pernod 牌 pastis](https://skillet.lifehacker.com/add-a-little-pastis-to-your-vesper-in-honor-of-melissa-1843483318)；我用绿色代替黄色黄绿色；我没有喝杏酒，而是喝了桃子酒。

黄色鹦鹉的配料很奇特，它是一种非常简单的鸡尾酒——你只需在碎冰上搅拌等份的所有东西整整一分钟，然后过滤更多的冰，用柠檬皮装饰(当然我也没有柠檬皮)。我的路很清晰:用摆在我面前的不太合适的原料做一只几乎黄色的鹦鹉。我决定称它为“阴暗的鹦鹉”，因为当你稀释它时，Pernod 会变得浑浊。

在将 pastis 与绿色黄绿色和桃子利口酒混合了整整一分钟后，我得到了一种美味的酒饮料，这种饮料的味道让人想起欧洲糖果——甜味、草药味和甘草味。我一直在想“这有点太甜了”，但我也无法停止喝它，这意味着我现在有点醉了，甚至还没到中午。

要制作你自己的黑色鹦鹉，你需要:

*   1 盎司意大利面
*   1 盎司绿色黄绿色
*   1 盎司桃子利口酒

将所有材料倒入加满碎冰的混合杯，搅拌一分钟。滤入盛有冰块的低杯，在杯子边缘摩擦之前，在饮料顶部挤出一条橙子或柠檬皮中的油。(这里不要跳过配菜；这一口柑橘油需要抵消饮料更像糖果的品质。)
