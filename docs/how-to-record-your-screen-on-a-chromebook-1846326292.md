# 如何在 Chromebook 上录制您的屏幕

> 原文：<https://lifehacker.com/how-to-record-your-screen-on-a-chromebook-1846326292>

谷歌将在未来几周内为所有 Chromebooks 添加内置屏幕录制功能。新工具让用户无需额外的应用程序或外部硬件就可以捕捉设备屏幕上的视频。



屏幕记录器应该对所有用户都有用，但谷歌称它实现了专门针对 教育环境的工具:教师可以用它来准备演示，和学生可以用它来记录讲座。

作为谷歌计划在 2021 年 3 月更新 Chrome 操作系统的一部分，该功能将在未来几周内推广到 Chromebooks 上。谷歌没有提到具体的发布日期，所以它可能需要几个星期才能为你的设备做好准备——但如果你选择进入 Chrome OS 测试版，你可以提前试用。下面是如何做到这一点:

1.  在你的 Chromebook 上，进入**设置>关于 Chrome OS >其他详细信息。**
2.  在**“频道”**部分，将您的频道更改为**“Beta 频道”**
3.  点击**“更换频道”**
4.  出现提示时，更新并重启 Chromebook。
5.  重启后，打开 Chrome，进入**Chrome://flags/#截屏**。
6.  从下拉菜单中启用**“截屏测试”**，然后重启。

你现在可以使用 Chrome OS 屏幕录制工具 :

1.  按下 **CTRL + SHIFT +总览**打开记录仪。
2.  拖动你的鼠标来调整你想要录制的屏幕大小。
3.  准备好了就按下录制按钮。将会出现一个倒计时器。一旦它达到零，它将开始捕捉屏幕上的任何视频。在录制过程中，任务栏上会显示一个红色的录制图标。
4.  单击停止按钮结束录制。

不想加入 ChromeOS beta 版的用户可以安装 Chrome 浏览器扩展，如或 [Screencastify](https://www.screencastify.com/) ，同时等待稳定的频道补丁。两者都有局限性，所以一旦内置的 Chromebook 屏幕记录器可用，你可能会想切换到它，但它们同时也是不错的权宜之计。

Vidyard 提供了一个免费版本，允许你录制长达一个小时的视频，尽管你可以根据自己的需要录制尽可能多的视频。Screencastify 对所有视频都有五分钟的截止时间，但你可以免费录制、编辑和导出任意数量的视频(只要你的电脑上有空闲空间)。
