# 如何与非美国人谈论美国发生的事情

> 原文：<https://lifehacker.com/how-to-talk-to-non-americans-about-whats-happening-in-a-1846021473>

本周在华盛顿特区发生的事件引起了世界各国领导人的震惊和恐惧。愤怒的右翼极端分子包围美国国会大厦一栋建筑的照片招致了从英国首相鲍里斯·约翰逊到北约秘书长延斯·斯托尔滕贝格和 T4 的指责，伊朗、瑞典、新西兰、以色列和其他国家的领导人拒绝了政变企图。



这些场景有助于玷污美国的公众形象，以至于的国际讨论都集中在美国民主是否能忍受这样的攻击的问题上。然而，许多美国人——他们中的许多人倾向于贬低在所谓的“第三世界国家”发生的类似事件。

在美国，、政客、媒体人物和普通美国人经常支持我们的机构是不会出错的、不可动摇的，它们受到神圣不可侵犯的法律的约束，而这些法律是不可能推翻的。混乱和暴力在外国城市的街道上爆发。未遂政变在国外造成了破坏和流血，但在这里却没有——在这里法治占主导地位。但是周三表明，我们不能对我们的国家给予如此高的评价，尤其是在非美国朋友中，他们可能想揭开美国目前到底发生了什么。

如果你发现自己在和一个非美国朋友或家庭成员谈论美国当前的形势，以下是如何应对的方法。

### 解释美国例外论的神话

如果你长期饮用美国饮料，你就已经接受了美国例外论——这是一个长期流传的神话，认为美国是这个世界上正义的力量，因此值得在民主和公民自由可能受到威胁的地方维护其影响力。直截了当地说，这是一句华丽的废话，但尽管如此，它还是影响了许多普通美国人对其他国家的看法。

如果一个非美国人问这种想法来自哪里，你可以追溯到美国宪法，它宣扬某些不可剥夺的权利和人人平等(当然，如果你是一个拥有土地的白人)。纵观历史，这种观念已经在我们的民族精神中根深蒂固。各种各样的军事努力，比如两次世界大战——美国总统伍德罗·威尔逊在一战期间著名地宣称，美国正在让“民主的世界变得安全”——此外 [罗纳德·里根的“山顶上闪耀的城市”演讲](https://en.wikipedia.org/wiki/City_upon_a_Hill) 也进一步强化了这个概念。

当然，任何经常关注国际新闻的非美国人都知道，周三的事件绝对是前所未有的。但是美国人对我们自己的同胞表达的震惊和沮丧更有意义，当你明白我们被反复告知这样的事情不能在这里发生。

### 解释这些事件实际上是如何有意义的

亲美教条似乎在周三被粉碎了，但这一特殊事件并非前所未有——尤其是如果你在最近的背景下审视它。如果你想揭开周三的围攻到底是如何实现的，只需看看简单的事实就够了:即将离任的总统一直、谎称 2020 年的选举因猖獗的选民欺诈而被操纵；一个由网上阴谋论者和有影响力的人组成的强大网络在互联网上传播谣言，在那里它经常接收到网络和网站的播放时间；许多美国人认为他们被邪恶的精英小集团剥夺了自由民主进程，他们受到总统和其他立法者的鼓励进行反击。

从某种意义上说，周三是颠覆 2020 年选举结果的长期协调努力的高潮。我们从更理性的立法者和媒体人物那里听到的愤怒和怀疑的回应，至少在某些方面是由于美国人对我们自己和我们的国家根深蒂固的特殊感。对于非美国人来说，在这个镜头下可能更有意义，尽管它可能不会使它变得不那么令人困惑。
