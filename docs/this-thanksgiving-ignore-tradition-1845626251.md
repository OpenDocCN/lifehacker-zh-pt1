# 这个感恩节，忽略传统

> 原文：<https://lifehacker.com/this-thanksgiving-ignore-tradition-1845626251>

今年的感恩节将与我们经历过的任何一次都不一样。我们没有“破解”的办法:我们必须对庆祝这一天的方式做出真正的、实质性的改变，这意味着与更少的朋友和家人一起庆祝。除非你属于某个荒谬的财富家族，否则你可能没有资源要求每个人在乘坐私人飞机飞往私人岛屿 [之前都要经过“多重健康检查”，在短暂的时间里假装一切正常](https://www.cnn.com/style/article/kim-kardashian-birthday-party-island-scli-intl/index.html)

不要试图强迫一个大的(甚至是中等的)潜在危险的集会，你应该扔掉规定这个节日“应该”是什么的规则手册。em 掩盖这一切的渺小。尽可能自私。正如我在我们最近的邪恶一周提到的，疫情的可以作为一个 [的内置借口，不见任何人](https://lifehacker.com/use-the-pandemic-as-an-excuse-to-cancel-holiday-plans-1845499406) 或参加任何你不想参加的活动。我很喜欢，但我们可以进一步推进。

因为你很可能会和已经住在你家的人一起庆祝，菜单可以小得多，并且根据你实际想吃的东西量身定制。你像我一样讨厌火鸡吗？让今年成为你最后烤鸭的一年。完全放弃鸟类，买一些真正美味的牛排。你知道一只火鸡能买到什么样的牛排吗？这只*真的是*好看的那种。你知道一只火鸡的价格可以买多少熟食吗？磅——字面意思是磅。你知道一只火鸡能买多少香槟吗？嗯，只有一瓶，但这是一瓶你不必分享的酒。

菜单由你来改变，传统由你来忽略。如果你从来都不太喜欢肉，那就把所有的精力放在配菜上，做一顿土豆和砂锅菜。或者完全拒绝感恩节的整个美学，用这一天来解决一个大的烹饪项目，这个项目与火鸡或填料或任何那些乱七八糟的东西毫无关系。做肉馅卷饼，吃玉米卷。从头开始做拉面肉汤。做千层面。

无论你如何选择过自私的感恩节，确保它带给你快乐。中午或两点吃晚餐，或者晚上 10 点吃小吃。穿上你最好的衣服，或者穿着睡衣到处走。播放圣诞音乐，或者沃伦·泽方，或者享受家里没有很多人时的宁静。

然而，有一件事你不应该改变，那就是你烤的馅饼的数量。馅饼再多也不为过，而且——在我们经历了这一年之后——我认为我们每个人都应该有自己的馅饼。事实上我需要它。
