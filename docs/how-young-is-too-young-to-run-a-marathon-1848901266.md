# 多年轻才算太年轻不能跑马拉松？

> 原文:[https://life hacker . com/how-young-is-too-young-to-run a-marathon-1848901266](https://lifehacker.com/how-young-is-too-young-to-run-a-marathon-1848901266)

最近，肯塔基州的一个家庭发现自己处于争议的中心，因为他们允许他们6岁的儿子参加上周六在辛辛那提举行的飞猪马拉松 。许多人对这么小的孩子跑那么长的距离感到厌恶，质疑这对他的健康是否安全。在我们得出任何结论之前，让我们看看科学是怎么说的。

为了找到答案，我首先查阅了美国卫生与公众服务部发布的美国人体育活动指南。根据该指南，跑步作为一项活动推荐给6 岁及以上的儿童。这项活动不仅有助于儿童建立有氧基础，它也是一种骨骼强化措施。他们建议儿童和青少年每天锻炼一小时，还可以包括肌肉强化活动。跑步并不是HHS推荐的唯一活动。其他有氧运动包括跳绳、游泳或骑自行车。对于肌肉强化活动，他们建议进行非结构化的活动，比如在操场器械上玩耍或爬树。对于骨骼强化活动，他们推荐篮球、网球和跳房子。

好的，所以是的，孩子们显然可以跑。但是应该跑这么长的距离吗？那得看孩子。明尼苏达大学Phalen村诊所的威廉·奥·罗伯茨博士解释说，只要孩子们“遵循一个可接受的监督训练计划，强调乐趣和参与，而不是记录和快速时间”,跑步是允许的。他接着指出，只要“他们在训练中没有受伤或疼痛”,这应该不成问题。正如奥罗伯茨所指出的，事实是，关于青少年马拉松运动员的数据并不多。

虽然该领域的一些人认为青少年马拉松运动员受伤的风险更高，但没有证据支持这一说法。事实上， [奥罗伯茨写道，](https://sci-hub.se/https://doi.org/10.2165/00007256-200737040-00007) “有成千上万名18岁的<青少年参加了有组织的洛杉矶马拉松比赛，在双城马拉松比赛中有近300名参赛者。这些数据集中最小的孩子只有7岁。在这些事件中没有任何重大的医疗伤害。”

美国儿科学会 呼应了类似的观点，解释说，“儿童和青少年必须得到适当的训练，以避免体温过低或过高，过度训练，过度使用损伤和倦怠。”马拉松需要一个精心培养的安全增加每周总里程的计划，但只要考虑到这一点，你的孩子就有可能避免受伤。AAP 说，“最终，没有理由不允许一个年轻运动员参加一个正常运行的马拉松比赛，只要这个运动员喜欢这项活动并且没有任何症状。”

简而言之，对于你的孩子是否应该跑特定的距离，没有“一刀切”的答案。这要看孩子的年龄，动机，所在的训练团，不能一概而论的说，“这个完全是错了，”或者，“T他的还好，其实是。我们所知道的是，孩子们被鼓励去跑步，尤其是当他们想要跑步的时候。所以，如果你担心他们要走多远，或者担心他们是否安全，一定要监视他们。
