# 教孩子如何拿剪刀的最简单的方法

> 原文：<https://lifehacker.com/the-easiest-way-to-teach-kids-how-to-hold-scissors-1846573779>

当孩子们磨练他们精细的运动技能时，有很多基本的任务是他们通常很难学会的。例如[系鞋带](https://offspring.lifehacker.com/teach-kids-to-tie-their-shoes-with-the-big-loops-trick-1828838854)[穿上自己的夹克](https://offspring.lifehacker.com/teach-your-preschooler-how-to-put-on-a-jacket-with-the-1828497274) ，这些都是我们通过练习和耐心帮助他们掌握的技能。然而，当谈到使用剪刀时，我们可能更喜欢尽可能熟练、尽可能快地掌握它们。



适当地拿着一把剪刀对幼儿来说是不直观的。他们以尴尬的角度扭曲他们的手和手腕，以一种砍它而不是切它的方式攻击那张建筑纸。但是 [忙碌蹒跚学步](https://www.instagram.com/busytoddler/) 的苏茜·艾莉森有 [一个简单的小窍门，你只需要一个马克笔或一个小贴纸:](https://www.instagram.com/p/CM41sbblW3p/)

 [https://lifehacker.com/embed/inset/iframe?id=instagram-CM41sbblW3p&autosize=1](https://lifehacker.com/embed/inset/iframe?id=instagram-CM41sbblW3p&autosize=1) 

使用记号笔或贴纸标记他们惯用手的拇指甲。正如 [佳佳写的](https://www.instagram.com/p/CM41sbblW3p/) :

> 用记号笔画一个笑脸(永久洗掉就好了)，告诉你的孩子“保持笑脸朝上”这有助于使一个非常抽象的想法变得非常具体:孩子们可以立即“看到”如何帮助他们更成功地切割。

如果孩子们不在家练习剪刀，他们最终会在幼儿园或学前班学会正确的剪刀技术。但是，通过在学龄前几年对他们使用这个技巧，你可以给他们一个良好的开端，让他们更好地准备处理所有这些学校艺术项目。
