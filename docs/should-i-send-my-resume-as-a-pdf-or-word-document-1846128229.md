# 我应该发送 PDF 或 Word 文档形式的简历吗？

> 原文：<https://lifehacker.com/should-i-send-my-resume-as-a-pdf-or-word-document-1846128229>

如果你已经很久没有出去试水就业市场了，掸掉简历上的灰尘可能会感觉像是在挖掘几个世纪前的遗物。但是一旦你准备好了，你可能会想知道在所有想雇佣你的雇主中分发它的最佳潜在格式。

Watch

虽然使用 Microsoft Word 可能是过去几年的最佳实践，但现在以 PDF 格式发送简历更为普遍。使用 PDF 有许多好处，但仍有许多雇主出于自己的原因要求使用 Word 文档。

一旦你的简历达到了面试的标准，以下是如何应对简历格式潜在的求职困境。

### PDF 的一个优势是格式化

PDF 基本上是一个大小适合所有人的文件，不管你用什么设备下载，看起来都一样。对于 Word 文档来说，情况并非如此，因为你用来让你的简历闪现个性的一些诡计可能会从一台计算机传到另一台计算机。如果收件人使用的 Word 版本与您的不同，格式也会改变。

正如 [简历中蔻驰解释的](https://www.resumecoach.com/send-resume-word-or-pdf/) ，在不同设备间传递一个 Word 文档的怪癖会带来许多令人头疼的问题:

> 通常页边空白**大小不一**，一页的简历可能会溢出到下一页，**字体可能会出现不同**(因为程序可能没有你选择的字体)，或者更糟糕的是，你的简历可能只是看起来像**无法解读的代码**。
> 
> 如果你已经花了宝贵的时间在你的简历上，专注于 [简历格式](https://www.resumecoach.com/resume-format/) 和布局，雇主打开一份**凌乱的简历**而不是你的**精心制作的简历**，那将是悲剧。它给人完全错误的印象。

pdf 也可以被锁定和代码保护，这是一个奖金，如果你厌倦了有人，嗯，破坏你的候选人资格。

### 当你可能使用单词 doc 时

如果你要给招聘人员或潜在的经理发电子邮件或短信发送简历，我建议选择 PDF 格式，因为它有格式上的优势，而且打开文档也很容易。但是当你在很多公司网站上申请时，你可能会选择 route 这个词。

许多公司使用更容易扫描 Word 文档的申请人跟踪软件(ATS)。例如，你可能已经在一个在线工作列表上提交了一份 PDF 文件，但是一旦系统消化了它并把它退回来，你就会发现你的经历看起来是分散的和不可理解的。

正如 [ZipJob 解释的](https://www.zipjob.com/blog/pdf-vs-word-for-sending-your-resume/) :

> 在网上申请工作时，发送简历的最佳格式通常是 Word 文档。这种格式最容易被大多数 [**申请人跟踪系统**](https://www.zipjob.com/blog/ats-resume-test/) (或 ATS)阅读。虽然越来越多的公司投资更复杂的 ATS 软件来解析你的简历，但你可以确信几乎所有的 ATS 扫描都能读取. doc 文件。

如今大多数 ATS 系统都与 pdf 兼容，但多加小心也无妨。

### 在极少数情况下，Word 文档是可编辑的

这种情况在招聘人员中更常见，他们喜欢对简历进行编辑，这样可以更好地增加你获得工作的机会。因为 PDF 并不是一个任何人都可以更改的活文档，所以你可能希望将你的简历以 Word 格式发送给招聘人员，这样他们就可以对其进行修改，并加强必要的内容。

### 两者都在手边真好

我通常使用 pdf，但是有一个通用的最佳实践对每个人都适用:提交雇主要求的格式。每个公司都有自己的政策和工作流程，所以你可能会被要求在某一天提交一份 Word 文档，然后在第二天提交 PDF 文档。根据我的经验，大多数雇主都要求 pdf 格式，但是不知道你是否会在某个时候被要求不同的格式。两者兼得肯定是最好的。
