# 做砸了的泡菜暴力又好

> 原文：<https://lifehacker.com/making-smashed-pickles-is-violent-and-good-1847416739>

一片泡菜有它的时间和地点(例如，在汉堡上)，你盘子边上的尖矛可以给你的午餐带来熟食的氛围，但泡菜的形状不需要如此光滑、整洁、干净。事实上，完全放弃刀子有一个非常好的理由:朋友们，打碎你们的泡菜吧。粉碎它们，获得自由。

从什么中解脱出来？刀子是其中之一，但泡菜必须看起来像某种样子。就像 [里的黄瓜被砸成肉酱](https://lifehacker.com/smashed-cucumbers-make-better-salads-1826458158) 一样，被砸成肉酱的黄瓜比被切得干干净净的黄瓜更容易吸收味道。

粉碎对黄瓜细胞壁的破坏比剁碎或切片要强烈得多，细胞破坏是让黄瓜吸收味道的原因。(这也是为什么给你的泡菜 [快速按摩](https://lifehacker.com/make-quick-pickles-even-quicker-by-massaging-your-veggi-1836190277) 有助于加快腌制过程。)再加上，就是好玩而已。拿一个钝的物体去切一个坚硬但多汁的水果是一种令人满意的方式，而用刀是不行的。你甚至可以让孩子们参与进来——只是要确保在把黄瓜粉碎之前，把它们放进一个密封的冷冻袋里。

对于捣碎的泡菜，我没有严格的“配方”，因为它对任何盐水都有效。(云杉噬有，如果你对那种东西感兴趣的话，不过我一直用 [这种卤水](https://lifehacker.com/you-should-put-pickled-corn-on-everything-1827544519) 来做一切。)只要抓一些腌制的黄瓜，把它们放在某种可密封的袋子里，然后用小煎锅、木槌或沉重的勺子打碎。将这些零碎的东西和大蒜、香料或芳香剂一起装进罐子里，用热盐水覆盖。让盐水冷却到室温，然后冷却泡菜。

粉碎不仅让黄瓜准备好接受盐水的味道，它还将更多的黄瓜汁释放到盐水中，从而产生一种更多汁、更有黄瓜味的盐水。我从我的花园里砸碎了一根黄瓜，用我的 [腌玉米盐水](https://lifehacker.com/you-should-put-pickled-corn-on-everything-1827544519) 浸泡，那东西绝对被注入了味道。(事实上，低等生物可能会认为这些泡菜太美味了，但我喜欢一种特别有侵略性的泡菜。)
