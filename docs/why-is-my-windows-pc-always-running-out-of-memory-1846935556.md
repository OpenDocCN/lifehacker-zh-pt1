## **你的系统很好，除了诺顿本身**

如果你不介意的话，劳埃德，我回答你的问题时想稍微离题一下。我认为你根本不需要使用诺顿实用程序高级版。你每年要支付40美元来购买一套你并不真正需要的设施，往好里说，这些设施会导致你的系统资源堵塞。

不仅仅是我在地上插了一面旗子。我数不清有多少计算机极客、评论家、IT专家和权威人士对这种“一体化修复”应用程序大加挞伐。虽然Windows 10远非完美，但该操作系统完全在后台管理所有这些东西的效率相当高。你可以或应该做的并不多，来“清理”你的电脑，释放你从来不知道它有能力达到的性能水平。除了让更少的程序打开(在后台或其他地方)来消耗所述内存这一明显的步骤之外，也没有真正好的方法来“修复”“低系统内存”。

事实上，当我面对一个反应迟钝的系统，或者像你这样，系统内存不足的警告时，这是我第一个求助的地方。但是也要知道，系统内存低并不一定是一件坏事。这可能表明，如果你的*总是*运行缓慢(如果你甚至*可以*升级)，你可以为你的装备增加一点，但是Windows 和你的各种应用使用你的内存也是很常见的。如果你没有很多，很有可能你在电脑上做的任何事情都会占据很大一部分。

也就是说，我敢打赌诺顿实用程序高级版也会消耗大量的系统资源和/或内存。我的意思是，在它的描述 中，它实际上是这么说的 [:“当你的电脑空闲时，它会自动维护你的电脑。”](https://us.norton.com/norton-utilities)

如果我是你，我会卸载Norton Utilities Premium并取消订阅。它提供给你的许多“优势”你都可以自己实现。例如:

*   使用**任务管理器>启动**来查看当你的电脑启动时哪些应用程序启动。禁用任何你不需要的。
*   更普遍地使用**任务管理器**来检查消耗大量系统资源的应用程序。考虑一下是应用程序的问题，还是你的电脑动力不足。
*   使用Windows 10的内置**磁盘清理**实用程序快速轻松地扫描不需要的文件，您可以安全地删除它们
*   检查Windows 10的**优化驱动器**应用程序，确保您的系统根据需要自动整理/整理您的硬盘/固态硬盘。
*   确保您定期检查(并安装)操作系统的最新更新，以及最新的驱动程序和固件。
*   使用Windows的内置杀毒软件( **Windows Defender** ) ，并补充一个按需运行的恶意软件应用程序(比如Malwarebytes的产品)
*   使用浏览器的内置工具定期清除缓存、删除历史记录、删除cookies等。

这听起来很多，但这是简单的事情，你不需要一个付费程序来代表你做。我不会强调诺顿的警告。删除应用程序，停止为你不需要的东西付费，并开始感受你的电脑是否真的运行缓慢。如果是，考虑 [升级其内存](https://lifehacker.com/dont-buy-overpriced-ram-from-apple-1844778287)[升级到SSD](https://lifehacker.com/how-to-migrate-to-a-solid-state-drive-without-reinstall-5837543) 而不是机械硬盘(如果你还在用的话)，使用闪存盘 [作为RAM磁盘](https://www.howtogeek.com/123780/htg-explains-is-readyboost-worth-using/) ，甚至执行 [全新重装Windows 10](https://lifehacker.com/the-ultimate-guide-to-reinstalling-windows-from-scratch-1832897572) 。
