# 用完酥皮点心碎屑的三种方法

> 原文：<https://lifehacker.com/three-ways-to-use-up-puff-pastry-scraps-1849609367>

酥皮点心可以将最简单的配料变成令人眼花缭乱的一餐或甜点。可悲的是，似乎总是有无用的残羹剩饭。这种珍贵的全黄油材料买起来并不便宜，当然也不容易制作。不要扔掉分层酥皮金箔的剩余部分，把多余的部分用在其他东西上。

我总是本能地保存较大部件的碎片。即使我没有确切的计划，我也会把各种残羹剩饭扔进冰箱。在你用过第一批松饼皮后，不管是商店买的还是自制的，都要用一张羊皮纸把碎屑卷起来。(商店购买的版本通常已经用未上蜡的纸卷好了，所以只需重复使用这些纸来卷就可以了。)为了节省冰箱空间，也因为你不需要保持废料的形状，把这卷废料对折。把它们放在一个密封的袋子或容器里，放在冰箱里，直到你决定做这些神奇的零食。

## 五香面包丁

我先把最好的留了下来。这是我最喜欢的使用酥皮点心碎屑的方法，也是最快的方法之一。将剩余的糕点放在切割面上。如果边角料冻住了，让它们在柜台上坐五分钟；它们会很快解冻，但摸起来应该还是冷的。将废料切成大约半英寸大小的块。我用一个比萨饼切割机，从不同的角度把它们卷起来，直到我把它们都切成合适的尺寸。

将碎片倒入一个中等大小的碗中，淋上大约一茶匙油。如果你有一瓶油，喷五次就可以了。你不希望它们太油腻，稍微湿润一点，这样干调味料就有东西可以粘在上面。用叉子搅拌酥皮点心。加入任何你喜欢的调料。我喜欢加盐、辣椒粉、牛至、大蒜粉、洋葱粉和鲜味粉。再次搅拌，直到各片均匀覆盖。将碎片平铺在铺有羊皮纸的平底锅上，这样它们就不会重叠。在预热到 400 华氏度的烤箱中烘烤 10-12 分钟。在汤、沙拉、意大利面中加入油炸面包丁，或者作为松脆的零食轻咬。

## 肉桂糖猴面包

与上面的油炸面包丁类似，猴子面包是通过将层压面团放入干物质中制成的——除了这次我们使它变甜(并以不同的方式成型)。用同样的方法准备面团，将解冻的酥皮点心切成小块，大约半英寸到一英寸。将残渣放入碗中，加入少量油或融化的黄油，大约一茶匙左右。如果你有两杯废料，在上面撒两汤匙白糖或红糖，大约两茶匙肉桂和一撮盐。将碎片搅拌均匀，使其包裹在干配料中。

根据你用了多少废料，在面包盘上涂上黄油，或者在纸杯蛋糕上涂上黄油来制作单个的猴子面包。将烘焙容器装满大约一半的废料混合物。如果你在碗的底部有剩余的糖，撒在上面。在预热到 400 华氏度的烤箱中烘烤 10-15 分钟，取决于你使用的烤盘。当猴面包在上面烤熟后就完成了。趁热把面包取出来，因为冷焦糖会粘在平底锅上。

## 奶酪棒

如果你有很多长的碎屑，你可以考虑卷一些奶酪条。我发现这种方法最适合干奶酪，因为它们不会完全融化。将大约半杯碎帕尔马干酪或其他陈年奶酪倒在工作台上。用刀穿过切碎的奶酪，制成碎肉。用解冻的大约两到五英寸长的糕点碎屑，放一个在奶酪中，张开手掌，在奶酪中挤压和滚动糕点。奶酪会粘住并嵌入油酥面团中。用你所有的酥皮点心块做这件事。如果你有很长的片，在奶酪中卷起后对折，扭曲成漂亮的图案。将面包片放在铺有羊皮纸的烤盘上，在预热到 400 华氏度的烤箱中烘烤 10-12 分钟。

在冰箱里堆积一大堆酥皮点心碎屑，然后把这些点心变成更大的产品。如果放在平底煎锅里，猴面包会是一道特别诱人的主菜。
