# 国税局的那条短信是个骗局

> 原文：<https://lifehacker.com/that-text-from-the-irs-is-a-scam-1849625108>

收到一条来自“国税局”的短信，声称要对你的可疑行为提起诉讼，这是非常令人担忧的，而阻止逮捕令的唯一方法就是马上发送你的个人信息。但是你不应该这样做。九月下旬， [国税局警告公众](https://www.irs.gov/newsroom/irs-reports-significant-increase-in-texting-scams-warns-taxpayers-to-remain-vigilant) 以国税局为主题的“smishing”有所增加，目的是窃取你的个人和财务信息。美国国税局今年已经正式确认了数千个欺诈性的原始域名，过去几周的案例呈指数级增长。



“最近几个月，美国国税局报告了多次大规模的窃密活动，在几个小时或几天内传递了数千条甚至数十万条以国税局为主题的信息，远远超过了以前的活动水平，[国税局专员查克·雷蒂格在上个月的新闻稿中说](https://www.irs.gov/newsroom/irs-reports-significant-increase-in-texting-scams-warns-taxpayers-to-remain-vigilant)。

## **什么是“微笑骗局”？**

一些更先进的技术向你的手机发送一个字母数字的“验证码”,并要求背诵或通过短信或电话发回给他们。一旦骗子有了这个代码，他们就可以完全访问你的信息，并可以访问你名下的账户。不要将验证码发送回未经验证的来源。

## **如何识别虚假的国税局短信**

最重要的是要记住，国税局永远不会通过电子邮件或短信联系你，向你询问任何个人或财务信息或账号。他们首先通过官方的地面邮件联系。

短信声称来自国税局，提供假 COVID 减免、税收抵免或帮助设立账户等服务。如果你收到一个链接，提示你输入任何个人信息，即使你得到一些服务作为回报，这也是一个危险信号。

## **如果你收到这些短信该怎么办**

当你不确定是否要通过电话、短信或网络提供任何敏感信息时，一个很好的经验法则是拨打该组织的官方号码并相互参照，看看你被要求提供的信息是否真的有必要。如果组织没有需要你提供这些信息的记录，你可以确定你是一个微笑骗局的目标。

下一个最好的办法是向 phishing@irs.gov 报告这个数字。这将有助于国税局跟踪最新的策略，并向服务提供商报告，服务提供商可以采取行动打击这些欺诈性的域名，以防止其他人成为受害者。

以下步骤是 IRS 报告欺诈的官方指南:

*   给 phishing@irs.gov 发一封电子邮件。
*   复制来电显示号码(或电子邮件地址)。
*   将号码(或电子邮件地址)粘贴到电子邮件中。
*   按住 SMS/text message 并选择“复制”。
*   将消息粘贴到电子邮件中。
*   如果可能的话，包括收到消息的确切日期、时间、时区和电话号码。
*   将邮件发送给 phishing@irs.gov。
