# 你实际上可以从 Telegram 获得免费的无限云存储

> 原文：<https://lifehacker.com/you-can-actually-get-free-unlimited-cloud-storage-from-1848337055>

免费无限的云存储通常是一个神话。如果有人把它卖给你，通常会有很多警告——通常是隐私问题。但实际上有一种完全免费、无限制的云存储，只要它持续存在，这是相当合法的。它是通过电报来的。



Telegram 是一种流行的私人信息服务，以其无限制的文件共享和其他功能而闻名。原来，Telegram 也有一个 API，让开发人员可以访问云存储功能。在 [的开发者 TeleDrive](https://teledriveapp.com/) 使用这个 Telegram API 创建了一个使用这个特性的云存储服务。

这很简单:你用你的电报账户登录，把它和 TeleDrive 连接起来，你就可以免费上传你想要的文件了。你只受到带宽的限制。每天有 1.5 GB 的带宽，以及 2 GB 的上传和下载限制。如果你愿意的话，可以通过支付 10 美元/年的费用来删除这两个选项。

TeleDrive 还对隐私部分进行了分类:它使用 Telegram 自己的存储空间，因此默认情况下是隐私的，它也可以在 Telegram 应用程序的已保存信息部分中找到。

所有这些听起来都不错。那么有什么问题呢？我们用多个文件测试了这项服务，它运行良好。您甚至可以在浏览器中预览视频和音频文件。唯一真正的限制是这项服务依赖于 Telegram 的 API。如果 Telegram 决定限制或停止其云存储 API，这项服务就没有了。虽然 TeleDrive 网站可以免费使用，但它仍然有自己的服务器和开发成本。

目前，它得到了赞助商的支持，他们计划在网站上(而不是仪表板上)引入广告。只要 TeleDrive 有足够的钱维持运营，你就没问题。如果你打算使用 TeleDrive 上传大文件，我们建议你支付 10 美元/年的费用(从大的方面来看，这是相当合理的)。

默认情况下，上传到 TeleDrive 的所有文件都是私有的，但您可以将其公开并生成一个任何人都可以用来下载文件的 URL，或者您可以使用 Telegram 并与任何知道用户名的 Telegram 用户共享。这里没有真正的协作，但是您可以获得共享文件的消息传递功能。

那么你应该如何使用这项服务呢？这可能是一种共享大量大文件的好方法，或者是一种很好的二级或三级备份服务。但是考虑到它易变的本质，我们不建议您将它用作主要备份。为此，最好购买 iCloud、Google One 或 OneDrive。如果你想知道更多关于这项服务如何运作的信息，请阅读他们的 [FAQ 页面](https://teledriveapp.com/faq) 。
