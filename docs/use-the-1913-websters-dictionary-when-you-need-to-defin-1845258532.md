### 如何使用 1913 年韦氏词典

事实证明，弗劳恩菲尔德发现了 1913 年在线版的[*韦氏词典*](https://www.websters1913.com) 来自不同的时事通讯推荐: [星期五发现](https://www.perell.com/friday-finds?utm_campaign=Recomendo&utm_medium=email&utm_source=Revue%20newsletter) 来自大卫·佩瑞尔。

显然，Perell 使用这本有 107 年历史的字典是因为他认为“现代字典有懒惰的定义，过于注重简单而牺牲了精确。”Frauenfelder 同意这一评价，并补充说他喜欢“界面的极端简单性”，指出“它值得书签。”

所以二话没说，下面就把 [链接到 1913 年*韦氏词典*](https://www.websters1913.com) *。*
