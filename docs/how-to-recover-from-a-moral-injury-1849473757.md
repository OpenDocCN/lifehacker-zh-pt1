# 如何从精神伤害中恢复

> 原文：<https://lifehacker.com/how-to-recover-from-a-moral-injury-1849473757>

我们大多数人都熟悉 [创伤后应激障碍](https://www.mayoclinic.org/diseases-conditions/post-traumatic-stress-disorder/symptoms-causes/syc-20355967) ，或 PTSD，它是由目睹或经历创伤性事件引发的。患有创伤后应激障碍的人通常会经历一系列精神健康症状，如噩梦、闪回、严重焦虑和无法控制的事件记忆。心理学家开始发现，除了 PTSD ( 与身体安全感有关)之外，当一个人目睹或做出违背个人价值观和信仰的行为时，人们还会产生所谓的。

## **什么是“精神伤害”？**

精神伤害的概念最初是在治疗越战老兵时形成的，他们中的许多人都在处理自己或目睹他人所做行为的精神痛苦，无论是未能挽救他人的生命，参与或目睹暴力行为，在特别危险的时刻冻结，还是未能报告违反道德的行为。

尽管精神伤害是在治疗退伍军人的背景下发展起来的，但在其他一些情况下，如果一个人个人做出了违背自己信仰的行为，他可能会经历类似的经历； 经历因所爱之人(如父母或伴侣)的行为而产生的背叛感；或者发现以前信任的机构的不道德行为，比如你的教堂或学校。

道德伤害本质上是信任的丧失，无论是对自己、他人还是机构的信任丧失。“背叛是精神伤害的很大一部分，”治疗师 [米歇尔·德马科](https://micheledemarco.com/about/) 说。"在精神伤害的核心，它是非常相关的."

## 你如何从精神创伤中恢复过来？

这里的 [对于精神创伤的最佳治疗方法](https://www.ptsd.va.gov/professional/treat/cooccurring/moral_injury.asp)以及用于治疗创伤后应激障碍的策略是否也适用，仍未达成共识。事实上，遭受精神伤害的人往往会退缩和孤立，这使得治疗变得更加复杂。“他们觉得自己被污染了，或者世界被污染了，他们需要保护自己，”德马科说。

正如德马科所指出的，精神伤害会彻底终结一个人对自己和自己生活的世界的理解。“他们的整个存在主义定位完全被扭曲了，”德马科说。“最终，这是一种纯真的丧失。”

尽管我们仍有许多关于治愈道德创伤的最佳方法要学，但德马科指出，这个过程的一部分不可避免地必须包括接受你现在所经历的世界，这包括失去纯真。

德马科说:“有时候，这是出于善意的诚实，了解到生活中的事情有时只是悲伤，并找到一种方式去接受它。”

治疗的另一个重要方面是找到与他人联系的方法。德马科说:“愈合非常需要重新联系和结合。有时，这还包括评估哪些关系是重要的，哪些关系可以放手。

在她自己的工作中，德马科还发现，写作可以为遭受道德伤害的人提供救济，因为这可以让人们敞开心扉，即使只是对自己，讲述发生了什么，以及产生了什么影响。

这样做可以帮助人们获得“真相的小内核”、，然后可以用来理解发生了什么。正如德马科所言，最大的问题是:“我如何种植这些【真理的核心】，以一种不会让我停留在胎儿位置的方式，让我在新的光中成长？”
