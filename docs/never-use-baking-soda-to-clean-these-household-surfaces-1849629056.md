# 千万不要用小苏打来清洁这些家居表面

> 原文：<https://lifehacker.com/never-use-baking-soda-to-clean-these-household-surfaces-1849629056>

当谈到家务时，很少有产品像小苏打一样方便和通用。从烹饪、清洁到除臭，它能为如此便宜的东西做这么多事。



但是，有些情况下，小苏打并不能解决问题；事实上，它实际上会使事情变得更糟。所以在你开始把它洒在任何东西上之前，要熟悉它可能伤害的表面。以下是需要了解的内容。

## 不要在这些表面上使用小苏打

虽然小苏打可能感觉像细粉，但它实际上是一种磨料清洁剂——这意味着它不适合所有表面。以下是一些例子:

### 大理石

大理石 [和石英](https://www.rd.com/list/never-clean-with-baking-soda/)每次擦拭都会磨损顶部保护层，最终可能导致划痕或暗沉。

## 玻璃

说到你不想刮伤的表面，玻璃也在上面。但是不要担心:你有很多其他的选择来获得无条纹光泽的。

### 你的头发

这在理论上似乎是个好主意，但实际上，在你的头发上使用小苏打会让头发非常干燥。幸运的是，你可以在头发上涂上很多其他的东西来让情况变得更好，而不是更糟。

### 木地板和家具

再说一次，苏打粉对木头表面来说有点太磨人了。它会磨损木材表面的密封剂，甚至导致木材翘曲。

### 任何有划痕、裂缝、蚀刻或雕刻的东西

作为一种粉末，小苏打沉积在物体表面的的每一个角落和缝隙中。虽然它可能有助于清洁物品的其余部分，但你将很难将其从这些凹槽中取出。
