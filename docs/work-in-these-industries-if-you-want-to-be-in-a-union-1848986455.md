# 如果你想加入工会，就在这些行业工作

> 原文:[https://life hacker . com/work-in-these-industries-if-you-want-to-be-in-a-union-1848986455](https://lifehacker.com/work-in-these-industries-if-you-want-to-be-in-a-union-1848986455)

在过去的一年里，各种行业 出现了一股工会化热潮，以至于你可能甚至没有意识到现在有许多公司和行业都有加入工会的员工(或为加入工会而斗争的员工)。如果你喜欢在某个你能从工会提供的保护中受益的地方工作(或者如果在你目前的公司 [组织](https://lifehacker.com/how-to-unionize-your-workplace-1845991328) 不是你愿意接受的挑战)，这里有一些可以考虑的——包括一些你可能没有想到的。(星巴克还没有登上榜单，但它已经登上了 。)

## **大多数行业都加入了工会(包括制造业、建筑业和电信业)**

当你想到工会时，脑海中可能会立即浮现出工会。汽车工人有工会。木匠有工会。电工有工会。甚至化妆师和发型师也有工会。。虽然每个行业都潜伏着动荡、剥削和不确定性，但加入工会可以让你得到一些保护，因为你可以很好地利用自己的专业技能。

## **公共部门的员工是工会员工的五倍**

数百万公共部门的雇员加入了工会。事实上， [公共部门加入工会的雇员几乎是私营部门](https://www.bls.gov/news.release/pdf/union2.pdf) 的五倍。有邮政工人、消防队员、气象服务雇员等的工会。在公共部门工作本身是有回报的，并伴随着一种特殊的成就感，但是你也可以——也应该——享受一些工人保护。

## 教师工会很强大

在新冠肺炎限制最严的时候，关于“教师工会”的讨论充斥着所有的有线新闻，根据网络的意识形态，这个工会听起来就像一个巨大的恶霸或者一股巨大的正义力量。底线是教师工会是强大的。

这些联盟(像纽约州立大学、美国艺术学院和UFT大学，尽管你会不时听说其他的联盟)代表和支持美国成千上万的教师。这很好，因为众所周知，教育工作者报酬低，支持少，尽管他们在塑造我们国家的未来方面发挥着巨大的作用，他们指导着有朝一日将成为领导人——以及工人——或许在他们自己的工会中。

## 大多数主要航空公司的飞行员都加入了工会

JetBlue、Delta、United等公司的飞行员通过代表64，000多名飞行员的航空公司飞行员协会成立了工会。最近，随着新冠肺炎限制在全国范围内实施，这些工会变得更加明显。例如，西南航空公司飞行员协会去年9月起诉了该航空公司，指控其通过改变工作条件、改变规则和改变费率的方式违反了联邦劳动法。该工会代表公司的10，000多名飞行员。你乘坐的航班上的服务人员可能也加入了工会——达美航空是唯一一家没有乘务员工会的美国主要航空公司。无论你是驾驶飞机还是协助乘客登机，都有理由认为这是工会的工作。

## 大多数娱乐工作都加入了工会

无论你是想在电影、电视和录音中表演、导演或制作，都有工会为你提供的选择。在许多情况下，工会成员是在集合中工作的一个要求。从美国导演协会，到电影演员协会、作家协会和IATSE(国际戏剧舞台雇员联盟)，你可能对这里的一些相关联盟很熟悉。这些组织为更公平的工资和确保片场安全而斗争。

## **卡车运输和运输工会仍然拥有一些权力**

虽然全国350万卡车司机中只有2%加入了工会 ，但卡车运输是另一个工会历史悠久的行业。以货运部门为例，它代表了整个北美卡车司机的利益，包括数百家当地商店和数万名卡车司机，其成员不仅包括卡车司机，还包括码头工人、机械师、办公室职员等等。简单地说，即使你不想驾驶真正的卡车，这个行业也能为你提供有工会保障的就业机会。

## **媒体越来越工会化**

虽然许多报纸已经加入工会几十年了，但近年来，有一股强大的力量推动数百万人每天阅读和收听的数字媒体(包括网站和播客)加入工会。Lifehacker自己的编辑人员组成了工会。《内幕》、《大西洋》、《石板》、Gimlet Media、《洛杉矶时报》以及许多其他公司的员工也是如此。请记住，即使有工会的保护，媒体职业也很容易充满不确定性，因为行业的格局一直在变化。但是，虽然裁员的威胁将永远是一个现实，在一个工会组织的出路，你将能够稍微轻松一点，因为你知道你将有保障的保护，如遣散费。
