# 使用过滤器为您的缩放聊天增添趣味

> 原文：<https://lifehacker.com/spice-up-your-zoom-chats-with-filters-1846230111>

你知道你陷入了疫情生活的永无止境的循环中，当你再也无法鼓起热情去改变你愚蠢的虚拟变焦背景。我过度使用 Zoom 的默认“空间”图像已经有一段时间了——但是想象一下当我前几天去切换它的时候我有多惊讶，发现这个应用程序现在包括了一堆更多的虚拟背景*和*效果供每个人尝试。

Watch

要访问它们，请确保您已将 Zoom 更新到应用程序的最新版本。这样做在您的移动设备上应该是不言自明的。在桌面上，单击 Zoom 主窗口右上角的个人资料图像，然后选择菜单下的**检查更新**。

(虽然不能保证升级到最新版本的 Zoom 就能获得额外的功能，但掌握最新版本的更新会给你带来最好的机会。它对变焦安全性也很有帮助。好了，回到正题。)

点击左下角视频开关旁边的箭头，启动缩放呼叫并调出虚拟背景功能。选择**“选择虚拟背景”，**，你现在应该会看到一些新的选项——一个可爱的动画海滩，以及一个更简单的“背景模糊”功能。我不能决定我更喜欢哪个。

但是我们还没完成。点击右下角的小**“Studio Effects”**链接，你会得到一个弹出工具条，让你给自己指定新眉毛、小胡子和/或胡须，以及唇色。很可爱。谁不想要一双的尤金·利维眉毛？

确保你取消了“应用于所有未来的会议”,这样你就不用担心忘记来调整这些效果，当你从你的朋友周末聚会切换到你的下一次工作聊天时。
