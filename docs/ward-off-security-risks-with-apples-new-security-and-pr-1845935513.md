# 使用苹果新的安全和隐私指南抵御安全风险

> 原文：<https://lifehacker.com/ward-off-security-risks-with-apples-new-security-and-pr-1845935513>

我们试图涵盖你能用 iPhone 做的所有事情，但即使你觉得你已经掌握了苹果的智能手机，你也总能学到一些新东西。为了帮助对可以访问你的设备、数据和账户的人(和公司)保持警惕，苹果发布了一份超级有用的支持指南，带你了解各种苹果设备的大量关键功能。

Watch

这可能不是最精彩的阅读——除非你是我——但是 [这是一份文件](https://manuals.info.apple.com/MANUALS/1000/MA1976/en_US/device-and-data-access-when-personal-safety-is-at-risk.pdf) 你绝对应该花至少 10 分钟浏览一遍，然后或者将它加入书签或者保存到你的系统中，这样你就可以在以后需要帮助锁定你的设备(或者 Apple ID)时参考它。或者，更糟糕的是，如果你真的遇到了某种安全或隐私问题，需要尽你所能来解决它。

虽然我强烈建议你跳到苹果文件的第 18 页(在那里苹果进入了快速点击的安全和隐私清单)，但我也浏览了苹果“当个人安全受到威胁时的设备和数据访问”文件的全文，并提取了我能找到的所有有用的提示。

我最喜欢的包括(强调我的):

### 保护您的 Apple ID 的安全

> 不要和任何人分享你的 Apple ID 密码，即使是家人。如果您共享一个 Apple ID，则意味着您允许其他人访问您的所有个人数据和内容。**如果别人设置了你的 Apple ID 和密码，你应该更改你的密码。**您可以使用“家人共享”来共享信息和服务，而无需共享帐户。

> *对您的 Apple ID 使用双重认证。**双因素认证旨在确保即使有人知道您的密码，您也是唯一可以访问您帐户的人。**借助二元身份验证，您需要提供您的密码和六位验证码，当您首次登录新设备时，该验证码会自动显示在您信任的设备上。*
> 
> 关注关于你的 Apple ID 的通知。**当你的账户发生变化时，苹果会通过电子邮件、短信或推送通知通知你，比如当你第一次在新设备上登录时，或者当你的密码被更改时**，所以保持你的联系信息最新是很重要的。

### 仔细检查你的应用程序隐私设置

> *在设置>隐私中，你可以看到你允许访问某些信息的应用程序——如定位服务、联系人、相机、文件&文件夹等——以及授予或撤销对这些信息的任何未来访问权限。*
> 
> *从列表中选择一种数据类型，查看哪些应用程序请求了使用该数据的权限。**一个应用程序在请求权限之前不会出现在列表中，你可以授予或移除任何请求访问的应用程序的权限。仅当您授予应用程序权限时，应用程序才可以使用设置中的数据类型。**T3】*

### 确认没有人通过“查找我的”跟踪您

> 若要查看与您共享位置的人，请转到“联系人”选项卡。当有人与您分享他们的位置时，您可以选择分享或不分享您的位置。
> 
> 您可以在“联系人”标签中停止与特定的人共享您的位置。只需选择联系人，向下滚动并选择停止共享我的位置。或者，您可以通过关闭“我的位置”选项卡中的“共享我的位置”来停止与所有人共享。
> 
> 如果您停止在“查找我的”中共享您的位置，此人将不会收到通知，但他们将无法在他们的好友列表中看到您。如果您重新启用共享，他们会收到您已开始与他们共享您的位置的通知。
> 
> *如果您通过 iMessage 而不是“查找我的”应用程序共享您的位置，并且您停止了共享，您停止共享的人会看到您已停止共享您的位置的通知。*

### 阻止应用和服务访问你的位置

> *要停止与应用程序和服务共享您的位置，即使是很短的一段时间，请前往设置>隐私>位置服务并关闭位置共享。这将阻止设备上的所有应用程序(如“地图”)使用您的位置。**如果您关闭定位服务，不会通知任何人，但是如果无法访问您的位置，功能可能无法按预期运行。**T3】*
> 
> *在设置>隐私>位置服务中，您还可以授权单个应用程序访问您的位置，并指示该应用程序使用您的位置的频率。*

### 停止与他人共享您的照片

> *要查看您与谁共享了相册，请前往照片>相册>共享相册。选择一个共享相册，然后轻按“人物”标签以查看共享相册的所有者以及相册的共享者。*
> 
> *如果您是相册所有者，并且想要停止共享，请点击订阅者的姓名查看选项。*

### 停止与其他人共享您的日历事件

> *若要查看您与谁共享了日历，请前往日历>日历。选择一个共享的日历，然后轻按“简介”图标以查看与谁共享了该相簿。*
> 
> *如果您是日历所有者，并且想要停止共享，请轻按订阅者的姓名查看选项。如果您是订阅者，您可以从屏幕底部选择删除日历。*

### 确定您的 iPhone 或 iPad 上没有任何未知的配置描述文件

> *如果您看到您的设备上安装了一个描述文件，但不知道安装的原因，您可以移除它并删除任何相关的应用程序。如果您的设备属于您的学校或组织，请在删除必要的应用程序或描述文件之前咨询您的系统管理员。*
> 
> *要从 iPhone、iPad 或 iPod touch 中移除描述文件或 MDM 配置，请前往设置>通用>描述文件&设备管理。*选择描述文件，轻按“删除描述文件”，然后按照屏幕指示进行操作。重新启动您的设备。*
