# 煮鸡蛋的“5-5-5”即煮法是一种令人厌恶的烹饪方法

> 原文:[https://life hacker . com/the-5-5-5-5-instant-pot-method-for-hidden-eggs-is-a-1847676087](https://lifehacker.com/the-5-5-5-instant-pot-method-for-hard-boiled-eggs-is-a-1847676087)

我没有做过正式的研究，但我猜我写的关于鸡蛋的东西比我写的其他任何东西都多(除了我自己)。鸡蛋是一个奇迹。它们非常多才多艺，能填饱肚子，有营养，而且它们能以各种令人兴奋的方式对热和化学处理做出反应。我喜欢它们——尤其是金蛋黄——这就是为什么“5-5-5”即食锅法的流行总是让我感到厌烦。

Watch

对于那些不熟悉这种方法的人来说，它非常简单:你在高压下煮鸡蛋五分钟，然后让它们在封闭的速溶锅里再呆五分钟，然后再冷藏五分钟。我对前五分钟和最后五分钟没有疑虑——他们可以留下来——但中间的五分钟对我来说是个问题。

你得到的不是许多食谱博客承诺的“奶油般金黄的蛋黄”，而是处于某种可怕边缘的蛋黄。上面的蛋黄在加入大量蛋黄酱时很好，但太干了，只能用一点点盐来吃，这是你应该能够吃一个煮熟的鸡蛋的方式。

在烹饪时间过去后，鸡蛋在速溶罐中停留五分钟，总是处于获得硫磺灰色环的边缘，这意味着只需要一分钟的遗忘就可以召唤它，但这也意味着你错过了刚刚凝固的真正奶油蛋黄的美丽。有些人可能会称之为“半熟”，我猜你也可以这样做，如果你必须这样做的话，尽管我仍然认为上面照片中的蛋黄是你应该让你的蛋黄变得最硬的。

我的意思是，不要低估自己。归结为偏好，但我的偏好是正确的。虽然你可能已经被无数的烹饪书、博客和杂志告知，完美的煮熟的鸡蛋看起来像本博客中的第二张鸡蛋照片，但我敦促你挑战旧的方式，想想什么味道好。如果你必须把蛋黄煮到接近粉状才能吃，也许你并不那么喜欢蛋黄！(唯一一次我会接受更硬的蛋黄，如果它伴有酥脆、黄油煎白、，但即使这样也有 [有限的应用](https://lifehacker.com/hard-fried-eggs-deserve-love-too-1846181764) 。)

拒绝“5-5-5”，拥抱“5-5”快速五分钟的高压，然后立即手动释放和冰浴(或者如果你打算立即食用它们，在冷水下冲洗)是你所需要的全部，因为煮得如此完美的煮鸡蛋，你不需要折磨它们就可以食用。(他们还会做更好的芥末蛋，这是额外的收获。)
