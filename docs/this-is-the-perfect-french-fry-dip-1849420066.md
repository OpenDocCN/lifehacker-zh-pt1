# 这是完美的薯条蘸酱

> 原文：<https://lifehacker.com/this-is-the-perfect-french-fry-dip-1849420066>

作为一个社会，我们对炸薯条做了很多事情。美国人把它们浸在番茄酱(有时还有芥末)里，英国人用麦芽醋泼它们，大部分欧洲人用蛋黄酱蘸它们，加拿大人用肉汁蘸它们。我们用辣椒、奶酪和辣椒奶酪覆盖它们，有些人用牧场调料旋转它们，但你很少——如果有的话——看到它们蘸黄油，这很奇怪，当你想到薯条是土豆，而土豆喜欢黄油。



在最近一次去(又热又粘的)阿拉巴马海湾的旅行中，我点了一磅蓝色的蟹爪，里面放了一池难以置信的大蒜黄油。我的男朋友选择了炸虾和炸薯条，我冒昧地偷了一些炸薯条蘸了蘸大蒜蟹黄油。

我不认为知道这个组合很好吃会让你震惊。滑溜的、辛辣的、注入海鲜的黄油渗透到酥脆的油炸食品中，创造了一种非常令人愉快的、有点油腻、咸咸的黄油味道。再说一次，土豆和黄油的组合并不新鲜，这就是为什么我这么困惑，用黄油蘸薯条不算是“一件事”

幸运的是，*我们*可以把它变成“一件事”我们要做的就是开始用黄油蘸薯条。(对我们来说容易！)你可以将薯条蘸上软化的室温黄油(就像我处理薯片一样)， [拉丝黄油](https://lifehacker.com/the-difference-between-clarified-butter-and-ghee-1849143100) ， [褐黄油](https://lifehacker.com/the-secret-to-the-best-browned-butter-1840046998) ，蒜泥黄油，或者 [复合黄油](https://lifehacker.com/how-to-make-steakhouse-butter-at-home-1847102251) 。我想他们会特别喜欢 [黑蒜酱](https://lifehacker.com/how-to-make-black-garlic-butter-1833065260) 。

如果你希望加入额外的乳制品，你可以在 [中加入一点冰岛酸奶](https://lifehacker.com/why-you-should-add-a-little-yogurt-to-your-butter-1832877504) 来制作丹吉尔蘸黄油，或者 [将你的黄油和布里干酪](https://lifehacker.com/brie-butter-is-the-best-compound-butter-1833264190) 混合来制作奶酪。但是先从一块不起眼的脱脂黄油开始。我想你会发现它非常令人愉快，一点也不无聊。
