# 如何免费直播奥运会

> 原文：<https://lifehacker.com/how-to-stream-the-olympics-for-free-1847345397>

2020 年奥运会— 将于 2021 年— 举行，在运动员 [新冠肺炎](https://www.usatoday.com/story/sports/olympics/2021/07/22/tokyo-olympics-team-flight-investigation-covid-cluster-czech-team/8052341002/) 检测呈阳性，以及许多日本公民表示 [他们不希望奥运会继续](https://www.latimes.com/world-nation/story/2021-05-25/tokyo-olympics-loom-fears-mount-over-thousands-visitors) 的不安定气氛的背景下，今天正式开始了一个可疑的开端。



但是，尽管有许多小插曲，奥运会正在东京举行，这意味着你可能应该观看他们，因为即使这是一个阴郁的、冲淡的奥运会，它仍然是奥运会。如果你没有有线电视，你仍然可以观看大量的活动。

## 没有有线电视怎么看奥运

NBC 是美国奥运会的官方转播商，你可以通过普通的电视天线收听一些大型赛事。不过，这只能让你观看在 NBC 主要网络频道播出的大型赛事，比如游泳、体操、田径等。(如果你有有线电视，奥运会将通过以下网络播出:美国全国广播公司、美国全国广播公司、美国消费者新闻与商业频道、奥林匹克频道、和高尔夫频道)。

如果你没有电视，但有联网设备，你可以在 NBC 的孔雀应用 T3 上免费观看一些比赛。同样，提供的项目会有些少，因为非订阅用户只能选择体育项目，包括篮球、游泳、田径、和其他旗舰项目。如果你不是付费用户，准备好观看广告，但是孔雀是一个很好的地方来弥补你可能错过的任何东西。

如果这两个选项都不可行，华盛顿邮报的[](https://www.washingtonpost.com/technology/2021/07/22/how-to-watch-tokyo-olympics-free/)**推荐 [Locast](https://www.locast.org/) ，它可以帮助人们通过互联网免费收看当地的广播电台。**

## **如何直播奥运会**

**虽然观看比赛的许多更详尽的选项不会完全免费，但它们也不一定那么贵。首先，会员模式会员费为每月 4.99 美元或一年 49.99 美元，任何计划都有七天免费试用。**

**奥林匹克网站和 NBC 体育应用也提供流媒体服务，但这两个选项都需要有线电视订阅。根据*邮报*的报道，一旦你用你的有线电视凭证登录，你就可以获得大量的内容，或者大约 5500 小时的价值。然后，当然是更昂贵的流媒体选项，你可能会订阅远远超出奥运会的服务，如 Hulu Live TV、富博 TV 和 YouTube TV，所有这些都是每月 64.99 美元。**

**免费观看全部比赛几乎是不可能的，但是免费观看部分比赛——或者以低价观看全部比赛——是可能的(T2)。**
