# 如何使用 Firefox 87 的“智能块”进行私人浏览

> 原文：<https://lifehacker.com/how-to-use-firefox-87s-smartblock-for-private-browsing-1846546913>

火狐的粉丝们会想要最新版本的浏览器(87)，因为它带有一个全新的 SmartBlock 功能，旨在用本地抄袭脚本替换潜在的有问题的 web 脚本。重点是？确保浏览器正确阻止网站跟踪*，而*不会在此过程中破坏网站。

Watch

换句话说，你可以两全其美:当你的浏览器禁止某些脚本运行时，强大的隐私保护和较少的破坏网站的错误。

正如 Mozilla [中写的](https://blog.mozilla.org/security/2021/03/23/introducing-smartblock/) :

> 在私人浏览窗口和严格模式中建立这些超强的隐私保护时，我们遇到了一个根本性的问题:引入一个直接阻止网络跟踪器的政策，不可避免地会阻止一些网站正常运行所必需的组件。这可能导致图像不显示、功能不工作、性能差，甚至整个页面根本无法加载。

尽管该公司在其帖子中没有深入讨论 SmartBlock 的技术细节，但它指出，这些“替身与 Firefox 捆绑在一起:根本没有加载追踪器的实际第三方内容，所以他们没有机会以这种方式追踪你。”当然，浏览器用来替换跟踪脚本的替身本身是不受任何跟踪的。

提醒一下，当你启用 Firefox 的严格跟踪保护模式时，你将从 SmartBlock 中受益。打开火狐的**选项**，点击隐私&安全。从那里，确保你已经启用了它的**增强跟踪保护**的“严格”版本:

请记住，您也已经这样做了，因为如果您访问的网站持续中断——即使 SmartBlock 自动启用——您可能会想要返回并将增强跟踪保护下调为“标准”

Firefox 87 的其他变化包括，当你使用浏览器的查找功能中的“高亮显示”选项时，滚动条上会出现新的刻度。此外，如果你在 macOS 上使用 Firefox，浏览器现在完全支持 VoiceOver 辅助阅读技术。你可以通过在 macOS 的**系统偏好设置>辅助功能**中启用该功能来验证这一点。
