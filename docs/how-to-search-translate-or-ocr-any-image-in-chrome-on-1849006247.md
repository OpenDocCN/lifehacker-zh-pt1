# 如何在桌面上搜索、翻译或 OCR Chrome 中的任何图像

> 原文：<https://lifehacker.com/how-to-search-translate-or-ocr-any-image-in-chrome-on-1849006247>

谷歌眼镜是一个了解你周围世界的有用工具。你可以用它来扫描一个 [二维码](https://lifehacker.com/you-should-know-how-to-scan-a-qr-code-from-a-screenshot-1848929906) ，用你在商店里看到的一件衣服的图片搜索网页，或者用外语翻译文本。虽然我们许多人在我们的 Android 和 iphone、上使用这一功能，但它也可以通过 谷歌 Chrome 在桌面上使用。



在你的电脑上，谷歌镜头的工作方式略有不同。你可以在任何保存的图像上使用该功能，或者在你正在浏览的任何网站上使用该功能，而不是实时摄像头视图。谷歌眼镜可以扫描你正在看的任何东西，找到相似的图片，识别物品等等。如果图像中有文字，你可以选择并复制它，或将其翻译成英语(或任何其他语言)。

只要你使用最新版本的 Chrome，你就会发现谷歌镜头功能被塞进了右键菜单。在你想要扫描的的图像上点击右键，选择**用谷歌镜头**搜索图像。 " 如果您想在页面上扫描多张图像，请右键单击屏幕的任意空白部分，然后选择 " **使用谷歌镜头搜索图像**。

触发搜索将在侧边栏中调出谷歌镜头功能，或者你可以使用右上角中的**展开**按钮在新标签中打开标题。您将在图片下方看到三个主要选项卡:“**搜索**、“**文本**”和“**翻译**”搜索功能将帮助您找到图像，或识别图像中的项目。

要快速复制图像中的任何或所有文本，请切换到“**文本**功能。图像中任何识别出的文本将自动高亮显示。U 用光标选中要复制的文本块，和可以点击**复制**按钮。

S 切换到“**翻译**功能打开谷歌翻译。识别出的 t ext 将自动在图像中翻译，你还会看到它出现在右侧的文本框中。在那里，您可以更改翻译语言，而可以复制翻译后的文本。如果您选择“听”，该功能还会为您读出翻译后的文本
