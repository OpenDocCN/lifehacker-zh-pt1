# 重新考虑那个匿名的情人节礼物

> 原文：<https://lifehacker.com/reconsider-that-anonymous-valentines-day-gift-before-it-1846258995>

去年情人节，你写给暗恋对象的善意匿名情书不会让任何人神魂颠倒，今年也不会。也许你认为发送一封深情、隐晦的信会让收件人头脑转动，激发希望，有人(你)注定会让他们神魂颠倒，但不幸的是，你可能想忘记这一切。在我们这个时代，爱情并不遵循故事书的脚本，如果它曾经有过的话，这也是为什么一封匿名的情书、花束或一盒巧克力会砰的一声落入某人的垃圾桶的原因。



### 可以看作是不成熟

当你在三年级时，对告诉别人你喜欢他们感到害羞是一回事，但当你是一个成年人时，表现出害羞的三年级学生的情感成熟是另一回事。如果你真的想开始一段感情——即使你只是想试试水——最好是直截了当。回避你的感受并不意味着你一定会成为一段关系中的情感基石。

你也必须问问自己，模糊你的身份有什么意义。是因为你表白爱的那个人正在谈恋爱，而你不想从中挑拨离间，尽管你是。你是否试图通过让他们感受到礼物的飘动，然后让自己和礼物配对，来欺骗他们喜欢你？你在情人节签一封匿名情书并没有真正帮到任何人。

### 这可以被视为令人毛骨悚然(尤其是对女性而言)

正如电影中发生的那样，一朵匿名的玫瑰如果送给一个陷入无爱婚姻的女人，可能会产生奇迹。也许发送者似乎真的理解这位女士，而她的丈夫可能不太在乎。这是一个流行文化比喻的原因:因为它很少，如果有的话，以这种方式出现。

如果你是个男人，你还必须考虑到一个女人可能会看到一封匿名信，担心自己被跟踪。食肉动物的存在是一个可悲的现实，知道给你寄爱心包裹的人可能并没有真正的好意，这可能是一个冰冷的安慰。

### 你可能会被误认为是别人

你愿意允许多少事与愿违的空间？在几个可能的灾难场景之一中，你爱的对象可能会错误地认为你的邮件是别人发的。如果你的暗恋对象找到了那个人呢？显而易见的是，如果你故意忽略你的名字，你的感情可能会被忽视，谁会真正从中受益呢？

所有这些都是说，你应该感到舒服，在这个情人节送某人一张情书或礼物，只是要确保它也是完全透明的，礼物来自谁。
