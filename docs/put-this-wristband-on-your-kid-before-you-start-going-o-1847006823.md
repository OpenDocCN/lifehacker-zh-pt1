# 在你再次进入人群之前，把这个腕带戴在你的孩子身上

> 原文：<https://lifehacker.com/put-this-wristband-on-your-kid-before-you-start-going-o-1847006823>

既然今年夏天我们可能真的能够在自己家外面做一些事情，我们可以回去担心正常的、非疫情式的养育问题，比如:如果我们在人群中丢失了我们的小孩，我们如何找到他们？ [抖音](https://www.tiktok.com/@adorethemparenting/video/6964410099121048838) 的一位家长设计了一个腕带形式的固体解决方案。



[宠爱他们养育](https://adorethemparenting.com/book/) 正在把那些我们经常与音乐会、交易会或节日联系在一起的有点烦人的纸腕带和重新利用它们成为儿童定位设备。只需购买一包特卫强腕带 并将它们(和一支记号笔)放在你的车里或包里。当你到达你的大型或拥挤的目的地时，在手镯上写下你的名字和电话号码，并固定在他们的手腕上，[like so](https://www.tiktok.com/@adorethemparenting/video/6964410099121048838):

 [https://lifehacker.com/embed/inset/iframe?id=tiktok-6964410099121048838&autosize=1](https://lifehacker.com/embed/inset/iframe?id=tiktok-6964410099121048838&autosize=1) 

此外，t 每个你的孩子 1)如果他们与你走散，呆在原地不动，2)向路过的成年人( [最好是另一位父母](https://lifehacker.com/teach-your-kids-to-find-a-parent-if-they-get-lost-1838253539) )展示他们的腕带，以便成年人可以打电话给你，告诉你他们被找到了，并告诉你他们在哪里。

如果你不想整个夏天都使用一次性纸质腕带，你可以用反复使用的硅胶腕带做同样的事情(如果上面已经印了另一条信息，只需将它翻过来写在空白的一面)。你家里可能已经有一些这样的东西了，但是你也可以在网上购买。
