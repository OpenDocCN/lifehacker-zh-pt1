# 如果你需要更多的肉，可以做这种以胸肉为基础的辣椒

> 原文：<https://lifehacker.com/make-this-brisket-based-chili-if-you-need-the-most-meat-1845666823>

大家好，猫咪们，欢迎来到另一个美味的煎锅辣椒烹饪比赛，这个季节性烹饪比赛让我(和我的冰箱)充满了辣炖菜。

Watch

本周的竞争者，来自评论员乔希，与我们迄今为止品尝过的美食完全不同。虽然任何一碗辣椒的焦点通常是肉，但在这种特殊的食谱中，这种焦点是激光般锐利的。不仅里面没有豆子，而且其他东西也很少，而且很明显，每一种非肉类成分都是为了让一口大小的胸肉和鸡胸肉看起来很好。

只有洋葱、西红柿和一些辣椒(墨西哥胡椒和烤波布拉诺)，没有太多的肉藏在后面，这是一件好事。香料很少，但都是精心挑选的。辣椒粉是凤尾鱼，红辣椒是烟熏的，孜然是烘烤的，然后*再*磨成粉末。也有一些干百里香，这是我没有想到的，但它的贡献有点翠绿的草药效果很好。所有的东西都为这种肉增色不少:有一点烟熏味，有点酸味，还有一点点辣味，还有几个手工碾碎的西红柿和略带水果味的辣椒带来的一丝甜味。简而言之，这种辣椒的味道和含量都很丰富，而且很棒。

在肉开始变褐并炒洋葱和墨西哥胡椒后，加入香料、番茄和烤波布拉诺，以及大量的廉价啤酒，所有东西都炖几个小时，直到肉变嫩，蔬菜几乎溶解成黑色和美味的不完全肉汤。

像辣椒，乔希的评论是削减和直截了当。我把它放在这里，在另一边和你碰面。
