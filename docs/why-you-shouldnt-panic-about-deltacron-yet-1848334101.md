# 为什么你还不应该对“Deltacron”感到恐慌

> 原文：<https://lifehacker.com/why-you-shouldnt-panic-about-deltacron-yet-1848334101>

上周是 [氟罗娜](https://lifehacker.com/what-is-flurona-and-should-we-be-worried-about-it-1848320894)T3。本周的主题是“deltacron”当我们听到一个新的变种时，我们已经准备好去“哦，妈的”,因为我们已经被相当多的变种烧伤了。但是这些可怕的名字比我们*实际上需要害怕的东西还要多。* 



不久前，我们被警告说“ [”德尔塔加上](https://lifehacker.com/the-difference-between-delta-delta-plus-and-lambda-a-1847423234)“T3。”这最终不了了之。另一方面，德尔塔是真实的，它的尖峰信号尤其糟糕，至今仍未消失。奥米克隆是真实的，而且肯定仍然是一个问题。那么，我们如何知道需要担心哪些变体呢？ 

我给你一个大线索:恐慌的时候不是有人给病毒起了一个吸引人的新名字的时候。你可能认为恐慌的时间是当一个变种进入 [世界卫生组织的“关注变种”名单](https://www.who.int/en/activities/tracking-SARS-CoV-2-variants/) 。目前，该列表有五个条目:α、β、γ、δ和ω。(还有第二个列表，感兴趣的变量，目前只包含 Lambda 和 Mu。一个变种可能从 VOI 开始，然后升级为 VOC。)

## 为什么变体即使不是什么大事也会成为头条新闻

有两个理由不要过早恐慌。一个是变体必须在现实世界中相互竞争。这是自然选择在起作用:如果一个变种不善于自我复制，它就不会走得太远。请记住，通常引起关注的变异是因为它们比以前更容易传播。Delta 只是比原味 COVID 更容易抓到。奥米克隆可能比德尔塔更容易传播。因此，如果有一种新的变种看起来很可怕，但它很难传播开来，它不太可能成为大问题。

但还有另一个原因:有时新的可怕名字*甚至不是指真正的病毒*。我们上周才讨论过这个，记得吗？几个人同时感染了 COVID 和流感，突然出现了一百个关于“氟罗娜”的头条新闻

[氟罗娜不是杂交病毒，也不是独特的威胁](https://lifehacker.com/what-is-flurona-and-should-we-be-worried-about-it-1848320894) ，只是一个偶然的、不幸的巧合。医生和科学家当然应该关注合并感染，例如在合并感染需要特殊治疗的情况下。但现在还不是恐慌的时候。

这让我们想到了 deltacron，它乍看起来像是两种 COVID 病毒的实际杂交体。塞浦路斯的一些科学家报告说，他们发现了包含奥米克隆的 Delta *和*突变的 RNA 序列。但专家对此表示怀疑。

“这几乎可以肯定不是 Delta 和 Omicron 血统的生物重组，”COVID 基因组学专家 Jeffrey Barrett 对英国科学媒体中心 说。他指出，假设的突变与测试过程中可能出现的 [已知技术问题](https://wellcomeopenresearch.org/articles/6-305/v1) 一致，导致了混血儿的表面外观。

其他科学家指出，新的 deltacron 序列在所有 COVID 变体的族谱上都不合适。如果德尔塔和奥米克隆真的是在一次反常的事故中走到一起，现在有 25 人感染了这种新病毒，你会希望所有 25 个样本看起来彼此相似，并在大的病毒家族树上彼此相邻出现。他们没有。大多数专家在这些数据中看到的是一个强有力的暗示，即实验室中存在污染，并且污染出现在另外 25 个非常不同的样本中。

所以 deltacron 不是真的？嗯，*大概*不会。最先发现新序列的团队说 [他们仍然认为他们发现了真实的](https://www.bloomberg.com/news/articles/2022-01-09/cypriot-scientist-says-covid-19-variant-deltacron-not-an-error) ，这将需要更多的研究来弄清楚到底发生了什么。但是即使真的有 deltacron，我们也没有任何理由相信它是一种异常特殊或危险的病毒。如果是的话，我们很快就会知道了。在没有任何值得担心的证据之前，没有必要担心。
