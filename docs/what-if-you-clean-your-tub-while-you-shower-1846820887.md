# 如果你洗澡的时候清洗浴缸会怎么样？

> 原文：<https://lifehacker.com/what-if-you-clean-your-tub-while-you-shower-1846820887>

清洁浴缸的每一步 看起来都被设计得尽可能的烦人。你必须跪下来给它适当的擦洗，这在最好的情况下是身体上的笨拙，在最坏的情况下是痛苦的。清洁墙壁是不可能的，除非你真的踩进去，事实上我只记得后的*我已经清洁了浴缸。冲洗应该是容易的部分，但它需要很长时间——即使是用手持淋浴头。*

避免这一切的最好方法是在洗澡时清洗你的浴缸。它更快更容易，原因有很多，包括蒸汽和热水可以软化肥皂浮渣和其他残留物，这样你就不用那么用力擦洗了，而且你可以触及每一寸墙壁，因为你就站在那里。

清洁沐浴用品很容易:将洗发水和其他物品放在浴缸的一端，擦洗搁板，然后在放回瓶子之前快速冲洗一下。说到冲洗，如果你不在乎淋湿，那就容易多了——尤其是如果你的淋浴头有手持附件的话。(如果不行，带上一个大的不易碎的杯子，用它来冲洗墙壁。)整个过程可能需要五分钟，所以你更有可能经常这样做，以保持淋浴和浴缸清洁。

一根充满肥皂的洗碗棒 让淋浴清洗更加容易，但你并不需要。带上一些表面清洁剂和一块海绵或布，然后去镇上。避免使用可能刺激皮肤的清洁产品——我用的是稀释过的 BioKleen，和其他产品一样，用的是超细纤维毛巾。大多数清洁产品并不比洗发水或护发素泡沫更滑，但是如果在湿漉漉的淋浴中俯下身听起来有点太危险，那么在踏入浴缸之前，快速擦洗一下浴缸的四周。当你出现的时候，你的浴缸将会和你一样干净。
