# iPhone的“焦点过滤器”可以帮助你更好地忽略干扰

> 原文:[https://life hacker . com/the-iphone-focus-filters-can-help-you-ignore-distracti-1849354328](https://lifehacker.com/the-iphones-focus-filters-can-help-you-ignore-distracti-1849354328)

iPhone的聚焦模式是一个增压版的勿扰模式，随着iOS 16的更新，它获得了新的上下文功能。在过去的一年里， [对焦模式](https://lifehacker.com/you-should-definitely-use-ios-15s-new-distraction-squa-1847671349) 已经成为想要将工作与生活分开或进行特定活动的用户生活中不可或缺的一部分。现在你可以使用一个叫做焦点过滤器的新功能，在使用焦点模式时自动隐藏特定的消息或电子邮件——，你可以 [自动执行这个操作，也可以](https://lifehacker.com/10-ways-you-re-not-using-focus-on-iphone-but-should-be-1848497964) 。

它的工作原理是这样的——你启用一种专注模式，比如睡眠或锻炼，iPhone可以自动隐藏你的工作账户中的信息、电子邮件、Safari标签和日历约会。它不仅会静音收到信息的通知，当你打开应用程序时，就像对话或信息从来没有存在过。

(**注**:这是iOS 16更新中的新功能，目前有一个[public beta](https://lifehacker.com/how-to-install-the-ios-16-and-ipados-16-betas-1849023051)的版本。预计在f all 中会有更广泛的展示。)

进入**设置** > **对焦**，选择想要定制的**对焦模式**。从**对焦滤镜**部分，点击“**添加滤镜。** " 你会在这里看到四个应用过滤器:

*   **日历**:选择一个特定的日历，在焦点模式下隐藏所有的约会。
*   **邮件**:如果你在邮件应用程序中使用多个账户，你可以选择在焦点模式期间隐藏单个邮件账户。
*   **消息**:消息应用程序可以帮助你过滤掉来自除选定联系人之外的所有人的消息。例如，你可以使用“家庭”焦点，只接收来自朋友和家人的消息，过滤掉工作消息。
*   **Safari** :如果你使用Safari中的标签组功能，你可以选择在使用焦点模式时隐藏特定的标签组(这也适用于共享标签组，)。

花些时间自定义应用过滤器。完成后，点击**完成，**和c 选择“**添加**选项，将滤镜添加到对焦模式。您可以在不同的聚焦模式下为多个应用过滤器执行此操作。
