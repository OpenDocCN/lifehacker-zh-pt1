# 哪里适合摸人？老白人的解释者

> 原文：<https://jezebel.com/where-is-it-appropriate-to-touch-people-an-explainer-f-1847415950>

纽约州总检察长办公室的一份官方报告发现，安德鲁·科莫的确骚扰了许多对他提出指控的女性，这让纽约州州长大为光火。有益的是，他用一个视频 回应了调查结果，坦率地承认，是的，他确实经常以可能被解释为不恰当的方式触摸他遇到的似乎每个人。

“我和每个人都这样做，”库莫对着镜头说，一张幻灯片展示了他把自己的手放在没有任何迹象表明他们愿意触摸的人身上。“黑人和白人，年轻人和老年人，异性恋者和同性恋者...朋友，陌生人，我在街上遇到的人。”总的来说，所有的人类都遭受过科莫不必要的触摸，所以抱怨的女性并不特别。

这种辩护对于某些一代白人男性(包括我们的现任总统)来说是恰当的，他们认为只要他们令人毛骨悚然的触摸不是抓屁股或摸乳房，就不构成性骚扰。对这些人来说，擦肩、轻抚上臂，甚至轻轻抚摸脸颊都是可以的，他们似乎从来没有被教导要考虑别人潜在的不适。他们似乎相信，地球上的每一个人都肯定能接受他们的触摸。

小时候，我们镇上的中老年人最喜欢玩的一个游戏是问小女孩是否想“看看马是怎么吃苹果的”。然后，在得到回答之前，他们会抓住我们裸露的膝盖，挤压我们，直到我们不由自主地大笑起来，让自己假装这个孩子正在享受对她身体自主权的侵犯。事实上，这是我第一次意识到我想以取笑这些混蛋为职业。

所以谢谢你让我开始我的职业生涯，你们这些可怕的老变态。现在，请允许我回报你，给你一张清单，上面列着不要接触他人的地方。

如果你不在工作，问对方是否愿意亲吻脸颊。

在工作场所之外，大多数你不熟悉的人不希望你把手放在他们的脸上。

询问任何不是家人或密友的人，他们是否愿意在所有不涉及空中飞人的场景中牵手。

除非你是专业按摩师。即使这样——只有当这个人已经明确支付、要求和/或同意这些服务时。

发疯吧。这是任何人的事。
