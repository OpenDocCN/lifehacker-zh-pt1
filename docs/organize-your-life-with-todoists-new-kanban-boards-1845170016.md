# 用 Todoist 的新看板板组织你的生活

> 原文：<https://lifehacker.com/organize-your-life-with-todoists-new-kanban-boards-1845170016>

Todoist 已经非常适合管理项目和待办事项列表，但该应用程序新推出的将这些列表变成了可定制的——和协作的——看板。这项新功能面向所有人，包括免费用户。



Todoist 的 board view 包含了你期望从 [看板风格的项目管理工具](https://lifehacker.com/productivity-101-how-to-use-personal-kanban-to-visuali-1687948640) 中得到的所有功能。用户可以使用额外的列和过滤器来自定义他们的板，以适应他们的项目的工作流程，并向他们的板添加其他用户。您还可以随时在新看板视图和 Todoist 的传统列表布局之间切换。

所有用户都可以访问 board view 的核心功能，但免费用户和高级用户的更多功能有所不同:

*   免费用户可以获得一个预先制作的过滤器列表，他们可以使用这些过滤器来组织任务，例如“分配给我”和几个优先级。
*   高级用户可以创建额外的过滤器，在卡片上应用自定义标签，导入或导出白板模板，发送电子邮件任务，查看“日历提要”中的项目， [和更多](https://todoist.com/kanban-board) 每月 4 美元。这大约比 Trello 或 Asana 便宜 10 美元。

对高级会员的细粒度组织控制是一个不错的奖励，但寻找 Trello 或 Asana 替代品的免费 Todoist 用户也应该拥有他们需要的一切。

### 如何使用 Todoist 的董事会观点

1.  登录您的 Todoist 帐户，然后从左侧菜单中选择一个预先存在的项目或**“添加新项目”**。还有一个“试板”教程，里面的专栏和任务会引导你完成试板的核心功能。
2.  单击或轻触右上角的三点菜单图标，然后选择**“查看为板”**(您也可以按键盘上的**“Shift+V**)在视图之间切换)。

以下是 board view 中的一些主要功能:

*   选择**“Add Section”**为您的项目创建一个新列。您可以通过单击列名称右侧的三点图标来编辑、复制、移动、删除或存档列。
*   选择**“添加任务”**创建一张新卡片。您可以为每个任务卡添加计划、标签、优先级和提醒(以及在高级模式下添加过滤器)。
*   单击任务名称旁边的三点图标可添加子任务、注释、查看卡活动历史记录以及编辑卡的详细信息。
*   拖移卡片以在列之间移动它。
*   单击右上角菜单中的人形图标添加协作者。

Board view 现在面向所有 Todoist 用户推出。这是一个服务器端的更新，这意味着你不必做任何事情就可以将 boards 功能添加到你的帐户中，但需要由 Todoist 打开它。如果你还没有接触到董事会，给它一天左右的时间来展示。
