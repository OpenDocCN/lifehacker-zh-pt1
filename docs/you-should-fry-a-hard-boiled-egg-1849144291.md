# 你应该煎一个煮熟的鸡蛋

> 原文：<https://lifehacker.com/you-should-fry-a-hard-boiled-egg-1849144291>

我最喜欢的一些食物是蛋白质的实用来源。农家干酪、牛肉干、煮鸡蛋——当我意识到我早就不饿了的时候，我可以抓起这些东西往我脸上塞。(事实证明，成为一名美食作家并不会自动让一个人擅长识别饥饿信号。)煮鸡蛋是我最喜欢的早餐之一，但有时我想吃点温暖、有质感的东西，一个冰冷、光滑的煮鸡蛋是不够的。

幸运的是，煮熟的鸡蛋非常适合快速煎炸。

煎一个煮熟的鸡蛋很简单。将鸡蛋切成两半，撒上少许盐，在不粘锅中用中高火融化一些黄油(每个鸡蛋大约一汤匙)。一旦大部分泡沫消退，黄油即将开始变黄，加入鸡蛋，切下一面，让它们煎大约两分钟，直到边缘变黄变脆。尽量不要来回翻动，因为那会导致剧烈的砰砰声。(每次翻转鸡蛋时，水分都会滞留在鸡蛋扁平的一面之下，引起一次小爆炸。)一旦切面变成褐色，翻转鸡蛋，让底部受热。

这个鸡蛋是给谁的？它是为那些想要一份热早餐，但又不能忍受鸡蛋里有一点点松软的人准备的；这是给那些想要一个热鸡蛋，但已经吃过饭的人准备的，他们的鸡蛋已经煮得很熟了；是为了我，一个总是在寻找新的有趣的吃鸡蛋方法的人。

简而言之，边缘的酥脆质地和棕色黄油的烘烤味道让这个鸡蛋变得更加有趣和诱人。它本身很好，可以放在沙拉上或饭碗里，但它也可以用来做一个有趣的鸡蛋沙拉，或一个令人难以置信的 [未调味的鸡蛋](https://lifehacker.com/un-devil-your-eggs-this-easter-1848778518) 。油炸让一种明显功利的食物变得不那么功利，这在繁忙的工作日早晨是一个受欢迎的举动，特别是如果你容易忘记给自己喂食的话。
