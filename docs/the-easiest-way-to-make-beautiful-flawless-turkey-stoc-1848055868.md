# 制作美丽无瑕的火鸡高汤的最简单方法

> 原文：<https://lifehacker.com/the-easiest-way-to-make-beautiful-flawless-turkey-stoc-1848055868>

如果你还没有开始准备感恩节大餐，做火鸡汤是个不错的开始。它很省力，而且让你的家充满了一种美妙的香味，暗示着肉汁就要来了。但是当你打开高压锅，它看起来...不太好。你有没有想过，为什么你可以搅拌你原本完美的肉汁 20 分钟，它仍然是块状的？



遇到:硬块*。*肉不能很好地溶解在水中，通过将这些残渣从你的汤料中取出，你最终会得到那块- 免费的肉汁，以及更清澈的汤和天鹅绒般柔滑的酱汁。

那里有很多东西，即使通过筛子或脂肪分离器过滤，也会导致一些碎片通过。但是不要害怕，我的朋友们，因为有一个漂亮股票的秘密武器:坚果牛奶袋。

虽然它主要用于狄莺 alt 牛奶，但它本质上是带拉绳的尼龙粗棉布。它不贵——多包售价不到 10 美元——也许最重要的是，它可以用洗碗机清洗。虽然我仍然会用粗棉布来做任何需要长时间烹饪的东西，比如花束 garni *，*坚果牛奶袋最适合过滤，非常适合乳清干酪，希腊酸奶，切黄瓜丝做 tzatziki，夏天浆果糖浆，最重要的是，至少这个月——做非常美味的鸡肉或火鸡高汤。

使用袋子很容易。撇去或分离出脂肪后，将你的原料倒入一个内衬塑料袋的容器中。(四杯量杯是这一过程的绝佳容器，它使测量和倒入储存容器变得更加容易。)你会对你的发现感到惊讶。由于尼龙不像粗棉布那样具有吸收性，液体在坚果牛奶袋中流动更快。

一旦你把汤里的小块过滤掉，你就可以把它冷冻起来，直到你准备好做肉汁(或者用来做汤)。两者都是很好的选择，而且都不会有讨厌的、难看的碎片。
