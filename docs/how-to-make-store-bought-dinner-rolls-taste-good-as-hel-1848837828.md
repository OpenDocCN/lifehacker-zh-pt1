# 如何让商店里买来的晚餐面包卷尝起来美味无比

我最喜欢的黑客风格之一是“提升”商店购买的东西。制作一盒 [蛋糕](https://lifehacker.com/how-to-make-a-boxed-cake-mix-taste-homemade-1839165427) 或 [核仁巧克力混合](https://lifehacker.com/how-to-make-boxed-brownie-mix-taste-homemade-1829269063) 口味自制、 [精选蛋黄酱](https://lifehacker.com/how-to-make-store-bought-mayonnaise-taste-homemade-1848402829) 、用快餐鸡肉制作 [鸡肉沙拉](https://lifehacker.com/transform-leftover-fried-chicken-into-a-super-flavorful-1784395666)——这类东西是我的拿手好戏，宝贝。

商店里买的晚餐面包卷适合这种即兴表演。标准的做法包括用黄油(通常是大蒜黄油)刷它们，并在它们上面放东西(香草和奶酪等)，但你可以用它变得更奇怪，你应该这样做。毕竟，你只是通过外包面团节省了一大堆时间；你可以多花点时间去 zhoosh。

## 混合(分层)你的脂肪

在把面包卷放进烤箱加热之前——你应该经常加热它们——融化一点脂肪，然后刷在面包卷上面。黄油面包卷之所以受欢迎是有原因的:黄油味道不错，但其他脂肪味道也不错，如果不给它们一个机会，那将是一个错误。

融化的鸭油、鸡肉酱和培根油可以用来给你的面包注入烟熏味。就像涂黄油一样刷上去。然后，当它们从烤箱里出来时，再给它们刷一遍。你可以保持脂肪不变——做双份鸭油或培根油——或者你可以混合它们，用黄油结束，或者，如果你想要一种咸甜的体验，蜂蜜黄油(或者这个 [黄油航班](https://lifehacker.com/impress-your-guests-with-a-butter-board-1848138546) 的任何黄油)。

## 把柔软的东西放在里面

最好的晚餐卷在上面是棕色的，但是把新鲜的香草、大蒜和其他配料放在你的晚餐卷外面会导致这些香草和配料烧焦(或脱落)。把这些东西放在面包里可以防止这种情况，软化香草而不是烧焦它们；融化，而不是变褐，任何奶酪。

最容易做到这一点的是完全煮熟并附在一起的面包卷(像国王的夏威夷卷)，因为你可以一次将它们全部切开(就像我们对 [这些早餐滑块](https://lifehacker.com/make-these-sheet-pan-breakfast-sliders-for-christmas-mo-1831061103) 所做的那样)。将面包卷切成两半，注意不要将它们分开，然后在两面刷上一些油脂。撒上你的香草、奶酪、磨碎的大蒜等。放到下半部分，然后将另一半放回顶部。在顶部刷上脂肪，最后放上脆脆的盐片或帕尔马干酪(一种变褐后味道非常好的奶酪)，然后将它们放入烤箱或烤盘中烤熟。
