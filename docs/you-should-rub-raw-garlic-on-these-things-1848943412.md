# 你应该把生大蒜抹在这些东西上

> 原文：<https://lifehacker.com/you-should-rub-raw-garlic-on-these-things-1848943412>

生大蒜是一种强大的力量，可以用来创造惊人的微妙效果。最简单的方法之一是将生的、去皮的丁香切成两半，然后在碗、盘子或浅盘中摩擦。这样做可以给这道菜带来一点大蒜的味道和香气，而不会压倒其他更微妙的味道。



[美国的试验厨房](https://www.americastestkitchen.com/articles/5531-why-you-should-rub-your-salad-bowl-with-garlic) 建议为他们的超级简单沙拉准备碗，用切成两半的丁香摩擦它，给它一点活力。他们在金属、木制和陶瓷沙拉碗上尝试了这一举措，他们的品尝者“证实了无论材料如何，这都增加了明显的改善。”

但是沙拉碗并不是唯一受益于洋葱按摩的上菜器皿。在填充之前摩擦砂锅菜——用 [奶油脆皮](https://lifehacker.com/the-best-potatoes-au-gratin-dont-contain-any-cheese-1825751769) 、通心粉和奶酪，或 [牧羊人馅饼](https://lifehacker.com/crown-your-shepherds-pie-with-tater-tots-1845154781)——会给它一种诱人但温和的大蒜味。你也可以在任何上菜的盘子、碗或碟子上摩擦大蒜，然后再往里面装食物。把你的牛排放在抹了大蒜的盘子里，把烤芦笋堆在抹了大蒜的大盘子里，把汤和炖菜舀进抹了大蒜的碗里。

一旦你的餐具已经用大蒜适当地按摩过，你可以用剩下的蒜瓣涂在一些面包上。将一瓣生大蒜放在一片硬面包上摩擦是制作大蒜面包最简单、最有效的方法之一。面包片粗糙的表面会磨碎大蒜，吸收其汁液，使其渗透到面包中。简单地吃，在上面涂一些盐黄油，或者磨碎一些西红柿，让平底锅变得有味道。

哦，如果你担心你的木制沙拉碗和大盘子闻起来像大蒜，美国的测试厨房建议通过 [【烘烤】碗过夜](https://www.americastestkitchen.com/articles/3959-there-is-a-reason-you-should-be-baking-your-wooden-salad-bowl) 来去除这些气味。(我还没有尝试过，但可能要用我的几个更臭的砧板来做。)
