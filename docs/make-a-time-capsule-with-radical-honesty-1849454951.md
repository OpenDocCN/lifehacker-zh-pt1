# 用彻底的诚实制作一个时间胶囊

> 原文：<https://lifehacker.com/make-a-time-capsule-with-radical-honesty-1849454951>

我小时候很喜欢时间胶囊的想法。我过去常常抓起我不再想要的动作玩偶和弹簧玩具，埋在标有“20 年内不要打开 T2”的盒子里，想象未来的地球人 T4 穿着银色宇航服苦思冥想这些奇怪的东西是什么意思。当然，我忘了我把他们都埋在哪里了。



现在我老了，我对时间胶囊失去了所有的热情。这个想法是可靠的，但是我们放进去的东西却不可靠。无论是像由 [保罗·里维尔和山姆·亚当斯](https://www.youtube.com/watch?v=YRWerw7i8Ao) 埋葬的“官方”时间胶囊，还是在育儿节 中经历 [文化时刻的“家庭时间胶囊”，我们对它们包含的秘密的期待总是伴随着对其内容的失望。我们用精心策划的、讨人喜欢的纪念品——婴儿鞋、旧硬币、假期照片——填满时间胶囊，努力向未来展示我们最好的自己。时间胶囊不是历史，而是圣书——自私、无聊且毫无意义。但他们不必如此。](https://proudhappymama.com/15-family-time-capsule-ideas-what-to-put-in-them/)

## **用彻底的诚实制造更好的时间胶囊**

如果你是一个正派的父母，你会向你的孩子展示一个精心编辑过的你。你在洗澡的时候哭，而不是开车送他们去学校的时候。如果你对你的后代诚实地说出你有多么失落，每天你是如何即兴发挥的，那将是*可怕的*。

然而，当孩子们长大后，他们不想要一个粉饰版的爸爸妈妈。他们想要泥土。他们想把他们的父母看作真实的人，有缺点等等，以此来理解为什么事情会像他们那样发展。这就是我的诚实时间胶囊的用武之地。

## **在你诚实的时间胶囊里放些什么**

一个诚实的时间胶囊应该只包含你对你的孩子、配偶和其他爱人保守的秘密的纪念品。当然，具体情况取决于你个人的失败，但这里有一些想法可以帮助你开始:

*   一只蟑螂，纪念你在蹒跚学步的岁月里尽可能经常吸食大麻
*   你的实际高中成绩，包括 d 和 f，以及所有的纪律通知
*   列出所有你真正讨厌的人，以及为什么
*   一份你为治疗抑郁症、焦虑症以及其他任何疾病而服用的处方药
*   各种未付账单和驱逐通知
*   你婚姻咨询会议的记录
*   解释咬了隔壁小孩的小狗到底发生了什么(不，他没有被带到 T2 北部的农场)
*   一张便条，解释你暗地里讨厌你孩子的哪个朋友
*   一本你十年级时写的“小说”
*   给你配偶之前的任何伴侣的情书
*   与其在圣诞节尝试你的曾祖母的火腿食谱，不如保留一份你实际上每天喂你的孩子吃的冷冻鸡块和酸奶的清单，因为他们不会吃其他任何东西
*   你年轻时派对上的罪证照片
*   破产文书/逮捕记录
*   来自朋友和亲戚的手写纸条，他们被要求对你的品格做出诚实的评价 ( *在任何情况下，你都不应该自己阅读这些纸条*
*   如果你有不止一个孩子，最后透露你最喜欢的

## **把你诚实的时间胶囊埋在哪里**

你可以把一个“普通”的家庭时间胶囊塞到壁橱后面，因为除非妈妈强迫，否则没有人会打开它，但是你必须格外小心你的诚实的时间胶囊。如果你做得对，早期的揭露可能会导致心理创伤、勒索或逮捕，所以确保你把它藏在一个没人会发现的地方，直到你死后。也许是一个标有“报税收据”的盒子，或者是你十年级时写的小说的注释。

## **何时打开你诚实的时间胶囊**

这是至关重要的:不要让你诚实的时间胶囊被打开，直到你死后。你不想解释。但是你要确保里面有一个说明，说明这个材料要传给后代。我在想你是否知道你曾祖母保守的所有秘密。T2:这难道不比火腿食谱和古铜色婴儿鞋更有趣吗？
