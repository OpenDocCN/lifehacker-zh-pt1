# 如何组织感恩节砂锅交换

> 原文：<https://lifehacker.com/how-to-organize-a-thanksgiving-casserole-swap-1845715960>

作为一个来自南方的女人，我发现没有什么比一顿美味的砂锅菜更令人欣慰的了，尤其是在每年的这个时候。( [这个](https://skillet.lifehacker.com/an-ode-to-jiffy-corn-casserole-1830231857) 是我的最爱。砂锅菜几乎是一种爱的语言——如果有人给你一个砂锅菜，他们会说，“给，朋友，这是一顿完全不需要你做的饭。”我认为那很好。



今年我们也许不能聚在一起，但是我们肯定可以交换(也就是砂锅菜),这样,就能感受到的亲密，而*就不会*亲近。现在，取决于你的朋友和家人在哪里，这可能不像把一个砂锅放在他们家门口那么简单，但可以根据距离进行修改。无论参与者相距多远，都适用相同的规则:每个人都被随机分配一个砂锅伙伴(在线使用 [名称抽取生成器](https://www.drawnames.com) )，你要么将完整组装的砂锅交付给你的伙伴，要么向他们提供制作所述砂锅的方法。

### 如果你爱的人超级亲密

这是最简单的。做一个砂锅，用锡纸包好，放在他们的门口(确保他们先回家)。然而，放下一个热砂锅需要接收者准备好吃它，这就是为什么我热衷于将一个组装好的(但不是烘焙的)砂锅冷冻在一次性铝制盘子里。这样，你的朋友或家人可以在闲暇时享受一顿热腾腾的舒适的饭菜。只要确保任何肉类在冷冻前都完全煮熟。基本上，在把它放进烤箱之前，做你通常会做的一切事情，让它变成棕色，变得漂亮，然后用双层箔纸紧紧地包起来，冷冻起来。在锡纸上写下烹饪温度和时间，你就可以开始了。

### 如果你没那么近

如果你离你的目标太远，不能把砂锅放在他们的门廊上，考虑做一个砂锅护理包。你可以自己组装和运输不易腐烂的原料，或者使用杂货店送货服务，让所有的原料直接运到你的砂锅菜伙伴家。确保你也发送详细的说明，并使它们成为一个有趣的。如果你是用蜗牛邮件寄的，打印或手写在好纸上，用贴纸或其他东西装饰(我不狡猾)。如果你在网上发送，使用漂亮的模板，或者使用一些照片编辑软件，让它看起来*特别*。

如果你想让你的礼物变得特别，包括一些可爱但相关的额外物品，比如挂钩、餐具或漂亮的厨房毛巾。每个人都喜欢漂亮的厨房毛巾，就像喜欢砂锅一样。
