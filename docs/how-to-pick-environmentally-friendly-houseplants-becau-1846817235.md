# 如何挑选环保的室内植物，因为你的不是

当我们想到我们个人对环境的影响时，我们的第一想法可能是改用可重复使用的购物袋或者节约用水用电。但是我们的室内植物呢？

过去几年里，无数的文章列举了室内植物的好处——特别是声称它们能够“”净化空气但是除了有 [少到没有](https://www.nationalgeographic.com/science/article/houseplants-dont-purify-indoor-air) 科学 [证据支持](https://www.theatlantic.com/science/archive/2019/03/indoor-plants-clean-air-best-none-them/584509/) “事实” [《每日电讯报》](https://www.telegraph.co.uk/gardening/problem-solving/dark-side-houseplant-obsession/) 最近的一篇文章详细描述了一些室内植物实际上是如何危害环境的。以下是需要了解的内容。

### 泥炭是问题所在

简而言之，大多数室内植物生长在泥煤中，马修·艾波将其描述为“一种由腐烂的有机物随着时间的推移而形成的营养丰富的土壤状物质”听起来不错，但是泥炭必须来自某个地方。按:

> 泥炭开采现在被广泛指责为不可持续、破坏环境和排放碳。像煤或石油一样，它实际上是一种有限的资源。它确实能再生，但只能以每年1毫米的速度形成。

尽管植物种植者可能很清楚化石燃料对环境的影响，但艾波说,很少有人意识到他们种植的泥炭也是一种有限的资源。

### 如何选择环保的**室内植物**

虽然公众对泥炭问题的认识远不如种植杏仁的对杏仁奶的环境影响那样广为人知，但有一些证据表明，这种现象似乎正在增长——至少在英国是如此。

具体来说， [哈里特·汤姆森](https://www.harrietsplants.co.uk/) ，一位在文章中采访的艾波园艺家，她宣传并销售无泥炭室内植物，她说，2021年前三个月对她的室内植物 的需求比2020年全年高出200%。

如果泥炭开采的前景让你想成为一个更道德的室内植物购买者，这里有一些方法可以确保你的植物尽可能环保:

1.  购买不含泥炭的植物。这将比逛当地的劳氏百货商店和仔细阅读园艺中心花费更多的时间和精力，所以准备好谷歌一下吧。
2.  用不含泥炭的堆肥重新种植你现在的室内植物。“简单地去掉旧的堆肥，把它们换成不含泥炭的堆肥，”汤普森说——但是也许等到你的植物确实需要换盆的时候，这样你就不会无意中制造更多的废物了。无论哪种方式，确保在移植时对根部温柔。
3.  购买不含泥炭的堆肥。不购买新工厂，而是继续维护现有工厂？下次你买堆肥的时候检查一下标签，找一个不含泥炭的。
4.  种植你自己的室内植物(插条或种子),而不是买新的。需要帮助吗？我们之前已经介绍过 [用插枝](https://lifehacker.com/steal-cuttings-to-save-money-on-plants-1839470340) 种植植物，以及 [用厨余](https://lifehacker.com/how-to-grow-vegetables-from-kitchen-scraps-1842858616) 种植蔬菜。

在 [《每日电讯报》的文章](https://www.telegraph.co.uk/gardening/problem-solving/dark-side-houseplant-obsession/) 中，Thompson注意到室内植物相当于园艺中的快速时尚:“我们都这样做过——你看到一种植物出售，觉得‘很好’，你就不假思索地买了下来。”如果没有别的，从这些角度考虑室内植物——园艺的H&M——可能是有帮助的。
