# 如何使用 Microsoft Edge 找到最佳在线价格

> 原文：<https://lifehacker.com/how-to-find-the-best-online-prices-using-microsoft-edge-1847010075>

微软的 Edge 浏览器已经成为谷歌 Chrome 的主要竞争对手，为用户提供类似的浏览体验，这种体验存在于之外的谷歌的跟踪和广告基础设施中，并且对资源的要求更低。不过，Chrome 的一个挥之不去的好处是它在桌面和移动设备上的内置谷歌购物工具，让你可以轻松地在网上买到最低价——但微软正在稳步缩小差距，推出了 Edge Chromium 版本 91 更新。此次更新为桌面用户增加了 [两个新的购物功能](https://blogs.windows.com/windowsexperience/2021/05/27/new-shopping-features-to-help-you-save-money-for-summer-fun-and-all-your-back-to-school-needs/) :内置价格跟踪器和 Bing 返利集成。

Watch

虽然有许多第三方价格追踪器插件可用于 Edge，但微软的浏览器现在可以在你在线购物时自然地向你显示产品的价格历史。要访问，在任何一家在线零售商上打开产品页面，然后点击蓝色价格标签图标，你会在地址栏的末尾看到。这将打开 Edge 价格跟踪器，它将向您显示一个图表，指示该商品过去的售价以及当前价格的对比情况。“与其他零售商比较”按钮允许您查看该商品在其他网站上的价格。

该菜单还会提醒您是否有在线优惠券或其他折扣代码，或者您的购买是否符合 Bing 返利计划，当您通过 Bing.com 购物标签或从精选在线商店购物时，该计划可为您赢得现金返还奖励。您必须 [注册并启用 Bing 返点](https://www.bing.com/rebates) *和*在购买时使用微软账户登录 Edge，以便利用该计划。

Edge 91 更新中提供了新的在线购物功能，还为 s 提供了其他几个功能和更新，包括将的内存节省能力提高 82%的睡眠标签功能，新的浏览器主题颜色，以及具有增强的个性化内容推荐的全新标签页。进入**设置>关于微软 Edge** 下载更新，或者从 [微软的下载页面](https://www.microsoft.com/en-us/edge) 抓取全新安装。
