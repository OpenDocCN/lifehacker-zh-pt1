# 为什么社交媒体名称游戏是一个安全风险

> 原文：<https://lifehacker.com/that-social-media-meme-might-pose-a-security-risk-1845877978>

在这一点上，这应该是常识，但它们在社交媒体上继续流行的事实表明，很多人需要提醒:停止参与那些要求你提供看似无害的个人信息(你的全名+你长大的街道+你的第一辆车等)的怪异模因。)来生成你的脱衣舞娘名，或者你的色情明星名，或者你的证人保护名。不仅没有人真正关心，而且这些“有趣”的小 [互联网浪费时间的](http://www.namenerds.com/uucn/listofweek/yourname.html) 构成了很大的安全风险。



至于为什么，那*应该*就很明显了。当你试图登录一个安全的在线帐户时，许多这样的测验或愚蠢的互联网连锁信息会问你同样类型的帐户恢复问题。我已经数不清有多少次，我看到一个应用程序或服务要求提供“你长大的街道”、“你出生的城市”、“童年小学”或其他类似的问题，作为其帐户恢复机制的一部分。

是的，这需要攻击者已经知道一些关于你的账户的信息——比如你用于特定服务的电子邮件——并且*也*在社交媒体上跟踪你，看你是否回答了类似这样的问题。尽管如此，自动化工具来做到这一点似乎不是不可能的。如果你公开发布这种信息，你只会把自己置于危险之中。

正如 1Password 写的 [早在 2018 年](https://blog.1password.com/how-social-media-games-put-your-security-at-risk/) :

> *玩游戏，参加小测验，甚至评论“你 X 叫什么？”帖子可以为罪犯提供您的安全问题的答案。也许你把你母亲的娘家姓和你第一辆车的颜色结合起来，发现如果你是一名街头赛车手，你会用菲茨帕特里克·戈尔德的名字。这可能会授权他们访问你的帐户，在那里他们可以窃取你的信息，其中一些游戏甚至走得更远。*

> 有些测验会把你带到一个你可以直接添加信息的网站，或者一个游戏会要求你连接到你的社交媒体账户来访问你的数据。他们获取这些信息，然后说出一个有趣的名字，一个神话中的生物，或者最符合你个性的早餐麦片。然后你可以在你的页面上分享结果，鼓励其他人也这样做。
> 
> 不幸的是，并非巧合的是，他们要求的许多信息与你注册账户时网站询问的安全问题的答案是一样的。如果您忘记了密码或被锁在门外，这些问题可以作为备用。但是如果其他人正确回答了这些问题，他们可以更改您的密码并控制您的帐户。

如果你已经回答这类问题有一段时间了，所有的希望都没有失去。首先，花些时间弄清楚你通常把这些东西放在哪里，如果可能的话，删除它们。如果你在一个特定的社交网络上把你的帖子限制在只有朋友的状态，这不是一个问题，但是如果你只是让你的名字组合在公共领域变得松散，那就更有问题了。(换句话说，你真的应该勇往直前，删除所有你从未看过的旧推文 。)

你还可以检查你的主要账户，确保你已经设置了某种双因素认证，如果有人试图重置你的密码，这应该有望给你增加一层安全保障。此外，我认为大多数主要服务会将密码重置链接(或新密码)直接发送到您与该服务关联的电子邮件地址。所以，至少要确保*你已经用 2FA 和非常强大、独特的密码锁定了你的电子邮件账户。如果您已允许应用程序或服务访问您的电子邮件帐户，但您不再使用它们，请撤销这些权限。*

最后，要知道你永远不必诚实。我不知道为什么人们认为他们有诚实回答安全问题的能力。你不知道。你长大的那条街可能是“辛纳邦”你母亲的娘家姓可以是直接从 [埃隆·马斯克](https://jalopnik.com/elon-musk-and-grimes-named-their-kid-x-a-12-so-heres-1843261276) 剧本里取的。你可以使用一个超级复杂的密码*作为安全问题的答案*，然后 [将答案](https://lifehacker.com/use-your-password-manager-for-security-answers-too-1829498257) 存储在你最喜欢的密码管理器(或者纸笔笔记本，或者其他什么)中。相信我，没人会猜到你的第一只宠物名叫*“3d 4 $ j87 # jdFd！9.12 英镑(9RF32 英镑，84 美元)*

那些在线测验、迷因和其他病毒式内容生成者可能看起来天真有趣，但你在公共环境中发布的任何个人数据可能会在某一天回来困扰你。即使你认为你只是在参与一个友好的玩笑，你的答案可能比你意识到的要多一点——即使是像你的“脱衣舞娘的名字”这样看似愚蠢的事情既然你已经处于警戒状态，也要避免的随机网上投票，来自未知来源的调查，以及其他任何想让你提供信息以换取无意义娱乐的事情。
