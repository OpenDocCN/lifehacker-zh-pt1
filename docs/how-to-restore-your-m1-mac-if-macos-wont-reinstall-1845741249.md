# 如果 macOS 无法重新安装，如何恢复您的 M1 Mac

> 原文：<https://lifehacker.com/how-to-restore-your-m1-mac-if-macos-wont-reinstall-1845741249>

苹果公司的 [第一代 M1 苹果电脑](https://lifehacker.com/do-all-apple-m1-macs-perform-the-same-1845668676) 是华而不实，但一些早期用户遇到了一个奇怪的错误，可能导致新电脑无法使用。在升级到 macOS Big Sur 之前，那些试图在装有 M1 的 Mac 或 MacBo 上执行 macOS 完全恢复的人可能会反复遇到系统停止警告:“准备更新时出错。无法个性化软件更新。请再试一次。”



大部分人不会用恢复工具在全新的系统上擦除重装 macOS。但是如果你这样做了，确保你首先 [更新到 macOS 大苏尔](https://lifehacker.com/how-to-get-your-mac-ready-to-upgrade-to-big-sur-1845644605) 以确保 Mac 恢复工具将正常工作。

那些已经冒险尝试恢复他们的机器的人——即使是出于合法的原因，比如有一个干净的空白版本的操作系统可以使用——可能会担心他们永久地损坏了他们全新的 MacBook。令人欣慰的是，苹果有一些方法来修复“未能个性化软件”的错误。第一种选择可以在家里进行，对大多数人都适用:

*   **从可引导的 USB 驱动器**安装 macOS。这是两种 DIY 方法中比较容易的一种，但它需要另一台 Mac 和一个足够大的空 USB 存储设备来保存安装程序文件。苹果的支持页面可以 [引导你完成](https://support.apple.com/en-us/HT201372) 的过程。

如果这个方法由于某种原因不起作用，你有两个选择:

*   **使用 Apple Configurator 2 来“复活”您的 Mac。**这个过程还需要第二台 Mac 电脑，外加一根 USB 线连接两台设备。有了这些材料， [用苹果的指南](https://support.apple.com/en-ca/guide/apple-configurator-2/apdd5f3c75ad/mac) (希望)复活你的 M1 Mac。
*   **在 macOS 恢复工具中修复终端问题。**如果您没有备用 Mac 或合适的 USB 配件，您可以尝试使用 Mac 恢复工具中的终端命令来解决这个问题。这是一个相当技术性的过程，但如果你 [按照苹果的详细预排](https://support.apple.com/en-us/HT211983) ，应该可以。

如果这些恢复方法都不适合你，那么最后一个选择是让你的新 M1 Mac 在当地的苹果商店或认证维修店进行维修。无论如何，确保你 [联系苹果支持](https://support.apple.com/) 以了解你的选择——特别是那些你可以在家处理的选择，这些将在这些时间里尽可能保证你的安全。
