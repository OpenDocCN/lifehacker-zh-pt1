## **本周集锦**

#### **来自 Nedra Glover Tawwab 的采访:**

关于为什么你不应该对设定界限感到内疚:

> 这并不总是让别人失望，因为有时当我们有不健康的界限时，我们会开始回避别人。我们开始让他们处于“我不想这个人在身边”的状态，不接他们的电话等等。当我们开始实施界限时，我们可能会开始看到他们的行为和关系中的一些变化，我们会更开放地与他们建立更多的联系。所以不总是这种尴尬。这可能是一种他们靠得更近的情况。

关于为什么你应该小心不要设置太多严格的界限:

> 我认为有了严格的界限，有时你会有所有这些规则，比如，“我不和别人这样做。我只做这个。人们不能这样对我。事情是这样的……”你认为它让你安全。但它也让你处于泡沫之中，让人们远离你。不同的人有不同的界限是非常健康的，因为有些人甚至不需要相同的界限。或者你想在这个边界上更加灵活。但是，如果你有这些硬性规定，从不同意某些事情或从不尝试某些事情，那么你可能很难接受新的体验，甚至很难发现在关系中带给你快乐的事情，而这是你甚至都没有想过的。

如何开始认识你自己的界限:

> 我认为模糊的怨恨是一个很好的开始，因为那是一种感觉...只是注意你的感受。所以不管你是生气还是沮丧，你都会感到沮丧。你会说“他们为什么要这么做...?"所有这些都有可能成为新的边界。不要让这些事情溜走，再给他们一次机会。听起来好像有你更喜欢的东西，而不是已经发生的事情。所以去发现那个东西是什么。
