# 这个特性让 AirPods Pro 物有所值

> 原文：<https://lifehacker.com/this-feature-makes-airpods-pro-worth-the-cost-1849601971>

你最喜欢 [AirPods Pro](https://lifehacker.com/21-clever-airpods-pro-settings-everyone-should-be-using-1847590845) 的哪一点？对于我们这些需要在听音乐时保持清醒的人来说，透明模式可能是最好的选择。幸运的是，对我们来说，苹果公司已经将透明模式 s *o* 做得比好得多，仅仅是这个功能就值得挑选一双。



## AirPods 的透明模式如何工作？

如果你没有 AirPods Pro 或 AirPods Max，透明模式会使用 AirPods 上的麦克风输入你周围的声音。合适的话，就好像你根本没有戴耳塞一样:关闭透明模式，声音就会消失，变回你期望的紧密密封的低沉安静。

当你想听音乐或播客但需要保持警惕的时候,功能非常适合,比如当你走在繁忙的街道上时,或者当你在一个房间里时不时有人需要你的注意时。使用透明模式，无需取出耳塞就能轻松进行通话。

但是这种模式从来没有考虑到你周围声音的强度。如果有什么声音向你袭来，比如警笛或喇叭，你的 AirPods 就会直接把那种声音传入你的耳朵。我偶尔也会遇到这种情况，这并不好玩。

## 自适应透明保护您的耳朵

然而 AirPods Pro 2 引入了*自适应*透明。这种透明模式的迭代降低了任何超过 85 dB 的声音的强度，同时保持其他声音的真实音量。这就像你的耳朵里一直有一个混音器，确保你周围的声音是平衡的。几乎没有延迟:苹果声称 [新的 H2 芯片以每秒 48000 次](https://www.apple.com/airpods-pro/) 的速度处理这些声音。

公平地说，苹果并不是第一个提出像自适应透明这样的功能的公司。Bose 的 QuietComfort 耳塞 II 也有类似的功能，但根据 PCMag 的说法， [，它没有苹果的产品开发得好。](https://www.pcmag.com/reviews/bose-quietcomfort-earbuds-ii)

[TechRadar 的 Al Griffin 甚至穿着 AirPods Pro 2 去了一场演唱会](https://www.techradar.com/opinion/i-wore-airpods-pro-2-to-a-loud-concert-and-you-should-too) 。他说音乐会达到了 114.7 分贝的峰值，但是由于自适应透明，他的耳朵再也听不到任何超过 85 分贝的声音。虽然 AirPods 肯定比一副简单的耳塞贵，格里芬认为耳塞抑制了 *太多的音乐*，而他的 AirPods 允许他清楚地听到一切。虽然这不是一次完美的体验，但格里芬表示，他肯定会穿着 AirPods Pro 2 参加他的下一场演唱会。

但看起来你可能不需要买一双新的 AirPods Pro 来使用自适应透明。据报道，苹果的 iOS 16.1 目前正在测试中， [为第一代 AirPods Pro 提供了自适应透明设置切换](https://www.macrumors.com/2022/09/29/ios-16-1-adaptive-transparency-airpods-pro/) ，尽管没有第二代 H2 芯片，而苹果称该芯片为 AirPods Pro 2 提供了该功能。我的猜测是如果该功能实际上在最初的 AirPods Pro 上工作，它不会工作得那么好。也许声音减弱会延迟一些，因为 H1 芯片不能以每秒 48000 次的速度处理声音。尽管如此，如果我能获得自适应透明的大部分好处，而不用花 250 美元买一副新的 AirPods，我会很高兴。

如果苹果*给第一代 AirPods Pro 带来了自适应透明，也许它也会给 AirPods Max 带来自适应透明。我们需要等待 iOS 16.1 全面推出才能确定。*
